# 菲国突发：中国籍女嫌犯涉跨国绑架案落网

🚨 菲国突发：中国籍女嫌犯涉跨国绑架案落网
7月14日，菲律宾警方在帕赛市成功逮捕了一名25岁中国籍女性嫌疑人赵婷，她涉嫌参与一起针对中国牙医的跨国绑架案，并勒索赎金。
据调查，赵婷平时从事加密货币兑换及网络销售，而同日被捕的23岁菲律宾籍男子查理·布班则是一名外卖骑手。两人在赎金交易过程中被证人指认，成为案件的关键线索。
警方现场缴获了65万比索赎金、点钞机及多部手机。目前，两名嫌疑人已被移送菲律宾司法部，面临绑架勒索、违反反洗钱及金融诈骗法等多项指控，并已配合调查。
菲律宾反绑架大队代理局长表示，警方将持续打击绑架犯罪，强化情报收集与跨部门协作，以确保当地社区的安全稳定。
