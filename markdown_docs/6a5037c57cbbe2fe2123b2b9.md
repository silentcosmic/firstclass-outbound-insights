# 越南支付新纪元：9Pay直联Visa，赋能跨境商家掘金市场

9Pay已通过CyberSource与Visa实现直接连接，此举为境外支付服务商和商户进入越南的银行卡支付市场开辟了新路径。
这一进展使9Pay成为越南少数几家拥有符合国际运营标准银行卡基础设施能力的支付中介机构之一。
此次合作将Visa的全球网络与9Pay的本土处理能力相结合，为国际支付网关、支付服务提供商（PSP）及跨国公司提供了进入越南市场的额外通道。
对于境外PSP和商户而言，9Pay提供合规的资金流管理解决方案，使其合作伙伴无需在当地设立法人实体或获取越南支付中介牌照即可开展业务。
此外，9Pay还负责外汇协调、风险管理以及税务咨询服务，其中包括境外承包商税项。
通过消除中间环节，该基础设施有助于降低处理延迟、提高交易成功率并减少购物车弃置现象。
越南的终端用户可以使用越南盾进行支付，且无需承担隐藏的货币转换费用，同时也能使用Apple Pay、Google Pay和Click to Pay等服务。
借由与Visa和CyberSource的直接连接，9Pay拓展了其银行卡处理能力，能够为具有复杂技术要求的行业提供定制化解决方案。“随着数字商业持续跨境扩张，Visa致力于与越南当地合作伙伴携手，提升支付受理能力、安全性及交易体验，以符合全球标准。9Pay通过CyberSource与Visa的直接连接标志着一项重要的进展，有助于强化越南与国际支付生态系统的互联互通。我们相信此次合作将支持企业更有效地触达客户，同时在数字交易中培育信任与便利。”
9Pay的系统支持多种现代商业模式，其中包括带有分账机制的电商平台、大规模数字广告支付、循环订阅服务以及客户或商户发起交易（CIT/MIT）。
其运营流程已获得PCI DSS一级认证，这是全球银行卡支付行业中最高的安全标准。
该公司还支持多种集成方式，涵盖支付链接、重定向、Iframe、API以及点对点API系统。
9Pay表示，其基础设施已为在越南运营的跨国公司提供了大容量数据处理和资金流管理支持，其中包括跨境资金转移业务。
该公司近期凭借其一体化支付解决方案，荣获2026年“Sao Khue奖”，该奖项是越南科技领域一项享有盛誉的荣誉。
9Pay还通过世界知识产权组织（WIPO）在11个主要市场获得了商标保护。
该公司指出，与Visa的直接连接将有助于优化成本，并为寻求在越南数字经济中扩张的全球合作伙伴提供支持。
