# ️ Telegram短链域名突遭暂停：制裁误伤全球用户

⚠️ Telegram短链域名突遭暂停：制裁误伤全球用户
7月14日凌晨，Telegram官方短链接域名 t.me 遭遇突发解析暂停（serverHold状态），导致全球大量频道、群组、机器人及邀请链接无法正常打开，故障持续数小时后恢复。
此次事件的核心原因在于，美国OFAC制裁了一家名为 First VPN Service 的VPN服务商，其制裁名单中包含一个特定的Telegram频道：t.me/FirstVPNService。注册局在执行OFAC合规操作时，错误地将措施扩大至整个 t.me 域名，从而引发了全球性影响。
此事件暴露了Telegram对单一短链接域名 t.me 的高度依赖所带来的脆弱性。这意味着，一旦某个频道因合规问题被针对，相关措施可能误伤整个域名。需明确的是，此次并非Telegram平台本身宕机，仅影响了外部浏览器链接的访问。
