# 谷歌搜索核心交互25年首度巨变：AI重塑搜索范式，解锁商业新机遇

逾二十五年来，谷歌搜索框一直是计算机领域最具标志性的交互界面之一：一个细长的白色矩形，一个闪烁的光标，寥寥数字的输入，以及一连串蓝色链接构成的结果。然而，谷歌近日正式宣告这一传统范式即将落幕。
在其年度发布活动上，谷歌宣布对搜索框本身进行全面革新——每日数十亿次搜索的起点，那个文字输入框，正从一个简单的关键词输入工具，蜕变为一个由AI驱动的动态对话式入口。它将支持文本、图像、PDF文档、视频，甚至可以识别已打开的Chrome浏览器标签页作为输入内容。同时，谷歌还将“AI总览”（AI Overviews）与“AI模式”（AI Mode）功能整合为一个无缝的搜索流程，此举消除了用户此前在传统搜索结果页面与AI驱动体验之间进行选择的障碍。
谷歌搜索业务副总裁兼负责人利兹·里德（Liz Reid）在周一的新闻发布会上表示，这是“自25年前首次亮相以来，我们标志性搜索框迎来的最大升级”。
这一重要发布伴随一系列其他重磅消息，包括全新的Gemini模型、名为“Spark”的个人AI助手、智能购物车以及重新设计的开发者平台。然而，搜索框的重新设计或许是最具深远影响的举措。它清晰地预示着，谷歌正将旗下旗舰产品的未来，从一个用户输入碎片化关键词的场所，重塑为一个用户能与由整个互联网支持的AI系统进行开放式、多模态对话的交互界面。
新搜索框：可扩展、接受文件并引导用户提问
这些变革揭示了谷歌对用户与其核心产品交互方式的根本性预期转变，而这款产品正是Alphabet公司绝大部分收入的来源。
现在，搜索框本身可以动态扩展，以适应更长、更具对话性的查询。旧版界面微妙地鼓励用户输入简洁的关键词串（通常两到三个词），而新设计则邀请用户以详尽的颗粒度充分表达复杂的疑问。此外，它现在直接支持多模态输入。用户可以直接从主搜索界面上传图像、PDF、文件和视频，或从Chrome标签页中拖入内容。此前，部分功能已存在于AI模式中，但需额外步骤才能触达。现在，它们位于主要的入口。
谷歌还部署了一个被其描述为“超越自动补全”的AI驱动查询建议系统。该系统不再仅仅根据热门搜索预测用户可能输入的下一个词，而是帮助用户构建复杂且细致的查询——实质上是引导他们提出AI模式最擅长处理的详细问题。
新搜索框已在所有支持AI模式的国家和语言中立即开始推广。
谷歌整合AI总览与AI模式，打造无缝体验
或许比搜索框本身更重要的是其背后发生的架构变革。谷歌正在统一“AI总览”（AI Overviews）——即出现在传统搜索结果顶部的AI生成摘要面板——与“AI模式”（AI Mode）——谷歌一年前推出的更沉浸式、对话式的搜索体验。
这一整合体验将全面上线移动端和桌面端。用户可以输入问题，在获得传统结果的同时看到AI总览，然后直接进入来回对话的AI模式，提出后续问题——所有这些都无需切换到独立的界面。
里德在新闻发布会上阐释了这一逻辑：新的AI搜索框是“对我们传统搜索框的升级，因此结果将直接引导至主搜索界面，而非AI模式”。她指出，虽然一些高级用户会主动寻找AI模式，但“对大多数用户而言，他们实际上并不想考虑是选择更传统的页面还是AI优先的搜索体验”。
她表示，目标是确保“对大多数用户而言，他们不必考虑去哪里，只需前往他们熟悉的搜索框，就能获得最佳体验”。
十亿用户与翻倍查询量：搜索行为加速变革的证据
谷歌决定重新设计其最重要产品的核心界面，并非凭空而生。该公司在发布会上分享了一组使用数据，揭示了用户行为正以惊人的速度发生变化。
“AI模式”在推出后的第一年内，月活跃用户已突破十亿。自上线以来，“AI模式”的查询量每个季度都在翻倍。“AI总览”作为更轻量级的AI摘要，目前已触达超过25亿月活跃用户。而整体搜索查询量在上一季度创下历史新高——这一数据该公司此前已在其财报电话会议上披露。
谷歌首席执行官桑达尔·皮查伊（Sundar Pichai）将这些数据视为AI功能对搜索使用量具有增益而非蚕食作用的证据。他表示：“当人们在搜索中使用我们的AI驱动功能时，他们会更多地使用搜索。”他还补充道，他“乐见搜索不再仅仅是独立的查询，而更像是一场持续的对话，为用户提供更深刻的洞察，并将其与浩瀚的互联网相连接。”
里德进一步强调：“这不仅仅是人们搜索更多了，更是他们的搜索方式正在改变。他们正以详尽的细节充分表达问题，提出追问，并跨模态进行搜索。”
Gemini 3.5 Flash模型赋能谷歌AI搜索，实现规模化高速响应
在幕后，新的搜索体验由谷歌在本年度发布会上推出的最新AI模型Gemini 3.5 Flash驱动。谷歌将AI模式的底层模型升级至3.5 Flash，以提供里德所描述的“更强大的AI搜索体验”。
Gemini 3.5 Flash是今年诸多发布中的核心驱动力。谷歌声称，它在几乎所有基准测试中都超越了其先前的前沿模型Gemini 3.1 Pro，同时在每秒输出tokens方面，其运行速度比同类前沿模型快四倍。皮查伊将其描述为在“人工智能分析指数”的“右上象限”中“独树一帜”，这意味着它以显著降低的延迟提供了接近前沿模型的质量。
这种速度对搜索而言至关重要。一个迟缓的对话式AI搜索体验，对于每日处理数十亿次查询的产品来说，将是“胎死腹中”。通过将重新设计的界面与兼顾质量和吞吐量优化的模型相结合，谷歌正试图让AI驱动的搜索体验如同旧版关键词搜索一样即时，同时功能大幅提升。
搜索功能突破：实时构建互动视觉与定制迷你应用
重新设计的搜索框也是一套新功能的入口，这些功能将搜索远远推向了基于文本答案的范畴之外。谷歌宣布了所谓的“生成式UI”（generative UI）——即搜索能够根据用户的具体问题，实时动态地构建定制小部件、互动可视化图表，甚至迷你应用程序。
里德在发布会上提供了一个具体例子：用户可以提问“黑洞如何影响时空？”，然后在AI总览中获得一个互动视觉图，将这一概念生动呈现。后续问题将触发系统实时动态生成全新的视觉内容。她解释说，这得益于“我们与谷歌DeepMind团队合作构建的新颖实时代码生成系统”，该系统运行在Gemini 3.5 Flash之上。生成式UI功能将在今年夏季免费向所有人推出。
但谷歌的雄心不止于此。对于持续性任务——如筹备婚礼、组织搬家、追踪健身计划等——用户将能够在搜索中构建可定制、有状态的体验，这些体验由其Antigravity开发平台提供支持。这无需任何编程专业知识。用户只需用自然语言描述他们想要的功能，搜索即可自动构建。这些体验将在未来几个月内推出，首先面向特定用户群体。
全天候网络监控AI代理将融入搜索结果
此次重新设计也为谷歌所称的“信息代理”（information agents）打开了大门——用户可以直接在搜索中配置AI代理，使其24/7监控网络以识别特定条件，并在条件满足时提供综合更新。
例如，用户可以设置一个代理来追踪特定行业的市场动态，并设定具体参数。该代理将创建监控计划，利用实时金融数据，并在条件满足时主动通知用户——通知中包含链接和上下文，以便用户进一步研究。其他用例包括找公寓、追踪运动鞋发售，或监控用户关心的任何主题。信息代理将首先在今年夏季面向特定用户推出。
这些代理位于谷歌在发布会上阐明的一项更宏大的战略转向之中：该公司正全面投入AI系统，这些系统不仅能回答问题，还能主动代表用户执行任务。除搜索之外，谷歌还推出了Gemini Spark，一个运行在谷歌云专用虚拟机上的24/7个人AI代理。它还发布了“通用购物车”（Universal Cart），一个智能的跨商家购物车。同时，它宣布了“代理支付协议”（Agent Payments Protocol），用于代理进行安全采购。此外，其Antigravity开发者平台也扩展成为一个用于构建自主AI代理的完整生态系统。
出版商、广告商及SEO专业人士面临新现实
此次重新设计对围绕旧版关键词搜索和蓝色链接模式构建的庞大生态系统——包括出版商、广告商和SEO专业人士——提出了深刻的问题。
如果用户越来越多地以完整的对话句子而非碎片化关键词来表达需求，那么搜索引擎优化这一整个学科将需要演进。当AI解析的是自然语言意图而非匹配字符串时，关键词密度策略的相关性就会降低。以权威方式解答深度、细致问题的内容将变得更有价值；而那些为两三个词的关键词片段而设计的内容则价值递减。
对出版商而言，其生存岌岌可危。“AI总览”已经能够综合网络信息并直接在搜索结果中呈现，从而减少用户点击进入源材料的需求。新的无缝AI模式整合深化了这种动态：用户现在可以获得AI生成的答案，并在不离开搜索页面的情况下提出多个追问。谷歌一直坚称其AI功能会为出版商带来更多流量，但此次重新设计使得搜索结果页面更具自我包含性，这让此论断面临重新审视。
对广告商而言——他们是谷歌绝大部分收入的来源——从关键词到对话的转变改变了广告定位的计算方式。对话式查询包含更丰富的意图信号，这可能使广告定位更精确、更有价值。但它们也带来了新的模糊性：当用户正在与AI模式进行多轮对话时，广告如何自然地融入其中？谷歌在发布会上并未详细说明其广告模式的变化，但界面结构的转变将不可避免地重塑广告的呈现和衡量方式。
搜索框：不止是产品，更是数十亿人的习惯
谷歌选择重新设计搜索框，而非仅仅在其背后添加新功能，自有其深层原因。搜索框在此时已不仅仅是一个产品元素，它是一种文化符号——少数被几乎所有互联网连接世界所使用的数字基础设施之一。改变它，无疑传递出公司对于计算未来走向的明确信息。
在过去25年里，搜索框训练了数十亿人以关键词思维进行表达——将他们的好奇心压缩成尽可能短的词串。而新搜索框则邀请他们反其道而行之：大声思考，上传他们正在看的内容，提出追问，让AI系统来处理信息压缩。
皮查伊将公司的宏大抱负与一个惊人数据联系起来：谷歌平台现在每月处理超过3.2万亿个tokens，比一年前增长了七倍。该公司预计2026年的资本支出约为1800亿至1900亿美元——大约是四年前310亿美元的六倍——主要用于支持这次AI转型所需的基础设施。当被问及传统搜索的未来时，他直言不讳：“搜索是世界上使用最广泛的AI产品。”
谷歌搜索框中闪烁的光标依然邀请你输入。但在教授世界如何用关键词交流25年后，谷歌现在要求它用句子表达——并押注约1900亿美元，相信这一转变将成功。
