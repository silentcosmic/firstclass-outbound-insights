# 菲律宾“假菲籍”清查升级：老华侨理发途中被捕

🚨 菲律宾“假菲籍”清查升级：老华侨理发途中被捕
菲律宾当局正全面升级对“假菲律宾国籍”及“假出生证明”的清查行动。6月27日，一名在菲生活经商逾30年的中国老华侨，在计顺市一家理发店理发时，被十余名执法人员现场带离，行动迅速且目标明确，引发周边华人社区关注。
消息人士透露，该男子疑因涉嫌“假菲籍”问题而接受调查。由于事发周末，菲律宾相关执法部门尚未公布案件详情及后续处理进展。
此轮清查正值菲律宾政府持续加强对冒用菲律宾身份、伪造出生证明及非法获取国籍等案件的整治。目前，菲律宾移民局、国家调查局等多个执法机构正联合展开行动，长期居留菲律宾的外籍人士及其身份资料已成为重点核查对象。
随着专项行动的深入，预计未来将有更多涉及“假菲籍”和“假出生纸”的案件浮出水面。
