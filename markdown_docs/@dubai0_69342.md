# 巴基斯坦：波音货机坠毁搜救升级，总理下令全速推进

🚨 巴基斯坦：波音货机坠毁搜救升级，总理下令全速推进
巴基斯坦救援人员正在加紧搜寻一架疑似坠毁的波音货机，搜寻工作已在相关水域全面展开。
据市场消息人士透露，巴基斯坦海军与商船均已参与到失踪飞机的搜寻行动中，并有军用飞机提供空中支援。此前，巴基斯坦总理已指示该国有关部门，务必在阿拉伯海域加强搜救力度，并调动一切可用资源，以确保搜寻工作取得进展。
