# 迪拜重金悬赏AI公园设计：重塑未来城市绿地

💡 迪拜重金悬赏AI公园设计：重塑未来城市绿地
迪拜正加速其智慧城市转型步伐。近日，迪拜市政府正式启动一项总奖金高达20万迪拉姆的AI公园设计挑战赛，旨在面向全球顶尖人才，利用人工智能技术对Al Safa 2公园进行前瞻性改造，打造未来城市绿地新标杆。
该项目是迪拜王储谢赫哈姆丹殿下此前公布的三大城市升级计划之一，彰显了迪拜在基础设施建设与技术创新融合上的决心。挑战赛广邀城市规划师、设计师、学生及人工智能创新者参与，共同探索AI在公共空间设计中的无限可能。
参赛者需在8月15日前，通过迪拜市政府官方平台提交设计方案。
