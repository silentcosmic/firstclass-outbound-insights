# 马来西亚破获跨国诈骗：21名中国籍嫌犯落网

🚨 马来西亚破获跨国诈骗：21名中国籍嫌犯落网
马来西亚马六甲警方近期成功破获一起重大跨国投资诈骗案，行动中逮捕了21名中国籍嫌疑人。据初步调查，该团伙专门针对中国公民，通过设立虚假的线上投资项目进行诱骗，从中非法获利。警方在现场查获了包括电脑、手机在内的大批网络设备。
