# 西港星汇海景湾突查：涉毒与无证人员引关注

🚨 西港星汇海景湾突查：涉毒与无证人员引关注
据市场消息，柬埔寨西哈努克省西港市“星汇海景湾”（STAR BAY）大项目于7月12日（周日）发生突发事件。警方对其中一栋大楼的617号房间展开调查，发现涉嫌非法储存、贩运及吸食毒品的行为。
进一步消息指出，该地点还被指秘密窝藏了多名无护照的中国籍人员，并疑似从事网络诈骗活动。据称，此次行动由特定线索引发，现场已有至少两名涉案人员被带走调查。鉴于当前形势，有行业观察人士提醒，在柬无证人员应提高警惕，注意自身合规风险。
