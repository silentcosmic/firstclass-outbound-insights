# 美驻阿联酋使领馆：因安全局势，非紧急人员已撤离

🚨 美驻阿联酋使领馆：因安全局势，非紧急人员已撤离
美国驻阿布扎比大使馆及驻迪拜总领事馆已宣布取消7月13日至15日的所有领事预约，主要原因在于地区安全局势的升级。使馆提醒已预约的民众无需前往，后续将安排重新预约。
目前，美国驻阿联酋的使领馆仍处于“指令性离境”状态，这意味着所有非紧急的美国政府人员已被调离阿联酋境外。因此，使馆方面目前仅能为美国公民提供非常有限的领事协助。
此外，阿联酋境内的所有常规美国签证服务将继续暂停，直至另行通知。
