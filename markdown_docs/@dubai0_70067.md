# ️ 马来西亚入境警示：特定护照或面临严查与遣返

⚠️ 马来西亚入境警示：特定护照或面临严查与遣返
近日有市场消息指出，一名持有中国护照的旅客在从曼谷飞往吉隆坡时，于自助通关环节遭遇异常，随后被马来西亚移民局人员带至审查室。目前该旅客已与使馆取得联系，正滞留遣返室等待进一步处理。
据了解，马来西亚边境管理部门近期对特定人群的入境审查趋于严格。有反馈称，凡持有中国护照且曾有柬埔寨出入境记录的旅客，尤其可能成为重点盘查对象，面临更高的入境风险。
