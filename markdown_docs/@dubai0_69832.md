# 境内“杀猪盘”惊现湖南：团伙被捣毁，5人落网！

🚨 境内“杀猪盘”惊现湖南：团伙被捣毁，5人落网！
近日，湖南衡阳县公安局成功捣毁一处长期盘踞本地的“杀猪盘”电信网络诈骗窝点，抓获5名涉案嫌疑人，有力打击了此类犯罪的嚣张气焰。
经查，该团伙自今年三月以来，在衡阳县内通过网络广泛引流，伪装成单身优质男女与网友建立“恋爱”关系，随后诱导受害人投资其搭建的虚假高回报平台，实施诈骗。
衡阳县警方在接到线索后迅速行动，先在当地居民小区内抓获3名嫌疑人，查获大量作案工具。随后，办案民警跨省追缉，辗转广西柳州、广东肇庆、湖南永州等地，最终将潜逃的两名主犯抓捕归案。
目前，5名犯罪嫌疑人均已被依法刑事拘留。案件正在进一步深挖，警方正全面梳理全国受害群众信息，追缴涉案赃款，并追查上游虚假平台开发及洗钱关联团伙。
