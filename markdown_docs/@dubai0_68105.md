# 新加坡查封逾6亿新元资产，太子集团洗钱案涉国际合作

🚨 新加坡查封逾6亿新元资产，太子集团洗钱案涉国际合作
新加坡警方近日通报，正就一起涉嫌洗钱的案件展开深入调查，该案关联柬埔寨太子集团。两名中国籍男子胡小伟（44岁）和邱伟仁（38岁）被指控涉案，两人在警方去年采取行动前已离境。
目前，警方已对涉案人员的房产、银行及证券户头等金融资产发布禁止处置令，累计查封关联资产逾6亿新元（约合19.16亿令吉）。据悉，新加坡警方正与包括美国在内的国际伙伴紧密协作，共同推进案件调查。
在此之前，警方已逮捕了包括陈友杰在内的三名新加坡籍男子。此外，警方还对一名涉嫌伪造账目和企图欺诈的女子陈秀玲发出了逮捕令。此案仍在进一步调查中。
