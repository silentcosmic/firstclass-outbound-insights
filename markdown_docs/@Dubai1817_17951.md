# ️ 马来西亚巴生启动外籍清查行动，逾200人被带走

⚠️ 马来西亚巴生启动外籍清查行动，逾200人被带走
据市场消息，今日下午马来西亚巴生（Klang）区域疑似开展了针对外籍人员的集中检查行动。现场有大量执法人员进行排查，对外籍人员身份进行核验。
消息称，行动中部分外籍人员被带离现场进行深入调查，目前已有约200多人被带走。截至发稿，马来西亚官方尚未就此次行动发布任何正式通报，具体原因、被检查人员的国籍及是否涉及违法事项仍待确认。
鉴于此，提醒在马工作生活的企业和个人，务必确保护照及签证有效，严格遵守当地法律法规，并积极配合执法部门的正常身份检查，以规避潜在风险。
