# NTT Data引入AI编码平台Cursor，赋能全球软件工程加速旧系统现代化

全球技术服务巨头NTT Data正着手在其遍布全球的软件工程运营体系中，全面部署AI编码平台Cursor。
此举旨在通过将人工智能技术深度嵌入其开发环境，显著加速企业客户传统系统的现代化改造进程。
此次战略合作核心在于，将Cursor的多模型AI代理无缝集成至NTT Data用于代码设计、构建与重构的工程层。
此次部署特别强调企业级治理与控制，包含全组织范围的隐私模式、集中化管理以及严格的策略执行，以确保AI在软件开发过程中得到妥善且规范的应用。“通过我们与Cursor的深度合作，我们将把AI技术置于我们工程与交付模式的核心位置，从而有效加快系统现代化步伐，在规模化应用中提升一致性，并为我们的客户创造更大价值。”
NTT Data首席执行官兼首席AI官Abhijit Dubey阐述道。
NTT Data的初期计划是率先在内部应用此项技术，以优化并缩短其项目交付周期，随后再逐步将其能力推广至与共同服务的客户群体。
这项深度集成旨在协助全球企业在安全、可控的前提下采纳AI技术，同时确保其应用与企业内部既有政策及规章高度契合。“NTT Data正将AI作为工程师现代化复杂系统的核心驱动力。”
目前，NTT Data已在关键工程团队中启动Cursor Enterprise的试点部署，并已制定了全球范围内的推广计划。
为进一步标准化该平台在其各项行业实践中的应用，NTT Data还计划成立一个Cursor卓越中心。
