# 越南Vingroup旗下出行平台GSM注册资本翻番至20亿美元，业务版图显著扩张

越南领先企业Vingroup旗下绿色智慧出行服务平台Green and Smart Mobility (GSM) 近期提交的备案文件显示，其注册资本已从2025年末至2026年6月期间实现翻倍增长。这一举动标志着该平台在资本实力和战略布局上的显著跃升。
根据上周五提交的最新文件，GSM的注册资本已增至54万亿越南盾（约合20.5亿美元）。在新注入的总资本中，94.7%（即51.15万亿越南盾）以越南盾形式出资，剩余部分则被归类为“其他资产”。GSM以其独家采用Vingroup旗下汽车制造商VinFast生产的电动汽车而闻名，这彰显了其在可持续出行领域的战略定位。
这一最新数字较2025年末的25万亿越南盾增长了一倍以上，这意味着GSM在2026年上半年已投入超过10亿美元的注册资本，进一步强化了其市场竞争力。
除了资本层面的显著增长，2026年6月的备案文件还揭示了GSM注册业务范围的重大扩张。在原有的车辆租赁、道路客运与货运、车辆维保等业务线基础上，公司新增了包括软件出版、计算机编程、IT基础设施建设、数据处理、教育支持服务等一系列新兴领域。此举预示着GSM正从单一出行服务商向更广泛的科技与服务综合平台转型。
此外，修订文件还反映了公司在2026年至今的领导层变动。阮国俊（Nguyen Quoc Tuan）被任命为总经理，范秋香（Pham Thu Huong）则担任董事会主席，两人分别接替了阮文成（Nguyen Van Thanh）和潘青龙（Phan Thanh Long）的职务。
阮国俊和范秋香在Vingroup生态体系中均担任核心要职。阮国俊现任Vingroup副总经理，而范秋香不仅是Vingroup的副董事长，亦是Vingroup创始人兼董事长范日旺（Pham Nhat Vuong）的配偶，这进一步凸显了GSM在集团战略中的重要地位。
