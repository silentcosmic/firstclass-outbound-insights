# 泰国破获非法运送中国人链条：8人落网

🚨 泰国破获非法运送中国人链条：8人落网
泰国公路警方于6月25日在武里南府南荣县成功拦截一起非法运送外国人的案件。在24号公路附近设卡监视时，警方发现两辆可疑车辆一前一后行驶，车内载有多名疑似外国公民。
进一步检查确认，车内共有6名中国公民未能出示任何合法入境文件。警方随即控制了这6名中国公民以及两名泰国籍司机。据初步供认，这两名泰国司机受雇从乌汶边境地区接人，并负责将他们送往巴吞他尼府，每趟可获得8000泰铢的报酬。
泰国警方表示，此案中两名泰国人涉嫌协助非法入境外国人藏匿和转运，而6名中国公民则面临未经许可入境、居留及逾期居留的指控。此次行动是泰国警方打击非法运送外国人和跨境犯罪网络的重要一环，后续将深入追查其背后组织和涉案人员。
