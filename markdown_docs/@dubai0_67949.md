# ️ 迪拜网络隐私警示：擅传私照面临巨额罚款

⚠️ 迪拜网络隐私警示：擅传私照面临巨额罚款
近日，迪拜法院对一起网络隐私侵犯案作出判决：一名女子因未经朋友同意，擅自分享其私密照片及聊天记录，被勒令支付85,000迪拉姆（约合人民币16.8万元）罚款。此案再次强调了阿联酋在网络空间个人隐私保护上的严格立场。
事件源于该女子私自截存朋友照片，并在与第三人八卦时，将截图与私密聊天记录一并转发，最终导致内容被进一步公开。受害人发现后，立即报警处理。调查人员通过数字痕迹确认了未经授权的保存与传播行为。
依据阿联酋2021年联邦法令第34号《打击谣言和网络犯罪法》第44条，任何利用技术手段侵犯他人隐私的行为，包括未经同意拍摄、保存、传输、披露或发布他人照片，都可能面临至少六个月监禁及15万至50万迪拉姆的罚款。此判决为在迪拜运营的企业及个人敲响警钟，务必审慎处理数字内容，严格遵守当地法律法规。
