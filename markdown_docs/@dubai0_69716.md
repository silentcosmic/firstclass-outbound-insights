# ️ 华尔街警惕沙特阿联酋裂痕：3万亿投资或受冲击

⚠️ 华尔街警惕沙特阿联酋裂痕：3万亿投资或受冲击
今年中东地区冲突备受关注，然而，在华尔街银行家和私募股权高管的私下交流中，一个鲜少被公开讨论、却可能颠覆全球投资格局的隐忧正日益浮现：沙特阿拉伯与阿联酋之间不断扩大的分歧。
作为海湾地区两大经济巨头，两国共计掌控逾3万亿美元主权财富，其关系走向对全球投资至关重要。高盛、摩根士丹利、贝莱德、Brookfield Corp. 和 KKR & Co. 等金融巨头已在该地区深耕多年，密切关注两国关系进展。
近期，沙特和阿联酋在也门、苏丹及非洲之角等地的内战和危机中支持不同阵营，政治摩擦不断。国际金融机构认为，这些裂痕正构成严峻挑战。早在围绕伊朗的地区冲突加剧之前，两国间的紧张关系已显露端倪：沙特曾试图削弱阿联酋在也门及红海地区的影响力。随后，在伊朗相关冲突最激烈时，利雅得对伊朗采取了相对缓和的态度，而阿布扎比初期则立场强硬。今年4月，阿联酋意外宣布退出欧佩克，令沙特措手不及，也对该组织造成了广泛冲击。
一系列商业摩擦也随之浮现。在也门分歧后不久，一些总部位于阿联酋的公司反映沙特商务签证申请遇到问题。最近几周，亦有企业和个人指出，从沙特到阿联酋的银行转账出现异常延迟。尽管原因尚不明确，但时机巧合加剧了市场对政治裂痕可能蔓延至商业领域的担忧。
这种担忧已促使全球投资银行乃至该地区至少一个政府的官员，开始进行内部评估，以应对经济竞争加剧的局面。受访企业高管普遍不认为会发生直接对抗，但随着两国立场趋于强硬，企业正日益担心未来可能被迫在利雅得和阿布扎比之间做出更艰难的选择。风险管理公司Crownox首席执行官Hussein Nasser-Eddin指出：“越来越多的成熟机构将海湾国家之间的地缘政治分歧视为一个值得关注而非可以忽视的场景。”他透露，一些企业已开始制定应急计划，包括在两国分别建立后勤系统、审查合同中的不可抗力条款，并评估当地关系对另一中心业务的影响。
对此，一位阿联酋官员表示，两国“保持着深厚而长久的经济和商业联系”，经济部尚未收到有关银行转账问题的投诉。沙特央行也声明，该国金融业在健全监管框架下运作，未对特定国家直接设限。
