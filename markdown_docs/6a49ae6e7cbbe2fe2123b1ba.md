# 聚焦亚太能源转型：施耐德电气与Seraya Partners联合启动5亿美元Faraday能源基金

全球能源技术巨头施耐德电气与总部位于新加坡的新一代基础设施投资机构Seraya Partners，近日共同宣布启动“Faraday能源”平台。这一全新能源基础设施平台是双方战略合作的结晶。
Seraya Partners在一份声明中指出，Faraday平台将施耐德电气在全球能源技术领域的领先优势，与Seraya在基础设施资本运作及平台建设方面的专业经验深度融合。Seraya承诺将投入高达5亿美元的股本资金，以支持Faraday平台在亚太地区及其他选定国际市场的业务拓展。
Faraday平台的业务范围涵盖分布式及客户侧能源基础设施项目的开发、融资、所有权管理与运营。
这些项目将以运营支出（OpEx）为基础的模式，为商业和工业设施、电力及电网基础设施运营商以及数据中心提供能源和脱碳解决方案。
依据长期合约安排，Faraday平台将提供与特定供应商无关、具备高度可扩展性且面向未来的能源解决方案和管理服务，助力客户实现向低碳运营模式的转型。
施耐德电气将作为Faraday平台的核心技术伙伴，充分发挥其全球研发能力、本地化实施网络以及长期积累的客户关系优势。
Faraday与施耐德电气将紧密协作，共同发起、构建并执行项目。这些项目旨在将成熟的技术解决方案与基础设施级别的融资能力以及长期的资产所有权相结合。
这种合作模式使客户能够通过一种资本效率高、基于运营支出的结构，并辅以长期投资支持，来获取施耐德电气的先进能源技术。
Seraya Partners将主导Faraday平台的资本运作、治理结构及机构发展，其经验来源于在数字基础设施和能源转型领域成功构建大型基础设施平台的实践。
双方的联合行动，共同打造了一个可扩展的商业模式，有效弥合了能源技术创新与长期基础设施资本之间的鸿沟。
该声明进一步指出，数字化基础设施电力需求的持续增长、工业电气化进程的加速以及企业对脱碳的坚定承诺，共同推动了对可靠且低碳能源解决方案的迫切需求。
与此同时，众多企业也正寻求在保持资产负债表灵活性的前提下，优化其总拥有成本（TCO）。
Faraday平台通过在一个单一框架内整合技术部署、运营专长和长期融资能力，精准回应了市场上的这些需求。
该平台将为长期能源基础设施资产提供资金并持有其所有权，而客户则能从长期合同下可预测的运营支出（OpEx）付费模式中受益，并实现总拥有成本（TCO）的优化。客户正在寻找既能提供卓越技术能力，又能确保长期交付稳定性的可信赖合作伙伴。通过Faraday平台，我们能够将尖端能源技术解决方案延伸至那些同时需要创新与资金支持的客户。
施耐德电气新加坡及文莱区域总裁Yoon Young Kim如此阐述。
与此同时，Seraya Partners的执行合伙人兼首席投资官James Chern指出，企业对能源解决方案的需求日益增长，它们不仅要技术先进，更要具备财务可持续性。Faraday平台赋能客户以“基础设施即服务”（IaaS）的模式，采用先进的能源基础设施。通过将施耐德电气的全球技术领导力与Seraya的长期资本相结合，我们正在构建一个具备长期增长潜力的可扩展平台。
James Chern补充道。
伴随Faraday平台的正式启动，Peter Goh已被任命为该平台的首席执行官，他将负责在目标市场中构建并扩大平台规模。
Peter Goh此前担任施耐德电气全资子公司NaviX Solutions的首席执行官兼董事会成员，他在能源行业拥有超过16年的丰富经验。
