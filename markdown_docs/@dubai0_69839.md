# 柬埔寨总理洪玛奈抵达北京：开启工作访问

⚡ 柬埔寨总理洪玛奈抵达北京：开启工作访问
柬埔寨总理洪玛奈率领的代表团于今天下午抵达北京，正式开启对中国的工作访问。此次访问旨在进一步深化中柬两国的全面战略合作伙伴关系。
中国外交部副部长华春莹、中国驻柬埔寨大使汪文斌等中方高级官员在机场迎接，凸显中方对此次高层互动的重视程度。此次工作访问预计将聚焦双边经济合作、区域发展及共同关心的国际议题。
