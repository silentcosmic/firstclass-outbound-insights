# 新加坡警方六月突击：超1.7万人受查，519人被捕

🚨 新加坡警方六月突击：超1.7万人受查，519人被捕
新加坡警方于今年6月联合多个执法部门，在全国范围内开展了一场大规模综合执法行动。此次行动共检查逾1.7万人，其中1776名涉嫌违法人员被带回调查，最终519人被依法逮捕。超过5600名执法人员参与了全岛1300余次突击行动，重点打击非法就业、卖淫、交通违法、电子烟及移民相关犯罪。
在多项重点打击领域中，娱乐场所成为此次行动的焦点之一。警方突击检查了上十字街一处KTV及Cecil Street的另一家娱乐场所，共逮捕了包括多名持旅游签证非法就业的女性在内的34名女性和1名男性。这些人员涉嫌违反《妇女宪章》、《外国人力雇佣法》及《移民法》，部分场所还被发现存在消防安全隐患。行动中，一名女子因涉嫌袭击执法人员被捕，另有两人因持有电子烟接受调查。
此外，警方亦针对非法赌博和无牌按摩院展开了专项打击。在三巴旺路和义顺的行动中，17名涉嫌非法赌博者被捕，查获超过4740新元现金。同时，警方在Jalan Mas Puteh附近突击了两家无牌按摩店，逮捕8名涉嫌卖淫及非法就业的女性。在打击交通违规方面，陆路交通管理局（LTA）与警方合作，在盛港和榜鹅查获25宗违法行为，并扣押15辆违规使用的个人代步工具（PMD）。值得注意的是，警方还逮捕了3名涉嫌秘密会社（黑社会）成员的青少年。
交通警察在酒驾专项执法中，对66名驾驶员进行了酒精检测，共逮捕15名酒驾司机。新加坡警方行动部总监陈良俊强调，警方将持续与各执法机构紧密合作，对各类违法犯罪行为保持高压打击态势，以维护社会治安，并重申任何违法者都将受到严肃处理。
