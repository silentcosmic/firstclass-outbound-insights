# 越南特大卖淫案：网红佳丽涉案，Telegram跨省组织，单次收费1.5亿越南盾

🚨 越南特大卖淫案：网红佳丽涉案，Telegram跨省组织，单次收费1.5亿越南盾
越南警方近日成功侦破一个利用Telegram私密群组进行跨省组织卖淫的特大中介团伙。该团伙主要在胡志明市和河内等主要城市活跃，并在河静省一家高档度假村内当场抓获涉案人员。
初步调查显示，该团伙在多个私密群组中发布了约50名女性的信息，其中不乏被包装成选美冠军、亚军乃至知名TikTok网红的人员。其运营模式高度线上化，从招揽客户、中介联络到交易安排，几乎全部通过加密通信软件完成，并利用USDT及银行账户进行资金结算。
据警方披露，客户需支付50万至200万越南盾不等的会员费方可浏览群组内女性资料。团伙提供的服务价格从单次500万越南盾起，而持续2至5天的“陪游”服务，最高收费可达1.5亿越南盾，彰显其高端定位。服务内容包括安排女性从河内前往越南南部多省进行“陪游”，每日收费约1800万越南盾。
警方已初步认定阮氏草（Nguyễn Thị Thảo）为该团伙的主要组织者及多个Telegram群组的管理员，并逮捕了另一名活跃中介人员。目前，两人已被立案并面临“卖淫中介罪”的刑事指控，警方正持续追查其他涉案人员。
