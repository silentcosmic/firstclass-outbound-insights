# 新加坡金管局设立未来金融研究院，引领AI与代币化技术迈向深度应用

新加坡金融管理局（MAS）正着手设立一个未来金融研究院，旨在推动金融机构的人工智能（AI）和代币化项目突破试点阶段，迈向更广泛的实际应用。
该研究院初期将聚焦于AI和代币化技术，汇聚金融机构、科技企业、研究人员及金融科技参与者，以期在整个行业内实现更广泛的采纳与部署。
未来金融研究院将依托金管局此前在AI和代币化领域已主导的多项创新举措，其中包括MindForge人工智能风险管理工具包、PathFin.ai、“守护者计划”（Project Guardian）以及“兰花计划”（Project Orchid）。
金管局表示，该研究院将成为新加坡金融创新下一阶段的统筹协调机构。
金管局将继续负责制定政策和监管框架，而研究院则将与业界紧密合作，致力于降低技术采纳壁垒，并支持不同规模机构的技术部署。
其核心工作内容包括构建一个知识共享中心，汇集经过验证的用例、部署指南及解决方案供应商信息。
此外，研究院还将通过设立创新工坊（innovation garage）来促进产业协作，提供沙盒环境以测试可编程货币、代币化资产及AI驱动工作流等前沿技术，并针对代理式AI（agentic AI）和可编程合规（programmable compliance）等领域开发相应的工具包。
研究院将设立一个理事会，其成员将涵盖金管局、主要金融机构、科技企业及学术界的代表。同时，研究院也将吸纳具备丰富行业和技术经验的实践者加入。
关于未来金融研究院的战略规划与治理结构的更多细节，预计将于今年晚些时候对外公布。
