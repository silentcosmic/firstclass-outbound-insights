# 印尼破获跨国“订购婚姻”团伙：3名中国籍嫌犯遭遣返

🚨 印尼破获跨国“订购婚姻”团伙：3名中国籍嫌犯遭遣返
印尼雅加达苏加诺-哈达国际机场移民局近日通报，执法部门成功侦破一起以跨国婚姻为幌子、涉嫌人口贩卖的“订购婚姻”犯罪案件。此案中，3名中国籍涉案人员已被抓获、遣返回中国广州，并被列入印尼入境黑名单，禁止其再次入境。
案件线索始于6月4日，一名印尼女子FNR在办理护照时，因陈述疑点被移民官员深入询问。FNR随后承认其真实目的地是中国，并计划通过印尼籍中介AN的安排与一名中国男子结婚。
移民情报执法部门迅速介入调查，锁定并于6月12日在苏加诺-哈达国际机场抓获该团伙主要协调人——中国籍男子CS。随后，执法人员于6月17日在丹格朗一处公寓逮捕了另外两名中国籍涉案人员FG和CX，同时发现3名印尼籍女性SA、PY、PO，经核实均为该“订购婚姻”案件的受害者。
调查显示，该团伙长期以婚后优渥生活条件为诱饵，诱导印尼女性参与跨国婚配。有意迎娶印尼女性的中国男子需向团伙支付约6万元人民币（约合1.5亿印尼盾），其中约2万元人民币（约合5000万印尼盾）作为彩礼支付给女方家庭，其余款项则由犯罪团伙支配，用于办理护照、签证、未婚证明、住宿及出境等相关费用。
6月26日，CS、FG、CX三名涉案人员已被依法驱逐出境。苏加诺-哈达国际机场移民局局长加利赫强调，印尼政府对利用跨国婚姻实施人口剥削、违反移民法律法规的行为始终坚持“零容忍”态度，并将依法严厉打击相关犯罪活动。目前，印尼移民部门仍在追查涉案中介AN及其他相关人员，以进一步深挖背后的跨国犯罪网络。
