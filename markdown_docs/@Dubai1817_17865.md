# 马来西亚：4人涉线上赌博案，全员否认控罪获保释

🚨 马来西亚：4人涉线上赌博案，全员否认控罪获保释
马来西亚吉打州双溪大年近日一宗备受关注的案件进入司法程序。四名男子被控涉嫌共同经营网络赌博中心，并于7月9日在当地推事庭出庭应讯。面对指控，四名被告均当庭否认有罪，并获法庭批准以每人6000令吉及一名担保人保释候审。
根据控状，这四名年龄介于25至61岁的被告（包括一对兄弟）被指控于2026年4月7日，在双溪大年甘榜拉惹花园的一处住宅单位内非法经营公开赌场。他们被控触犯马来西亚《1953年公开赌场法令》第4（1）（c）条文，一旦罪名成立，将面临最高5万令吉罚款及3年监禁。
法庭最终采纳了辩方降低保释金的请求，并定于2026年8月11日再次过堂，以推进后续司法程序。
