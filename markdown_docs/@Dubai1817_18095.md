# ️ 芭堤雅公园不雅事件：公共形象受损，当局介入调查

⚠️ 芭堤雅公园不雅事件：公共形象受损，当局介入调查
近日，泰国芭堤雅一则拍摄于公共公园的不雅视频在社交媒体上广泛传播，引发当地民众高度关注。视频内容显示，一对男女疑似在公共场所做出不当行为，目前当地相关部门已介入调查。
事件发生地点位于芭堤雅考帕塔纳克山上的查洛姆帕拉基亚特公共公园，该公园是当地居民及游客日常散步、慢跑和休闲的重要场所。流传视频画面显示，涉事男女在一处石质野餐桌旁行为不雅。
公园保安维奈透露，事发当天凌晨3时20分左右，他在巡逻时曾看到一名男子独自坐在石桌附近，当时并未察觉异常。直至同事向他展示网络视频后，维奈才确认视频场景正是自己巡逻区域。他表示，过去公园内偶尔有情侣在偏僻区域逗留，但从未发生过如此直接在公共区域的不雅行为。
事件曝光后，大量网友呼吁有关部门加强夜间巡逻力度，并在公园偏僻区域增加监控设施，同时尽快查明涉事人员身份并依法处理。目前，视频中男女双方的身份、国籍及具体情况尚未公布，当地警方正展开进一步调查。
