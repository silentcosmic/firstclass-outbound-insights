# 泰国特大电信诈骗案告破：中籍团伙涉案超千万

🚨 泰国特大电信诈骗案告破：中籍团伙涉案超千万
近日，泰国网络犯罪调查局（CCIB）召开新闻发布会，通报侦破一起涉案金额高达1100万泰铢的跨国电信诈骗及洗钱大案，并新增抓获三名中国籍团伙成员。
该案源于2025年11月，一名19岁男生在视频通话中遭诈骗团伙冒充公职人员恐吓，被迫开启母亲保险箱，窃取金条、黄金首饰、钻石等全部贵重财物，直接损失高达680万泰铢。
警方调查发现，该团伙分工明确，从实施恐吓、线下收赃、转运贵金属到变卖兑现，最终通过USDT虚拟货币进行层层洗白，形成完整的犯罪链条。此前，该团伙已累计洗白218,350枚USDT，折合逾720万泰铢。
自2025年11月首次抓获收赃嫌疑人并追回350万泰铢赃物起，警方持续扩大侦查。至2026年6月，执法人员在突击搜查中起获380万泰铢现金、100公斤银粒及奔驰S500e豪车等，使全案扣押财物估值突破1100万泰铢。本次新增落网的三名中国籍嫌犯，主要负责现金兑换数字资产及大额USDT转账洗白，虽审讯中否认涉案，但转账记录和监控证据已锁定其罪行。
目前，该案已有四名核心嫌疑人归案，包括三名中国公民和一名泰国本地人员，均被指控合谋洗钱罪名并移交司法起诉。警方正持续追捕其余在逃人员，并全力追查全部非法所得资产。
