# 斯里兰卡监狱暴乱：致4死35伤

🚨 斯里兰卡监狱暴乱：致4死35伤
当地时间7月6日，斯里兰卡尼甘布监狱发生严重暴乱，已造成4人死亡、至少35人受伤。死者包括一名监狱狱警、两名监狱看守以及一名在押囚犯。
目前，斯里兰卡有关部门已介入调查此次事件原因，并加强监狱安全管控。所有伤者已被送往当地医院接受救治，事件具体细节仍待官方进一步公布。
