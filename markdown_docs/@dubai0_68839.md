# 泰国破获AI深度伪造赌博案，年流水25亿

🚨 泰国破获AI深度伪造赌博案，年流水25亿
泰国网络犯罪调查局（CCIB）于7月1日在暖武里府宣布，成功侦破一起利用AI深度伪造技术进行宣传的特大网络赌博案件。该非法赌博网站年资金流水高达25亿泰铢，其宣传视频冒充知名乡村歌手兼演员哈伊·阿帕蓬（Hai Apaporn Nakhonsawan）。
调查显示，一家支付网关公司涉嫌为该平台提供资金收付服务，目前正接受进一步调查。此案凸显了AI技术滥用带来的新型犯罪风险，以及支付服务提供商在合规监管方面的挑战。
