# 伊朗革命卫队：已对美军85个目标发起导弹无人机袭击

🚨 伊朗革命卫队：已对美军85个目标发起导弹无人机袭击
今日，伊朗革命卫队对外宣称，已对中东地区85个美国军事目标发动了协调一致的导弹与无人机袭击。据声明，受攻击目标包括巴林境内的美国第五舰队、科威特的阿里·萨勒姆空军基地以及萨勒曼港。
