# 泰国百亿比特币洗钱案：跨国灰产网络浮出水面

🚨 泰国百亿比特币洗钱案：跨国灰产网络浮出水面
泰国特别调查厅（DSI）近日深度扩大了一起重大非法“比特币挖矿”案的调查。该案表面伪装成窃电挖矿，实则揭露了一个由华人主导、涉嫌利用电诈和网络赌博进行跨国洗钱的庞大灰产网络，其年资金流水惊人地超过100亿泰铢（约合20亿元人民币）。
警方在行动中缴获了逾6390台加密货币挖矿设备，仅给泰国电力局造成的损失便高达近10亿泰铢。调查显示，该团伙分工明确，不仅有泰国公职人员提供便利，还雇佣缅甸籍人员每日在泰国银行取现高达3000万至5000万泰铢。
此案已串联至美国，美国特勤局已冻结该网络核心中国籍负责人王某名下价值超过1780万美元的数字资产。目前，泰国警方已对4名中国籍出资人和4名缅甸籍成员发出逮捕令，跨国清查工作仍在持续推进中。
