# 洪森主席将访华：中柬高层互动再升级

⚡ 洪森主席将访华：中柬高层互动再升级
中共中央对外联络部于23日发布消息，柬埔寨人民党主席、参议院主席洪森应中国共产党邀请，定于6月25日至27日对中国进行正式友好访问。此次高层互访被视为进一步深化中柬两国执政党及国家间战略伙伴关系的重要举措，预计将聚焦双边合作与地区发展议题。
