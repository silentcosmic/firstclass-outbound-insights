# 中国商人斯里兰卡机场被捕：涉千万卢比香烟走私案

🚨 中国商人斯里兰卡机场被捕：涉千万卢比香烟走私案
一名中国籍商人在斯里兰卡卡图纳亚克国际机场被海关拘捕，原因系其涉嫌非法携带大量外国品牌香烟入境。此次查获的香烟总价值高达约1300万斯里兰卡卢比（约合人民币30余万元）。
此事件再次警示出海企业及个人，在国际商业活动中务必严格遵守目的地国家的法律法规，以规避潜在的运营及个人合规风险。
