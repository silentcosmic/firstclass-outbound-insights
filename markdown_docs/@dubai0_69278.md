# ️ AI深伪助长迪拜“王子”骗局：一年血汗钱被卷走

⚠️ AI深伪助长迪拜“王子”骗局：一年血汗钱被卷走
近日，一名菲律宾籍家政工玛丽亚不幸遭遇高科技网络诈骗，被冒充“迪拜王储”的骗子卷走一年积蓄。骗子利用先进的AI深度伪造技术，在视频通话中呈现出逼真的虚假形象，辅以甜蜜情话，诱骗玛丽亚支付了10万比索（约合1625美元），声称用于办理结婚证和“皇室会员卡”。
玛丽亚的案例揭示了日益猖獗的“迪拜假王子”骗局生态。犯罪分子利用迪拜王储在社交媒体上的巨大影响力，通过伪造账号、发布虚假信息，并借助AI技术进行情感诈骗。调查发现，部分诈骗活动与尼日利亚犯罪团伙有关。专家警告，AI技术正使实时视频深度伪造日益逼真，未来非面对面交流的真伪将难以辨别，极大地增加了网络诈骗的识别难度。
这类骗局并非孤例。据全球反诈骗联盟估算，去年全球消费者因各类诈骗损失高达4420亿美元，其中婚恋诈骗占比不小。面对AI驱动的新型诈骗威胁，国际上已开始出现自发的防骗团体，呼吁提高警惕，防范这类利用高科技手段进行的金融欺诈。
