# 美国施压柬埔寨：加速追责电诈主谋

🚨 美国施压柬埔寨：加速追责电诈主谋
近日，美国驻柬埔寨大使馆临时代办亚历克斯·齐特尔与柬埔寨国家打击网络诈骗委员会秘书长蔡西那烈举行会谈，聚焦打击针对美国公民的网络诈骗犯罪。
齐特尔在会谈中对柬方过去一年在突击搜查行动中取得的成果表示认可，但同时敦促柬埔寨政府加速行动，深挖并追究这些诈骗园区背后实际运营者的法律责任，以有效遏制此类跨国犯罪活动。
