# 中菲联手捣毁千万级电诈团伙，6名嫌犯被遣返

🚨 中菲联手捣毁千万级电诈团伙，6名嫌犯被遣返
据中国驻菲律宾大使馆7月16日消息，中菲两国执法部门近日紧密协作，成功捣毁一个盘踞在菲律宾的特大电信诈骗窝点。该团伙涉案金额高达近千万元人民币，6名嫌疑人悉数落网，均系中方通缉在逃人员，涉嫌电信诈骗、洗钱及掩饰、隐瞒犯罪所得等多项重罪。
行动中，执法人员在窝点现场查获大量作案工具，包括电脑、手机、假印章等，同时缴获手枪、毒品等违禁品。在中国驻菲使馆的全力协调下，菲律宾移民局已于7月14日将这6名嫌疑人押解遣返回中国。
中国驻菲使馆表示，将持续深化与菲方的执法合作，坚决打击跨境违法犯罪，切实保障海外同胞的生命财产安全。
