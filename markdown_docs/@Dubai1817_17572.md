# 柬埔寨出海公司遭员工实名举报：拖欠工资引发威胁！

🚨 柬埔寨出海公司遭员工实名举报：拖欠工资引发威胁！
近日，一份来自前员工的实名举报在社群中流传，矛头指向一家曾活跃于东南亚的出海公司。该员工指控公司长期拖欠工资，并在其回国后彻底失联。
据爆料人透露，其曾服务于一个名为“财通俊壹暴富团队”的股票盘操作团队，业务地点涵盖波呗、维加斯、白宫及马来西亚新园区等地，并在金边永旺二商场附近设有“外盘打野点”，声称为公司创造了可观价值。然而，在工资屡次被拖延后，爆料人目前已无法联系到公司任何负责人。
面对僵局，该员工发出严厉警告，表示将不惜承担法律后果，持续爆料公司各级领导的照片、藏身点及“打野点”信息。爆料人还披露了多位核心人员的身份信息，包括老板方伟（外号阿伟）、后勤财务方鸿（外号草人）、总监梁宝（外号阿宝），以及金主林（外号老虎弟），均据称为福建籍人士。
