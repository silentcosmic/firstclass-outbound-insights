# 老挝突袭网络诈骗窝点：80名外籍嫌犯落网

🚨 老挝突袭网络诈骗窝点：80名外籍嫌犯落网
老挝占巴塞省公安厅外国人管理警察部门于7月8日上午10时采取行动，根据群众举报，对巴江县会盆村一处涉嫌利用计算机系统实施违法犯罪活动的窝点进行了突击检查。
此次行动共控制了80名外籍嫌疑人，并现场查获了80台用于非法活动的电脑设备。此案再次凸显了东南亚地区在打击跨境网络诈骗方面的持续努力和面临的挑战。
