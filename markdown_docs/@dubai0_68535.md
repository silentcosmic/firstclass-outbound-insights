# 泰韩联手突袭曼谷诈骗窝点，7名韩籍嫌犯落网

🚨 泰韩联手突袭曼谷诈骗窝点，7名韩籍嫌犯落网
泰国移民局调查人员近日与韩国警方合作，在曼谷展开了一场跨国联合执法行动。此次行动成功突袭了两处联排别墅，捣毁了一个涉嫌从事跨国诈骗的犯罪团伙，并逮捕了7名韩国籍嫌疑人。
