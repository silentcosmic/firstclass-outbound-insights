# 阿联酋首条客运铁路试运营，中企参与共建

⚡ 阿联酋首条客运铁路试运营，中企参与共建
阿联酋首条客运铁路——阿提哈德客运铁路已于6月30日启动试运营，初期开通阿布扎比至富查伊拉线路，标志着阿联酋正式迈入铁路客运时代。
中国驻阿联酋大使曾继新指出，阿提哈德铁路车站项目是中阿两国高质量共建“一带一路”的标志性成果，为深化双边全面战略伙伴关系及各领域务实合作树立了成功典范。
