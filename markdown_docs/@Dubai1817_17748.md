# 泰国失业男为9000泰铢成“人头账户”被捕

🚨 泰国失业男为9000泰铢成“人头账户”被捕
泰国中央调查局（CIB）近日逮捕一名43岁清莱籍男子，该男子因失业，为赚取9000泰铢报酬，出售了3个银行账户给网络贷款诈骗集团。目前，他已被控涉嫌共同诈骗及向计算机系统输入虚假信息等罪名，并涉及两起法院通缉案件。
警方调查显示，该诈骗团伙通过Facebook等社交平台发布虚假“线上贷款”广告，诱骗急需资金者下载假贷款APP。两名受害人因此被骗取共计13万泰铢，其中一人在申请5万泰铢贷款后，被以“资料填写错误”为由，陆续支付各项费用累计4万泰铢，但始终未获贷款。另一名受害人则在支付9万泰铢保证金后，贷款未到账，诈骗分子随即失联。
嫌疑人供述，其在2021年失业期间，受人介绍以每个账户3000泰铢的报酬开设并交出银行账户及ATM卡。他否认诈骗指控，但警方已掌握其作为“人头账户”参与诈骗的证据。
