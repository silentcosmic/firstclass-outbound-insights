# 缅甸诈骗集团南迁：泰缅边境现异动，当地居民忧心

🚨 缅甸诈骗集团南迁：泰缅边境现异动，当地居民忧心
近期观察显示，缅甸掸邦南部的勐奈镇出现了大量华裔面孔，他们频繁采购大量建材与食品，且不议价，物资流向并非中国境内，而是转向泰缅边境的特定区域。此现象引发当地居民对北部诈骗集团（俗称KK园区）势力南迁的深切担忧。
据当地居民透露，这些异常采购活动与去年“1027行动”后，大量诈骗团伙从缅北转移至掸邦南部及泰缅、老缅边境地区的趋势高度吻合。此前，缅甸军方与佤联军已在不同区域逮捕并移交了数百名涉嫌网络诈骗的中国籍人员，凸显了该区域的治理挑战。当地民众呼吁武装力量及时介入清查，以避免类似诈骗活动再次泛滥，对区域稳定与社会秩序造成冲击。
