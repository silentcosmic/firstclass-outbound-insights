# 星展银行与三星证券深化合作，共拓全球财富管理新格局

知名金融机构星展银行（DBS）近日与韩国三星证券（Samsung Securities）签署谅解备忘录，确立了一项旨在深化财富管理领域的战略合作关系。
此次合作旨在充分发挥三星证券在韩国资本市场的深厚根基，并结合星展银行覆盖全球的财富管理平台，从而显著拓宽双方客户的跨境投资渠道。
两家领先的金融机构将围绕四大核心领域展开深度协作。
首先，星展银行的客户将能够通过三星证券提供的专业投资解决方案，便捷地进入充满活力的韩国市场。
相应地，三星证券的客户亦将享受到星展银行所提供的一系列多元化、全球化的财富管理产品与服务。
此外，双方还将探讨联合提供专业的财富管理咨询服务，以全面满足客户日益复杂的全球资产配置需求。
更进一步，两家机构将积极推动知识共享与联合创新，特别是在人工智能（AI）技术应用以及财富管理前瞻性理念等领域，共同探索行业发展的新路径。
鉴于高净值客户对专业财富增长建议的需求持续攀升，此次战略合作的根本目标在于为两家机构的客户创造更广泛的跨境投资机遇。“此次合作标志着一个重要的里程碑，它将韩国投资者与全球市场紧密相连，同时也将全球的优质机遇带给我们的客户。”
星展银行首席执行官谭素珊（Tan Su Shan）指出，亚洲正迅速崛起为全球财富增长的核心引擎。她强调，当前客户对资产的跨市场无缝管理有着越来越高的期望。“通过融合三星证券在韩国资本市场的领导地位与星展银行的全球化平台，我们能够为双方客户提供在亚洲乃至全球范围内都鲜有匹敌的高水准投资机会。”
谭素珊进一步表示，财富管理仅是此次合作的起点。星展银行的长远目标是，通过这一伙伴关系，最终能将其全面的企业及财富服务体系惠及所有客户。
