# ️ 迪拜华人圈警示：前中企员工涉嫌卷款潜逃

⚠️ 迪拜华人圈警示：前中企员工涉嫌卷款潜逃
近期，迪拜华人社区出现一起涉嫌金融欺诈的警示事件。据多名受害者反映，一名曾在中国公司任职、并与当地华人社群有联系的个人，在向多人借款后失联。
其中一名受害者透露，其个人损失达1200美元。综合市场信息估算，该涉事人员通过类似手段累计借贷金额可能已达15000美元。此事件提醒在迪拜的华人及出海人员，务必谨慎处理个人借贷关系，防范潜在的财务风险。
