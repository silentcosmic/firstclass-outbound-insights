# AI时代，出版商如何确保内容权威性与文化价值：新框架SAIL的实践

对于出版商而言，其内容被AI聊天机器人引用已不足以满足其需求。他们更希望AI的回应能够准确体现其编辑判断，并承载其所服务社群的文化特质与价值观。
一项名为SAIL（Standardized Agentic Intelligence Ledger，标准化代理智能账本）的创新方案应运而生，旨在同时实现这两个目标。它不仅在AI抓取出版商内容时提供补偿，更确保AI的输出能遵循出版商在自身内容中贯彻的文化标准。
该框架由AI内容授权平台Next Net，联合Sundial媒体与技术集团共同设计。Sundial旗下拥有《Essence》、《Refinery29》和《Afropunk》等多家知名出版品牌。
Sundial首席执行官柯克·麦克唐纳（Kirk McDonald）表示，SAIL是一个数字记录系统，能够追踪AI解决方案如何使用出版商内容，以及如何将其与其他来源混合。其核心目标是在AI引用高品质出版商内容时，同时结合那些虽不严谨、但仍具文化相关性的用户生成内容时，保护前者的固有价值。
麦克唐纳先生将此框架描述为一种更具协作性的替代方案，以取代与AI供应商进行一次性授权谈判的模式，或者避免因未经授权的内容抓取而采取诉讼的极端手段。
出版品牌历经数十年构建了与受众的真实联结，在AI时代，它们理应拥有维护这种联结的机制。
例如，麦克唐纳指出，Sundial旗下品牌被业界尊为报道非裔社群新闻的顶尖资源。特别是在涉及非裔女性议题时，《Essence》杂志常被AI回应引用为最具权威的媒体来源。因此，Sundial希望在AI应用日益普及的背景下，确保这种权威性得到保护，并获得AI合作伙伴的恰当估值。
麦克唐纳警示，AI存在误读或扁平化内容的风险，尤其是在专业制作内容与缺乏文化敏感性、可能助长有害刻板印象的用户生成内容混合使用时。
为应对这一风险，SAIL提供了一座技术桥梁，连接AI供应商与出版商内部已预先进行文化敏感性训练的AI工具。
例如，Sundial已开发出其专属的市场自动化与研究智能平台、内容创作者平台以及面向消费者的聊天机器人界面。麦克唐纳介绍，在训练这些工具时，Sundial吸收了其旗下出版品牌一个世纪以来积累的文化专业知识。
通过将内部系统与第三方AI工具集成，SAIL将Sundial深厚的文化理解直接融入到第三方的训练模型中。
Next Net首席执行官富兰克林·里奥斯（Franklin Rios）表示，Next Net的平台运行在与许多第三方AI供应商相同的NVIDIA基础设施上，这使得它能更便捷地促成这些连接。他透露，自今年一月以来，Next Net一直在“向量化”Sundial的内容目录，即将内容转化为AI模型可理解的软件语言。
然而，技术只是SAIL框架的一部分。麦克唐纳强调，该框架更旨在让出版商在AI工具训练方式上拥有话语权。
他直言不讳地指出：“如果我不能参与到成果的设计中，那么我就只是一个内容的供应商，而主导权将完全由机器掌控。”
那么，这个框架究竟如何为出版商带来收益呢？
SAIL会追踪AI系统每次利用出版商内容来生成答案的记录，其账本系统与IAB Tech Lab的CoMP AI框架兼容，后者采用“按查询付费”模式，即只要出版商内容被用于AI回应，出版商即可获得报酬。
Next Net将从支付给出版商的收入中抽取分成，具体条款根据个案协商确定。
麦克唐纳重申，SAIL的意义远不止于为出版商的内容付费。他解释道，出版商投入大量时间与精力，才得以成为其社群中的权威声音，而这种价值往往被自动化系统所忽视，SAIL也为此提供了补偿。
由于该框架指示AI系统将某些出版商视为更具权威性的来源，这些媒体更有可能在AI回应中被引用，进而，如果用户点击引用链接，便会为原始来源带来引荐流量。
麦克唐纳认为，尽管引荐流量具有一定价值，但它远不及在AI驱动的内容发现机制中拥有发言权更为重要。
他指出，出版商真正需要的是一份“源文件”，能够展示AI答案是如何构建的，包括它从出版商那里提取了哪些资源（例如文章、视频或现场活动洞察），以及还有哪些其他来源共同塑造了AI的回应。
这种可追溯性赋予了出版商权力，让他们能够对AI输出内容进行最终的认可。麦克唐纳表示：“我认为没有人希望自己的作品以一种他们无法认同的方式被修饰或扭曲。”
麦克唐纳先生拥有市场双方的丰富经验，包括曾任GroupM北美首席执行官、AT&T时代Xandr首席商务官以及PubMatic总裁（与出版商紧密合作）。他补充说，这种模式也为媒体所有者与AI供应商之间日益紧张的对抗关系提供了一个受欢迎的替代方案。
他亲身见证了自动化和利益冲突如何导致不同利益相关者创造的价值得不到充分回报。因此，他坚信，出版商保护其内容权威性和价值的最佳途径是与AI公司协作，而非对抗。
麦克唐纳总结道：“如果你开始进行‘大卫与歌利亚’式的争论，我其实并不想打败巨人。我更希望我们能以合作者的身份坐到谈判桌前，共同设计一种体验，解决任何意外产生的偏见，并引入能让所有参与方都负起责任的技术。”
