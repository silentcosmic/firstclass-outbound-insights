# ️ GCC秘书长谴责伊朗袭击巴林科威特

⚠️ GCC秘书长谴责伊朗袭击巴林科威特
海湾合作委员会（GCC）秘书长贾西姆·穆罕默德·阿尔布达维（Jasem Mohamed Albudaiwi）近日发表强烈声明，谴责伊朗对巴林和科威特发动的袭击。阿尔布达维秘书长指出，这些袭击行为反映出伊朗持续破坏地区及国际社会为促进安全、和平与解决危机所做的努力，严重威胁了海湾地区的稳定。
