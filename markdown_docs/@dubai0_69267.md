# 阿联酋彩票再爆3000万迪拉姆头奖得主！

💰 阿联酋彩票再爆3000万迪拉姆头奖得主！
阿联酋彩票近日公布了第二位3000万迪拉姆头奖得主的身份。这位幸运儿是来自阿布扎比的52岁印度居民Sunil Kumar Sadasivan。
这是自阿联酋彩票今年初推出“Lucky Day”新赛制以来，第二次派发如此高额的头奖。此前，首位3000万迪拉姆得主是26岁的尼泊尔籍保安Tayab Khan，他曾与四位朋友合买彩票并共同分享了这笔巨款。
