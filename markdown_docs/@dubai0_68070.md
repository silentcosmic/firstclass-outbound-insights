# ️ 泰国严打“保关”服务：1.3万人被拒，这类游客风险最高

⚠️ 泰国严打“保关”服务：1.3万人被拒，这类游客风险最高
泰国移民局近日启动紧急调查，严查网络上再度活跃的“保关”服务。此类服务声称能协助旅客应对移民审查、确保顺利入境，并根据旅客风险等级收取高昂费用，从普通旅客的4500泰铢至高风险旅客的12000泰铢不等。
据披露，有柬埔寨、老挝、越南、马来西亚等周边国家旅行记录者，以及15天内频繁往返泰国的旅客，被系统性认定为高风险对象。市场消息显示，泰国已拒绝约1.3万人入境，凸显了此次整治的决心。
泰国移民局已明确表态，已成立专项调查委员会，一旦发现执法人员参与其中，将依法追究刑事及纪律责任。此举对依赖此类服务的旅客及相关产业构成重大风险。
