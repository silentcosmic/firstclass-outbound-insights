# Paymonade获MiCA牌照，加密服务版图扩展至欧洲30国

新加坡背景的Paymonade公司现已成功获得单一MiCA牌照，获准在欧洲30个市场提供受监管的加密服务。
该牌照由列支敦士登金融市场管理局（FMA）颁发，其运营实体为Damoon Technology (Europe)，即Paymonade。
这一批准使得Paymonade能够在整个欧洲经济区内，通过单一可跨境授权（passportable authorisation）提供加密资产服务。
Paymonade的核心业务是为加密货币交易所、金融科技公司及支付服务商提供法币与加密货币之间双向兑换的基础设施。
截至2026年上半年，该公司报告的年化交易量已达到18亿美元。“轻度监管的加密时代正在走向终结。在行业内绝大多数公司尚未获得此类牌照之际，我们能成功取得这一许可，充分展现了我们所构建机构的强大实力。”Paymonade表示，“我们相信，下一代数字资产领域的领导者将是那些能够将创新与监管信任相结合的企业，我们决心成为其中一员。”
Paymonade计划在未来12个月内将其欧洲团队规模扩大一倍，并力争在2027年中期将年化交易量提升至60亿瑞士法郎。
