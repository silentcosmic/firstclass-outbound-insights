# 金边突发：疑犯急抛电脑销毁证据，数据安全引关注

🚨 金边突发：疑犯急抛电脑销毁证据，数据安全引关注
近日，柬埔寨金边森速区警方在一次公寓楼排查行动中，遭遇一紧急情况：有人试图通过将电脑从高处抛下以销毁潜在证据。警方随即在楼下收集了该物品。
此前，金边警方在其他执法行动中也曾遇到类似情况，有当事人试图通过将电脑抛入水中来销毁证据。然而，与传统物品不同，电子设备即使被物理破坏，其内部数据仍有可能被专业技术手段恢复，这给执法部门的证据收集工作带来了新的挑战。
