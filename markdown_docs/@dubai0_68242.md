# 中国商人涉加密货币洗钱，泰国国际通缉高管

🚨 中国商人涉加密货币洗钱，泰国国际通缉高管
泰国特别调查厅（DSI）近日对中国籍商人王益承发出逮捕令，指控其涉嫌通过非法加密货币挖矿，为跨境网络赌博及“杀猪盘”诈骗团伙进行大规模洗钱活动。
此案牵涉的非法挖矿活动据称盗窃电力，价值高达2800万美元。此外，美国司法部此前也曾将其列为数字资产欺诈嫌疑人，并于2023年扣押了其名下账户中来自美国受害者的50万美元加密货币。警方披露，王益承曾利用其亚泰经济交流总会常务副会长的身份，结交泰国政警界高层。
目前，王益承已被指控触犯《计算机犯罪法》等多项罪名，并已潜逃出境。泰国当局正联合国际刑警组织，在全球范围内对其展开追捕。
