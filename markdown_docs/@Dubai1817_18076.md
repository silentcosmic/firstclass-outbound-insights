# ️ 柬埔寨前将军绑架勒索案上诉再延期

⚠️ 柬埔寨前将军绑架勒索案上诉再延期
曾震惊柬埔寨社会的前国家警察总署反毒局副局长、二星中将万高索（Van Kosal）涉嫌绑架、勒索案，其上诉程序近日出现新进展。原定于7月14日上午在金边高等法院进行的二审开庭，因辩护律师临时申请，法院已决定推迟审理，新的开庭日期将择日公布。万高索及其四名下属目前仍被羁押，等待下次庭审。
回顾案情，此前金边初级法院一审认定万高索涉嫌策划绑架、非法拘禁、勒索财物及洗钱等罪名，判处其17年有期徒刑；其四名下属因参与加重绑架勒索，被分别判处15年有期徒刑。五名被告均不服一审判决并提出上诉，案件遂进入二审程序。此案因涉及前高级警官打着缉毒名义进行绑架勒索，持续引发社会对司法公正及公权力滥用的关注。
