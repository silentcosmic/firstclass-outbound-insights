# 印度侨民在迪拜抽中一套房：12年租客变房东

⚡ 印度侨民在迪拜抽中一套房：12年租客变房东
近日，一位在迪拜生活了12年的印度籍金融从业者阿迪蒂（Aditi），在参与当地购物抽奖活动“Win Your Home in Dubai”后，幸运赢得一套全新的单间公寓，从租客一跃成为房主。
据悉，阿迪蒂女士此前在阿尔巴沙（Al Barsha）的Damas门店购物，成为该全城促销活动的第三位幸运获奖者。此案例再次凸显了迪拜市场消费活动的活跃与居民获得意外惊喜的可能性。
