# 洞察东南亚：新加坡倡议东盟深化AI应用，警示数据壁垒与主权误区

新加坡数字发展和信息部长约瑟芬·张（Josephine Teo）近日指出，凭借日益完善的数字基础设施、充满活力的年轻技术人口以及不断深化的区域合作，东南亚在加速人工智能（AI）普及方面拥有独特的优势。
在雅加达举行的数字峰会上，张部长强调东盟在全面部署AI所需的关键支柱领域取得了显著进展，这些支柱涵盖了基础设施建设、劳动力能力提升、治理框架构建以及国际协作。
“整体而言，东南亚在广泛应用AI方面具备良好条件，” 她明确表示。“然而，这一进程并非水到渠成。”
张部长进一步阐释，预计到2025年至2030年间，该区域的数据中心容量将实现三倍以上的增长，同时，不断扩展的海底光缆网络正持续增强区域内的数字互联互通能力。
除了有形的基础设施，她着重强调了软件与政策层面的关键作用，并以SEA-LION为例——这是一个开源的东南亚语言模型，其下载量已超过20万次，充分展现了区域在软件生态方面的活力。
她观察到，各行各业的企业正加速将AI技术融入日常运营：金融机构利用AI进行欺诈检测，而制造商则将其应用于供应链优化，以减少停机时间。
诸如Kata.ai等公司也正在积极开发AI工具，旨在提升印尼语环境下的客户服务体验。
张部长对今年东盟轮值主席国菲律宾在推动微型、小型和中型企业（MSMEs）采纳AI方面的努力表示赞赏。
她补充道，东盟基金会的目标是为10万家中小微企业提供必要的数字化工具支持。
张部长还特别提及了来自基层的创新案例，包括文莱和印度尼西亚大学生开发的AI驱动平台，这些平台分别用于支持痴呆症护理和提供实时洪水预警服务，体现了AI在社会福祉领域的广阔应用前景。
然而，张部长也警示，东盟在AI发展道路上面临两大潜在挑战：限制性的数据政策以及对技术主权的狭隘理解，这两者可能阻碍区域AI雄心的实现。
她强调数据是“AI的生命线”，企业需要获取高质量数据并建立可信赖的跨境数据流机制，才能开发出可靠的AI系统，并实现其业务在区域内的规模化发展。
她指出，虽然保护个人信息和国家安全需要必要的保障措施，但过度的限制可能会扼杀创新，尤其对于那些缺乏资源来应对复杂合规要求的小型企业而言。
张部长将拟议中的《东盟数字经济框架协议》（DEFA）视为一项关键举措，旨在确立共同的数字贸易规则，并促进可信赖的跨境数据流通。
她呼吁：“无论是雅加达、万隆、河内还是曼谷，我们都应携手合作，共同帮助这些小型企业蓬勃发展并实现规模扩张。”
关于AI主权问题，张部长告诫各方，不应将其局限于对整个AI技术栈的完全所有权，这包括芯片、模型、数据和应用程序等所有环节。
她认为：“这种观点虽然可以理解，但对大多数国家而言，既不现实也无益。” 她同时指出，实现完全所有权的成本高昂，且AI供应链本身就具有全球性特征。
相反，她建议各国应聚焦于三大优先事项：确保能够有效治理和部署AI以造福公众；在技术合作与采购决策中保持自主性；并通过研究机构、开发者社区和投资网络来强化国内AI生态系统。
展望未来，张部长透露，新加坡将于明年接任东盟轮值主席国，并计划在现有倡议基础上，进一步推动企业和员工更广泛地采纳AI技术，投资于语言模型和治理工具包等共享数字公共产品，并强化区域内的数据流通与AI治理机制。
她强调：“这不仅仅是新加坡的议程，更是整个东盟的共同议程。” 她补充道，尽管区域内存在多样性，但合作才是东盟力量的源泉。
“只要东盟成员国步调一致，我们就能取得更丰硕的成就。”
