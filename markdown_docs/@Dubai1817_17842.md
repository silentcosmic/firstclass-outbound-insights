# 泰政府重申大麻仅限医用，反批反对党曾提娱乐试点

🚨 泰政府重申大麻仅限医用，反批反对党曾提娱乐试点
泰国政府发言人拉莉达于7月8日，就人民党党魁纳塔蓬对政府大麻政策的批评做出强硬回应，重申其核心立场：大麻仅限于医疗和健康目的，绝不支持娱乐化。拉莉达指出，非法走私大麻属犯罪行为，不应归咎于政府政策，并强调执法部门正加强打击力度。
她同时反驳称，人民党前身前进党曾提议设立娱乐大麻试点区，质疑其当前批评的立场矛盾。目前，政府正加速推进新大麻管制法案，旨在全面监管种植、生产、销售及使用，并严惩违规行为。泰国卫生部已着手整顿全国大麻商店，对无证或违规店铺进行取缔。
