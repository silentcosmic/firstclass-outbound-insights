# 阿联酋气温逼近50°C：极端酷热引关注

🔥 阿联酋气温逼近50°C：极端酷热引关注
6月20日，阿联酋全国气温普遍攀升至接近50°C，其中艾因（Al Ain）地区的斯韦汉（Sweihan）在当地时间下午12点45分录得全国最高气温49.4°C。这一极端高温对当地居民生活及户外作业带来了显著挑战，预示着今年夏季可能将比往年更为炎热。
