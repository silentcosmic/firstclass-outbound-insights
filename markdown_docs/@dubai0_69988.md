# 迪拜辟谣市中心爆炸报道：官方称不实

🚨 迪拜辟谣市中心爆炸报道：官方称不实
针对路透社此前发布的关于迪拜市中心传出“巨响”并引发爆炸传闻的紧急警报，迪拜媒体办公室已迅速出面驳斥，明确指出该报道“不实”。
迪拜媒体监管机构同时强调，公众应仅以官方渠道获取准确信息，并严正警告所有媒体及个人，切勿传播未经证实或带有误导性的谣言，以维护信息环境的清晰与稳定。
