# 金边突袭：13名外籍人员涉网络诈骗被捕

🚨 金边突袭：13名外籍人员涉网络诈骗被捕
据最新消息，7月9日下午，柬埔寨金边雷西胶区警方采取突击行动，在当地一处名为“维曼金边城”的住宅区内，成功抓获13名涉嫌网络诈骗的外国籍人员。
当地政府透露，此次行动并非仓促之举。警方此前已进行了数日的严密监视，在掌握充分证据后，才联合多部门对目标场所的5处房屋实施了突查。现场证据进一步证实，该场所确系网络诈骗活动的窝点。
