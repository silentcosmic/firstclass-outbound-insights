# 东帝汶端掉90人电诈窝点：租30年合同掩盖犯罪

🚨 东帝汶端掉90人电诈窝点：租30年合同掩盖犯罪
7月13日，东帝汶国家警察（PNTL）采取统一行动，成功捣毁位于帝力科莫罗和利基萨乌尔梅拉片区的两处跨境电信诈骗及非法网络赌博呼叫中心窝点，共拘留90名外籍嫌疑人。
此次行动采取双点同步突击，其中利基萨窝点抓获76人，帝力窝点抓获14人。涉案人员中，88人为印尼籍公民，另有2名中国大陆籍人员分别在两处窝点被查获。
警方深入调查发现，该犯罪团伙为掩盖其电信诈骗和非法网络赌博的真实目的，长期对外伪装成车辆仓储及汽车销售门店。他们不仅向当地房东提供虚假租赁理由，甚至不惜与业主签订长达30年的租赁合同，每月支付高达20750美元的租金，并聘请当地律师协助办理正规手续，试图通过复杂的商业包装来掩盖非法活动。
