# ️ 杭州小伙险失百万，警惕网络‘干爹’投资陷阱

⚠️ 杭州小伙险失百万，警惕网络‘干爹’投资陷阱
近日，杭州一名22岁的生物科技公司财务人员小陈（化名），在社交平台遭遇网络诈骗，险些被骗走103万元人民币。骗子通过扮演‘干爹’角色，以海外经商、生活奢华为诱饵，逐步取得小陈信任后，诱导其进行所谓‘投资’。
警方提示，网络上突然出现过度关心并承诺带你赚钱的人，往往是诈骗的信号。凡是要求通过购物卡、礼品卡、陌生链接或非正规平台进行充值投资的，基本都是典型的诈骗手段，公众应保持高度警惕。
