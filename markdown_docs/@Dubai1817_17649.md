# ️ 开云公司陷劳资纠纷：被指涉嫌福利倒扣与汇率压榨

⚠️ 开云公司陷劳资纠纷：被指涉嫌福利倒扣与汇率压榨
近日，有匿名市场消息指出，出海企业开云公司内部面临严重的劳资争议。爆料员工指控公司在发放世界杯福利后，通过高额罚款制度变相收回福利，并涉嫌在内部汇率上进行不当操作，导致员工实际收入受损。
据称，部分员工在为世界杯项目加班长达两个月、每日工作12小时后，获得约七千元人民币的福利。然而，公司随后却以各种制度名义实施罚款，动辄数百U（美元），而员工月薪仅约1800U，罚款金额与薪资严重不符，引发员工强烈不满。
此外，爆料还指出，开云公司在每月工资结算时，内部汇率固定为7.2，而同期市场汇率约为6.66。这一汇率差额导致员工每月损失逾千元人民币，一年下来相当于白白损失了一个月的工资。员工普遍认为公司管理层存在严重腐败和压榨行为，并呼吁潜在求职者警惕该公司。
