# 映界科技：以时空感知构筑具身智能新基石

尽管人形机器人在舞台上展现出令人惊叹的运动灵活性，甚至能在马拉松赛道上疾驰，然而一个不容忽视的现实是，它们往往“能动”却“不善劳作”。这背后，前者彰显的是精密的运动控制实力，而后者则亟需机器人对物理环境形成深层理解，包括准确记忆物体方位与推断空间关联。
这种感知能力上的瓶颈，几乎是当前所有具身智能领域参与者共同面临的挑战。为此，成立不到三个月的映界科技（MirrorSpace）正致力于填补具身智能产业链中的这一关键空白。这家公司由一支平均年龄仅24岁的博士团队创立，专注于具身机器人视觉空间感知系统的算法与硬件研发，其目标是成为未来世界模型时代的核心基础设施供应商。
“我们观察到，市场上目前缺乏相对成熟且完善的空间感知综合解决方案，”映界科技创始人金依力指出，“大多数传感器制造商仅将雷达、摄像头等设备的原始数据提供给本体厂商，由其自行处理。我们的愿景是提供一套完整的空间感知能力，从而弥合机器人在开放场景应用落地中的空白。”
今年前半段，映界科技成功完成了由松禾资本与奇绩创坛共同领投的千万元种子轮融资。松禾资本管理合伙人汪洋曾评价，空间智能是推动机器人从单纯的自动化设备向具备物理AI能力的智能体演进的关键基础设施层，他尤其看重映界科技团队“超越其年龄的深邃判断力”。而奇绩创坛则青睐于该团队所具备的“从学术研究到产业实践的完整认知链条”，以及其产品在“工业应用场景中已显现的真实付费潜力”。
映界科技创始人金依力持有麦吉尔大学博士学位，在空间智能与多媒体系统领域耕耘多年。两位联合创始人胡凯源和段西泽亦为麦吉尔大学博士生，分别专注于多模态空间场景、全息视频通信及智能系统工程研究。值得一提的是，该创始团队的导师刘学教授，不仅是加拿大工程院院士、麦吉尔大学终身教授及IEEE Fellow，更曾担任三星北美研究副总裁和Tinder首席科学家，在空间智能与机器学习交叉领域积累了近二十年的深厚经验。
金依力透露，映界科技即将发布其“软硬一体、即插即用”的感知模组原型——MirrorSense。尽管该产品尚未正式面市，却已收到多类客户的明确合作意向，公司预计到2026年订单总额将突破千万元。
在多模态感知技术层面，MirrorSpace超越了传统的视觉与激光雷达（LiDAR）方案，创新性地将温度这一非视觉特征纳入空间表征矩阵。通过在边缘端实现异构传感器的原始数据级深层前融合，该系统能确保RGB、深度信息与热度数据的无损时空对齐，进而为机器人在夜间、逆光、低纹理及遮挡等严苛环境中，提供更为稳定可靠的空间感知输入。
区别于传统空间感知侧重静态三维建模，MirrorSpace在进行空间建模时，巧妙地将“时间”作为第四维度引入。此举旨在构建时空连续感知流，有效解决了传统静态三维建模在处理动态场景时所面临的“盲点”问题。
“我们的核心工作是赋能机器人实现对空间的更深层次、更精准理解，并在此基础上积累时空数据，最终使其能够形成类似人类的记忆功能，”联合创始人胡凯源阐释道。他进一步解释，通过Mirror-Mind决策中枢，系统能够将融合多重模态后的4D高斯表征与视觉语言大模型（VLM）进行深度对齐，从而将语义信息高效压缩存储，形成可供大语言模型调用的空间记忆。
以机器人制作咖啡为例，借助映界科技的技术，机器人能够依据此前对杯子、咖啡机和桌面之间空间关系的观察与记忆，自主制定行动方案，并精准执行“前往厨房取回已加热的杯子”等任务。即便在杯子被遮挡的情况下，机器人依然能够结合历史观测数据和对空间关系的推理，准确判断其具体位置。
当前，映界科技的业务模式主要分为两大类：其一是直接向机器人本体制造商提供空间感知解决方案，旨在替换或显著增强机器人原有的感知模块。目前，公司已与多家本体厂商确立了合作意向。其二是与行业集成商及平台服务商携手进行项目交付，目前正在积极探索的落地应用场景涵盖电力巡检、石油化工等传统工业领域，以及海外高端地产、商业零售和全息全景体育等新兴泛娱乐产业。
在持续深耕现有应用场景，并成功打通量产交付闭环的同时，映界科技还积极与高等院校合作，共同推动空间智能领域行业标准的制定。其长远目标是使自身的技术成为未来所有机器人在物理世界中进行交互的标准配置及核心基础设施。
除实际应用落地之外，映界科技还规划了一项更为宏大的长期布局：MirrorSense模组本身即被设计为一款高效的时空数据采集器。随着该模组在更广泛场景中实现规模化部署，映界科技将持续汇聚真实物理世界中的多模态时空数据，为下一代“世界模型”的训练构建坚实的数据基础。“空间感知不仅仅是机器人的‘眼睛’，它更是世界模型理解物理世界的关键入口，”金依力强调。
金依力坦率承认，对于映界科技这支年轻的团队而言，当前面临的核心挑战在于工程化落地能力尚需提升。团队需要持续在硬件制造、供应链管理及实际项目交付等多个层面积累宝贵的实战经验。
金依力表示：“在公司发展的早期阶段，我们选择不直接参与一些大型客户的招投标和实际项目交付，正是为了将核心精力聚焦于世界模型的研究和产品的持续研发。我们深知，最终客户选择的关键在于算法的强大与产品的易用性，而我们在科研能力上具备显著优势，因此我们希望首先将产品与技术精益求精。”
尽管在当前的视觉空间感知领域，市场竞争激烈，既有奥比中光、海康等上市公司凭借成熟且可量产的感知硬件占据一席之地，亦有宇树科技等本体厂商积极投入自研感知解决方案，然而年轻而充满信心的映界科技，坚信能够在此竞争格局中构筑起自身独特的技术壁垒，并最终经受住市场的严格检验。
