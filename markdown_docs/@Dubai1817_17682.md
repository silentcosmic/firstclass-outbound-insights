# 马来西亚“壹汇环球”特大骗局：50亿资金蒸发，3万人受害

🚨 马来西亚“壹汇环球”特大骗局：50亿资金蒸发，3万人受害
市场消息指出，一个名为“壹汇环球”（GHC）的马来西亚籍外汇投资项目涉嫌跨境诈骗，导致大量中国投资者蒙受巨额损失。该项目被指控为典型的“资金盘”模式，涉案金额已超50亿元人民币，受害者达3万余人。
据披露，“壹汇环球”的幕后操盘手为马来西亚籍男子Wong Khoay Shen，外号“黄董”。该平台自2017年前后开始面向中国投资者推广，以高额固定收益、团队奖励及外汇交易为幌子，诱导投资者不断投入资金并发展下线。项目初期宣传攻势猛烈，吸引了中国多个省市的投资者。
然而，从2021年左右，平台问题开始集中爆发，包括提现困难、资金冻结、收益无法兑付，大量投资者的本金血本无归。尽管项目表面声称进行外汇投资，但其运作模式实则依赖新入场资金填补前期投资者的收益，一旦资金链断裂，便迅速崩盘。目前，该平台的运营地点被指设在马来西亚吉隆坡蒲种一带。
当前，受害者们正积极整理银行流水、聊天记录、宣传资料等证据，期待推动中马两国警方正式立案调查，并追查涉案人员及资金流向，以期最大程度挽回损失。
在此提醒广大投资者，凡是打着“稳赚不赔”、“高收益低风险”、“拉人头获高额佣金”旗号的境外投资项目，务必提高警惕。在不了解其金融牌照、公司背景及资金去向的情况下，切勿盲目投入。
