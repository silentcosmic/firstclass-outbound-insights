# 具身智能新突破：前自动驾驶与机器人专家创立纽娲机器人，获数千万天使轮融资，打造“世界通行模型”

本刊获悉，具身智能领域新锐企业纽娲机器人近期成功完成5000万元天使轮融资，由蓝湖资本领投，不同资本及共青城朴一投资参与跟投。在此前的两个月，纽娲机器人已获得一笔由一支头部创新加速基金领投的种子轮融资。
纽娲机器人（下称“纽娲”）成立于2026年2月，在短短半年内，已迅速获得多家财务及产业基金的投资。公司创始人杨睿刚博士，在3D视觉、仿真技术、自动驾驶与机器人领域深耕多年，曾担任国内领先科技公司自动驾驶与机器人实验室主任，并曾任自动驾驶卡车领军企业嬴彻科技CTO，成功推动L3级自动驾驶重卡实现量产落地。其丰富的产业与创业经验，结合其目前上海交通大学特聘教授的身份，为纽娲的发展奠定了坚实基础。
业界领先企业普遍认同“分层解耦”是具身智能发展的关键路径，但在解耦的具体维度上则各有侧重。国际上的先行者多倾向于按推理时间尺度进行解耦，例如将“慢思考”的规划层与“快反应”的控制层加以分离，以提升系统效率。国内的多数新兴力量则更侧重于按能力类型进行拆分，如将运动、操作、交互能力独立建模，或将感知、学习、规划、执行等核心功能分层解耦，也有企业致力于模拟类脑分区结构。
基于对行业趋势的深刻洞察，纽娲选择从机器人最基础的“通行能力”切入，前瞻性地提出了“具行智能”（Embodied Mobility Intelligence）的概念。其核心在于通过自主研发的“世界通行模型（WTM）”，赋能机器人像人类一样，在复杂多变的人类社会环境中实现自主移动、精确导航、自然交互，并最终高效完成各项任务。
这项核心能力不仅适用于人形机器人，还能够兼容四足机器人、AGV（自动导引车）、无人配送车等多种移动机器人平台。创始人杨睿刚博士强调，无论是社区配送、园区物流，亦或是文旅度假服务，所有这些应用场景的底层都指向一个共同的关键挑战——机器人如何在人类环境中安全、高效且优雅地完成任务。这正是纽娲“世界通行模型（WTM）”所具备的根本价值。
业界当前有大量世界模型的构建聚焦于视觉生成和内容生成，而纽娲则独辟蹊径，认为机器人真正缺乏的是对物理规律的深刻理解及其指导行动的能力。公司将自身定位为“具行智能”大脑的提供者，构建了一套从高保真仿真、到数据生成、再到模型训练的完整技术闭环体系。
具体而言，在渲染层面，纽娲通过先进的三维重建与AIGC（人工智能生成内容）技术，确保空间的一致性。在仿真层面，则结合显式三维结构与自主研发的物理引擎，精确表达物理规律。数据层面，公司则充分利用前两层的能力，持续生成可交互、物理正确且三维一致的高质量合成数据，从而为世界通行模型的训练构建起一个强大的数据飞轮。
值得关注的是，纽娲自主研发的高保真物理仿真引擎SimWeaver，相较于市场主流仿真平台，在生成速度上实现了3倍的提升，同时将仿真到真实世界的迁移误差大幅降低了20%。在多项柔性体操作任务中，其零样本（Zero-shot）成功率达到了91%，展现出卓越的技术实力。
在运动控制（Locomotion）能力方面，纽娲的自研模型已能稳定应对55度空心楼梯等极具挑战的复杂场景。杨睿刚博士阐释，这项能力的实现，得益于视觉感知与本体感知的深度融合。与当前部分主要依赖运动控制策略进行“盲爬”的现有方案不同，纽娲的系统具备理解哪些区域可承载、哪些区域不宜触碰的能力，从而在复杂环境中显著提升了机器人的运动稳定性与安全性。
导航能力（Navigation）方面，当前众多物流机器人仍高度依赖高精地图进行路径规划。纽娲则致力于推动无图或轻地图导航方案的落地，仅凭小区平面图或通用导航服务提供商提供的基础地图数据，即可实现室内外、平地与楼梯之间的三维自主导航，极大地拓展了机器人的应用边界和部署灵活性。
此外，“社会行为合规性”（Social Ethic）被纽娲视为其世界通行模型的另一关键要素。自动驾驶系统主要在结构化道路环境中运行，而机器人未来将更直接地融入人类生活空间。如何在电梯、商场、园区等公共场所中遵循人类默认的行为规范，是机器人实现规模化应用不可回避的核心挑战。目前，纽娲正重点攻克电梯场景中的社会行为学习，通过海量数据训练机器人理解并执行“先下后上”、“排队等待”、“主动礼让”等社会规则，确保其在公共环境中的行为得体合规。
创始人访谈摘录杨睿刚博士指出：“数据闭环对于机器人和自动驾驶而言，其本质是共通的。从数据采集、标注，到模型迭代，整个逻辑体系是完全一致的。自动驾驶领域成熟的工程化体系，如CI/CD（持续集成/持续部署）、仿真测试、灰度发布等环节，已经发展得非常成熟和严格。我们过去在实现L3级重卡量产的过程中，这套体系得到了充分验证，并非纸上谈兵。这些宝贵的经验可以直接应用于机器人领域。”
他进一步阐述，广义而言，自动驾驶旨在解决结构化道路上的自主通行问题。而世界通行模型（WTM）则将这一能力向前推进了一大步——将其从结构化的高速公路环境，延伸至楼道、电梯、门禁等更为复杂、更具“人性化”的室内外环境。虽然其基础框架有所继承，但无疑在难度上迈上了一个新的台阶。杨睿刚博士强调：“真正需要从头构建的，是‘具身操作’这一核心能力——即机器人如何与物理世界进行真实交互，例如开门、按电梯、搬运物品等。这在自动驾驶领域完全没有对应的经验可循，必须从零开始。我们目前的策略是充分利用团队在三维重建和物理仿真方面的深厚积累，通过合成数据来大幅加速这部分的训练进程，这也是我们实现快速突破的关键优势所在。”针对世界模型的理论框架，杨睿刚博士引用李飞飞教授的观点：“李飞飞教授将世界模型划分为三层：渲染器（Renderer）、仿真器（Simulator）和预测器/规划器。其中，渲染器主要处理观测数据，实现‘观测输入到观测输出’的转化，例如VLA模型可以直接从图像进行规划；仿真器则在隐空间中进行未来状态的预测；而真正的生成式世界模型，则致力于融合渲染与仿真，通过三维对齐和视频动力学习来驱动规划决策。”杨睿刚博士阐述了纽娲在此领域的独特路径：“我们首先将世界模型构建为一个高效的数据生成器。在渲染器层面，我们利用三维重建与AIGC技术，确保生成数据的空间一致性；在仿真器层面，则在显式三维结构的基础上，叠加自主研发的SimWeaver仿真器，以确保牛顿力学等物理规律的正确表达。通过这种方式，我们能够生成海量、可交互、物理正确且空间一致的高质量数据，进而用于训练我们的世界通行模型（World Travelers Model / WTM）。”他进一步强调：“我们并非追求一个大一统的端到端世界模型，而是采用一种更具工程化可行性的方案：将渲染器与仿真器进行解耦并协同工作，首先打通数据生成的飞轮，继而让我们的‘大脑’（即WTM）在海量高质量数据中习得真正的‘世界通行能力’。这正是我们持续推进的核心技术路线。”
未来规划与投资方洞察谈及未来的发展规划，杨睿刚博士透露：“2026年，我们计划将世界通行模型率先部署到1至2个真实的落地场景，这包括物流客户、产业园区以及文旅度假区等。届时，我们将重点验证两项核心能力：一是无图导航，二是环境零改造部署，这意味着无需额外安装传感器或对现有门禁系统进行改造，即可实现模型的顺利落地。”
进入第三阶段，纽娲规划实现自主研发机器人本体的量产，并通过软硬件深度协同，有效降低整体系统成本。同时，公司将向更多系统集成商开放其世界通行能力，使其能够兼容AGV底盘、双足机器人等各类异构平台，最终逐步形成一套通用化的机器人通行解决方案。蓝湖资本合伙人魏海涛对此次投资表达了高度认可：“作为专注于投资新质生产力的基金，我们选择投资纽娲并坚定看好杨睿刚教授团队，是因为其‘具行智能’精准击中了具身智能领域的一个核心痛点——机器人不仅要‘会操作’，更要‘能抵达’。团队以自主研发的显式3D几何与物理规律为基础，构建了世界模型底座，并聚焦于打造机器人开放世界通行的通用能力模型（World Traversal Model），旨在实现自主、安全、优雅的智能导航，从而打通行业规模化的关键瓶颈。蓝湖资本非常荣幸能领投本轮融资，并将长期陪伴纽娲，共同开启‘具行智能’的新纪元。”不同资本投资人同样强调了纽娲的独特优势：“杨教授团队兼具世界模型研发与产业真实落地的双重背景，通过其在3D视觉、物理仿真、自动驾驶与机器人等领域积累的丰富经验，开创性地提出了‘具行智能’的技术概念。团队聚焦于机器人在开放人类环境中的通行、导航与社会行为理解，并以高质量合成数据、自主研发的物理仿真引擎和世界通行模型为核心构建技术闭环，我们相信，纽娲有望率先实现‘一脑多型’机器人在真实生活场景中的规模化落地。”
