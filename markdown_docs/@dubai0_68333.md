# 巴林谴责伊朗无人机袭击，保留自卫权

🚨 巴林谴责伊朗无人机袭击，保留自卫权
巴林方面日前通过国家通讯社发布声明，强烈谴责伊朗无人机于上周六对其领土发动的袭击。巴林政府强调，将保留捍卫国家主权和安全的全部权利。此事件无疑将对地区安全局势带来新的挑战。
