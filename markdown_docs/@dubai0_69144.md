# 迪拜谢赫扎耶德路餐厅火灾：火势已控，无人员伤亡

🔥 迪拜谢赫扎耶德路餐厅火灾：火势已控，无人员伤亡
周一上午，迪拜谢赫扎耶德路一家餐厅突发火灾，现场浓烟滚滚，紧急救援部门迅速抵达靠近首个立交桥的事发地进行处置。
截至上午10:50，现场指挥官确认火势已得到有效控制，目前正在进行冷却作业。此次事件未造成人员伤亡。
