# 迪拜珠宝商以900颗钻石重塑世界杯奖杯，彰显创意精神

✨ 迪拜珠宝商以900颗钻石重塑世界杯奖杯，彰显创意精神
迪拜珠宝商Hemant Karamchandani近日将一个由其10岁儿子搭建的乐高世界杯奖杯模型，用约900颗钻石及18K金部件进行奢华重塑。这项耗时近一个月的艺术创作，估值高达30万迪拉姆（约合人民币60万元），且不作出售。
Karamchandani先生表示，这一项目最初是为了鼓励孩子而生，最终却演变成对迪拜独特创意精神的致敬。他计划将这座“钻石世界杯奖杯”带到当地学校和足球俱乐部进行巡展，希望以此激励更多青少年，并力求未来能进入博物馆展出。
