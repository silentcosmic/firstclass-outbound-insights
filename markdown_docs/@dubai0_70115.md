# ️ 迪拜警方紧急警告：谨防虚假签证诈骗！

⚠️ 迪拜警方紧急警告：谨防虚假签证诈骗！
迪拜警方近日向居民发出紧急警告，提醒公众谨防社交媒体和聊天应用上泛滥的虚假签证广告。这些诈骗信息声称可提供工作、居留或旅游签证，并要求支付费用。
诈骗分子通常冒用未经许可的公司名义，甚至假冒政府官员，以此诱骗受害者支付款项办理所谓的“签证”。警方强调，市民切勿轻信此类虚假承诺，以免遭受财产损失。
