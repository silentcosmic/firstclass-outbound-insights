# Stripe与Advent联手：PayPal或迎巨额收购要约

⚡ Stripe与Advent联手：PayPal或迎巨额收购要约
支付巨头PayPal正面临一场潜在的重量级收购。市场消息指出，金融科技巨头Stripe正与私募股权公司Advent International联手，向PayPal发出了一份非约束性要约收购，估值高达530亿美元。
此次要约收购价格较PayPal近期股价溢价约28%，凸显了收购方对数字支付领域整合前景的强烈信心。若交易达成，这将是全球金融科技领域近年来最大规模的并购案之一，预示着行业格局可能迎来深刻调整。
