# 比特币之父身份再引关注：天价资产纹丝不动

🔍 比特币之父身份再引关注：天价资产纹丝不动
比特币创始人“中本聪”的真实身份再次成为焦点。美国《纽约时报》今年4月刊发的一项调查报道，将英国密码学家亚当·贝克列为最有可能的中本聪人选之一。
这项由曾揭露美国血液检测公司Theranos骗局的记者约翰·卡雷鲁参与完成的调查，深入分析了超过13万篇早期密码学论坛文章、电子邮件及网络记录。报道指出，亚当·贝克与中本聪在写作习惯、英式拼写、技术理念和活动时间等方面存在诸多相似之处，其1997年研发的“哈希现金”技术也被中本聪写入比特币白皮书。
然而，亚当·贝克已公开否认自己是中本聪，并表示对此身份毫不知情。目前，相关证据主要基于文本和时间线分析，缺乏私钥签名等决定性验证。
中本聪于2008年发布比特币白皮书，并在2009年启动比特币网络后逐渐淡出公众视野，真实身份至今仍未得到确认。根据早期挖矿数据推算，中本聪可能控制着约110万枚比特币，这些资产多年来基本未曾移动，当前估值接近690亿美元，其价值随市场波动。
