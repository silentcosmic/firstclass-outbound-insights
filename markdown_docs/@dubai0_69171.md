# 金边破获跨国杀猪盘：美元冥币成诈骗工具

⚡ 金边破获跨国杀猪盘：美元冥币成诈骗工具
金边消息，当地执法部门近日在朗哥区成功捣毁一处大型跨境网络诈骗窝点，抓获8名外籍嫌疑人，包括7名孟加拉国男子和1名缅甸女子。此次行动由金边市长坤盛与国家警察总署副总监兼金边市警察局局长尊那林联合指挥。
经查，该团伙以柬埔寨为据点，通过“杀猪盘”（交友诱导虚假投资）的方式，对来自美国、巴西、墨西哥和哥伦比亚等国境内的目标实施诈骗。现场查获12台笔记本电脑、17部手机、路由器，以及40多张用于投资诈骗的“美元冥币”，凸显了该犯罪团伙的欺诈手法之奇特与大胆。目前，所有嫌疑人及证物已移交金边市警察局，将依法进行审理。
