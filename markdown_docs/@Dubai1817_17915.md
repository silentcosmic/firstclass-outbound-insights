# 哈利法塔艺术征集再启：作品或将点亮全球最高楼

✨ 哈利法塔艺术征集再启：作品或将点亮全球最高楼
伊玛尔集团（Emaar）近日再次启动备受瞩目的哈利法塔艺术作品征集活动，为全球艺术家、设计师及创意人士提供独一无二的展示平台。
此次活动旨在遴选杰出创意作品，将其投射到迪拜标志性的全球最高建筑——哈利法塔的外墙之上，为世界呈现一场视觉盛宴。有意参与者，需准备好视频素材及创意说明，并在截止日期前发送至指定邮箱：opencall@emaar.ae。
