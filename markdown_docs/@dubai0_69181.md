# 柬埔寨新省长上任首巡：被查封电诈园区

🚨 柬埔寨新省长上任首巡：被查封电诈园区
7月4日上午，柬埔寨蒙多基里省新任省长孔金尼亲自带队，对此前被执法部门查封的网络诈骗窝点进行了实地视察。新任省警察局局长及省宪兵司令等主要官员陪同。
此次视察背景特殊，此前该省的前任省长、省警察局长及宪兵司令均因打击电诈不力而被革职。此举被外界解读为新一届领导班子对打击网络诈骗犯罪的决心和力度将显著加强。
