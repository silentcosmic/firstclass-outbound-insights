# 美国退休老人加密杀猪盘：22万积蓄被骗

🚨 美国退休老人加密杀猪盘：22万积蓄被骗
近日，美国阿拉巴马州佛罗伦萨市一名退休男子不幸落入典型“杀猪盘”加密货币骗局，毕生积蓄中的22.2万美元几乎被洗劫一空。目前，美国联邦政府已成功查扣部分涉案加密资产，并启动民事没收程序。
骗局始于一名自称“Bella”的女性主动接触受害者，通过数周乃至数月的感情培养，逐步建立信任。随后，骗子将话题引向高收益的加密货币投资，并指导受害者将银行资金转入Coinbase，购买USDT后，最终汇入诈骗团伙控制的虚假钱包和交易平台。
案发后，美国执法部门迅速通过区块链追踪，成功查扣部分涉案加密资产，并已向联邦法院申请民事没收。然而，涉案嫌疑人身份尚未公开，被追回资产能否返还受害者仍是未知数。执法机关指出，“杀猪盘”团伙多藏匿境外，身份隐蔽，抓捕难度极大，因此优先采取资产冻结与没收手段。
安全机构数据显示，2024年全球因加密货币“杀猪盘”造成的损失已高达55亿美元，且案件数量持续增长。执法部门再次警示，此类骗局往往通过长时间的感情或信任培养，诱导受害者投资虚假平台，公众务必对网络交友中涉及高收益投资或加密货币的建议保持高度警惕。
