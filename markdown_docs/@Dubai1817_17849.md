# 老挝金三角特区突查：中方警力疑似参与

🚨 老挝金三角特区突查：中方警力疑似参与
据市场消息，近期老挝万象及金三角特区（包括特外区域）出现多起针对中国企业的突击检查行动。有报告指出，此次行动疑似有中国警方参与，直接针对特定目标。
消息人士透露，已有数家公司被查封，多名人员被捕。部分企业在行动前两天收到预警并及时停业撤离，避免了损失；但仍有硬着头皮继续运营的公司遭到查处。据称，此次行动中，部分被捕人员是中国警方点名要求抓捕的，且中国警员亦在现场参与。
此次整治行动显示出更强的力度，涉及人员恐难通过常规途径解决。建议在该区域运营的企业和个人务必提高警惕，密切关注当地动态。
