# 马来西亚捣毁电诈窝点：18名中国籍嫌犯落网

🚨 马来西亚捣毁电诈窝点：18名中国籍嫌犯落网
近日，马来西亚森美兰州警方成功侦破一宗跨境电信诈骗案，捣毁了一个藏匿于瓜拉比拉迪欧工业区的虚假贷款电诈窝点。
据悉，警方在此次行动中，共逮捕了19名涉案男子，其中18名为中国籍人员，另有1名本地男子。这起案件再次提醒出海企业及个人，需警惕海外诈骗风险，加强安全防范意识。
