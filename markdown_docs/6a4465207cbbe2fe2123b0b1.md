# 印尼政商界震动：Gojek创始人兼前部长纳迪姆·马卡里姆贪污罪名成立

印尼一家法院周二裁定，科技巨头Gojek的联合创始人、前教育部长纳迪姆·马卡里姆（Nadiem Makarim）贪污罪名成立，并判处其十年监禁。有消息指出，这一判决在印尼政商界引发广泛关注。
现年41岁的马卡里姆坚称自己无罪，认为此案具有政治动机。他的这一主张获得了部分学者和人权活动家的支持。
雅加达反腐法院公布了此项裁决，主审法官普尔万托·S·阿卜杜拉（Purwanto S. Abdullah）同时判处马卡里姆缴纳10亿印尼盾（约合5.5万美元）罚款。在雅加达中央地方法院举行的听证会上，五名法官组成的审判小组宣读了判决书，当地媒体对此进行了报道。
周二傍晚，马卡里姆通过其个人社交媒体平台发表声明，表示将立即提起上诉，并为他的孩子、家人以及他深爱的印尼“继续抗争”。“判决已下达，结果正如我所担心的那样糟糕：十年监禁，如果我无法支付8090亿印尼盾的赔偿金，还将额外增加五年零六个月的刑期。我没有那笔钱——从未有过，也从未触碰过——检察官和法官对此心知肚明，”他在声明中写道。“所以，这实际上是长达十五年半的刑期。”“我将立即提起上诉，为了真相，为了那些年轻人、专业人士，以及所有被无辜定罪的人而战，”他强调。据了解，马卡里姆的社交媒体页面由其法律团队管理并代为发布内容。
消息指出，马卡里姆被指控与2020年至2022年期间为学校采购逾百万台搭载谷歌ChromeOS的Chromebooks电脑的决定有关。检方指控他高价购买笔记本电脑和软件，忽视内部评估结果，并涉嫌通过该采购项目收取8090亿印尼盾的回报，同时将谷歌对其前公司的投资作为利益冲突的证据。
马卡里姆否认了所有指控，辩称采购价格低于市场水平，并且内部评估结果已经过时，与新冠疫情期间的紧急教学需求不符。他的律师团队则指出，所谓的“不当所得”是标准的首次公开募股（IPO）前行政重组资金，从未流入他个人账户。
辩方还强调，检方未能证明马卡里姆存在恶意企图、个人致富行为，或笔记本电脑采购项目与谷歌对Gojek相关实体的投资之间存在直接关联。
马卡里姆于2019年离开Gojek，加入时任总统佐科·维多多（Joko Widodo）的内阁。他此前曾警告，此案可能阻碍专业人士投身公共服务，并令担忧印尼法律不确定性的投资者望而却步。
据了解，检方或辩方均可对该判决提起上诉。
