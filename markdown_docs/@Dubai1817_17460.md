# 缅泰边境诈骗园：超5千人仍被困，国际救援受阻

🚨 缅泰边境诈骗园：超5千人仍被困，国际救援受阻
据路透社援引人权组织最新报告，缅甸与泰国边境地带的诈骗园区内，仍有超过5300人被困。这些受害者主要为外籍人士，深陷由当地武装组织（DKBA）控制的至少四处据点。
反人口贩运民间社会组织（CSNHTV）已致信泰国警方，敦促其介入解救。据估算，被困人员中包括约1600名中国公民、200名缅甸公民、20名泰国公民，以及来自菲律宾、台湾、巴西和肯尼亚等地的多国公民。该组织强调，由于未能彻底铲除犯罪网络，这些诈骗活动持续猖獗，受害者遍布全球。
截至目前，DKBA及缅甸军政府尚未对此事件作出公开回应。
