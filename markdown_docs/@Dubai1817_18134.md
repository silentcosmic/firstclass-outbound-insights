# 特朗普将亲临世界杯决赛现场

⚽ 特朗普将亲临世界杯决赛现场
白宫于7月16日宣布，美国总统特朗普将于本周日前往纽约-新泽西体育场，现场观看备受瞩目的阿根廷与西班牙世界杯决赛。
白宫发言人卡洛琳·莱维特表示，特朗普总统对此次出席充满期待，其到场将为美国历史上观众人数最多的世界杯画上圆满句号，凸显了对这项全球体育盛事的高度重视。
