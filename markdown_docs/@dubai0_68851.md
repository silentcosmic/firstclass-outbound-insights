# 迪拜智能系统发威：18个月揪出900余份假证件！

🚨 迪拜智能系统发威：18个月揪出900余份假证件！
迪拜近期在边境安全与身份验证方面取得显著进展。在过去18个月里，其高科技检查系统已累计检查了20,307份旅行证件，成功识别出902份伪造或修改过的护照、签证等文件。
具体数据显示，在较早的阶段，共查验15,746份证件并发现689起违规；而在最近的半年内，则在4,561份检查中揪出213起假证。这表明迪拜在打击身份欺诈方面效率持续提升。
迪拜身份与外国人事务总局（GDRFA）下属的“文件检查中心”已从传统人工查验升级为先进的数字系统，能够实时比对全球80多个国家的护照样本，精准识别高科技伪造的假证。该中心还获得了国际ISO认证，吸引了包括韩国、德国、欧盟及海湾国家在内的多个国家前来学习其先进经验。
