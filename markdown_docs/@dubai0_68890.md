# 亚马逊年内推卫星互联网，直面星链竞争

⚡ 亚马逊年内推卫星互联网，直面星链竞争
科技巨头亚马逊计划于今年底推出其首批低轨卫星互联网服务，正式向SpaceX的“星链”发起挑战。随着近期“宇宙神5型”火箭成功发射，亚马逊在轨卫星数量已接近400颗。该项目旨在通过部署超过3200颗卫星，构建覆盖全球的网络，服务普通用户、航空公司及政府机构。
据悉，亚马逊的卫星互联网服务将率先在南北极附近区域落地，并逐步向赤道推进。尽管目前其在轨卫星数量远低于“星链”的万颗规模，但亚马逊已承诺投入至少820亿美元，预订约100次火箭发射以全力追赶。然而，该项目也面临挑战，上月合作方新型火箭发生爆炸故障，导致后续发射计划遭遇技术阻碍，目前团队正联合展开调查以确保项目顺利推进。
