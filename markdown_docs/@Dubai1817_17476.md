# 洪森访华：铁杆友谊背后，三张账单待北京清算

🔥 洪森访华：铁杆友谊背后，三张账单待北京清算
洪森今日起对中国进行访问，此次行程远超表面上的“铁杆友谊”，其背后实则承载着柬埔寨亟待解决的三大核心议题：电诈治理、运河项目融资以及权力平稳交接。
首先是电诈问题。近年来，柬埔寨因网络诈骗、在线赌博等灰色产业而备受国际关注。随着中国持续加大打击力度，美国也扩大制裁范围，洪森此行首要任务或是向北京展现肃清电诈的决心，以期重塑国家形象，吸引中国投资和游客回流，填补灰色经济退场后的空白。
其次是德崇扶南运河项目。作为柬埔寨最大的基础设施工程及洪森重要的政治遗产，该运河的推进离不开中国的资金、技术与政治支持。在越南保持警惕及西方国家提出质疑的背景下，洪森此时访华，暗示运河项目可能已进入关键的资金及支持争取阶段。
最后是权力接班。尽管已卸任首相，洪森仍是柬埔寨核心权力网络的实际掌舵者。其子洪玛奈执政近三年，但外界普遍认为，柬埔寨政治稳定的真正象征依然是洪森本人。此番亲自出面，不仅是为儿子背书，更是向国际社会传递信号：无论谁执掌首相之位，柬埔寨的内外政策路线将保持连续性。
洪森的北京之行，无疑是在为柬埔寨的未来发展、政治稳定和经济复苏寻找新的支点。
