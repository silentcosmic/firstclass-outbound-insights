# 中老边境反诈新进展：中国警方成功押解嫌犯回国

🚨 中老边境反诈新进展：中国警方成功押解嫌犯回国
据最新消息，中国警方昨日在老挝磨丁关口成功完成了一批涉嫌网络诈骗犯罪嫌疑人的押解回国工作。目前，现场交接已全部完成，相关人员正等待进一步的调查处理。
此举再次表明，中国警方将持续加大对跨境网络诈骗犯罪的打击力度，以切实保障国民财产安全。
