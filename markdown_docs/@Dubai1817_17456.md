# 泉州警方斩断“养号”诈骗链条，6嫌犯落网

⚡ 泉州警方斩断“养号”诈骗链条，6嫌犯落网
福建泉州警方近日成功捣毁一个专门为网络诈骗提供引流服务的黑灰产窝点，依法刑事拘留6名犯罪嫌疑人。该团伙通过“养号”机制，为诈骗活动提供大量活跃的社交媒体账号。
据警方披露，该窝点以高薪兼职为诱饵，吸引人员注册并长期维护大量社交账号。这些账号经过“养号”操作，保持活跃度以规避平台监管，随后被团伙高价出售。部分出售的账号最终被用于诈骗推广和引流等非法活动。
行动中，警方查扣了30余部涉案手机及笔记本电脑，并掌握了团伙招聘、注册、养护及出售社交账号的完整证据链。目前，以叶某前、叶某胜为首的嫌疑人因涉嫌帮助信息网络犯罪活动罪已被刑拘，案件仍在进一步侦办中。
