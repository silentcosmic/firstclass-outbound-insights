# ️ 逃离西港电诈炼狱：中国公民柬埔寨再陷危局

⚠️ 逃离西港电诈炼狱：中国公民柬埔寨再陷危局
中国公安部刑侦局近日披露一起案件，一名中国男子因债务缠身，轻信所谓“赴柬掘金”诱惑，抵达西港后被迫从事股票诈骗。他展示身上多处电击伤痕，证实其在园区内曾遭受殴打。
在柬期间，他不仅分文未得，逃离西港后又被“朋友”诱骗至波贝，囚禁于“中转站”并遭毒打，最终被转卖至其他园区。最终，他成功逃出，却因未随身携带护照原件，在宪兵检查时暴露，被遣返回国。
此次事件再次凸显柬埔寨政府打击电信诈骗的决心。近年来，柬埔寨副首相兼内政部长苏速卡透露，已累计遣返约36万名涉嫌电诈、非法入境、居留或务工的外国人。西港、波贝、柴桢、金边等曾是柬埔寨电诈活动的重灾区。
随着中柬及区域执法合作深入，大量涉诈人员被遣返或移交。中国公安部此前通报，近年来，通过国际执法合作，已有5.8万名涉诈嫌疑人被移交中方。缅北“四大家族”犯罪集团已被彻底铲除，陈志、佘智江、李雄等重大跨境赌诈集团头目也已成功押解回国。然而，此类犯罪正呈现出更隐蔽、更分散的趋势，甚至有向周边国家转移的迹象。
