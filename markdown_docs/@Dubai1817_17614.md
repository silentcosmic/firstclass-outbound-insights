# ️ 缅甸诈骗园区逃离者泰被拘：受害者身份引争议

⚠️ 缅甸诈骗园区逃离者泰被拘：受害者身份引争议
近日，13名从缅甸网络诈骗园区逃出的外籍人员在进入泰国后被警方拘留。此事件不仅涉及非法入境，更再次聚焦了对诈骗园区受害者身份认定的国际争议。
据悉，这些人员分别来自七个不同国家，此前被高薪诱骗至缅甸克伦邦，被迫在由民主克伦佛教军（DKBA）控制区域内的诈骗园区内从事洗钱、电信诈骗等非法活动。调查发现，他们长期处于受控状态，且身上有不同程度的伤痕，存在遭受长期暴力虐待的迹象。
泰国警方正以非法入境罪名对他们展开调查。然而，总部设于泰国的反人口贩运非政府组织对此提出质疑，认为这13人更符合人口贩运受害者的特征，呼吁泰国政府依据国际原则进行身份甄别，并提供必要的医疗、心理及法律援助，而非简单作为移民案件处理。
此次事件再次凸显缅甸部分武装组织控制区域内诈骗园区普遍存在的强迫劳动、人身控制及暴力胁迫问题。国际社会正持续呼吁加强跨境执法合作，加速营救受困人员，并严厉打击跨国电信诈骗网络。
