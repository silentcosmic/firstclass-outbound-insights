# 全球穆斯林旅游指数发布：阿联酋位列第六

📣 全球穆斯林旅游指数发布：阿联酋位列第六
万事达卡与CrescentRating近日联合发布了第11版《全球穆斯林旅游指数》，深度剖析人工智能、数字信任、区域流动性及目的地准备度如何共同塑造穆斯林友好型旅游的未来格局。
报告预测，今年全球国际穆斯林游客数量将达到2.08亿人次，且高于2025年预期的1.96亿人次；到2030年，这一数字有望攀升至2.62亿人次，年消费总额预计将高达3100亿美元。
2026年版指数沿用了“可达性、沟通、环境与服务”（ACES）评估框架，覆盖了全球超过98%穆斯林游客所前往的150个目的地。新版报告将重点放在人工智能准备程度、数字曝光度、智慧目的地基础设施、游客信心以及日益动荡的全球环境下的韧性规划。
在此次榜单中，阿联酋以75分位列第六，彰显其在清真旅游、穆斯林友好服务及目的地营销方面的综合实力。报告将阿联酋归入“先锋者”象限，该象限包含了27个在接待穆斯林游客方面处于领先地位的全球目的地。
