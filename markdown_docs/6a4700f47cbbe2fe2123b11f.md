# 摩根大通携手NPCI，打造更透明的UPI跨境支付体验

UPI用户在境外支付时，有望通过印度国家支付公司（NPCI）与摩根大通支付部门的深度合作，享受到更为透明的汇率。
双方正携手致力于为UPI跨境支付提供实时的货币兑换与结算支持。
具体而言，此次整合将把摩根大通支付部门的外汇交易平台及其API接口能力，与NPCI的UPI基础设施进行无缝对接。
这将使得多币种交易能够实现实时兑换与结算。
UPI作为印度领先的即时账户间支付系统，随着越来越多国家向印度旅行者开放其受理服务，其国际市场版图正在持续拓展。摩根大通支付部门印度支付业务主管Guhaprasath Rajagopal对此表示：“这项合作能够显著减少跨境支付中的摩擦，将客户与UPI日益扩大的覆盖范围连接起来，同时结合我们实时的外汇交易能力和基于API的集成方案。我们的核心目标是协助客户在各种货币和支付通道中，实现更佳的客户体验、更高的透明度、更强的规模效应和更完善的控制。”NPCI方面指出：“UPI已为安全、无缝的数字支付树立了坚实标杆。通过与摩根大通支付部门的合作，我们致力于消除跨境支付的障碍，为印度旅行者提供即时、可靠且值得信赖的支付体验。”
目前，UPI已在九个国家和地区实现可用，包括新加坡、阿联酋、尼泊尔、不丹、毛里求斯、法国、斯里兰卡、柬埔寨和卡塔尔。
此次合作预计将进一步助力UPI在全球市场的拓展步伐。
