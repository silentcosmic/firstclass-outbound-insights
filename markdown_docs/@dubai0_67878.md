# 特朗普警告伊朗：不收敛将面临更严厉打击

🚨 特朗普警告伊朗：不收敛将面临更严厉打击
美国前总统特朗普近日就中东局势向伊朗发出严厉警告。他要求伊朗立即制止其在黎巴嫩的代理人继续制造事端，并明确表示，如果伊朗未能有效约束其地区行动，美国将采取比此前更强硬的报复措施。
