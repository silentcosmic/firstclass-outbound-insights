# ️ 柬埔寨汇旺债权遭低价收购，洪图律师现身引猜测

⚠️ 柬埔寨汇旺债权遭低价收购，洪图律师现身引猜测
近日，金边一家名为“债务解决服务有限公司”的企业引发市场关注。该公司发布公告，公开以债权金额一至三折的超低价收购汇旺储户尚未结算的美元债权，并积极联系债权人进行办理。
有储户反映，该公司客服声称拥有官方背景，并掌握柬埔寨国家银行提供的储户名单及联系方式。然而，有维权人士前往该公司了解情况时，发现洪森亲侄洪图的律师在场，这不禁引发外界猜测该公司可能与洪图存在关联，意在利用储户急于套现的心态，低价收购债权以赚取巨额差价。
据知情人士透露，自今年5月以来，金边已陆续出现多家收购汇旺债权的公司。尤其是在4月27日有维权人士被捕后，不少当地储户因担心维权风险，被迫选择低价出售手中的债权。
