# 东亚外卖市场2025年GMV预计达386亿美元：运营商执行力重塑竞争格局

据市场分析机构预测，2025年东亚四大市场的餐饮外卖平台商品交易总额（GMV）预计将达到386亿美元。
该机构一份报告指出，其中韩国市场以283亿美元的规模，占据了总额的约73%。
与此同时，日本、中国台湾和中国香港地区的GMV则分别贡献了41亿美元、36亿美元和26亿美元。
尽管这些区域在城镇化水平、居民收入、餐饮服务发展以及高强度工作文化方面存在诸多相似之处，然而餐饮外卖的渗透率却呈现出显著差异——从日本约3%的水平，到韩国超过20%的惊人比例。
这清晰地表明，市场最终表现并非仅由市场成熟度决定，更深层次地取决于运营商的执行能力。“人们通常认为，外卖服务的成功与否主要由市场发展水平决定。然而，东亚市场的案例明确地反驳了这一观点。”
一位资深行业分析师对此评论道，并指出从纸面上看，这四个市场具有惊人的相似性，但实际的市场表现却截然不同。“市场准备度仅仅是先决条件。但市场并不会自我增长——真正推动市场发展的，是运营商不懈的努力和持续的推进。”
他补充道。
他还强调，经过多年在全球竞争最为激烈的餐饮外卖市场中摸爬滚打，亚洲本土运营商已日益具备了这种强大的执行能力。
值得关注的是，东亚餐饮外卖行业正经历着十余年来最大规模的领导力格局变迁。
在过去十余年间，Delivery Hero曾通过一系列战略性收购，构建了该区域内最为广泛的配送业务版图。
然而，时至今日，这一扩张策略正面临瓦解。据最新报告显示，Foodpanda台湾业务正被出售予Grab，Baemin也已进入市场待售状态，Foodpanda在香港市场失去了领先地位，并且该公司已彻底退出日本市场。
与此同时，以美团（Meituan）、酷澎（Coupang）和Grab为代表的新一代亚洲本土运营商，正凭借其在竞争更为激烈的本土市场中锤炼出的运营策略，重塑该区域的竞争格局。
报告指出，过去曾使Delivery Hero在亚洲占据主导地位的并购驱动型策略，如今正受到那些通过激烈竞争而非单纯整合发展起来的运营商的严峻挑战。
有分析指出，美团旗下的Keeta于2023年进入香港市场，其核心策略围绕补贴一人份餐食和快速扩张网络密度展开。
在短短29个月内，Keeta便实现了盈利，超越了Foodpanda，并从根本上将市场竞争的焦点从补贴支出转向了运营效率的比拼。
与此同时，中国台湾市场多年来一直维持着一种盈利但相对安逸的双头垄断格局。随着竞争压力的减弱，其市场渗透率多年来徘徊在10%左右，增长速度也放缓至5.5%。
Delivery Hero多次试图退出该市场，包括2024年未能成功出售给Uber Eats的交易，都表明台湾市场的竞争结构已抵达转折点。Grab在东南亚市场积累的竞争经验，为其提供了重振台湾外卖市场、使其恢复两位数增长的有力契机。
另一方面，韩国高达283亿美元的市场规模，其基础是数十年来消费者形成的配送习惯，而非仅仅依赖平台的创新和普及。
尽管Baemin目前仍是市场领导者，但酷澎外卖（Coupang Eats）正通过充分利用酷澎更广阔的生态系统，不断扩大其市场份额，这无疑为Baemin的任何潜在未来所有者提出了重要的战略考量。
日本虽坐拥富裕的经济、高人口密度和成熟的餐饮生态系统，但其餐饮外卖渗透率却一直维持在约3%的低位。
日本发达的便利店网络和根深蒂固的独自用餐文化，有效缓解了餐饮外卖在其他市场通常解决的诸多痛点。
酷澎旗下的Rocket Now正积极探索，通过提供价格更亲民的单人配送服务，能否为日本外卖市场开启下一个增长阶段。
报告强调，与上一代全球性整合者不同，如今的亚洲本土运营商在学会扩张之前，首先学会了如何在激烈竞争中生存和取胜。
这些领先的亚洲运营商并非仅仅通过收购资产来扩张，而是通过在高度竞争的市场中持续对抗，磨练并发展出了自身的核心能力。
报告认为，这种在网络密度建设、单位经济效益、客户获取和生态系统整合等方面的运营经验，正成为这些企业拓展国际市场时的关键竞争优势。“所有权的更迭改变的是资产负债表，而非竞争动态。”
该分析师总结道：“无论谁最终掌控这些资产，都将不得不与那些在高度竞争市场中摸爬滚打多年、深谙制胜之道的运营商展开较量。”
