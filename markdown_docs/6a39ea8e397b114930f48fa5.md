# 行业风向 | 圆桌论坛：在没人相信之前，乘风破浪的创er | WAVES2026新浪潮

每一个创业者，都有一段“没人相信”的日子。但正是那段日子，把他们推向了浪尖。创业是鲜活的，接下来这一场，我们听一听那些在质疑声中长大的创业者，怎么把“不可能”变成“不，可能”。
我是本站后浪研究所的许嘉婧，我们这场研究的主题叫：“在没人相信之前”。我们后浪研究所之前也是报道过很多年轻创业者的故事，最有趣的就是在这些早期还处在水下阶段的项目，我们会找到很多带有冲突、戏剧性和有挣扎感的故事，因为这个阶段充满了不确定性。所以今天我们也很荣幸请到了6位嘉宾，有一个数据是在2024年全国大概新注册的企业有600多万户，但是拿到A轮以上融资的不到2%，大多数的项目都死在了黎明之前，今天就请这2%的人聊一下他们是怎样活到这场战役之后。
首先在我左手边的是来自光子易商务副总裁徐子悦。光子易是一家在2026年1月完成B轮融资的公司，投资的公司包括IDG资本。它们主要做的是海外支付赛道，其实这个赛道很难用一句话跟普通人解释清楚，而且在早期很少有人专门做这件事。但是做到现在，光子易在全球有14个运营中心500多人，帮助了很多企业把跨境资金成本砍掉75%以上，今天想向徐总抛出一个问题，你们在早期做这件事情的时候，是不是会被很多人质疑为什么是你们来做，包括你是在什么时候意识到这件事是可行的？
徐子悦：跟大家简单做个分享。国内支付行业发展了十几年，大家对这个赛道的固有印象就是拼价格 —— 这也是行业长期以来的惯性。在很多人眼里，支付算不上什么 “性感” 的生意，再加上我们对接的客户部门，大多是风格保守、风险偏好低、决策周期偏长的财务团队，就更显得这件事没什么想象空间。
但对我们来说，最大的命题恰恰是扭转用户的这份认知：很多人觉得支付是必备、但和业务增长无关的基础配套，我们要做的，就是让它成为中国出海企业的必修课。
举个例子，现在 AI 出海、Token 相关的新业务机会不断涌现，我们是 “春江水暖鸭先知” 的一方 —— 我们是最先看到企业真实境外收入现金流的机构。所以我们一直在做的，就是陪伴中国出海企业接住科技与市场变化带来的新增长机会，确保企业的海外收入能安全、顺畅地收回来。
第二件事，是中国出海企业要面对各国不同的监管、税务、数据安全等要求，我们通过一站式解决方案，帮企业老板和财务团队搞定多收款方式、多币种、多国家的收、付、汇、兑全链路，大幅降低跨境资金流转的摩擦。这些能力不是凭空来的，都是我们服务各行各业、各个国家的出海客户，从一个个真实案例里沉淀出来的。
再拿最近的电商市场举例子：2020 年以来全球电商高速增长，目前发达国家的增速有所放缓，但以巴西为代表的南美电商增长非常迅猛。受当地税务政策影响，大量巴西电商都采用本土店模式运营，这就给我们的客户带来了一连串问题：怎么把电商平台的资金提现，怎么合规地从巴西跨境回款给国内供应链，同时还要覆盖巴西本地的各项费用支付。
更现实的难点是，按巴西当地政策要求，企业需要以本地主体身份开立多个本地账户，但很多出海企业根本没有条件在当地开立银行账户。如果没有合适的解决方案，企业只能把巴西雷亚尔跨境兑换成美元转出，中间既要承担汇兑损失和手续费，还可能因为金融机构的风控调单导致资金冻结，没法灵活使用。更何况企业在巴西当地还有税款、仓储费、广告费等各类支出要付，整个资金链路的摩擦非常大。
这还只是巴西一个市场、美客多一个平台的情况，而美客多的业务覆盖整个南美，当同样的问题放大到墨西哥、智利等五六个国家，复杂度会呈指数级上升。
所以我们做的事，就是实实在在帮企业解决一个又一个细碎的痛点，最终切实提升资金使用效率，降低财务团队的时间成本，大幅缩短资金周转周期。这就是我们面对这份看似 “不性感” 的生意，给出的答案：把基础的支付服务，变成真正能帮企业实现高速增长的核心解决方案。
现在请出第二位嘉宾是KOOK语音的程总，KOOK的前身其实2019年就上线了，叫“开黑啦”，其实KOOK也在今年2月的时候宣布要全面拥抱AI，进行一个AI的整合品牌升级。但是在早期的时候有一个问题，因为海外语音社交这一块很多人都用Discord，尤其是青少年，是很习惯这种语音社交的平台，但是中国其实一直没有长出这样一个主打语音社交的平台，很多人说"中国的社交已经被定义"，你们偏偏还要坚持做这件事。
今天想问程总的是，在早期的时候，你是怎样能特别坚信这个方向是正确的？你内心深处真的相信"中国会有自己的Discord"吗？如果让你重新解释KOOK是什么，你会怎么讲？
回答主持人的问题，Discord无疑是非常伟大的产品，但如果今天让我定义KOOK，我认为“KOOK不只是中国的Discord”。Discord在发展中有一些不错的创新，比如服务器组织形式和开放的生态，这些是我们尊重且欣赏的地方。
对KOOK而言，早期为了让第一次接触的用户更容易理解，我们会说自己是“中国的Discord”，但我们内心希望把KOOK定义为面向中国游戏玩家及泛娱乐人群的实时社群基础设施，这个定义我需要稍作解释。
首先是“中国的游戏玩家”，KOOK想做的不是泛社交，也不是办公协作，而是游戏。中国游戏玩家的生态、游戏内容生态以及游戏开发者生态，与国外有很大区别，这也决定了KOOK与Discord之间有许多差异，我们不是简单复制，而是从中国玩家真实场景里长出来的。
另一个是“实时”。现在普遍使用的社区产品大多基于异步，比如浏览、发帖、评论。但我们发现，游戏玩家和泛娱乐人群有很强烈的实时沟通需求，他们会在语音频道中组织各种活动，一起打游戏、看电影、听歌。所以对KOOK而言，我们认为“实时”对游戏玩家和泛娱乐人群是非常重要的一个点，这与普通社区产品有明显区别。
第三，我想说“基础设施”。基础设施说起来简单，做起来很难。KOOK最开始叫“开黑啦”，最早被大家当作游戏开黑语音工具，但发展过程中越来越多用户不只把它当作开黑工具。我们从第一天提供的基础能力，包括即时通讯、实时音视频会话，从第一天就坚决开放底层API所形成的丰富开发者生态，还有基于此产生的活跃机器人生态，以及越来越多引入的AI能力，我觉得这其实都是向着同一个目标在前进——为中国游戏玩家和泛娱乐人群提供简单、可靠、易扩展的实时社群沟通基础设施，让大家能基于此做出更多有意思的东西。
说到AI，其实我们对AI的态度是非常欢迎的，也一直在拥抱AI。但对KOOK而言，AI不是取代人，而是让人与人之间的连接更丰富、好玩、有趣。例如我们有很多几万人、甚至几十万人同时在线的社群，管理这么多人对社群管理者来说很困难，所以我们会用AI提供方便管理的小工具。同时因为有丰富的开发者生态，我们用户也会基于AI做出一些非常有意思的东西。比如在频道里给大家提供一些有意思的语音助手，还可以基于AI框架把聊天频道变成一起玩自制小游戏，这些都很有意思。
回到你刚才的问题，所谓“中国的社交已经被定义”，我们并不这么认为。对KOOK而言，基于游戏用户和泛娱乐用户的实时社群，正处在蓬勃发展、快速前进的状态。这里可以和大家简单透露一下，KOOK在去年年底已突破250万DAU，累计注册用户超2400万，日均语音时长突破5亿分钟。我觉得除了这些数字之外，更重要的是这些用户的增长我们几乎没有投入一分钱做推广。对我们来讲用户拉用户，对一个好的社群、一个好的社区生态，其实是最健康的增长方式。
从商业化的角度来看，我们有两个自然的商业化路径，一个是增值服务，包括BUFF会员、屏幕分享等增值服务；二是游戏厂商和用户之间的连接价值，这两条路都已走通。所以我们商业化收入增长也非常迅猛，今年预计能增长200%以上，用户的增长可能是70%左右。在这些数据面前，我认为中国的社交生态，至少基于游戏用户和泛娱乐用户的社交生态还远未结束，正在一个蓬勃发展的阶段，甚至可以说才刚刚开始。
回到最开始的问题，我其实不太希望KOOK被定义为“中国的Discord”。KOOK真正的定义，不是在“对标谁”的商业故事里，而是在“被用户需要”的每一个日常里。我们并不急着证明自己是个大故事，我们要先证明用户真的离不开你，这也是KOOK希望能一直做下去的事。
许嘉婧：刚才险些打断您，主要是我们都是游戏玩家，觉得这个切口还是很不错。确实因为中国的游戏市场跟国外生态有很大的区别，现在也是从游戏社交切口打泛娱乐社交更大的平台，很期待KOOK之后的改变。
下面是第三位嘉宾，来自智域基石的CTO徐良威。智域基石这家公司成立时间很近，是到2025年底成立的，但不到4个月已经拿到上亿订单，现在我们就来到具身智能赛道。
智域基石是有灵初、穹彻这几家比较大的具身智能公司都获得了投资，但他们做的不是机器人本身，而是做数据方向的，就相当于做卖金矿的人的“铲子供应商”这么一个角色。这个赛道会面对很多投资人的质疑，比如你为什么要在金矿还没有挖掘出来的时候就提前做铲子，你们是怎么去选择这样一个具体的切口去做行业布局的呢？
智域基石是一家希望用工业化的方法，把人类的行为数字化成高质量、可用于具身智能的训练输入数据，这是智域基石正在做的事情。简单说，就是给这些模型公司提供高质量的用于机器人训练的数据。
刚刚提到用挖金矿、卖铲子的比喻，我把它拆成两部分：第一，对于一个金矿来说，什么时候能挖到金矿，因为只有有了这么一个明确的目标，卖铲子才会有意义。第二个在大家往金矿走的路上，什么时候去卖铲子？
先讲第一个，第一个是具身智能整个大的问题，虽然说这个问题很大，但是大家其实是有一个答案的。就是我们挖到金矿，金矿一定存在，因为具身智能是当前或者下一代真正的技术革命，才能够产生社会结构和整个生产力的变化。但是它的发生可能没有那么快，因为毕竟要技术的发展，需要和产业迭代才能真正让具身智能落到真正生产场景里面去。
所以我们关注的第一个信号是什么时候能够让人真正从原先需要从重要的，非常多的体力劳动的过程中让机器人能够替代过去，这是我们作为具身智能数据的数据生产基础设施公司这么一个角色，希望能看到的变化。
因为我们正在看到已经有大量的模型公司，他们已经把原先还在实验室里的DEMO逐步一点点往家庭，往工厂，往各种场景里面去，我们看到了这么一个现象，所以我们判断这些公司有可能在今年，也有可能明年，也有可能是3—5年，他们很快就要挖到金子了。
我们又看到大家虽然在往这条路上走，但是所有做具身智能模型的公司还没有一个趁手的工具，因为具身这个行业实在是太短了，在这么两三年的时间里面需要完成这么大的突破。他们还在借用原先可能在实验室，或其他行业的工具在完成从原先的demo，在往真实场景落地的一个过程，这时候我们就看到了自身的价值——提供一整套完整、工业化的数据生产基础设施，将人类劳动和行为转化为可用于模型训练的高质量数据。
第一个这些模型公司他们是在做研究，还是真正落一个产品，这个毋庸置疑，大家都是从研究逐渐转变到真正的落地场景里面去，这是我们看到的第一个现象。
第二个现象，模型公司的需求从你们有什么数据，我们来买，还是说有一个持续的、稳定的，能够长期和他们的目标挂钩的一个数据需求？这个事情也是在逐步发生的。如果说关注具身智能行业的可以发现，从2023年开始大家就已经开始提数据了，但是对于数据指标、数据的类型，它需要什么任务，以及这个数据怎么采，它又怎么去评价这个数据好不好，其实是非常分散的，在这两到三年的时间里面逐渐的开始有一点收敛。也就是说，在下游的这些做模型的公司里面，他们已经开始去关注到把自己任务的需求、数据的需求收敛成一个持续订单，这是我们看到的第二个现象。
第三个现象是刚才提到的订单，在以前的具身智能行业里面，他来找一个供应商，他很多时候说我们交流一下，但现在我们发现上来他不是说交流一下，而是说我们有一个订单，希望你来做，这是我们看到的第三个现象。
这三个现象结合起来，说明具身智能已经从“遥远的地方有一个金矿，大家用着不同的工具去做”，逐渐发展到“金矿可能在不远或可预见的未来就会挖到”，同时大家又在寻找一个足够好的铲子。所以智域基石选择在这个大环境下，发挥我们的优势，提供高质量的数据生产基础设施，以此推动整个具身智能的发展。
万拿机器人他们主打的点是在于机器人的灵巧操作技术，要帮助机器人获得真正的劳动能力，我觉得专做部件这件事情挺有意思的。抛给程总的问题是，为什么你们想去主打做部件，而不是做整机，你们是觉得部件比整机更有壁垒，还是说暂时不想做整机？
我觉得这不是一道“整机和部件二选一”的题。整机有整机的壁垒，核心部件也有核心部件的壁垒。真正重要的是，你判断未来机器人产业最大的价值和瓶颈会出现在哪里。
先简单介绍一下万拿机器人。我们是具身智能灵巧操作专家，专注于微型伺服电缸、仿生多自由度灵巧手等具身智能核心部件，以及具身机器人灵巧操作场景解决方案的研发与产业化，是国内较早聚焦机器人灵巧操作能力产业化落地的科技创新企业。我们不仅关注核心部件本身，更关注机器人如何在真实场景中完成操作、创造价值，希望让机器人真正拥有一双灵巧而有力量的双手，推动具身机器人规模化应用。
过去几年，机器人本体的发展非常快，已经涌现出一批优秀企业。整个行业在运动控制、本体设计和智能化水平上持续突破。但当机器人真正进入工厂、物流、商业服务等真实场景时，行业面临的核心问题正在发生变化——竞争焦点正从“会不会动”转向“会不会干活”。
因为现实世界本质上是围绕人的双手建立起来的。工具是给手设计的，设备是给手设计的，工业流程和社会基础设施同样是围绕人的双手构建的。机器人如果想真正获得劳动能力、创造生产力，最终绕不开灵巧操作能力。
基于这样的产业判断，也基于团队多年来在灵巧手和核心执行器领域持续积累的技术基础，我们选择聚焦灵巧操作赛道。我们不是因为整机做不了而选择部件，而是相信未来机器人产业的竞争，不仅是大脑的竞争，更是操作能力的竞争。机器人能不能站起来，决定它像不像人；机器人能不能干活，决定它有没有价值。而万拿机器人选择的，正是机器人从“会动”走向“会干活”、从技术验证走向规模化应用的关键环节。
