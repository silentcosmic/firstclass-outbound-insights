# 越南破获跨境电诈案：专骗法国投资者，11名中国籍嫌犯落网

🚨 越南破获跨境电诈案：专骗法国投资者，11名中国籍嫌犯落网
越南北宁省警方近日成功捣毁一处跨境电信诈骗窝点，抓获11名中国籍嫌疑人，现场查获笔记本电脑、手机及大量电子数据。该团伙专骗法国投资者，其作案手法精密，涉及虚假平台与数字资产洗钱。
警方调查显示，该团伙藏匿于居民区，利用翻译软件伪装成法国人，通过虚假社交媒体账号与法国受害者建立信任。随后，嫌疑人诱导受害者进入名为“Lucky Coin Mall”的虚假投资平台，并拉入Telegram群组，由“托儿”营造高收益假象，诱骗受害者持续转账投资。
当受害者尝试提现时，诈骗团伙便以“账户冻结”、“缴税”或“验证费”等理由，进一步骗取资金。所有非法所得随后通过数字资产转移，以掩盖资金流向。警方表示，该团伙组织严密、分工明确，目前案件正在深入调查中，正追查幕后网络及各嫌疑人的具体分工。
