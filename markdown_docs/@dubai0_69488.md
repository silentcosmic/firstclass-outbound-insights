# 深圳特大“荐股”诈骗案：245人被捕，冻结2400万

🚨 深圳特大“荐股”诈骗案：245人被捕，冻结2400万
近日，深圳市公安局龙岗分局成功捣毁一个盘踞于龙岗坂田街道的特大“荐股引流”电信网络诈骗团伙。此次行动冻结涉案资金高达2408万元，并彻底斩断了一条集非法买卖公民信息、电话引流、虚假荐股及诱导转账于一体的完整诈骗产业链。
行动中，深圳警方精准端掉全部诈骗窝点，现场抓获247名涉案嫌疑人。目前，已有245名嫌疑人被依法刑事拘留，案件正在进一步侦办中。
