# 阿联酋“买断”停火疑云：百亿资金换中东和平？

⚡ 阿联酋“买断”停火疑云：百亿资金换中东和平？
路透社援引多方消息称，阿联酋已同意向伊朗解锁100至200亿美元资金，以换取后者在中东地区的停火。首批约30亿美元据称已到位。资金性质尚不明朗，或为阿联酋自身投入，亦或是长期冻结在迪拜银行的伊朗账户。
然而，阿联酋外交部于6月13日断然否认此消息，称其“纯属捏造”，但声明中未提供任何反驳细节。这一消息若属实，将彻底改写此前市场对部分中东国家“越挨打越涨”逻辑的解读。
分析人士指出，“政府+钱包托底+花钱买和平”或许才是其市场韧性的真实机制。例如，阿联酋通过潜在的30亿美元“停火交易”迅速买断了风险溢价，使得迪拜和阿布扎比证券交易所（DFM/ADX）在一个月内重回正增长。相比之下，卡塔尔因未能达成类似协议，其市场持续笼罩在Al Udeid空军基地事件的阴影下，QE指数一年内下跌6.82%。而巴林由于经济规模较小，市场波动性被流动性有效抑制，表现出相对稳定的反弹。
由此可见，“是否谈成”而非“是否被打”，正成为影响中东地区市场走向的关键变量。值得注意的是，本报道基于路透社、半岛电视台、CNBC、新华网及澎湃新闻等公开信息，但对资金是否实际到位，本平台未进行独立核实。
