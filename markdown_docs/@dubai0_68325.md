# 重磅！柬埔寨电诈“盘总”雅加达落网

🚨 重磅！柬埔寨电诈“盘总”雅加达落网
印尼警方近日宣布，成功抓获一名被国际刑警组织中国国家中心局列为头号通缉人员的中国籍电信网络诈骗核心骨干——郑荣景。这位长期盘踞柬埔寨诈骗园区、被称为“盘总”的嫌犯，于6月24日乘坐亚航航班抵达雅加达苏加诺-哈达国际机场时，被印尼警方与移民执法人员当场逮捕。
据悉，中方早在今年3月就通过国际刑警渠道向印尼警方发出协查请求，对郑荣景展开持续追踪。印尼国家警察国际刑警局秘书长表示，目前正深挖其入境印尼的真实目的，初步研判已有当地人员为其安排藏匿窝点，以图继续实施犯罪活动。
此次抓捕行动，是印尼警方与移民机构紧密联动，并依托中印尼两国国际刑警警务协作的又一成功案例，彰显了国际合作打击跨境犯罪的决心与成效。
