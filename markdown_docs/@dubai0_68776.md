# ️ 北京飞机撞楼事件：官方定性系个人原因危害公共安全

⚠️ 北京飞机撞楼事件：官方定性系个人原因危害公共安全
针对此前引发广泛关注的北京中信大厦轻型飞机撞楼事件，朝阳区官方于7月2日公布了调查结果。事件被定性为一起因个人原因造成的危害公共安全案件。
调查详情披露，涉事航空器为一架阿若拉SA60L单发双座螺旋桨轻型飞机。驾驶员刘某某，66岁，持有私用飞行执照。事发当日，刘某某从平谷区一通用机场起飞后，在独立飞行中偏离预设航线并与机场失联，最终撞上高层建筑，当场身亡。事件共造成13人受伤，其中1人已出院，其余伤者经救治均无生命危险。
深入调查显示，驾驶员刘某某长期受失眠和焦虑困扰，其个人日记中多次出现意图“结束生命”的表述。这些发现最终促成了官方对事件“因个人原因造成危害公共安全”的定性。
