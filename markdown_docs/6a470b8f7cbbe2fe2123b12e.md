# 大华银行斥资4.5亿美元，在越南胡志明市国际金融中心启建新总部

新加坡大华银行（UOB）旗下的越南大华银行已正式启动其在胡志明市大华银行广场的建设工作。这座耗资4.5亿美元的摩天大楼将坐落于越南国际金融中心（VIFC）核心区域，标志着大华银行成为首家在该区域内投资建设总部的外资金融机构。
在周三举行的奠基仪式上，胡志明市人民委员会主席阮文育（Nguyen Van Duoc）强调，大华银行广场未来不仅将成为大华银行在越南的全新总部，更将扮演越南与整个区域之间重要的金融、商业及投资联结点。
大华银行副主席兼集团首席执行官黄一宗（Wee Ee Cheong）表示，大华银行在全球19个国家及地区设有分支机构。此番新总部的建设，充分彰显了该行对越南发展前景的坚定信心，并与胡志明市建设国际金融中心的战略愿景高度契合。
该项目预计于2030年竣工，这将是大华银行在越南长期发展战略中的一个新里程碑。越南已被该行确立为东盟地区的重点市场。
胡志明市期望，这座国际金融中心能吸引全球性的投资基金、投资银行、资产管理公司及金融科技中心入驻，从而引导长期资本流向越南的战略基础设施、可再生能源及高科技产业。
此次奠基正值胡志明市大力推动将VIFC-HCMC打造成为吸引新一代外国直接投资（FDI）的战略工具之际。目前，该市已吸引来自152个国家和地区的20,259个FDI项目，累计投资总额近1420亿美元。
仅在2026年上半年，胡志明市就吸引了超过68亿美元的FDI，完成了年度目标的62%。FDI部门约占全市社会投资总额的20%，并贡献了超过50%的出口总额。
