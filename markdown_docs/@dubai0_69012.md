# 特朗普：美曾可斩首伊朗高层，但为核谈而克制

🚨 特朗普：美曾可斩首伊朗高层，但为核谈而克制
美国前总统特朗普近日披露，美国曾有机会在伊朗最高领袖阿里·哈梅内伊的葬礼期间，对出席的伊朗领导层实施打击。然而，美方最终放弃了这一行动，旨在维持与伊朗的核谈判渠道。
特朗普直言，“我们本可以一击将他们全部消灭，但我们没有，因为那样就没有谈判对象了。”此番言论揭示了美国在地缘政治博弈中，为外交斡旋而采取的战略克制。
