# 东南亚华人涉嫌跨国黑产：多名受害者呼吁彻查

🚨 东南亚华人涉嫌跨国黑产：多名受害者呼吁彻查
市场消息指出，一名代号“阿三”的华人陈君法，被指控在国内涉嫌高利贷、强奸及组织黑社会等严重罪行，并携妻何婉霞潜逃至东南亚多国。
据爆料，陈君法及其团队在老挝期间，从事人民币换汇业务，专以“黑钱”为目标，导致多名同乡银行卡被冻结。随后，其团队转战柬埔寨，与张国龙、曾海宾、黄俊豪等人涉嫌盗取同乡社交通讯账号，并被控与合伙人进行商业合作时，诈骗数百万U的资金。
消息还透露，尽管其团队内部曾有员工意图将骗取资金退还股东，但款项最终被陈君法私自截留。此外，陈君法被指控在向股东哭穷并借款逾十万U的同时，秘密携带1500万人民币回国。在缅甸期间，他更涉嫌将同乡诱骗至当地的“黑园区”后不闻不问。
目前，相关证据已提交至国内反诈部门。多名受害者呼吁社会各界警惕此人，并希望相关部门能尽快介入调查，依法处理。
