# 全球基础设施赋能本土市场：BTSE印尼持牌数字与加密资产交易所启航

专注于区块链资产交易与技术解决方案的BTSE集团，近日正式宣布BTSE印尼平台的上线。该平台是一家受监管的印尼数字与加密资产交易所，由BTSE集团与PT Aset Kripto Internasional通过合资形式成立。
据BTSE方面声明，此次上线是继印尼领先的受监管数字资产平台NVX成功品牌重塑后的重要一步。
在合资结构下，BTSE集团将作为交易平台的技术中枢，提供世界级的交易基础设施、深厚的流动性以及稳健的技术支持，确保为用户带来一流的交易体验。
BTSE印尼则将重心放在用户获取、市场推广、关键伙伴关系建立以及销售业务上。凭借对本地市场的深入洞察和广泛的人脉网络，BTSE印尼致力于加速其在该国的业务增长。
BTSE印尼已获得印尼金融服务管理局（Otoritas Jasa Keuangan/OJK）的正式许可，获准作为受监管的数字金融资产和加密资产交易运营商（Pedagang Aset Keuangan Digital – PAKD）开展业务。
作为少数几家获准在印尼开展加密资产交易的实体之一，BTSE印尼通过健全的客户资产保护机制、严格遵守反洗钱法规以及透明化的运营，为用户提供了更高的保障。
该牌照使BTSE印尼能够与当地银行及支付网关合作，提供受监管的数字金融产品，其中包括印尼卢比（IDR）的充值、提现及兑换服务，以及以印尼卢比计价的交易对。这些核心能力对于该国迅速增长的加密用户群体至关重要。
此外，此举亦赋予BTSE印尼拓展其产品组合的能力，未来有望根据印尼当局的法规和指导方针，引入期货交易及其他创新服务。BTSE集团首席运营官Jeff Mei表示：“印尼拥有成为亚洲下一个主要加密枢纽的所有要素：庞大的人口基数、旺盛的市场需求以及健全的监管框架。当前所需的是全球基础设施与本土专业知识的完美结合，而这正是我们此次合资所能提供的。”
与此同时，BTSE印尼首席战略官Stephanie Kusnadi则认为，与BTSE的整合为印尼数字资产领域开启了新篇章。“凭借我们深厚的本地专业知识和全球交易所巨头的强大支持，BTSE印尼独具优势，能够提供一个安全、具有竞争力且完全符合当地法规的交易平台。”她补充道：“我们很高兴能将全球标准带给我们的本地用户，并为印尼数字经济的增长贡献力量。”
另有消息称，印尼Polibeli集团计划在泰国建设一座100兆瓦的人工智能计算中心。
