# 亚马逊AWS核心高管变动：计算与机器学习负责人戴夫·布朗任期19年后离任

亚马逊云科技（AWS）宣布，其计算与机器学习业务主管戴夫·布朗（Dave Brown）将于7月底离职，该职位将进行人事更迭。
亚马逊电商基础业务高级副总裁戴夫·特雷德韦尔（Dave Treadwell）将于8月1日正式接管该部门。
特雷德韦尔目前在亚马逊担任电商基础业务高级副总裁一职。
戴夫·布朗在AWS任职近19年，此次选择离开亚马逊，寻求外部发展机会。其下一家雇主及具体职位，亚马逊方面尚未对外披露。
布朗和特雷德韦尔均为亚马逊S团队成员，该团队直接向首席执行官安迪·贾西（Andy Jassy）提供战略咨询。
戴夫·布朗于2023年加入S团队，并于2016年4月晋升为高级副总裁。
