# ️ 4名中国公民逃离柬埔寨诈骗园区后在马被捕

⚠️ 4名中国公民逃离柬埔寨诈骗园区后在马被捕
马来西亚玻璃市州警方近日破获多起偷渡案件，逮捕了包括11名当地人贩子及31名非法入境者。其中，有4名中国籍男子被捕。
警方调查显示，这4名中国公民均是从柬埔寨诈骗园区逃离后，非法偷渡进入马来西亚的。此外，警方在搜查两名涉案人贩子时，分别查获了格洛克半自动手枪、史密夫与威森手枪、弹匣以及数十发子弹。
目前，所有涉案人员均已被依法起诉。
