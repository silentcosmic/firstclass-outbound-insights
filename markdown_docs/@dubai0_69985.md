# 特朗普：中东富国应向美国支付“保护费”

⚡ 特朗普：中东富国应向美国支付“保护费”
美国前总统特朗普近日公开表示，沙特阿拉伯、阿联酋、卡塔尔、巴林和科威特等中东富裕国家，应当向美国支付“保护费”。他指出，美国长期以来为这些国家乃至以色列的安全提供保障，并为此投入巨额资金。因此，他认为由受保护国分担相关成本是合理且必要的。
特朗普进一步强调，美国拥有丰富的石油资源，若将委内瑞拉等地的供应计算在内，美国掌握的石油量已超过全球一半。这意味着美国自身并不依赖中东石油，其在中东的行动主要是为了保护盟友。
