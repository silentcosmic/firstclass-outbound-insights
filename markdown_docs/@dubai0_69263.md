# 印尼严打跨境诈骗：92名国人遣返并终身禁入

🚨 印尼严打跨境诈骗：92名国人遣返并终身禁入
印尼方面于7月6日证实，92名涉嫌参与网络赌博及投资诈骗的中国籍人员已于7月5日被集体遣返回中国。这些人员同时被印尼列入终身禁止入境名单。
此举是印尼近期持续加大打击跨境电信网络诈骗犯罪力度的重要行动之一，凸显了区域内对非法网络活动的零容忍态度。
