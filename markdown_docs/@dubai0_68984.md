# 公安部警示：警惕加密私聊陷阱，多款诈骗APP曝光

🚨 公安部警示：警惕加密私聊陷阱，多款诈骗APP曝光
公安部刑侦局于7月3日发布紧急提醒，指出当前婚恋交友诈骗案件仍高发。骗子惯常伪装成退役军人、公职人员或企业高管等身份，在初步建立信任后，便诱导受害者转至第三方应用进行私密沟通。
这些骗局中，诈骗分子常以“保护隐私”、“加密沟通”或“平台不便深聊”为由，诱骗受害者下载陌生应用，进而逐步实施诈骗。近期被高频利用的包括：旺商聊、企快快、金石录、火鱼、深度专注、小知聊、情绪森林岛、反向碎片等。
