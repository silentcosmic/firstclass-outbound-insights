# 跨境偷渡组织者被遣返：国家移民局通报

🚨 跨境偷渡组织者被遣返：国家移民局通报
国家移民管理局近日发布通报，宣布两名长期盘踞境外、涉嫌组织他人偷越国（边）境的犯罪嫌疑人章某和雍某进已被成功抓获并遣返回国。
此项行动是在我国执法部门的协调推进下完成的，旨在严厉打击跨境人口贩运和非法移民活动，维护边境管理秩序。
