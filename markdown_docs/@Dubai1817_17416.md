# 中国升级“信息战”防御：全面封锁境外广告链接

🚨 中国升级“信息战”防御：全面封锁境外广告链接
2026年6月22日，中国国家安全部发布严厉警告，指控部分外国情报机构正利用互联网“弹窗广告”收集中国公民个人信息，并随即升级网络安全措施。新规要求全国各大网络平台全面封锁来自不明来源的外部广告链接，同时限制或禁止无法核实来源的网站访问。此外，平台需加强对广告内容、来源及跳转页面的审查，并具备快速删除或拦截涉嫌威胁国家安全广告的能力。
此举正值中国与西方国家在科技及信息安全领域竞争日益白热化。北京方面明确指出，网络空间已成为现代“信息战”的关键战场，其战略重要性与军事和经济领域同等。分析人士认为，尽管官方将此举定位为保护数据与国家安全，但也反映出中国正进一步加强对数字空间的监管力度，尤其是在人工智能和信息技术竞争加剧的背景下。此次行动传递出明确信号：数据已被中国视为21世纪与军事、经济、能源同等重要的战略资源。
