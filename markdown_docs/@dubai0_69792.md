# 柬埔寨旬内遣返逾千名非法外籍人士

🚨 柬埔寨旬内遣返逾千名非法外籍人士
据柬埔寨内政部移民局总局最新披露，自7月初至7月10日，柬埔寨已累计驱逐1362名涉嫌各类违法行为的外国公民。这些违法行为包括非法越境、网络诈骗、人口贩运及毒品犯罪等。
此次遣返行动涉及来自中国、印度尼西亚、孟加拉国、泰国、越南、缅甸、尼日利亚等24个国家和地区的公民。其中，170名中国籍人员已通过德崇国际机场搭乘专机被遣返。
