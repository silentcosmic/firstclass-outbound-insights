# 中信银行高管爆仓疑云：驾机撞击中国尊？

🚨 中信银行高管爆仓疑云：驾机撞击中国尊？
近期市场流传一则未经证实的消息，指称中信银行一名高级管理人员疑因交易损失惨重导致“爆仓”，愤而驾小型飞机撞击公司位于北京的总部大楼。据传，涉事者为中银金融及业务客户管理部总经理助理刘俊华，其此前曾接受过驾驶飞机训练。
受撞击的建筑据称是北京地标“中国尊”，事件发生后大楼 reportedly 遭到封锁。目前，官方正全面封锁相关信息，仅有部分内部谈话内容在社交媒体平台上零星外泄。此事件的真实性及具体细节仍有待进一步核实。
