# 中菲联手遣返跨境赌诈主犯，涉案数亿

🚨 中菲联手遣返跨境赌诈主犯，涉案数亿
近日，中菲两国执法部门成功合作，遣返了一名涉嫌组织跨境赌博及电信诈骗犯罪的逃犯林某。此举标志着两国在打击跨国犯罪方面的进一步深化合作。
据查，林某所属的犯罪集团在菲律宾、柬埔寨等多地设立窝点，利用网络赌博平台及色情诱导应用实施跨境赌博、电信诈骗活动，涉案金额高达数亿元人民币。
中方强调，将持续加强与菲方的执法合作，严厉打击各类跨国犯罪，以维护民众财产安全，并为两国间正常的人员往来提供坚实保障。
