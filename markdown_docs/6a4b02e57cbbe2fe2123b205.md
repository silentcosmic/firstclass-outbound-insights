# 小米前高管打造餐饮具身机器人，「影智XBOT」获数亿元融资，林斌、黎万强等知名投资者参与

融资动态与核心团队
据《出海·头等舱》了解，专注于通用餐饮具身机器人领域的「影智XBOT」近期已连续完成两轮总计数亿元人民币的融资。其中，A轮2亿元融资由香港简坤资本GPTX领投；B轮融资规模达3至5亿元人民币，汇集了多家政府基金、美元基金以及产业投资方的共同注资。
在此之前，「影智XBOT」已获得一轮阵容豪华的天使轮融资，参与投资的知名人士包括腾讯高级副总裁张小龙、小米联合创始人黎万强、黄江吉、林斌、洪锋，以及香港科技大学首席副校长郭毅可等。
公司创始人唐沐先生，其职业生涯履历辉煌，曾先后担任金山软件与腾讯CDC（用户研究与体验设计中心）总经理，主导了QQ、QZone等核心产品的体验设计工作。随后，他出任小米生态链副总裁，成功孵化并推出了小米路由器、小爱音箱等多款出货量达千万级的标志性爆品。唐沐先生在产品研发与市场化方面积累了长达25年的丰富经验。
具身智能赛道的新路径
在当前具身智能领域普遍聚焦人形机器人进入家庭与工厂应用场景的宏大叙事时，「影智XBOT」却另辟蹊径，选择了一条看似低调，实则能够迅速实现产品化与商业落地的道路——让机器人走进商业空间，专注于咖啡制作。
与大多数主流人形机器人企业有所区别的是，「影智XBOT」将其技术架构围绕餐饮垂直场景进行深度定制，并以其自主研发的“XOS 3.0具身操作系统”为核心，构建了一种“一脑多形”的创新架构。
唐沐先生指出，业界普遍认为VLA（视觉-语言-动作）模型的泛化能力不足，根源在于试图让单一模型同时处理叠衣服、烹饪等截然不同的任务。而「影智XBOT」的策略是严格限定应用场景，仅在出现异常情况时才调用VLA模型作为应急处理，从而有效节省算力并确保系统运行的稳定性。
该系统在实验室环境下进行仿真预训练，随后在真实商业场景中开展在线强化学习。其性能持续优化，得益于超过400万杯咖啡的真实作业数据积累；同时，通过RLHF（人类反馈强化学习），系统对齐了逾300万次人类偏好数据，持续提升模型表现。
产品线介绍
XBOT C3咖啡机器人： 该设备占地面积仅1.83平方米，配备6轴机械臂，重复定位精度高达±0.02毫米。每小时可制作80杯咖啡，在满电状态下可持续出品150杯。其搭载43英寸数字人屏幕及“爱宝店长”Agent智能助理，市场售价为21.9万元人民币。
XBOT I3冰淇淋机器人： 此款机器人体积更为小巧，每小时可制作60杯以上冰淇淋，制作失败率低于千分之五，设计使用寿命长达25万杯。它支持添加4种酱料和4种脆粒，售价为17.9万元人民币。
XBOT X1通用餐饮人形机器人： 该款机器人配备7轴双臂，双臂协同精度小于等于±1毫米，搭载单颗辉羲R1芯片，可提供500TOPS的本地算力。产品计划于2026年底实现量产，能够完成拿取、制作、放置、递送的全链路闭环操作，预计同期正式发售。
商业模式与市场表现
以在浙江义乌某商场投放的Lite系列咖啡机器人为例，根据公司介绍，该设备售价在十几万元人民币，日均出杯量约200杯，平均客单价约20元，单月累计营收可达6万元以上，净利润超过3万元。由此推算，设备的投资回本周期仅需6至8个月。考虑到机器的设计寿命为5年，扣除回本周期后，剩余运营时间将基本处于纯利润阶段。
唐沐先生以网约车平台滴滴为例进行类比：一位滴滴司机需要花费十几万元购车，并全天候投入运营，而一台同等价格的咖啡机器人仅需每天进行一次物料补给与维护。从深层商业逻辑来看，咖啡机器人本质上是一种更为高效的生产资料。
基于上述考量，「影智XBOT」并未沿用行业内普遍的租赁模式。唐沐先生认为，租赁模式在机器人时代并非最优解。为此，公司积极推行RaaS（Robot as a Service，机器人即服务）模式。
据唐沐先生介绍，未来加盟商在支付机器人设备成本后，每月还需缴纳三类费用：物料订阅费（涵盖咖啡豆、奶制品等耗材）、Agent Token费（即“爱宝店长”数字人服务费用），以及设备的维修保养费。
值得关注的是，目前行业中，「影智XBOT」是唯一一家持有“全国全品类食品经营许可证资质”的企业。这一资质使其能够合法合规地掌控供应链，进而构建并巩固RaaS模式的完整闭环。
同时，「影智XBOT」已与多家知名企业建立合作，为京东七鲜咖啡、亦庄机器人、菲柯机器人（FICO咖啡）、吉旅控股（吉林冰雪文旅）以及博物天行（文博场馆）等提供定制化解决方案。此外，公司正与一个拥有四千多家门店的亚洲咖啡品牌携手，共同拓展海外市场。
截至目前，「影智XBOT」的咖啡机器人已在全球100多个城市成功部署1000多台，累计制作咖啡超过400万杯。公司2025年营收已突破1亿元人民币。唐沐先生透露，2026年，「影智XBOT」已获得的订单金额接近3至5亿元人民币。
「影智XBOT」的运营团队核心成员汇聚了来自知名连锁咖啡品牌的精英力量，其中包括曾负责超大规模区域门店连锁运营、品牌标准化体系及连锁培训体系搭建等关键工作的曹锐锟、于涛等多位资深人士。
创始人访谈：为何选择咖啡机器人？唐沐先生阐述了这一决策的清晰推导过程：“第一步，我首先排除了面向C端的家庭场景。当时观察行业趋势，许多团队投入巨资训练机器人进行叠衣服、烹饪等任务，但效果并不理想。我的结论是，除了像扫地机器人这种用途被严格限定的品类，通用型家庭机器人在短期内是不可行的。”“第二步，我们排除了B端的工业场景。在‘黑灯工厂’中，传统的工业机械臂已然具备极高的效率与稳定性，完全没有空间让一个行动迟缓、行为不可预测的人形机器人去拖累整体生产节奏。工业场景的核心诉求是效率，而非人形。”“第三步，我们最终锁定了B端的商业场景。既然家庭和工厂都不适用，那么就需要在商业领域寻找最大、最高频、且具有强劲增长潜力的刚需业务。经过深入推演，咖啡业务被认为是最佳选择——它是一种全球通用的高频饮品，标准化程度高，并且正处于快速增长期。因此，我们选择研发咖啡机器人并非一时兴起，而是经过严谨的逻辑推导，在排除了所有不适合的选项之后，得出的必然结果。”当谈及人形机器人的必要性时，唐沐先生表示：“埃隆·马斯克曾提出人形机器人符合‘第一性原理’，但在许多实际工作场景中，人形设计反而可能成为一种累赘。仅仅为了‘看起来像人’而拥有长腿，本身就会消耗大量的算力和电力，却不一定能解决实际问题。我们的核心逻辑是，在许多情况下，非人形机器人能够完成人类无法胜任的任务，这才是更具价值的。”关于商业化应用的核心，唐沐先生强调：“商业场景的首要准则是稳定性，而非单纯的炫技。例如，五指灵巧手虽然宣称能开合10万次，但在商场环境中，一旦出现抓取失误导致咖啡泼洒，当天的运营就会受到严重影响。因此，我们仅采用经过工业级验证的夹爪。关节设计亦是同理，行星减速器仅适用于‘大致动作’的摆动，而谐波减速器在重复定位精度、量产成本和使用寿命方面均表现更优。商业领域绝不接受实验室的‘差不多’，必须追求确定性的‘精准’。”对于具身智能的未来，唐沐先生认为：“想象力并非在于场景的广度，而在于能否将其深度打透。过去一段时间，业界曾一度认为‘VLA已死’，那是因为试图让它在通用领域同时完成叠衣服和烹饪等多种任务，这确实极具挑战。然而，在垂直领域内，只要将任务范围限定明确，它就能发挥出色的作用。就像‘黑灯工厂’中的机械臂，甚至无需安装视觉系统，只需高效、低成本地完成单一任务即可。”唐沐补充道：“我们甚至在多数情况下都不需要触发VLA模型，仅在处理异常情况时才调用，以此节省算力。未来几乎所有产品都将实现具身化，但这并不意味着它们都将演变为传统意义上的机器人。「影智XBOT」的目标是深耕餐饮这一垂直场景，通过‘作业-数据-模型’的飞轮效应，最终成为具身智能时代的SaaS（软件即服务）公司。”
