# AI重塑影视产业格局：非科班新锐崛起与资本竞逐新篇章

今年5月，一位来自云南玉溪、拥有地产行业中专背景的刘梓瑜，凭借其短片《丧尸清道夫》创下了全网过亿的播放量，被誉为“国产爱死机”。这部作品甚至为他带来了好莱坞制作人的跨洋合作邀约。尽管这部短片配有英文字幕，但据一位接触过刘梓瑜的业内人士透露，他的英语水平并不突出，甚至不曾使用“翻墙”工具。
仅仅一个月后，这位生于1997年的刘梓瑜便在一次行业活动中，被评选为“新兴未来导演”。在他的社交账号下，一位自称“AAA云还路**建材批发发哥”的网友评论道：“太棒了，这也算是登上了AIGC这艘诺亚方舟，我们这些普通人什么时候也能迎来转机？”评论后还附上了一个“大哭”的表情，流露出对机遇的渴望。
与刘梓瑜一样，迎来人生“翻身”的还有出生于2004年的孟柯。他最近用一部在网吧完成的AI短片《霉》，摘得了北京国际电影节AIGC最佳导演奖项。非科班出身的孟柯在接受采访时坦言：“如果没有AI，我可能一辈子都无法踏足电影节的舞台，AI彻底改变了我的命运。”
孟柯组建了一个约二十人的团队，成员几乎都是“00后”，其中不乏大学生。他直言，如果一辈子的努力只是为了一套价值一两百万的房子，那这样的生活未免缺乏价值。
回溯过往，成为一名电影导演的道路充满了严苛的门槛与挑战，几乎容不得半点差池。首先，导演们通常只能毕业于北京电影学院或中央戏剧学院等顶尖院校，在学期间必须不断参赛、获奖，寄希望于能建立人脉关系，才有可能换来进入剧组的机会，而这通常也只是从场务等小角色做起。
即便成功跻身圈内，导演与导演之间亦有着天壤之别。国内头部视频平台的一位首席内容官曾在内部场合坦言，若非爆款短剧《盛夏芬德拉》的横空出世，影视行业与该剧导演张大马恐怕不会有任何合作的可能——在此之前，张大马仅是一名广告导演。
当前国内影视行业面临成本高企、风险巨大、回报周期漫长的困境。项目立项愈发依赖知名导演、编剧和流量明星，这又进一步推高了制作成本。一位资深导演举例指出，如今一个项目若想“过会”，即获得投资方的资金，通常需要一名及以上的S级明星坐镇，否则合作几乎免谈。而这类明星的片酬动辄高达数千万元，甚至超过整个项目总成本的四分之一。
高昂的成本并未必然带来更好的内容，使得押注影视项目犹如一场豪赌。然而，AI技术的出现，正将影视制作的成本降低“一个数量级”，正如某头部视频平台CEO所言。
创意在落地过程中的损耗也大幅减少。在传统行业中，一位从业者评价称：“很多时候一个优秀的剧本，经过资本方、导演和演员各方意见的掺杂，往往会从80分的水准降至60分。”
一群年轻、非主流的AI导演，正借助日益成熟的AI模型，凭一己之力或少数人团队，完成了过去需要数十人乃至上百人团队才能实现的工作——影视行业中“1%的人将取代大部分人工作”的惊悚预言，似乎正在变为现实。
以往，传统影视从业者需要耗费数年时间，才能完成剧本打磨、平台对接、融资、演员签约、主创团队洽谈等一系列工作，最终实现作品产出并登上主流平台。而AI导演们却能迅速抹平这些障碍。“这就好比游泳比赛，我们讲究入水姿势、划水动作，但他们只管游到终点。”
AI创作的幕后：细节与情感的捕捉
新年刚过，刘驰拖着尚未痊愈的身体，躺在家中摇椅上，仅用一周时间，便将自己在巴厘岛潜水时遭遇船撞九死一生的经历，写成了剧本《七日浮生》。该片随后斩获北京国际电影节AIGC最佳长片，而此时距离报名截止不过一个多月。
节后复工第一天，刘驰便与同事们进行剧本围读——女主角被困在起诉肇事者可能打破一个完整家庭的道德困境中，律师安慰她说：“他们的自由没有比你的生命重要。”读到此处，小团队中一片沉默，甚至有女生红了眼眶。那一刻，刘驰确信，这个故事值得被拍出来。
由于时间紧迫，团队几乎动用了市面上所有可用的视频生成大模型和工具。每天下班前，刘驰和同事会把尽可能多的分镜脚本输入AI，第二天上班时便能“收割”生成好的片段，再进行筛选和汇总。
然而，分镜脚本也并非能原封不动地喂给AI。“AI无法准确理解剧本中的上下文关系、人物关系和情感，所以需要对每个场景进行细致拆解。”刘驰解释道。她会像传统影视导演给演员讲戏那样，为团队成员讲解角色：“比如我在剧本中写‘他的眼睛望向地面’，我需要小伙伴理解他为什么这样做，是出于愧疚还是在思考，而不是机械地输入提示词。”
在AI短剧创作中，一个典型的机械式提示词可能这样描述：“女主在极度欢乐中笑岔了气，秒睡在男主肩头。喧闹瞬间静谧，男主满眼怜惜，将她极其温柔地以公主抱姿势横抱而起。”
与之形成鲜明对比的是，同样获奖的AI短片《牡丹记》的创作者刘雨晴，其提示词则充满细节与构图考量：“以图片1为首帧，躺在床上的人突然咳嗽睁眼向画面左边翻身（运镜参考视频1），旁边两个人很害怕地往画面左边逃跑，镜头推进到床上那个人的这个角度（插入图片2）。”这是片中主人公恋人病重的一段呈现。
《牡丹记》的美术风格甚至先于故事而存在。学习漆画艺术的孙锦涛，对各类材料充满好奇。AI出现后，她开始探索能否将“宣纸”的质感融入到立体的人物和场景中，并在社交媒体上更新系列图片作品“宣上生形”，但苦于没有故事作为载体。后来，孙锦涛在咨询申博信息时，结识了在香港大学读书的刘雨晴。刘雨晴主动提议可以制作一部风格相似的AI影视作品，以放大现有热度，两人一拍即合。由孙锦涛负责美术，刘雨晴负责剧本和视觉内容生成。
刘雨晴热爱电影，她最欣赏的导演是法国新浪潮代表人物之一侯麦。起初她对AI并不感兴趣，因为早期AI在视频生成方面过于粗糙和不稳定。但随着大模型的不断进步，她发现自己想实现的效果可以通过AI来完成。于是，她以晚唐诗人鱼玄机为原型创作了剧本，讲述谣言对女性的伤害，以及在流言蜚语中，人们几乎遗忘了女性的才华。
为了确保风格的准确和统一，这部7分钟的短片，每分钟大约10个关键帧，几乎都需要回炉重造。“无论是P图还是重新绘制，都得我们自己动手。”孙锦涛表示。正因如此，《牡丹记》的制作周期长达半年之久，期间两位默契合作的女性创作者，几乎只见过一面。
今年3月，AI短片《纸手机》火遍全网，引得各大模型和工具厂商争相宣称是其技术成果。其中一个“穿帮镜头”最终被保留了下来——主角的手并未拨动按键，电话却已接通。因为这一版本中人物的微表情、情绪张力以及镜头衔接都十分到位，导演不忍心修改。
技术与艺术的边界：成本、审美与市场博弈
《牡丹记》的两位创作者粗略估算了这部作品所需的成本，约为2000元，主要用于购买两个大模型平台的三项会员服务。而在传统影视行业，数千万元的成本、上百人的团队，通常才会被称为“小成本”制作。其他一些AI创作者甚至无法说出具体的成本数字——因为那不是一个值得在意的投入。
“观众对一个艺术作品的消费，并不只在于内容本身，同时也在于制作过程，我们将其称之为劳作美学。”中国传媒大学动画与数字艺术学院院长王雷指出，实拍内容的巨大工作量，也成为了其独有的优势。
宫崎骏的影迷或许都曾听说过《悬崖上的金鱼姬》的创作故事——整部电影动用了17万张画稿，其中仅一只水母浮出水面的片段，就手绘了1613幅画。观众在欣赏作品内容的同时，也热衷于讨论这部动画电影背后的巨大工作量。
AI导演们也普遍认为，许多优秀的表达和表演仍然需要通过实拍来呈现。影视作品承载着人类的情感与经验，乃至历史记忆和对未来的想象，我们无法将这样的内容完全托付给AI的“黑箱”。
一位业内观察人士透露，《纸手机》走红后，某头部科技公司旗下的AI团队以强势姿态抢夺了冠名机会，并追加了数十倍于其他平台的宣传费用；另一家AI视频应用公司，为争取AI导演合作，甚至在未达成协议的情况下，主动承担了对方的差旅和参赛费用，包揽创作所需的计算资源更是轻而易举。
“有头部模型公司的代表第二天就找上门来，希望在宣传中带上他们的名字，可我实际上并未使用他们的模型。”一位在北京国际电影节获奖的AI导演无奈地表示。对方随即回应，即使没有使用也没关系，只需在宣传时提及自己是该公司的“超创”即可。
一家大模型公司负责“超创运营”的小七，最近没少遭遇“截胡”。为了抢到合适的档期和独家合作，她竭尽全力提前联系对方。然而，在绝对的资金和资源面前，原本谈好的合作往往转瞬即逝。如某头部模型公司和另一知名AI视频应用公司，就曾多次从她手中抢走合作机会。
实际上，对于模型和工具公司而言，“超创”带来的直接商业转化率并不高，更多是扮演“展示窗口”的角色，以彰显模型的能力。但对于某些平台而言，“超创”还承担着另一层作用：协助平台攻克影视级内容，沉淀更多数据资产和模板，进而吸引更广泛的用户。一位AI视频产品从业者表示。
然而，模型能力处于断层领先地位的第一梯队玩家，面对AI导演们却显得稳如泰山。一位同样获奖的AI创作者透露，她们获奖后，国内几乎所有知名的模型和工具公司都主动找上门来，唯独某大型科技集团旗下的模型平台例外。
在《出海 · 头等舱》接触的AI导演中，仅有一位与该头部平台建立了合作，且是该导演主动寻求的。为了达成合作，这位“超创”需要提交简历和项目书，如同应聘一般。她能感受到，该平台更倾向于寻找“在海外国际电影节获过奖、有过长片制作经验”的导演。
传统巨头的进退：押注AI，或固守IP
激进如博纳影业，近期发布的招聘岗位中，AI相关的职位占比超过七成。博纳同时也在积极推进AI原生电影项目，力争夺取“首部院线AI电影”的称号。在此之前，该公司已连续四年亏损，AI被视为其扭转困局的希望。
相比之下，光线传媒则选择了不同的道路。一位行业分析人士指出，光线传媒的核心战略仍旧聚焦于IP和动画电影——2025年，光线传媒总营收预计达到40.40亿元，其中影视剧及相关衍生业务贡献了36.82亿元的收入，主要由爆款动画电影《哪吒之魔童闹海》拉动。“大家虽然都知道AI的重要性，但目前仍处于观望状态。”
刘梓瑜走红之后，光线传媒也曾与他接触，但双方未能达成合作。光线传媒希望刘梓瑜能加入到动画电影的制作中，但他本人更希望能打造个人作品的IP。5月16日，刘梓瑜在社交平台上表示，《丧尸清道夫》的游戏版本已经正在制作中。
比起传统影视巨头，那些兼具技术、产品和IP优势的互联网公司，正成为最有潜力的新玩家。本刊此前曾独家了解到，某头部小说平台正在推动AI动画电影登陆院线，而某互联网巨头也补齐了影视制作链条的最后一环——发行。大浪淘沙之后，若有AI原生的IP能够沉淀下来，它大概率会属于这类互联网科技公司，而非任何一家传统影视企业。
获奖后，孙锦涛和刘雨晴的社交媒体账号接到了不少商业合作订单，其报价远高于市场上一些S级短剧的收购价。6月，她们又更新了一部同样美术风格的作品《画皮》，点赞量超过3万，评论区有不少观众期待她们能将其打造成“聊斋”系列。
刘驰最近正与独立制片人刘蜜合作，计划利用AI制作一部有望登陆院线的动画电影。刘蜜预计，这部影片的AI生成内容占比大约在50%左右，并不算高。因为在她们的经验中，AI在镜头调度、人物一致性、清晰度和配音方面，仍不足以达到大银幕的放映标准。
然而，技术的快速进步可能很快改写刘驰团队的计划。6月底，某科技巨头旗下的计算平台在一次大会上推出了最新版大模型，可以输出4K视频，在场景调度和精细度上都有巨大飞跃。这一备受瞩目、可能进一步颠覆行业的新模型，预计将在7月内正式上线。
冷静思考：AI冲击下的内容价值回归
对AI技术持悲观态度的人也大有人在。在中国传媒大学动画与数字艺术学院院长王雷看来，爆红的《丧尸清道夫》在叙事、调度和剪辑方面的水平，仅属于毕业生作品的“中等偏下”。他认为，AI导演的热度主要来源于大众的“AI冲击感”，一旦这种冲击感褪去，作品的粗糙之处便会显露出来，届时能留下来的创作者和作品将寥寥无几。
最近一部已登陆院线、被称为全球首部95分钟AI电影的作品是《HELL GRIND》，由旧金山一家AI创业公司在某AI模型公司的赞助下制作完成。人们在惊叹其技术突破的同时，也对其剧情单薄、缺乏艺术性提出了批评。
观众真正渴望的是打动人心的好内容，而非仅仅打着AI或名导旗号的作品。当“AI冲击感”逐渐消退，当AI作为一项工具平等地呈现在所有人面前时，最重要的依然是：什么样的作品能够被真正喜爱、被尊重、被铭记。
