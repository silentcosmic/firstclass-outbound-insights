# 特朗普回应加密暴富质疑：全民皆在获利？

💰 特朗普回应加密暴富质疑：全民皆在获利？
昨日，美国前总统特朗普在搭乘专机时，就其2025年通过加密货币业务获利12亿美元的质疑公开作出回应，否认利用职务之便敛财。
特朗普表示，其财富增长主要归因于股市的强劲表现以及此前成功的商业运作，并强调“大家都在获利”。
公开文件显示，特朗普预计在2025年将从其子参与创立的加密货币公司World Liberty Financial（WLF）获得近5.5亿美元，并从“$TRUMP”代币授权中赚取6.35亿美元。在其任内，美国曾放宽加密货币监管政策，此举被视为推动相关资产价值大涨的因素之一。《福布斯》杂志估计，特朗普的个人净资产已从23亿美元激增至65亿美元。
