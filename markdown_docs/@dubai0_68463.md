# 柬埔寨遣返170名中国籍诈骗嫌犯

🚨 柬埔寨遣返170名中国籍诈骗嫌犯
柬埔寨方面近日依照高级别指示，对一批涉嫌网络诈骗、非法居留及非法工作的中国籍人员实施了集中驱逐。此次行动共遣返170人，其中包括4名女性。
据了解，此次大规模驱逐行动由柬埔寨副首相兼内政部长苏速卡及内政部高层领导直接指示，移民总局局长Sok Veasna负责协调调查与执法部门，并与中国驻柬埔寨大使馆紧密合作。所有被遣返人员已通过德崇国际机场口岸，乘坐专机离境。
