# 日本破获2亿日元跨境洗钱案，7名中国籍嫌犯被捕

🚨 日本破获2亿日元跨境洗钱案，7名中国籍嫌犯被捕
日本警方近期成功侦破一起跨境洗钱案，逮捕了包括核心成员林军在内的7名中国籍男子。该团伙被控利用日本企业银行账户，通过伪装国际贸易货款等手段，为跨境网络诈骗集团洗钱，涉案金额高达约2亿日元。
此次调查源于一名美国受害者遭遇网络诈骗，损失约2800万日元。这笔被骗资金汇入日本一家企业银行账户后，随即被迅速拆分并多层转移，疑似通过“洗白”操作以规避监管。
警方指出，44岁的中国籍男子林军被视为该团伙的核心人物，负责管理涉案银行账户并协调资金流向。目前，所有嫌疑人均已依法被捕，案件仍在深入侦办。警方怀疑，涉案企业账户在半年内累计流入的2亿日元，可能与多起跨境电信诈骗及网络诈骗案件有关，远不止本案中的美国受害者损失。
日本警方正持续追查资金来源与流向，并深入调查是否存在更多涉案人员，同时已与相关国际执法机构展开合作。
