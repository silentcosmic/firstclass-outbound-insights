# 迪拜警方联手比亚迪：AI智能巡逻车亮相

⚡ 迪拜警方联手比亚迪：AI智能巡逻车亮相
迪拜警方近日发布了其第二代Ghiath智能巡逻车，这款整合了先进人工智能技术的新型车辆，旨在全面支持一线执法行动，并显著提升紧急响应效率。
据悉，该新巡逻车是迪拜警方与Al-Futtaim集团携手开发的成果。Al-Futtaim作为比亚迪在阿联酋的独家经销商，此次交付的车辆将基于比亚迪高端子品牌DENZA（腾势）的B8插电式混合动力车型。
