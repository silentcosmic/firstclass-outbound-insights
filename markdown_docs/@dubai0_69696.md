# 阿联酋军事城仓库起火：军方迅速控制无伤亡

🔥 阿联酋军事城仓库起火：军方迅速控制无伤亡
近日，阿联酋扎耶德军事城内一处仓库突发火灾。据军方消息，起火原因系木材与旧弹药燃烧。
阿联酋武装部队消防队迅速响应并成功控制了火情。国防部随后确认，此次事件未造成任何人员伤亡，并提醒各界以官方渠道发布的信息为准。
