# ️ 阿联酋央行再罚外资银行182万迪拉姆：合规监管持续收紧

⚠️ 阿联酋央行再罚外资银行182万迪拉姆：合规监管持续收紧
继此前2000万迪拉姆的反洗钱罚单之后，阿联酋中央银行近日再次对一家在阿联酋运营的外资银行分行开出182万迪拉姆的罚单。
央行声明指出，此次处罚源于该分行未能按规定在七天期限内出具责任函，此举违反了阿联酋中央银行的市场行为及消费者保护法规与标准。尽管涉事银行名称未被披露，但央行重申，其监管职责旨在确保所有金融机构及其人员严格遵守阿联酋法律法规，以维护当地银行业的透明度和金融体系的诚信。
