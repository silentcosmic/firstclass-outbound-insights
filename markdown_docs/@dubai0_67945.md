# 太子集团高管日本被捕：中方回应引关注

🚨 太子集团高管日本被捕：中方回应引关注
据日本媒体报道，东京警方近日逮捕了一名名为“胡石”（音）的男子，该男子被指与柬埔寨太子集团存在关联。太子集团此前曾被控运营多处电信诈骗园区。
针对路透社记者就此逮捕行动及涉案人员遣返的提问，中国外交部发言人郭嘉昆在例行记者会上回应称，具体问题建议向中方主管部门询问。郭嘉昆同时强调，打击网络赌博和电信诈骗犯罪是国际社会的共同责任，中方期望周边国家能与中国加强执法合作，共同维护人民生命财产安全及区域交往秩序。
