# 生成式AI冲击东南亚劳动力市场：近八千万人面临变革浪潮

最新研究洞察指出，生成式人工智能（GenAI）的崛起，正深刻触及东南亚国家联盟（ASEAN）近八千万劳动者的职业生态。
尽管如此，该研究报告强调，目前仅有一小部分劳动力身处暴露风险最高的职业领域，并且截至目前，尚无大规模失业的明确证据。
展望2025年，据测算，东南亚地区总就业人口的22.9%（相当于近八千万劳动者）所从事的职业，将面临生成式AI一定程度的潜在影响。
然而，实际处于“最高暴露类别”的劳动力仅占总数的3.3%，约合1170万人。
与此同时，约67%的就业岗位目前尚未显现出受到生成式AI影响的迹象。
在有数据支持的九个东盟国家中，新加坡的劳动力面临生成式AI影响的比例最高，达到总就业人口的42.2%。
紧随其后的是菲律宾（28.1%），这部分反映了该国偏向服务业和信息技术（IT）产业的经济结构；印度尼西亚（21.7%）；越南（20.8%）；以及泰国（20.6%）。
该项研究还观察到，在东盟地区，高暴露风险职业的就业规模仍在持续增长。
分析报告强调：“劳动力市场转型的潜力巨大，但目前尚未出现普遍性的颠覆迹象。”
当前，生成式AI的采纳仍处于早期且不均衡的阶段，其应用主要集中在技术密集型职业，而在办公室和行政管理等高暴露岗位中的普及率相对有限。
报告特别指出，年轻劳动者（15-24岁）与成年劳动者在生成式AI暴露程度上表现出大致相似的水平。然而，在AI暴露方面存在显著的性别差异。
女性在高生成式AI暴露职业中的就业可能性是男性的两倍多，这与其在文书、行政及专业岗位上的集中度密切相关。
这份研究还发现，区域内存在不均衡的“准备度差距”。其中，新加坡凭借其先进的数字基础设施、充足的人才储备以及政府主导的全面实施战略，正迅速崛起为一个全球领先的AI生态系统。
为确保人工智能创新能切实促进生产力增长、创造高质量就业，并在东盟区域内实现包容和公平的惠益，该报告概述了若干区域性优先事项。
这些优先事项涵盖：以人为本的治理体系；通过扩大技能提升和再培训项目（特别关注女性和青年）来实现包容性技能发展；支持微型、小型和中型企业（MSMEs）克服AI采纳障碍；以及加强成员国之间的知识交流与人力资源开发协调。
“要充分利用生成式AI的益处，仅仅依靠技术获取是不足的。生产力提升有赖于对人力资本和社会保障的投资。归根结底，未来劳动力市场的表现将不仅仅取决于技术暴露程度，更取决于构建劳动者、企业和机构准备度和韧性的政策选择。”——报告主笔经济学家Christian Viegelahn强调。
这份名为《生成式AI与东盟劳动力市场：显著暴露、有限颠覆、不均准备》的报告，通过审视职业暴露程度和生成式AI的新兴采纳模式，深入探讨了其对东盟11个成员国就业和劳动力市场的影响。
