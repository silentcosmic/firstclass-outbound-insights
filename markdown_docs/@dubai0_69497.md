# 广州警方成功阻断280万复合型投资诈骗

🚨 广州警方成功阻断280万复合型投资诈骗
近日，广州市公安局黄埔分局反诈中心通过银行预警机制，成功劝阻一起新型网络投资理财诈骗，为一名市民挽回经济损失280万元。
据悉，黄埔区某银行发现客户侯某异常预约支取280万元现金，遂将线索及时通报警方。民警迅速赶赴银行现场拦截，发现侯某正遭遇“网络交友+虚假投资+线下取现”的复合型诈骗。
受害者侯某因深度洗脑，初期对警方劝阻表现出强烈抵触，甚至误认为警方干预了其“投资大计”，并扬言将通过每日小额取现规避监管。面对僵局，反诈民警随后上门开展二次深度劝阻，耐心细致地逐条拆解诈骗套路。侯某最终彻底醒悟，并承认此前已按骗子指令进行了三次线下取现操作。所幸警方及时介入，成功保住了这笔巨额资金。
