# 诈骗三天即遭解雇，马籍男子狮城获刑16个月

🚨 诈骗三天即遭解雇，马籍男子狮城获刑16个月
近日，一名30岁马来西亚籍男子叶志明因参与柬埔寨跨国网络诈骗集团，被新加坡法院判处监禁16个月零2周。该集团在一年内对新加坡公民实施了多达528起冒充官员诈骗，涉案金额高达4110万美元。
法庭文件披露，叶志明此前于2024年11月前往金边，在每月1800美元底薪及提成的诱惑下加入该集团。尽管集团为其提供了新加坡受害者名单及诈骗话术，但叶志明因“无法自信说出台词”而表现不佳，仅工作三天便被集团辞退，期间未成功骗取任何款项，也未获得报酬。
此后，叶志明于2025年2月前往新加坡务工，同年9月被当地警方逮捕归案，并于6月26日接受审判。
