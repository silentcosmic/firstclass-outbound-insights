# 阿提哈德铁路客运爆火：两天预订破5000张

⚡ 阿提哈德铁路客运爆火：两天预订破5000张
阿提哈德铁路客运服务预订自开放以来，仅两天内便吸引超过5000名乘客完成座位预订。同时，其官方应用程序也迅速攀升至阿联酋免费应用下载榜首，显示出市场对这一全新交通方式的强烈需求。
据悉，首趟客运列车将于6月30日从阿布扎比驶往富查伊拉，标志着阿联酋国家铁路客运时代的正式开启。
