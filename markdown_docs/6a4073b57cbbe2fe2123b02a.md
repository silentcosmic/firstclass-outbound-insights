# 新加坡金管局与银行公会携手擘画PayNow Gen2，定义即时支付系统新浪潮

新加坡金融管理局（MAS）与新加坡银行公会（ABS）周四宣布，双方将共同启动“PayNow Generation 2”（PayNow Gen2）研究项目，旨在探讨新加坡国家即时支付基础设施的四大升级方向。
金管局在一份声明中指出，PayNow的潜在升级将更好地满足消费者、商户、企业及公共机构的需求，使新加坡的国家支付基础设施对用户而言更加顺畅、高效且易于使用。
此次研究通过与37家机构的广泛咨询，并参照11个司法管辖区的最佳实践，明确了四大潜在的升级领域。
首先，提升PayNow与NETS QR之间的二维码互操作性，确保消费者无论使用何种支付方案，都能够扫描并向任何商户完成支付。金管局与银行公会计划在2026年底前启动该项试点。
其次，通过PayNow二维码内的深度链接功能，优化消费者的在线结账体验。
此举将有效缩短支付流程，让消费者在线支付更为流畅，同时也能帮助在线商户减少因支付环节繁琐而造成的销售损失。金管局与银行公会预计在一年内完成此项功能的部署。
第三，通过设定适当交易限额与安全保障措施的沙盒试点，赋能政府机构处理更大金额的公共部门PayNow交易。
这将为消费者和企业向政府机构进行大额支付提供更多便利。金管局与银行公会计划于明年与政府机构一道启动该沙盒项目，并同步落实各项安全保障。
第四，拓展支付能力以满足新兴业务需求，包括“请求支付”（request-to-pay）功能；用于自动化对账的结构化数据字段；微支付通道；更广阔的跨境连接；通过离线支付能力等方式增强支付运营的韧性；以及促进代理商业（agentic commerce）的新功能。
这些功能更多着眼于长期发展，金管局与银行公会将于今年启动相关的基础性工作。
第一阶段的研究成果已详细载于题为《PayNow Generation 2：定义新加坡即时支付系统的下一波创新浪潮》的报告中。
在接下来的工作阶段，金管局与银行公会将与行业参与者紧密合作，将这些洞察和倡议转化为具体的实施步骤，包括为部分选定的升级项目开发试点。
声明还指出，金管局与银行公会将在2026年底前发布一份详细的实施路线图。
新加坡副总理兼贸易与工业部部长、金管局主席颜金勇在演讲中表示，新加坡将通过“PayNow Generation 2”项目，对其即时支付基础设施进行全面升级，使其系统更加无缝衔接、互操作性更强，并为未来发展做好准备。
他强调：“‘PayNow Generation 2’的核心在于确保新加坡的国家支付方案和基础设施能够满足当前需求，并为未来的挑战做好充分准备。”
颜金勇指出，金管局、银行公会及各行业合作伙伴将携手拓展PayNow的功能，其中包括提升不同支付网络之间的互操作性。
他进一步表示：“我们将试点推动这些网络间的互操作，让消费者无论商户采用何种方案，都能扫码支付。”
该项倡议还将通过二维码深度链接技术，优化数字结账体验，有效减少完成支付所需的步骤。
金管局和行业参与者也将探索更多新功能，例如“请求支付”以及支持自动化对账的结构化数据功能，以帮助企业减少人工处理环节。
他补充道，政府还将与各机构合作，针对PayNow在公共部门大额交易中的应用进行沙盒测试。
颜金勇总结道：“尽管如此，我们仍将在今年启动基础性工作，以确保我们的支付基础设施能够适应未来商业模式的演变。”
