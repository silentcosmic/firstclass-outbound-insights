# 东盟齐聚暹粒：誓言联手打击网络诈骗

🚨 东盟齐聚暹粒：誓言联手打击网络诈骗
第29届东盟移民局长暨外交部领事司长会议（DGICM）及系列相关会议近日在柬埔寨暹粒省拉开帷幕。本次会议的核心议题聚焦于区域内日益严峻的跨国犯罪和非法移民问题，特别是对整个地区构成严重威胁的网络诈骗活动。
柬埔寨副总理兼内政部部长苏速卡在主持会议时，敦促东盟成员国以高度责任感，加强协作，共同应对这些挑战。他特别强调，除了区域内部合作，东盟还通过此次会议机制，进一步深化了与中日韩（10+3）的合作，以期有效提升边境管理效能及区域安全水平。
据柬埔寨移民总局局长苏威斯纳介绍，自1996年成立以来，DGICM已成为东盟强化区域对话、便利人员往来及防范跨国犯罪的关键平台。据悉，本届会议期间还将举行多场与中日韩（10+3）的配套会议，进一步巩固合作成果。
