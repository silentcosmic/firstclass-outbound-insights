# 中东四国重申地区安全：美伊协议与巴勒斯坦问题

🌍 中东四国重申地区安全：美伊协议与巴勒斯坦问题
埃及、巴基斯坦、沙特阿拉伯和土耳其四国外长在开罗举行协商会议后发表联合声明，对美伊双方于6月18日签署的《伊斯兰堡谅解备忘录》表示欢迎。声明指出，该备忘录是缓解地区紧张局势、化解此前威胁到地区安全稳定、能源市场、关键航运通道、全球供应链及国际贸易冲突的建设性一步。
四国外长呼吁尽快完成下一阶段谈判，就悬而未决的问题达成“持久、可核查且双方均可接受的解决方案”。他们强调，这一进程应充分考虑地区各国，特别是海湾阿拉伯国家和黎凡特地区的安全与稳定关切，以期强化集体安全、促进地区长治久安。
声明中，四国外长还重申巴勒斯坦问题仍是实现中东持久和平与稳定的核心关键，并重点关注加沙地带、被占领的约旦河西岸及东耶路撒冷的人道主义与政治局势。他们重申支持巴勒斯坦人民的合法权利，包括民族自决权，以及根据1967年6月4日边界建立以东耶路撒冷为首都的独立巴勒斯坦国，此举符合联合国安理会相关决议。
