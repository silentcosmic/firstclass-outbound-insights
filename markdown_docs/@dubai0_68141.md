# 柬埔寨刘任案：深挖跨境诈骗与政商网络

🔍 柬埔寨刘任案：深挖跨境诈骗与政商网络
柬埔寨打击网络诈骗行动持续深入之际，近期被引渡回中国的中国籍商人刘忍（Liu Ren）再次成为焦点。此案不仅牵出刘忍涉嫌参与跨境网络赌博和电信诈骗的指控，更引发外界对柬埔寨境内部分政商关系与犯罪集团之间可能存在关联的广泛质疑。
据了解，50余岁的刘忍长期在柬埔寨经商，曾担任安徽省驻柬埔寨商会主席，并投资经营金边黄山国际酒店等项目。中国公安机关指控其为陈志犯罪集团核心成员，涉嫌自2016年起受陈志指使，组建“金贝集团”并在西哈努克省运营多个网络赌博及诈骗平台，甚至涉及非法拘禁和暴力控制人员。
值得关注的是，刘忍此前曾活跃于柬埔寨商界及慈善领域，甚至以商会主席身份参与人才培养项目并获得官方支持。然而，随着调查深入，部分观察人士质疑，犯罪集团可能正是通过商业投资、慈善捐赠等“合法”活动建立影响力，从而为非法业务赢得发展空间。柬埔寨民主组织人士森·森卡鲁纳（Seng Sencaruna）呼吁，应加强对外国投资背景及资金来源的审查，以防犯罪资金渗透。
国际社会对柬埔寨网络诈骗问题一直高度关注。此前，国际特赦组织曾点名西港部分园区涉嫌大规模网络诈骗，并存在强迫劳动等侵犯人权问题。2025年，美国和英国已对“金贝集团”及关联企业实施制裁，理由包括涉嫌网络诈骗、人口贩运及强迫劳动。美国和平研究所（USIP）报告指出，柬埔寨网络诈骗产业每年或非法攫取逾125亿美元，已成为区域跨境犯罪治理的严峻挑战。
目前，柬埔寨政府正持续推进打击网络诈骗专项行动，已查处多处涉诈窝点。刘忍案的后续调查进展，仍有待中柬两国执法部门进一步公布。
