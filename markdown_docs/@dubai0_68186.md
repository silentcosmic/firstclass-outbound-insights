# 洪森访华：边境争端非核心，金边另有深意？

🔍 洪森访华：边境争端非核心，金边另有深意？
柬埔寨首相洪森近期访华，引发外界对其与泰国边境争端是否会成为核心议题的猜测。然而，据柬埔寨媒体Fresh News独家报道，目前并无迹象表明边境问题将是洪森与中国领导人会谈的核心焦点。
文章指出，尽管如此，此次访问的地缘政治时机仍备受关注。此行正值柬泰边境局势紧张之际，而柬埔寨一直强调通过外交和国际法和平解决争端。在此背景下，加强与中国这一重要战略伙伴的关系，被视为金边方面在地区局势不明朗时期，巩固自身外交与战略地位的宏观布局。
分析认为，洪森此访主要目的在于进一步巩固柬中全面战略合作伙伴关系，深化两国政治互信，而非意图建立针对任何第三方的联盟。
