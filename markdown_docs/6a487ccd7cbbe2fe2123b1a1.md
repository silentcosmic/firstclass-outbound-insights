# 比尔·盖茨洞察：为何人工智能无法取代竞技体育中的人类核心价值

近年来，生成式人工智能（AI）已在职场中确立其地位，尤其在处理重复性任务方面展现出强大的替代能力。去年，Anthropic 首席执行官达里奥·阿莫迪（Dario Amodei）曾指出，AI可能导致入门级白领岗位减少高达50%，这无疑会为Z世代进入劳动力市场带来新的挑战。
即使是微软人工智能首席执行官穆斯塔法·苏莱曼（Mustafa Suleyman），也近期对其关于AI将在18个月内淘汰部分白领工作的论断进行了澄清。他强调此前的表述有所夸大，并进一步阐明，AI的真正作用在于赋能并优化重复、单调的工作流程，而非全面取代人类。
展望2025年，微软联合创始人比尔·盖茨（Bill Gates）曾预测，AI将接手大部分由人类完成的工作。然而，这位科技巨擘及慈善家也同时指出，人类如何选择与这项颠覆性技术互动，其主动权仍掌握在我们自己手中。
值得注意的是，盖茨在参与吉米·法伦（Jimmy Fallon）的《今夜秀》节目时，或许不经意间揭示了第四个极难被AI取代的职业——专业运动员。他直言不讳地指出：尽管人工智能能够替代诸多人类工作，但没有人愿意观看电脑进行棒球比赛。
由此可见，即使在全球瞩目的世界杯等体育盛事中，我们依然将沉浸于欣赏钟爱球员在赛场上挥洒天赋、奉献精彩绝伦表现的时刻。人类运动员所带来的情感共鸣与不可预测性，是冷冰冰的算法无法模拟的。
随着AI技术的广泛应用与普及，它将不可避免地重塑我们的工作模式，甚至深刻地改变“工作”一词的定义。微软研究院去年发布的一项研究报告就曾警示，在全面进入人工智能时代后，多达40种职业将面临被淘汰的风险，其中包括客户服务代表、专业翻译、技术文档撰稿人、内容编辑以及网页开发人员等。
