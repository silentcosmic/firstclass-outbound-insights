# 新中资本市场监管深化：MAS与CSRC签署人员交流协议

🤝 新中资本市场监管深化：MAS与CSRC签署人员交流协议
新加坡金融管理局（MAS）与中国证券监督管理委员会（CSRC）近日正式签署了一项人员交流协议。此举旨在全面深化两国在资本市场领域的监管合作，预计将显著提升新中两地金融市场的互联互通水平与协同发展潜力。
