# 人工智能浪潮驱动马来西亚科技腾飞：产业前瞻与战略布局

多家研究机构指出，人工智能（AI）和数据中心相关的需求将持续成为推动马来西亚科技产业在2026年下半年增长的关键动力。这一趋势得益于超大规模计算厂商的持续投入，以及全球半导体产业在AI投资热潮中的强劲表现。
一份最新报告分析指出，AI及数据中心相关的需求仍是推动马来西亚科技产业下半年增长的核心要素，其背后是超大规模计算厂商持续的资本支出投入。
该分析机构预期，大多数马来西亚科技企业将取得符合市场预期的业绩表现。报告尤其看好与AI/数据中心有直接关联的企业，而关联度较低的细分领域则可能面临相对缓慢的增长。
除了AI和数据中心，该报告还强调了三个额外的增长方向：光子学与光模块正逐步成为主流增长动力；先进冷却、测试及检测解决方案迎来新的投资周期；以及先进封装技术在本地市场获得关注，这预示着有能力成功实施相关战略的企业将实现显著的能力升级和潜在价值重估。
鉴于中东冲突的持续影响，报告对软件行业保持乐观态度。该行业主要立足于国内及区域运营，这为其提供了抵御全球地缘政治风险的防御性缓冲。
“我们对软件行业的乐观预期不变，这主要得益于广泛数字化趋势带来的结构性利好。在此背景下，加之需求的持续支撑，我们预计软件行业在下半年将保持韧性，”该研究机构表示。
报告同时指出，马来西亚在其城市数字化进程中已抵达关键转折点，发展叙事正从“智慧城市”明确转向“AI城市”的构建。
这一转变标志着通过采纳人工智能技术提升城市效率的重大步伐，旨在将马来西亚打造成为该地区领先的“AI城市”强国。
与此同时，另一份近期报告指出，全球半导体产业正迈入人工智能超级周期的全新阶段，预计市场销售额将在2026年突破1.5万亿美元，并于2027年接近1.9万亿美元。
“增长预计将由AI基础设施、高带宽内存（HBM）和先进计算领域的加速投资所驱动，”该研究机构分析道。
与以往主要由个人电脑（PC）、智能手机及消费电子产品复苏驱动的上升周期不同，当前周期日益受到结构性AI资本支出、更高的半导体含量以及在尖端逻辑芯片、存储器和先进封装领域创造更大价值的支撑。
“马来西亚凭借其成熟的半导体生态系统，已准备好参与到这一转型浪潮中，尽管目前其在更高价值的先进封装技术领域的涉足程度仍相对有限，”有分析师评论道。
然而，也有分析机构对马来西亚科技产业持中立观点。该机构认为，半导体产业的基本面正因人工智能的深远影响而变得更加有利。
这与全球半导体月度销售数据保持一致，其持续向世界半导体贸易统计组织（WSTS）预测的2026年1.5万亿美元销售额目标迈进。
“现阶段来看，AI驱动的收入尚未在总收入中占据最大份额。不过，我们预计其最终将占据更大的比重，”该机构补充道。
该机构同时指出，智能手机市场持续面临严峻挑战，内存短缺和地缘政治紧张是主要因素。
但其认为，苹果、三星和华为等巨头相比竞争对手处于更有利地位。此外，关税和外汇波动也可能阻碍企业的盈利表现，报告进一步强调。
另一家分析机构也预期半导体市场的积极势头将得以维持，人工智能相关的需求将继续扮演核心增长引擎的角色。
根据该机构的预测，到2027年，全球半导体销售额有望突破1.9万亿美元，这主要受AI基础设施和加速计算平台持续投资的驱动。
相比之下，传统终端市场如PC和智能手机的前景则较为谨慎，主要原因在于持续存在的内存供应短缺问题。
援引全球数据机构的报告，该分析机构指出，2026年全球PC出货量预计将同比下降11.3%，而全球智能手机出货量则预计同比下滑13.9%，降至约11亿部，这将创下有记录以来的年度最大跌幅。
“内存成本的上涨预计将推高设备价格，这可能进一步抑制消费者需求，”报告补充道。
该分析机构还强调，马来西亚国家半导体战略（NSS）的实施正在稳步推进，政府在加强该国在全球半导体价值链中的地位方面取得了持续进展。
援引马来西亚投资、贸易和工业部副部长沈志强（Sim Tze Tzin）的讲话，该报告提到，在2024年1月至2025年9月期间，国家半导体战略已吸引了599亿马来西亚林吉特（约合146.9亿美元）的半导体相关投资，其中包括568亿马来西亚林吉特（约合129.3亿美元）的外国直接投资和31亿马来西亚林吉特（约合7.6亿美元）的国内直接投资。
展望未来，国家半导体战略将继续聚焦于强化高价值技术领域、培养高素质人才，以及促进本地知识产权的拥有和商业化。
该战略的一个核心目标是，通过从传统的后端活动转向集成电路（IC）设计和晶圆制造等更高价值的前端环节，来提升马来西亚在半导体生态系统中的地位。其中一项重要举措是与总部位于英国的Arm Holdings plc的战略合作，政府近期已向三家本土企业授予Arm设计代币，以加速并增强其IC设计能力。
此外，政府还在2026年预算案中拨出近20亿马来西亚林吉特（约合4.9亿美元），用于支持国家半导体战略的全面实施，推动半导体相关发展计划。
“我们相信这些举措将进一步巩固马来西亚的半导体生态系统，提升本土技术能力，并为本地企业参与更高附加值活动创造广阔机遇，”
“从长远来看，国家半导体战略有望帮助马来西亚在半导体价值链中实现更高层次的攀升，提升该行业的全球竞争力，并使国家在全球AI、先进计算和半导体投资周期中获益，”分析师总结道。
