# Airwallex在H轮融资后强化财务领导力：前高管Pranav Sood回归出任首席财务官

金融科技巨头Airwallex近日宣布，前高管Pranav Sood已再度回归公司，出任首席财务官（CFO）一职。此举旨在其近期成功完成融资后，进一步强化核心财务领导团队。
Sood先生此前任职于贝恩资本（Bain Capital）的成长型股权基金，专注于与英国及欧洲地区的科技企业合作。
此次任命标志着Sood先生重返Airwallex。他此前曾深度参与公司在欧洲、中东和非洲（EMEA）地区的业务拓展，并负责全球市场营销职能。
凭借其丰富的投资经验以及对Airwallex商业运营模式的深刻理解，Sood先生此次回归将为公司带来宝贵的战略洞察。
Sood先生的办公地点将设在伦敦。伦敦作为Airwallex国际化扩张战略的关键枢纽之一，其战略地位日益凸显。Sood先生表示：“Airwallex是一家引领时代变革的企业，正在重塑全球银行业的基础架构。此次回归Airwallex，如同回到一个比以往任何时候都更为重要的使命之中。”
他进一步指出：“我的目标是确保我们的财务职能成为推动创新与规模化增长的催化剂，提供坚实的框架支撑，助力我们巩固在全球金融经济中的基石地位。”Airwallex首席执行官兼联合创始人Jack Zhang评论道：“随着Airwallex迈入新的增长阶段，我们需要一位既是商业战略家又是财务管理者的高级财务负责人。Pranav正是这样一位难得的领导者，他能将严谨的财务纪律与建设者思维完美结合。”
Zhang先生补充道：“他在贝恩资本的经验，结合他对公司商业驱动因素和金融科技垂直领域的深刻洞察，使他成为引领我们下一阶段财务战略的理想人选。”
Airwallex近期完成了一笔3.2亿美元的H轮融资，公司估值由此达到110亿美元。
此外，Airwallex已承诺投入11亿美元用于其在欧洲、中东和非洲（EMEA）地区的扩张，并计划在英国和荷兰招聘约100名高级工程技术人员。
