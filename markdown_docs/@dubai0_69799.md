# 武汉女投资者遭诈骗：警方截获百万现金！

🚨 武汉女投资者遭诈骗：警方截获百万现金！
近日，武汉青山区警方成功侦破一起电信诈骗案。一名女投资者轻信网络“证券分析师”推荐，被诱骗投资并损失五十万元现金，警方迅速行动，不仅抓获涉案嫌疑人，更现场截获高达一百万元的涉案资金。
据警方通报，受害人张女士在网上结识一名自称“证券分析师”的男子，对方以高额收益为诱饵，劝说其投资。7月9日，张女士将五十万元现金交付给一名自称证券公司员工的崔某。然而，在未等到承诺收益并被要求追加投资时，张女士警觉发现异常，随即于7月11日报警求助。
武汉青山区警方迅速成立专案组，通过沿线监控锁定嫌疑人崔某。警方发现崔某居无定所，并已购买离汉火车票，有携款潜逃迹象。在铁路公安协助下，警方于当日下午成功截停正乘坐网约车转移赃款的崔某，现场查获装有一百万元现金的行李箱及六套伪造的证券公司工作证件。
崔某交代，其按照上线安排，冒充“证券分析师”负责线下取款，并将现金转交接头人，从中获取约9000余元跑腿费。目前，崔某因涉嫌掩饰、隐瞒犯罪所得，已被警方依法刑事拘留。此案提醒广大投资者，务必提高警惕，防范各类金融诈骗。
