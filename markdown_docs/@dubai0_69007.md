# 涉案超31亿！逃亡12年韩国网赌头目阿联酋落网

🚨 涉案超31亿！逃亡12年韩国网赌头目阿联酋落网
韩国当局近日宣布，已成功从阿联酋引渡两名涉嫌运营大型网络赌博团伙的头目。此举标志着韩国在打击境外网络赌博方面国际合作的深化。据韩国中央日报7月4日消息，一个由多部门组成的跨国犯罪工作组与阿联酋方面合作，最终将嫌疑人遣返回韩国。
其中一名主犯被控运营非法赌博网站，涉案金额高达约4.8万亿韩元（约合31.4亿美元），并涉嫌洗钱。调查显示，此人自2014年逃离韩国后，在菲律宾、马来西亚、柬埔寨等地藏匿长达12年，最终在阿联酋被捕。该嫌疑人还面临约660亿韩元的逃税指控，以及提供吸食毒品、与赌博相关的嫖娼等多项罪名。当局表示，后续还将调查此案是否与2018年一名韩国公民在马来西亚的死亡案件存在关联。
另一名嫌疑人运营的赌博网站涉案金额约5000亿韩元。检方指出，此人通过韩国境内的同伙，招募初高中学生充当销售代理，为非法赌博平台招揽客户，手段极其恶劣。
