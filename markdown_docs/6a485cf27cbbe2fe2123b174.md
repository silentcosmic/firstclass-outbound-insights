# 出海情报速递 | 东南亚创投月报：2026年6月融资额创历史新高，晚期轮次主导市场

2026年6月，东南亚区域的月度融资追踪报告揭示了一次资本部署的创纪录激增。当月总融资额高达38.4633亿美元，共完成18轮融资。这一数字与2026年5月的2.2412亿美元相比，环比增长了惊人的1616.2%；而与去年同期（2025年6月的2.4503亿美元）相比，增幅更是达到了1469.8%。
尽管融资轮次数量仅实现温和增长（本月18轮，而2026年5月为14轮，2025年6月为19轮），但此次资本激增几乎完全由少数几笔十亿级和亿级美元的晚期轮融资推动，而非由广泛的交易量扩张所致。
6月份的资本呈现出压倒性地集中于成熟企业。晚期融资占据了当月总资本的绝对主导地位，贡献了86.3%（约33.2亿美元）。早期融资占比10.4%（约4.0093亿美元），而种子轮融资则构成了剩余的3.3%（约1.254亿美元）。这里所指的晚期阶段涵盖了C轮及C轮之后的融资以及私募股权（PE）投资。
当月，区域内共录得1宗首次公开募股（IPO）事件，相较于5月份的2宗有所下降；同时发生了1宗并购交易，与5月份持平，但远低于2025年6月的7宗。
本月的融资头条无疑是DayOne完成的一轮巨额融资，仅此一笔就占据了当月总融资额的约65%。DayOne与Supabase两家公司的融资总和，更是构成了全部募资的约78%。
主要融资交易包括：DayOne获得25亿美元；Supabase获得5亿美元；Airwallex获得3.2亿美元；SiliconFlow获得2.94亿美元；Striding AI获得1亿美元；Respond.io获得6250万美元；Gero获得1700万美元；Libeara获得1360万美元；kAIgentic获得1000万美元；以及BeeX获得770万美元。
新加坡继续作为东南亚初创生态系统的核心引擎，几乎吸纳了当月全部的区域资本。
具体来看，Airwallex在新加坡完成3.2亿美元融资；Monk’s Hill Ventures对新加坡的BeeX投资770万美元；JRE Ventures向新加坡的H3Zoom.AI注资360万美元。更引人注目的是，Hillhouse（高瓴）投资了新加坡的DayOne，金额高达25亿美元；GIC（新加坡政府投资公司）则对Supabase进行了5亿美元投资，该公司同样位于新加坡。
核心洞察
本月融资创下历史新高：6月总融资额较5月激增逾17倍，这主要得益于少数几笔超大规模的晚期和私募股权融资。如果剔除DayOne和Supabase这两笔巨额交易，当月融资额约为8.46亿美元，这仍是一个强劲但更符合常规水平的月份。
晚期融资主导市场：超过33亿美元的资金，占总资本的逾86%，流入了成熟企业。其中，DayOne的25亿美元数据中心项目融资和Supabase的5亿美元募资最为突出。
新加坡主导地位显著：新加坡的初创企业占据了当月录得美元价值的约98%，凸显了该城邦在晚期和私募股权流动性方面的集中优势。
AI基础设施热潮延续：除了几宗超大型交易，资本还集中流向了人工智能和基础设施领域的企业，包括SiliconFlow、Striding AI、kAIgentic和Gero。这预示着投资者对该地区AI技术栈的兴趣依然浓厚。
