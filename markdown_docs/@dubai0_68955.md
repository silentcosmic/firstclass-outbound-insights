# 美国货币新变局：特朗普签名首登美元，肖像钞提案引争议

⚡ 美国货币新变局：特朗普签名首登美元，肖像钞提案引争议
近日，美国财政部宣布，为纪念建国250周年，将首次发行印有现任总统签名的美元纸钞。最新公布的100美元新样张显示，美国总统特朗普的亲笔签名赫然位于财政部长斯科特·贝森特签名之上，此举打破了以往仅由财长和国库局长签名的惯例，标志着美国货币设计的一项重要突破。
在此背景下，共和党议员此前曾提出一项争议性法案，旨在突破“只有逝者才能登上美钞”的法律限制，提议推出印有特朗普肖像的250美元建国纪念钞。然而，该提案在国会仍需争取民主党人的支持，其能否最终通过，仍是未知数。
