# ️ 柬埔寨遣返维权人士：或永久禁止入境

⚠️ 柬埔寨遣返维权人士：或永久禁止入境
据市场消息，一名此前在柬埔寨被拘的年逾六旬的“汇旺”相关维权人士，或将面临快速遣返。
分析人士指出，柬埔寨当局此举或意在通过遣返措施，对相关活动人士实施永久性入境禁令，从而有效遏制潜在的“麻烦制造者”。
尽管现场并非仅此一人，但当局选择性拘捕并迅速遣返，可能旨在树立一个典型案例，以震慑其他潜在的维权参与者。此类遣返通常不设申诉渠道。即便要求解释，当局也可能以“扰乱公共秩序”为由回应，但实质上，遣返决定更多体现了其行政裁量权，无需详细说明。
有观察人士预判，若街头维权活动持续，柬埔寨当局或将采取进一步行动，通过小规模、快速的逮捕及连夜驱逐，直至平息此类集会。
