# ️ 柬埔寨遣返亲历者忠告：平安回国为上，切勿口无遮拦

⚠️ 柬埔寨遣返亲历者忠告：平安回国为上，切勿口无遮拦
鉴于当前柬埔寨局势复杂，近期有从该国遣返的亲历者分享经验，提醒仍在等待遣返或办理手续的人员，务必将平安回国放在首位。若未涉及国内重大案件，切勿抱有侥幸心理，应尽快配合流程返乡。
这位五月完成遣返流程的亲历者强调，在整个过程中务必保持冷静，谨言慎行，切勿夸大自身身份或经历。其核心建议包括两点：
首先，在接受调查时务必保持低调。集中遣返后，相关人员可能会被分流至户籍地或办案地公安机关进一步调查。面对询问时，应避免吹嘘曾担任组长、主管等管理职务，更不可替他人承担责任，以免因言语不慎招致不必要的麻烦。同时，面对诱导性提问或心理施压，切勿慌张，更不能为求速结而承认不属于自己的事情。个人所做、所处单位及工作时长等信息，应提前梳理清晰，并如实说明。
其次，正式做笔录时须保持清醒，确保表达清晰一致。进入正式询问程序后，应认真倾听每个问题，回答时务必有条理、前后连贯。笔录完成后，务必仔细核对内容，若发现与自身陈述不符，须当场提出修改，确认无误后再签字。切勿编造经历或与他人串通口供。涉及案件事实，应如实陈述；如遭遇不当对待、诱导认罪或其他异常情况，可依法提出异议并申请法律援助。
总而言之，平安归国才是最重要的。亲历者指出，即使回国后从事普通工作，每月收入数千元，也远胜于在海外终日提心吊胆、将个人安危置于险境。境外的所谓高薪往往伴随着巨大的未知风险，安全至上，尽早回家，切勿再以生命为赌注。
