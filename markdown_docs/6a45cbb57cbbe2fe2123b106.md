# 智能视觉创新突破：众见科技获数千万融资，韶音、高秉强等巨头布局智能变焦眼镜赛道

市场痛点：视觉退化与现有方案局限
据本刊独家获悉，在智能视觉科技领域崭露头角的众见科技已成功完成数千万元人民币的三轮融资。此轮战略投资的参与方阵容强大，包括知名科技品牌韶音、高秉强教授旗下的高锋耐心基金、昊辰资本、SEEFund、A股上市公司光峰科技，以及高秉强教授个人投资。据悉，此次募集资金将主要用于核心技术的研发深化与产品迭代，目前其初代智能变焦眼镜产品已成功推向市场。奇域资本在此轮融资中担任众见科技的独家财务顾问。
众见科技的创始人赵鹏先生，拥有卓越的学术和产业背景。他曾荣获第26届全国物理竞赛金牌，并在清华大学电子工程系完成了本科及博士阶段的学习，期间对新型显示技术和微纳光学光电子应用进行了深入研究，并发表了多篇高水平SCI论文。在创办众见科技之前，赵鹏先生曾在光峰科技（股票代码：688007）担任研究院负责人，主导公司前沿技术的研发与产业化进程，尤其在新型显示和AR眼镜领域深耕。他的职责涵盖从团队组建、技术创新、产品规划、研发落地到商务对接的完整闭环，对智能眼镜的上下游产业链具备深刻理解，并积累了丰富的传统眼镜行业资源。
当前，伴随信息技术浪潮的迭起，80后、90后群体中近视患病率显著攀升。更值得关注的是，“近视老花共存”已成为部分步入初老阶段消费者所面临的普遍痛点。传统的解决方案通常涉及佩戴两副不同功能的眼镜，或选择市场上的渐进多焦点眼镜。然而，渐进多焦点方案作为一种度数分区设计，其单一镜片虽能同时覆盖远、中、近视野，却普遍存在可用视野受限、易产生眩晕感等弊端，同时，其在渠道验配和终端售价方面成本亦相对较高，未能有效解决用户体验及经济性上的挑战。
技术破局：电致变焦光学器件的产业化机遇
与此同时，随着光学技术工艺和相关产业的持续精进，电致变焦光学器件的潜力日益凸显。该技术能够实现单片镜片的无缝变焦，大幅提升集成度，并显著减小产品尺寸与重量，同时有望大幅降低生产成本。这种能够提供更优佩戴体验的电致变焦镜片，正逐步迈向产业化的关键临界点，为未来智能视觉产品大规模进入终端应用市场奠定了坚实基础。敏锐洞察到这一变革性机遇，赵鹏先生毅然决然投身创业浪潮，旨在将此创新技术付诸实践。
众见科技的创新原理：电控变焦技术解析
赵鹏先生对众见科技电控变焦眼镜的核心原理进行了深入阐释。他指出，其核心在于通过精密的电信号控制，实现镜片的动态变焦。
“我们的智能眼镜内部集成了多个传感器，它们协同工作，实时感知用户的运动状态、方位及视角。其中，测距传感器是关键部件，它能根据用户注视物体的远近，自动调整镜片度数，实现无感切换。”
在谈及为何选择智能眼镜赛道时，赵鹏先生分享了他的个人洞察与行业情结：
“我与眼镜产业的渊源颇深，有亲戚在眼镜制造重镇丹阳工作。大学毕业后，我选择加入一家智能眼镜公司，当时便坚信眼镜的智能化是未来的大趋势，因为它天然是人类获取信息的重要视觉入口。”
“我一直在思考，人们佩戴眼镜的根本目的是什么？除了附加的智能化功能，眼镜最基础的视光学功能——即帮助人们看得更清晰——在现有技术下，并未得到完美满足。我观察到身边不少40多岁的朋友已开始出现老花，业内常说‘43岁，过眼关’。老花的本质是人眼动态对焦能力的逐渐丧失，此时他们急需一副可变度数的眼镜。因此，我希望能通过创新的技术和光学发展，为大家解决这些实际的视力问题。”
克服挑战：动态环境下的精准变焦与用户体验
面对用户对于动态环境下变焦精准度和佩戴体验的疑问，赵鹏先生进一步解释了其技术方案：
“这是一个非常好的问题。首先，我们的眼镜内部集成了惯性传感器和姿态传感器。当用户进行简单的手部动作或微小移动时，这些传感器的意图是可识别的。因此，在非主动变焦的场景下，眼镜会智能过滤掉大量的‘噪声’信号，避免不必要的变焦操作。”
“其次，我们的智能变焦眼镜在佩戴体验上，其变焦度数的切换与单光眼镜的感知类似，这意味着它的适应性远优于市场上传统的上下分区式渐进多焦点眼镜。目前，我们还在积极开展眼动追踪相关技术研究，未来有望通过眼球注视点来调节度数，进一步有效避免佩戴者的眩晕感。”
技术壁垒与众见科技的创新优势
谈及智能变焦光学眼镜研发面临的核心挑战，赵鹏先生直言，光学设计是其最大的难点所在。
“核心挑战在于如何将复杂的光学系统巧妙地集成到眼镜这一轻薄载体中。不同于手机或相机可拥有较大的光学模组空间，眼镜的设计限制极大。为此，我们建立了全栈自研的光学设计能力和工艺流程来攻克这一难题。”
“此外，变焦速度和功耗也是决定产品用户体验的关键因素。人眼对响应速度有极高要求，快速变焦是解决眩晕感的必要条件。我们为此自主研发了核心变焦组件，旨在实现全球领先的功耗控制、响应速度和光学性能表现。”
“当前，智能变焦眼镜仍是一个新兴领域。尽管海外有一些初创企业涉足，但尚未出现真正成熟且具备市场竞争力的产品。部分现有产品存在光学口径过小、视野范围受限等问题，实用性不足。众见科技能够领先于市场和同行的核心优势，在于我们在动态变焦光学领域拥有深厚的底层创新，这亦是我们投入时间与精力最多的研发方向。”
产品演进路径与广阔的市场前景
展望未来，赵鹏先生详细描绘了众见科技的产品发展蓝图及市场策略：
“我们的终极目标是让智能变焦眼镜在形态上趋近于普通眼镜。然而，任何新品类都有其自身的发展节奏。在早期阶段，我们可能会优先以强调功能性的方式推出产品，确保核心功能率先实现。当然，我们同样重视产品外观设计，初期将更倾向于科技感风格，并集成全自动化与强交互体验。随着后续迭代，产品形态将逐步向轻量化、简洁化方向发展。”
“从长远来看，仅现有光学眼镜市场便已是一个千亿级的庞大市场。在产品推广初期，我们将通过品类创新来切入并深耕这一市场。未来，我们计划在现有的核心功能之上，进一步叠加人工智能（AI）功能，从而拥抱一个更具规模、更广阔的市场空间。”
