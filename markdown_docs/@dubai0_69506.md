# ️ Gate追赃风波：资金竟流入受制裁洗钱平台

⚠️ Gate追赃风波：资金竟流入受制裁洗钱平台
此前，加密货币交易平台Gate.io曾表态将协助受害者追回被盗资产。然而，知名链上追踪专家ZachXBT的最新调查却揭示了令人担忧的走向。
据ZachXBT披露，这些受害者的资金已被转移至Xinbi平台。Xinbi被指为亚洲最大的洗钱黑市之一，并曾受到英国政府的制裁。与此类平台发生交易，通常会导致正常用户面临来自各大交易所及监管机构的严格风控审查乃至制裁。
