# 大马重拳整治非法外籍：超1.7万人被捕

🚨 大马重拳整治非法外籍：超1.7万人被捕
马来西亚民众对吉隆坡街头大量外籍人士聚集深感担忧，尤其在节假日，吉隆坡市中心等核心区域常被外籍人群挤满，引发社会对非法滞留问题的广泛争议。
对此，马来西亚移民局总监拿督扎卡利亚·沙班召开新闻发布会澄清，指出节假日外籍人员集中出行造成视觉冲击，并强调绝大多数外籍人士持有合法居留证件。他透露，当前马来西亚合法外籍人口已突破300万大关。
然而，官方披露的执法数据仍显严峻。截至今年5月25日，全国已累计开展5010场专项清查行动，筛查71734人。行动中，共计逮捕了17313名非法移民，同时有789名雇主因聘用无证外籍人员被捕，主要违规行为包括签证滥用及长期逾期滞留。
被捕的无证外籍人员中，印尼籍以5578人居首，其次是孟加拉（3533人）、缅甸（2856人）、菲律宾（1990人）和巴基斯坦（811人）。
扎卡利亚坦言，民间对外籍人员涌入的不满情绪持续高涨。移民部门将持续加大巡查和抓捕力度，但执法过程中，无证外籍人员常四散逃窜，给一线工作带来巨大挑战。
