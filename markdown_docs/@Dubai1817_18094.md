# 东南亚回国普查升级：警惕出境限制

🚨 东南亚回国普查升级：警惕出境限制
近期市场消息指出，国内各省市县级公安机关已陆续接到上级指示，要求对从东南亚地区回国的人员进行全面核查。
有此前从菲律宾回国两年的人士反映，上周突然接到当地派出所电话，被要求核实其在菲律宾期间的活动及未来出行意向，并对手机通讯记录进行了检查。据多方交叉证实，这并非孤立事件，多位同期回国人员也接到了类似的核查通知，表明这是一项正在全国范围内展开的普遍性行动。
业内人士分析，此次普查可能旨在加强边境管理及维护社会稳定。对于相关回国人员，建议在配合调查时，务必注意个人信息保护。同时，若被问及是否会再次前往东南亚，策略性地表达可能有助于避免潜在的出境限制风险。
