# 杜罗夫第四次受讯，Telegram深陷法律漩涡

🔍 杜罗夫第四次受讯，Telegram深陷法律漩涡
Telegram创始人帕维尔·杜罗夫近日在巴黎接受了长达六个多小时的第四次司法讯问。自2024年8月被正式指控以来，这是他第四次面对法国司法部门。杜罗夫的三名律师随后强调，近两年过去，法国及欧洲司法机构仍未能提供任何有效证据支持相关指控，并已就此提出投诉。
法国当局指控杜罗夫未能充分审核Telegram平台上的内容，且未配合执法部门，认为其平台涉嫌与儿童色情传播、毒品交易和诈骗等犯罪活动存在关联。尽管巴黎检察官今年5月表示调查仍在进行中，且杜罗夫仍受司法控制，但自2025年11月起，法国已解除对他的行动限制，他目前可自由离境。
