# 斯里兰卡警方重拳：严查“奢华生活”不明财富

🚨 斯里兰卡警方重拳：严查“奢华生活”不明财富
斯里兰卡警方近期进一步强化了对非法所得资产的调查力度，并向社会公众发出呼吁，积极举报那些生活方式异常、财富来源不明的可疑人员。此次行动重点关注涉嫌通过违法犯罪活动获取巨额财富的本国公民及在斯外国人。
警方明确指出，凡是短期内资产暴增、频繁出入高档场所、驾驶豪车、居住豪宅，却无法合理解释合法收入来源的个人，都将成为执法部门的重点调查对象。此举旨在协助查处涉嫌非法获利、洗钱及其他跨国犯罪活动，以维护金融秩序和国家安全。
