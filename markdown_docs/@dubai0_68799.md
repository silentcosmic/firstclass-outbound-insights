# 缅甸猛拉老街严打升级：不亚于柬埔寨风暴

🚨 缅甸猛拉老街严打升级：不亚于柬埔寨风暴
据市场消息，缅甸掸邦东部的猛拉（Mong La）与果敢地区的老街（Laukkai）近期正经历一场大规模的打击行动。此次行动的力度与范围，被业内人士认为丝毫不弱于此前在柬埔寨针对网络诈骗及非法产业的整治风暴。
这预示着东南亚地区针对非法经济活动，尤其是电信诈骗和跨境赌博的跨国联合执法态势正在持续加强。
