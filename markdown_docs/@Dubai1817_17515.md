# 美伊谈判取得突破：同意建立直接军事沟通渠道

🤝 美伊谈判取得突破：同意建立直接军事沟通渠道
美国参议员JD Vance近日透露，美伊双方在瑞士谈判中取得一项重要突破：原则上同意建立直接军事沟通渠道，旨在有效避免未来地区局势升级。Vance指出，此举旨在伊朗方面设立一个冲突降级沟通机制。
据Vance称，伊朗已同意派遣伊斯兰革命卫队代表前往多哈，与美国中央司令部人员会面，以处理潜在争端。
