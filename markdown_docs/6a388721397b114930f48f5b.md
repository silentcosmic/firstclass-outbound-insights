# 上市公司股权收购警示：*ST海利紧急修订交易条款应对监管问询

6月21日晚间，*ST海利就其控股子公司以9360万元收购泽尔顿公司60%股权的交易，紧急签署补充协议，以回应上海证券交易所的监管问询。此项交易的核心争议点在于标的公司高达352%的估值增值率，与低于2025年预期的业绩承诺形成显著反差。
进一步审视显示，泽尔顿公司高达90%的收入依赖海外销售，其中超过七成通过间接出口实现，其核心客户信息披露亦存在不透明状况。面对这些质疑，*ST海利方面坚称其业绩承诺具备合理性。同时，为强化自身权益保障，公司对原协议中的部分关键条款进行了紧急修订。
娱乐产业观察：本年度端午档电影市场表现亮眼
根据最新市场数据显示，截至本年度6月21日晚21时，中国电影市场在端午假期档取得了显著成就。本年度端午档总票房达到3.94亿元人民币，观影总人次突破1063.7万，而总场次更是创下145.4万场的历史新高，刷新了中国影史端午档的最高场次纪录。
在票房表现方面，本年度端午档的五部领先影片分别为《玩具总动员5》、《给阿嬷的情书》、《抓特务》、《火遮眼》以及《我看见两朵一样的云》。这些影片的联合贡献，共同推高了假期档的市场热度与整体表现。
