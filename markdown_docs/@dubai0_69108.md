# ️ 警惕：厦门旅行社涉嫌熟人诈骗，资金风险预警

⚠️ 警惕：厦门旅行社涉嫌熟人诈骗，资金风险预警
近期，市场流传一则针对旅行社同业及客户的警示信息，指出厦门一家名为“厦门任意飞CC”的旅行社及其负责人谢锦芬（又称谢CC）涉嫌诈骗。据消息人士透露，谢锦芬长期活跃于厦门、迪拜、吉隆坡等地的商业圈，被指专门利用熟人关系进行欺诈。
有受害者爆料称，其于6月26日向谢CC转账7万USDT，并有聊天记录证实收款。然而，次日该受害者即遭对方拉黑。谢CC随后以“钱包被盗”为由，仅退还3万人民币，其余款项则拒不承认，被指设局侵吞资金。该信息提醒广大同行及客户，务必提高警惕，规避潜在的合作风险。
