# Fortinet任命资深高管西蒙内利，擘画亚太区销售新格局

全球领先的网络安全解决方案提供商Fortinet近日宣布，任命资深行业专家卢卡·西蒙内利（Luca Simonelli）为公司高级副总裁，全面负责其亚太区的销售组织，业务范围涵盖日本、北亚及南亚市场。
西蒙内利先生将常驻新加坡，其领导职能还将扩展至东南亚、印度、南亚区域合作联盟（SAARC）成员国以及澳大利亚和新西兰等关键市场。
在新岗位上，西蒙内利先生的核心职责将集中于制定区域市场战略、深化客户与合作伙伴关系，并推动整体业务的持续增长。
西蒙内利先生拥有逾三十年的科技行业深耕经验，尤其在网络安全、网络解决方案、安全访问服务边缘（SASE）、云安全以及托管服务等领域积累了深厚的专业知识。
在此次履新之前，他曾在GCX Managed Services担任高级领导职务。
此外，西蒙内利先生的职业足迹遍布欧洲、中东、非洲及亚太地区。他此前曾在Fortinet任职近八年，并成功领导了公司在欧洲、中东和非洲（EMEA）区域的业务拓展。“我非常高兴能再次加入Fortinet。我期待与亚太区的客户、合作伙伴及团队紧密协作，在公司现有强劲发展势头的基础上，助力更多企业安全地迈向数字化转型的下一阶段。”西蒙内利先生谈及此次回归时表示。
