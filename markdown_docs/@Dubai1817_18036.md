# 东南亚移民新政：签证滥用成打击重点

🚨 东南亚移民新政：签证滥用成打击重点
东南亚多国正全面收紧移民执法与口岸审查，重点打击利用旅游签、免签及落地签进行非法务工、长期居留或频繁“跑签”等行为。
马来西亚移民局近期在全国范围内展开大规模联合执法，共检查2260名外籍人士，扣留503名涉嫌逾期居留、滥用准证等违规人员。执法范围已扩大至工厂及用工单位，柔佛州突查工厂并扣留80名外籍人员即是最新例证。
柬埔寨亦持续推进清查，仅7月上旬便驱逐了来自24个国家和地区的1362名外籍人员，主要涉及违反移民规定及非法居留。
泰国正进一步强化机场及边境口岸入境审核，重点核查资金证明、返程机票、住宿地址、旅行计划及频繁出入境记录。对于入境目的不明或材料不全者，移民部门有权拒绝入境。此外，泰国内阁已批准调整免签及落地签政策，包括取消部分国家60天免签待遇，并缩减30天免签及落地签适用范围。具体实施时间待《皇家公报》公布。
行业观察人士提醒，前往东南亚的旅客务必提前了解当地签证及准证适用范围，严禁利用旅游签从事工作或经营活动。最终入境许可仍由口岸移民官员依法决定。
