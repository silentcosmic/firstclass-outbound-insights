# 迪拜RTA推出上门验车服务：便利升级，全阿联酋覆盖

📣 迪拜RTA推出上门验车服务：便利升级，全阿联酋覆盖
迪拜道路与交通管理局（RTA）近日正式启动“Aber”移动验车服务，旨在大幅提升车辆年检的便利性。这项创新服务允许客户在阿联酋全境内的家中、办公室或指定地点进行车辆检测，省去了前往验车中心的麻烦。
用户可通过RTA官网、官方App或其他数字平台轻松预约。服务涵盖车辆年检、过户验车、出口验车及车辆信息更新等多种类型，并由RTA认证的专业检验员携带全套设备上门提供服务。单次预约最多可同时检测9辆车。
该服务的起步价为370迪拉姆，具体费用会根据所选项目有所调整。
