# 新加坡气枪伤人案：嫌犯家中搜出大量武器

🚨 新加坡气枪伤人案：嫌犯家中搜出大量武器
近日，新加坡淡滨尼发生一起气枪伤人案，一名38岁男子因涉嫌使用气枪袭击一名37岁男子，已被警方逮捕。受害者在此次事件中耳朵和额头受伤。
警方通过初步调查和闭路电视监控，迅速锁定了嫌疑人身份并实施逮捕。随后，警方在其住所内查获了大量武器和违禁品，包括多支气枪、弹药、指节套环以及疑似电击装置等。该男子将于7月1日被控“使用危险武器蓄意伤人”罪，最高可判处七年监禁，并可能面临鞭刑或罚款。
