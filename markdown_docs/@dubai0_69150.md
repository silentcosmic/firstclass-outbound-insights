# ️ 马来西亚酒店未挂中国国旗引争议，视频已删除

⚠️ 马来西亚酒店未挂中国国旗引争议，视频已删除
7月4日，一位中国抖音博主发布视频，称在马来西亚某酒店早餐区发现，其走廊悬挂的各国国旗中唯独不见中国国旗。该博主随即当场质问酒店工作人员，表示：“如果你们想赚中国人的钱，那你们就要悬挂中国国旗。”
该言论迅速引发网友热议。目前，相关视频已被删除，且该博主在抖音平台已关闭搜索功能，其账号无法通过关键词检索。此事件再次凸显了海外旅游服务业在面对中国游客时，需审慎处理文化敏感性与品牌形象管理。
