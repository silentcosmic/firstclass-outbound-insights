# 金边森速区夜查：重点打击毒品枪支

🚨 金边森速区夜查：重点打击毒品枪支
2026年6月28日晚，柬埔寨金边市森速区联合行政指挥部开展了一场专项检查行动。
此次行动在森速区郭克良分区1019号路森林管理局办公点前设立检查站，主要针对过往人员和车辆进行毒品、枪支及爆炸物检查，旨在维护当地社会治安，确保人民群众的生命财产安全。
