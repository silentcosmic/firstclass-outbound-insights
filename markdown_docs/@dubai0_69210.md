# ️ 阿曼苏哈尔别墅区突袭：近200人被捕

⚠️ 阿曼苏哈尔别墅区突袭：近200人被捕
市场消息显示，7月6日凌晨4时左右，阿曼苏哈尔多个别墅区遭遇突击检查，据称近200人被捕。
知情人士透露，被捕者以越南籍人士为主，另有少量中国籍人员。目前尚不确定后续处理方式，但大概率将面临遣返。此次行动或早有部署，不排除是长期摸排后的收网行动，亦有消息指向举报。据悉，5日下午已有部分别墅收到检查通知，至凌晨时分，未能撤离的区域随即遭到突袭。
