# ️ 越南胡志明周边电诈团伙覆灭：多名通缉犯被遣返

⚠️ 越南胡志明周边电诈团伙覆灭：多名通缉犯被遣返
近日，有市场消息指出，越南胡志明市富美地区一处电信诈骗窝点被当地警方成功捣毁，现场20余名涉案人员被悉数抓获，其中包括一名公司股东。据了解，此次行动是在社区邻居与物业的举报协助下，由警方上门查处。
一名涉案人员透露，他于今年1月通过非正规渠道自老挝万象入境越南，后在富美找到一家“精聊盘”（情感诈骗）公司。该团伙在别墅内办公，曾一度获利，并向国内汇寄资金。
越南警方对此类案件处理高效。涉案人员被捕仅3天后，警方即与中国大使馆核实身份，发现其中多人系在东南亚多国流窜5年以上的中国境内通缉犯。根据中越两国的情报共享机制，特别是针对电信诈骗犯和通缉犯，一旦被捕，通常会在一个月内完成遣返程序。
目前，该团伙所有成员已于本月被遣送回中国。此事件再次警示海外不法从业者，跨境执法合作日益紧密，相关人员回国后亦将面临法律追责。
