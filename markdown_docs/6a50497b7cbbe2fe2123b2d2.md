# 新加坡金融管理局优化审批流程 助力新型基金市场准入提速

新加坡金融管理局（MAS）提出了一系列监管改革，旨在加快新型零售基金产品的审批速度。此举意在促进投资管理行业的创新发展，同时坚守对零售投资者的保护机制。
该局于周四发布了一份征询意见稿，就《集体投资计划守则》（CIS Code）的拟议修订方案征求各方反馈。该守则主要规范在新加坡发行的零售投资基金。
这些拟议的调整背景在于投资者成熟度的提升及投资需求的多样化。市场对更广泛基金产品的需求日益增长，而这些产品可能不再完全契合现有的监管框架。
新加坡金融管理局指出，此次修订旨在通过赋予现有投资要求更大的灵活性，并吸纳更多创新型基金结构，从而推动基金产品创新，拓宽投资者选择。
一项关键提议是在《集体投资计划守则》中引入新的“另类基金附录”。该附录将为创新型基金产品提供一个专门的框架，将其与传统零售基金区分开来，并根据其独特的风险特征设定具体的监管要求。
金融管理局强调，投资者保护措施仍将是新框架的核心要素。这包括针对不同基金类型量身定制的特定要求，以及强化的信息披露义务，以确保投资者获取充分信息，做出明智的投资决策。
同时，适用于零售基金的核心监管标准将继续生效。这涵盖了资产保管、流动性管理和公司治理等方面的要求，以及基金管理人和分销商在投资产品设计和分销过程中需遵守公平交易原则的义务。
新加坡中央银行表示，此次拟议的框架旨在平衡鼓励金融创新与维护投资者保护之间的关系。
在新的审批模式下，新加坡金融管理局预计将在约三个月内完成对大多数新型基金产品的评估，并确立必要的保护措施。一旦这些指导原则确定，符合既定要求的同类型后续基金有望在三周内获得批准。
这一精简化的流程预计将显著缩短创新型基金产品的上市时间，并为基金管理人开发新投资产品提供更清晰的监管指引。
近年来，随着投资者对新投资主题、策略和资产类别的需求不断增长，金融行业对拓展零售基金产品范围的兴趣也随之提升。
新加坡金融管理局认为，部分拟议的基金产品可能无法完全符合当前《集体投资计划守则》下的现有投资指引，这凸显了建立一个更具灵活性监管框架的必要性。
这项提案是新加坡金融管理局与新加坡交易所（SGX）为共同打造一个更具活力和竞争力的金融产品生态系统所付出广泛努力的一部分。
它也借鉴并发展了此前已宣布的多项举措，其中包括精简新加坡的复杂产品框架，以及强化产品亮点说明书，旨在向投资者提供更清晰的关键产品信息披露。
金融管理局表示，最新的提案旨在持续支持新加坡基金管理行业的创新，同时确保在新投资产品进入市场时，零售投资者能够获得充分的保护。
目前，该监管机构正邀请行业参与者及其他利益相关方，就这些拟议的修订方案提供反馈意见。
