# 以色列防长：以军已备好独立打击伊朗

🔥 以色列防长：以军已备好独立打击伊朗
以色列国防部长卡茨近日在与军事记者的闭闭门谈话中明确表示，以色列国防军已做好对伊朗采取独立军事行动的准备。
卡茨同时强调，以军将在黎巴嫩南部维持长期军事存在，并就其在黎巴嫩、伊朗及加沙地带的最新地缘军事战略进行了深入阐述。
