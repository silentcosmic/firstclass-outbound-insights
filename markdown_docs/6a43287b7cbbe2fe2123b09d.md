# 深度解析 | AI浪潮下白领职场变革：AWS高管驳斥“末日论”

亚马逊云科技（AWS）业务首席执行官马特·加曼近日指出，当前关于人工智能（AI）将导致大规模失业的“末日论”被严重夸大。他强调，AI固然会重塑大量白领岗位，但这种“改变”与“消失”之间有着本质区别。
加曼明确驳斥了AI将全面替代劳动者的观点。他预计，大约一半的白领工作内容或将因AI而发生转变，但这并非意味着这些岗位会彻底消失。他以经典的Microsoft Excel为例阐释：新技术往往会深刻重塑我们的工作模式，而非直接将其消弭。他进一步指出：“消失与改变是两个截然不同的概念。关键在于，我们不能仅仅停留在当前静态的职场图景，从而武断地断言某个岗位将不复存在，从业者亦将失业。事实上，新的工作机会正不断涌现。”
加曼观察到，AI已经开始催生出一系列新兴职业类型。在全球日益担忧AI将率先冲击初级岗位的背景下，他依然高度重视并看好初级人才的价值。他向亚马逊内部员工传递的信息是：“未来职场仍将提供大量工作机会。”
亚马逊今年在全球范围内计划招聘超过1.1万名软件开发工程实习生和初级软件开发工程师。加曼坚信，新入职的血液将为团队注入充沛的活力、饱满的热情以及审视问题的新颖视角。他认为，如果一个团队长期由同一批共事十五年之久的人员构成，这种不可或缺的活力、热情与创新思维将难以激发。
加曼强调，只要劳动者保有学习新技能的意愿，便能在AI时代稳固自己的职业生涯，尽管未来的工作内容可能与当前大相径庭。他告诫所有员工：“如果你将两年前的工作与两年后的工作进行比较，其内容将截然不同。但你依然拥有工作，并且很有可能是一份更令人振奋、更具意义的工作。然而，你必须始终保持学习的姿态。”
加曼进一步预测，未来职场中，适应能力的重要性将可能迅速超越对特定技能的掌握。他表示：“我认为，我们在招聘过程中所关注的焦点，将不再仅仅是应聘者掌握了哪些具体技能，而是他们是否具备强大的学习能力和适应潜力。”
