# 中老联合行动：71名中国籍嫌犯被遣返，66人涉电诈

🚨 中老联合行动：71名中国籍嫌犯被遣返，66人涉电诈
7月15日，中老两国执法部门在磨丁—磨憨国际口岸举行联合移交仪式，成功将71名中国籍犯罪嫌疑人遣返回国依法处理。据透露，这批嫌犯中，有66人涉嫌电信网络诈骗。
