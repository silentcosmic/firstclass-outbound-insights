# 伊朗宣称袭击美军目标：中东局势骤然升级

⚡ 伊朗宣称袭击美军目标：中东局势骤然升级
据市场消息，伊朗革命卫队今日对外宣称，已对中东地区多个美国军事目标发动了大规模的协调打击。声明指出，此次行动针对了巴林的美国第五舰队、科威特的阿里·萨勒姆空军基地，以及萨勒曼港的共计85个美军设施，涉及导弹和无人机。此举若属实，无疑将加剧地区紧张局势，对中东地缘政治格局及区域安全稳定构成严峻挑战。
