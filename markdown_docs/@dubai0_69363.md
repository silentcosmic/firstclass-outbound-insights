# 维权群主卷款跑路，受害者群聊竟被转卖！

🚨 维权群主卷款跑路，受害者群聊竟被转卖！
据市场消息透露，自去年12月汇旺平台暴雷以来，大量受损储户为追讨资产成立了维权社群。然而，近日该社群却传出惊人变故：其负责人“表哥”在从未实际参与维权行动的情况下，卷走社群筹集的维权资金并潜逃回国。
更令人震惊的是，该负责人随后竟将整个维权社群及其相关账号，以12万USDT的价格打包出售给了一家博彩平台。此举导致众多受害者在投入大量精力与时间后，最终沦为他人渔利的工具。有评论指出，此事件再次凸显了在某些海外市场，投资者面临的复杂风险和信任挑战。
