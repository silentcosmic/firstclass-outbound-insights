# ️ 特朗普警告：或将再次打击伊朗

⚠️ 特朗普警告：或将再次打击伊朗
美国总统特朗普近日发表强硬声明，表示不排除今晚再次对伊朗采取军事打击行动，但强调在最终决定前将密切观察事态发展。
特朗普指出，此前针对伊朗的军事行动是对其“极其恶劣”行为的回应，特别是在霍尔木兹海峡发生船只袭击事件后。他明确表示，美国对伊朗的立场并非旨在寻求政权更迭，而是聚焦于防止伊朗发展核武器，强调“伊朗绝不能拥有核武器”这一核心红线。
