# 出海巨头Sea集团：进军美国消费级AI市场，秘密测试聊天机器人Migoo

为寻求在既有核心业务之外的增长突破，有迹象表明，Sea集团正在美国市场秘密测试一款名为Migoo的AI聊天机器人。
据市场信息显示，Migoo已在全球多个市场提供服务，然而无论是该应用本身，还是其官方网站，均未公开披露Sea集团的参与信息。
该聊天机器人能够利用用户偏好及个人特质的储存数据，进行个性化对话，从而提升用户体验。
此外，用户还可通过Apple的iMessage等主流通信服务渠道与Migoo进行互动。
Sea集团至今未就此项目进行官方发布，并拒绝置评。然而，据知情人士透露，Migoo有望获得更广泛的推广，其全面推出将为集团带来新的增长点。
此举将使Sea集团进入竞争激烈的美国消费级AI市场，当前已有OpenAI和字节跳动等众多科技巨头在此领域积极争夺年轻用户群体。
有内部消息指出，Sea集团总裁冯恺（Chris Feng）是Migoo项目的核心负责人，凸显了公司对此战略方向的重视。
Sea集团资深高管王冰玉（Bingyu Wang）也被认为参与了该项目的研发工作，进一步证实了其战略层面的投入。
在美国市场，这款聊天机器人通过Marvelous Technology Inc.提供服务，该公司在加利福尼亚州注册成立。
相关备案文件进一步显示，这家企业与一家在新加坡注册的实体存在关联，王冰玉女士在该实体中登记为董事，揭示了其背后复杂的股权结构。
Migoo的加入，为Sea集团日益增长的AI应用版图增添了一款消费级产品。目前，该集团已将AI技术广泛应用于商品推荐以及Shopee平台卖家工具等多个领域，以优化运营效率和用户体验。
在此之前，Sea集团于今年二月份扩大了与Google的合作，共同为旗下Shopee、Garena和Monee平台开发更深层次的AI应用。
这项深度合作成果之一，便是包含了一个AI购物原型，旨在显著支持和优化用户的商品搜索及购买体验。
与此同时，Sea集团还专门成立了一个团队，负责评估潜在的AI公司投资机会，这表明其在AI领域的战略布局不仅限于内部研发，更着眼于外部生态系统的构建。
此次传闻中的产品测试，恰逢Shopee在全球范围内裁减数百名开发人员，团队规模缩减约8%。然而，目前尚无明确证据表明此次裁员与Sea集团更广泛的AI战略存在直接关联。
