# ️ 印度驻阿联酋领馆服务停摆，数万侨民受困

⚠️ 印度驻阿联酋领馆服务停摆，数万侨民受困
近期，印度驻阿联酋迪拜及阿布扎比领事馆面临严重运营混乱，导致大量印度侨民无法正常办理护照及其他领事服务，现场排队秩序混乱，民怨沸腾。
此次服务中断是多重因素叠加的连锁反应。首先，领事服务费用自6月20日起突然大幅上涨，其中成人护照费用飙升近58%，且事先毫无预告。紧接着，从7月1日起，原有的两家外包服务供应商被新公司取代，新旧系统交接不畅，导致服务骤然中断。雪上加霜的是，被淘汰的供应商就招标公正性提起诉讼，印度法院颁布“维持现状”禁令，使得新旧供应商皆无法正常运作，领馆被迫临时收回所有业务自行处理。
受此影响，领馆一度改为现场排队、先到先得的现金办理模式，但因人手不足和需求激增而难以应对。随后紧急调整为线上预约制，无预约者不予接待。数万名印度侨民，包括急需为新生儿办理护照者，多次奔波无果，情绪激动，对领馆的处理方式表达强烈不满。
