# 浙江金华警方破获假体彩赌球案，涉案500万，18人落网！

🚨 浙江金华警方破获假体彩赌球案，涉案500万，18人落网！
近日，浙江金华市公安局江南分局成功捣毁一个假冒“中国体彩”名义的非法赌球犯罪团伙，抓获涉案人员18名，涉案金额高达500余万元。
警方通过线索摸排和大数据研判，锁定了这个长期活跃于网络、专门招揽网民参赌的团伙。该团伙利用世界杯赛事热潮，搭建高仿正规体彩界面的非法赌球APP，并在线上多渠道引流，诱导球迷和网民下注，手段隐蔽且极具迷惑性，严重扰乱了网络秩序和社会治安。
在全面查清该团伙组织架构和运作模式并固定犯罪证据后，警方多路同步展开收网行动，将所有涉案人员抓获。目前，18名嫌疑人均已被依法采取刑事强制措施，案件正在进一步侦办中。
