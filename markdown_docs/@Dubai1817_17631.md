# 泰国突查赌博窝点：月流水7000万泰铢

⚡ 泰国突查赌博窝点：月流水7000万泰铢
6月30日，泰国网络警察在达府湄索县成功捣毁一处名为“Las Vegas LV177”的在线赌博网站运营窝点，并逮捕两名涉嫌负责运营管理的缅甸籍女子。
据警方披露，该赌博网站月资金流水高达约7000万泰铢，且正计划借2026年世界杯赛事之机进一步扩大运营规模。被捕的两名缅甸籍女子主要负责网站客户沟通、会员管理及客服工作。现场查获电脑、手机、现金及金饰等涉案证物。
目前，两名嫌疑人已被控非法经营网络赌博及相关罪名，案件正在进一步调查中。
