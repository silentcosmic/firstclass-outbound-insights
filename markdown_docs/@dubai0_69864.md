# 中东战事升级：美伊互袭，霍尔木兹海峡航运引关注

🚨 中东战事升级：美伊互袭，霍尔木兹海峡航运引关注
美国夜间对伊朗发动了长达七小时的空袭，并同步恢复了对霍尔木兹海峡的海上封锁。此举标志着地区紧张局势的显著升级。
作为回应，伊朗随即对约旦、科威特及巴林境内的目标实施了打击。此次冲突互袭，预计将对中东地区的地缘政治格局和国际航运市场带来深远影响。
