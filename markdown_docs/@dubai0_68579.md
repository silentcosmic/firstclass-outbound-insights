# 涉1.92亿“牛”骗贷主犯泰国落网

🚨 涉1.92亿“牛”骗贷主犯泰国落网
泰国移民局近日在清迈成功逮捕一名涉嫌巨额贷款诈骗的中国籍嫌疑人柴继鹏（CHAI JIPENG）。此案涉及金额高达1.922亿元人民币（约9.6亿泰铢），嫌疑人曾以1.2万头牛作为抵押，在中国骗取银行贷款后潜逃至泰国。
据泰国警方通报，34岁的柴继鹏因涉嫌贷款诈骗被中国公安机关通缉。他于2026年6月25日持旅游签证入境泰国，企图逃避法律制裁。泰国移民局第五区调查部门联合清迈国际机场移民检查站，在接到中方通报后迅速锁定其行踪并实施抓捕。
中方调查资料显示，2025年9月至2026年3月期间，柴继鹏在吉林省白山市浑江区利用一家农业畜牧企业，向银行江源支行先后申请8笔贷款，总计约1.5亿元人民币，抵押物为1.2万头牛。然而，贷款发放后，嫌疑人便失联，导致银行蒙受近1.922亿元人民币的经济损失。
目前，泰国当局已依法撤销柴继鹏的在泰居留许可，并正协调将其遣返回中国。此次行动彰显了泰国打击跨国犯罪的决心，旨在防止泰国成为国际犯罪分子的避风港，并进一步深化与外国执法机构的合作。
