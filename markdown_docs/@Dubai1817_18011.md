# 聚币创始人刘源泰国落网：豪宅藏匿百亿资金迷雾

🔥 聚币创始人刘源泰国落网：豪宅藏匿百亿资金迷雾
近日，涉嫌大规模金融欺诈的虚拟货币平台JuCoin（聚币）核心创始人刘源，在泰国曼谷的豪华宅邸中被成功抓捕归案。泰国警方对其住所进行搜查，据称查获了包括五本不同国家伪造护照、多条金条、高端手机、多国银行卡、大额外币现金以及泰国精英居留卡等大量贵重物品及作案工具。然而，存储大额比特币私钥的关键U盘目前仍下落不明。
此前，刘源为逃避追查，曾利用伪造护照长期潜逃境外。JuCoin平台曾以交易所为外壳，诱导投资者将比特币、USDT等主流数字资产强制兑换为平台原生代币JU，随后恶意砸盘导致JU币价格几近归零。平台随即关闭提现通道、锁定用户账户，并要求追加资金以解锁。据统计，该平台涉嫌卷走资金规模逾百亿人民币，令海内外大量普通投资者蒙受巨大损失。
此次操盘主脑的落网，让众多受害者感到振奋，也预示着针对此类跨境虚拟货币诈骗的打击力度将持续加强。
