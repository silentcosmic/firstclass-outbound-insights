# 菲律宾绑架案5分钟锁定：中国公民成功获救

🚨 菲律宾绑架案5分钟锁定：中国公民成功获救
7月9日晚间，菲律宾国家调查局（NBI）接报一起紧急绑架案：一名中国公民在马尼拉冈田酒店附近公寓遭四名绑匪非法拘禁。
NBI迅速响应，仅用五分钟便成功锁定受害人具体位置，并立即启动解救行动。目前，该中国公民已安全获救，案件仍在进一步调查中。
