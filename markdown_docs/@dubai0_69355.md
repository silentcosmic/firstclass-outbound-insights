# H-Pay清算程序生变：庭外清算终止，转入司法程序

🚨 H-Pay清算程序生变：庭外清算终止，转入司法程序
H-Pay的清算公司REACHS & Partners今日发布通知，鉴于H-Pay目前已处于资不抵债状态，即无足够现金清偿现有债务，此前由柬埔寨国家银行任命清算人主持的庭外清算程序现已正式关闭。
据通知，清算公司已向法院提交申请，以启动法院监督下的司法清算程序。此举旨在由具有管辖权的法院依法指定新的清算人，继续按照适用法律履行清算职责，确保法律程序的持续推进。
所有持有相关索赔要求的个人和机构，务必密切关注具有管辖权的法院及法院指定清算人的后续最新进展。
