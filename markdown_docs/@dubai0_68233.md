# 迪拜“黑美元”骗局：老人深信不疑，损失惨重

🚨 迪拜“黑美元”骗局：老人深信不疑，损失惨重
迪拜警方近日破获一起“黑美元”诈骗案，一名老人在长达数月的时间里反复向诈骗团伙支付巨额款项。骗子谎称，通过“特殊化学药剂”，装满黑色纸张的保险箱可将其转化为真实货币。令人震惊的是，即便在迪拜警方成功抓获所有涉案人员后，受害者仍执迷不悟，向警员询问如何处理那些所谓的“黑美元”。
