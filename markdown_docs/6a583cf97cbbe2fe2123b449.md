# 澳大利亚设立国家人工智能办公室，加速战略布局并规范数据中心能耗

澳大利亚联邦政府近日发布重磅举措，宣布将正式推出国家人工智能标准，并于总理和内阁部（Department of the Prime Minister and Cabinet）内部设立专门的人工智能办公室。此举旨在全面统筹澳大利亚在人工智能领域的战略发展，确保国家能够牢牢把握这一颠覆性技术带来的历史性机遇，同时有效保障公民的数字安全与福祉。
与此配套的，是针对大型数据中心审核流程的优化与规范。根据新政要求，这些关键的算力基础设施在选址时，必须优先考虑最为适宜的区域，并深入征求当地社区的意见与反馈。此外，数据中心运营方将需要承担相应的电力基础设施建设成本，并在电网面临压力时，承担减少用电的责任，以增强国家电网的整体弹性。同时，政策亦强调数据中心必须尽可能采取措施，最大限度地实现水资源的高效利用与节约。
