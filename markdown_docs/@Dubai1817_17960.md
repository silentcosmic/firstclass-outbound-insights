# 金边突发：中国籍女性诊所内遇害，疑遭奸杀

🚨 金边突发：中国籍女性诊所内遇害，疑遭奸杀
据柬埔寨当地消息，金边市棉芷区冠树新世界城紫云诊所今日凌晨6时30分发生一起恶性案件。一名中国籍女性受害人疑遭强奸后被杀害。
案发后，当地警方已迅速赶赴现场展开勘查与调查。截至目前，案件的具体细节、作案动机以及嫌疑人情况仍在进一步调查核实中，有待警方后续正式通报。
