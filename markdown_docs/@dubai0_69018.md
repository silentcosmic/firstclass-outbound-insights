# 阿联酋总统贺美国独立日，赞美“强大联盟”

🤝 阿联酋总统贺美国独立日，赞美“强大联盟”
阿联酋总统谢赫·穆罕默德·本·扎耶德·阿勒纳哈扬于美国独立日（7月4日）向美国总统唐纳德·特朗普及美国人民致以诚挚祝贺。他特别赞扬了两国之间“强大的联盟”，并祝愿美国持续进步与繁荣。
