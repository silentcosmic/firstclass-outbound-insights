# 中柬深化执法合作：联合打击电信诈骗，维护区域安全

🚨 中柬深化执法合作：联合打击电信诈骗，维护区域安全
近日，中国国家主席习近平在北京会见柬埔寨人民党主席兼参议院主席洪森。双方就深化中柬命运共同体建设、加强经贸与安全合作等议题深入交换意见，并一致同意持续加强执法协作，共同打击跨国电信网络诈骗犯罪。
习近平主席强调，中方高度赞赏柬埔寨政府打击电信诈骗的坚定决心，愿持续深化执法合作，保持高压打击态势，彻底铲除此类跨国犯罪“毒瘤”，共同维护两国人民生命财产安全及地区稳定。洪森主席亦表示，清剿电信网络诈骗是柬埔寨当前重要工作，柬方愿进一步深化与中方在执法安全领域的务实合作，严厉打击包括人口贩运在内的各类跨国犯罪活动，以维护两国共同利益。
