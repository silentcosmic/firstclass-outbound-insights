# 具身智能仿生扑翼机器人新纪元：00后上海交大博士团队三个月内完成三轮融资，引领流体智能突破

近日，在具身智能机器人领域，一家专注于仿生扑翼飞行技术的创新企业「鹰瞰智翼」宣布完成数千万元人民币A轮融资。此轮融资由元禾璞华领投，孚腾资本与厚雪资本联合跟投，标志着该公司在短短三个月内已连续完成第三轮融资，彰显了资本市场对其前瞻技术与商业化潜力的强烈信心。本轮募集资金将主要用于首款消费级产品的量产备货与市场拓展，同时支持核心团队的扩充，并持续投入下一代具身智能扑翼机器人及流体仿真引擎的深度研发。
「鹰瞰智翼」于2025年3月在深圳正式成立，是全球范围内较早将目光聚焦于具身智能扑翼机器人赛道的先行者之一。创始人陈昊先生系上海交通大学机械工程专业直博生，其核心研发团队占比超过75%，成员汇聚了来自上海交通大学、大疆、华为、字节跳动等顶尖学府及科技企业的精英，兼具前沿科研实力、全球市场洞察和互联网运营经验，致力于加速流体智能技术的产业化进程。
颠覆性产品策略：从借力飞行到主动驾驭
当前，「鹰瞰智翼」正同步推进两条清晰的产品发展路线。其首款消费级产品，代号“Eagle X”（原代号“丹丹”），其核心设计理念在于“借助流体飞行”。该产品通过精密的扑翼结构，实现滑翔与拍打的混合飞行模态，巧妙地利用气流借势而行。截至目前，该产品已累计完成逾3000小时的飞行测试，性能表现稳定，目前已顺利进入量产阶段，计划于今年第三季度通过国际众筹平台KickStarter进行全球发布。
另一条路线则聚焦于产业级产品，其技术核心是从“被动借力”向“主动驾驭”的跨越式升级。这款产品拥有约15个自由度，使其能够主动感知并高效利用复杂气流，而非仅仅依靠被动滑翔。支撑这一突破性能力的关键，在于公司自主研发的机器人流体仿真引擎Vortrix。
Vortrix引擎的价值不仅限于飞行控制，更在于其独特的数据飞轮效应。每一台飞行器在实际空中作业中采集到的真实流体数据，都将实时反哺至仿真引擎，用以持续校正与优化模型，确保其与物理世界的高度对齐。陈昊先生将这一数据驱动的闭环迭代过程，形象地类比为特斯拉的自动驾驶数据飞轮，他强调：“采集到的流体数据可能比样机本身更具战略价值。”
在市场拓展层面，「鹰瞰智翼」的初期战略是深耕海外C端市场。陈昊先生介绍，“Eagle X”主要面向全球的极客群体。产品设计上高度强调开放性与可玩性：机身预留了多种标准接口，用户可以根据自身需求，自由设计并更换不同的翅膀、烧录定制程序，甚至挂载相机等扩展模块。团队旨在将其打造为一个开放的创新平台，而非封闭的硬件产品，以此吸引全球开发者社区，共同探索多元化的应用场景与玩法。
谈及当前企业发展面临的最大挑战，陈昊先生直言是市场拓展能力的建设。“目前，许多客户是主动通过产品信息找到我们，成交率非常高，但这并非我们期望的长期模式。我们不能一直处于被动接单的状态。”
随着产品量产与众筹活动的逐步推进，「鹰瞰智翼」团队计划在今年下半年进行适度扩张，重点将加强市场营销、渠道建设及售后服务等关键力量。
深度解析：Vortrix引擎的突破性与未来愿景
关于具身智能仿生扑翼机器人的核心技术难点，陈昊先生详细阐述了其复杂度：“挑战在于非线性、非稳态的气动力建模。传统的旋翼无人机，其动力学特性相对线性，已有成熟的PID控制方法。然而，扑翼机器人在每一次拍打过程中，升力与推力是高度耦合且动态变化的，并且极易受到实时气流的显著影响。传统的解决方案往往依赖于长时间的风洞试飞与反复调参，整个研发周期漫长。”
「鹰瞰智翼」的应对策略是自主研发的Vortrix流体仿真引擎。该引擎首先通过真实风洞数据结合粒子法进行精确建模，随后在高度仿真的虚拟环境中，运用强化学习算法对机器鸟进行数百万次的训练，使其能够自主学习在不同气流条件下如何智能调整翅膀角度和拍打频率。最终，经过充分训练的模型可直接部署到真实飞行器上。陈昊先生表示，若没有Vortrix引擎作为支撑，快速且高效地进行包含流体动力学在内的强化学习训练几乎无法实现。
在Vortrix引擎的商业化策略上，陈昊先生透露，在早期阶段，该引擎将不会对外开放。其主要作用是赋能「鹰瞰智翼」自有的具身智能仿生扑翼机器人产品线。用户可以通过订阅公司的具身智能飞行控制方案，获得包括悬停、落叶飘、倒飞、长续航飞行以及大载重飞行等多样化的飞行能力。
然而，从长远来看，陈昊先生展望了Vortrix引擎更广阔的应用前景：“在未来合适的时机，我们可能会考虑将Vortrix引擎对外开放，使其能够服务于各类飞行机器人的强化学习训练，乃至固定翼飞行器、民航飞机，甚至风电叶片的气动优化。我们希望Vortrix能够成为整个低空经济发展的重要基石。”
从“会飞的机器”到“具身智能体”：自由度与AI学习的深度融合
对比当前市场上的多数扑翼机设计，陈昊先生指出其普遍存在的局限性：“市面上的扑翼机大多以追求极致轻量化为目标，翅膀往往只能进行对称拍打，自由度极低。这类设计虽然能实现飞行，但它们无法主动应对复杂的外部气流。当遇到突发气流时，它们只能‘被动借力’，而缺乏足够的能力调整姿态以‘主动驾驭’气流。”
而「鹰瞰智翼」的第二款产品则将自由度大幅扩展至约15个。这意味着其翅膀可以像真正的鸟类一样，独立调整角度、进行扭转，甚至折叠。陈昊先生解释道：“这种足够大的状态-动作空间，为强化学习算法提供了广阔的发挥天地。您可以将此理解为，我们首先构建了一个足够灵活的‘身体’，然后让人工智能去学习如何高效地运用这个身体进行飞行。”目前，该整机的翅膀和腿部结构已分别申请了专利，并进行了全球专利布局。这一系列创新，标志着公司正从单纯的“会飞的机器”向更高级别的“具身智能体”迈出关键性的一步。
投资方观点：低空经济与具身智能的交汇蓝海
元禾璞华（领投机构）表示：“从半导体底层技术，延伸至AI全域硬科技的深度布局，是璞华长期坚持的核心投资主线。当前地面人形机器人与四足机器人赛道竞争日益激烈，而仿生飞行机器人领域仍处于产业萌芽阶段，具备高技术壁垒、广阔的大众消费市场潜力以及长期的产业延展性，成长空间巨大。从技术层面看，鹰瞰智翼凭借其自主研发的Vortrix流体仿真引擎与全栈强化学习算法，实现了仿生飞行本体与空中具身智能的深度融合。其产品具备低噪音、高安全性与生物兼容性等天然优势，完美解决了近人场景中的核心痛点。团队层面，核心成员均为00后上海交通大学博士，在顶尖学术研究、整机结构设计、AI飞行控制及流体仿真基础方面，展现出全链路的自主研发能力。我们坚定领投鹰瞰智翼，看好其在新兴近人消费赛道的开辟能力，前景广阔，未来可期！”
孚腾资本指出：“传统旋翼无人机在应用于近人场景时，长期面临噪音、安全隐患以及难以融入环境等问题。鹰瞰智翼通过仿生扑翼技术另辟蹊径，其自主研发的Vortrix流体仿真引擎结合强化学习训练，使得机器鸟能够在虚拟环境中自主学习并掌握应对复杂气流的飞行姿态，并通过AI进行自主决策。创始人陈昊先生及其联合创始团队均为00后上海交通大学硕博，他们用短短一年时间便将学术验证快速推进至样机量产筹备阶段，其执行力令人印象深刻。我们对鹰瞰智翼将技术优势转化为实际产品的潜力充满信心，并对扑翼机器人市场的未来发展抱有积极预期。”
厚雪资本认为：“仿生鸟类空中具身智能是低空经济与具身智能两大前沿领域的黄金交叉赛道。鹰瞰智翼凭借自研的扑翼控制技术与空中具身算法，打造出低噪音、高机动性的产品，非常适用于巡检、生态监测等多种场景。当前地面物理AI赛道竞争日趋白热化，而空中具身智能仍是一片广阔的蓝海市场。本轮融资将有力助推鹰瞰智翼在技术研发与商业化进程上的提速，使其有望成为国产仿生飞行机器人领域的领先标杆。”
