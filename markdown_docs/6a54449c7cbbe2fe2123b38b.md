# 突破价格内卷：新一代出海企业以创新定义高端产品赛道

黄强帅清晰记得，曾为了争取一个重要客户的订单，他在发烧一夜后，打了点滴便立刻搭乘航班飞赴海外，因为客户两天后就需见到样品。最终，订单顺利拿下。
然而，黄强帅深感，在传统外贸模式下，企业往往过于被动。他指出：“客户提出什么需求，我们就做什么，公司完全受制于客户的硬性要求。”
作为浙江帅帅电器科技有限公司的副董事长，黄强帅是典型的“厂二代”。他在留学期间便开始在亚马逊平台销售产品，并发现电商评论区蕴藏着工厂从未直接听到的用户心声。他意识到，正是这些宝贵的用户反馈，为帅帅电器提供了比客户更早一步开发新产品的根基。
根据海关总署近年来的跨境电商业态统计，中国自主品牌出口在2024年和2025年连续实现增长，占中国出口总值的比重分别提升了0.8和1.4个百分点。截至2025年底，自主品牌出口已达到中国出口总值的23.2%。
以洗碗机为例，黄强帅分享道，近年来洗碗机在海外家庭日益普及，但传统金属锅具在洗碗机酸碱环境下易被腐蚀氧化。在传统外贸反馈链条中，从客户投诉到门店，再从门店退货至品牌商，品牌商最终反馈给工厂，这一过程漫长而滞后。
通过亚马逊上的自有品牌商城，帅帅电器能够实时获取海外消费者的反馈。基于这一洞察，帅帅电器有针对性地进行了材料和工艺技术创新，推出了洗碗机清洗不氧化的锅具。尽管其价格比同类产品高出十几至二十美元，却反而更受市场欢迎。
更深层的积极变化体现在，自有品牌有效盘活了帅帅电器的工厂运营，产能利用率从60%提升至85%。过去工厂只能被动等待订单，无订单时便空置；现在，帅帅电器的自有品牌使其在没有外部订单时，工厂也能持续运转。而当海外订单量激增时，自有品牌产品也可灵活采购其他工厂的产能，实现更高效的资源配置。
深耕供应链：以极致品质定义母婴市场
2023年7月，哈库拉玛塔达品牌的天丝婴儿睡衣意外成为爆款。原因是，一位皮肤科医生在宝妈论坛上推荐了两种对婴儿湿疹友好的面料，孙翔公司旗下的天丝产品恰好是其中之一。然而，补货后不久，一条差评赫然出现：“我儿子一直穿你们的衣服，但这次购买的，让他身上起了很严重的疹子。”
经过一番追查，孙翔发现是供应商偷换了面料。97%天丝含量的面料工艺复杂、市场需求量小、成本高昂，很少有工厂愿意承接，即使有工厂愿意生产，也常会掺杂更廉价的面料，这直接导致了婴儿的过敏反应。
为此，孙翔决定深入供应链源头。为了确保原料来源纯正可追溯，他将目光锁定在四川的生态种植园；为了制定纱线和印染参数标准，他的团队长时间驻扎在工厂一线；在成衣环节，由于传统制衣工厂难以规模化处理这种极其柔软的新型面料，孙翔找到了一家疫情后转型生产医用防护服的工厂，将最贴肤的生产工序转移至医疗级生产车间，并与工厂共同采购设备，建立了一套严苛的婴儿服装生产管理标准。
与过去跨境商家通常直接从工厂采购不同，新一代跨境创业者对供应链拥有更强的主导权和话语权。正是这种对工厂的深刻掌控，决定了产品用料和制作工艺，进而从根本上定义和创造了产品。而更关键的是，他们明确了为谁创造产品。
孙翔选择婴儿服饰赛道，源于他妹妹孩子的亲身经历。婴幼儿皮肤敏感，即使购买了昂贵的有机棉产品也无法解决湿疹问题，唯有穿着天丝面料才能有所好转。以此为切入点，孙翔推出了天丝面料睡衣。销售一段时间后，他意识到问题远不止面料本身——湿疹导致宝宝夜晚无法安睡，进而也影响了妈妈的睡眠。这才是更深层的用户需求。
第二个爆款产品的诞生更富戏剧性。一位新生儿重症监护室（NICU）护士在评论区留言说，她们一直在为NICU的宝宝寻找一件合身的衣服。孙翔由此才意识到，市场上所有婴儿服饰均以0-3个月起步，早产儿体型过小，根本没有合适的尺码。
从投资回报的角度来看，早产儿产品线几乎是一项亏本的投入。工厂生产不同尺码衣服的工序成本基本一致，但早产儿衣物需求量小，利润空间接近于零。即便如此，孙翔仍坚持投入生产，他表示：“就是为了让所有妈妈在任何时候，我们都可以成为她选择的对象。”
以用户需求驱动：声学专家破局助听器市场
世界卫生组织数据显示，全球约有15亿人存在不同程度的听力损失，然而助听器的全球佩戴率不足10%。在欧美等成熟市场，这一数字约为三成，而在中国甚至尚不足一成。
苗健彰决心探究其背后的原因，于是他在公司楼下开设了一家助听器门店。门店不销售自家产品，而是销售市面上已有的产品，并安排听力师和工程师共同驻守，每天与真实的听损人群面对面交流。
通过与用户的深度沟通，他们发现传统助听器存在诸多痛点：例如使用纽扣电池，对于手指不灵活、视力不佳的用户来说更换极为困难；许多助听器缺乏蓝牙功能，无法连接手机、接打电话或观看视频；在外观上，传统的耳背机体积明显，许多人宁愿忍受听力不佳，也不愿让他人发现自己佩戴助听器。
他们曾遇到一对父女。父亲听损二十年，年轻时口才出众，人缘极佳，但后来因抗拒佩戴不美观的助听器，逐渐退出了社交生活，变得沉默寡言。女儿感慨道，如果你们早些认识他，会觉得那是截然不同的两个人。
在创立听象科技之前，苗健彰曾主导一家专注于声学AI技术的B2B公司，通过机器学习实现通话降噪、提升语音清晰度，为小米、OPPO、vivo、联想等知名品牌提供服务。
“做B端相对容易，问题通常是客户提出的，你只是在回答一个有标准答案的问题，比拼的是团队的执行力和效率。”苗健彰强调，“但转到C端，首先要自己发现并提出问题，这才是极具挑战性的事情。”
2023年，苗健彰创立听象科技，面向C端推出了昂听ELEHEAR品牌。最初上线的第一代产品Alpha，如今看来尚不成熟。然而，它的市场投放为团队提供了与用户直接沟通、收集真实反馈的入口。在摸清用户需求之后，才真正开始了技术攻关。
对于技术出身的苗健彰而言，听损人群表达“听不清”时，通常是生活化的感受。他需要将用户这种模糊的“听不清”翻译成精确的技术语言和工程语言，才能将其转化为实际的产品功能。
助听器的技术约束远高于手机或蓝牙耳机。助听器需佩戴十几个小时，因此功耗必须极低；与人面对面交流时，算法延迟需控制在几毫秒以内，稍有偏差用户便会感觉到声音和口型不同步；拾音距离需超过两三米，对远场信噪比有着更高的要求。
昂听ELEHEAR历时四年，完成了算法、验配系统、硬件产线的全部自主研发和建设，推出了第二代旗舰产品Beyond。其中，昂听ELEHEAR自研的VOCCLEAR®AI声学算法，能够在低功耗助听器上每秒完成约500次环境分析，并完成4.5亿次运算。其验配系统支持用户在家通过App连接，实现远程实时验配，无需亲赴线下门店。
Beyond上线后，昂听的退货率远低于行业平均水平的一半，在助听器品类中逐渐站稳了脚跟。苗健彰认为，真正难以被复制的并非某个单一技术点，而是这整套以用户为中心、快速迭代的全栈自主研发闭环能力。
新一代跨境企业的产品力：穿透、垂直、技术
EcoFlow创始人王雷、拓竹创始人陶冶，皆是大疆走出的技术专家，如今已成为跨境领域的佼佼者；近一年内，更有不少来自大疆的技术人才投身外骨骼、陪伴机器人、消费级具身智能等新型跨境赛道。
这些新一代出海企业具备的核心优势在于：供应链穿透力，让他们能掌控产品如何制造；场景垂直力，让他们清晰知道为谁服务；以及强大的技术能力，让他们能够创造出此前根本不存在的创新产品。三者叠加，共同构筑了新一代跨境创业者的核心产品力。
然而，正如苗健彰所言，从B端转向C端，最大的转变在于需要企业自身去发现并定义正确的问题。这对工程师背景的创业者而言尤其是一项挑战。技术积累固然能打造出优秀的产品，但如何让用户看见这款新产品、如何建立品牌、如何维护用户关系，则是另一套截然不同的能力体系。
在全球供给充裕的时代，即使是美酒，也可能因身处深巷而无人问津。正如黄强帅在那趟发烧的航班上所意识到的，过去是企业追着订单跑，而现在，则应是让产品为自己开辟市场。制造产品只是第一步，产品成功推出后，真正的市场竞争才刚刚开始。
