# 印尼北苏门答腊：跨国“网恋诈骗”团伙38人落网

🚨 印尼北苏门答腊：跨国“网恋诈骗”团伙38人落网
印尼北苏门答腊省执法部门近期开展联合专项行动，成功打击一个涉嫌实施跨国“网恋诈骗”（Romance Scam）的犯罪团伙。据悉，共有38名涉案人员被控制，其中包括6名中国籍、1名越南籍及31名印尼籍嫌疑人。此案凸显了区域内打击跨境网络犯罪的决心，目前案件仍在进一步调查中。
