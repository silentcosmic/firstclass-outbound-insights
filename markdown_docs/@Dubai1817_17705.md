# 柬埔寨现代奴役揭露：5名受害者获救

🚨 柬埔寨现代奴役揭露：5名受害者获救
蓝龙儿童基金会近日宣布，成功从柬埔寨解救了5名遭受现代奴役的受害者。这批受害者包括1名36岁女性及4名年龄介于21至33岁的男性，他们此前均被虚假的高薪工作承诺诱骗至柬埔寨。
抵达当地后，这5人随即被贩卖给一个犯罪团伙，遭到非法拘禁和残酷奴役。其中，该名女性在遭受酷刑威胁下，被迫在妓院从事性交易；而另外4名男性则被囚禁于大型园区内，被迫参与网络诈骗活动。据披露，拒绝工作的受害者会遭受殴打甚至断绝食物供应。
在受害者设法向警方求助后，蓝龙儿童基金会迅速与当地执法部门合作，成功将这5名受害者安全营救并协助他们返回家园。目前，他们正获得该基金会全球社区提供的全面支持，包括心理康复、生活安置及其他后续援助，以帮助他们逐步恢复正常生活。
