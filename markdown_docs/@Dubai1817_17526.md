# 阿联酋总统会晤美国国务卿：共促中东地区安全稳定

🔍 阿联酋总统会晤美国国务卿：共促中东地区安全稳定
近日，阿联酋总统谢赫穆罕默德·本·扎耶德·阿勒纳哈扬在阿联酋会见了正在当地访问的美国国务卿马尔科·鲁比奥。此次会晤是鲁比奥中东之行的一部分。
双方回顾了阿联酋与美国之间的战略合作，并探讨了在多领域深化协调以支持共同利益的途径。会晤重点关注了双方共同关心的地区及国际议题，特别是中东局势的最新发展。在此背景下，谢赫穆罕默德与鲁比奥国务卿强调，应共同致力于巩固地区安全、稳定与持久和平的基础。
阿布扎比王储谢赫哈立德·本·穆罕默德·本·扎耶德·阿勒纳哈扬以及多位阿联酋高级官员出席了此次会晤。
