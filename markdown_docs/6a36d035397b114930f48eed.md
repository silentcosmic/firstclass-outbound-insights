# Sumsub发布AI智能代理工具，赋能企业高效构建合规验证流程

Sumsub近期推出了一项创新性AI代理集成方案，旨在赋能合规团队，将复杂的反洗钱（AML）政策直接转化为可配置的验证工作流程。
这项基于“模型上下文协议”（Model Context Protocol）的集成，允许企业将内部的反洗钱政策或各类监管要求上传至主流AI工具，例如Claude、ChatGPT或其他领先模型。
随后，AI代理便能依据这些上传的文档，在Sumsub平台内部自动配置验证等级、风险问卷以及新用户入职（onboarding）流程。
Sumsub方面指出，此举能将原先耗时数天甚至更长的配置工作，大幅缩短至短短数分钟。即便面对包含国家特定风险等级、评分表和条件规则的复杂政策，效率提升也同样显著。
公司强调，这项集成旨在最大程度地减少将合规政策手动转化为平台具体设置的繁琐工作。
此外，智能AI代理还能协助企业，将Sumsub的验证流程无缝嵌入到客户应用程序中。
合规团队可以利用这些代理工具，高效地审查申请人、执行数据分析、生成验证链接，并快速响应不断变化的监管要求。
Sumsub负责人表示：“我们的智能代理体验彻底改变了传统模式，它通过将AI代理直接连接到平台的配置层，使得团队只需将反洗钱政策提交给AI代理，即可自动构建完整的合规环境。这与此前市场上任何可用功能都截然不同，代表了一个全新能力范畴。”
这项集成方案具备模型无关性，旨在兼容各类主流AI代理。同时，Sumsub已在GitHub上发布了开源的代理技能，进一步推动行业协作。
“模型上下文协议”（MCP）的集成，是Sumsub现有AI工具生态的进一步拓展，其中包括专为合规与反欺诈团队设计的AI副驾驶工具Summy。
访问权限通过独立设置进行严格控制，所有敏感操作均在隔离沙箱环境中运行。此外，任何配置更改仍需经过人工审核与批准，确保最终决策的准确性和安全性。
此项集成服务现已全面上线。Sumsub补充指出，其是首个正式入驻ChatGPT Apps平台的验证服务商。
相关的文档和代理技能已可通过Sumsub的开发者资源平台获取。
