# 中国高校战略转型：为AI腾挪空间，外语专业何去何从？

当前，多所高校正在逐步削减翻译及外语专业课程，转而增设具身智能、人工智能和机器人学等新兴学位项目。
在全球经济被人工智能深刻重塑的背景下，中国高等教育机构正迅速调整其学科设置，以适应新的知识与人才需求格局。
近几个月来，国内多所大学已积极减少外语及翻译专业的招生名额，同时大力推出具身智能和低空经济等新兴专业方向。
近期一项针对70所中国高校的调查显示，在它们公布的下一学年专业调整计划中，与语言相关的课程削减幅度显著：包括8个日语专业、5个德语专业以及5个翻译研究专业。
业界分析认为，外语专业曾是中国高校中增长最快的学科之一。然而，全球格局的变迁以及人工智能翻译工具的迅速崛起，正迫使这些学科重新审视其人才培养模式。
这些调整揭示了中国高等教育领域正如何努力为学生应对人工智能驱动的未来做好准备。当前，全球大学教育正面临日益增长的压力，需证明其在一个人工智能日益能完成写作、编程、翻译及信息分析的世界中的价值与关联性。
这一趋势并非中国独有。普渡大学应用语言学博士后方韶华指出，全球范围内的大学都在探索重新设计其语言课程的路径。
方韶华补充道，人工智能的快速发展无疑影响了学生修读语言专业的意愿，因为传统上与这些学位相关的职业道路——例如翻译和语言教学——正面临日益严峻的挑战。
与美国大学在很大程度上自主决定学术设置不同，中国高校的专业调整需要政府部门的审批。
雪城大学社会学教授马颖毅指出，在中国，应对这一变化的策略更趋于集中和自上而下。人工智能正被纳入国家教育规划和新专业设置中。而在美国，其应对方式更为分散，各大学、学院、系以及教职员工的自主性和变通性巨大。
今年4月，中国教育部批准了9所高校开始招收“具身智能”专业的学生。这一术语在中国语境下，指的是自动机器和人形机器人等实体人工智能技术。
教育部为即将到来的学年总共批准了38个新专业，其中大部分聚焦于技术或数字化领域。
新专业涵盖了商业人工智能、数据智能等与AI相关的学科，以及低空经济与管理、半导体装备工程、稀土科学与工程等前沿领域。
根据教育部的官方声明，这些新设学科旨在支撑国家战略性产业发展，推动传统行业的转型升级，并为未来经济增长储备人才力量。
中国当前进行的专业结构调整，与此前政府发展目标导向下的多轮课程改革一脉相承。
相关数据显示，2020年至2024年间，电子商务相关专业是中国顶尖大学中削减幅度最大的学科之一。这反映出在经历阿里巴巴和京东等公司多年高速扩张后，中国互联网经济正趋于理性发展。
展望2025年，市场营销专业面临的削减数量最多，在70所大学中共有16个项目被取消。相关分析指出，这些削减主要由劳动力市场需求的变化以及纠正此前过度扩张的努力所驱动。
有统计显示，2022年至2026年间，全球范围内人工智能硕士项目数量几乎翻倍。数据显示，目前有304所美国机构提供人工智能学位，其中包括193个本科学位项目。
马颖毅教授认为：“这揭示了两国社会之间更深层次、更根本性的差异。”
她指出：“[中国]的优势在于在特定领域培养人才的速度和规模。风险则在于过度修正和人才过剩：一些领域可能在其长期重要性被充分理解之前就被低估了。”
马颖毅表示：“美国的优势在于多元化。由于其教育体系分散，这使得更多的实验、机构多样性以及自下而上的创新成为可能。然而，其劣势在于碎片化和不平等。在人工智能时代，这种不平衡可能成为一个严重的国家人才问题，并催生新的不平等源头。”
尽管人工智能相关学位数量激增，但美国科技界和商界巨头却认为，人工智能实际上可能提升人文学科毕业生的价值。
英伟达CEO黄仁勋曾将英语专业描述为“最有可能成功的专业”，因为其正是人工智能的“编程语言”。全球最大的资产管理公司贝莱德首席运营官罗伯特·戈德斯坦去年也表示，该公司更看重那些学习历史、英语及其他人文学科的毕业生。
中国高校也在努力保持人文学科和艺术学科的时代相关性，但侧重点在于将人工智能融入这些专业课程中。
作为国内领先的艺术与传媒院校之一，中国传媒大学削减了摄影、漫画和视觉传达设计等五个专业，并新增了“智能影像艺术”等融入人工智能的新项目。
在中国传媒大学，奥曼云教授着一门有着六十年历史的课程，名为“斯瓦希里语翻译：理论与实践”。她表示，学生们曾向她质疑，当人工智能工具可以即时完成翻译任务时，人工翻译是否仍有价值。
奥曼云教授强调：“我们的目标不再仅仅是教授学生如何翻译，而是培养他们指导和管理人工智能翻译工具完成复杂翻译任务，并界定与评估翻译质量的能力。”
方韶华博士，其研究聚焦于人工智能对外语学习的影响，对语言课程在大学 curricula 中的未来地位持乐观态度。
方韶华表示：“对于那些高度专注于某种语言，或希望从事以该语言为主要支柱的职业的学生而言，显而易见的是，人工智能无法完全取代他们。仅通过AI学习语言，也无法替代大学教育所提供的深度培养。”
