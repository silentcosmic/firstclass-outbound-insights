# 阿根廷足协遭FBI调查：数亿美元资金疑云

🔍 阿根廷足协遭FBI调查：数亿美元资金疑云
在阿根廷国家队成功晋级2026年世界杯八强之际，其足球协会（AFA）却突陷国际风波。据市场消息，美国联邦调查局（FBI）已对阿根廷足协在美国境内的商业和财务活动启动调查。
此次调查核心围绕数亿美元资金流向及多笔用途不明的转账交易。这起事件无疑给阿根廷足坛乃至国际体育界投下阴影，引发对体育机构财务透明度的广泛关注。
