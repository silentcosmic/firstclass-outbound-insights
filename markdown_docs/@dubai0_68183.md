# 阿联酋更新落地签政策：新增六国享60天停留期

📣 阿联酋更新落地签政策：新增六国享60天停留期
阿联酋近日宣布更新落地签证政策，将60天停留期的资格扩展至更多国家公民，此举有望进一步促进区域内的商务与旅游往来。
根据最新标准，印度尼西亚、越南、泰国、菲律宾、肯尼亚和南非六国公民现已符合申请此类60天停留期落地签的资格。此前，该政策主要面向印度公民开放，此次扩容显示出阿联酋在全球人才和市场吸引力方面的战略布局。
