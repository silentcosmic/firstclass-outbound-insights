# 马来西亚今夜大规模执法：严查酒驾毒驾及通缉犯

🚨 马来西亚今夜大规模执法：严查酒驾毒驾及通缉犯
据市场消息，马来西亚将于今晚启动一项大规模联合执法行动。该行动定于2026年7月4日晚间23:00至次日凌晨05:00进行，旨在全面打击酒驾、毒驾、违规电子烟，并追查通缉人员及在案调查人员。
本次行动为综合性临检，覆盖范围广泛且执法力度严厉。建议当地居民及出海人士在上述时段内避免不必要的出行，以确保个人安全并配合执法。
