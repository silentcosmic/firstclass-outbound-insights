# 越南酒店突发：台籍住客坠亡，警方介入调查

🚨 越南酒店突发：台籍住客坠亡，警方介入调查
近日，越南胡志明市发生一起外籍人员死亡事件。一名37岁的中国台湾籍男子于6月27日傍晚被发现坠亡于当地安富坊一酒店门前，医护人员到场后确认其已无生命体征。
据越南媒体报道，警方初步调查显示，死者是该酒店的常客，疑似从酒店高层坠落。目前，胡志明市警方已封锁现场，正对酒店工作人员及目击者进行询问，并调取监控录像，以进一步还原事发经过并查明具体死因。死者遗体已被送往法医检验，警方正设法联系其家属。
