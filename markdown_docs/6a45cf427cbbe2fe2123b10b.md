# 深度解读：5亿月活的印度流媒体巨头JioHotstar，如何通过短剧战略引领内容市场新风向？

在近期举办的APOS 2026行业峰会上，印度领先传媒集团JioStar娱乐首席执行官Kevin Vaz先生，首次对外公布了其流媒体平台JioHotstar的最新运营成果，揭示了这一市场巨头的增长轨迹与战略布局。
回溯至2024年11月，印度电信业巨擘信实集团（Reliance）与迪士尼公司完成了在印度市场的媒体业务整合，将旗下的JioCinema平台与迪士尼+ Hotstar两大流媒体资产进行合并。此举催生了全新的JioHotstar平台，迅速崛起为印度本土规模领先的流媒体服务商。
JioHotstar不仅承袭了迪士尼的丰富内容版权，更整合了Star电视网络、印度板球超级联赛（IPL）等顶级体育赛事版权，以及海量本土影视作品与娱乐节目。这使其成为一个涵盖长视频、体育赛事直播及传统电视节目等多元内容形态的超级娱乐生态系统。
根据官方披露数据，JioHotstar当前内容库已超30万小时，支持19种语言，每周触达逾8亿受众，其庞大体量已使其成为全球领先的流媒体平台之一。
该平台旗下的短剧品牌“Tadka”，被明确定义为“微内容”（Micro Content），并已深度融合于JioHotstar应用程序之中，而非以独立应用形式推出市场。
官方数据表明，Tadka上线后，用户日均观看时长实现五倍增长。其用户画像分析显示，42%的用户年龄在24岁以下；同时，二线城市与一线城市各贡献了约四成的观看时长，这清晰地揭示了短剧内容正快速渗透至印度更广泛的消费群体。
根据FICCI-EY发布的《2026媒体与娱乐报告》洞察，印度市场微短剧用户中有60%至75%来自二线及三线城市。本地语言内容被视为驱动用户付费的关键因素——超过半数的观众偏好观看母语内容。报告进一步指出，当角色使用口语化的泰米尔语、博杰普尔语或泰卢固语，而非标准印地语或英语时，用户的付费意愿呈现显著提升。
JioHotstar并未将短剧视为一个独立的新赛道，而是将其定位为整个娱乐平台内的重要内容入口。用户即便初衷仅为观看一场IPL赛事、一部迪士尼电影或电视剧，也极易顺手浏览并沉浸于数集时长仅一分钟的短剧内容。
AppsFlyer近期发布的《2026营销人员订阅状况报告》指出，全球短剧市场正在经历一场显著的战略重心转移。
印度次大陆贡献了全球新增付费安装量的49%，几乎占据了全球增长的一半。这表明，越来越多的海外短剧内容提供商正在将其市场预算从欧美区域转向印度市场。
据Lumikai发布的《2025印度互动媒体报告》披露，印度微短剧行业在2025年已实现约3亿美元的市场规模，年度累计下载量突破4.5亿次，月活跃用户数超过1亿。该报告预计，到2030年，印度短剧市场规模有望增至45亿美元，而2026年行业增速预计仍将维持在91%的高位。
JioHotstar平台本身坐拥超过5亿的月活跃用户，这赋予了其短剧业务在上线伊始便可直接获取海量自然流量的巨大优势。相较于独立短剧App需要投入数千万乃至数亿美元持续进行用户获取，JioHotstar所拥有的这种平台优势，是其他市场参与者难以复制的。
对于ReelShort、DramaBox等专注于独立运营的短剧平台而言，当前的市场格局可能意味着，未来的竞争对手将不仅仅限于同业公司，更包括那些拥有数亿用户生态的超级娱乐综合平台。
在印度市场，JioStar的战略并非孤例。包括Balaji Telefilms在内的多家印度本土影视制作公司，也已着手将微短剧内容融入其OTT（Over-The-Top）内容生态系统，而非选择独立应用模式进行运营。
