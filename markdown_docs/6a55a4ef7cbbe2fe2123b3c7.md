# 行业风向 | 丨海南将成中国首个禁售燃油车省份；长鑫科技部分员工入股成本低至1毛；SK海力士暴跌15%

近日，海南省人民政府对外发布《“十五五”海南国家生态文明试验区规划（美丽海南建设“十五五”规划）》提出，平稳推进2030年禁售燃油车，预计届时海南新能源汽车保有量占比从2025年的23.75%提升至45%。这意味着海南将成为中国第一个不再卖燃油车的省份。
“十四五”时期，海南清洁能源岛建设深化升级，新能源成为海南第一大电源，新能源汽车市场渗透率、保有量占比分别居全国省级地区第一、第二位。在交通运输领域，到2030年，海南全省公共服务和社会运营领域新增和更换车辆（特殊用途除外）清洁能源占比、私人用车领域新增和更换车辆新能源占比达到100%，车桩比保持2.5:1以下。（中国新闻网）
国内存储芯片龙头长鑫科技冲刺科创板IPO，招股书披露公司分两批落地员工持股方案，累计授予股份覆盖6760人次，数千名技术、生产与管理核心员工共享企业成长红利。整套激励计划分两期稳步推进，首期自2021年9月启动，历经九轮授予，惠及3596名员工，对应每股注册资本作价1.05元；二期2023年6月落地，完成七轮股份授予，覆盖3164人次，入股成本低至0.108元。
激励范围不局限管理层，研发骨干、产线关键技术工人、业务核心人员均纳入其中，重点倾斜芯片研发团队。截至2025年末，公司员工近两万名，研发人员比超三成，硕士及以上高学历人才规模突出，股权激励成为留住高端芯片人才的重要抓手。
本次员工持股采用长期绑定机制，设置多层锁定期，引导员工与企业长期同步发展。除此之外，公司董事长朱一明作出重磅让利承诺，自愿拿出自身持有的7.68亿股用于全员激励，计划分十年向全体在职员工分批发放，对应股份市场价值有望突破200亿元，创下A股个人出让股权激励员工的规模纪录。（经济观察网）
