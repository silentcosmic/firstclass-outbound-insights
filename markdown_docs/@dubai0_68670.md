# 柔佛新山移民局突袭：夜市商圈大规模抓捕行动

🚨 柔佛新山移民局突袭：夜市商圈大规模抓捕行动
马来西亚柔佛新山市区夜市商圈于6月29日晚间遭遇移民局大规模突击行动。现场画面显示，整条街道被移民执法车辆管控，大量外籍人员排队等候证件核查，场面一度拥挤，多名外籍人士当场被捕。
据悉，该商圈餐饮小吃店林立，吸引了众多外籍务工人员。近期，柔佛移民局执法力度空前，巡查范围覆盖工业区与夜市商圈，不分昼夜展开行动，此次突袭是其持续高压态势的又一体现。
