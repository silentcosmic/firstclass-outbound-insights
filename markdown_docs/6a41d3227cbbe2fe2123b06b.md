# 深度解析：会员制特卖如何重塑奢侈品库存管理与渠道生态

消费者初次接触到折扣奢侈品的准入门槛时，可能会感到意外。例如，在申请加入某会员制平台的限时特卖活动时，提交申请后数日仍未收到邀请消息，这与通常的购物体验截然不同。
按照该平台的规则，特卖活动需凭邀请码入场。若申请迟迟未能获批，则可能需要邀请两位好友注册，方有机会进入候补名单。至于最终能否获得邀请码，其机制并不完全透明。
消费者所尝试进入的，正是近年来在中国一线城市崭露头角的一种新型渠道——会员制奢侈品特卖平台。其中一家在2016年成立于香港，以限时快闪的形式销售奢侈品、设计师品牌及生活方式产品，是该领域颇具代表性的参与者。与传统奥特莱斯不同，这类平台不持有品牌所有权，也不运营长期折扣门店，而是直接与品牌方合作，在限定时间内消化其库存商品。正如其对外宣称的理念：“最精明的购物，往往不公开示人。”
近几年，类似的平台在上海、北京、香港、新加坡等国际都市不断涌现。它们的线下门店多选址于核心商圈附近，采用会员邀请制、预约制或限时快闪模式，这与消费者熟悉的郊区奥特莱斯形成了鲜明对比。
这些平台更倾向于自称为“特卖平台”，而非“奥特莱斯”。从商业逻辑层面分析，它们本质上已构成一种不同的业态。传统奥特莱斯更侧重地产运营：其收益主要依赖品牌入驻费、租金及联营扣点。而城市特卖平台则更趋近于纯粹的零售业务：它们面积更小、资产更轻，其核心竞争力在于买手能力、供应链管理效率以及库存周转速度。
这类新兴渠道的出现，与奢侈品行业近年来日益沉重的库存压力密切相关。例如，开云集团（Kering）计划在2025年削减8%的库存，并设定目标在2026年继续压降；在今年4月的资本市场日上，公司提出的“ReconKering”计划明确要在2026年底前净削减约10本站元库存，并将库存水平控制在销售额的20%以内。目前，旗下Gucci品牌业绩持续低迷，本季度销售额下降了8%。
对于奢侈品牌而言，库存积压所带来的成本远高于普通消费品行业。仓储、物流与资金占用仅是显性问题，更深层的影响在于，庞大的库存将拖累现金流，并对未来的价格体系构成潜在压力。
一位长期服务奢侈品牌的供应链专家指出，过去品牌更倾向于将货品保留在仓库中，等待下一轮销售机会。但如今，多数企业对库存周转天数的考核标准日趋严格，“与其任由商品贬值，不如尽快变现。只要渠道足够隐蔽，不影响正价体系，品牌方愿意提供比以往更大的折扣。”
奢侈品行业的高毛利率，也为这种特卖模式提供了生存空间。业内普遍认为，奢侈品牌的定倍率通常在十倍以上，部分顶级品牌甚至可高达数十倍。即便商品以原价的两到三折出售，品牌仍有机会覆盖成本并有效回收现金流。
据一位开云集团内部人士透露，目前类似会员制平台与品牌合作主要存在两种模式。其一是买断批发，即平台直接采购库存商品；其二是代理销售，由品牌掌握定价权，平台则收取佣金。后者通常能带来“更为可观”的利润。相较于批发模式，奢侈品牌通常更偏爱代理模式，因为它既能有效控制价格体系，也能精准掌握库存流向。
“渠道管理越严格的品牌，其快闪活动的表现通常越突出。”有业内观察人士表示，一些热门品牌的快闪活动虽然仅持续四五天，但单日销售额可能达到数百万元甚至更高。在整个奢侈品行业增速放缓的背景下，这一销售数字绝非等闲。
当然，特卖并非新鲜事物。早在二十多年前，欧美时尚界便存在“Sample Sale”（样品特卖）的形式。其参与者通常是品牌员工、媒体编辑、买手及熟客群体，更像是行业内部的一种福利，而非面向公众的零售活动。在那个信息不透明的时代，信息差本身就是一道门槛，知晓活动的举办地点、时间及参与资格，往往比购买能力更为关键。
随着奢侈品行业规模扩大、产量增长以及库存持续累积，这种原本封闭的小圈子活动逐渐演变为标准化的库存处理机制。疫情之后，这一趋势进一步加速。麦肯锡预测，奢侈品牌的折扣市场在2025年至2030年间的增长速度预计将是正价市场的五倍。
如果说十年前的特卖渠道主要承接的是上一季甚至数年前的库存，那么今天出现在快闪名单里的品牌，已经越来越贴近当下消费者关注的热门品牌。从Miu Miu、The Row到Lemaire、Jil Sander，不少品牌的产品仍处于市场热度高位。这表明特卖渠道不再仅仅是处理“滞销商品”，而正在成为品牌库存管理体系中的常规环节。即便仍有市场需求的产品，只要错过了最佳销售周期，也会被纳入更高效的周转体系。
有消费者在体验某会员制平台香港门店的活动时发现，从原价三万折扣后仅需8000元的皮衣，到现价不到1000元的格纹短裤，这些货品并非外界想象中的过时款式或设计平庸之作，相反，从实际穿着角度来看，其性价比颇高。
然而，对于平台方而言，快闪特卖终究是一门高度依赖项目制的生意。每一次活动都意味着需要重新寻找货源、重新获取客户、重新搭建销售场景。当市场规模扩大后，仅依靠几天一次的特卖活动显然难以支撑持续增长。因此，对于平台而言，最大的挑战并非如何销售商品，而是如何获得稳定、持续的库存来源。
Openmaxx开时仓的创始人Dylan Wang曾就职于字节跳动，在纽约读研期间曾为当地特卖零售商Prive代理跨境电商业务。2022年回国后，Dylan在国内复刻了Prive的模式，在上海成立了Openrack Flash Sales。初期专注于奢侈和轻奢品牌的快闪特卖，两年内业务扩展至六座城市。
2025年，Openrack Flash Sales更名为Openmaxx，业务范围开始扩展至家居、宠物、美妆、运动休闲、香氛等品类。其运营模式也从最初的限时快闪转向常驻门店与快闪相结合，同时主动缩减了门店规模。目前，其常驻门店主要集中在重庆和上海。
这种演变不仅反映了一家企业的发展路径，更折射出会员制特卖行业长期存在的内在张力：奢侈品牌需要这样的平台来解决库存问题，但这类平台却很难仅凭奢侈品单一品类实现持续生存与发展。
从这一视角审视，奢侈品牌与特卖平台之间，恰如城市地表与地下错综复杂的排水系统。正价门店负责展示品牌价值，奥特莱斯则承接规模化折扣，而这些会员制特卖平台，则精准消化那些既不宜长期陈列于橱窗，又不适合公开大幅降价的库存。它们虽不直接创造市场需求，却维系着整个奢侈品销售生态的运转效率。
对于消费者而言，这是一场寻找高性价比商品的寻宝游戏；对于品牌而言，这是库存管理与渠道控制的关键一环；而对于平台方而言，这则是一场关于品类扩张、会员留存以及商业模型能否持续成立的长期考验。
