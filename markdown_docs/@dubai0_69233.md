# 斯里兰卡监狱暴乱致25人死亡，安全风险升级

🚨 斯里兰卡监狱暴乱致25人死亡，安全风险升级
据当地媒体报道，斯里兰卡西部城市尼甘布的一所监狱近日发生严重暴乱，已造成至少25人死亡，约100人受伤。死者包括5名监狱官员和20名囚犯，伤者中亦有数十名警方安全部队成员。
事件最初于当地时间5日下午因两伙囚犯冲突引发，造成2名囚犯死亡和30余人受伤。警方一度宣布局势得到控制，但6日早晨，监狱内部再度爆发大规模冲突。斯里兰卡警方已向该监狱增派警力以稳定局势。对于在当地开展业务的企业或个人，此类事件无疑提示了潜在的安全与合规风险。
