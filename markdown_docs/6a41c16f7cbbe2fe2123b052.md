# 狮集团战略注资1200万美元，深化印尼稳定币及数字金融基础设施布局

狮集团控股有限公司（以下简称“狮集团”），作为一家综合性交易平台运营商和数字资产管理方，日前宣布向印尼科技企业PT Nusantara Bumi Sangkara（以下简称“NBS”）注资1200万美元。NBS专注于数字金融解决方案，业务范围涵盖发行与印尼盾挂钩的稳定币NIDR。
狮集团周一发布声明指出，根据与Meili Capital Management Limited（以下简称“Meili资本”）达成的投资参与及经济利益安排协议，狮集团将通过非现金交易方式，即发行其普通股或其他股本挂钩证券，提供高达1200万美元的投资对价。
交易完成后，狮集团将通过Meili资本或其指定的投资载体，获得NBS公司10%的间接经济权益或同等参与权。
此次交易架构与此前各方在战略合作备忘录（MOU）中确立的框架保持一致。
NBS公司正在开发NIDR稳定币，该稳定币将以1:1的比例锚定印尼盾，并由安全且流动性强的储备资产支持，旨在实现合规运营。
NBS致力于提供低成本的跨境资金转移和价值交换服务，同时构建更广泛的数字金融基础设施，深度融合区块链、智能合约、AI驱动的风险控制以及自动化决策等前沿技术。
近期，NBS已获得印尼金融服务管理局（OJK）的批准或确认，这使其有望成为市场上首批合规的稳定币发行方之一。
此次投资标志着狮集团“数字资产储备（DAT）”与“Aquila AI基础设施”双轨增长战略中的关键一步。
通过参与一个合规且由真实世界资产（RWA）支持的稳定币及数字支付平台，狮集团进一步实现其数字资产储备的多元化与强化。
这与公司现有持仓及对Hyperliquid、比特币等高潜力生态系统的储备再配置形成互补，旨在提升狮集团在东南亚高增长市场中代币化价值交换、支付以及链上基础设施领域的市场敞口。
与此同时，NBS公司对AI驱动风险控制、智能分析、欺诈检测以及自动化流动性管理的整合，与狮集团拟议收购总部位于美国的全球AI基础设施平台Aquila Hash, Inc.（其专注于AI工厂、GPU云平台和AI原生服务）形成直接协同效应。
综合来看，这些举措使狮集团站在AI与Web3融合的前沿，从而在数字金融、去中心化物理基础设施网络（DePIN）、真实世界资产代币化以及可扩展金融基础设施等领域实现更先进的应用。“通过Meili资本进行的此次战略性参与，标志着我们在拓展东南亚数字经济版图方面迈出了重要一步，”狮集团控股有限公司首席执行官Wilson Wang表示，“通过采用非现金交易中的股权对价，我们得以在保留资本的同时，获得一个前景广阔的稳定币及数字支付平台的敞口，这与我们的数字资产储备战略和Aquila AI基础设施愿景完美契合。”
