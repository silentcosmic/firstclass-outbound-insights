# 数字支付平台OKPay遭黑客攻击，官方称漏洞已修复

🚨 数字支付平台OKPay遭黑客攻击，官方称漏洞已修复
数字支付平台OKPay近日遭遇一起黑客攻击事件。据了解，一名自称为“方阳”的黑客利用平台漏洞，试图进行一笔高达45万OKPay的非法转账。OKPay官方迅速响应，已将该笔交易作废，并声明资金系非法获取，交易无效。平台同时表示，相关安全漏洞已得到修复。
此次事件再次凸显了数字资产平台面临的网络安全挑战。该黑客声称OKPay仍存在多处安全隐患，并透露其本次攻击可带来的潜在非法收益高达300万USDT。OKPay方面藉此提醒用户，在进行大额数字资产交易时，务必确保资金直接存入个人钱包，以最大限度保障资产安全。
