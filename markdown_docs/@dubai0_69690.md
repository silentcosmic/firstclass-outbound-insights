# 泰国总理阿努廷即将访华，深化区域合作

📣 泰国总理阿努廷即将访华，深化区域合作
在柬埔寨首相洪玛奈访华之后，东南亚外交再度升温。中国外交部于7月13日宣布，应国务院总理李强邀请，泰国总理阿努廷将于7月16日至20日对中国进行正式访问。
此次高层互访，继柬埔寨之后，进一步凸显了中国与东南亚国家间日益紧密的战略互动与合作深化趋势。
