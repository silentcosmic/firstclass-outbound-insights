# 出海情报速递：SK海力士预测2027年存储行业剧烈短缺；苹果对OpenAI提起商业机密诉讼；赛力斯上半年业绩承压

SK海力士：2027年存储芯片供应或迎历史性紧张
美东时间周五（7月10日）晚间，SK海力士首席执行官郭鲁正（Kwak Noh-jung）对外表示，预计2027年将成为存储行业有史以来供应最为紧张的一年。他指出，当前客户纷纷寻求与SK海力士签订长期供货协议，显示出持续增长的市场需求与公司现有产能之间的紧张关系。尽管公司正竭力扩大生产规模，但预计即使到2030年以后，客户需求仍可能超越公司的供应能力。郭鲁正还透露，美国在公司晶圆厂选址的候选名单之中，但尚未做出最终决定。公司将优先考虑成本、土地、水电及技术工人等综合条件优越的地区，目前美国、日本和东南亚均在其评估范围内。
苹果指控OpenAI窃取商业机密并提出法律诉讼
苹果公司已对OpenAI提起诉讼，指控其通过协同行动窃取苹果未发布产品的相关信息，构成商业机密盗窃。苹果声称，OpenAI曾鼓励其员工分享未发布产品的具体信息、零部件、设计图纸及其他专有资料，以助其开发自家的全系列硬件产品。苹果要求OpenAI立即停止此类行为，销毁所有涉及苹果专有资料的内容，并对其待发布产品进行重新设计，以确保其中不包含任何苹果的技术。对此，OpenAI方面回应称，公司对其他公司的商业秘密不感兴趣，其核心关注点仍在于持续打造创新技术。
赛力斯上半年预计亏损15亿至18亿元，问界汽车业绩波动
赛力斯日前发布公告，预计2026年上半年归母净利润将出现15亿元至18亿元的亏损，而去年同期则实现盈利29.41亿元。公司指出，亏损主要受多重因素影响：存储芯片、工业金属、碳酸锂等主要原材料价格上涨，导致公司生产成本显著增加；同时，基于审慎原则，并为进一步夯实整体资产质量，结合资产后续收益预期，公司对部分因技术迭代、车型换代导致适配性有限的存量资产调整了账面价值。受上述因素综合影响，公司核心子公司问界汽车的业绩出现波动，由盈利转为亏损，进而导致公司合并口径归属于上市公司股东的净利润为负值。

长江存储IPO辅导工作迎来首期进展更新。7月10日，证监会官网披露了其辅导工作第一期报告，其中一个细节尤为引人关注——中信证券、中信建投两家券商合计派出了31人组成辅导团队，如此规模在近年来的IPO项目中实属罕见。
鹰瞰智翼完成数千万元A轮融资，加速具身智能扑翼机器人商业化与研发
具身智能仿生扑翼飞行机器人公司“鹰瞰智翼”已于近期完成数千万元A轮融资。本轮融资是该公司在三个月内完成的第三轮融资，由元禾璞华领投，孚腾资本、厚雪资本跟投。募集资金将主要用于首款消费级产品的量产备货、市场拓展、团队扩充，以及下一代具身智能扑翼机器人与流体仿真引擎的研发。
