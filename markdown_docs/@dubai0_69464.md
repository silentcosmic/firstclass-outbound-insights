# 中亚出海突发：承诺落空，97人遭拘留！

🚨 中亚出海突发：承诺落空，97人遭拘留！
一份来自匿名渠道的投稿，为所有出海企业敲响了警钟。据披露，一个曾辗转缅甸、柬埔寨、马来西亚的中国团队，在寻求“稳定发展与保护”的承诺下，最终落脚中亚吉尔吉斯斯坦。然而，当初的保障承诺并未兑现，包括6名女性在内的97名团队成员，近期被当地国家安全委员会拘留，并被带往临时拘留所。
投稿人透露，此前承诺提供安全保障的当地官员最终背弃了诺言，将团队成员移交给了国家安全委员会。更令人担忧的是，团队负责人“阿洪”在事件发生后完全失联，对员工的求助信息置之不理，彻底拒绝提供任何援助。目前，所有被拘人员均面临巨大的精神压力。
鉴于此，投稿人呼吁所有从事海外业务的同行务必提高警惕，尤其是在异国他乡，切勿轻易相信任何承诺。同时，被拘人员表示，若无法脱困，将考虑向中国警方提交涉事老板信息并公开发布其照片，以寻求帮助。
