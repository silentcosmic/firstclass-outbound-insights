# ️ 泰国入境骗局频发：使馆提醒警惕“保关”陷阱

⚠️ 泰国入境骗局频发：使馆提醒警惕“保关”陷阱
中国驻泰国大使馆近日发布紧急提醒，揭露了近期在泰国口岸盛行的“保关”骗局。不法中介利用泰方加强边境管控之机，散布“付费通关”、“内部渠道”等虚假信息，诱骗赴泰公民购买高价服务，已导致部分人员蒙受财产损失。
泰国有关部门已对此类违法行为启动调查，并明确强调，不存在任何合法的付费“保关”服务，将严厉查处涉事中介及相关人员。
使馆特别提醒中国公民，务必提高警惕，切勿轻信此类虚假宣传。赴泰应通过合法途径办理入境手续，备齐有效护照、返程机票、住宿及资金证明等材料，并如实配合移民官员的查验。同时，严禁参与非法务工、电信诈骗、网络赌博等违法活动，共同维护良好的出入境秩序。如遇入境受阻，请保持冷静，依法寻求帮助，必要时可联系中国驻泰使领馆寻求领事协助。
