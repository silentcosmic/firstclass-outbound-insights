# FBI通缉缅甸诈骗主脑：涉数十亿美金，勾结军方

🚨 FBI通缉缅甸诈骗主脑：涉数十亿美金，勾结军方
市场消息指出，美国FBI通缉的缅甸诈骗集团核心人物“阿兴”（福建永安人），被指控实际掌控缅甸多个大型诈骗园区，包括妙瓦底交克（现智慧城）、新泰和（又称炮弹坑）、水沟谷东方物业以及亚太6号地物业。这些园区据称强制控制了逾万名中国籍、香港及马来西亚籍人员，通过军事包装、投资理财、加密货币和跨境电商等名义，诱骗美国、香港、新加坡及中国大陆数百万受害者，涉案金额高达数百亿美金。
爆料称，在全球严厉打击电信诈骗的背景下，阿兴集团仍能勾结亚太地区军方保护伞，新建如亚太6号地物业等关押多国公民的诈骗园区。其与亚太军方的合作关系被认为堪比此前从泰国移交回国的诈骗首脑佘志江。
消息还披露，阿兴曾因私吞股东分红并伪造账目，指使其集团二把手“图图”在皇潮国际KTV内强迫财务吸毒，待其昏迷后指使保安将其从亚太东方物业楼顶抛下，制造跳楼假象。目前，“图图”据称仍在亚太6号地物业强制被诱骗人员进行诈骗活动。
尽管亚太军方对外宣称爆破违建以清除电诈，但私下却被指与这些诈骗集团主脑勾结，提供保护。阿兴目前据称藏匿于马来西亚和泰国，相关部门正持续关注其动向。
