# 柬埔寨纸箱厂老板回应家务纠纷：承认出轨，否认弃家

🔍 柬埔寨纸箱厂老板回应家务纠纷：承认出轨，否认弃家
柬埔寨永佳纸箱厂有限公司老板赵某华近日首次公开回应此前其妻女提出的多项指控，承认部分事实，但否认关键指控。
赵某华承认婚内出轨，并表示愿向妻子及三个孩子道歉。但他否认长期不顾家庭、抛弃妻儿，解释称与第三者徐某春的关系发生在其在柬埔寨工作经商二十多年、长期与妻子两地分居期间。
为反驳长期不给生活费的指控，赵某华提供了大量聊天及转账记录，显示二十多年来累计向妻子汇款逾200万元人民币，款项用于家庭开支、购房购车及患病儿子的医疗费等，单笔最高达34万元。他同时否认曾说过“希望患病儿子早点死”的言论，并表示一直关心儿子的治疗。
关于殴打女儿的指控，赵某华承认因第三者问题与女儿争吵时情绪失控掌掴女儿，对此感到后悔。他也承认抓奸视频中冲突基本属实，并称当时曾拿起水果刀，后被警方带走调查。
针对外界质疑其依靠妻子娘家发展，赵某华承认初期通过妻子娘家亲属介绍赴柬工作，并于2013年与包括妻子娘家亲属在内的10人共同投资创办吉安纸箱厂。但他强调目前永佳纸箱厂已由其独资控股，公司正常运营。
赵某华推测妻女公开曝光可能与财产分配担忧有关，并表示若妻子坚持离婚，将尊重决定并通过法律程序处理婚姻及财产问题。目前，此案多方说法仍存在明显分歧，或需法律途径解决。
