# 千米高楼技术已成熟：哈利法塔工程师揭示真正挑战

⚡ 千米高楼技术已成熟：哈利法塔工程师揭示真正挑战
全球目光聚焦于谁能率先建成首座千米摩天大楼之际，迪拜哈利法塔的首席结构工程师比尔·贝克却提出了一个颠覆性观点：人类的建筑高度远未触顶，真正的瓶颈已不再是技术本身。
这位SOM公司的咨询合伙人直言，现有技术足以支撑数公里高的摩天大楼，甚至未来超高强度混凝土等新材料还将持续刷新上限。他强调，从纯粹工程角度看，我们远未达到极限。
贝克指出，真正的挑战已转向建造更高建筑的价值所在。换言之，问题已从“我们能否建造”转变为“我们是否真的需要建造”——这无疑为全球超高层建筑的未来发展方向提供了新的思考维度。
