# 越南胡志明市警方突击：多名中国人涉诈被捕

⚡ 越南胡志明市警方突击：多名中国人涉诈被捕
近期，越南胡志明市警方在入境检查高峰期加强巡查，成功破获多起跨国诈骗案件。警方发现有中国籍人员租用酒店及民宅，涉嫌安装设备从事诈骗活动，严重扰乱社会秩序。
6月21日，胡志明市警方依法逮捕了涉嫌非法组织犯罪活动的中国籍嫌疑人范海明、曹嘉良，以及阮鸿草二（34岁），目前案件正在深入调查中。
警方同时在一处名为“宝丽小屋”的地点查获数十名非法居住的中国籍人员，并发现了大量用于诈骗的设备。警方提醒公众务必提高警惕，积极配合执法部门，共同打击此类跨境犯罪行为。
