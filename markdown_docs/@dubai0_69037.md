# 菲律宾回国经商者：护照被剪，限制出境三年！

🚨 菲律宾回国经商者：护照被剪，限制出境三年！
近日，一名长期在菲律宾经营餐饮生意的中国公民，在回国处理家事后，遭遇了出境限制。据当事人自述，其护照被剪角作废，并被告知将面临长达三年的出境禁令。
该男子表示，他曾携带营业执照和经营流水等材料，意图证明其在菲律宾的合法经营活动。然而，回国次日即被当地警方传唤调查，重点核查其境外工作经历与资金往来，以及是否涉及电信网络诈骗、跨境赌博等违法犯罪活动。最终，他被告知因长期停留于涉电信网络诈骗高风险国家，已被列为重点管控对象。
尽管当事人坚称仅从事正常餐饮经营，未涉任何违法活动，但此次限制已对其后续工作与生活造成重大影响。此案也再次凸显，在我国持续加大对电信网络诈骗及跨境犯罪打击力度的背景下，从高风险区域回流人员面临的身份核验与风险评估将日益严格，可能对个人出入境及证件办理产生实质性影响。
