# 退役13年仍吸金：贝克汉姆世界杯期间狂揽1.7亿

💰 退役13年仍吸金：贝克汉姆世界杯期间狂揽1.7亿
尽管已退役13年，传奇球星大卫·贝克汉姆在本届世界杯期间展现了惊人的商业吸金能力。据《福布斯》和Sportico等机构统计，在约40天的赛事期间，贝克汉姆仅凭商业代言及品牌合作，就进账约2500万美元（折合人民币约1.7亿元），超越了所有现役参赛球员。需要明确的是，这笔收益并非其全年收入，而是世界杯赛事期间新增的商业净利。
数据显示，贝克汉姆的商业收入遥遥领先，位居榜首。紧随其后的是梅西，收入约2100万美元；姆巴佩以约1800万美元排名第三。即便与当红球星如贝林厄姆（约1200万美元）和凯恩（约1000万美元）相比，贝克汉姆的商业价值也展现出压倒性优势。
这充分证明，尽管已远离绿茵场多年，贝克汉姆凭借其强大的个人品牌影响力和全球号召力，其商业价值在世界足坛依然稳居顶流。
