# 美伊冲突白热化：特朗普欲扩大打击，油路面临双重封锁

🔥 美伊冲突白热化：特朗普欲扩大打击，油路面临双重封锁
中东紧张局势骤然升级。据伊朗官方媒体消息，美军新一轮袭击已导致伊朗境内五座桥梁受损，造成平民伤亡。与此同时，巴林与卡塔尔首都多哈相继拉响防空警报，科威特军方也报告拦截“敌对”导弹及无人机威胁。作为回应，伊朗陆军宣称已对美国驻巴林谢赫伊萨空军基地实施无人机袭击，目标包括美军直升机与P-8侦察机部署区域。
在此背景下，美国总统特朗普正考虑扩大对伊朗的军事打击范围，以迫使其重返谈判桌。据《华尔街日报》援引知情人士透露，特朗普已与高级幕僚讨论了包括夺取哈尔克岛、打击“镐山”地下核设施以及扩大对伊朗能源设施空袭等方案。尽管特朗普尚未作出最终决定，并曾表示更倾向外交解决，但伊朗未按美方要求交出核材料库存，促使其倾向于升级打击。美国国防部一份内部评估显示，对伊军事行动的实际支出可能已高达1000亿美元，远超官方公布的300亿美元。
更令人担忧的是，有消息称伊朗已指示也门胡塞武装做好准备，一旦美国袭击伊朗关键基础设施，便封锁红海关键石油通道曼德海峡。此前，受冲突影响，霍尔木兹海峡航运已受阻，沙特阿拉伯已将原油出口主要转向通过曼德海峡的延布港。若曼德海峡也被封锁，全球11%的海上贸易将受影响，中东两大主要石油出口通道可能同时中断，对全球能源供给造成巨大冲击。
