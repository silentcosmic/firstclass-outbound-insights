# 网易《遗忘之海》深度解析：七年磨砺如何重塑开放世界体验与商业生态

7月9日，网易旗下Joker工作室推出了他们耗时七年精心打磨的巨作——《遗忘之海》，一款宏大的海洋奇遇角色扮演游戏。PC端已率先启动公测，移动端版本也计划于7月23日与玩家见面。
这款独具匠心的怪诞木偶风格航海游戏，在正式启航前便已吸引了广泛关注。公测前夕，其全网预约量已突破3600万，并在多个主流游戏预约平台预约榜单上冲至首位。游戏实机演示视频在视频平台上播放量破千万，有市场分析机构预测其首年营收潜力可观。
网易高层管理团队连续三个季度在财报会议上对该项目表示坚定支持，称其“令人耳目一新”。历经七年精雕细琢，实现PC、移动端数据互通，并投入五千万元启动创作者扶持计划——从研发资源到生态构建，这款产品几乎汇集了网易近年来最集中的投入与期望。
作为打造了《第五人格》的团队，Joker工作室首次涉足开放世界领域，并延续了其标志性的怪诞美术路线。在当前国产游戏普遍追求极致写实或扎堆唯美二次元的潮流中，Joker工作室反其道而行，巧妙地运用明亮、高饱和度的色彩，构建出一个由木偶造型与波普艺术融合而成的独特海洋大世界。
对于Joker工作室而言，这种一以贯之的美术风格为《遗忘之海》带来了极高的辨识度。当今开放世界游戏市场在写实与二次元两大方向上已趋于饱和，在这样的格局下，《遗忘之海》的木偶主角形象在美术风格上独树一帜，成功实现了与其他产品的差异化竞争。
然而，对于一款开放世界游戏，真正能长期吸引并留住玩家的，依旧是其核心玩法。在深入体验《遗忘之海》后，我们认为其独特之处，或许正源于一套高度自洽且环环相扣的生态循环设计。
航海与陆地：双线战斗的策略深度
游戏的世界观设定在一片被诅咒的海域——任何坠入海水的人，都将失去记忆与才能。玩家扮演一位失忆的木偶船长，携手一位身份不明的少女艾丝共同出海，在反复的探索中，逐步找回自己和伙伴遗失的记忆片段。
每一次出海，本质上都等同于一局Roguelike冒险。海上收集的所有物品，一旦回港或角色倒下，便荡然无存，角色属性亦回归初始状态。若被击败导致“记忆值”归零，当前探索将强制终止。唯一能带回主城的，仅有少量与长期养成挂钩的稀有资源。每次返航所积累的，是对风向的判断、对骰子机制的计算，以及对每个岛屿特性的熟悉。
返航即归零，这种周而复始的循环设计，使得每一次“出海”都成为需要慎重对待的决策。风向、补给、航线——每一个选择都可能直接影响本次航程的远近，因为玩家面临的是真正意义上的“失去”。
在海上，战斗采用即时制。玩家需要亲自掌舵、规避风暴，并利用洋流和风向抢占有利战位。同时，必须时刻关注天气变化，因为暴风雨确实会使炮弹轨迹偏离。遭遇海怪时，则需迅速集中火力攻击其触手，发光区域通常是其弱点所在。船只本身可通过升级龙骨、火炮等模块进行强化，船员也能分配至不同炮位，形成战术分工。
一旦登岸，战斗模式立即切换为策略回合制，并融入了骰子检定玩法。玩家可主动消耗数量不等的骰子来触发技能判定，点数高低决定效果强弱，消耗骰子越多，技能强化幅度越大。
敌人身上还会显示一个固定的破绽数值，玩家投掷的骰子点数之和若能超越此数值，便可触发额外行动回合。如果运气连贯触发，理论上能实现“我的回合，依旧是我的回合”的连续攻击循环。此外，攻击还能削弱敌人的架势条，打空后将造成高额伤害并延后其行动节奏。
单一操作层面的节奏变化，难以支撑长期的重复体验。无论海战如何刺激，骰子机制如何惊险，当返港清点战利品时，一个核心问题仍会浮现：这些物品除了用于角色升级，是否还有更深层次的、更具活力的意义？
交易行与生态循环：一场大胆的社会化实验
在海域自由探索中免费获取的殿堂级宝藏和金罐，转手出售即可获得专属流通货币“好金砖”。高付费玩家可以通过出售多余时装换取养成资源，而低付费玩家则可依靠日常“搬砖”积累“好金砖”，进而兑换原本需要付费获取的内容。从逻辑上，这使得“理论上白嫖所有付费内容”成为可能。
更深层次地思考，交易行的存在，实质上是对“搜打撤”玩法机制的巨大正向反馈。它让每一次出海探索的收获，不再仅仅局限于默默服务于自身的角色养成，而是具备了更广泛的社会价值和流通属性。这极大地增强了“出海”行为的驱动力——玩家不仅是在为自己付出，也可能挖掘到他人急需的“珍宝”。这种隐约存在的可能性，有时比任何固定奖励都更能触动人心。
然而，将此设计置于内容驱动型开放世界中，无疑是一步险棋。交易行机制在大型多人在线游戏（MMO）赛道已被反复验证，网易凭借《梦幻西游》等长线产品，对供需调控的底层逻辑驾轻就熟。但核心挑战在于，MMO的成功经验能否直接移植到一个以Roguelike循环为核心的开放世界中，目前尚无定论。
交易行若要维系生态稳定，必须具备两个硬性前提：足够庞大的用户基础，以及一套精密且能灵活响应的供需调控机制。前者，《遗忘之海》上线后的用户数据将给出答案；后者，则是一场没有终点、需要长期动态平衡的治理考验。有行业观察者将此解读为该产品“挑战开放世界此前不敢触碰的生态领域”。
这种诱惑背后潜藏的风险同样引人注目：当付费玩家的“特权”能够通过市场行为逐步流向低付费玩家时，玩家整体的付费意愿最终会倒向哪一边，仍是未知数。但从另一个角度看，如果这个生态真能稳定运行，它为“搜打撤”循环带来的正反馈效应将是实实在在的。这一步，网易迈得足够大胆，也足够令人好奇。
《遗忘之海》甚至在角色获取方式上也透露着一股“不按套路出牌”的坚韧。它并未照搬传统的抽卡模式，而是通过通关噩梦深潜层数来免费获取。玩家每周完成一次“遗忘之旅”，就有50%的概率获得SSR角色；即使运气不佳，每月也能稳定获取一个SSR。付费点则主要集中在外观定制和加速养成方面。
玩家在“搜打撤”过程中积累的资源，最终服务于角色养成；而构建成熟的强大队伍，又能让玩家在下一轮“遗忘之旅”中走得更远，获得更丰厚的奖励。战斗为养成提供了验证场景，养成则为战斗提供了数值支撑，二者共同构成了一条清晰的玩家成长心流。
《遗忘之海》的核心玩法“遗忘之旅”以七天为一个周期，玩家在各个岛屿上搜刮、构建角色，最终抵达遗忘之海进行奖励结算。多周目的效率曲线几乎呈指数级增长：有玩家反馈，首次周目可能耗时近七小时才勉强完成构建，而第二周目则缩短至不到三小时，到第三周目时，一小时左右便能跑完一轮。这种效率的压缩并非源于难度降低，而是基于玩家对游戏理解的加速深化。
这正是Roguelike游戏最引人入胜之处：变强的并非角色数值，而是玩家自身。每个角色都可以自由搭配职业技能、船歌、职业天赋和装备，玩家可以本轮选择持续伤害辅助流，下一轮切换为主攻击输出，再下一轮尝试一套奇特的组合。每一次“遗忘之旅”，都如同重新洗牌，而玩家手中的“好牌”，则来自于对牌局越来越深刻的洞察。
奥托皮亚：一座会呼吸的城市社区
主城奥托皮亚，或许是这款游戏最具颠覆性的设计之一。大多数游戏的主城，更像是一个功能堆砌的中转站，提供强化、兑换、任务交接等标准化服务，玩家很少会主动漫无目的地闲逛。然而，奥托皮亚并非如此。它是一座真正可以被玩家探索与影响的城市。
玩家可以自由出入任何建筑，主动与NPC互动、寻找乐趣，甚至可以设法“送走”某些角色。NPC消失后，他们所照看的店铺会歇业或由他人顶替。玩家的行为将真实地影响这座城市的兴衰，以及不同势力的此消彼长。
制作团队甚至埋藏了一些彩蛋设计。例如“荒凉结局”，如果玩家执意送走大部分主城NPC，整个奥托皮亚将变得破败潮湿，终日阴雨，玩家还会获得一把“伞”玩具作为纪念。更令人匪夷所思的是隐藏的“宕机结局”：游戏直接进入类似404的异常状态，无法养成、无法出海，甚至无法充值，只能联系客服进行回档。这种玩法，在主流商业作品中几乎不可能出现。它透露出一种“我不想讨好所有人”的任性，而这种任性，恰恰是当前同质化市场中最为稀缺的特质。
与此一脉相承的，是游戏的叙事方式。它并未沿袭传统线性主线的旧路。如果玩家只专注于推进主线，剧情可能会显得相当单薄。开发团队将更多的笔墨隐藏在碎片化叙事和支线选择之中。当玩家在一座雾蒙蒙的小岛上翻到一本旧日记，寥寥数段文字便拼凑出一个早已消失的船员的一生，那种寂静的触动，远比站桩式对话来得更为强烈。
这种高度的自由度，使得主城更像一个会呼吸的社区，而非一张冰冷的功能菜单。当然，追求快速剧情体验的玩家或许会感到迷茫，但如果愿意放慢脚步细细探索，或许会被某些角落里隐藏的细节轻轻触动。
《遗忘之海》在海域风貌上也刻意做了分化处理。白沙海域以明媚的寻宝路线为主，而渡桥海则笼罩在浓雾之下，以水枪博弈为核心玩法。两种截然不同的海岛调性，使得每一次靠岸都像抽取一张未知的牌。7月底还将解锁更为隐秘的海域，进一步拓展这张海域图谱的纵深与探索空间。
然而，海洋题材真正的魅力，在于它天然契合了开放世界探索的终极奥义：延续心流，提供沉浸式的放松体验。正如在开放世界中，玩家既可以选择直奔任务目标，也可以漫无目的地闲逛一样，在《遗忘之海》中，玩家既可以为了资源直抵目标岛屿，也可以单纯享受航海本身的乐趣——感受风向变化，欣赏夕阳将海面染成金色，偶遇一场暴雨，或是发现一处地图上并未标注的隐藏洞穴。
一方面，这款产品的野心是真切而具体的。多周目Roguelike构建、海陆双线战斗的节奏对撞、主城生态的可干预性、碎片化叙事——这些设计单独来看，每一项都算不上惊天动地，但当它们被同时整合进一个框架内，并试图彼此咬合时，玩家能清晰地感受到这个团队并不满足于做一款“合格的开放世界”，他们所追求的，是让开放世界的每一个子系统都能相互滋养、彼此支撑。
但另一方面，潜在的风险同样具体。交易行的供需平衡能否支撑长线运营？多周目构建的重复感是否会在数百小时后产生反噬？复杂的双战斗体系是否会劝退轻度用户？这些疑问，目前尚无人能给出确凿的保证。
七年磨一剑，从非对称竞技到宏大开放世界，无论最终的市场反馈是热烈还是平淡，《遗忘之海》的这次出海，本身就已经为国产开发者们提供了一个全新的思路：开放世界的未来，本就不该只有一种固定的模样。
