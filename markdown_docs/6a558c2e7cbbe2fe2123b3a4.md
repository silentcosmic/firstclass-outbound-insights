# 新加坡与日本深化能源市场监管合作，共促区域稳定与创新

新加坡能源市场管理局（EMA）与日本电力及天然气市场监管委员会（EGC）近日宣布，将全面深化在能源市场监管领域的合作。此次合作旨在推动两国在天然气与电力市场信息方面的互通共享，并促进双方在市场监测方法、监管框架设计以及系统稳定性措施等关键领域的经验交流与知识共享。
新加坡能源市场管理局于本周一发布声明指出，已与日本电力及天然气市场监管委员会正式签署合作备忘录。该协议确立了一系列合作机制，包括两国监管机构间的常态化对话、技术交流、互访考察以及联合研究等。这份具有里程碑意义的协议在新加坡签署，由新加坡能源市场管理局总裁傅伟基（Puah Kok Keong）与日本电力及天然气市场监管委员会秘书长新川达也（Tatsuya Shinkawa）共同见证并签署。
新加坡能源市场管理局总裁傅伟基强调，与日本监管机构的交流是该局为确保能源市场高效运作及电力供应可靠性所持续努力的一部分，尤其在全球能源格局快速演变的大背景下。傅伟基进一步表示，新加坡能源市场管理局期待在即将于2026年10月新加坡国际能源周期间举行的“监管机构@SIEW”活动中，与日方展开更为紧密的合作。
日本电力及天然气市场监管委员会秘书长新川达也指出，日本与新加坡在各自的能源领域拥有诸多共性，并且在保障可靠能源供应和构建高效电力市场方面面临相似的挑战。他表示，日本监管委员会期望通过此次合作，进一步强化两国间的协同，提升消费者权益保障水平，最终实现互利共赢。
