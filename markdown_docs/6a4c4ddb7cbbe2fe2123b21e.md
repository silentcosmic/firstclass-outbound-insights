# 马来西亚数字银行Boost：智能代理AI驱动客户服务变革，效率提升九成

Boost，一家领先的马来西亚数字银行及金融科技企业，近日宣布对其智能代理AI平台“Boba AI”进行全面强化升级。此平台旨在通过其在Boost应用及Boost银行应用中的聊天和语音交互功能，赋能客户高效完成各项日常银行操作。
Boost在最新声明中披露，此次升级的关键进展是“Boba Voice”的正式上线。这项24/7全天候AI驱动的语音服务已于2026年5月投入使用。自推出以来，“Boba Voice”已成功处理了高达90%的客户咨询，其中超过80%的问题在首次交互中即得到妥善解决，无需转接人工坐席。客户等待时间也因此大幅优化，最短可缩减至三秒。
自2026年1月以来，已有逾3万名客户通过Boost应用与“Boba AI”进行了深度互动，累计生成了近10万次服务请求。如今，“Boba AI”的功能已远超简单的问答范畴，它能协助客户开立银行账户、顺利完成入职流程与电子身份验证（eKYC），并通过实时获取欠款信息完成账单支付，并在一次对话中解决多种复杂的查询。这些能力显著简化了客户过去在多界面间切换或手动完成常规流程的繁琐操作。
Boost集团首席执行官Sheyantha Abeykoon强调，人工智能正在超越单纯的回答问题阶段，迈向实际执行任务的新里程碑。作为马来西亚数字金融服务领域中智能代理AI能力的早期实践者，Boost正持续精进其AI代理，致力于使银行服务变得更简单、更迅速、更具直观性。
Boost未来计划持续拓展“Boba AI”的功能边界，未来的增强重点将聚焦于赋能AI代理，代客户处理更广泛、更复杂的金融任务。
