# 马来西亚重拳出击：网络犯罪法案最高监7年、罚50万

⚡ 马来西亚重拳出击：网络犯罪法案最高监7年、罚50万
马来西亚副首相拿督斯里阿末扎希近日在国会下议院提呈《2026年网络犯罪法案》二读时指出，该法案旨在应对数字时代快速发展带来的挑战，弥补现有法律空白，以提升马来西亚法律的时效性和适应能力。
这项新法案将全面加强打击网络犯罪的执法机制，覆盖预防、侦测及应对日益复杂的网络威胁。根据法案要点，若犯罪行为涉及致人伤害、羞辱、胁迫或威胁，最高可判处7年监禁，或罚款50万令吉，或两者兼施，大幅加重了刑罚。此外，法案明确将未经同意制作或传播含性性质的私密照片或视频列为刑事罪行，并要求服务提供商承担平台责任，确保其平台不被滥用于网络犯罪活动。法案还计划设立一个具备法定职能的打击网络犯罪专门委员会。
