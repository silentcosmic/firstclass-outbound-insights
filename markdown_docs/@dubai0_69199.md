# 地区冲突冲击阿联酋就业：低薪岗位招聘锐减

📉 地区冲突冲击阿联酋就业：低薪岗位招聘锐减
一份最新报告揭示，受地区军事冲突影响，阿联酋招聘市场显著降温，其中低收入群体受到的冲击最为剧烈。
数据显示，今年4月至5月，月薪低于1万迪拉姆的职位招聘量同比大幅减少26%。相比之下，高薪岗位表现出更强的韧性：月薪1.1万至2万迪拉姆的岗位下降约22%，2.1万至4万迪拉姆的岗位下降约19%，而4.1万至8万迪拉姆的岗位仅下降11%。月薪8万至15万迪拉姆的高薪职位几乎未受影响，仅微降5%。尽管月薪超过15万迪拉姆的极高薪岗位招聘量也出现23%的降幅，但行业观察人士指出，这更多是由于基数较小导致的百分比放大，雇主对这类高级别人才的需求依然旺盛。
自2月28日地区军事冲突爆发以来，阿联酋及海湾地区经济增长已明显放缓，旅游、贸易、观光等关键行业均受波及。受此影响，当地私营企业纷纷采取裁员或放慢招聘节奏等措施以应对市场变化。
