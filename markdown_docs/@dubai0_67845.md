# 迪拜涉债纠纷：暴力催收案致多方入狱

🚨 迪拜涉债纠纷：暴力催收案致多方入狱
市场消息披露，一桩迪拜本地的涉债纠纷近期浮出水面，其暴力催收行为已导致多名涉案人员面临法律制裁。
据一匿名知情人士自述，他因卷入一起为“上岸钱”（涉赌或诈骗所得）的债务追讨行动，被指控抢劫并已服刑数月。该人士透露，他曾驾车送两名同伴至一别墅进行债务追讨，虽未入内，但其同伴在屋内对欠债人实施了殴打，并抢走手机，随后协商还款日期离去。然而，欠债人随即报警指控抢劫，导致该人士因车辆涉案被捕。
目前，其一名同伴“上岸”已落网服刑，而另一名涉案人员“阿疯”则仍在迪拜潜逃，据称其为迪拜黑户，无护照且在国内被通缉，已有三次在迪拜入狱记录。据称，被追讨的欠债人本身亦涉嫌诈骗活动。
此案再次凸显了海外华人社区在非正规债务纠纷处理中面临的巨大法律风险，尤其是在迪拜等对金融犯罪和暴力行为持零容忍态度的司法管辖区。企业及个人应严格遵守当地法律法规，避免卷入任何可能导致严重后果的非法活动。
