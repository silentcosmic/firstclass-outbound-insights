# 泰国警方逮捕中国籍非法代孕网头目

🚨 泰国警方逮捕中国籍非法代孕网头目
泰国移民局近日通报，两名涉嫌在中国境内组织大规模非法代孕网络的中国籍核心头目，已在泰国芭提雅落网。此案揭示了跨境非法医疗服务的严峻挑战。
涉案嫌疑人分别为43岁的鄢某和34岁的米某。据中方警务部门通报，两人涉嫌在近期（原通报提及2026年3月至5月期间）利用“杭州甥宝医疗健康科技公司”名义，在未取得医疗执业许可的情况下擅自设立医疗点，非法提供人工授精、胚胎移植及代孕服务以牟取暴利，涉案人数众多。
案发后，两人潜逃至泰国。泰国移民局接报后迅速行动，依法撤销其在泰签证，并联合中央调查局在芭提雅某豪华酒店成功将二人抓获。目前，嫌犯已被羁押，正等待遣返回国。
