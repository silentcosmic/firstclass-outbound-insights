# 中国网文平台AI应用困境：从拥抱效率到规范内容生态

32岁的中国工程师盛戈（音译，Gordon Sheng）拥有长达二十年的网络小说阅读经验，如今他已开始借助人工智能创作自己的作品。他利用DeepSeek勾勒了一个戏剧性的离婚故事大纲，随后通过AI写作工具仅用五分钟便生成了完整文本。这部短篇小说发布于番茄小说平台，在短短10天内便获得了超过5500次阅读。盛戈表示：“即使AI写作存在诸多不足，它的表现也远胜于我自己动笔。”
网络小说起源于上世纪90年代的中国互联网论坛，通常以连载形式呈现，专为智能手机用户设计。其内容往往围绕复仇、时间旅行、武侠争斗或浪漫情感等主题，故事情节扣人心弦，引人入胜。“很多人内心都有一个故事的火花，但不知道如何将其表达出来。AI的出现，弥合了我们与创作之间的鸿沟。”——盛戈，工程师兼作者
如今，越来越多的作者正转向人工智能工具，以期批量生产具有传播潜力的故事。虽然网文平台最初鼓励作者利用AI提升创作效率，但当前它们正面临巨大压力，需对可能威胁原创性并导致读者流失的AI生成内容进行严格规范。
网文平台堪称AI写作领域的先行者。2023年，腾讯旗下的阅文集团推出了一款AI工具，能够生成角色名称并将粗略的情节线索发展为更完整的故事。时任首席执行官侯晓楠曾形容其为“从人工驾驶向辅助驾驶的转变”。阅文集团拥有起点和晋江两大中文网络文学平台，前者深受男性读者青睐，后者则主要面向女性读者。这两个平台均通过订阅服务和销售作品改编权实现盈利。
此外，阅文集团还在其国际站WebNovel上发布了超过17000部经过AI翻译的故事，美国、印度和巴西是其主要的海外市场。
字节跳动旗下的番茄小说以免费阅读模式运营，通过广告获取收益。该平台于2024年推出了一款“作者助手”，旨在协助作者研究故事的历史或科学背景，并在创作遇到瓶颈时提供持续支持。
此后，AI工具的迭代升级日益显著。北京的开发者马军贤（音译，Junxian Ma）近期构建了InkOS，一个由多个人工智能代理驱动的自动化小说写作系统。其中包括负责构建大纲的“架构师”、生成章节内容的“写手”，以及检查逻辑错误和过多破折号使用的“审计员”。人类作者仅需偶尔介入，引导故事情节走向。
马军贤透露，该系统已有超过5万用户下载。其中部分用户已通过在番茄小说发布AI生成的故事而获得收益。他指出：“现在，人类拥有更多的时间和精力去思考他们真正想讲述什么样的故事。”
然而，AI写作的泛滥也引发了部分读者的不满。在一些社交媒体平台上，用户分享了小说中意外遗留AI提示语的截图。还有读者表示，每当他们遇到奇怪的比喻或诸如“它不是……而是……”之类的句式时，便会怀疑作品中存在AI创作的痕迹。对于一些读者而言，AI写作的明显特征令人望而却步。一位爱好奇幻小说的软件开发者杨舟（音译，Yang Zhou）表示：“阅读快速生成的作品是在浪费我的时间。但如果一部小说是随着时间推移，缓慢地创作和更新，我会更愿意花时间去品味它。”
此外，作者群体也担忧AI技术助长了版权侵权行为。2024年，番茄小说曾要求作者授权其作品用于AI训练，此举引发了网文作者的抗议，平台随后允许作者选择退出该计划。一些AI工具甚至被发现助长抄袭，其推广的功能能够提取流行故事的情节，并将其重写为“新”小说。
面对这些挑战，平台方正逐步收紧对AI生成作品的监管。晋江创始人黄艳明在2025年的一份声明中，要求作者仅将AI用于故事研究和校对，并鼓励读者举报疑似包含AI写作内容的小说。然而，黄艳明也坦承，要精确判断一部作品中AI的介入程度仍十分困难。
番茄小说目前已对每个账户每日允许发布的字数设定了上限。该平台近期一份声明显示，仅在六月份，就拒绝了超过10.4万篇“低质量”投稿，其中包含了部分由AI创作的作品。
悉尼大学研究中国数字出版产业的高级讲师任翔（音译，Xiang Ren）指出，网文平台正努力在提升作者生产力的AI潜力与原创性受到的威胁之间寻求平衡。他进一步分析，像番茄小说这类免费阅读平台，相对于需要通过高质量内容吸引读者付费的订阅制平台（如晋江），往往对AI写作表现出更高的容忍度。
拥抱AI的创作者认为，这项技术帮助他们克服了专业写作训练的不足。盛戈指出，尽管AI为他完成了写作，但故事背后的价值观和逻辑构架仍源于他本人。他也承认需要修正AI写作中的“漏洞”，例如，他发现自己短篇小说中的父亲，尽管与女儿近在咫尺，却仍然发送短信与她交流。他总结道：“许多人心中都有故事的萌芽，却苦于不知如何表达。AI的出现，为我们填补了这一空白。”
