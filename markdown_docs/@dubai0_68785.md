# 中国高层代表团赴伊朗，出席哈梅内伊葬礼

🔥 中国高层代表团赴伊朗，出席哈梅内伊葬礼
中国外交部周四（7月2日）发布消息，全国人大常委会副委员长何维将代表中方，于周五（7月3日）赴德黑兰出席伊朗已故最高领袖哈梅内伊的葬礼。此次高层出席，凸显了中方对地区局势及中伊关系的重视。
