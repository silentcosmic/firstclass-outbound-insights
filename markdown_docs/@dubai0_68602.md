# 泰航空姐澳洲涉毒被捕：藏匿1.13公斤海洛因

🚨 泰航空姐澳洲涉毒被捕：藏匿1.13公斤海洛因
据澳大利亚联邦警察局消息，泰国国际航空一名26岁空姐因涉嫌走私毒品在墨尔本机场被捕。该乘务员于6月25日被海关人员查获，在其随身携带的多个手提包中藏匿了1.13公斤海洛因，市值估算约34.33万美元。
目前，该女子正面临非法进口及持有毒品的指控。若罪名成立，每项罪名最高可判处25年监禁。此案再次引发了对航空业内部安全管理及人员背景审查的关注。
