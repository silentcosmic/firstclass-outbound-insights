# 优衣库Uniqlo U系列十年之变：Lemaire与迅销的战略交汇与分流

在巴黎巴士底歌剧院尚未竣工的音乐厅内，Lemaire最新一季大秀落下帷幕。然而，真正牵动时尚界神经的，是其幕后一场可能重塑行业格局的重大变动。
权威行业媒体最新消息指出，核心设计师Christophe Lemaire与Sarah-Linh Tran或将告别优衣库Uniqlo U系列。据透露，2026秋冬系列很可能成为这对组合与优衣库长达十年合作的终章。
与此同时，有迹象表明，迅销集团已调降了对Lemaire同名品牌的持股比例。据了解，迅销内部曾探讨Uniqlo U系列在设计师离任后独立运作的可能性，但这一设想最终未能获得创始人柳井正的认可。截至本报告发布，Lemaire品牌与迅销集团均未就此置评。不过，消息人士也提示，未来数月情势仍可能出现变数。
此前，优衣库与德国设计师Jil Sander的+J系列未能长期维系，被柳井正视为一大憾事。尽管Uniqlo U系列落幕的消息尚未尘埃落定，但行业内部普遍预测，这段长达十年、被业界观察者誉为“坚定且互惠”的合作关系，已接近尾声。
Uniqlo U系列的前身是2015年启动的联名项目“Uniqlo and Lemaire”。首次尝试便获得巨大成功后，双方于2016年将合作关系升级：Lemaire受邀担任优衣库巴黎研发中心的艺术总监，Uniqlo U系列由此正式创立。
这位拥有近四十年行业经验的法国设计师，其职业生涯始于Christian Lacroix和Yves Saint Laurent的工作室，后于2000至2010年间执掌Lacoste，并于2010年接替Jean Paul Gaultier，出任爱马仕女装艺术总监。正是其在爱马仕的这段经历，使得Uniqlo U系列在中国社交媒体上赢得了“爱马仕平替”的称谓。彼时，Lemaire品牌本身在国内的知名度尚不高。
Lemaire本人曾坚持不在产品标签上署名。他在2016年接受一家国际时尚刊物采访时回忆道：他们曾提议，“不如将您的名字印在标签上，称作Christophe Lemaire for Uniqlo如何？”我的回答是，不。时尚圈的人或许会在意，但优衣库的消费者并不会……这并非一次短暂的设计师联名。我们的构想，是创造一个能够持续存在的、与优衣库主线产品形成互补的新产品层级。
对于优衣库而言，双方合作赋予了其一套更为现代、内敛且富有欧洲审美特质的设计语汇；而对Lemaire品牌而言，优衣库则成为了其发展壮大过程中至关重要的助推平台。
据权威商业媒体报道，Lemaire品牌的销售额在2019年至2024年间实现了十倍增长，成功突破1本站元大关。以美元计，约合1.16亿美元，而疫情前这一数字仅为1000万欧元。
自2023年起，Lemaire品牌加速全球布局，相继在中国大陆、日本和韩国市场开设直营门店。2024年，其中国大陆首店在成都太古里盛大开业。市场反馈显示，该店开业后月均销售额高达约700万元人民币，其标志性的牛角包系列产品更是长期处于供不应求的状态。
消息源进一步披露，Lemaire品牌预计2026年销售额将达到1.6本站元。更值得关注的是，这一高速增长是在品牌核心特质未受丝毫影响的前提下实现的，这使得Lemaire成为私募股权基金及战略投资者眼中极具吸引力的理想标的。正如行业分析师所言，该品牌已跻身“每一个战略买家都在密切关注的独立奢侈品牌候选名单”。
与此同时，Lemaire品牌内部也进行了高层人事调整：此前主导亚洲市场扩张的首席执行官Laetitia Mergui于今年早些时候卸任，由Lemaire资深员工、曾任Jacquemus首席运营官的Clarisse Godbillon接任。所有迹象均表明，该品牌正为未来的独立发展积极进行战略部署。
十年U系列：不只是爆款，更是优衣库的“设计实验室”
在过去的十年间，Uniqlo U系列凭借其标志性的低饱和色调、宽松剪裁与实用廓形，成功构建了优衣库产品体系中辨识度最高的支线之一。虽然优衣库官方从未单独披露Uniqlo U系列的具体销量，但依据其全量、全渠道铺货的规模估算，该系列年度零售额理论上可达到9亿至10亿美元，约占优衣库品牌2025财年总营收的5%。
其影响力更体现在一系列堪比主力产品线的爆款单品上。例如，Uniqlo U系列圆领T恤曾自2024年5月至2025年9月连续17个月蝉联优衣库全球T恤销量榜首，这一斐然成绩甚至被直接写入了优衣库的商品介绍页面。
然而，销量只是最直观的价值体现。Uniqlo U系列更深层次的、难以量化的贡献在于，它实际上充当着优衣库整个产品体系的“设计实验室”角色。凡是在U系列中获得市场验证的设计、剪裁或创新材质，往往会被移植到主力产品线中，这一内部机制被称为“连锁效应”。
这种效应虽然通常不易察觉，但早在2016年，时任优衣库全球品牌体验创意总监的Shu Hung在接受一家权威媒体采访时便亲口证实：随着时间推移，这个系列将日益壮大，并开始对优衣库的其他产品线产生影响。Christophe为Uniqlo U所做的一切，对整个优衣库产生了连锁效应。他真正的目标，是创造全新的基本款。
无缝3D针织技术曾是Uniqlo U系列的标志性工艺。她当时曾特别指出，“这种技术在大众零售领域并不常见，通常成本高昂且产量有限。”然而，如今3D针织连衣裙和毛衣已成为优衣库主线的常规商品，可见其创新成果的深度渗透。
换言之，Uniqlo U系列以一个相对边缘的试验平台，为优衣库这个体量庞大、行动相对迟缓的主品牌，完成了本应由自身承担的审美探索。Uniqlo U系列作为优衣库成本效益最高的创新试验场，若失去它，优衣库不仅将损失可观的销量，更将失去其产品线更新迭代的核心策源地。
新旧迭代：柳井正的B计划
Lemaire潜在离去的时机，恰逢优衣库全球化战略迈入新阶段。在过去的二十年里，中国市场一直是迅销集团海外业务增长最快的引擎。然而，近年来集团的增长重心已显著向欧美市场转移。
自2022年起，大中华区在优衣库海外业务中的比重持续走低，从接近一半的48.1%降至上一财年的34%。与此同时，欧美市场的占比则逐年攀升：2022至2025财年，欧洲在集团总收入中的份额从5.6%几乎翻倍至10.9%，北美地区也从5%增至8%。
相较于在中国和日本市场以密集门店网络实现覆盖，欧美市场则更侧重于旗舰店模式。优衣库在伦敦、巴黎、纽约等全球核心城市持续布局大型门店，旨在通过强化品牌影响力来提升单店运营效率，而非仅仅依赖门店数量的扩张。
高盛分析师河野翔（Sho Kawano）对优衣库此番“西征”的判断是：过去十年，行业大举涌向快时尚，这反而巩固了优衣库的市场地位。优衣库全球创意总裁John Jay的表述更为直接：我们完全站在了时代精神所应处的位置。因为对我们而言，品质是绝不妥协的底线。
在某种程度上，优衣库可以被视为“节制版”的快时尚。零售分析机构数据显示，优衣库约三分之一的商品在货架上的停留时间长达六至九个月，而Zara 66%的商品上架不足三个月即下架。柳井正曾明确指出：我们不追逐潮流。人们常误认为优衣库是快时尚品牌，但事实并非如此。
在这场跨洋扩张的战略冒险中，优衣库比以往任何时候都更需要Lemaire所代表的、一套现代且富有西方美学特质的设计语汇。Uniqlo U系列的潜在终结，无疑将为这一进程带来一定的挑战。
然而，面对Lemaire可能离任的局面，柳井正显然早有布局。2023年推出的Uniqlo:C系列，常被视为Uniqlo U系列的“继任者”——其设计理念、命名方式乃至定价策略，几乎完全沿袭了Uniqlo U所开创的模式。更进一步，C系列的掌舵人、前Givenchy与Chloé创意总监Clare Waight Keller，已于2024年秋季正式晋升为优衣库全线创意总监，全面负责从HEATTECH系列到男装的全部主线产品。优衣库研发负责人胜田幸宏对此的解释是：她是世界上少数能够在创意构想与商品运营之间实现高度平衡的创作者之一。
这种“先设实验、后主全局”的晋升路径，正是Uniqlo U系列留给优衣库最深刻的遗产：首先通过一个“设计实验室”验证其市场吸引力，随后由成功的掌舵者接管整个产品线。如今，当新的主理人已经就位，原先“实验室”的创始人或许便完成了阶段性使命。毕竟，一山难容二虎。
十年前，Christophe Lemaire的愿景并非一个短暂的联名企划，而是“一个能够持续存在的产品层级”。时至今日，这一目标显然已然达成。即便未来Uniqlo U系列最终谢幕，其所确立的创新方法论——即通过独立设计团队进行试验，并将成功经验复制推广至主线产品——已然深植于优衣库的产品开发体系之中。
对Lemaire品牌而言，优衣库无疑是其品牌成长路上的重要推手；而对于优衣库，更艰巨的挑战在于证明：当核心设计师离任之后，其所建立的创新体系是否依然能够独立、高效地持续运作。
