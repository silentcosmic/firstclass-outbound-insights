# 新加坡金融机构与反诈中心高效联动，两月内遏制逾3800万新元诈骗损失

新加坡反诈骗中心与当地五家核心金融机构联手，在为期两个月的专项行动中，成功遏制了逾600起诈骗活动，为公众挽回了超过3800万新元的巨额潜在损失。
此次卓有成效的协同作战由隶属于新加坡警察部队网络司令部的反诈骗中心牵头，并于2026年5月1日至6月30日期间，携手星展银行（DBS）、大华银行（UOB）、华侨银行（OCBC）、渣打银行（Standard Chartered Bank）以及GXS银行共同展开。
这项联合部署充分利用了机器人流程自动化（RPA）技术，显著提升了诈骗识别与干预的速度和效率。其覆盖范围广泛，包括冒充政府官员诈骗、投资诈骗、招聘诈骗、电子商务诈骗以及租赁诈骗等多种常见类型。
在此次行动中，反诈骗中心警员与银行工作人员紧密配合，累计向逾3300名被识别为潜在受害者的客户发送了超过3800条手机短信警示，有效开展了预警工作。
据警方披露，此次跨机构协作的核心优势在于实现了与银行之间更为迅捷的信息共享。这使得金融机构能够及时标记潜在受害者，并在资金被转移至诈骗分子账户之前进行有效干预，从而最大程度地减少损失。
同时，公众也被再次提示，应主动采取措施提升自我保护能力，包括启用各类安全防护功能、警惕诈骗警告信号，并及时举报可疑遭遇。
警方特别建议用户启用多重安全保障，例如安装ScamShield应用程序、开通双因素认证（2FA），并为网上银行和PayNow支付设置交易限额，以增强账户安全性。
此外，警方敦促公众对任何索要个人信息或要求转账的要求保持高度警惕，尤其是在面对那些听起来好得令人难以置信的线上优惠时，务必进行仔细核实。
如遭遇诈骗，市民可直接向银行、通过ScamShield应用程序或向警方举报。
公众亦可访问ScamShield官方网站，或拨打ScamShield求助热线1799获取更多相关信息。
