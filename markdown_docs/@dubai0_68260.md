# 中柬联手打击电诈：誓言彻底铲除跨国犯罪毒瘤

🚨 中柬联手打击电诈：誓言彻底铲除跨国犯罪毒瘤
近日，国家主席习近平在北京会见柬埔寨人民党主席、参议院主席洪森。双方就深化中柬命运共同体建设、加强经贸与安全合作等议题深入交换意见，并一致同意将持续加强执法协作，联合打击跨国电信网络诈骗犯罪。
习近平主席指出，中方高度赞赏柬埔寨政府打击电诈的坚定决心，愿同柬方持续强化执法合作，保持高压打击态势，彻底铲除电信网络诈骗等跨国犯罪“毒瘤”，共同维护两国人民生命财产安全和地区稳定。
洪森主席回应表示，清剿电信网络诈骗犯罪是柬埔寨当前的重要工作，柬方愿进一步深化与中方在执法、安全等领域的务实合作，持续严厉打击包括人口贩运在内的跨国违法犯罪活动，以维护两国共同利益。
