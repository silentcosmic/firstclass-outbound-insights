# 马来西亚中国留学生遇害：同居男子被捕

🚨 马来西亚中国留学生遇害：同居男子被捕
马来西亚雪兰莪州八打灵再也双威镇一栋高层公寓近日发生一起震惊命案。一名20岁的中国籍女留学生被发现陈尸公寓单位内，身上带有多处伤痕。警方接报后迅速行动，当场拘捕了与其同住的一名27岁中国籍男子，并已援引谋杀罪名展开调查。
据警方初步通报，案件发生于6月23日上午11时许。初步调查显示，死者身上有明显外伤，现场情况存在可疑之处，警方随即将其列为刑事案件处理。被捕男子与死者据称相识多年，曾是恋人关系，并长期共同居住于涉案公寓，两人均为在马求学的中国籍学生。
警方透露，案件可能涉及感情纠纷及日常经济开支矛盾，但目前仍属调查方向之一，尚未得出最终结论。鉴证人员已在现场提取多项证物，包括疑似作案工具送检，以协助还原案发经过。死者遗体已送往医院进行法医解剖，具体死因及更多案发细节有待法医报告完成。
6月24日，死者家属已从中国赶赴马来西亚处理善后事宜，对案件深感震惊，并表示此前不认识被捕男子。马来西亚警方目前已援引《刑事法典》第302条（谋杀罪）立案调查。随着调查深入，更多细节预计将陆续披露。
