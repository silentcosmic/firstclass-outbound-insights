# 出海·头等舱独家 | 前西门子、罗罗电动飞行团队开创航空电驱系统，连续完成数千万元融资

据《出海·头等舱》独家获悉，开普动能航空技术有限责任公司（下称“开普动能”）近日已连续完成种子轮与天使轮融资，累计融资金额达数千万元人民币。本轮融资由零以资本领投，新鼎资本跟投，唯快资本继续担任长期财务顾问。
当前，全球eVTOL（电动垂直起降飞行器）产业正逐步迈入工程化实施阶段，航空电驱系统在其中扮演的角色日益凸显，并被视为产业链上技术壁垒最高的关键环节之一。作为电动飞行器“三电系统”不可或缺的核心组成，航空电驱系统在整机成本中占据高达30%-40%的份额。其设计不仅要满足高功率密度、高扭矩密度及卓越效率的综合需求，更需符合严苛的适航标准，包括多通道冗余设计、单点失效保护、复杂热管理能力等在内的极致安全性、可靠性与可测试维护性。此外，电机控制器的软硬件还须通过DO-178C和DO-254等一系列严格的适航认证，这无疑对研发团队的航空工程经验构筑了极高的门槛。
审视当下国内eVTOL电驱领域，尽管已有诸多企业积极布局，但中高端产品的国产化率仍显不足。在轻量化设计、系统可靠性以及适航认证经验等关键维度上，与国际领先水平之间尚存一定差距。整个行业目前仍处于研发与认证并行的阶段，市场竞争格局远未尘埃落定。
开普动能得以切入这一高门槛赛道，其核心竞争优势在于创始团队深厚的航空电驱技术积累与完整背景。公司的核心成员阵容强大，汇聚了前西门子、罗尔斯-罗伊斯（简称罗罗）电动航空部门的总监、首席工程师等资深技术专家，以及来自中国商飞和国内领先eVTOL主机厂的资深低空技术与适航负责人。团队成员平均拥有超过十年的航空工业实践经验。
自2016至2017年间，该团队便投身电动飞行项目，专注于航空电驱系统设计。他们曾与空客等国际领先的主机厂进行深度合作，累计参与了逾20个固定翼及多旋翼电动飞行平台的开发工作，为多项前瞻性项目提供了纯电及混合动力系统设计方案。其电驱样机的功率覆盖范围广阔，从30kW至2MW不等，且大部分项目已成功进入试飞阶段，积累了逾1500架次的电动飞行宝贵经验，并协同创造了多项电动飞行领域的全球纪录。
民用载人航空电驱系统根据应用场景不同，大致可划分为两大主要方向：一是服务于区域空中交通（RAM）的电动固定翼飞机，二是面向城市空中交通（UAM）的eVTOL。值得关注的是，这两大方向对电驱系统的设计理念和技术指标提出了截然不同的严苛要求。
开普动能展现出同时深耕这两条技术路线的研发实力，当前阶段的战略重点聚焦于UAM方向。公司创始人张嶔崟先生曾担任罗尔斯-罗伊斯电动固定翼部门的总监，在其任职期间，主导了Tecnam P-Volt和MDA1 EViator九座电动飞机等重要项目。而其联合创始人则分别曾任罗尔斯-罗伊斯电动UAM与RAM部门的首席工程师，专项负责不同应用场景下的电驱系统设计工作。
在产品研发进度方面，开普动能计划于今年内正式推出其第一代技术验证样机（1.0版本），旨在全面检验并优化设计方案。基于详尽的测试数据，公司将迅速迭代升级至具备试飞能力的1.5代样机，预计该版本将于明年第一季度发布，并有望在明年第三季度携手主机厂共同进行首次飞行测试。当前，所有样机开发均紧密围绕eVTOL（UAM）的应用场景需求。
商业化拓展方面，开普动能已深入走访国内逾十家eVTOL主机厂，通过与潜在合作伙伴的紧密沟通，全面掌握了行业前沿需求，并获得了积极的市场反馈。目前，公司已与部分意向客户初步达成合作共识。同时，在电动固定翼领域，开普动能亦成功与多家海外厂商签署了战略合作协议，进一步拓展了国际市场布局。
张嶔崟先生指出：“我们的核心优势主要体现在三大层面。首先，便是团队深厚的航空基因与长期沉淀。我们的核心成员早在2016年至2017年间，即已加入西门子电动航空团队，并始终专注于航空电驱这一特定领域，而非从其他行业‘跨界’而来。目前，团队在各个专业方向上均配备了资深专家，并曾与空客等全球顶尖主机厂进行深度协作，累计参与了超过20个电动飞行平台的驱动系统开发工作，积累了行业内极为丰富的项目实战经验。”
“我们之所以强调自身并非‘跨界玩家’，而是沿着航空工业体系逐步成长起来的专业团队，根源在于航空产品的研发与普通工业产品存在显著不同。前者对可靠性、安全性以及冗余设计等各方面均有着更为严苛的要求，航空电机更需要通过严格的适航认证。所有这些，都离不开长期不懈的经验积累。”
“其次，是我们的技术创新实力。若仅限于开发100至150千瓦级的常规电驱系统，全球范围内的供应商可谓众多；然而，真正能满足航空领域严苛应用需求的企业则寥寥无几。航空电驱面临的最大挑战，是如何在轻量化、高性能与高可靠性之间寻得精妙平衡，这在业内也被形象地称之为‘不可能三角’。”
“此外，我们长期以来围绕这一核心难题持续开展研发工作，并与各大主机厂保持着深度协作关系，因此对客户的实际需求有着极为透彻的理解。无论是在轻量化设计、可靠性保障，还是在复杂热管理等关键技术环节，我们都已形成了自身独特的技术沉淀与创新领先优势。”
张嶔崟先生继续阐述：“我们在欧洲航空电驱领域深耕逾十年，对当地的供应链体系有着极为深刻的认知。与此同时，自去年起，我们已着手系统性地梳理国内供应链生态，并专项组建了负责供应链建设与优化的专业团队。”
“经过严谨的市场调研，我们欣喜地发现，中国在诸多航空电驱关键材料领域已展现出显著优势。例如永磁体、软磁体、高性能电工钢等，其中部分领域甚至已跻身全球领先行列。鉴于此，我们在这些核心材料的选择上，更倾向于采用优质的国内供应商，以期获得更为稳固的供应保障和成本竞争力。目前，我们已在绕组材料、磁钢、永磁体等多个生产环节完成了国产供应商的精细化筛选，并正与优秀的本土合作伙伴开展深度协作。诚然，在航空轴承及部分高端功率模块等领域，目前仍存在一定程度的进口依赖，我们亦在积极评估与推进国产替代方案。然而，对于那些短期内难以实现替代，且直接关乎产品性能与可靠性的核心部件，我们仍将审慎选用成熟可靠的海外供应商。”
“从长远战略视角审视，我们致力于达成‘欧洲先进设计能力与中国优质供应链’的深度融合。这不仅意味着充分发挥团队在技术创新和系统设计方面的独特优势，更要有效利用中国制造业的强大产业基础，从而打造出兼顾卓越性能、合理成本与极致可靠性的航空电驱产品。”
“我们的团队本身便拥有强大的国际化基因与背景，因此，我们并非遵循传统意义上‘先立足国内，再逐步出海’的发展路径，而是在公司成立之初，便同步启动了国内外市场的战略布局。当前，我们正积极与中国本土主机厂进行深度对接，同时亦与海外主机厂展开紧密合作。在语言沟通、跨文化理解及项目协同效率方面，我们具备得天独厚的优势。”
“事实上，我们已与部分海外客户取得了实质性的合作进展。举例而言，目前已进入签约意向阶段的电动固定翼项目，便是由一家海外知名厂商发起。这不仅有力验证了中德航空专业团队协同研发低空‘心脏’——航空电驱系统——的实践可行性，更凸显了我们国际化战略布局的深远价值与市场核心竞争力。”
零以创投创始人郑连发先生对此表示：“我们此次对开普动能进行战略投资，主要基于三大核心考量。其一，航空电气化已是大势所趋，低空经济正加速崛起。航空电驱系统作为这一变革的核心环节，不仅是规模达千亿的蓝海市场，更兼具高成长性与稀缺性，其在中高端产品的国产化率尚待提升，亟需实现自主技术突破。其二，开普动能的核心团队由前罗尔斯-罗伊斯电动部门的高级管理人员领衔，创始人张嶔崟博士及其核心成员均具备国际顶尖的研发与工程实践经验，集技术实力、行业资源与卓越执行力于一身。其三，公司明确聚焦于高功率、高扭矩重量比的适航电驱系统研发，在技术创新上已成功构建起差异化竞争壁垒，能够快速响应并适配主机厂的定制化需求，商业化路径清晰可见。零以创投将秉持‘资本+产业资源’的全面赋能理念，全力支持开普动能的快速产业化进程，助力其抢占全球电动航空的先发优势，共同推动中国低空经济的高质量发展，最终实现资本与产业的互利共赢。”
新鼎资本董事总经理印石澔先生亦指出：“低空经济已然成为我国培育发展新质生产力的关键战略支点，而航空电驱系统作为电动飞行器的‘动力核心’，其技术壁垒极为高企。这片万亿级蓝海市场中的核心环节，兼具极强的稀缺性与广阔的成长空间。开普动能的核心团队拥有来自西门子、罗尔斯-罗伊斯等全球顶尖航空发动机企业的完整研发及适航认证经验，其技术实力与工程化能力均位居行业第一梯队。我们对开普动能所构建的‘欧洲顶尖技术标准与中国极致供应链及市场需求’这一独特优势组合充满信心。此次投资，是新鼎资本在低空经济核心硬科技供应链领域的重要战略部署，我们将倾力赋能团队，助力公司深化与国内主机厂及供应链体系的协同，加速适航取证与商业化项目的落地实施，最终推动我国低空经济核心部件实现自主可控。”
