# 柬埔寨木牌突查：涉诈窝点被捣毁，中柬嫌犯落网

🚨 柬埔寨木牌突查：涉诈窝点被捣毁，中柬嫌犯落网
近日，柬埔寨柴桢省与巴域市联合执法部队在省检察官协调下，对巴域市巴域洛分区的“春兴城”小区展开突击检查，成功捣毁一处涉嫌网络诈骗的窝点。
行动中，执法人员对该小区内的6栋排屋进行了排查，发现17名持有合法护照和签证的外籍人员，其中包括15名中国人和2名越南人。随后，在锁定的DL06号涉嫌电诈排屋中，逮捕了3名嫌疑人，分别为1名中国男子和2名柬埔寨女子。现场缴获4台笔记本电脑、19部手机、2辆摩托车及1本护照等涉案物品。
目前，涉案窝点已被查封，所有嫌疑人已移送省警察局，等待进一步的法律审理。
