# 阿联酋法院：非法闯入私人住宅者被判赔2万迪拉姆

🚨 阿联酋法院：非法闯入私人住宅者被判赔2万迪拉姆
阿联酋艾因地方法院近日审理了一宗非法闯入案件。一名男子因未经许可擅自进入一女性家中，被法院判决需向受害人支付2万迪拉姆（约合人民币3.8万余元）的经济赔偿。
