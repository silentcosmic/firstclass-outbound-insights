# 波贝华福酒店遭突袭：警方联合行动

🚨 波贝华福酒店遭突袭：警方联合行动
柬埔寨波贝市华福大酒店近日成为警方联合行动的目标。据市场消息，国家警察与班迭棉吉省联合部队于2026年6月25日，在该市奥克罗夫桑卡特区克巴尔斯宾1村的一栋建筑物内展开了突击行动。
此次行动的具体细节和目的尚未完全披露，但此类警方介入通常与当地的治安整顿或特定案件调查有关，值得在柬埔寨运营的企业和个人持续关注。
