# 芭提雅少女遇害藏尸案：澳籍嫌犯机场落网，将追加谋杀指控

🚨 芭提雅少女遇害藏尸案：澳籍嫌犯机场落网，将追加谋杀指控
近日，泰国警方侦破一起芭提雅少女遇害藏尸案。一名17岁泰国少女被发现遇害并藏尸于行李箱内，警方迅速行动，在素万那普国际机场成功抓获一名涉嫌作案的46岁澳大利亚籍男子，当时该男子正准备搭机离境。
警方调查显示，受害少女于6月25日凌晨最后一次出现在监控画面中，与该澳大利亚男子一同进入芭提雅某公寓。约15小时后，嫌疑人独自拖着一个大型黑色行李箱离开公寓，并骑摩托车行驶约4.2公里，将其遗弃在铁路附近一处监控盲区后返回。
随后，警方在公寓内发现疑似打斗痕迹，并循线在铁路旁找到了被遗弃的行李箱。打开后，箱内藏有一具赤裸女性遗体，初步勘验显示死者面部有严重伤痕，口鼻有血迹。法医鉴定将进一步确认具体死因。同时，警方在嫌疑人颈部及手臂发现明显抓痕，推测可能为与受害人冲突时所致。
目前，该澳大利亚籍男子已被依法逮捕，但他否认与案件有关。警方表示，已获取逮捕令，正结合现场物证、监控录像及法医鉴定结果完善证据链，后续将追加相关刑事指控。案件仍在进一步侦办中。
