# 全球商业瞭望 | 大众汽车重塑德国生产布局；半导体代工价格策略调整；国家超算核心节点投入运营

全球半导体产业动态：LG Innotek越南设厂与三星代工价格调整
韩国电子元件巨头LG集团子公司LG Innotek已获越南海防经济区管理委员会批准，将投入10亿美元在海防自由贸易区内的DEEP C海防2工业园新建一座半导体基板制造基地。该项目规划占地约33公顷，预计将于今年第三季度启动建设，并计划在明年第四季度完工投产。此举标志着LG Innotek在全球半导体供应链布局上的重要战略落子。
与此同时，全球晶圆代工市场正经历价格策略调整。三星电子已将其晶圆代工服务对新客户的供货价格上调约15%，主要集中在市场需求旺盛的先进制程节点，如4纳米和5纳米工艺，以及部分车用8纳米制程。此前，台积电也已知会其主要客户，计划将供应价格提升5%至10%。市场分析显示，相较于台积电的全面性提价策略，三星的调整更侧重于对需求集中的特定节点进行价格优化。
智能出行与汽车制造：Waymo拓展服务与大众工厂优化
Waymo近日宣布，将在未来数周内逐步在美国四个新城市推出其无人驾驶出租车服务，进一步强化其在本土市场对竞争对手的领先地位。新增服务城市包括圣迭戈、拉斯维加斯、佛罗里达州坦帕以及丹佛。Waymo表示，新服务将首先面向Alphabet内部员工开放，随后逐步扩展至普通公众。此项扩张计划是公司去年首次对外披露的战略部署。
在传统汽车制造领域，大众汽车执行董事会正在审议一项计划，拟在未来五年内逐步停止其位于德国茨维考和埃姆登两家工厂的生产。此外，汉诺威的商用车工厂计划于2032年关闭，内卡苏尔姆的奥迪工厂则计划于2034年关闭。这些调整将影响德国境内约4万名员工，体现了大众汽车在应对产业转型和提升生产效率方面的深层考量。
奢侈品美妆格局重塑：古驰美妆授权易主欧莱雅
开云集团（Kering）于7月7日宣布提前回购科蒂集团（Coty）持有的古驰（Gucci）美妆授权协议。紧随其后，依据开云集团与欧莱雅集团在2025年10月19日达成的战略联盟条款，欧莱雅正式签署了一项为期50年的独家美妆授权协议。该协议将赋予欧莱雅负责古驰品牌香氛和美妆产品的研发、生产与分销业务的全面权限。此项新授权协议预计在获得监管批准后，将于2027年7月1日起正式生效。届时，欧莱雅将全面接管古驰美妆的全球业务发展与运营管理，此举将深刻影响全球奢侈品美妆市场的竞争格局。
科技巨头战略调整：Xbox重组与OpenAI语音模型升级
微软公司旗下Xbox业务正在进行新一轮大规模重组。内部消息指出，Xbox所属的黑曜石工作室已取消多个在研项目，其中包括原计划开发的《宣誓》续作。该工作室的重心将转向开发备受期待的《辐射》系列新作。此外，此次重组还涉及黑曜石工作室约四分之一的员工裁撤。
在人工智能领域，OpenAI正式发布了新一代语音模型GPT-Live-1和GPT-Live-1 mini，全面升级了ChatGPT的语音交互能力。新模型采用“全双工”架构，能够实现同步听取和语音输出，用户可以在AI回应过程中随时插话打断，使得人机对话体验更接近真实的自然交流。
前沿科技投融资：量子计算与AI领域获资本青睐
“逻辑比特科技”近期正式公布完成了数亿元人民币的A轮融资。本轮融资由康君资本和混沌投资联合领投，国开科创、元璟资本、上海未来产业基金、IDG资本、中芯聚源、芯联资本、弘晖基金、TCL创投及厚雪资本等机构跟投。原有股东经纬创投、东方嘉富、达晨财智、华控基金、深创投、浙大校友基金藕舫天使及华夏恒天在本轮中持续增资。本轮募集资金将主要用于研发和建设新一代高性能超导量子芯片及自动化产线，加速部署国际领先的量子计算云平台，并持续吸纳高水平人才。
北京月之暗面科技有限公司近期也发生了工商变更，新增社保基金长三角科技创新股权投资基金（上海）合伙企业（有限合伙）和深圳和谐成长三期科技发展股权投资基金合伙企业（有限合伙）等战略股东，同时公司注册资本由100万人民币增至约152.4万人民币，显示出其在AI领域的强劲发展势头和资本市场的高度认可。
全球投资视野：淡马锡的加密货币立场与战略布局
淡马锡全球投资总裁纳吉·哈米耶明确表示，受加密货币交易所FTX巨额亏损事件的影响，该公司正在积极消化此前投资造成的损失，目前加密货币投资“不在考虑范围之内”。同时，他也分享了淡马锡在人工智能、欧洲市场、中东地区及防务赛道上的战略布局思路，彰显其在全球复杂经济环境下的审慎与多元化投资策略。
国家战略与产业发展：碳达峰方案与新能源汽车增长
国务院近日印发了《“十五五”碳达峰行动方案》，逐步明确了重点用能行业可再生能源消费的最低比重目标。该方案提出，到2030年，抽水蓄能装机容量将达到1.6亿千瓦左右，新型储能装机容量力争达到3亿千瓦，全国虚拟电厂最大调节能力超过5000万千瓦，电力需求响应能力达到最大用电负荷的5%以上，为中国绿色转型设定了明确而宏大的目标。
中国新能源汽车市场继续保持强劲增长态势。2026年上半年，我国新能源汽车产销分别完成743.8万辆和744.6万辆，同比均实现显著增长，其中纯电动汽车销量占比已升至67%左右。6月份，新能源汽车产销分别完成159.8万辆和164.3万辆，同比增长均超过两成，新能源汽车新车销量占汽车新车总销量的比例进一步提升至58.5%。上半年，我国汽车累计出口509.6万辆，其中新能源汽车出口235.5万辆。6月份单月汽车出口量首次突破100万辆，同比增长75.1%，其中新能源汽车出口52.3万辆，同比增长高达1.6倍，充分体现了中国汽车产业在国际市场上的竞争力。
国家算力基础设施建设：超算互联网核心节点上线
在7月9日于郑州举行的2026河南省人工智能大会上，国家超算互联网核心节点正式上线运行。该项目旨在构建一个覆盖全国的计算资源统筹调度体系，承担运营管理、资源调度等核心职能，并整合供需对接、产业孵化等综合服务功能。据了解，该节点可对外提供超过10万卡的国产AI算力，是国家超算互联网平台上线以来接入的全国最大规模的单体国产AI算力资源池，对推动我国人工智能产业发展具有里程碑意义。
