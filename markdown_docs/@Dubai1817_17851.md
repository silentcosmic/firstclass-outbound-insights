# 俄籍男子假冒飞行员诈骗泰女超400万泰铢被捕

🚨 俄籍男子假冒飞行员诈骗泰女超400万泰铢被捕
近日，泰国警方成功侦破一起“浪漫诈骗”案，逮捕了一名31岁的俄罗斯籍男子。该嫌疑人涉嫌伪装成飞行员，通过社交平台骗取多名泰国女性信任，诈骗金额累计超过400万泰铢。
据警方调查，嫌疑人长期在社交媒体上伪造飞行员身份，以建立恋爱关系为名接近受害人。在取得信任后，他多次以各种理由诱导受害人转账，骗取巨额资金。
北榄府法院已对该男子签发逮捕令，指控其涉嫌共同诈骗公众及向计算机系统输入虚假信息等罪名。警方根据线索，在春武里府邦拉蒙县的一处公寓内将其抓获，抓捕过程未遇反抗。
目前，嫌疑人已被移交至北榄府警方，案件正在进一步调查中，并将依法追究其法律责任。
