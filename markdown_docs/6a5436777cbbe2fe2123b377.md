# 淡马锡重磅出击AI赛道：2031年投资组合中AI配置目标高达15%

新加坡主权财富基金淡马锡（Temasek）计划在2031年3月之前，将其直接人工智能（AI）投资在其当前投资组合中的占比，从约6%提升至10%至15%。
淡马锡首席执行官Dilhan Pillay在周三发布的《淡马锡2026年回顾》声明中指出：“我们投资组合中剩余的85%必须专注于AI应用，以保持市场竞争力。”
截至2026年3月31日，淡马锡的净投资组合价值已达到5180亿新元（约合4020亿美元），在过去十年中实现翻番。基于当前估值，这一目标意味着其AI净资产将从目前的约310亿新元，跃升至520亿至780亿新元。
淡马锡公开披露的AI投资已覆盖了完整的技术栈。在能源与数据中心领域，其通过AI基础设施合作伙伴关系以及Aligned、Neoen和PCS Power进行布局。半导体持有包括AEM、d-Matrix、英伟达（NVIDIA）和台积电（TSMC）。而在云服务方面，则涵盖了阿里云、亚马逊云服务（Amazon Web Services）、谷歌云（Google Cloud）和腾讯云。
在基础模型开发者方面，淡马锡的投资组合包括Anthropic、OpenAI、Mistral和Advanced Machine Intelligence。在AI应用和软件基础设施领域，投资了cusp.ai、Databricks、Momenta和Physics X。支撑这些投资的是贯穿整个产业链的垂直整合型超大型企业，如Alphabet、亚马逊（Amazon）、微软（Microsoft）、阿里巴巴、腾讯、字节跳动和SpaceX。此外，淡马锡还设立了自身机构以构建能力，包括Temus、Aicadium和Resaro。
四大战略支柱推动AI转型
淡马锡在推动AI转型方面确立了四大核心支柱。
第一支柱，“AI赋能自身”（AI-Enabling Ourselves），侧重于内部变革。投资团队利用定制化的AI工具进行尽职调查和投资组合监控，并通过一项覆盖全公司的“数字素养”计划提升员工的AI认知水平。
第二支柱，“AI强化投资组合”（AI-Proofing Our Portfolio），核心是帮助被投企业适应AI浪潮。淡马锡为此成立了专门机构，包括拥有AI工厂的Temus以及AI工程公司Aicadium，旨在合作提升被投企业的质量控制和运营效率。
第三支柱，“扩大AI投资”（Scaling Our AI Exposure），承载着淡马锡的投资雄心，涵盖五大重点领域：能源和数据中心、半导体、云服务提供商、基础模型，以及AI应用和软件基础设施。其投资范围从Alphabet、亚马逊和微软等超大型企业，到英伟达和台积电等芯片制造商，再到Anthropic、OpenAI和Mistral等基础模型开发者。
第四支柱，“支持AI普及”（Supporting AI Diffusion），旨在淡马锡生态系统和公共部门中推广AI应用，其核心是2023年成立的AI保障公司Resaro。
Pillay强调，表面上的投资配置仅是故事的一部分。他指出，AI最持久的价值并非仅仅来自模型本身，而是源于该技术如何被规模化地采纳和应用。作为一项战略目标，投资组合中剩余约85%至90%的资产必须成为AI的采纳者，才能维持竞争力。他表示，投资AI公司固然重要，但真正的价值捕获在于传统企业将AI深度融入其运营之中。
Pillay反复提及此次转型中的“人”的维度，强调AI是取代任务而非取代工作，因此重点必须放在员工的再培训和技能提升上。此举意义重大，因为淡马锡在全球范围内拥有约1000名员工，其投资组合公司在新加坡雇佣超过16万人，全球范围内则有约40万人。这使得淡马锡有责任确保此次转型以人为本。
