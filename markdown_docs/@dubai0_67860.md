# 马来西亚新山移民突查：大批外籍人士被捕待遣返

🚨 马来西亚新山移民突查：大批外籍人士被捕待遣返
昨晚，马来西亚新山地区开展了一场针对外籍人员的专项清查行动。行动中，执法部门现场抓获了大量签证失效、无证滞留以及非法务工的外籍人士。
所有被查获人员已统一转运至苏丹依斯干达CIQ出入境大楼进行临时扣留，并在此逐一核对身份信息及登记笔录。完成初步手续后，这些人员将被移送至专门的移民扣留所集中关押，等候后续的统一遣返安排。
