# ️ 迪拜核心地铁站车道封闭至2026年底，配合扩建

⚠️ 迪拜核心地铁站车道封闭至2026年底，配合扩建
迪拜道路与交通管理局（RTA）近日宣布，位于哈利法塔/迪拜购物中心地铁站的公交和出租车专用车道将暂时关闭，直至2026年底。此次封闭旨在配合该站的大规模扩建工程。
该站是迪拜地铁最为繁忙的枢纽之一，每日服务数万名通勤者及前往迪拜市中心、迪拜购物中心和哈利法塔的游客。RTA已制定周密的交通管理方案，确保公共交通用户能安全便捷地进出车站，并呼吁驾驶者留意替代停车区域指示，施工期间该路段仅限工程车辆通行。
此次扩建项目旨在将该站的客运能力提升65%，每小时客流量将从7250人次增至1.232万人次，日均处理能力可达22万人次。此举将显著缓解新年夜及重大节假日等高峰时段的交通压力，提升城市公共交通效率。
