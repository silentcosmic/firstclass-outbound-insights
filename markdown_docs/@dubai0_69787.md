# 巴林间谍案：三名被告为伊朗提供攻击坐标被判终身

🚨 巴林间谍案：三名被告为伊朗提供攻击坐标被判终身
巴林法院于当地时间7月14日作出判决，两起独立案件中的三名被告因间谍罪被判处终身监禁。他们被指控为伊朗伊斯兰革命卫队效力，协助其实施针对巴林王国的敌对行动。法院同时下令没收所有涉案物品。
据指控，其中一名主犯在境外遥控境内代理人，搜集巴林关键设施情报，意图在伊朗发动袭击时将其作为打击目标。另一起案件的被告则被查出与隶属于伊朗革命卫队的网络账户保持联系，不仅传送了伊朗袭击后的现场视频，更提供了巴林境内多处可能遭攻击的关键地点坐标，直接威胁到当地居民的生命安全和国家稳定。
巴林检察机关在接到报告后迅速展开调查，通过讯问被告、听取证人陈述及技术设备检查，证实被告向革命卫队提供的数据和信息，构成了伊朗对巴林关键设施实施敌对攻击的重要依据。此判决正值本月以来伊朗对巴林持续进行一系列袭击的敏感时期。自7月8日首次大规模袭击以来，巴林防空系统已多次成功拦截进入领空的导弹和无人机，巴林军方明确指责德黑兰以平民为攻击目标。
