# 缅甸监狱枪击：1名中国籍涉诈人员身亡，5人伤

🚨 缅甸监狱枪击：1名中国籍涉诈人员身亡，5人伤
7月14日，缅甸掸邦东部景栋监狱发生枪击事件，导致1名中国籍人员死亡、5人受伤。据悉，此次事件与狱中关押的中国籍涉诈人员发生骚乱有关。
据了解，涉事人员多为此前在大其力被捕的电信网络诈骗嫌疑人。骚乱中，监狱警戒人员开枪控制局势，造成1死5伤，死者被证实为一名中国籍涉诈人员。消息人士指出，景栋监狱主要关押军政府控制区内被捕的非法入境中国籍人员，他们通常在完成司法程序后等待移交中方。
事件发生后，涉事警戒人员已被更换，目前相关情况仍在进一步调查中。
