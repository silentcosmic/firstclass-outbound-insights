# 老挝跨境执法：174名涉诈嫌犯移交中国

🚨 老挝跨境执法：174名涉诈嫌犯移交中国
近日，老挝警方与中国警方开展跨境执法协作，并于2026年7月4日完成了一批涉案中国籍人员的移交工作。此次行动依据老挝《出入境及外国人管理法》及两国公安部门的合作协议，旨在共同打击跨境违法犯罪。
据悉，老挝公安部移民管理部门联合中老警务合作中心，在老挝相关部门配合下，在琅南塔省波顿国际口岸向中国警方正式移交了174名中国籍嫌疑人，其中包括4名女性。这些人员已被中国警方接收，并将依法开展后续调查取证及处理。初步调查显示，他们涉嫌违反老挝出入境及外国人管理法律法规，涉及非法出入境、非法务工、非法经营、非法滞留等行为，部分人员还涉嫌参与电信网络诈骗等跨境犯罪活动。
此次联合行动由老挝多个执法部门共同参与，并向中方同步移交了一批涉案物证，包括电脑、手机等电子设备，为后续案件侦办和证据固定提供了有力支持。老挝警方表示，将继续加强与中国等国家的警务合作，持续打击跨境电信网络诈骗、非法务工及非法出入境等违法犯罪活动，维护地区社会治安秩序。
