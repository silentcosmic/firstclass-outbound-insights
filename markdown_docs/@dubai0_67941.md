# 中柬联合执法：319名中国籍涉诈人员遭遣返

🚨 中柬联合执法：319名中国籍涉诈人员遭遣返
6月20日，柬埔寨移民总局与中国驻柬埔寨大使馆联手，在柬埔寨副首相兼内政部长阿皮森迪班迪的指导下，成功遣返了319名涉嫌参与网络诈骗、非法居留及非法务工的中国籍公民。
柬埔寨移民总局局长苏克·瓦萨纳已要求相关部门依法推进后续程序，并与中国驻柬使馆持续保持密切协调，以确保此次联合执法行动的顺利完成。
