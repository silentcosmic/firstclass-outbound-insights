# 宜昌警方拦截10万现金，揭露中东石油投资骗局

🚨 宜昌警方拦截10万现金，揭露中东石油投资骗局
湖北省宜昌市夷陵区警方近日成功破获一起虚假投资诈骗案。警方在快速路上及时拦截，追回一名受害女子准备交付给诈骗分子的10万元现金，有效避免了群众的重大财产损失。该案件揭示了以“投资中东石油”为诱饵的典型骗局模式。
