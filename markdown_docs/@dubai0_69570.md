# ️ 韩国调整柬埔寨旅行警示：三地仍建议撤离

⚠️ 韩国调整柬埔寨旅行警示：三地仍建议撤离
韩国外交部于7月9日宣布，已调整对柬埔寨部分地区的旅行警告级别。此前被列为第4级旅行禁区（即禁止前往）的波哥山、波贝和巴域（木牌）三地，现已下调至第3级“建议撤离”警告。
这意味着，虽然对这三个地区的旅行禁令已解除，但韩国政府仍保持高度戒备，强烈建议国民避免前往并尽快撤离。根据韩国法律，此前未经批准进入第4级禁区的公民将面临刑事及行政处罚，此次调整后，相关法律限制有所放宽，但官方警示依然显著。
