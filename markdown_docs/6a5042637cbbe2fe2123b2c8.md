# 新加坡推出资本市场研究新动议，旨在巩固全球金融中心地位

新加坡管理大学（SMU）于日前正式启动了“新加坡资本市场倡议”（Singapore Capital Markets Initiative, SCMI），旨在将新加坡在全球快速演变的资本市场格局中推向领先地位。
该大学在一份声明中指出，这项全新的研究倡议隶属于SMU杨邦孝法学院（YPHSL）亚洲商法中心（CCLA），将致力于推动法律与金融交叉领域的研究、政策对话及人才培养，重点关注新加坡乃至整个亚洲区域。
SCMI的研究议程涵盖广泛，包括股权与债券资本市场、风险投资与私募股权、私募信贷与另类金融、公司治理与投资者保护、金融监管与市场基础设施，以及新兴技术在塑造未来资本市场中的作用。
SMU观察到，当前全球资本市场正处于一个关键的拐点。在许多经济体中，首次公开募股（IPO）活动显著下滑，与此同时，私募股权、风险投资和私募信贷正在重塑企业融资格局。
该机构进一步指出，亚洲正位于这一转变的中心。该地区已成为全球上市公司的主导力量，并逐渐发展为全球风险投资活动的主要引擎，同时也是资本市场发展与改革的焦点。SMU杨邦孝法学院院长李佩婉教授表示：“SCMI将作为一个跨学科研究和政策参与的平台，致力于将SMU定位为资本市场研究领域的国际领导者，同时确立新加坡在法律与金融领域作为资本市场研究、思想领导和政策影响力的枢纽地位。”她补充道：“这项倡议建立在法学院现有CCLA平台的成功基础之上，其中包括新加坡全球重组倡议（SGRI）和新加坡税务学院研究倡议（STARI）。这些平台已经证明了专注型研究倡议的价值，它们将卓越的学术成果与政策制定者、行业领袖以及国际组织的有效互动相结合。”
这项新的倡议将由Aurelio Gurrea-Martínez教授牵头，并由法学院的Dan Puchniak教授和Nydia Remolina助理教授共同领导。SCMI负责人Aurelio Gurrea-Martínez教授指出：“资本市场在资助创新和经济增长方面扮演着至关重要的角色。随着资本市场和企业融资模式的持续演变，对法律与金融交叉领域进行严谨研究的需求达到了前所未有的高度，尤其是在亚洲。如今，全球超过55%的上市公司以及半数以上的全球风险投资活动都集中在亚洲地区。”
