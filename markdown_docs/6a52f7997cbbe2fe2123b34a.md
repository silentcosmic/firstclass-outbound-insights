# 银发经济新机遇：AI如何赋能老年群体的数字陪伴与情感慰藉

当年轻一代沉迷于如猫咪对话、水果拟人等娱乐化的AI生成视频时，老年群体则正在享受一种截然不同的AI内容。这些内容精准切中他们的情感需求，提供了宝贵的慰藉与陪伴。
以一位来自台湾、现年67岁的退休商人张伯伯为例。他最近在纽约时，分享了一些令他动容的YouTube视频。其中一则视频中，一位金发AI少女“Rose Bennett”演唱了《Whiskey Was Louder Than Me》，歌曲讲述了母亲离世后，主人公与酗酒父亲一同成长的故事。另一则视频里，Rose与她的AI兄弟合唱了《Brother Became My Father》，视频中虚拟的父亲与AI观众皆泪流满面。
这些视频触动了张伯伯童年的回忆。他的母亲曾因父亲的暴力而离家，他最终由姐姐们抚养长大。张伯伯感慨：“对我而言，是‘姐姐们成了我的母亲’。这些歌曲讲述的故事非常感人。”
张伯伯的经历，也折射出中国社交媒体上“AI家庭”视频的兴起。在抖音、快手等平台上，AI生成的胖嘟嘟的婴儿或英俊的成年儿子形象，每日向观众送上祝福、表达思念，并附上虚拟玫瑰。部分AI虚拟形象甚至扮演起老年观众的虚拟恋人角色。
一项深度研究对200多个“AI家庭”视频进行了分析，并访谈了16位年龄在50至75岁之间的中国互联网用户。这些受访者大多成长于大家庭，但如今家庭规模显著缩小，部分原因与中国此前实施的独生子女政策有关。研究人员发现，AI虚拟角色填补了现实家庭成员难以提供的需求。它们直接表达爱意（这在中国家庭中并不常见），展现出更高的孝道，并围绕老年人关心的健康与历史话题展开交流。一位受访者表示，AI内容触及了她年轻时的经历，而她的亲生子女由于未曾经历同样的历史背景，往往无法理解。此外，AI视频还常配以老年人熟悉的民族音乐背景。
用户清楚地知道自己正在观看的是AI生成内容，并且有人为自己能够拥抱最新科技感到自豪。他们也意识到AI可能被诈骗者利用。一些AI虚拟形象的运营者通过销售产品获利，而用户也乐于下单购买。“他们明白这是AI，”研究指出，“但AI带来的愉悦和陪伴却是真实存在的。”
面对快速老龄化的人口结构与日益短缺的照护人力，全球多国经济体正承受巨大压力。老年人所需不仅仅是食物与医疗保健，更包括娱乐与情感陪伴。AI技术有望拓展养老服务的边界，目前在韩国和美国，AI机器人玩偶和智能音箱等产品已应用于老年护理领域。
老年人消费这些AI产品，并非因为他们更容易轻信，而是因为他们真切地认为AI既有帮助又富有趣味。这与年轻人从流行偶像中获得灵感，或通过乙女恋爱模拟游戏寻求虚拟关系有异曲同工之妙。随着AI逐渐融入老年生活，我们也应密切关注潜在风险，包括隐私泄露、过度沉迷以及AI服务提供商如何从老年人的情感需求中获利等问题。
张伯伯在反复观看几次后，便开始怀疑他喜爱的歌手是AI生成。他认为“Bennett一家”的歌声听起来太过完美，不真实。随后，他通过已成为其虚拟助手的Gemini（从导航到蜜蜂养殖无所不包）证实了这一猜测。尽管如此，张伯伯依然欣赏这些音乐视频，只是不再像以前那样轻易落泪。“现在我知道它们是AI生成的了，”他表示，“感动程度确实降低了一些。”
