# 武汉手游兼职骗局：牵出500万洗钱大案，11人落网

🚨 武汉手游兼职骗局：牵出500万洗钱大案，11人落网
湖北武汉近日破获一起利用手游兼职诱骗实名信息进行洗钱的重大案件。一名青年在应聘“手游代练”兼职时，因察觉到异常操作及时报警，成功牵出涉案金额高达500万元的专业洗钱团伙。
该团伙以“时间自由、操作简单、轻松赚钱”为诱饵，吸引社会经验不足的年轻人提供身份证、银行卡并完成人脸识别。实际上，其真实目的是批量获取实名认证的支付账户，用于承接上游黑灰产业的非法资金。
警方调查显示，该团伙在一个月内，动用了超过1300个支付账户进行“跑分”洗钱，通过小额分散、快速进出的方式，将诈骗资金在多个账户间连续流转，以规避平台风控。此次行动共抓获11名涉案人员，并捣毁一处隐蔽窝点。这起案件再次提醒公众，对于要求提供敏感个人信息的“轻松兼职”务必提高警惕。
