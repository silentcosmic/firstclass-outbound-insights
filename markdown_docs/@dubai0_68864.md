# 苹果印度代工厂泄密：iPhone 18 Pro细节恐已外泄

🚨 苹果印度代工厂泄密：iPhone 18 Pro细节恐已外泄
据路透社昨日报道，苹果在印度重要的制造伙伴塔塔电子遭遇大规模数据泄露。此次事件导致苹果尚未发布的iPhone 18 Pro系列机型的供应商名单、零部件清单及疑似跌落测试照片等敏感信息外泄。塔塔电子作为苹果在印度最重要的制造伙伴之一，不仅供应零部件，也承担了约三分之一的印度iPhone组装业务。
名为World Leaks的黑客勒索组织已声称对此次攻击负责，并在暗网发布了逾20万份文件，总规模超过630GB。这些泄露数据不仅包含苹果的组件设计文件、供应商对应关系及质量检测标准，还涉及特斯拉Model 3改款项目的工程图纸，以及台积电、高通等公司的疑似敏感资料，甚至包括部分员工的护照扫描件、邮件和事件日志。
