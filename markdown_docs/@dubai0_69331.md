# 日本首例跨国非法器官案：中国医生团队涉案

🚨 日本首例跨国非法器官案：中国医生团队涉案
日本警视厅近日侦破了该国首例非法有偿跨国器官移植案件，三名日本籍中介因涉嫌组织患者前往柬埔寨进行地下肾脏移植手术而被捕。此案揭示了一个横跨多国的非法医疗链条。
据案情披露，今年1月，一名70余岁的日本患者在柬埔寨首都金边接受了活体肾脏移植，供体据信是一名20多岁的柬埔寨女性。此次手术的焦点在于，一个由中国籍医生和协调员组成的团队在金边负责对接并主刀。患者除了向中方医生团队当面支付约2500万日元（约合15.7万美元）现金外，还向日本中介汇款936万日元作为“谢礼”。
目前，日本警方正联合柬埔寨方面，对这一复杂的跨国非法医疗网络展开深入调查。
