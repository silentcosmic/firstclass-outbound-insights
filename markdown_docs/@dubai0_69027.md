# 泰国严打航空走私：出境安检全面升级

🚨 泰国严打航空走私：出境安检全面升级
鉴于近期发生的泰航空乘涉嫌跨国运毒案件，泰国政府正全面收紧出境毒品管控。泰国总理兼内政部长阿努廷于7月3日主持国家禁毒委员会会议，宣布将推出六项新措施，旨在大幅提升机场安检标准，尤其针对航空业人员。
新规明确指出，航空公司机组人员若违规代带或代寄物品，将面临包括吊销执业资格在内的最严厉纪律处分。此外，泰国将全面强化对出境旅客及其托运行李的X光检查和警犬缉毒力度，并严格检查机组人员的随身及托运行李。
为进一步堵塞漏洞，政府要求各航空公司制定更严格的内部规定，禁止空乘人员为他人携带或邮寄物品。同时，泰国还将推动成立旅客信息单位（PIU），完善国际旅客姓名记录（PNR）信息共享机制，并提升机场安检人员的专业能力。未来，海关、机场管理部门及相关机构的数据库也将实现互联互通，以提高风险预警和执法效率。此举意味着所有从泰国出境的旅客，尤其是航空公司员工，都将面临更为严格的安检和审查。
