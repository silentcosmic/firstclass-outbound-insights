# ️ 冲突后首航：德黑兰-迪拜直飞航线重启

✈️ 冲突后首航：德黑兰-迪拜直飞航线重启
6月29日，一架从伊朗首都德黑兰起飞的航班在迪拜国际机场顺利降落，标志着自今年2月地区冲突爆发以来，两地间首个直飞航班的恢复。该航班由伊朗航空公司FlySepehran执飞，当日下午1时18分抵达迪拜国际机场2号航站楼，并计划于同日晚些时候执飞返程航班。
此次航线重启前，阿联酋与伊朗之间每周有数十个航班往返。然而，目前包括迪拜航空和阿拉伯航空在内的阿联酋主要航空公司尚未恢复该航线运营，航班选择仍旧有限。FlySepehran官网信息显示，下一班德黑兰至迪拜的航班定于7月1日，后续航班安排在7月8日。
