# 星展银行千万新元民生援助计划启动，聚焦日常必需品购物优惠

亚洲领先金融机构星展银行（DBS）及其旗下储蓄银行（POSB）已正式推出其斥资千万新元的社区援助计划首阶段举措，旨在通过每周提供必需品购物优惠，直接惠及新加坡民众。此轮焦点集中在日常消费中的鸡蛋与大米。
自2026年7月18日起至8月29日，消费者在每周六均可享有特定自有品牌鸡蛋及大米产品各3新元的折扣。
这项优惠覆盖新加坡全岛的昇菘（Sheng Siong）及巨人（Giant）连锁超市，每位消费者单次购物最高可节省6新元。
这项名为“星展周六优惠”（DBS Saturday Savings）的活动，可在129家指定门店享受。消费者仅需使用任意星展银行或储蓄银行的信用卡或借记卡进行支付，即可自动获得折扣。
每周预计提供约9万份优惠名额，这一数字相较于去年同期该银行推出的同类超市购物优惠活动，已扩大逾三倍。
此轮计划在覆盖范围上显著升级，不仅将巨人超市纳入合作体系，更将参与资格从原先的储蓄银行卡持卡人拓宽至所有星展银行及储蓄银行的信用卡与借记卡用户。
星展银行新加坡消费金融业务主管黄庆祥（Calvin Ong）指出：“今年早些时候，我们公布了一项价值1000万新元的援助倡议，其核心承诺便是为新加坡居民带来实实在在的经济助益，助力他们提升消费购买力。我们深知，对于众多家庭而言，周末的超市采购已成为一种固定的生活习惯。因此，通过在每周六提供此类优惠，我们期待能让这些日常购物变得更为经济实惠。”
回溯至去年，星展银行曾在昇菘超市推出过0.60新元的大米与食用油特别优惠，为本年度计划奠定了基础。
此外，星展银行及储蓄银行将于今年9月再次启动其颇受欢迎的PayLah! 3新元周六返现活动。具体实施细则，预计将在临近上线时对外披露。
值得一提的是，PayLah!返现活动最初主要面向小贩中心（hawker meals）的餐饮消费，随后逐步扩展至社区商铺及传统湿货市场，覆盖面日益广泛。
本次超市购物折扣将秉持“先到先得，售完即止”的原则。每位客户每日针对每件指定商品，仅限兑换一次。
