# 迪拜无人驾驶出租车上线：5迪拉姆体验未来出行

⚡ 迪拜无人驾驶出租车上线：5迪拉姆体验未来出行
迪拜正积极推进其智慧城市愿景，近日，迪拜交通局（RTA）旗下无人驾驶出租车服务已正式投入运营。这项由Apollo Go提供技术支持的Robotaxi服务，目前已覆盖朱美拉和乌姆苏奎姆地区，单程票价定为5迪拉姆，为当地居民及访客提供了体验未来交通的便捷途径。
此举不仅是迪拜智慧城市战略的重要组成部分，也预示着中东地区在未来出行解决方案上的持续探索与布局。
