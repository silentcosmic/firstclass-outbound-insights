# 柬埔寨电诈难除：VOD揭腐败根源

🔍 柬埔寨电诈难除：VOD揭腐败根源
柬埔寨独立媒体VOD近日发布署名文章，深入剖析了洪玛奈政府长期未能根除网络诈骗产业的深层原因。该报告核心指出，电诈产业之所以难以铲除，根本在于其与柬埔寨政治利益的深度交织、系统性腐败以及对灰色经济的严重依赖。
文章详述，电诈网络与执政党高层及部分寡头财阀（如受美国制裁的参议员李永法）存在直接或间接的利益链条，彻底整治恐将动摇现有执政根基。此外，系统性腐败使得地方官员成为电诈活动的保护伞，而电诈及洗钱所得的巨额灰色资金，已成为支撑柬埔寨房地产等支柱产业的重要经济来源。
在外交层面，柬埔寨政府选择掩盖真相并压制国内异见，导致其国际形象受损，外交陷入孤立。这不仅可能为邻国（如泰国）以“维护数字安全”为由干预柬埔寨主权提供借口，更迫使柬埔寨在西方制裁下过度依赖单一大国，面临丧失外交独立性的巨大风险。
对此，柬埔寨政府方面迅速回应，称此文“不值一驳”，否认其指控。
