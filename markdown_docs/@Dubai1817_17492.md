# ️ 柬埔寨“杀猪盘”新变种：中国外籍劳工深陷网恋骗局

⚠️ 柬埔寨“杀猪盘”新变种：中国外籍劳工深陷网恋骗局
近日，一名在柬埔寨务工的中国籍男性深陷情感诈骗，损失逾3000美元。这位50岁的受害者，在当地水电安装行业不景气、失业近四个月的困境中，通过社交平台Facebook结识了一名自称19岁的柬埔寨女性。对方主动示好，迅速发展成“夫妻”关系，声称不介意年龄差异。
取得信任后，该女子及其“妹妹”轮番编造悲惨身世，从母亲重病至病危离世，辅以虚假病床照片及视频博取同情，并以此为由急需资金置办葬礼。在短短二十天内，受害者在自身经济拮据的情况下，不惜向工友及老板借贷，累计向对方姐妹转账逾3000美元。
然而，在所有款项转出后，对方随即态度冷淡并最终失联。受害者随后发现诸多破绽，包括女子实际年龄与谈吐不符、婚姻生育说辞前后矛盾，以及对方多次以各种理由拒绝视频连线、上门探望或与家人沟通。目前，受害者已妥善保存所有聊天记录及转账凭证，并表示若对方确有真实困境，仅要求归还2000美元；但若证实为精心策划的感情骗局，他将请求当地警方彻查真相，核实涉事人员的真实身份及背景。
