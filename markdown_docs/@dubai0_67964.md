# ️ 迪拜商家警惕：新型汇款骗局致千U损失

⚠️ 迪拜商家警惕：新型汇款骗局致千U损失
市场消息显示，有迪拜商家遭遇新型诈骗。骗子以购买商品为名，在加密货币（U）转账时故意多付一笔款项，随后谎称转账失误，要求商家退回多余金额。
据受害者描述，一名迪拜电话卡商家因此损失超过1000U。在商家按要求退款后，骗子随即失联。值得注意的是，骗子在交易过程中还盗用他人U地址截图，手法隐蔽，增加了识别难度。
