# 柬埔寨警示：Telegram等平台附件藏木马风险

🚨 柬埔寨警示：Telegram等平台附件藏木马风险
柬埔寨邮电部部长谢万德近日向公众发出紧急呼吁，提醒民众在使用Telegram等即时通讯软件时，务必对文件附件保持高度警惕。
谢万德部长指出，当前网络黑客正利用特定后缀名的附件，意图在数字设备上植入木马病毒，从而窃取敏感数据。他特别强调，公众切勿打开、点击或运行带有`.exe`、`.bat`、`.vbs`、`.ps1`、`.sh`、`.msi`、`.scr`等后缀的文件。这类文件极可能内含恶意软件，一旦运行，设备可能被远程控制，个人隐私、账户密码乃至银行信息将面临被窃取的巨大风险。
