# 洞察潜意识：顶尖咨询机构Triggers如何解码消费者品牌关联，并质疑传统细分策略

营销咨询机构Triggers或许比你自己更了解你的真实想法。
这听起来可能有些令人不安，但与其他科技平台相比，Triggers在理解消费者对品牌的认知和感受方面，采用了一种侵入性较低的方法。它通过一种名为“记忆唤起”（memory elicitation）的流程，从广泛的受访群体中收集面板数据。
Triggers首席执行官Leslie Zane打趣道，尽管这个词听起来“像是出自《哈利·波特》或科幻小说”，但这仅仅是Triggers数据收集过程的一种独特称谓。每个品牌约有300名受访者，他们以文字和图片的形式，将自己对特定品牌的记忆上传至Triggers平台。
这些记忆可以是任何形式，小到儿时与祖母一同前往麦当劳享用薯条的特定事件，大到像“Nathan's”关联到“热狗”，再到“棒球”的词语联想。
上周四，Triggers宣布推出一项新功能，名为“品牌本能监测器”（Brand Instinct Monitor，简称BIM），旨在追踪品牌关联随时间推移的变化。
知名食品巨头卡夫亨氏（Kraft Heinz）是Triggers多年的客户，也是BIM的首批试用者之一。卡夫亨氏消费者洞察负责人Doug Healy表示，这项新工具让公司对“驱动消费者选择的潜意识线索和关联”有了更深入的理解，从而能够基于需求状态、动机和特定场景，制定更精准的创意和信息传递策略。
BIM还为卡夫亨氏描绘了一幅更真实的消费者痛点图景。Healy透露，最初公司对消费者担忧存在“内部假设”，这些假设来源于问卷调查以及社交媒体和面对面交流中听到的轶事，比如人们声称避免食用包装食品或高碳水零食。但他强调：“当我们审视实际行为时，会发现消费者仍然在做许多他们嘴上说着不做的事情。”因此，对于卡夫亨氏而言，理解客户的潜意识关联至关重要，而非仅仅依赖“有意识的、自我报告的反馈”，因为这些反馈“可能具有误导性”。
Triggers的核心在于其专有的“品牌联结组”（Brand Connectome）工具，它承载着品牌忠诚者（即对品牌抱有最积极关联的受访者）所分享的“累积记忆和关联”，Zane解释道。
Zane指出，BIM运用专有的数据科学模型，量化“哪些关联对品牌增长影响最大”，从而帮助品牌在营销中集中发力于这些驱动因素。品牌随后可以持续衡量这些关联的变化，评估其营销工作的成效。这些结果会与品牌既有的“核心”（即面板受访者中最普遍的关联）进行对比。
Triggers的受访面板由外部招募公司构建，但其目标是尽可能广泛的受众。Zane明确表示：“我认为市场细分是浪费时间。”她补充说，对品牌忠诚的人，无论其身份如何，往往持有相似的关联。如果一个品牌无法对所有人都意味着同样的事情，那么它就不再是一个真正的品牌。
Zane认为，除了理解消费者如何真实感知品牌外，营销人员还需要洞悉这些认知如何随时间演变，因为关联和记忆并非一朝一夕就能消逝。例如，多年来，洗碗精品牌Palmolive曾塑造了一个名为Madge的角色，她是一名美甲师，并用该品牌的肥皂浸泡客户的指甲。
Zane提到，这个角色在1990年代被停用，大约三年后，该品牌“销量直线下降”。她补充说，传统的品牌健康追踪器存在“滞后效应”，但如果当时品牌能够追踪这些关联，或许会更早意识到Madge是许多Palmolive用户坚持使用该品牌的一个关键原因。
凭借BIM揭示的结果，卡夫亨氏成功避免了犯下类似的错误。Healy表示，千禧一代的父母曾报告称，Lunchables（一款儿童午餐盒产品）让他们感觉“老式、过时，像是过去的遗物”。然而，同一群体所报告的潜意识关联却是“积极的、怀旧的记忆”，联想到产品标志性的黄色包装盒和泡泡字体。
Healy指出，如果卡夫亨氏当时仅关注有意识的信号，“我们可能已经彻底改造了品牌，并因此失去了有价值的品牌资产。”
