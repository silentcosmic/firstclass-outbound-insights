# ️ 阿联酋修法：疫情感染员工带薪禁止上班

⚖️ 阿联酋修法：疫情感染员工带薪禁止上班
阿联酋联邦国民议会近日通过一项关键法律修正案，明确规定在官方宣布的疫情或大流行期间，感染传染病的员工将被禁止上班，但在此期间将获得全额薪资，且法定休假权益不受影响。
此次修订旨在强化国家在传染病预防、监测和控制方面的法律框架，同时更全面地保障公共卫生及疫情期间受影响个人的权益。修正案强调，此举是为了防止传染病进一步扩散，并确保遵守公共卫生要求的员工不会因此遭受经济损失。
该法律修正案旨在通过厘清个人、雇主和相关部门的责任，提升国家应对传染病疫情的准备和响应能力。在支持国家公共卫生策略、鼓励民众遵守隔离措施的同时，也保障了受影响员工的工资和就业福利不受削减。
