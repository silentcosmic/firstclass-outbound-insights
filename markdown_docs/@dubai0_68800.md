# 柬埔寨首相洪玛奈：承认网络诈骗，力挽投资者信心

🚨 柬埔寨首相洪玛奈：承认网络诈骗，力挽投资者信心
6月27日，柬埔寨首相洪玛奈在金边出席“全国中小微企业日”活动时，公开承认该国存在网络诈骗问题，并强调必须严厉打击，以重塑国家声誉和投资者信心。
洪玛奈指出，执法部门必须正视网络诈骗的客观事实，并协同展开打击行动。他警告称，若低估当前形势，将可能毁掉柬埔寨战后数十年的发展努力。他引用一位投资者访问西哈努克省后，担忧“来这里投资是否会被绑架”的疑问，以此说明网络诈骗已严重损害国家形象，甚至影响到海外柬埔寨人的声誉。
