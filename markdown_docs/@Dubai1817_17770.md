# ️ 旅业警示：厦门“熟人”诈骗致巨额资金损失

⚠️ 旅业警示：厦门“熟人”诈骗致巨额资金损失
近日，有业内消息指出，厦门一家名为“任意飞CC”的公司及其负责人谢CC，近期被指涉嫌财务纠纷，引发业界关注与警示。据悉，谢CC长期活跃于厦门、迪拜、吉隆坡等地的商业圈，被指专门利用熟人关系进行交易。
据爆料，有客户在6月26日向谢CC支付了7万USDT，并获得明确收款确认。然而，次日该客户即遭对方切断所有联系方式。事发后，谢CC以“钱包被盗”为由进行搪塞，目前仅退还3万人民币，剩余款项至今未予处理，也未给出合理解释。
此事件被指控并非偶发失误，而是有预谋地利用熟人信任，采取快速收款、长期拖延、寻找借口等手段，意图侵吞客户资金。鉴于此类缺乏诚信的商业行为，建议旅行社同行及广大客户务必提高警惕，审慎选择合作方，防范潜在的交易风险。
