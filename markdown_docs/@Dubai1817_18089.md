# 中菲联手：跨境电诈头目林某落网，涉案数亿

🚨 中菲联手：跨境电诈头目林某落网，涉案数亿
今日下午，中菲两国执法部门成功完成跨境移交，将涉案金额高达数亿元人民币的电信网络诈骗犯罪集团核心头目林某遣返回中国，接受进一步调查。
中方相关部门强调，将持续深化与菲律宾、柬埔寨、越南、老挝等东南亚国家的执法合作，严厉打击跨境电信诈骗、人口贩运等不法活动，并依法追究幕后组织者的法律责任。
