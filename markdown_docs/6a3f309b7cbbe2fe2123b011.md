# 出海 · 头等舱晚报 | 具身智能独角兽估值飙升，小鹏自动驾驶全球化提速，苹果预警AI内存需求激增

五粮液公司近期发布公告，披露了曾从钦先生因故无法正常履职的董事职务变动事宜。公司于2026年6月26日召开了2025年度股东会，并审议通过了《关于免去董事及补选第七届董事会非独立董事的议案》。根据决议，自股东会通过之日起，曾从钦先生不再担任公司董事职务，其原兼任的董事长、董事会专门委员会主任委员及委员职务亦同时解除。其原定任期至公司第七届董事会届满之日，即2028年6月19日。公司明确指出，曾从钦先生离任后将不再在公司及其控股子公司任职，并强调此次人事变动不会对公司正常经营产生影响。截至公告披露日，曾从钦先生未持有公司股份，亦无任何未履行承诺事项。
京东集团在医药零售领域的布局持续深入。近期，京东大药房（烟台）有限公司正式设立，法定代表人为张媛媛。该公司注册资本为10万人民币，经营范围涵盖了药品零售、药品互联网信息服务、第三类医疗器械经营以及医疗器械互联网信息服务等核心业务。股权结构显示，新公司由京东大药房（青岛）连锁有限公司全资持有。
埃隆·马斯克旗下的太空探索技术公司（SpaceX）正着手为美国消费者拓展全新的星链移动服务。SpaceX总裁兼首席运营官格温·肖特韦尔透露，公司正积极筹备“推出星链零售产品”，并可能在美国本土构建自有地面移动网络。若此战略得以全面实施，星链的全球卫星通信系统将直接面向个人客户提供移动通信合约，从而开发新的零售市场，这将使其直接与美国市场的主要运营商，包括Verizon Wireless、AT&T和T-Mobile展开激烈竞争。
小鹏汽车董事长何小鹏近日公开表示，第二代VLA（Valet Parking Assist，代客泊车辅助系统）的全球化进程已进入确定性模式。他进一步指出，联合国世界车辆法规协调论坛（WP.29）批准的DCAS UNR 171系列02法规将在未来六个月内生效，并成为欧盟的强制性法规。这意味着，从2026年底起，自动驾驶技术将能够合法地在全球范围内推广应用。何小鹏还提前预告，展望2027年，小鹏汽车的VLA和VLM（Valet Driving Memory，代客驾驶记忆）功能将在海外市场亮相并提供用户体验，届时海外用户有望体验到中文与当地语言混合的车载对话及车辆控制功能。
在具身智能领域，一家专注于灵巧手研发的创新企业——临界点，近期完成了近10亿元人民币的融资，使其估值一举跃升至10亿美元，成功跻身独角兽企业行列。这家从智元机器人分拆而出的公司，被业界视为具身智能赛道中成长最为迅猛的企业之一。令人瞩目的是，临界点仅用了短短五个月时间，便实现了独立发展并达到10亿美元的估值里程碑。在这极短的周期内，该公司已密集完成了四轮融资，吸引了包括百度、蓝驰创投、高瓴创投在内的多家顶尖资本积极投资布局。
根据其向香港交易所提交的公告，北京同仁堂医养投资股份有限公司计划在香港市场公开发售1.08亿股股份。此次发售价格区间定于每股5.48港元至6.21港元，公司预计通过此次发行最多可筹集6.716亿港元资金。预计该公司股票将于7月7日开始在香港联交所进行上市交易。
苹果公司近期指出，人工智能数据中心的迅猛扩张正引发内存需求呈现爆炸式增长。该公司观察到，此前从未有任何单一零部件的价格出现如此大幅且快速的上涨。这一趋势预示着整个电子产品行业正面临前所未有的供应链及成本挑战。
通用具身智能机器人领域的创新企业“无界动力”于6月26日宣布成功完成逾2亿美元天使轮融资。本轮融资由京东关联基金、C 资本、弘毅投资、盛宇投资、丰源投资等多家知名机构联合领投，线性资本、红杉中国、华业天成、雅瑞资本等现有股东继续追加投资。据悉，所筹资金将主要用于支持具身通用大脑的深入研发、核心技术基础设施的建设，以及推动全球范围内的规模化产品交付。
“深度机智”公司近日宣布完成新一轮数亿元人民币融资。本轮融资由国寿长三角科创基金领投，既有股东普华资本、诚通科创基金持续加注，同时吸引了蓝湖资本、博彦科技、磐谷创投、朝晖资本、财鑫资本、道禾长期投资、易高资本、明德资本等多家市场化及产业资本的共同参与。值得关注的是，该公司目前下一轮融资已进入收尾阶段。此前，深度机智的所有产品线均已获得商业订单，累计签约金额已达数千万元。
国内超高清技术应用领域的领先企业4K花园日前正式对外宣布，其自主研发的AI短剧画质提升工具“蝶变3.0”将于7月1日正式上线并面向公众开放公测。伴随此次发布，4K花园同步启动了“AI短剧超高清专项扶持计划”，旨在推动AI短剧内容的高质量发展。
商络电子公司透露，其部分AI服务器电源客户已开始涉足液冷技术。公司目前正与这些客户合作研发液冷项目，为其提供关键元器件产品及配套技术解决方案。然而，商络电子也谨慎提示投资者，当前该类业务未来的订单增长量及对公司业绩的贡献仍存在不确定性，敬请投资者关注潜在的投资风险。
工业和信息化部近期发布公示，将《开源鸿蒙设备统一互联接入与控制接口》等3项行业标准，以及《Ku频段船载卫星通信地球站通用测试方法》等9项推荐性国家标准计划项目予以公开。此次公示的目的是广泛征求社会各界意见，公示期定于2026年6月27日至2026年7月26日。
当前，具身智能产业的上下游协同、闭环创新路径正加速成型。从核心零部件制造、整机系统集成，再到多元场景应用，一条完整的产业链生态正在逐步完善。据行业监测数据，中国具身智能市场规模预计在2025年将达到约9150亿元人民币，并有望在2026年进一步增长至10904亿元。截至目前，全国范围内具身智能相关企业数量已突破一万家。这一产业链的日益成熟，正在驱动行业创新效率发生根本性的变革。
