# ️ 阿联酋ADCB银行突发大面积故障，服务受阻

⚠️ 阿联酋ADCB银行突发大面积故障，服务受阻
今日，阿联酋阿布扎比商业银行（ADCB）遭遇大面积系统故障，导致其多项核心金融服务中断。
受影响的服务范围广泛，包括移动应用程序无法正常使用、转账交易受阻（甚至出现转账成功但资金未到账的情况）、ATM取现功能失效、以及银行卡支付中断。此外，部分客户反映，已开出的支票也遭遇了强制退回。
