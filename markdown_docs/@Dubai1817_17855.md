# 西港省政府联手民间力量：共筑数字反诈防线

🚨 西港省政府联手民间力量：共筑数字反诈防线
柬埔寨西港省政府与内阁办公厅民间社会组织联盟论坛今日在西港市联合举办“数字化时代增强民间社会组织能力”对话论坛，旨在鼓励民间力量积极参与，共同打击日益猖獗的虚假信息和网络诈骗。
论坛由民间社会组织联盟论坛常务副主席杜威拉与西港省副省长隆迪蒙共同主持。杜威拉在会上强调了数字转型的重要性，并指出洪玛奈总理领导的第七届政府始终将民间组织视为推动各领域发展及提升基层民众福祉不可或缺的合作伙伴。隆迪蒙副省长则建议各组织持续学习新技术，加强内部数据安全管理以防范网络犯罪，并善用数字平台提升与社区及地方政府的沟通效率与工作透明度。
