# 百家巨头联手：Visa、Stripe共建开放美元稳定币新生态

⚡ 百家巨头联手：Visa、Stripe共建开放美元稳定币新生态
数字支付领域正迎来一场由头部玩家主导的变革。日前，由Visa、Stripe等逾百家行业巨头共同发起的Open USD稳定币联盟正式亮相，剑指传统稳定币发行模式。该联盟计划通过共享储备金收益和取消交易费用，打造一个开放、低成本、高效率的数字支付新生态。
