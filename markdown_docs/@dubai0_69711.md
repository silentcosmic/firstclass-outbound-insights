# iPhone 20 重磅曝光：告别刘海，回归玻璃机身

⚡ iPhone 20 重磅曝光：告别刘海，回归玻璃机身
据市场消息，苹果计划在2027年为纪念iPhone问世20周年，跳过“iPhone 19”命名，直接发布全新的iPhone 20系列。该系列内部代号为V73和V74，预计将配备6.3英寸和6.9英寸屏幕，尺寸与现有Pro机型相近。
外观上，iPhone 20系列将迎来重大革新，有望回归一体化玻璃机身设计，并放弃此前两代采用的铝合金材质。更引人注目的是，新机将采用四微曲屏，并集成屏下Face ID技术，彻底告别沿用多年的“刘海”和“挖孔”屏，实现近乎无边框的真全面屏视觉效果。
供应链方面，有消息指出相关生产场地已完成翻新，等待设备进场。玻璃素材供应商预计以伯恩光学为主，辅以比亚迪和蓝思科技。核心配置上，iPhone 20系列将首发台积电2nm制程的A21 Pro芯片，并预装iOS 28系统，预示着性能的显著飞跃。
