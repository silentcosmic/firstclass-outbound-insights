# 柬波贝“杀猪盘”窝点被端：45名外籍嫌犯落网

🚨 柬波贝“杀猪盘”窝点被端：45名外籍嫌犯落网
柬埔寨卜迭棉芷省联合执法工作组于6月26日对波贝市的3处涉嫌网络诈骗窝点进行了突击清查，成功捣毁了这些非法运营场所。
此次行动中，执法人员共查获45名外籍嫌疑人，分别来自印度（20人）、泰国（14人）和印度尼西亚（11人），同时控制了2名柬埔寨籍涉案人员。据查，这些团伙主要利用排屋等隐蔽地点，通过俗称“杀猪盘”的网恋诱导投资加密货币方式实施网络科技诈骗。现场缴获了包括80部手机、笔记本电脑在内的大批作案工具。
