# 莱纳斯与JS Link携手马来西亚，共建稀土永磁产业新版图

澳大利亚稀土巨擘莱纳斯（Lynas Rare Earths）与韩国JS Link公司宣布达成战略合作，双方将携手在马来西亚关丹共同开发一座稀土永磁工厂。这一重大项目是在2025年7月签署谅解备忘录后，双方伙伴关系实质性推进的体现。
莱纳斯在周二发布的一份声明中指出，此次合作标志着两家公司建立了长期的战略伙伴关系。
根据协议条款，JS Link将在莱纳斯位于关丹的现有先进材料工厂附近，设立一座专注于钕铁硼烧结永磁体生产的工厂，规划年产能将达到3,000吨。该新建工厂预计将在马来西亚创造多达400个就业机会，其生产的永磁体将主要面向韩国、马来西亚以及其他核心市场的汽车、风能和电子产品制造供应链。
此外，莱纳斯计划向JS Link注资约5000万澳元（约合3470万美元），以普通股形式持有其约4.58%的稀释后股权。这笔资金将直接用于支持马来西亚新工厂的建设，所持股份将设定为三年的托管期。
根据双方签订的独家供应协议，莱纳斯将以商业条款向JS Link位于韩国礼山的现有磁铁工厂，以及未来在马来西亚新建的工厂，持续供应稀土材料，此协议有效期将延续至2038年1月。
莱纳斯稀土公司临时首席执行官Pol Le Roux先生指出，此次战略合作汇聚了莱纳斯在稀土加工领域的深厚专长与JS Link卓越的磁体制造能力。这不仅将在马来西亚催生一个全新的制造业态，更重要的是，它将助力莱纳斯实现其核心增长目标，即积极拓展中国境外金属和磁体供应链的布局。
JS Link首席执行官Jun Y. Lee先生强调，与莱纳斯的携手合作对于构建一个稳定且高效的永磁体价值链至关重要。这不仅能增强全球稀土供应链的韧性，也将为马来西亚、澳大利亚和韩国三国的经济发展带来积极的推动作用。
