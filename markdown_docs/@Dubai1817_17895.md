# 吉隆坡豪宅诈骗窝点被捣毁：9名中国籍嫌犯落网

🚨 吉隆坡豪宅诈骗窝点被捣毁：9名中国籍嫌犯落网
7月10日中午，吉隆坡警方根据情报在满家乐（Mont Kiara）一处豪华公寓展开突击行动，成功捣毁一个运营约两个月的网络诈骗呼叫中心，并当场逮捕9名中国籍男子。
落网嫌犯年龄介于24岁至41岁之间，其中1人负责管理窝点，其余8人则担任“客服人员”，每人每月薪资约3000令吉。
初步调查显示，该团伙主要以“股票投资”为幌子，通过互联网锁定中国境内的受害者。嫌犯以高收益投资项目诱导受害者投入资金，随后由客服人员持续跟进，完成整个诈骗链条。
行动中，警方共起获23部手机、9台笔记本电脑、网络路由设备及2500令吉现金等涉案物品。目前，所有嫌犯已被带回警局接受进一步调查。警方表示，将继续追查案件背后的组织架构、资金流向及是否涉及跨境犯罪网络。
