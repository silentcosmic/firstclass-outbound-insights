# 越南G-Group重磅投资3亿美元，河内将迎AI数据中心综合体

越南G-Group科技集团宣布，将斥资7.6万亿越南盾（约合3亿美元），在河内建设“G-Campus Hoa Lac”高科技中心及Tier III数据中心综合体。该综合体将专注于人工智能（AI）和高性能计算（HPC）领域。
G-Group周二发布声明，证实该项目已于前一日的河内投资促进大会上获得了投资许可证。
“G-Campus”占地逾3.8万平方米，规划为四个功能区，旨在支撑高科技价值链。其中两个区域将聚焦于AI、大数据、物联网（IoT）、高性能计算及网络安全领域的研发、测试与设计。剩余两个区域则将容纳一个Tier III数据中心，专门服务AI和HPC工作负载，初期IT电力容量达20兆瓦。
根据规划，未来扩建完成后，总容量预计将达到30兆瓦，从而为国内外机构提供完善的数据基础设施、云计算以及AI和HPC商业服务。
G-Group董事长Phung Anh Tuan表示，“G-Campus”致力于与河内携手迈入AI时代，构建一个能够支持研究、技术开发、数字化转型以及大规模AI应用的综合性平台。
在获得投资许可证的同时，G-Group还与河内市人民委员会及OTECH（阿曼ICT解决方案提供商Omantel集团旗下成员）签署了一项战略合作协议。三方将共同探讨下一代AI数据中心发展模式、云计算基础设施、高性能计算、GPU计算、数据服务、信息安全以及数字化转型解决方案。
依据合作框架，河内市人民委员会将负责对接使用需求并支持关键技术项目。OTECH将贡献其在国家级数据基础设施运营和主权云治理方面的宝贵经验，并协助资金筹措。G-Group则将全面承担在越南市场的投资、技术资源、运营及商业化职责。
此项重大投资的背景是，河内市设定了雄心勃勃的目标：到2030年，每位居民至少实现四项物联网连接；到2035年，在大数据和AI平台上运行数字化政府。这些目标将直接催生对至少两个超大规模数据中心，以及更多专业AI和边缘数据中心的需求。
与此同时，东南亚地区正经历一场价值300亿美元的数据中心投资热潮，但该区域也正加速面临电力供应的严峻挑战。
