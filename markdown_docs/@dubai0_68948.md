# ️ 阿布扎比重罚：开车玩手机将面临800迪拉姆罚款

⚠️ 阿布扎比重罚：开车玩手机将面临800迪拉姆罚款
阿布扎比警方近日发出明确警告，针对开车时使用手机的行为将处以800迪拉姆罚款并扣除4个黑点。警方呼吁所有驾驶员务必保持高度专注，强调即使是在缓慢行驶或交通拥堵时，短暂的分心也可能导致严重交通事故。
交通和巡逻部门特别通过真实事故视频提醒司机，在交通缓慢或停车时更需警惕路况变化，以防追尾。
