# ️ 英国TikTok网红迪拜机场被捕，卷入谋杀疑云

⚠️ 英国TikTok网红迪拜机场被捕，卷入谋杀疑云
据英国媒体消息，一名23岁的英国TikTok网红Brooke George于2026年6月22日凌晨在迪拜机场被捕，原因系其涉嫌在当地犯下谋杀罪。她是在准备离境时被警方控制的。
Brooke George来自英国Gravesend，在TikTok上拥有一定粉丝基础。有消息称，她通过社交平台结识了26岁的英国籍迪拜常驻居民Bill Treeby，两人随后发展为网恋关系。男方曾多次邀请女方前往迪拜，提供奢华的旅行体验，女方也曾公开表示对其迪拜之行感到非常满意。
英国媒体已梳理出案发当日的部分时间线。同时，多家媒体暗示，此案可能涉及男方扣留女方护照、仅为其预订单程机票以及两人相处中存在的暴力行为。这些细节与近年曝光的年轻女性在迪拜遭遇诱骗、胁迫的案例有所关联，引发了业界对跨国关系中潜在风险的警惕。
截至2026年6月25日晚，Brooke George仍在迪拜被拘留，调查仍在进行中。阿联酋官方尚未发布任何详细通报或警方声明。英国外交、联邦及发展办公室（FCDO）已通过媒体表态，确认正与被拘留的英国公民保持联系，并向其家属提供支持，同时与当地当局进行沟通。
