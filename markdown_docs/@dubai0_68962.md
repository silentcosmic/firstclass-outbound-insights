# 柬埔寨新省长突查电诈窝点，严防死灰复燃

🚨 柬埔寨新省长突查电诈窝点，严防死灰复燃
柬埔寨蒙多基里省省长兼省统一指挥部主席功金尼今日上午亲自率领多部门联合执法小组，前往该省乌良县，对前期已被依法查封的一处网络诈骗窝点进行实地复查与督导。
此次联合行动集结了省警察局、省宪兵队、省军事行动区指挥部以及乌良县地方当局与警力，旨在强化安全管控，严防已被取缔的电诈窝点死灰复燃。新任省长此举，彰显了其确保地方治安与公共秩序长期稳定的坚定决心，也预示着该地区在打击犯罪方面的执法力度将进一步加强。
