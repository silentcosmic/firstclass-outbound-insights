# GCC紧急发声：伊朗袭击油轮，国际航行安全告急

🔥 GCC紧急发声：伊朗袭击油轮，国际航行安全告急
海湾合作委员会（GCC）近日对伊朗在霍尔木兹海峡袭击沙特“Wadiyan”号和卡塔尔“Al Rekayyat”号油轮的行为，发出了最严厉的谴责。GCC声明指出，此举不仅直接威胁船员生命安全，更是对国际航行和全球能源供应安全的公然侵犯。
声明同时谴责伊朗对巴林和科威特的一系列袭击，强调这些行为严重违反国际法、联合国安理会第2817号决议，并公然违背了美伊关于霍尔木兹海峡停火与重开的谅解备忘录。
GCC成员国重申，将依据《海湾合作委员会宪章》和《联合防务协定》展现充分团结和共同防御的决心。GCC强调，对任一成员国的袭击均视为对全体GCC国家的直接攻击，并保留依据《联合国宪章》第五十一条行使单独或集体自卫的权利。
GCC指出，伊朗必须为这些袭击及其带来的后果承担全部责任。其持续的敌对行为正严重损害地区及国际和平与安全，对国际航运和全球能源市场的稳定构成巨大风险。
GCC再次呼吁国际社会，特别是联合国安理会，谴责此类袭击，并采取坚定立场，确保霍尔木兹海峡的航行自由和安全通行，且不得征收任何过境费或服务费。这对于保障全球能源、粮食和医药供应以及贸易流通至关重要。
GCC敦促采取一切必要措施，促使伊朗立即无条件停止所有敌对行动，实现霍尔木兹海峡的持续开放，并全面遵守联合国安理会第2817号决议及美伊谅解备忘录，以维护地区及全球的和平与繁荣。
