# 泰国收紧免签政策：93国60天待遇取消，安全考量升级

🚨 泰国收紧免签政策：93国60天待遇取消，安全考量升级
泰国总理府副发言人普洛塔莱近日宣布，内阁已审议并批准了针对现有免签政策及签证便利措施的重大调整方案。此次调整的核心在于取消此前93个国家和地区享有的60天免签入境待遇，旨在堵塞安全漏洞，防止签证权益被滥用，并减少政策重叠，坚持“一国一权益”原则。
根据新方案，共有59个国家和地区将获得30天旅游免签待遇，其中包括印度、克罗地亚等6个新增国家，调整后欧盟27个成员国将全部享有统一的30天免签。此外，毛里求斯和塞舌尔两国则获15天免签待遇，未来将根据游客数据进行评估。
落地签政策亦有调整，阿塞拜疆、白俄罗斯和塞尔维亚三国仍可办理，但印度游客的落地签资格因其已获免签待遇而被取消。为加强边境管理，泰国安全部门将升级“泰国数字入境卡”系统，强化背景筛查和出入境记录追踪，并推动跨部门数据共享，以评估入境风险并监督签证使用情况。
普洛塔莱强调，此次审查并非为降低旅游吸引力，而是力求在促进经济、便利游客、维护国际关系与保障国家安全之间取得平衡，防止签证政策被不法分子利用。调整完成后，享有各类签证便利的国家和地区总数将达65个，相关新规将在《政宪公报网站》公布15天后正式生效，此前已入境的游客不受影响。
