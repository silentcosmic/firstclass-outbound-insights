# 柬埔寨警方严打网诈：私营安保被警告绝不包庇

🚨 柬埔寨警方严打网诈：私营安保被警告绝不包庇
今日下午，柬埔寨国家警察总署私营安保管理局局长林苏卡拉斯美在一次培训班闭幕式上明确指示，全国私营安保人员必须与警方紧密合作，严厉打击网络诈骗活动。
林苏卡拉斯美指出，凡是存在大量外国人聚集、囤积电脑与手机且行为隐秘的场所，均应被视为可疑窝点，安保人员需立即上报。他严厉警告，私营安保人员绝不允许与电诈分子勾结，严禁为其提供看守、庇护，或充当“内鬼”监视警方执法。
他强调，柬埔寨政府已颁布相关法令，并在全国范围内开展专项行动，誓将网络诈骗彻底清除。作为维护社会治安的重要辅助力量，安保行业必须提升专业素质，共同落实“安全村社”政策，以保障国家利益与安全。
