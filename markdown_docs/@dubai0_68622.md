# ️ 金边永旺诈骗警示：警惕“落难同胞”利用善心设局

⚠️ 金边永旺诈骗警示：警惕“落难同胞”利用善心设局
近日，一名在柬埔寨金边永旺商场遭遇诈骗的中国公民向市场消息透露，其被一名假借“落难同胞”身份的男子骗走100美元现金及一部备用手机。该事件再次凸显海外华人需对求助信息保持高度警惕。
据受害者描述，骗子以手机丢失、急需现金支付嘟嘟车费为由，利用同胞情谊博取信任。在初期被拒绝兑换USDT后，骗子巧妙升级求助，最终通过伪装出的极度窘迫，成功获取受害者的金钱和手机。该男子在取得财物后迅速失联。
此案例提醒广大海外华人，当前不法分子的诈骗手法日益隐蔽且善于利用人性弱点。面对类似借钱或借用物品的求助，无论对方言辞多可怜，都应保持清醒判断，切勿因一时心软而放松警惕。
