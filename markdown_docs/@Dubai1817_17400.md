# 太子集团要员日本被捕：或牵出更多海外资金网络

🚨 太子集团要员日本被捕：或牵出更多海外资金网络
日本媒体6月22日报道，太子集团核心人物胡小伟日前在大阪被日本警方逮捕。这位44岁的塞浦路斯籍男子，同时被认定为美国制裁名单中的“陈小二”。
公开资料显示，胡小伟长期以胡晏铭、陈小二等多个化名及不同国家身份文件活动。调查机构此前已通过多方信息交叉比对，确认“陈小二”与胡小伟为同一人，他被认为是太子集团商业体系中的重要成员，与集团创始人关系密切。
此次日本警方采取行动，并非直接针对其商业活动，而是因其涉嫌向东京有关部门申报虚假住址信息，试图办理长期居留手续。然而，胡小伟早已被美国列入制裁名单，英国方面也已对其部分资产采取冻结措施。
其名下商业布局横跨柬埔寨、英国、塞浦路斯以及英属维京群岛等多个国家和地区，通过离岸公司控制大量海外资产，并拥有私人飞机及多处房地产。随着电子设备及相关资料被扣押，日本警方后续调查是否会牵出更多资金往来和关联网络，仍有待进一步观察。
