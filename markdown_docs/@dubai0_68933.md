# 缅甸反诈引争议：25组织反对美方接触军政府

🚨 缅甸反诈引争议：25组织反对美方接触军政府
7月2日，美国与缅甸军政府就清剿诈骗园区问题进行接触，引发缅甸25个民间组织的强烈反对。这些组织公开表态，认为不应将军政府视为反诈合作伙伴。
反对组织指出，缅甸边境地区的诈骗园区问题根深蒂固，牵涉军方盟友武装、地方势力及跨国犯罪网络，并非简单清场即可解决。他们普遍担心，如果美国以打击网络诈骗为由与缅甸军政府合作，恐将为军政府提供“洗白”机会，使其借打击诈骗之名重塑形象。
组织方面强调，真正要有效打击诈骗园区，关键在于追查幕后保护伞、切断资金链并营救被困人员，而非赋予军政府国际合法性。尽管缅甸军方近期屡次宣称清剿诈骗中心，但外界普遍质疑其行动仅为表面功夫，未能触及真正的幕后利益链。
