# 美国放宽对阿联酋AI芯片出口限制

⚡ 美国放宽对阿联酋AI芯片出口限制
美国商务部近日发布通知，正式放宽对阿联酋的出口管制。此举将为阿联酋购买包括发展人工智能所需半导体在内的各类先进技术扫清障碍。
商务部指出，鉴于阿联酋为保护美国敏感技术所采取的措施，以及其在区域安全合作中对美国的支持，该国现已符合更宽松的出口待遇条件。这意味着，阿联酋政府及部分获批准的商业机构，包括科技旗舰企业G42，未来在采购英伟达和AMD的先进人工智能芯片时，将不再需要美国政府的许可证，且新规未对数量设定上限。
截至目前，美国商务部、阿联酋驻华盛顿大使馆、英伟达和AMD均未就此事立即置评。
