# 派安盈在印度设立前沿创新中心，加速全球AI平台发展

国际领先的支付与金融服务提供商派安盈（Payoneer）近日在印度古尔冈设立了一处创新中心。该中心旨在集中力量，深度聚焦于人工智能（AI）前沿技术、工程开发、全球平台构建及核心业务运营。
这一战略枢纽汇聚了来自多个关键领域的顶尖团队，包括工程技术、产品研发、数据分析、人工智能、合规科技、商业运营以及劳动力管理，形成了一个协同高效的创新生态。
古尔冈团队将承担核心平台能力的开发重任，为派安盈日益壮大的企业级市场、多元化的支付解决方案及劳动力管理服务提供全面的技术支撑与创新动力。
此外，该团队还将与派安盈遍布全球的研发团队紧密协作，确保产品开发能够实现全天候不间断推进，以满足全球市场的快速变化与客户需求。派安盈首席产品与技术官Oren Ryngler表示：“印度在全球业务未来布局中占据核心地位。随着公司致力于转型为一家AI原生企业，古尔冈的各项工作将直接塑造我们的全球平台。这一创新中心是我们为企业级市场客户、金融服务及劳动力管理构建关键能力的关键阵地，同时也是我们拓展下一代核心竞争力的重要引擎。”派安盈印度高级副总裁兼平台现场负责人Gaurav Gupta指出：“印度如今汇聚了全球最深厚的金融科技与人工智能工程人才储备之一，这使得其在人工智能从实验阶段迈向企业级部署的关键时刻，占据了独特的战略优势。我们在此吸引顶尖的人工智能与工程专家，旨在大规模解决复杂问题，这不仅使我们能更贴近客户需求，也极大地加速了全球企业的创新步伐。”
派安盈选择古尔冈，正是看中了其丰富的工程技术与人工智能人才资源，以及其在全球不同时区之间高效支持协作的独特能力，这将极大提升跨国项目执行的灵活性与效率。
在此背景下，派安盈近期还获得了印度储备银行作为支付聚合商运营的原则性授权，并成功收购了Skuad，该公司现已更名为派安盈劳动力管理部门，进一步拓展了其服务范围与市场影响力。
