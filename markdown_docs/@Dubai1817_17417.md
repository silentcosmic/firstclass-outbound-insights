# 越南“杀猪盘”大案告破：中国籍主谋操控，涉案逾70亿

🔍 越南“杀猪盘”大案告破：中国籍主谋操控，涉案逾70亿
越南北宁省公安机关近日通报，成功侦破一起跨国网络诈骗案件，并已依法对五名涉嫌“诈骗侵占财产罪”的越南籍嫌疑人提起刑事诉讼并实施拘留。调查显示，该团伙长期在老挝金三角经济特区活动，受雇于由中国籍人员控制的诈骗组织，专门针对越南国内民众实施网络诈骗，涉案金额累计超过70亿越南盾（约合人民币200万元）。
警方此前通过外事安全调查发现，一批越南公民频繁往返老挝和柬埔寨，虽鲜少在家乡居住，却持续向家人汇款用于购置高价值资产，资金来源异常，引起执法部门高度关注。深入调查后确认，这些人员涉嫌参与跨国网络诈骗活动。该犯罪组织采用典型的“杀猪盘”模式，嫌疑人利用非法获取的手机号码注册大量虚假社交账号，伪装成事业成功的女性企业家或高收入人士，通过长期聊天建立感情，逐步获取受害者信任。
在取得信任后，诈骗人员以高额回报为诱饵，诱导受害者参与虚假数字货币投资项目，并推荐其使用名为“Bitsten”的平台。受害者转账后，资金随即被侵吞，联系方式也迅速失联。目前，警方已确认至少五起受害案件，其中不乏单笔被骗金额超过20亿越南盾的案例。案件仍在进一步深挖调查中，警方正追查幕后组织成员及其他涉案人员。
