# ️ 马来西亚跨境诈骗案：49名被告拒不认罪，案件续审

⚠️ 马来西亚跨境诈骗案：49名被告拒不认罪，案件续审
7月7日，马来西亚纳闽法院首次开庭审理一起备受关注的跨境网络诈骗案，该案涉及57名嫌疑人。庭审中，在控方修改部分控状后，49名被告——包括48名中国籍和1名马来西亚籍人员——当庭否认全部控罪，案件正式进入司法审理程序。
此案源于马来西亚警方今年6月开展的一次跨部门联合执法行动。当时，警方突击搜查了纳闽一处涉嫌从事跨境网络诈骗活动的窝点，共逮捕了57名嫌疑人，其中包括48名中国籍、8名越南籍及1名马来西亚籍人员。警方调查显示，该犯罪团伙涉嫌通过网络平台，以韩国民众为主要作案目标，实施投资诈骗、电信诈骗等跨境诈骗活动，并以有组织犯罪集团模式运作，引起马来西亚及韩国执法部门的高度关注。
此次开庭标志着该案自6月大规模抓捕行动以来取得重要司法进展。法院将择期继续审理案件，审视涉案人员是否涉及串谋诈骗、参与跨国有组织犯罪等相关指控。马来西亚警方表示，案件调查工作仍在持续进行，不排除进一步追查幕后组织者及跨境犯罪网络。
