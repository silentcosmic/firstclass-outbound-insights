# Penske Media 集团完成对 Vox Media 剩余资产的收购，数字媒体版图再调整

上月，詹姆斯·默多克旗下的Lupa Systems完成了对Vox Media约一半股权的收购，具体涵盖了《纽约杂志》、Vox品牌及其播客网络。如今，Vox Media的剩余资产也找到了新的归属，由Penske Media Corp.接手，标志着媒体行业整合的又一关键进展。
