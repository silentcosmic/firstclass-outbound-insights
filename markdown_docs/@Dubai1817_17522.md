# 吉隆坡诈骗头目遭公开警告：证据已提交国内部门

🚨 吉隆坡诈骗头目遭公开警告：证据已提交国内部门
近日，有市场消息指出，一名在马来西亚吉隆坡活动的网络诈骗团伙头目遭到公开警告。爆料方不仅披露了该头目的身份、具体犯罪活动，更声称已将相关证据提交至国内相关部门。
据称，该头目名为张良，外号“汪洋”，籍贯湖北恩施。他长期从事网络诈骗活动，主要目标为美国投资者，涉及美国股票诈骗及“精聊”（社交工程）等多种形式。爆料人表示已掌握张良大量犯罪证据，包括其手机数据及相关资料，并已整理上交。
消息强调，张良被给予最后三天时间，以面对其在马来西亚的犯罪行为，否则将面临严峻后果。其中特别提到，张良向其家人汇款逾百万元人民币，此举不仅可能牵连其家人，也足以让其幕后老板受到牵连。此事件再次凸显了海外网络诈骗活动的复杂性和跨境打击的紧迫性。
