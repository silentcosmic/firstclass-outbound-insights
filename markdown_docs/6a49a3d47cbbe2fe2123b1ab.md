# 微软豪掷25亿美元，组建AI工程新部门，深度赋能企业级应用

微软宣布将斥资25亿美元，成立一支全新的AI工程业务部门。该部门将集结6000名行业专家，专门面向企业级客户提供深度服务。
这支被命名为“微软先锋公司”（Microsoft Frontier Company）的新部门，核心使命是协助企业客户构建并规模化部署AI系统，同时确保其数据安全及知识产权受到严密保护。
“微软先锋公司”将整合行业专长、变革管理经验以及顶尖的AI工程支持能力，旨在设计并优化与可衡量业务成果紧密挂钩的AI系统。
该部门的目标客户群体，是那些正从AI试点项目阶段迈向大规模商业化部署的企业。
客户将能够运用自身的专有数据、业务流程及核心专长来构建AI工具，而无需担忧丧失对其竞争情报的控制权。
“微软先锋公司”将为各类特定应用场景提供多元化的AI模型支持，这其中涵盖了来自OpenAI、Anthropic、微软AI以及开源提供商和专业行业模型。
此外，该部门还将与埃森哲（Accenture）、凯捷（Capgemini）、安永（EY）、毕马威（KPMG）以及普华永道（PwC）等全球知名合作伙伴紧密协作，共同为跨市场、跨行业的客户提供全面支持。
目前，微软已与伦敦证券交易所集团（LSEG）、陆湖（Land O’Lakes）、联合利华（Unilever）和诺和诺德（Novo Nordisk）等企业成功开展了多项企业级AI项目合作。
以伦敦证券交易所集团（LSEG）为例，微软协助其将AI能力融入LSEG Workspace平台，使得金融专业人士能够针对复杂的金融内容进行高效提问与分析。
罗德里戈·凯德·利马（Rodrigo Kede Lima）已被任命为“微软先锋公司”的总裁。
利马先生拥有长达30年的行业经验，此前六年一直在微软领导美洲及亚洲地区的企业转型工作。
微软商业业务首席执行官贾德森·阿尔索夫（Judson Althoff）表示，新部门的核心目标是帮助客户提升AI投资回报，同时确保其核心情报与知识产权的安全无虞。
