# ️ 长期居菲者回国遭管控：护照失效，三年内限制出境

⚠️ 长期居菲者回国遭管控：护照失效，三年内限制出境
近日，一名长期在菲律宾经营餐饮的山东威海籍男子，因家中事务返华后，意外被当地警方列入重点管控名单。尽管该男子携营业执照、经营流水等材料力证合法经营，仍未能避免后续调查。
据该男子自述，返乡次日即被传唤问询，警方重点核查其境外经历与资金往来。随后，他被告知因长期停留在涉电信网络诈骗高风险国家，护照被现场剪角作废，并被限制出境三年，相关信息已录入出入境管理系统。
该事件凸显了中国持续打击跨境电信网络诈骗的力度。对于曾长期在涉诈高风险国家或地区工作、经商的人员，回国后可能面临更为严格的身份核验与风险评估。一旦被纳入重点管控名单，出入境及证件办理等均可能受到限制，具体措施将视个案情况及法规执行。
