# 缅北电诈集团终结：中国跨国打击行动大捷

🔥 缅北电诈集团终结：中国跨国打击行动大捷
中国公安部牵头的“打击缅北涉我犯罪专项工作组”近期取得了重大突破，彻底瓦解了盘踞缅北、长期危害中国公民的电信诈骗犯罪集团，标志着一场国家级反诈行动的大获全胜。
2023年以来，缅北地区成为中国电信网络诈骗案件的主要源头，以果敢“四大家族”为首的犯罪集团操控大规模电诈园区，严重侵犯了中国人民的生命财产安全。为应对这一严峻形势，公安部于2023年7月遵照党中央部署，联合多部门成立专项工作组，展开跨国打击行动。
工作组集中资源、创新战法，仅用两个月便推动缅方清剿大批电诈园区，并移交多批涉诈嫌疑人。然而，果敢“四大家族”负隅顽抗，于2023年10月20日制造了震惊中外的“10·20”惨案，导致中国籍涉诈人员4死4伤。为彻查此案，公安部迅速组织数万精锐警力赶赴边境，克服重重困难，高效开展排查核实与证据收集。
在战火频仍、地雷密布的缅北地区，专案组成员冒着生命危险，多次深入犯罪现场勘察取证，查获被害人遗体、骨灰、枪弹痕迹等关键证物。他们足迹遍布全国20余省份，查清包括“10·20”惨案在内的多起命案及上万起电诈案件。在强大的证据链面前，明家犯罪集团核心成员于2023年11月被公开通缉。
最终，在缅甸各方配合下，相关犯罪头目或被抓获移交，或畏罪自杀。2024年1月，随着白所成等10名电诈头目被押解回国，为害多年的缅北“四大家族”彻底覆灭。截至目前，缅方已累计向中方移交涉诈犯罪嫌疑人5.8万名，缅北规模化电诈园区已基本实现清零，所有相关犯罪分子均依法受到严惩。此次行动彰显了中国打击跨境犯罪、维护公民安全的坚定决心与强大实力。
