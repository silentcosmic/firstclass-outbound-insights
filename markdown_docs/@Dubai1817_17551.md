# 马来西亚槟城夜场冲突：疑涉华人暴力事件

🚨 马来西亚槟城夜场冲突：疑涉华人暴力事件
市场消息指出，马来西亚槟城卡帕尔·辛格路（Karpal Singh Drive）一处名为Tsin Livehouse的娱乐场所近日发生一起肢体冲突事件。据称，两波涉嫌与海外灰色产业相关的中国籍从业人员爆发激烈冲突，期间有投掷物品并出现人身攻击行为。
此次事件引发了当地对治安环境的担忧，并对海外华人社群的形象造成负面影响。
