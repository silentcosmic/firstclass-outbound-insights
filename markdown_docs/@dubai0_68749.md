# 泰国成立特别行动部队，重拳打击边境犯罪

🚨 泰国成立特别行动部队，重拳打击边境犯罪
6月30日，泰国军方宣布正式组建“特别行动部队”（Special Task Force），旨在全面加强对边境地区电信网络诈骗、科技犯罪及人口贩卖等跨境违法犯罪活动的打击力度。
据了解，这支特别行动部队将重点部署在与缅甸接壤的“西线”、与柬埔寨接壤的“东线”以及相关海上区域，通过强化边境巡逻、深化情报共享和开展联合执法行动，进一步遏制跨境犯罪活动的蔓延。
泰国军方表示，成立特别行动部队是提升边境安全管理和打击跨国犯罪的重要举措，未来将持续加强各相关单位的协同配合，维护国家安全和社会稳定。
