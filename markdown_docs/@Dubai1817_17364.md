# 敏昂莱：缅中友谊深化，全球须合力打击电诈

🚨 敏昂莱：缅中友谊深化，全球须合力打击电诈
缅甸国家管理委员会主席敏昂莱在接受央视高端访谈时表示，缅中两国作为山水相连的友好邻邦，传统友谊源远流长，双方始终保持密切合作与相互支持。缅方高度重视发展对华关系，将继续推动两国在经济、贸易、安全及人文交流等领域深化合作，以实现互利共赢与共同发展。
谈及国际社会普遍关注的电信网络诈骗问题，敏昂莱指出，电信诈骗已成为全球最棘手、最复杂的跨国犯罪挑战之一。其犯罪网络跨越多个国家和地区，涉及人员流动、资金转移及网络技术等多个环节，单靠任何一个国家都难以彻底解决。
敏昂莱强调，打击电信诈骗不仅是缅甸面临的重要任务，更是国际社会共同面对的安全挑战。他呼吁各国执法部门应进一步加强情报共享、联合执法和跨境协作，共同打击电信诈骗、网络赌博、人口贩运及相关跨国犯罪活动，切实维护地区安全稳定和人民群众合法权益。缅甸愿继续深化与中国及周边国家的执法合作，共同推进跨境犯罪治理，为地区和平稳定和经济发展创造更安全的环境。
