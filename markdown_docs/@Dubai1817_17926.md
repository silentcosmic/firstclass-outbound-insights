# ️ 缅甸边境警报：诈骗集团疑南迁至勐奈镇

⚠️ 缅甸边境警报：诈骗集团疑南迁至勐奈镇
近期，缅甸掸邦南部的勐奈镇引起当地居民高度关注。大量华裔面孔频繁现身，大宗采购建材及食品等物资，且交易时从不议价，这引发了对北部诈骗集团（俗称“KK园区”）势力可能南迁的强烈担忧。
据市场消息，这些异常采购的物资并未运往中国方向，而是转向了泰缅边境由SSS组织控制的区域。此前，缅甸军方曾在勐班镇逮捕56名涉嫌网络诈骗人员，其中52名为中国公民；佤联军也曾移交中方192名中国籍诈骗人员，显示出缅甸境内打击电诈的持续行动。
自2023年“1027行动”以来，缅北地区的诈骗集团大量向掸邦南部及泰缅、老缅边境地区转移已是公开的秘密。鉴于此，当地民众呼吁武装力量及时介入清查，以避免区域再次沦为诈骗活动的温床。
