# Uber巨资收购Delivery Hero：全球外卖市场重塑

⚡ Uber巨资收购Delivery Hero：全球外卖市场重塑
全球出行巨头Uber近日正式宣布，已达成协议收购德国外卖巨头Delivery Hero SE。这笔交易对Delivery Hero的估值高达129亿欧元，标志着全球食品配送行业迄今最大规模的整合案之一。
根据协议，Uber将以每股41.50欧元的现金价格完成此次收购，证实了市场此前的传闻。此举将进一步巩固Uber在全球配送市场的领先地位，Delivery Hero旗下包括阿联酋配送平台talabat在内的多个知名品牌也将纳入Uber生态。
