# 跨境诈骗案开审：49名被告当庭否认控罪

🚨 跨境诈骗案开审：49名被告当庭否认控罪
7月7日，马来西亚纳闽法院首次开庭审理一起备受关注的跨境网络诈骗韩国人案件，共有57人涉案。庭审中，在控方修改部分控状后，包括48名中国籍被告及1名马来西亚籍被告在内的49人当庭否认全部控罪，该案件已正式进入司法审理程序。
