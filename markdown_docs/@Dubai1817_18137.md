# ️ 金边街头惊现异象：男子路边诡异举动引关注

⚠️ 金边街头惊现异象：男子路边诡异举动引关注
近日，有市场消息指出，在柬埔寨金边莫尼旺大道旁，一名男子被目击做出异常举动。据现场观察者描述，该男子蹲在路边，手持一根拼接而成的长吸管，并使用打火机对着地面下水道吸食不明物质，其行为全程旁若无人。
此举引发了路人的困惑与关注，其行为模式与常理相悖，具体动机与所吸食物质尚不明确。该事件凸显了城市公共空间中可能存在的社会管理挑战。
