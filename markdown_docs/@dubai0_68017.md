# 柬埔寨打击网诈：千余人被捕，房东遭重罚近2亿瑞尔

🚨 柬埔寨打击网诈：千余人被捕，房东遭重罚近2亿瑞尔
柬埔寨内政部近日通报，在6月上半月的全国性执法行动中，共逮捕1332名涉嫌高科技网络诈骗人员，涉及16个不同国籍。此次行动覆盖全国12个省市的78处目标地点，显示出当局打击网络犯罪的决心。
在被捕人员中，已有133人移交移民总局处理，87人已送交法院面临起诉，其余1112人仍在接受深入的法律调查。行动中，警方还成功捣毁了一个由16人组成的反侦查团伙。该团伙专门为诈骗主谋提供情报，充当“眼线”，搜集警方行动信息。
此外，因未能遵守关于集中居住地安全管理的“03号法令”，多个涉案场所的业主及管理者被依法处以总计1.97亿瑞尔（约合4.8万美元）的巨额罚款。
