# 华为携手泰国CP All，创新引领7-Eleven可穿戴支付体验

华为与泰国零售巨头CP All正联手将创新的可穿戴设备忠诚度计划与支付功能引入其在泰国的7-Eleven连锁店。
作为泰国7-Eleven的运营方，CP All在华为2026年开发者大会期间，于华为东莞园区与华为签署了战略合作备忘录，此举标志着双方在数字化零售领域的深度融合。
这项合作的核心在于将7-Eleven的会员码以及TrueMoney的支付码无缝集成至华为的可穿戴设备中。
消费者凭借这一创新功能，可在店内通过单一的可穿戴设备界面，便捷地完成会员积分累积与支付交易。
双方企业计划持续优化该服务，并积极探索更多日常场景中的技术应用潜力。
此外，此次合作还将借助华为的快应用服务，为在泰国的中国游客提供更紧密的互联体验，覆盖购物、支付及忠诚度奖励等多个环节。
展望未来，此次战略合作的范畴有望进一步拓展至零售分销领域，包括华为设备可能通过7-Eleven在泰国的广泛门店网络进行销售。
