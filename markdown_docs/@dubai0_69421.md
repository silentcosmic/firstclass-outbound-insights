# ️ 金边支付平台CoolCash通知客户紧急提现销户

⚠️ 金边支付平台CoolCash通知客户紧急提现销户
近日，天旭国际科技有限公司旗下支付平台CoolCash（小易支付）通过其官方脸书发布紧急通知，敦促所有代理商、商户及客户尽快提取账户余额并办理销户手续。
根据通知，相关方需携带有效身份证明文件，前往公司位于金边市桑园区万景岗1分区第2村310路48-50号（G楼）的总部办理。办理时间为每周工作日（周一至周五）上午九点至下午五点。
CoolCash方面强调，此通知自发布之日起生效，直至公司另行通知。客户如有疑问，可拨打联系电话023 988 777 或 081 600 777进行咨询。此举或预示着该平台在柬埔寨市场的运营策略发生重大调整。
