# Lightstorm、微软、新加坡电信与塔塔通信：携手打造连接印度、马来西亚、新加坡的I-2SEA海底电缆系统

聚焦于新加坡的Lightstorm、微软、新加坡电信（Singtel）以及印度塔塔通信（Tata Communications）将携手共建一条连接印度、马来西亚与新加坡的新型海底电缆系统。该财团合作伙伴预计此项目将于2029年第四季度完工。
Lightstorm作为一家专注于人工智能连接的平台，在本周四发布的一份声明中指出，这条名为I-2SEA的电缆系统将直接连接印度东海岸城市海得拉巴和钦奈，通往新加坡和吉隆坡。该系统预计从新加坡延伸至印度马奇利帕特南，全长约3600公里，并在钦奈南部设立第二个登陆点。
Lightstorm的客户将能够选择接入该公司长达3万公里的陆地网络，从而进一步延伸至海得拉巴、孟买以及全国80多个数据中心。
这条电缆系统能够为超大规模数据中心运营商、GPU基础设施提供商以及在印度-东南亚走廊运行AI训练和推理工作负载的企业提供服务。与Lightstorm在印度本土的低延迟回程网络相结合，I-2SEA将为新加坡、马来西亚至海得拉巴这一区域内对AI工作负载而言最具战略重要性的城市对，提供与任何现有电缆系统匹敌的卓越传输能力。
Lightstorm集团首席执行官兼董事总经理Amajit Gupta表示，公司现在可以将其SmartNet AI Fabric平台自然地延伸至海底领域，通过一个专为AI基础设施性能和规模需求而设计的、端到端的系统，连接印度、马来西亚和新加坡的AI核心区域。
日本电气株式会社（NEC Corporation）将担任该系统的供应商，这得益于其在全球范围内成功部署超过45万公里海底电缆系统的丰富经验。
总部位于新加坡的ASEAN Cableship公司将作为海洋安装合作伙伴。该系统的一大特点是采用深度埋设策略，在埋设区域目标深度达三米，以提供更好的保护和高可用性，同时在印度的两个登陆点均采用运营商中立的登陆基础设施。
印度塔塔通信此前已宣布，将投资1.52亿美元用于建设连接印度AI枢纽与新加坡的海底电缆项目。
