# 迪拜35分钟极速反击：路透社再陷虚假报道风波

🚨 迪拜35分钟极速反击：路透社再陷虚假报道风波
近日，国际知名通讯社路透社再次因对阿联酋发布不实报道而引发争议，迪拜官方以创纪录的35分钟速度迅速辟谣并发出严厉警告。事件发生于2026年7月16日晚，路透社向全球订阅用户发布快讯，援引“目击者”称迪拜市中心商业湾（Business Bay）区域听到巨大声响。
然而，迪拜政府媒体办公室（GDMO）在短短35分钟内即发布官方声明，否认市中心发生“爆炸”的报道，并敦促公众与媒体依赖官方信息源。随后，路透社更新报道转述了迪拜的否认，并最终于当晚22:30发出正式撤销通知。在此期间，GDMO态度强硬，警告将依据阿联酋本地及联邦法律，对发布虚假新闻、未经核实信息及传播地缘谣言的外国媒体机构采取法律行动，并已开始依法封禁违规的社交媒体账号。
此次迪拜官方的迅速反应，打破了以往相对滞后的行政流程，彰显了在当前地缘政治高压下，阿联酋对“信息战”和“金融做空谣言”采取的“高关注、零容忍”立场。鉴于此前阿联酋油轮遭袭以及周边地区持续的紧张局势，作为中东金融、航空与航运枢纽的迪拜，对任何可能恶化安全局势的传言都表现出极度敏感。此事件预示着阿联酋媒体监管机构可能大幅收紧对外媒的审核，并加强对境内信谣传谣行为的打击。
值得注意的是，这已是路透社在一个月内第二次针对阿联酋发布错误报道。就在此次“爆炸乌龙”前一个月，即2026年6月13日，路透社曾援引“匿名知情人士”报道称，阿联酋已与伊朗达成秘密财务安排，将解锁100亿至200亿美元被冻结资产作为“战争损失补偿”。对此，阿联酋外交部亲自出面，发表极其强硬的严正声明，公开指责该报道包含“毫无根据的指控”，并强调阿联酋绝无规避国际制裁或进行暗箱财务外交的行为。
两次事件表明，在复杂敏感的地缘背景下，即使是国际知名通讯社的失实报道，也可能迅速转化为对城市安全信誉的冲击。迪拜此次果断亮剑，不仅展现了其包容开放的姿态，更划清了守护城市秩序与公共信心的坚定底线。
