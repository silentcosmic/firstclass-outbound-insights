# 中老边境联手：严打网络诈骗深化执法合作

🚨 中老边境联手：严打网络诈骗深化执法合作
近日，中国与老挝两国公安部门举行了2026年第二季度联席会议，旨在进一步加强边境执法合作，尤其针对网络诈骗展开严厉打击。
此次会议由老挝琅南塔省孟新县公安指挥部与中国云南省西双版纳傣族自治州勐腊县边境管理大队共同召开。双方就第二季度边境安全形势、社会治安管理、打击跨境违法犯罪以及边境管控等工作情况进行了深入交流。
会议达成共识，将深化下一阶段的协作机制，共同维护边境地区的安全与稳定，持续高压打击各类跨境违法犯罪活动，特别是网络诈骗。
