# 沙特阿美直升机坠毁：14名公民遇难

🚨 沙特阿美直升机坠毁：14名公民遇难
沙特国家通讯社（SPA）近日援引该国能源部消息证实，沙特阿美公司（Saudi Aramco）一架直升机在东部城市拉斯塔努拉附近坠毁，机上14名沙特籍公民不幸全部遇难。
据官方消息人士透露，事故发生于当地时间6月28日（周日）上午约6时。目前坠机原因仍在调查中，沙特能源部正与相关部门紧密协作，已启动全面调查程序。
