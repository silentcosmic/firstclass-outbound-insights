# 阿布扎比狩猎展购枪：内政部公布持枪许可细则

📣 阿布扎比狩猎展购枪：内政部公布持枪许可细则
阿布扎比国际狩猎与马术展览会（ADIHEX）将于8月28日至9月6日期间举行。期间，阿联酋内政部宣布，有意在展会期间购买枪支并办理持枪许可的人士，可通过内政部官网或移动应用程序提交申请。
内政部明确了持枪许可的申请条件：阿联酋公民需年满21周岁。
对于非阿联酋公民，申请流程则更为严格，须提供由其本国相关部门出具的无异议证明，并经其本国驻阿联酋大使馆认证后方可提交申请。
