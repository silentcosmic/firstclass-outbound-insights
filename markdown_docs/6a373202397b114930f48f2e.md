# Roblox全球推出分龄账户体系，强化未成年用户保护与家长控制功能

全球领先的在线游戏平台Roblox宣布，已在全球范围内正式推出其全新分龄账户体系，其中包括“Roblox儿童账户”（Roblox Kids）和“Roblox精选账户”（Roblox Select）。此举旨在根据用户的年龄自动分配至相应的账户类型，并提供定制化的游戏、聊天及其他功能访问权限，以确保不同年龄段用户的体验适宜性与安全性。
此项创新举措此前已在澳大利亚、印度尼西亚、荷兰和新西兰等市场进行了初步试点，并取得了积极反馈。
根据Roblox的设置，“Roblox儿童账户”将执行最为严格的标准配置，例如默认禁用聊天功能。而“Roblox精选账户”则面向年龄稍长的儿童和青少年群体，提供适度更成熟的内容访问权限，并逐步解锁聊天功能。对于16岁及以上用户，标准Roblox账户则要求通过身份验证或面部年龄识别进行严格的年龄核查，以确保用户身份的准确性。
所有被纳入“儿童账户”和“精选账户”目录的游戏，都将在标准内容成熟度评级的基础上，接受额外的严格审核。平台对开发者提出了更高要求：必须完成身份验证，通过双重认证机制确保其账户安全，并在为年轻用户发布游戏前支付相应的订阅或发布费用。此举旨在确保内容的质量与安全性。
明确规定，凡是包含社交聚会、自由创作绘画功能或敏感内容的游戏，均不允许发布。此外，在发布初期，“Roblox时刻”（Roblox Moments）功能和社交媒体链接将不对“儿童账户”或“精选账户”开放。相关链接仅供那些已通过年龄验证的16岁及以上用户查看。
平台强调，无论用户自行声明的年龄如何，若未能完成必要的年龄核查，其将无法使用聊天功能。然而，用户仍可根据其自行声明的年龄被分配至“儿童账户”或“精选账户”类别，从而获得相应的受限体验。
Roblox致力于赋能家长。如果家长将其账户与子女账户关联，便可访问一套全面的家长控制功能。对于5至8岁的儿童用户，家长可以启用游戏内聊天功能，但其范围将被严格限制在同一年龄段用户或“信任朋友”之间。与此同时，平台内置的文本过滤和针对儿童剥削行为的监控机制将始终保持激活状态，提供持续的安全保障。
在孩子满13岁之前，家长拥有管理其好友列表以及设置屏幕使用时间和消费上限的全面权限。当孩子达到16岁之前，家长可以继续调整聊天设置，并对特定游戏进行批准或阻止操作。对于16至18岁的青少年用户，尽管直接控制权限有所减少，但家长仍可通过关于消费、屏幕使用时间和好友列表的通知，保持对孩子在线活动的可见性，实现更透明的监督。
Roblox全球家长委员会成员Anuradha Rao对此表示，新推出的账户类型为家长提供了更多参与孩子在线活动的机会，并能便捷获取丰富的在线安全资源。她强调，这些平台工具若能与家庭内部的积极亲子互动相结合，将有助于所有家庭成员共同培养健康、负责任的科技使用习惯。
