# 深度解析：估值110亿美元的金融科技巨头如何助力初创企业从起步即布局全球

时至深夜，一家羽翼未丰的AI创业公司，或许已在全球多个角落同步运作：服务远在旧金山的客户，采购首尔的技术服务，并与拉各斯的合作伙伴敲定合约。即便尚未配备首位全职财务人员，其业务版图却已横跨多重市场、币种与监管区域，展现出惊人的全球化速率。
人工智能的崛起正使这类创业路径成为现实。过去，企业需构建庞大的全球化市场、运营与客服团队方能完成的工作，如今借助智能体已能承担相当部分。新一代的创新企业不再受限于“先立足本土，再逐步出海”的传统扩张模式，而是能够从创立伊始，便直接面向全球客户与供应商。
然而，当AI的迭代速度达到毫秒级时，多数企业的金融基础设施却未能同步进化。这正是为何全球顶级资本正积极布局，重塑金融基础设施以适应AI经济新范式。
2025年6月，全球金融基础设施平台Airwallex空中云汇宣布完成3.2亿美元H轮融资，估值攀升至110亿美元，相较半年前的80亿美元显著增长。本轮融资由老股东Addition领投，并获得Baillie Gifford、T. Rowe Price、Amex Ventures等现有及新增投资者的跟投。
这笔融资传递出一个明确的市场信号：AI经济亟需一套与之匹配的金融操作系统。空中云汇作为先行者，不仅拥有坚实的全球合规基石，更在积极构建这套未来系统。
以往，AI更多扮演的是“建议者”角色，辅助人类决策应购买何物、支付给谁、选择何种汇款路径。而现在，AI智能体正加速成为“执行者”，无论是自动采购算力、订阅SaaS服务，还是向海外供应商完成款项支付，越来越多的商业操作正由AI自主发起。
国际市场研究机构Gartner的报告指出，2024年至2025年间，AI智能体辅助支付已从概念验证阶段迅速迈向实际部署。2025年9月，OpenAI正式在其ChatGPT中支持AI智能体辅助支付功能，Stripe亦推出了专为AI智能体设计的Agentic Commerce Protocol。然而，核心症结在于，AI的决策速度达到毫秒级，其所依赖的金融系统却仍旧遵循人工审批与T+2（交易日+2天）的传统结算周期。
对于具有海外采购需求的B端AI企业而言，这种速度上的矛盾尤为突出。试想一家快速扩张的中国AI公司，需同时向美国、欧洲、东南亚的多家云服务商和数据供应商付款，涉及美元、欧元、新加坡元等多种货币。每一笔跨境转账都可能触及不同国家的合规要求，而汇率波动、中间行手续费以及结算周期的不确定性，往往令财务团队疲于奔命。
更甚的是，随着企业规模扩大，簿记、报销、税务、银行系统各自为政的局面将以指数级放大管理成本。一位资深财务总监（CFO），曾服务多家出海AI企业，他描绘了一个常见场景：公司在全球十几个国家拥有供应商，每月仅对账一项就需耗费财务团队近一周时间，还要随时应对各地监管机构对资金流向的严格核查。
这意味着，财务团队最宝贵的时间与精力，正被消耗在原本可自动化的重复性劳动上。更深层次的风险在于，滞后的对账周期导致管理层长期处于财务盲区。决策所依据的数据往往已过时数日，一旦遭遇汇率剧烈波动或监管突击检查，企业几乎没有足够的反应窗口。
当然，一个再聪明的AI模型，或许能理解发票内容、识别异常并给出支付建议，但它无法凭空获得金融牌照，也无法替代银行合作关系及遍布全球的本地支付网络。唯有当这些核心金融能力被统一打通、整合，并以AI可调用的方式开放，AI的分析与判断才能真正转化为财务运营效率的跃升。
这也使得金融基础设施在AI时代被重新审视。过去，跨境金融平台主要关注企业如何更高效地进行收付款。如今，随着AI逐步融入财务工作流，企业不仅需要支付通道，更需要一套机制，允许AI在明确的授权范围内调用金融工具，同时确保整个过程符合内部制度与监管要求，并留下清晰可追溯的记录。
对于新一代AI公司而言，这套能力显得尤为关键。AI固然能让企业以更精简的团队服务全球客户，却并未消除跨境金融固有的复杂性。如果账户、支付与财务管理依然分散在不同的系统之中，那么所谓的“从Day 0走向全球”的宏图，极易在最基础的资金运营环节受阻。
Gartner预测，到2026年，33%的企业软件将集成智能体AI（Agentic AI）能力，其中财务职能将是渗透最深的领域之一。更关键的是，当AI智能体经济真正成熟时，谁的金融底座能支撑“代理自主交易”，谁就将掌握下一阶段的市场入场券。
空中云汇自2015年成立以来，历经十年布局，已在全球范围内铺设超过85张监管牌照，并整合了160个本地支付网络。据其披露，目前空中云汇的年化交易量已达2870亿美元，其中93%的交易可实现当日结算，业务覆盖全球200多个国家和地区。
这套历时十年打造的基础设施，在AI时代正被重新估价：其严密的合规网络与广泛的本地通道，恰好契合智能体经济对底层管道的需求。本次H轮融资，正是空中云汇在这套管道之上加盖“智能层”的战略举措，核心载体是其推出的两款全新产品：T:0与Airi。
重塑财务流程：T:0与Airi的创新实践
当前大多数企业的财务系统，往往是一个由分散工具拼凑而成的混合体：簿记、报销、税务、银行系统各自为政，数据在它们之间流转时不断产生摩擦。这导致财务团队大量精力消耗在重复性的核查与录入工作中。
T:0相当于一个自主运营的金融部门，其核心理念是从第一性原理出发重新设计系统：将簿记、预测、税务、合规、报告与对账整合进一个持续运行的统一平台，使财务职能如同一台永不停歇的引擎持续运转，从而将财务团队从繁琐的管理工作中解放出来，专注于更具战略意义的决策。
对于从零起步的创始人而言，T:0的价值在于，它意味着从企业创立的第一天起，便拥有一个内置合规逻辑的CFO级账簿，无需经历系统迁移的阵痛。这是因为T:0可直接调用空中云汇已有的全球支付通道、多币种账户与合规框架，使财务数据与资金流转在同一套基础设施内完成闭环。目前T:0正处于私测阶段，预计数周内将更广泛地推出市场。
Airi则旨在解决一个更现实的命题：当AI智能体开始替代人类执行购买行为时，商业世界所需远不止一个更智能的结账页面。空中云汇将其定义为“智能体商业”。
行业内不乏智能体布局的先行案例。2025年，OpenAI与Stripe联合推出了Agentic Commerce Protocol（ACP），为AI智能体执行购买行为提供标准化协议框架；Mastercard也在同年发布了面向智能体商业的支付解决方案。然而，这些方案存在一些共同局限，其本质是在传统卡支付通道上叠加一层AI界面，并未从根本上解决AI智能体在全球范围内安全调用和转移资金的底层问题。
Airi正是为填补这一空白而生。它从底层重新构建了一套完整的代理钱包能力：AI智能体可以持有专属的代理卡，在设定的支出限额内自主发起交易，并根据权限层级调用不同币种的余额。整个过程在受严格监管的框架内运行，使AI智能体真正具备代表人类完成跨境交易的能力。
目前，Airi已实现“一键结账”功能。消费者完成身份绑定后，AI智能体可在授权范围内直接发起支付，无需每次手动填写信息。早期测试数据显示，商家转化率因此提升了14%。展望未来，Airi将进化为完整的智能体钱包基础设施，配合空中云汇更广泛的智能体商业套件，打通从发现需求到完成结算的完整资金链路。
如果说T:0解决的是“钱如何管理”的问题，那么Airi则聚焦于“钱如何使用”的场景——前者面向企业财务团队，后者则服务于商家与AI开发者。两者共同依托空中云汇已建立的全球合规网络，构成了一套协议无关、天然全球化的金融操作系统。
在一个AI智能体可能同时运行于OpenAI、Anthropic、Google等不同生态的世界里，不被任何单一协议绑定，本身即构成一种重要的战略优势。
金融基础设施的护城河与战略机遇
空中云汇此次融资，可被视为行业新窗口期开启的明确信号。金融基础设施的AI化重构，正成为下一个真正的竞争主战场。这场较量的核心，已不再单纯比拼模型能力，而是谁的底层管道能真正驱动AI高效运转。
该赛道存在一道独特的“护城河”：监管资质与本地支付网络。一张支付牌照的申请周期往往以年计，本地清算网络的接入需要逐一谈判，而合规体系的建立更需要在每个市场进行长期投入，这些几乎无法在短时间内仅靠资金堆积完成。
空中云汇用十年时间积累的超过85张全球牌照和160多个本地网络，在AI时代演变为一种稀缺资产，时间壁垒成为其核心价值之一。这也是本轮融资能够吸引Amex Ventures入局的关键因素：作为传统金融巨头的战略投资部门，美国运通的投资是对这套合规基础设施价值的有力背书。
更值得关注的是，这笔融资是在空中云汇已经实现盈利的背景下完成的。根据空中云汇官方数据，其年化收入已达13亿美元，经常性收入（ARR）于2025年第四季度转正。全球超过67.5万家企业直接或通过其平台管理跨境金融运营。这些数字证明，其商业模式已成功跑通。此次H轮融资是在财务状况健康的前提下，主动加速进入一个新的战略发展阶段。
在智能体AI重塑财务职能的进程中，真正能够脱颖而出的，将是那些同时具备全球合规能力与AI原生架构的平台。空中云汇恰好集这两种条件于一身：一方面是其十年沉淀而来的合规基础设施，另一方面则是从第一性原理出发，为智能体经济重新设计的产品架构。两者叠加，构成了它在此行业窗口期中的独特优势。
AI时代已不再有严格意义上的本土市场。当AI智能体开始跨越国境自主交易时，企业所需的金融底座，必须从一开始就是全球化的。那些目前仍依赖传统银行账户、手动审批流程来支撑海外业务的公司，正在试图用昨天的基础设施去追赶明天的速度。当智能体经济的浪潮真正到来时，留给企业进行系统迁移和重建的时间窗口，可能远比大多数人预期的要短得多。
