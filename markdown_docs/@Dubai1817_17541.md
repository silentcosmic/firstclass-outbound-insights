# 马来西亚柔佛收紧外籍居留管理：多名人员被查处

🚨 马来西亚柔佛收紧外籍居留管理：多名人员被查处
近期，马来西亚柔佛州新山市开展了一系列大规模街头联合执法行动。据市场消息，执法部门在主要街道设立检查点，对外籍人员的身份、护照及签证等有效证件进行了严格核查。
行动中，发现部分外籍人士存在签证逾期、证件不齐或手续不完整等违规情况，相关人员已被依法带走接受进一步调查处理。当地执法部门强调，此举旨在强化对外籍人员的合法居留管理，维护社会治安与出入境秩序。
