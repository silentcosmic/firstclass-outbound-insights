# 老挝丰沙里省破获网络诈骗案：5名嫌犯落网

🚨 老挝丰沙里省破获网络诈骗案：5名嫌犯落网
近日，老挝丰沙里省警方成功侦破一起网络诈骗案件，将5名涉案嫌疑人（包括1名女性）移交省公安机关，进行深入调查和依法处理。
据了解，本泰县警方联合人民检察院于7月9日对Pho Thong村一处出租屋进行突击检查。此前，执法部门已发现该地点涉嫌从事网络诈骗活动。行动中，警方当场抓获5名老挝籍嫌疑人，并查获了一批涉嫌用于诈骗的作案工具，包括5台台式电脑、2台笔记本电脑、13部iPhone手机、5张老挝身份证、1辆电动自行车、1台游戏机、2张ATM银行卡及4本银行存折。
此次案件是老挝近期侦破的最新一起网络诈骗案件。目前，老挝执法部门正持续加大力度，打击境内高科技犯罪和网络诈骗团伙，以遏制此类违法犯罪活动蔓延。
