# 越南Masan高科技材料扩储1.15亿吨钨矿，剑指AI半导体供应链核心地位

越南Masan集团旗下子公司Masan高科技材料（MSR）正积极拓展其钨资源基础，旨在为全球人工智能（AI）、半导体及国防工业提供关键原材料保障。此次潜在的储备增量预计将达到1.15亿吨钨多金属矿石。
MSR周一发布声明指出，这笔新增储量足以支撑其采矿与加工业务额外运行20至30年。新探明的储备位于其Nui Phao矿区周边的勘探区域。
根据越南国家矿产总体规划，Nui Phao矿区扩展区域的资源量估计约为5519万吨，而Nui Chiem矿区则估计约有6000万吨。
该公司同时计划将氧化钨（WO₃）的产能提升至8000吨以上，以满足日益增长的客户需求。MSR表示，鉴于其现有的仲钨酸铵（APT）产能基础设施，此次扩产无需投入大量资本。
MSR董事长Danny Le表示，公司已在中国境外构建了一个涵盖采矿、深度加工及先进材料供应的领先一体化钨平台，服务领域包括AI、半导体、航空航天、能源及国防等关键产业。他强调，随着规模化、深度加工的非中国钨供应日益稀缺，此次进展将进一步巩固公司长期的战略地位。
MSR特别指出，此次扩张充分利用了现有基础设施和加工能力，避免了新建项目的巨额投入，从而支持了一条更为资本高效的增长路径。
