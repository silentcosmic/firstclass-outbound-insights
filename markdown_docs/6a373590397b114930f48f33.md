# 新加坡金管局局长警示：全球经济韧性背后，AI投资暗藏潜在风险

全球经济增长与股市估值正日益紧密地与人工智能（AI）的发展深度绑定。新加坡金融管理局局长周三指出，尽管全球经济在一系列重大冲击中展现出显著韧性，但这种对AI的依赖也同步孕育了新的潜在脆弱性。
在上海陆家嘴论坛上，新加坡金管局董事总经理孟文能（Chia Der Jiun）发出了警示：一旦市场对AI的预期进行重新评估，可能导致全球经济增长放缓或引发市场剧烈波动。孟文能强调：“全球经济增长和股票市场估值高度依赖于人工智能。如果对AI投资的假设进行重新评估，经济增长和市场表现可能面临急剧放缓甚至逆转的风险。”
他观察到，自2024年以来，AI领域的投资为全球经济注入了一股积极的需求动能，有力支撑了增长，有效缓解了地缘政治紧张、贸易中断及能源供应冲击带来的负面影响。然而，AI投资的实际回报仍存在不确定性，而能源和先进半导体芯片的成本却持续攀升。
孟文能还特别指出了AI技术普及过程中所伴随的长期隐忧，例如是否存在充足的安全保障机制，以及这项技术的经济红利能否得到广泛普惠。他进一步阐述：“中期来看，AI的应用充满了诸多不确定性。如果安全保障不足，并且其经济效益未能广泛共享，风险便会不断累积。”
孟文能发表此番言论之际，正值全球政策制定者与投资者之间，就大规模投入AI基础设施、数据中心及算力资源能否产生足够的回报以支撑当前高估值，展开日益激烈的探讨。
宏观视角下，孟文能回顾道，过去六年全球经济已历经五大供需冲击：包括新冠疫情、俄乌冲突、2025年的关税壁垒、AI投资热潮以及本年度的能源供应危机。
尽管面临重重扰动，全球经济依然展现出韧性：2023年起增长逐步复苏，2024年通胀水平趋近各国央行目标，而全球贸易在2025年也持续保持扩张态势。
孟文能警示，在历经数年通过财政支出应对连续冲击后，各国政府应对未来危机的财政空间已大幅缩减。当前，许多发达经济体的债务占GDP比重已逾100%，而平均财政赤字仍维持在GDP的5%以上。
他同时预警，尽管霍尔木兹海峡的重开取得进展，全球能源市场的不确定性依然挥之不去。若出现长期性中断，可能导致库存下降和成品油供应趋紧，从而在今年晚些时候产生更为显著的影响。
为增强经济韧性，孟文能敦促各国经济体坚守审慎的财政和货币政策，深化本土资本市场建设，实现供应链多元化，并积极推行促进增长的结构性改革。
此外，他呼吁加强区域合作，并指出中国与东盟之间的贸易额已于2025年突破1万亿美元。他认为，两大区域间更紧密的投资流动和资本市场互联互通，将有助于在全球不确定性中支撑经济增长。
孟文能强调，国际货币基金组织（IMF）、金融稳定理事会（FSB）以及清迈倡议等多边和区域性机制，对于维护全球金融稳定依然至关重要。他总结道：“面对频繁的冲击、持续的不确定性以及某些领域国际合作承诺的削弱，我们必须采取行动，要求每个经济体都构建自身韧性，并充分认识到加强合作所能带来的巨大益处。”
新加坡方面也积极倡导东盟各国拥抱人工智能技术，同时警惕数据安全与主权风险的潜在陷阱。
