# 阿联酋油价七月或迎大幅下调

⛽ 阿联酋油价七月或迎大幅下调
随着全球油价自本月初以来每桶下跌逾20美元，阿联酋汽油价格在连续四个月上涨后，预计2026年7月将迎来显著下调。此前，自2月28日中东战争爆发以来，阿联酋零售燃油价格已累计上涨超过60%，其中6月环比涨幅约8%。
这一预期下调主要得益于6月全球原油市场的下行趋势。布伦特原油价格在6月第一周约为每桶95美元，随后受美伊和平协议及霍尔木兹海峡有望重新开放的预期影响，迅速跌至每桶74美元以下。整个6月，布伦特原油平均收盘价从5月的约106美元降至约71美元。
盛宝银行大宗商品策略主管奥勒·汉森指出，油价持续回落，布伦特原油已跌破每桶76美元。市场关注的焦点已从此前中断的供应转向可能很快恢复的供应。随着霍尔木兹海峡船运交通的逐步改善，交易员正密切关注大量等待运输的原油货物，这预示着市场可能迎来一轮供应激增。
瑞士宝盛银行经济与新一代研究主管诺伯特·吕克尔也表示，石油市场的迅速反弹令人惊讶。他认为中东地区的石油出口量已恢复至危机前水平的80%以上，这意味着市场已从供应短缺转为供应过剩，库存可能会比预期更早开始回补。
