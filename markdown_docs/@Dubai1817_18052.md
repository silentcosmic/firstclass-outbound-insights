# 马来西亚柔佛遣返150名非法外籍，缅甸籍占八成！

🚨 马来西亚柔佛遣返150名非法外籍，缅甸籍占八成！
马来西亚柔佛州移民局近日通报，北干那那移民扣留中心于7月第二周完成了一批大规模遣返行动，共计150名非法外籍人员被分批送返各自原籍国。
此次遣返人员构成显示，缅甸籍公民占据绝大多数，达到127人，约占总数的八成半。此外，被遣返人员还包括7名孟加拉国籍、7名越南籍、5名巴基斯坦籍、2名中国籍及2名尼泊尔籍人员。所有人员均已通过吉隆坡国际机场（KLIA第一、第二航站楼）以及柔佛士乃国际机场，按国籍分批搭乘航班离境。
柔佛州移民局强调，所有被遣返人员均已被列入马来西亚移民系统黑名单，今后将不得再次入境。当局重申，将持续加大执法力度，严厉打击非法居留、非法务工等违法行为，并坚定不移地推进非法移民遣返工作。
