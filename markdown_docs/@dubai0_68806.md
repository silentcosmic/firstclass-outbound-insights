# 马来西亚“壹汇环球”涉巨额跨境诈骗，超50亿！

🚨 马来西亚“壹汇环球”涉巨额跨境诈骗，超50亿！
近日，一起涉及马来西亚外汇投资项目“壹汇环球”（GHC）的跨境诈骗案浮出水面。据称，该项目由马来西亚籍人士Wong Khoay Shen（又名“黄董”）主导，自2017年前后向大量中国投资者推广高收益外汇理财业务，导致巨额经济损失。
多名受害者反映，“壹汇环球”以高额固定收益、外汇交易及团队奖励为诱饵，鼓励投资者持续追加投入并发展下线，迅速吸引了中国多省市的大量参与者。该平台主要运营地点位于马来西亚吉隆坡蒲种。
然而，自2021年前后，平台陆续出现提现困难、资金冻结等问题，最终导致众多投资者血本无归。初步估计，此案涉嫌诈骗金额已超过50亿元人民币，受害者人数约达3万人。
目前，受害者正积极搜集证据，呼吁有关国家和地区执法机关加强跨境合作，对涉案人员及资金流向展开调查，以期追究法律责任并追回损失。反诈机构提醒，对于宣称“稳赚不赔”、“高收益、低风险”或带有传销性质奖励机制的境外投资项目，公众务必保持高度警惕，并核实其是否具备合法金融牌照。
