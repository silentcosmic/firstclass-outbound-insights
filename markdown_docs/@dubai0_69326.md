# ️ 沙特对阿联酋转账受阻：企业贸易受扰

⚠️ 沙特对阿联酋转账受阻：企业贸易受扰
近期市场消息指出，沙特阿拉伯多家银行出现延迟或阻碍从沙特账户向阿联酋（特别是迪拜）收款人进行银行转账的情况。多家企业反映，此前正常到账的款项现在被无故扣留数日后退回，严重干扰了正常的贸易活动。
这一异常现象已迫使部分受影响企业寻求替代解决方案，例如通过巴林转道汇款，或采用成本更高的支付方式，显著增加了运营成本及复杂性。
对此，沙特阿拉伯中央银行否认对特定国家实施限制，并表示各商业银行仅在执行标准的、基于风险的合规审查措施。然而，实际操作中的支付延迟和退回，已引发市场对两国间跨境支付稳定性的普遍担忧。
