# 阿联酋航空首架777-300ERSF改装货机投入运营

⚡ 阿联酋航空首架777-300ERSF改装货机投入运营
阿联酋航空近日宣布，正式启用全球首架波音777-300ERSF客机改装货机，此举使其成为全球首家部署此类机型的综合性航空运营商。这标志着阿联酋航空在货运机队扩张上迈出了重要一步，进一步强化了其全球物流能力。
这架注册号为A6-EBK的改装货机已正式启动商业运营，其首航任务成功执飞中国香港至迪拜航线，单次载货量突破100吨大关。
