# 越南金融科技巨头MoMo或出售半数股权，重塑投资者格局

本刊获悉，越南金融科技公司MoMo正吸引潜在买家对其高达50%的股权进行评估，此举或将重塑这家公司的投资者结构。
据知情人士透露，黑石集团（Blackstone）和三菱日联金融集团（MUFG）是正在考量MoMo股权的五家机构投资者之一。
本次交易进程由摩根士丹利（Morgan Stanley）和杰富瑞（Jefferies）担任顾问。
目前，谈判仍处于尽职调查阶段，最终交易是否达成尚存变数。
一旦交易顺利完成，部分长期支持者将有机会部分套现其所持股份。
据悉，MoMo的现有投资者阵容中，包括华平投资（Warburg Pincus）和Goodwater Capital，这两家机构曾于2021年共同领投其D轮融资。
其他投资者还涵盖了Kora Management、Tybourne Capital Management、Affirma Capital、麦格理资本（Macquarie Capital）、瑞穗银行（Mizuho Bank）以及高盛（Goldman Sachs）等知名机构。
华平投资持有MoMo 26%的股权，而Augusta Investments I Pte Ltd则持股超过13%。通过E轮融资加入的瑞穗银行，预计到2025年底将持有7.3%的股份。
2021年，MoMo在瑞穗银行牵头下完成了2亿美元融资，使其估值突破20亿美元大关。
MoMo最初以电子钱包业务起家，现已发展成为一款涵盖支付、投资、保险和资金管理等多元化金融服务的综合性应用程序。
该平台目前服务超过3000万用户，并积极围绕人工智能驱动的金融工具进行战略转型。
据此前市场信息，MoMo已于2024年实现全年盈利，净利润达3475亿越南盾，约合1340万美元。
根据相关文件显示，该公司此前一年曾录得亏损，但预计在2025年仍将保持盈利态势。
对于股权出售传闻，黑石集团、MoMo和华平投资均拒绝发表评论，而三菱日联金融集团则未予回应。
