# 加密市场8个月蒸发2.3万亿，总市值腰斩

📉 加密市场8个月蒸发2.3万亿，总市值腰斩
全球加密货币市场近期遭遇新一轮剧烈回调。最新数据显示，全球加密资产总市值已从2025年10月约4.3万亿至4.4万亿美元的历史高点，骤降至约2万亿美元。这意味着在短短八个月内，市场蒸发了约2.3万亿美元，跌幅超过最高市值的一半，创下有史以来同期最大回撤纪录。
