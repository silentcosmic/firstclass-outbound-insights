# 迪拜地铁蓝金双线官宣：2029、2032年分批开通

⚡ 迪拜地铁蓝金双线官宣：2029、2032年分批开通
迪拜的公共交通网络即将迎来一次重大升级。地铁蓝线目前已进入建设阶段，而金线也已提上规划日程。这两条新线路将显著拓展城市轨道交通覆盖，把更多新旧住宅区、商业中心及主要交通枢纽紧密连接起来，大幅提升居民和商务人士的出行效率。
蓝线预计将于2029年投入运营，共设14个车站，分为两条支线，主要服务迪拜东部及北部社区，并提供前往迪拜国际机场（DXB）的快捷通道。其中，Creek side支线将经过迪拜河港（Dubai Creek Harbour），该站建成后将以74米的高度成为全球最高的地铁站。两条支线最终将在International City 1汇合，该站将成为迪拜地铁网络中最大的地下换乘枢纽，占地超4.4万平方米，预计日客流量可达35万人次。
金线则计划于2032年开通，将是迪拜首条全地下地铁线路，共设18个车站，覆盖15个重点区域，从Al Ghubaiba延伸至Jumeirah Golf Estates，有效串联起迪拜的老城区与新城区。该线路将提供多个重要换乘点，包括与绿线在Al Ghubaiba换乘，与红线在Business Bay和Jumeirah Golf Estates换乘，并可展望未来与阿提哈德铁路客运服务在Meydan和Jumeirah Golf Estates实现连接。金线预计将服务约150万居民，并为55个大型房地产项目提供交通支持，受益区域涵盖Bur Dubai、Jumeirah、Jumeirah Village Triangle (JVT)等。
这两条新线的建设，无疑将极大改善迪拜多个热门住宅和商业区的通勤体验，进一步巩固迪拜作为全球互联互通城市的地位。
