# 柬埔寨拜林省长誓言：彻底清剿网络诈骗

🚨 柬埔寨拜林省长誓言：彻底清剿网络诈骗
柬埔寨拜林省省长班斯雷蒙博士近日在会议上明确表示，将打击网络诈骗列为当前首要任务，并誓言彻底清除省内及全国范围内的电诈犯罪。
班斯雷蒙省长强调，此举旨在响应洪玛奈总理的最高指示，要求各级执法部门强化责任，重点清查人员聚集区、外籍人员落脚点及赌场等高危区域，严厉打击外籍人员非法入境并从事网络诈骗的活动。
