# 卡塔尔前埃米尔逝世，阿联酋宣布全国哀悼

📣 卡塔尔前埃米尔逝世，阿联酋宣布全国哀悼
阿联酋总统事务法院近日发布公告，为悼念卡塔尔前埃米尔谢赫哈马德·本·哈利法·阿勒萨尼殿下的逝世，阿联酋全国将举行为期四天的哀悼活动。
根据公告，自7月12日（星期日）至7月15日（星期三），阿联酋所有官方机构以及驻外使领馆和外交代表机构均将降半旗志哀，以示敬意。
