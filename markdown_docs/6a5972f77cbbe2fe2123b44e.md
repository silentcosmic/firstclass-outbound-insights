# 迪拜虚拟资产监管局向Triple-A颁发经纪商牌照初步许可

支付科技公司Triple-A日前获得迪拜虚拟资产监管局（VARA）的初步批准，准许其在当地提供经纪商服务。
作为一家总部位于新加坡的支付企业，Triple-A接下来需满足一系列运营准备要求，并达到监管机构设定的其他条件，方能最终获得正式牌照。目前，其牌照申请的评估流程仍在持续进行。
作为迪拜虚拟资产领域的最高监管机构，VARA主要负责监督该酋长国境内外虚拟资产的供应、使用与交易活动。
事实上，Triple-A在全球的合规布局已初具规模，其在新加坡和欧盟也已依据当地金融管理局（MAS）及欧盟加密资产市场法规（MiCA）的要求，获得了相关运营许可。
此外，该公司在美国超过20个司法管辖区持有汇款牌照，并在美国和加拿大注册为货币服务企业，进一步彰显其在全球主要市场的合规深度。“此次里程碑式的进展，是基于我们为预见市场未来趋势而早已构建的支付基础设施之上。这亦是我们合规、法务及各跨职能团队数月来严谨协作的成果，同时，也深刻体现了我们致力于在健全监管框架内持续运营的长期承诺。”
