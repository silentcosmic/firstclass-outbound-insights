# 英伟达擘划河内研发中心新愿景：剑指全球顶尖AI工程枢纽

英伟达（NVIDIA）正致力于将旗下位于河内的越南研发中心（VRDC）打造成为全球领先的人工智能工程枢纽。此举不仅将为英伟达的全球业务运营注入强劲动力，更将对越南乃至整个东南亚地区的创新生态系统产生深远影响。
英伟达全球副总裁Steven Truong日前出席了河内百年总体规划与2026年投资促进大会，并于会上发表了相关观点。
Truong指出，英伟达的愿景是使VRDC成为驱动公司全球成就的关键力量，同时兼作区域乃至全球的创新焦点。这不仅有助于巩固越南作为AI先行者的地位，也将把河内塑造成为该国创新与数字化转型的重要增长极。
为达成这一宏伟目标，英伟达计划大力投入人力资源建设，涵盖学生、讲师及研究人员。此外，公司还将与越南本土伙伴，包括大学和研究机构，展开深度科研合作项目。英伟达的目标是通过建设现代化AI工厂及基础设施，构建全球最先进的AI生态系统之一，以此吸引顶尖人才汇聚。
Truong强调，河内正面临成为全球领先AI之都的独特历史机遇。他进一步阐释，以AI为核心的数字化转型将重塑河内未来百年的发展轨迹，最终目标在于提升市民生活品质、打造透明高效的治理体系、发展智慧城市并创造高价值就业机会。
谈及英伟达在越南的当前业务表现，Truong透露，2025年至2026年间，公司在越南的业务增长率达到了约130%。同期，VRDC自身的业务也实现了约90%的增长。该中心不断扩大其在河内和胡志明市的工程与研究团队，并吸引了来自顶尖大学、主要科技企业及专家社区的优秀毕业生。
他表示，VRDC的工程师和科学家们正直接参与英伟达多项全球重大战略项目，其中包括语音AI、Metropolis平台、BioNeMo生物制药平台、AI-RAN无线接入网络技术以及智能代理系统。此外，他们还在英伟达NeMo平台上开发了越南语大语言模型，该模型在VMLU排行榜上荣登榜首。
Truong总结道，河内有潜力整合AI基础设施、主权AI、AI工厂、世界级大学、研究中心、创新创业生态系统以及高素质人才，从而构建全球最先进的AI生态系统之一。
