# 美国调整缅甸策略：聚焦打击跨境诈骗与犯罪

🚨 美国调整缅甸策略：聚焦打击跨境诈骗与犯罪
美国正重新评估对缅甸的政策，并计划加强与缅甸境内相关组织的合作，以更有效地打击跨境电信诈骗及其他跨国犯罪。此举旨在维护美国国家利益，并促进印太地区的安全与稳定。
美国国务院东亚暨太平洋事务助理国务卿迈克尔·G·德桑布尔在国会听证会上明确指出，鉴于过去政策未能有效遏制缅甸持续多年的武装冲突及人权问题，美方正调整合作方向，重点聚焦于打击网络诈骗园区扩张、人口贩运及洗钱等犯罪活动。
美方披露的数据显示，2025年美国民众因网络诈骗造成的经济损失预计将高达125亿美元，较2024年增长约25%，凸显跨境电信诈骗已成为美国亟需重点应对的跨国犯罪之一。美国表示将持续强化与地区伙伴的合作。
