# ️ 迪拜启动夏季送清凉行动：百万饮品支援户外工作者

☀️ 迪拜启动夏季送清凉行动：百万饮品支援户外工作者
在迪拜酷热的夏季，一项旨在关怀户外工作者的社区公益行动“Al Freej Fridge”再度启动。鉴于当前迪拜正值酷暑，户外工作者面临严峻高温挑战，该项目已是连续第三年开展。
此次行动计划派发超过200万份冰水、果汁及冰淇淋等消暑饮品，持续至9月3日，以缓解高温对户外工作者的影响，体现了当地对劳动群体的关注。
