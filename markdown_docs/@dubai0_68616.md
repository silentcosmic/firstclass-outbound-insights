# 郭文贵被判30年监禁，巨额资产遭没收

🚨 郭文贵被判30年监禁，巨额资产遭没收
近日，潜逃美国的中国商人郭文贵因大规模金融诈骗等多项罪名，被纽约南区联邦法院判处30年监禁。同时，法院裁定没收其8.89亿美元（约合60.4亿元人民币）的非法所得。
联邦法官阿纳丽莎·托雷斯在宣判时指出，郭文贵在十多年前离开中国后，在美国实施了一系列诈骗活动，其骗取资金主要用于个人享乐。据市场消息，这起大规模金融诈骗案导致全球范围内超过1000名受害者损失数亿美元。
