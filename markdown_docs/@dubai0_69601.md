# 老挝“杀猪盘”团伙在华受审：11人当庭认罪

🚨 老挝“杀猪盘”团伙在华受审：11人当庭认罪
7月9日，湖南娄星区人民法院公开审理了一起备受关注的跨境电信网络诈骗案。黄某、李某某等11名被告人被控通过偷渡或骗领护照等方式出境，抵达老挝“金三角”一产业园区，参与针对中国境内居民的虚假投资理财诈骗活动。
公诉机关指控，该团伙在2023年至2024年间，利用微信等社交平台，虚构“高富帅”“白富美”身份，诱骗受害人进行投资，从中骗取巨额钱财。庭审中，11名被告人全部当庭认罪认罚，案件将择期宣判。
