# 槟城警方捣毁5处诈骗窝点，百余人落网并解救受害者

🚨 槟城警方捣毁5处诈骗窝点，百余人落网并解救受害者
7月16日晚间，马来西亚槟城警方成功捣毁了五处跨境虚假投资诈骗窝点，在威中县和槟岛同步行动，共计逮捕109名多国籍涉案人员，并成功解救一名被犯罪团伙控制、强迫务工的25岁喀麦隆籍女子。
据槟城警察局长林广庆拿督通报，此次行动线索源于喀麦隆社群代表的报案，称一名本国女子遭到非法拘禁与胁迫劳动。警方随即锁定目标，在首轮突袭中解救出受害女子，并扣留了29名无法出示合法证件的外籍人员，其中包括一名疑似该犯罪团伙金主兼实际运营负责人的中国台湾籍男子。
初步审讯显示，该喀麦隆籍女子此前在柬埔寨为同一团伙工作近一年，后被非法带入马来西亚。犯罪团伙以高薪工作为诱饵将其骗入，却未兑现薪资承诺，长期限制其人身自由。该女子最终设法联系上本国社群代表求助，才得以曝光案件。
随后，警方对槟岛的四处民宅式诈骗窝点展开收网行动，再抓获80名涉案人员，包括一名马来西亚本地男子和79名来自中国、越南、柬埔埔寨的外籍人员。经查，这些呼叫中心已秘密运营一个多月，既作为诈骗据点，也充当涉案人员的集体住所。
该团伙通过网络平台和境外中介发布每月3200至5000马币的高薪客服招聘信息，诱骗人员入职。工作人员日常通过社交软件联系陌生人，虚构理财项目，诱导境外受害者投入资金，实施跨境诈骗。警方在行动中查获了大量手机及其他作案电子设备。目前，所有涉案人员已被移交法院申请延押，以便进行更深入的调查取证。
