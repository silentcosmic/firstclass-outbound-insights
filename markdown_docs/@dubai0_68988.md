# 越南富国岛破获诈骗案：4名中国人非法入境被捕

🚨 越南富国岛破获诈骗案：4名中国人非法入境被捕
越南边防部队近日在富国岛拜蓬港成功拦截四名非法入境的中国籍男子。据越南媒体报道，这四人于昨日清晨5时许在一辆准备登船前往河仙市的出租车上被查获。
经调查，四名嫌疑人包括李中国（1980年生）、柴人尚（1992年生）、许新文（2000年生）及张文博（2005年生）。他们供认从柬埔寨非法潜入越南，并流窜至富国岛企图开展网络诈骗活动。
警方指出，该团伙涉嫌与一起利用高科技手段在富国特区进行网络诈骗的重大案件有关。目前，越南警方正对此案进行扩大调查和依法追办。
