# 迪拜金价回升：24K金重返500迪拉姆上方

📈 迪拜金价回升：24K金重返500迪拉姆上方
迪拜黄金市场近期再次呈现上涨趋势。受国际市场连续两日上扬推动，24K黄金价格已重新站上每克500迪拉姆关口。
据最新数据显示，24K黄金报价达到每克503.50迪拉姆，较前一交易日的494.75迪拉姆有显著提升。同时，22K黄金价格也攀升至每克466.25迪拉姆，高于此前的458迪拉姆。
此次价格回升使得迪拜金价大致回到6月第三周的水平。然而，与6月初的高点相比，当前价格仍有一定差距。例如，24K黄金在6月4日曾触及每克538.50迪拉姆的高位，而22K黄金当日也曾达到498.50迪拉姆。
