# 迪拜超豪宅销量破纪录：H1 2026交易额达51亿美元

🔥 迪拜超豪宅销量破纪录：H1 2026交易额达51亿美元
尽管地区地缘政治存在不确定性，迪拜的豪华房地产市场在2026年上半年创下新纪录。根据全球地产咨询公司Knight Frank的数据，共有296套住宅物业以超过1000万美元的价格售出，交易总额达到51亿美元。
Knight Frank指出，与2025年上半年相比，2026年上半年超1000万美元豪宅的交易价值增长了14%，交易数量增长了16%。若与2024年上半年相比，交易数量更是大幅增长了49%。
具体来看，迪拜在第一季度录得165套超1000万美元的豪宅交易，第二季度则有131套。值得一提的是，第二季度还包括创纪录的26套售价超过2500万美元的超高端豪宅，这凸显了市场对顶级住宅的持续强劲需求。
