# 缅泰边境打击升级：239名涉诈中国人被移交

🚨 缅泰边境打击升级：239名涉诈中国人被移交
据缅甸官方媒体7月9日报道，缅甸方面近日将239名中国籍人员移交给泰国。这批人员涉嫌参与网络赌博、电信诈骗及其他犯罪活动，其主要活动地点位于克伦邦妙瓦底镇区水沟谷一带。此次移交通过缅泰第二友谊大桥完成，是中泰缅三方合作打击跨境犯罪的最新进展。
缅方表示，今年以来已累计遣返逾1.3万名涉诈人员及人口贩运受害者，并正与周边国家和地区加强合作，严厉打击网络诈骗活动。然而，外界对缅甸打击电诈的决心仍存疑虑，因部分诈骗园区被指与当地武装势力及利益链条深度绑定，其灰色收入长期滋养地方势力。值得关注的是，随着边境清查力度加大，部分诈骗团伙已不再固守大型园区，而是转为分散藏匿于更小、更隐蔽的据点，以规避执法打击。
长期以来，水沟谷与妙瓦底一带被视为缅泰边境网络诈骗、线上赌博和人口贩运活动的集中区域。
