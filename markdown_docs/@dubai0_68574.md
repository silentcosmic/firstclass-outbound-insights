# 迪拜百万婚恋骗局：WhatsApp成关键证据

🔍 迪拜百万婚恋骗局：WhatsApp成关键证据
近日，迪拜法院审理了一起引人关注的案件：一名出身显赫的阿联酋女性遭遇婚恋及投资诈骗，损失近百万迪拉姆。
诈骗者以共同投资未来为幌子，诱导受害者投入巨额资金。由于缺乏书面协议，受害者只能将案件提交至迪拜法院。庭审中，一系列银行转账记录与WhatsApp聊天信息成为核心证据，这些数字痕迹耗时近一年才被完整整理并提交。
法院最终判决受害者胜诉，勒令诈骗男子偿还近百万迪拉姆本金，并支付相应损失赔偿、法定利息、诉讼及律师费用。此案凸显了数字通讯记录在现代法律诉讼中的关键作用，尤其是在缺乏传统书面证据的情况下。
