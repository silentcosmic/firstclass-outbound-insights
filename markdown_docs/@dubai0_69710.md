# 迪拜上半年逾5000人皈依：文化包容战略见成效

📣 迪拜上半年逾5000人皈依：文化包容战略见成效
今年上半年，共有5511名来自不同国籍的公民在迪拜选择皈依伊斯兰教。这一数据由隶属于迪拜伊斯兰事务与慈善活动部的穆罕默德·本·拉希德伊斯兰文化中心发布。
中心主任贾西姆·穆罕默德·阿尔·哈兹拉吉表示，这一成绩验证了迪拜长期坚持的策略：即营造一个包容多元国籍与文化的环境，并致力于扎稳相互尊重的根基。他指出，这并非仅仅是数字的呈现，其背后是一套从教育、文化到社会活动的完整支持体系。新穆斯林在此不只完成一项仪式，更会获得持续的陪伴与融入支持，包括《古兰经》学习、阿拉伯语培训、伊斯兰教法普及、各类教育课程，以及社区聚会和长期的跟进项目，确保他们在认知和生活上都能稳固立足。
阿尔·哈兹拉吉强调，这种积极的氛围本身就是迪拜所倡导的宽容与共存价值观的体现，也与伊斯兰事务与慈善活动部提升生活质量、构建更具认知度和凝聚力社会的长期目标高度一致。
