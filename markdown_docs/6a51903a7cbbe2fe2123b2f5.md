# 币安：欧洲战略持续推进，亚洲版图加速扩张

加密货币交易巨头币安正全力确保其在欧洲市场的战略部署稳步推进，同时也在亚洲地区积极筹划更广泛的监管布局。
币安联合首席执行官 Richard Teng 近期阐述了公司在全球的牌照获取规划。
Teng 透露，欧洲部分监管机构已鼓励币安通过其各自的监管框架申请许可，但鉴于谈判仍在进行中，他未指明具体是哪些机构。
此番表态之前，币安上月刚撤回了其在希腊提交的、依据欧盟《加密资产市场监管条例》(MiCA) 框架下的申请。
业界普遍预计，币安将把其在欧盟的牌照申请工作转移至其他成员国，毕竟 MiCA 批复是加密企业在欧盟全境运营的必要条件。
Teng 曾表示，币安此前认为其在希腊的申请已符合所有必要标准，理应获准进一步推进。
币安撤回申请，旨在避免欧盟客户因过渡期紧迫而面临不便。
与此同时，该交易平台也在积极深化其在亚洲市场的合规化布局。
其近期在亚洲区域的最新进展包括进军菲律宾市场，并与当地金融科技公司 Blockshoals Technologies 达成合作。
Teng 进一步指出，随着币安区域运营版图的扩大，更多牌照审批正在积极推进中。
