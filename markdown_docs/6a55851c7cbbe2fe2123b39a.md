# Agnes AI发布Agnes-2.5-Flash：以无限制免费访问，重塑智能体编程体验

Agnes AI正式发布了Agnes-2.5-Flash，这是其最新一代文本模型，专为编程及智能体任务设计。
该公司同时宣布了Agnes-2.5-Pro，这是一款更大型的旗舰模型，旨在处理更复杂、运行周期更长的智能体工作，预计将于本月晚些时候推出。
伴随模型发布，Agnes AI还推出了Agnes Code，这是一个桌面工作区，开发者可以在其中直接将新模型应用于实际项目。“智能体编程的真正制约并非能力，而是成本。Agnes-2.5-Flash的推出，彻底消除了这一瓶颈。”
当前行业趋势正转向自主运行、长时间工作的智能体，它们能够独立规划任务、跨文件进行编辑、执行指令并自我校验结果，而非仅仅局限于一次性响应一个提示。
对于已采用这种工作流程的开发者而言，这标志着软件构建方式的根本性变革。
然而，这种转变也伴随着一个显而易见的弊端：随着使用深度的增加，成本支出也随之飙升。
当智能体工作流变得不可或缺时，日常使用智能体的开发者们却面临着使用限制和日益膨胀的成本，这无疑是对行业正积极倡导行为的一种“征税”。
Agnes-2.5-Flash旨在彻底消除而非仅仅管理这笔“税费”，确保开发者可以根据工作需求，无顾虑地大规模运行智能体，无需时刻关注使用量。
Agnes-2.5-Flash是Agnes-2.0-Flash的直接继任者。其主要改进体现在智能体工作流所依赖的关键任务上：
底层的智能体编排层——Agnes Harness也得到了升级，显著提升了模型在执行多步骤任务时保持状态跟踪的可靠性。
目前，开发者已可通过Agnes AI开发者平台及API访问Agnes-2.5-Flash。
此前，通过API使用Agnes-2.0-Flash的逾58.5万用户曾指出，该模型在实际编程工作中表现出色，Agnes AI团队正是基于这些反馈进行了本次更新。“我认为Agnes Flash非常适合代码补全功能。我可能会在VS Code中使用它。它在CSS方面表现强劲，并且在通用编程中也相当稳健。”
Agnes-2.5-Flash正是基于开发者们所描述的这一模型基础之上进行了直接构建与优化。
若Flash模型侧重于速度和日常编程，那么Agnes-2.5-Pro则专为更严苛的工作场景而设计。
这类工作涵盖生产级系统、复杂的智能体管道，以及需要长时间深度推理的任务。
该模型的目标受众包括专业开发团队、独立开发者、研究人员及高级用户，他们对模型能力的需求远超快速模型所能提供的范畴。
Agnes-2.5-Pro将于本月晚些时候推出。其定价及访问详情将在临近发布时公布。
对于那些更青睐项目级工作区而非纯粹API访问的开发者，Agnes-2.5-Flash现已可通过今日同步发布的全新桌面应用Agnes Code获取。
Agnes Code允许开发者以自然语言描述任务，并能直接在真实项目中执行，包括读取文件结构、跨文件编辑及运行指令，所有结果将直接保存到项目中，而非分散于聊天记录。
它与Agnes AI的网页版及API产品共享账户、订阅和积分体系，因此开发者只需一次付费，即可在聊天、API和Agnes Code等所有平台使用其余额，避免了管理多份独立计划的繁琐。
Agnes AI的免费访问模式并非新鲜事物。然而，真正令人瞩目的，是其在Agnes 2.0模型系列下所达到的运行规模。
在过去七天内，Agnes AI的文本、图像及视频模型在现有免费访问计划下，累计处理了5.41万亿个tokens，其中包括3.25万亿文本tokens和2.16万亿多模态tokens，且无任何使用上限，也未设定截止日期。
举例而言，每周3.25万亿文本tokens的处理量，相当于数百万部完整长篇小说的文本量，并且持续不断，生成这些数据的开发者无需支付任何费用。
如此庞大的处理量，清晰地证明了消除成本障碍正在改变人们的构建方式。这并非偶然的实验。
这是一种可持续的生产级应用，且其规模已由现有基础设施顺利承载。Agnes-2.5-Flash正是在这种免费访问模式下发布，并将其拓展至Agnes AI迄今最强大的编程模型版本。
此次发布的所有产品均非最终版本。Agnes AI正将Agnes-2.5-Flash和Agnes Code提前交由真实开发者使用，并根据实际使用反馈来塑造后续迭代。
Agnes-2.5-Pro将于本月晚些时候推出，同时Agnes AI的图像和视频模型也将迎来更新，这些都将作为更广泛的Agnes全模态模型系列的一部分。
