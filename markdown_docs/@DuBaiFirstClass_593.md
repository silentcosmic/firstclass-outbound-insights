# 星展银行2.1亿美元融资，力促亚洲能源转型

⚡ 星展银行2.1亿美元融资，力促亚洲能源转型
星展银行近日宣布，已向ETAFCo提供2.1亿美元高级融资。这项关键举措旨在加速亚洲地区的清洁能源转型与基础设施建设，彰显了其在可持续发展领域的承诺。
此外，星展银行还是新加坡金融管理局（MAS）FAST-P倡议下，唯一一家支持两项合作的商业银行，这凸显了其在推动区域绿色金融创新中的领导地位。
