# 台湾竹联帮涉人口贩运：缅甸女子被诱骗囚禁逼娼

🚨 台湾竹联帮涉人口贩运：缅甸女子被诱骗囚禁逼娼
近日，台湾媒体揭露了一起震惊社会的人口贩运案。具竹联帮和堂背景的犯罪团伙，由臧维林、柯讲韬等人组建“金三角国际公司”，自2024年起在缅甸、泰国等地通过社交媒体发布虚假高薪招工信息，诱骗9名缅甸女子赴台。
这些受害者抵达台湾后，随即被囚禁并被迫从事性交易，其中最高纪录一天被迫接客达11次。目前，新北地检署已侦查终结此案，并依人口贩运防制法、组织犯罪防制条例及妨害风化等罪名对涉案人员提起公诉。
