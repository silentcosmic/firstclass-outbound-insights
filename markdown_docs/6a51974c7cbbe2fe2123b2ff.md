# 新加坡对AI接受度领先全球，为何日常应用却步不前？

一项最新调查结果显示，在全球范围内，新加坡案头工作者对人工智能（AI）的怀疑程度最低。仅有29%的受访者自认为是AI怀疑论者，这一比例显著低于全球37%的平均水平，更是远低于美国、英国和法国等国家53%的怀疑比例。
然而，尽管新加坡员工对AI持开放态度，但其在日常工作中的实际应用水平却令人担忧。根据这项覆盖四大洲逾1500名案头工作者的调研，仅有6%的新加坡员工表示AI已成为其日常工作的核心组成部分，这一比例几乎是全球11%平均水平的一半。
这种“态度积极，行动滞后”的显著落差并非源于员工本身的抵触，而是企业在AI部署实践中的表现未能达到预期。在曾经历AI试点失败的31%新加坡员工中，高达40%的人将“生成结果过于通用化”列为失败主因，这一比例远超所有受访市场，并高出全球平均水平30%整整十个百分点。此外，约38%的受访者对AI输出结果的信任度较低，高于全球28%的平均值；另有30%的员工认为AI成果缺乏业务关联性，而全球平均水平为22%。
这份研究进一步剖析了全球超过500名成功将AI从初步试点深度融入日常工作流程的案例。研究发现，实现这一转型的关键在于围绕AI工具构建完善的生态系统，这包括提供针对具体岗位的专业培训、将AI深度嵌入现有工作流程，以及确保不可妥协的数据安全保障。
以新加坡保险公司Singlife为例，该公司成功实践了AI的深度应用。通过将企业内部的产品手册、培训指南和常见问题解答（FAQs）等关键知识库作为其AI代理的基础，Singlife的客服人员现在能够几乎即时地获取准确且有来源依据的答案，从而取代了过去手动筛选大量文件的繁琐过程。
一位行业资深人士指出，新加坡员工并非阻碍AI发展，而是在等待真正能够为他们所用的AI解决方案。他强调，企业领导者必须超越通用型工具的局限，转而采用那些值得信赖、根植于具体业务场景并能深度融入日常工作流程的AI系统。
值得关注的是，新加坡通讯及新闻部部长杨莉明（Josephine Teo）与东南亚出行巨头Grab的创始人陈炳耀（Anthony Tan）已受邀加入“AI for Good全球委员会”，这预示着新加坡在推动AI向善发展方面的积极投入。
