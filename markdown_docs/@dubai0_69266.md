# 日本收紧柬埔寨、瓦努阿图及福建人员居留审查

🔍 日本收紧柬埔寨、瓦努阿图及福建人员居留审查
日本政府正着手加强对来自柬埔寨、瓦努阿图及中国福建省人员的居住身份审查程序。此举是为了应对近期东南亚和南太平洋地区针对日本的电话和电子商务诈骗犯罪激增。
据外交部宣布，自2026年7月15日起，针对上述国家和地区人员的签证签发和入境审查将变得更为严格。具体而言，柬埔寨公民在申请特定技术工人、技术实习培训和商务等居留身份时将面临更严苛的审查；瓦努阿图公民在办理旅游、商务、特定技术工人及家庭团聚等居留身份时亦将受到影响；而来自中国福建省的人员，特别是闽南籍居民，无论其具体居住地，都将面临不限居留身份的严格审查。此举旨在从源头上遏制诈骗活动，维护日本的社会治安。
