# 马来西亚突袭赌球窝点：32名中国籍人士被捕

🚨 马来西亚突袭赌球窝点：32名中国籍人士被捕
随着2026年美加墨世界杯淘汰赛阶段的深入，非法赌球活动愈发活跃。7月1日凌晨，马来西亚警方在吉隆坡蕉赖展开代号“非法赌球行动11”的突击行动，成功捣毁一个线上赌球运营中心，现场逮捕32名中国籍嫌疑人，并查获大量电子设备。
据悉，此次行动共逮捕29名中国籍男子及3名女子，年龄介于29至40岁。警方查扣了28台电脑、手机及数据机等设备，初步调查显示，该犯罪集团租用办公单位，通过网站“6288.com”经营世界杯赌球业务。被捕人员主要负责管理赌客账户、处理资金交易及推广平台。
目前，所有嫌疑人已被批准延长扣留4天以协助调查，警方正援引《1953年公开赌博法令》及《1959/63年移民法令》展开深入调查。吉隆坡警方强调，世界杯赛事期间将持续加强打击非法赌球行动，严厉遏制相关犯罪活动。
