# 菲律宾遭超强台风侵袭：暴雨引发山体滑坡致15人死亡

🚨 菲律宾遭超强台风侵袭：暴雨引发山体滑坡致15人死亡
超强台风“巴威”（当地命名“印黛”）近期横扫菲律宾，已造成至少15人死亡。该台风带来持续强降雨，导致菲律宾多地受灾。上周五，菲律宾南部棉兰老岛萨兰加尼省的一场暴雨引发山体滑坡，造成10人被掩埋身亡。当地民防部门官员向法新社表示，连日降雨致使土壤饱和，最终引发了致命的山体滑坡。
