# 科伦坡安全警示：中国公民夜间疑遭“执法人员”拦车索款

🚨 科伦坡安全警示：中国公民夜间疑遭“执法人员”拦车索款
近日，有网友投稿称，其在斯里兰卡科伦坡夜间出行时，遭遇自称执法人员的拦车检查，并被疑似索要高额“罚款”。
据该网友描述，其与同事乘坐网约车前往目的地途中，车辆被拦停。最初对方要求司机出示证件，随后多名人员围住车辆，并对车内人员及随身物品进行检查。在未发现其他问题后，对方将目标转向网友携带的免税香烟，并以此为由提出高达约6万斯里兰卡卢比（约合人民币1200元）的“罚款”要求，声称此为“严重问题”。
双方沟通约30分钟，期间对方将金额从6万卢比逐步降至3万卢比，但网友团队始终未予支付。最终，对方放弃纠缠并离开现场。事后，据该网约车司机透露，当地偶有类似情况发生，但该司机拒绝了可能存在的“分成”要求。
鉴于此类事件，提醒在海外的中国公民及商务人士，如遇不明身份人员检查，务必保持冷静，核实对方身份及执法依据，并注意保留现场证据。切勿轻易向陌生人员支付现金，如遇不合理索取，建议通过官方渠道寻求帮助或投诉。
