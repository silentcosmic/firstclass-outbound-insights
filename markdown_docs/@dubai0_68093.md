# 柬埔寨商博委：共商网赌诈骗查封资产处置

🔍 柬埔寨商博委：共商网赌诈骗查封资产处置
本周一，柬埔寨商业博彩管理委员会（CGMC）总秘书处在金边召开重要会议，专项讨论针对网络诈骗犯罪中查封、充公财产的收集、管理与处置机制。
此次会议由委员会总秘书长 Yeth Vinel 主持，柬埔寨经济财政部（MEF）国家财产与非税收入总局（GDSPNR）副总局长 Nom Sinith 亦共同出席。
与会顾问、部门主任及官员共同审查了相关程序，旨在加强跨部门协作，确保非法资产的收缴与处置工作严格符合现行法律法规，以进一步提升资产管理的透明度与问责制。
