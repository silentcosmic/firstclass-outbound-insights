# 阿联酋警告：未经许可播放世界杯将面临重罚

🚨 阿联酋警告：未经许可播放世界杯将面临重罚
阿联酋经济与旅游部近日向当地企业和场所发出明确提醒，禁止未经许可播放FIFA世界杯赛事。此举旨在确保商业活动符合相关版权法规，避免触及法律红线。
根据官方通告，所有计划播放2026年FIFA世界杯赛事的商业机构，必须从官方转播权持有者处获取必要许可，并仅使用经授权的频道和设备。违规者将面临经济处罚，甚至可能引发法律诉讼。
这一提醒正值阿联酋各地球迷热情高涨，聚集观看世界杯比赛之际。目前，许多酒店和娱乐场所已提前获得转播许可，并为方便球迷观赛延长了营业时间。当局强调，场所应妥善保管并随时出示许可证以备检查。
