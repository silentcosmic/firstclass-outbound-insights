# 老挝警方重拳出击：百余名跨国诈骗嫌犯落网

🚨 老挝警方重拳出击：百余名跨国诈骗嫌犯落网
2026年7月11日上午，老挝公安部打击网络诈骗专项工作组对万象市西萨达纳县蓬卡扬村的CAMCE Investment（Laos）有限公司（东盟别墅）出租房屋进行了突击搜查。此次行动旨在清剿盘踞在该地的电信网络诈骗团伙。
警方共检查了7栋出租房屋，成功控制涉嫌电信通讯诈骗的嫌疑人122名，并查获大量涉案物品，包括电脑、手机、护照等设备，以及多辆汽车和疑似毒品。值得关注的是，被捕嫌疑人国籍多样，涵盖中国、巴西、乌拉圭、巴基斯坦、马来西亚和缅甸等，凸显了此类犯罪的跨境性质和复杂性。
行动中，警方不仅扣押了大量作案工具，还对一名前来申请保释的中国籍人员实施扣留，并查扣了一辆黑色奔驰轿车。此次大规模的打击行动，再次彰显了老挝政府对跨境网络犯罪的零容忍态度，也为东南亚地区打击此类犯罪活动提供了重要案例。
