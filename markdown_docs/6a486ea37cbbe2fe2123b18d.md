# 省级实验室核心团队赋能电氢耦合：启航氢能获厦门高新投战略领投，加速绿氢技术商业化进程

“出海 · 头等舱”独家获悉，厦门启航氢能技术有限公司（以下简称“启航氢能”）近期宣布完成新一轮战略融资。本轮融资由厦门高新投领投，所获资金将主要用于电氢耦合关键装备及离网制氢相关核心技术的持续研发投入，并加速产品的迭代升级。
启航氢能成立于2024年4月，其技术根基深植于嘉庚创新实验室的电氢耦合研究团队。公司业务聚焦于电氢耦合技术及其应用场景的装备研发、生产与销售。其核心团队在电力电子技术、智能电网与电化学等前沿领域拥有近二十年的深厚研究积累。值得关注的是，嘉庚创新实验室作为福建省政府重点支持的高水平科研平台，其氢能产业集群由实验室主任郑南峰院士与氢能研究院院长方晓亮主导布局。
在新型能源体系中，氢能扮演着连接电力系统与终端用能的关键枢纽。特别是在风能、光伏等波动性可再生能源大规模并网的宏观背景下，如何实现高效、稳定的电力转化与消纳，已成为制约产业发展落地的核心瓶颈。其中，“电氢耦合”是提升绿氢制备效率与降低成本的关键所在。通过更精细的电源控制、更智能的系统级调度以及离网制氢/微电网制氢架构的创新设计，电氢深度耦合相关技术能够显著增强绿氢系统的运行稳定性与整体经济效益。
与传统电源巨头侧重于提供通用型电源解决方案的模式不同，启航氢能基于对电化学特性的深入洞察，成功开发出高度适配电解槽工况的智能电源系统。通过自主研发的控制算法，公司有效解决了电解槽冷启动电流冲击、运行期谐波损伤以及老化期内阻补偿等电解槽全生命周期运营中的关键挑战。
截至目前，启航氢能已成功完成IGBT制氢电源、离网制氢/微电网制氢解决方案及燃料电池逆变器三大产品系列的研发，初步构建起一套全面的电氢耦合核心电气产品体系。
其中，制氢电源产品涵盖液冷与风冷两大主流技术路线，功率覆盖从30kW至20MW的广泛区间，同时提供专为PEM电解槽设计的专用电源模块以及光伏直驱制氢解决方案，能够灵活适配从6到4000标方/小时的多元制氢规模场景。此外，公司已着手下一代更高效、更低成本制氢电源的前瞻性研发。
针对制氢微电网应用，启航氢能可提供200kW至50MW级的系统集成解决方案。该方案依托1500V直流或10~35kV交流中压架构，可实现电能的中远距离高效传输与灵活组网。同时，其标准化与模块化设计确保了方案的快速部署能力。
启航氢能明确自身定位为独立的第三方氢能电气装备及系统解决方案供应商。公司不直接涉足电解槽的生产制造，而是专注于为全球电解槽制造商与终端能源业主提供专业配套服务。目前，公司已与逾50家电解槽企业建立了深度合作关系，覆盖了国内主流厂商的绝大多数市场参与者，并同步积极拓展国内大型客户及广阔的海外市场。
启航氢能核心团队成员魏闻表示，公司的核心优势主要体现在三大方面。首先，是其卓越的工程化实践能力与国际一流的产品对标水平。团队成员均深耕产业一线，拥有十余年超大功率电力电子设备研发的丰富经验，在工程化落地方面积累了扎实的实践基础。他强调：“我们的对标目标非常明确，即全球顶级技术平台。在核心产品领域，我们已能与世界顶尖企业的产品保持在同等技术水平线上，而非仅限于实验室研究阶段。”
其次，是对电解槽材料机理的深入研究以及对前沿技术方向的强大拓展能力。厦门大学及嘉庚创新实验室汇聚了世界级的电化学研究团队与先进实验支撑平台，这为公司通过跨学科交叉融合，将学术界的最新研究成果与方法论高效导入研发体系提供了坚实支撑。魏闻补充道：“我们跳出了单一的电力电子视角来审视制氢电源，而是建立起一套覆盖电解槽材料机理、整槽特性、离网制氢乃至系统级解决方案的整体研究路线图，旨在打造‘更懂电解槽的制氢电源与电氢耦合解决方案’，从根本上驱动行业技术的迭代与演进。”
第三，则是其引以为傲的产品快速迭代能力。魏闻透露：“我们的组织机制相对灵活高效，能够更迅速地完成从研发到产品化的全链条闭环，并方便与行业头部客户联合开展定制化设计。行业内一代产品的常规开发周期普遍在三年左右，而我们内部设定的目标是将其压缩至一年半，最长不超过两年，这意味着公司能够比行业主流厂商至少提前一年推出新产品。”他总结道：“因此，我们不只是追求技术不落后，而是通过这种领先的产品开发节奏，逐步构建起在产品和技术上的独特优势。”
