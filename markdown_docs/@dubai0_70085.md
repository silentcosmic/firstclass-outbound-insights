# 东帝汶总理：网络诈骗威胁国安，将严打涉外犯罪

🚨 东帝汶总理：网络诈骗威胁国安，将严打涉外犯罪
东帝汶总理夏纳纳·古斯芒近日表示，跨国网络诈骗，尤其涉及部分外国公民（包括中国籍人员）的案件，已成为国家安全的一大威胁。政府将持续强化国家安全力量，全力遏制并打击此类犯罪活动。
古斯芒强调，尽管执法部门近期已成功抓获大量涉嫌参与网络诈骗的嫌疑人，但由于此类案件数量持续增长，政府将不会放松相关执法力度。他郑重呼吁全国安全执法部门，务必保持高压态势，持续深化打击行动，全力缉捕涉嫌参与网络诈骗的犯罪分子，以切实维护东帝汶的社会治安与国家安全。
