# ️ 菲索莱尔赌场舞弊案：主管涉嫌盗刷会员积分被控

⚠️ 菲索莱尔赌场舞弊案：主管涉嫌盗刷会员积分被控
菲律宾奎松市警方通报，索莱尔北区赌场一名33岁的赌桌主管因涉嫌操纵会员积分系统，已被警方以加重盗窃罪起诉。此案源于赌场内部调查。
调查发现，多名长期未使用的钻石级会员账户出现异常积分记录。涉事主管被指控利用职务权限，在赌桌关闭且无实际博彩活动的情况下，人为录入这些账户并添加玩家评级及约9900点游戏积分。这些积分可在约两小时内被刷入，用于兑换会员奖励和专属优惠，而相关会员并未参与任何博彩。
赌场方面表示，赌桌主管仅在会员真实参与游戏时才有权限录入积分，私自添加已严重违反管理规定。赌场确认异常后立即报案，警方已于7月13日在索莱尔度假村内将涉案主管控制。目前案件已移交检方，将依法展开司法程序。
