# 马来西亚娱乐场所遭突查：房东将负连带责任

🚨 马来西亚娱乐场所遭突查：房东将负连带责任
马来西亚Mikado KTV近日深夜遭遇警方联合突击清查，现场多名涉案人员已被转运至警局，接受进一步调查。
此次行动旨在全面整治夜间娱乐场所乱象，重点打击涉黄、毒品流通及非法雇佣外籍劳工等违法行为，并同步排查无证居留外籍人士。
警方强调，未来将长期开展常态化突击检查，范围覆盖KTV、酒廊及沿街出租房屋。所有商铺及住宅房东将被要求签署租客管理相关文书，一旦查实知情不报租客涉黄涉毒或无证居住，房东将面临严厉罚款。
