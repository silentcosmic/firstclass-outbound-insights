# ️ 香港性罪行法例拟修订：强奸定义扩至男女

⚖️ 香港性罪行法例拟修订：强奸定义扩至男女
香港保安局正推动性罪行法例的重大修订，旨在更新和现代化相关法律框架。核心在于，强奸罪的定义将不再局限于男性对女性实施，而是扩大至男女皆可成为受害人，女性亦可能被定罪为主犯，最高刑罚仍维持终身监禁。
咨询文件同时明确了“同意”的定义，强调性行为必须基于当事人的自愿并清楚表示同意。文件详细列举了11种无同意情况，包括睡眠、不清醒、遭受暴力威胁或滥用信任地位等。此外，法例还将以“性侵犯”取代现行的“猥亵侵犯”罪名。
