# ️ Coinbase放宽大陆注册：身份认证新动向

⚡️ Coinbase放宽大陆注册：身份认证新动向
7月15日，多家行业媒体报道称，美国大型加密货币交易所Coinbase已调整针对中国大陆用户的注册验证流程。现在，用户可直接使用中国居民身份证和大陆地址完成身份认证，不再强制要求提供中国护照或香港地址。此举迅速在加密货币圈引发广泛关注。
据最早披露消息的区块链媒体透露，已有用户成功完成注册。该媒体还表示，经向Coinbase相关人员确认，目前中国大陆用户可直接使用身份证和大陆居住地址进行KYC（身份认证）。此前，大陆用户注册Coinbase通常需要提供中国护照及香港地区地址等资料，注册门槛较高。
然而，Coinbase截至目前尚未正式宣布重新进入中国市场，也未发布针对中国大陆用户开放业务的官方公告。业内人士普遍认为，此次变化更多体现为身份验证环节的调整，并不意味着中国大陆已经允许加密货币交易活动。即便完成注册，用户仍需严格遵守中国现行对虚拟货币交易的严格监管政策。
受此消息刺激，Coinbase（COIN）股价一度上涨超过2%。市场认为这可能有助于其扩大国际用户基础。不过，分析人士指出，Coinbase正面临全球交易平台竞争加剧等挑战，相关影响仍有待观察。
