# 越南重拳打击赌球：两特大网络涉资1.33亿美元

🚨 越南重拳打击赌球：两特大网络涉资1.33亿美元
越南警方近日宣布，在全国范围内展开的打击世界杯非法赌球行动中取得重大进展，成功捣毁了两个特大地下网络赌博犯罪团伙。据法新社报道，这两个组织严密的网络累计非法交易额高达1.33亿美元，并有85名涉案嫌疑人被捕。
胡志明市警方声明指出，此次行动于6月底启动，目标直指这些管理严格的非法赌博集团。此前，越南公安部曾通报，在世界杯开赛前的20天内，全国已取缔73个非法赌博窝点，逮捕346名嫌疑人，涉案总金额达数亿美元，显示了越南当局对非法赌博活动的持续高压态势。
