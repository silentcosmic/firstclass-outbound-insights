# 中企承建阿联酋输油大动脉开工，再创里程碑

⚡ 中企承建阿联酋输油大动脉开工，再创里程碑
近日，由中国化学工程国际公司（CNCEC）总承包，并联合六化建、华陆公司共同实施的阿联酋阿布扎比国家石油公司（ADNOC）西东管线泵站工程已正式启动。这一战略性原油输送通道将横贯阿联酋五大酋长国，目前已全面进入快速施工阶段，标志着中资企业在中东能源基础设施领域再添一座里程碑。
