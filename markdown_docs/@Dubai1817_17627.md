# 中老边境执法升级：严打电信诈骗与偷渡

🚨 中老边境执法升级：严打电信诈骗与偷渡
2026年6月30日，老挝与中国执法机关进一步深化合作，持续加大对电信网络诈骗等跨境违法犯罪的打击力度。
当天，老挝琅南塔省孟新县公安指挥部与中国云南省西双版纳州勐腊县边境管理大队举行了第二季度边境执法联席会议。双方就边境安全形势、社会治安管理、跨境犯罪打击及边境管控协作等核心议题进行了深入交流，并总结了阶段性执法成果，分析了当前边境区域面临的新挑战。
会议一致决定，将持续强化情报信息共享、联合执法协作及边境联防联控机制建设。重点打击电信网络诈骗、非法偷渡、走私及其他各类跨境违法犯罪活动，旨在全面提升边境地区的联合治理能力，共同维护中老边境的安全稳定与正常的通关秩序。
随着两国执法合作的不断深化，针对跨境犯罪的联合打击行动正日益加强，这将为维护两国边境地区社会治安和人民群众财产安全提供更强有力的保障。
