# 柬埔寨：反诈新法联动，剑指人口贩卖新变种

🚨 柬埔寨：反诈新法联动，剑指人口贩卖新变种
在东盟-澳大利亚反贩卖人口项目（ASEAN-Act）支持下，柬埔寨国家反人口贩卖委员会近日召集多方代表举行讨论会。会议指出，当前人口贩卖犯罪正呈现危险新形式，常与网络诈骗及诱骗剥削相结合，引发各界高度关注。
与会代表深入探讨了2008年《反人口贩卖及性剥削法》与将于2026年4月生效的《反网络诈骗法》的协同执行机制。讨论内容还包括柬政府通过第03号政令清查网络犯罪的程序，以及针对涉案外籍移民的驱逐流程。包括澳大利亚驻柬使馆代表、柬劳工部国务秘书以及司法、警察部门和非政府组织在内的约70名代表出席，并表示将加强跨国协同合作，提升联合调查与受害者保护能力。
