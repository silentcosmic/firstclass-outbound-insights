# 柬埔寨勋爵高价出租物业，路透社揭露疑涉电诈园区

🔍 柬埔寨勋爵高价出租物业，路透社揭露疑涉电诈园区
路透社近期发布调查报告指出，柬埔寨知名商人林兴所属的赌场及房地产集团，曾将位于奥多棉芷省皇家山度假村内的三栋建筑，以高额租金出租给一名中国籍男子。
租赁文件显示，双方于2024年3月签署了为期两年的合同，月租金高达20万美元，远超当地市场水平。尽管目前无证据表明林兴集团直接参与电诈或人口贩运，但路透社记者在现场发现，涉事园区设有高墙和铁丝网，内部多间房屋被改造为新加坡警察局、越南银行网点等逼真场景，疑似用于“冒充公检法”实施诈骗。
现场还遗留有隔音电话间、诈骗话术本及大量宿舍床位。泰国安全部门估计，该园区此前曾容纳约3000人。
