# 宏桥公司加入Anthropic“翼影项目”：以前沿AI模型强化网络安全防御

宏桥公司（Broadridge）已正式加入Anthropic发起的“翼影项目”（Project Glasswing），旨在运用前沿人工智能模型，全面升级其网络安全防御体系。
这项全新的行业倡议汇聚了众多为关键基础设施（包括金融服务领域）构建或维护软件的顶尖机构。
参与各方将深度利用Anthropic尚未发布的尖端模型——Claude Mythos Preview，以全面提升构成全球共享网络攻击面核心部分的各项基础系统的安全韧性。
宏桥公司将把该模型深度融入其防御性安全工作流程，以构筑更坚实的安全屏障。
“网络安全是金融市场韧性与稳定的基石。我们此次参与‘翼影项目’，正是为了将前沿AI模型应用于我们的自有系统，从而有效应对不断演进的新型威胁，并为构建一个更为安全的金融生态系统贡献力量。”
