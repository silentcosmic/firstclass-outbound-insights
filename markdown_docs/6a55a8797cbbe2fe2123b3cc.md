# 昆仑万维出海新篇章：亏损承压下的短剧业务创新与扩张

据本刊观察，昆仑万维旗下第三款海外短剧应用HotDrama已低调上线。此举标志着昆仑万维在海外短剧领域的战略部署正由“双轨并进”升级至“三极鼎立”——在其长期雄踞海外应用素材投放榜前列的DramaWave和FreeReels之外，又布下重要一子。
HotDrama已于今年4月相继登陆Google Play与App Store。其开发者与早前的DramaWave、FreeReels相同，均为昆仑万维旗下的新加坡业务实体Skywork AI。
从产品功能层面看，HotDrama的首页界面简洁明了，仅设“剧集”和“动漫”两大类别。其中，动漫内容以2D/3D动画为主，而热门短剧则大量采用了AI生成技术。功能配置方面，该应用集成了竖屏Feed流、离线下载、多语言字幕等行业标准功能，并延续了DramaWave的弹幕互动设计——用户在观看剧集的同时，可实时发送浮动评论，深度体现了昆仑万维“社交化观看”的产品基因。
其核心差异体现在变现策略上：HotDrama创新性地采用了IAAP混合变现模式，即付费订阅与插播广告并存。这与DramaWave的纯IAP付费模式和FreeReels的纯IAA广告模式形成鲜明对比，使其成为昆仑万维短剧产品矩阵中独一无二的混合变现产品。
在广告投放与收入表现方面，HotDrama自5月6日启动广告投放以来，日素材量一度从零攀升至6月中旬的约2700组，随后回稳至约1600组。其下载量与收入数据在6月初呈现显著加速态势，6月15日单日下载量达到峰值18654次，次日（6月16日）单日收入也创下新高，录得18466美元。近30天内，该应用在双端累计收入达到20.76万美元，折合人民币约140.3万元。
在微短剧行业整体背景下，昆仑万维的这一布局已属相当进取。横向观察市场，枫叶互动（ReelShort）、短剧快照（DramaBox）、九州文化（ShortMax）等多数头部企业，通常只专注于运营一个主力海外短剧平台。昆仑万维则显得与众不同，是少数能够在海外市场同时驾驭IAP（应用内购）和IAA（广告变现）两种模式的参与者。
纯IAP与纯IAA模式之间，隐藏着一个长期被市场低估的“利基市场”。DramaWave在高付费意愿的欧美市场展现出卓越的运营效率，其中美国更是海外短剧收入的首要市场。FreeReels则凭借其免费策略，在价格敏感型市场实现高效大规模获客。然而，两者之外，尚存一个由“不喜广告、具备一定但非极强付费能力与意愿”的用户群构成的中端市场。
HotDrama所采取的IAAP混合模式，精准地填补了这一市场空白：通过“免费+广告”的组合拳降低用户获取门槛，再辅以付费订阅机制甄别并筛选出高价值用户。这种策略同时顾及了高消费市场中付费意愿相对较低的用户，以及价格敏感市场中具有较高价值的用户群体。
根据行业研究机构发布的《2025海外短剧行业报告（全年度）》所示，2025年跻身前十的腰部应用中，有半数出现了下载量和收入同步下滑的现象。这一“头部稳固、腰部松动”的市场格局，揭示了中端市场的结构性空白，为昆仑万维推出HotDrama提供了战略机遇。
此外，在当前获客成本居高不下的背景下，IAAP模式正逐步成为行业发展的主流趋势。最新行业月报显示，截至2026年5月，海外IAAP类短剧应用数量已增至286款，环比增长7%，其数量规模已显著超越了纯IAP和纯IAA模式的应用。
HotDrama的推出，既是对市场趋势的积极响应，也补齐了昆仑万维在海外短剧商业模式布局上的最后一块版图。至此，付费、免费、混合三种模式得以同步展开，这有助于公司进行投资回报率的横向对比，并探寻最优的市场解决方案。
然而，同时运营三款不同定位的应用，其核心挑战在于“成本控制”。过往海外真人短剧的制作成本居高不下，但当前正值AI短剧技术爆发期，大幅降低的制作成本使得“三极布局”的战略得以变为现实。
根据公开资料，昆仑万维在AI赛道布局较早，并凭借其自研AI技术显著降低了内容制作成本。公司2025年年报明确指出，将依托自研AI视频生成技术，构建短剧与AI短剧的工业化生产及全球化分发体系，旨在实现AI内容的规模化、低成本、高质量产出，从而大幅削减内容创作的投入。
具体数据来看，2025年昆仑万维旗下短剧及AI短剧平台的毛利率高达79.19%，这一数字远超行业平均水平，为公司同时推动三款应用提供了坚实的成本支撑。
昆仑万维2026年第一季度财报揭示，截至2026年3月底，公司短剧及AI短剧平台单月流水已突破4800万美元，年度经常性收入（ARR）超过5.7亿美元。其合计月活跃用户数（MAU）稳居海外短剧市场榜首，其中FreeReels的下载量更是位列海外第一。
回顾2025年，昆仑万维的销售费用同比大幅增长81.53%，达到41.82亿元。销售费用与研发费用合计支出58.58亿元，占总营收的71.46%。当年公司录得净亏损15.93亿元，已是连续第二年出现亏损。
从这一角度审视，DramaWave和FreeReels在2025年合计投放超过110万组素材，其背后逻辑仍属于“烧钱换市场份额”的阶段。一旦营销投放力度减弱，用户增长和流水表现很可能面临压力。
然而，情况正显现积极转变。一方面，2026年第一季度，昆仑万维归属于母公司股东的净利润为-8.87亿元，亏损同比收窄15.34%，显示亏损幅度正在收窄。另一方面，2025年公司短剧及AI短剧平台业务收入同比飙升864.92%，达到16.17亿元，占总营收的比重从2.96%跃升至19.73%。这预示着，随着微短剧业务收入规模的持续扩张和AI产品的商业化落地，前期巨额投入所带来的拖累效应有望逐步减弱。
综合上述分析，本刊判断昆仑万维正处于一个关键的战略拐点。尽管DramaWave和FreeReels所构建的海外短剧矩阵已充分展现其营收驱动潜力，但新的“三极”布局能否在今明两年内实现利润端的实质性改善，将取决于两大核心变量：首先是AI技术在内容降本方面的效果能否持续放大并有效传导至利润环节；其次是这三款应用能否在各自目标市场建立起无需巨额投放支撑的自然增长模式。
对HotDrama而言，其真正的战略重心并非仅在于当前的投放素材量或下载数据表现，更在于它能否与既有的DramaWave和FreeReels两款产品形成真正的差异化用户分流，从而实现整体商业效益的最大化。
