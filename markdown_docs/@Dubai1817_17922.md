# 福州警方捣毁“杀猪盘”引流团伙，10人落网！

🚨 福州警方捣毁“杀猪盘”引流团伙，10人落网！
福建省福州市公安局台江分局于7月8日对外通报，警方在6月10日根据前期线索精准研判，成功捣毁一个专门为“杀猪盘”提供引流服务的犯罪团伙。此次行动共捣毁两处窝点，抓获10名涉案人员，涉案金额累计超过30万元。
据警方介绍，该团伙手法狡猾隐蔽，设立两处作案窝点，招募专职话务员，利用上游提供的虚假人设和成套“话术剧本”，在各大社交平台物色目标。他们通过嘘寒问暖、刻意套近乎的方式博取网友信任，随后将受害人推送给上游虚假投资、刷单诈骗等“杀猪盘”团伙，从中非法抽取分成获利，沦为电信诈骗的“帮凶”。
目前，警方已对3名核心涉案人员依法刑事拘留，对其余7名涉案人员采取取保候审强制措施。案件的后续深挖侦办及线索拓展工作仍在持续进行中。
