# ️ 阿提哈德航空：台风致阿布扎比-台北航班延误

✈️ 阿提哈德航空：台风致阿布扎比-台北航班延误
阿提哈德航空（Etihad Airways）近日发布公告，受强台风巴威影响，其阿布扎比往返台北的航班运营面临延误。
具体而言，原定于7月10日从阿布扎比扎耶德国际机场（AUH）飞往台北桃园国际机场（TPE）的EY898航班，因恶劣天气及桃园机场运营中断而被迫推迟。
