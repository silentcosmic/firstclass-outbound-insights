# 柬日警方高层会晤：联手打击跨境网诈

🤝 柬日警方高层会晤：联手打击跨境网诈
当地时间今日上午，柬埔寨国家警察总署副总监兼中央治安局局长迪伟杰，与到访的日本警察厅有组织犯罪对策部总监大滨猛（Takeshi Ohama）举行了高级别会晤。双方围绕深化打击电信网络诈骗及各类跨国犯罪议题，进行了深入交流。
会谈中，柬日警方就当前电信网络诈骗犯罪的严峻形势、跨境执法合作机制及情报共享等关键领域展开了细致探讨。双方达成高度共识，未来将进一步加强警务合作与协调配合，以全面提升两国共同打击跨国有组织犯罪的能力和实效。
