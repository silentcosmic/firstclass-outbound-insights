# 阿联酋总统会晤美国务卿：聚焦中东战略合作

⚡ 阿联酋总统会晤美国务卿：聚焦中东战略合作
阿联酋总统谢赫穆罕默德·本·扎耶德·阿勒纳哈扬日前在阿布扎比会见了到访的美国国务卿马尔科·鲁比奥，此次会晤是鲁比奥中东之行的重要一站。
双方深入回顾了阿联酋与美国间的战略合作关系，并就如何在各领域进一步加强协调、以服务共同利益进行了探讨。会谈重点关注了中东地区局势的最新发展，双方领导人一致强调，巩固地区安全、稳定与持久和平的基础至关重要。
