# Amity重金布局新加坡：东南亚AI研发中心揭幕，斩获1亿美元D轮融资

企业级人工智能公司Amity正式在新加坡设立其东南亚区域总部及AI研发应用中心。
此举是在其完成1亿美元D轮融资后进行的，本次扩张由新加坡经济发展局（EDB）和新加坡企业发展局（Enterprise Singapore）的投资平台SG Growth Capital旗下的投资机构EDBI领投。
新设立的AI研发应用中心将专注于开发针对企业级应用优化的代理式AI（agentic AI）及垂直领域模型。
Amity致力于构建此类系统，以实现复杂商业流程的自主执行。
本轮融资所注入的资金将通过这一新中心加速产品研发进程。
同时，这笔资金还将支持Amity在欧洲和东南亚地区的战略性收购，并用于吸引顶尖的工程和商业人才。
预计在未来三年内，该公司管理层将在新加坡创造多达60个新职位。Amity高层表示：“新加坡是我们‘亚洲企业值得拥有为其现实需求量身定制的AI’这一信念得以实现的关键之地。”“新中心代表了我们致力于构建未来十年间驱动亚洲企业发展的核心AI能力的坚定承诺。”
此次区域性战略拓展，与新加坡将AI技术融入金融服务、先进制造、互联互通及医疗保健等特定经济领域的国家议程高度契合。
目前，全球范围内已有超过20,000家机构采用该公司的平台服务。
这家技术提供商设定的目标是，到2026年底，其年度化营收将达到2亿美元。
