# 阿联酋启动百万民众应急响应培训计划

📣 阿联酋启动百万民众应急响应培训计划
阿联酋近日启动一项名为“社区就绪”的重大国家计划，目标是培训100万名居民和学生，使其具备基础应急响应能力。此举旨在构建一个全民参与的应急响应体系，确保在专业救援力量抵达前，社区成员能有效展开自救互助，最大程度减少损失。
该计划将培训分为四个递进级别。从入门级的基础急救技能（如心肺复苏、止血、AED使用、灭火与疏散）开始，逐步深入到进阶级的现场管理、伤情评估与感染控制。此外，还设有领导级课程，旨在培养社区骨干的组织与指挥能力；以及师资级培训，以确保培训体系的可持续发展与广泛覆盖。
