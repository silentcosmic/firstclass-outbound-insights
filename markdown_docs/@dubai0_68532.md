# 越南警方重拳出击：捣毁万亿盾跨境洗钱团伙

🚨 越南警方重拳出击：捣毁万亿盾跨境洗钱团伙
越南河静省公安近日联合越南公安部及多地警力，成功破获一起特大跨境网络赌博及洗钱案件，共抓获36名涉案人员，其中包括5名中国台湾籍嫌犯。
据警方披露，该团伙自今年4月起，根据台湾地区幕后人员指令，在胡志明市租用三处公寓作为犯罪窝点。他们招募越南籍人员负责办理手机卡、银行账户，并操控赌资流转。为逃避侦查，该团伙采取封闭式管理，并频繁更换办公地点与人员。
其运作模式是通过雇佣当地人实名办理大量手机卡和银行账户，随后将账户信息回传台湾地区进行集中管理，专用于接收、拆分及转移网络赌博资金。初步调查显示，短短两个多月内，该团伙利用266个银行账户，洗钱金额已突破1万亿越南盾。
此次行动中，警方查获了266部涉案手机、56部个人手机、3台电脑及大量银行卡和账户资料等关键证据。目前，越南警方已对涉案人员以赌博和洗钱罪名立案调查，并正深入追查境外幕后主脑、相关赌博平台及资金流向。
