# ️ 迪拜华人警惕：两类新型骗局高发，已有损失！

⚠️ 迪拜华人警惕：两类新型骗局高发，已有损失！
近期，迪拜针对华人社区出现两类新型诈骗及抢劫案件，当地警方已发出紧急预警，并提醒在迪拜的中国公民及商家务必提高警惕。已有受害者因此蒙受财产损失。
第一类是“假冒警察”抢劫。骗子通常为外籍人员，会伪装成警察，以检查身份证件为由进行盘查。一旦发现受害者未随身携带证件，便会趁机抢夺手提包或随身物品，并声称需带回警局检查，随后迅速驾车逃离，甚至有案例显示骗子直接驾车出境。
第二类是针对沿街商铺的“送货诈骗陷阱”。据警方反馈，骗子会来店里下单，但要求店员将货物送至指定偏僻地点，并声称现场现金结算。当地警方已上门提醒商户，此举极有可能是专门针对商家的抢劫或诈骗行为。
为有效防范，警方建议：商家若遇陌生客户要求送货至可疑地点并现场结算，宁可放弃交易，切勿让员工冒人身和财产风险送货。个人外出时，尽量避免携带大量现金和贵重物品。如遇盘查，真正的迪拜警察会出示合规证件，且不会在路边直接抢夺包裹。遇到可疑情况，应立即前往人多区域，并尝试记录对方车牌，第一时间拨打999报警。
