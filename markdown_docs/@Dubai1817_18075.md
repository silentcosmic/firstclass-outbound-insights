# 斯里兰卡查获4.5亿卢比中国香烟走私大案

🚨 斯里兰卡查获4.5亿卢比中国香烟走私大案
斯里兰卡海关近日根据国际情报，成功查获一起特大香烟走私案。执法人员在进口冷库板中发现并截获约360万支中国走私香烟，其市场价值估算超过4.5亿斯里兰卡卢比，被认为是该国近年来最大规模的香烟走私案件之一。
此案由斯里兰卡海关中央情报局主导。海关透露，这批申报为“冷库板”的货物原计划运往德希瓦拉的一家公司。检查中，执法人员在冷库板内部查获了超过1.8万条香烟。
当局初步怀疑，部分在斯里兰卡居住或工作的中国公民可能与此走私活动有关联。此次走私若成功，将导致斯里兰卡政府超过4亿卢比的税收损失。目前，案件仍在深入调查中。
