# 中阿基础设施合作再升级：共建“一带一路”

⚡ 中阿基础设施合作再升级：共建“一带一路”
近日，由阿联酋投资部主办的“阿联酋—中国建筑与基础设施圆桌会”在阿布扎比举行。中国驻阿联酋大使曾继新与阿联酋投资部次长哈维等高级官员出席并致辞。
曾大使在会上强调，基础设施合作是中阿全面战略伙伴关系的重要支撑。他指出，中国企业在阿联酋积极参与了道路交通、建筑工程、港口物流、可再生能源及海水淡化等多个关键项目，并已完成一批标志性工程，为两国高质量共建“一带一路”注入了强劲活力。
