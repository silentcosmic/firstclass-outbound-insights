# 曼谷-南昌航班突发：乘客落地后遭警方带走调查

🚨 曼谷-南昌航班突发：乘客落地后遭警方带走调查
据旅客反映，昨日一架从泰国曼谷飞往江西南昌的航班在夜间落地时发生突发情况。乘务员要求所有乘客留在座位上等候，随后警方登机，将前排一名乘客带离，具体性别不详。
另有消息称，在乘客下机通过海关时，有多名警员在现场进行排查。
