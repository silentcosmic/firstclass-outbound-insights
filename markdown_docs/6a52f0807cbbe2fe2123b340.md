# 算力基础设施巨头比特大陆(Bitdeer)在美国内华达州破土动工，投资3600万美元建设先进电子制造工厂

作为一家专注于人工智能及比特币挖矿基础设施的纳斯达克上市公司，总部位于新加坡的比特大陆科技集团（Bitdeer Technologies Group）已在美国内华达州斯帕克斯为其先进电子制造工厂举行了奠基仪式。此项投资总计3600万美元，覆盖厂房、设备及建设。
该占地18.7万平方英尺的设施，是比特大陆在美国境内的首个本土制造与组装基地，将有效补充并强化其位于加利福尼亚州圣何塞的现有数据中心及创新中心。
工厂预计将于2026年底前竣工，建成后每月可生产10,000台比特大陆自主研发的SEALMINER计算硬件。此举还将为当地创造70个就业机会，涵盖工程、熟练技术员及支持等多个职位类别。
比特大陆工业公司董事长保罗·汉森指出，此项设施的建设，反映了公司致力于增强供应链韧性的战略方针，同时也能更紧密地服务并快速响应美国本土客户的需求。在美国内华达州生产先进计算硬件，是公司对构建产能和培养人才的长期承诺，旨在支持美国日益增长的数字基础设施运营。
目前，比特大陆的全球总部设于新加坡，美国总部位于圣何塞，并在美国、挪威、不丹和埃塞俄比亚等地运营着多个先进计算与技术中心。
