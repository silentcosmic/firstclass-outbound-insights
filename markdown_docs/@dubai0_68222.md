# 全球油价跳水，阿联酋7月汽油或大幅降价

⚡ 全球油价跳水，阿联酋7月汽油或大幅降价
随着全球油价自本月初大幅下跌逾20美元，阿联酋的汽油价格预计在2026年7月迎来显著下调，结束连续四个月的上涨趋势。此前，受中东地区局势影响，阿联酋零售燃油价格自2月28日以来累计涨幅已超过60%。
2026年6月，阿联酋零售燃油价格环比上涨约8%，其中Super 98汽油达到每升3.95迪拉姆。然而，受美国与伊朗达成和平协议及霍尔木兹海峡可能重新开放的预期影响，全球油价在整个6月呈下行趋势，布伦特原油价格从月初的每桶95美元跌至74美元以下，6月平均收盘价也从5月的106美元降至约71美元。
盛宝银行大宗商品策略主管奥勒·汉森指出，尽管中东地区此前经历了大规模石油供应中断，但市场焦点已转向即将恢复的供应。随着霍尔木兹海峡船运逐步改善，大量此前滞留的油轮正等待运输，预示着市场可能面临一轮供应激增。瑞士宝盛银行经济与新一代研究主管诺伯特·吕克尔也观察到，中东地区石油出口量已迅速恢复至危机前水平的80%以上，市场正从供应短缺转向供应过剩，库存有望更快回补，但明年仍可能出现库存过剩局面。
