# ️ 迪拜跨境劳务警示：涉嫌诈骗与胁迫案件频发

⚠️ 迪拜跨境劳务警示：涉嫌诈骗与胁迫案件频发
近日，有消息指出迪拜特定服务行业内部出现多起劳务纠纷及诈骗指控。据多名从业者反映，三名主要涉事人员——“菲菲”、“天一”（又名“王建”）及“小天”，涉嫌对多名女性从业者进行财务欺诈、克扣报酬并发出威胁。
其中，“菲菲”被指控在与客人结算时虚报收入，私自截留部分小费。而“天一”则被多方指认为惯犯，此前据称在柬埔寨、中国及泰国等地均有涉嫌诈骗、勒索甚至更严重犯罪行为的记录，并因名声败坏而频繁更换身份及活动地点，目前已转至迪拜继续其不法活动。
另一名涉事者“小天”，也被指控在全球多地故技重施，以各种理由克扣从业者工资，并涉嫌将其挪作私用。受害者称，在追讨报酬时，“小天”不仅拒绝支付，还曾以其在迪拜及国内的“人脉”为由进行恐吓，威胁若报警或回国将采取报复行动。目前，已有数名受害者表示仍有款项未被结清。
鉴于上述严重指控，行业观察人士提醒相关从业者务必提高警惕，注意甄别合作对象，以规避潜在的财务及人身风险。建议受害者通过合法途径维护自身权益。
