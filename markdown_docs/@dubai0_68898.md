# 柬埔寨严打人口贩卖：西港“旧账”浮现

🚨 柬埔寨严打人口贩卖：西港“旧账”浮现
市场消息指出，约半个月前，柬埔寨西哈努克港（西港）一名知名女商人被捕并已遣返回国。据传，其被捕原因与早年间涉嫌人口贩卖有关。
此案引发当地商界广泛关注。有分析认为，若此指控属实，恐将对西港一批曾明面经营实业、暗地里却在疫情期间通过贩卖人口维系生意的老板们构成警示。这些“陈年旧账”正逐步被警方重新审视。
随着大规模涉诈人员包机回国，以及柬埔寨与中国在打击跨境犯罪方面的合作深度加强，许多类似案件或将浮出水面。柬埔寨当局在引渡相关嫌疑人方面表现出高度配合，使得跨境追责更为迅速有效。
值得注意的是，对于部分涉事人员可能提出的“自愿”或“自卖”辩解，业内人士普遍认为，在法律审判中，受害者一方的证词更容易被采信，此类辩解恐难被司法机构接受。
