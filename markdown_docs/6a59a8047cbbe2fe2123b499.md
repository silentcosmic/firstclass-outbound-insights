# 苹果加速布局AI算力：全球寻觅芯片收购标的

长期以来，苹果公司在企业并购上向来审慎，通常不会大手笔进行外部收购。然而，随着其在全球AI竞赛中寻求显著提升自身AI处理能力，这一传统策略正面临调整。有迹象表明，苹果已开始积极与全球范围内的半导体制造商及金融机构展开洽谈，旨在通过潜在收购，迅速强化其在服务器端的性能表现。
目前，苹果在部分AI任务上依赖其M2 Ultra芯片驱动的服务器，但这些系统在处理更繁重的计算负载时，已暴露出明显的性能瓶颈。例如，驱动Siri背后复杂AI模型（如Gemini）的重度计算任务，目前似乎仍需借助谷歌云平台上英伟达芯片的强大算力。据了解，苹果曾尝试将此类高强度任务内部化，然而其现有基础设施显然未能满足需求。
尽管有消息指出，基于M7 Ultra的下一代服务器芯片预计要到2029年方能面世，但苹果正计划通过M5 Ultra芯片对现有基础设施进行中期升级。此前，苹果曾规划于今年推出代号为“Baltra”的下一代服务器芯片，但这一发布时间表似乎已有所推迟。在供应链层面，苹果近期也与博通公司达成了一项高达300亿美元的长期协议，以确保后者在美国生产的特定芯片的稳定供应。
苹果在芯片设计领域的专长，历来主要聚焦于消费电子设备。因此，该公司寻求在服务器端获得更强大的技术支持，以应对日益增长的AI计算需求，是其战略的必然延伸。苹果首次涉足芯片制造，可追溯至2008年以2.78亿美元收购PA Semi。即便并购风格一贯稳健，但近年来其在AI领域的投资明显提速。今年，苹果斥资近20亿美元收购了AI初创公司Q.ai，这已是继十多年前30亿美元收购Beats之后，苹果历史上的第二大并购案。
鉴于先进芯片技术在当前AI产业中的核心战略地位，苹果未来在这一领域的任何潜在收购，无疑可能需要付出显著的溢价。即便如此，苹果在财务层面拥有充裕的灵活性以支撑其战略雄心。截至今年3月底，苹果公司拥有高达456亿美元的现金及现金等价物，为其在全球AI芯片市场的布局提供了坚实的资金保障。
