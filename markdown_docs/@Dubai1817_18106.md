# 湖南警方捣毁裸聊引流窝点：跨境诈骗链条曝光

🚨 湖南警方捣毁裸聊引流窝点：跨境诈骗链条曝光
近日，湖南常德汉寿县警方成功捣毁一处裸聊引流电信诈骗窝点，抓获3名主要嫌疑人。此次行动系警方在日常线索摸排中发现异常后，于7月12日展开收网，现场查获电脑及VPN设备等作案工具。
经查，该团伙自6月26日起，由主犯帅某朋纠集人员，私自搭建平台对接境外电信诈骗团伙。他们利用境外社交平台注册上千个虚拟女性账号，诱导男性网民裸聊，同步窃取通讯录并录制不雅视频，随后将受害人信息移交境外团伙实施敲诈勒索，从中赚取引流佣金。
目前，帅某朋等3名嫌疑人因涉嫌非法利用信息网络罪已被刑事拘留。警方正同步深挖涉案流水、上下游犯罪链条，并推进追赃挽损工作。
