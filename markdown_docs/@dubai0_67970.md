# ️ X平台全球故障波及阿联酋：通知正常内容不载

⚠️ X平台全球故障波及阿联酋：通知正常内容不载
6月22日，社交媒体平台X（原Twitter）遭遇全球性故障，其中阿联酋用户受影响尤为显著。数万份故障报告指出，当地居民虽能正常接收到通知提醒，但点击后内容却无法加载。
许多用户点击通知后，页面显示“帖子不可用”的错误信息。此次大范围的服务中断给依赖X平台进行信息获取和商务交流的用户带来了不便，引发了广泛关注。
