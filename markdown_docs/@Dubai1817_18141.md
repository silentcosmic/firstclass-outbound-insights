# 柬埔寨免签满月：客流量不升反降，旅游业承压

📉 柬埔寨免签满月：客流量不升反降，旅游业承压
柬埔寨自6月15日对中国游客实施免签政策，然而，政策推行一个月后公布的数据却令人意外。全国旅客量不增反降7%，其中6月份单月旅客吞吐量同比锐减11%，仅录得39.5万人次。这表明，免签政策未能如预期般有效提振当地旅游市场。
与客运市场的冷清形成鲜明对比的是，柬埔寨航空货运表现强劲。上半年航空货运量同比增长29%，其中国际货运业务的97.9%经由德崇机场完成，物流业已然成为机场运营的重要支柱。然而，这一增长未能带动低迷的游客数量。
尽管旅游数据不尽如人意，但免签政策在促进中柬商务往来及长期合作方面展现出积极作用。它极大便利了中国商人赴柬考察投资，也为游客和探亲人士省去了繁琐的签证办理流程，有助于两国间营造更加友好的交流氛围。
然而，深层问题仍未解决。受中东局势影响，航班需绕道导致机票价格飙升，这无疑提高了旅行成本。同时，当地酒店、景区及交通等旅游配套设施建设滞后，难以满足市场需求。此外，政策在中国的宣传力度不足，许多潜在游客对此知之甚少。过度依赖单一客源市场，缺乏多元化战略，也使柬埔寨旅游业面临结构性挑战。
综合来看，高昂的机票价格、政策的商务侧重、配套设施的不足以及宣传的缺位，多重因素叠加使得单一的免签政策难以扭转局面。若要真正激活旅游市场，柬埔寨仍需在降低机票成本、升级旅游配套服务及加大市场推广力度等方面持续发力。
