# ️ 国泰航空官宣：2026年9月重返中东航线！

✈️ 国泰航空官宣：2026年9月重返中东航线！
国泰航空近日宣布，计划自2026年9月1日起，全面恢复其中东地区的客运服务。届时，将提供每日往返迪拜的航班，以及每周四班往返沙特阿拉伯首都利雅得的航班。目前，相关航班已开放预订。
公司方面同时指出，在此期间将持续密切关注中东地区局势的最新发展，以确保运营安全与服务质量。
