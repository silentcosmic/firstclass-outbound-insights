# 中国红通富豪在泰落网！

🚨 中国红通富豪在泰落网！
6月24日，泰国移民局第一分局成功逮捕了一名被国际刑警组织列入红色通缉令的中国籍嫌犯“冯某”。据了解，冯某曾是中国西部地区知名企业家，身家高达数百亿泰铢。他因涉嫌伪造公司印章等多项罪名受到中方通缉，并在案发后潜逃出境。
