# 日本情感诈骗：女大学生半年骗走200万日元

🚨 日本情感诈骗：女大学生半年骗走200万日元
日本横滨近日曝出一起离奇的情感诈骗案。一名22岁的女大学生通过交友软件，对一位70多岁的丧偶老人实施诈骗。在短短半年多时间内，该学生以学费、住院费、考试费等多种名义，从老人处骗取了近200万日元。
诈骗得手后，这名女大学生甚至假冒其母亲，向受害老人发送消息称“女儿已去世”，试图彻底切断联系。老人因此察觉异样并报警，事件才得以曝光。
涉案女大学生广重果音随后被捕并对罪行供认不讳。此案因其情节离奇以及广重果音在被捕时“极具冲击力”的眼神在社交媒体上引发广泛关注。此事件再次提醒公众，网络交友风险不容忽视，尤其涉及金钱往来时更需警惕。
