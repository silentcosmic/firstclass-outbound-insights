# 柬埔寨总理访华：深化“钻石六边”合作

⚡ 柬埔寨总理访华：深化“钻石六边”合作
柬埔寨总理洪玛奈于7月15日上午率领高级代表团启程，对中国展开为期三天的正式工作访问。此次行程将涵盖北京、上海两地，旨在进一步深化柬中双边合作，推动两国各领域交流迈向更高水平。
柬方明确表示，此访是落实两国高层交往共识的关键举措，将持续深化“钻石六边”合作框架，推动新时代柬中命运共同体建设取得更多务实成果，为双边关系发展注入全新动力。
