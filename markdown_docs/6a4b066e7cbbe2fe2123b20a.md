# 生物基皮革引领者贻如科技完成超亿元A轮融资，鄂尔多斯、和达金服联合领投

近日，生物基皮革创新企业贻如科技宣布完成超亿元人民币A轮融资。本轮投资由鄂尔多斯集团与和达金服共同领投，巢生资本跟投，易凯资本担任独家财务顾问。此轮募集资金将主要用于加速公司的商业化拓展与产能建设，并深度融合AI技术，以推动生物基皮革的技术革新与产品迭代。
贻如科技成立于2021年，其核心技术是基于合成生物学，旨在开发以生物基皮革为代表的新一代创新生物基材料。公司通过微生物发酵直接生产生物基树脂原料，再经加工制成皮革成品。这一创新工艺显著降低了碳排放，减排幅度超过80%。值得注意的是，部分产品的生产成本甚至低于传统油性PU产品，同时在物理性能（如耐磨性、透气性、柔韧性）上已达到真皮水准。这使得大众消费品所需的高性能材料能够从石油基转向生物基，实现可持续发展。
早在2024年，贻如科技便已完成由星空资本领投的数千万元Pre-A轮融资。当时，公司创始人苏睿曾预判商业化拐点“或许明年到来”。如今，这家曾长期低调、鲜少公开露面的公司，其全球范围内的商业落地已实现显著突破。截至2026年第一季度，其皮革产线年产能预计将突破1600万平方米，今年以来海外订单已接近亿元人民币，标志着其全球化布局的序幕正式拉开。
在过去十年中，生物基皮革，特别是以菌丝体为原料的“蘑菇皮”，一度是时尚可持续发展领域备受瞩目的方向。MycoWorks、Modern Meadow、Ecovative等海外公司曾获得顶级机构或奢侈品集团的青睐和投资。
然而，在初期热潮消退后，这些明星项目大多未能持续发展。尽管材料本身能够研发成功，但其高昂的成本和难以实现规模化量产的瓶颈，使其产品多停留于概念展示或小批量联名款。因此，“生物基皮革更昂贵”也成了消费者和品牌之间一个心照不宣的“常识”。
时至今日，传统皮革产业普遍面临着碳排放量高、原材料供给波动大以及品牌方ESG（环境、社会和公司治理）压力日益上升等问题，市场对生物基皮革的兴趣并未减退。但行业评估标准已经从“能否成功研发”转向“能否以低成本实现规模化生产”。
技术创新与成本优势
成本控制正是贻如科技当前的核心竞争壁垒。创始人苏睿曾详细阐述了公司的成本优势：在欧洲市场，品牌使用真皮的成本折合每米约为300-400元人民币；而贻如科技售价最高的皮革产品约200元/米，多数产品价格区间在80-150元/米。这意味着，其产品成本可压低至真皮的一半以下，同时还具备透水、透气且比真皮更耐用的特性。
这背后深层的原因在于，在全球范围内，能够将生物基皮革做到规模化量产的公司寥寥无几。苏睿表示，行业内能稳定交付百万米以上产品的企业屈指可数，贻如科技是极少数最早投入并长期坚持“液体发酵制备树脂、再涂覆成革”这条技术路径的企业之一。
推动成本曲线持续下探的另一项关键因素是AI技术。从菌株筛选、基因编辑设计，到液体发酵工艺参数的优化和量产良率的控制，贻如科技构建了一套全链路数据驱动的研发引擎，将生物材料的开发效率与放大成功率推向了行业前沿。
在产品布局方面，贻如科技以Mulkol系列实现了差异化定位：Mulkol-3系列定位于平价人造革替代品，采用偏硬质树脂，质感接近高强度工业用料，主要面向家具、装饰等平价人造革替代场景；Mulkol-5系列则定位于高级真皮替代品，采用更柔软的弹性体材料，触感更接近羊皮与真皮，主攻手袋、服装等中高端市场。苏睿还透露，为满足海外奢侈品牌客户的严苛需求，公司另外推出了更高端的HORA品牌。贻如科技通过稳定的产品供给与市场拓展，持续深耕奢侈品及品牌消费市场，逐步构建起成熟的业务基础与品牌影响力。
在此基础上，贻如科技正加速向更广阔的高分子生物基材料领域延展，逐步向上游材料端布局，着力构建覆盖全球的弹性体原料体系。同时，公司也在同步拓展膜材料与橡胶材料等多元应用方向。这些相关材料将进一步走向消费品包装、工业薄膜及各类终端应用场景，持续拓宽产业边界与应用空间。
贻如科技的自有研发团队围绕不同应用场景与性能需求，持续推进材料开发与技术迭代，不断完善弹性体原料的自主研发能力与产品体系。这推动公司从皮革业务出发，逐步迈向以原料开发为核心的材料与生物基创新平台型企业。在此方向上，鄂尔多斯集团旗下的电力冶金集团在可持续能源、低碳冶金及绿色化工材料领域积累深厚，并设有合成生物产业专业化平台，将与贻如科技在从上游原料、中游生产到下游产品转化的全链条上形成互补协同。
全球市场战略与未来展望
此外，欧洲市场是贻如科技最重要的战略区域之一。欧洲拥有众多高端时尚消费品牌，对低碳、可持续材料有着强烈且迫切的需求，使其成为生物基皮革最具价值的战场。
据悉，贻如科技已在法国设立全资子公司SYNMETABIO，以此作为欧洲区域总部支点，联动英国、德国、意大利、土耳其及北欧各地的合作渠道，全面辐射欧洲主要消费市场。在欧盟碳关税（CBAM）的政策背景下，贻如科技凭借其材料在降碳和可持续性方面的显著优势，能够为欧洲客户带来切实的成本优势和强大的品牌宣传效应。
苏睿透露，国际地缘冲突导致传统供应链成本上升，贻如科技的欧洲订单随之显著跃升。法国全资子公司SYNMETABIO累计接到的订单已接近亿元人民币，且“几乎全部来自奢侈品牌”，这充分印证了其在高端市场的竞争力。
目前，境外订单已占据贻如科技总业务量的约三分之二。为此，公司已组建海外质检、海关与运输团队，以应对日益增长的国际业务需求。近半年来，贻如科技的团队规模也从约20人迅速扩充至近70人。
业务的快速扩张在各个方面都有所体现，而支撑这些订单的核心在于产能保障。贻如科技与上市公司聚杰微纤以轻资产模式合资成立了安徽聚贻科技，目前皮革产线年产能已突破1600万平方米，二期工厂也正在建设中。据苏睿介绍，公司下半年还将陆续投产新的制成品工厂，为未来的增长奠定基础。
苏睿坦言，贻如科技将把2026年视为“筹备上市的第一年”。对于一家2021年成立、刚跑通工厂级规模的公司而言，这一目标显得颇具雄心。在当前行业从“环保叙事”回归“成本与量产”的背景下，一家能够将皮革成本控制在真皮一半以下，并已在欧洲市场斩获近亿元订单的中国公司，无疑为全球生物基材料产业提供了一个值得深入观察的新样本。
