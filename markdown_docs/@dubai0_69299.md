# 柬埔寨推广助理街头遇袭，疑涉离职纠纷

🚨 柬埔寨推广助理街头遇袭，疑涉离职纠纷
市场消息指出，柬埔寨近日发生一起街头暴力事件。一名在当地某云公司任职的推广助理，在街头遭遇袭击，伤势 reportedly惨重。事发地点据称在海悦附近。
有消息称，此事件可能与推广人员的离职流程沟通不畅引发的纠纷有关。据称，此举系另一位推广人员因离职流程沟通问题，情绪失控后所为。此次事件也再次警示出海企业，在人员管理和内部沟通上务必加强重视，以避免类似冲突，确保员工安全。
