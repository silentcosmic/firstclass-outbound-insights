# 比特币跌破6万：近18万人爆仓，市场蒸发近10亿！

⚡ 比特币跌破6万：近18万人爆仓，市场蒸发近10亿！
6月24日，比特币价格突然大幅下挫，盘中一度跌至约5.9万美元，创下数月以来新低。尽管随后有所反弹，回到6.1万美元附近，但整体市场波动依然剧烈。
受此轮急跌影响，大量使用杠杆的加密货币投资者遭遇强制平仓。在过去24小时内，整个币圈爆仓人数接近18万，总金额高达9.84亿美元。其中，押注比特币上涨的多头头寸损失最为惨重，爆仓金额约7.99亿美元。
分析认为，此次下跌与全球科技股走弱、美元走强、加密货币ETF资金持续流出，以及市场对美联储可能继续加息的担忧等多重因素交织有关。
