# 东南亚电诈链条遭重击：关键招募者在泰被捕

⚡ 东南亚电诈链条遭重击：关键招募者在泰被捕
泰国警方近日在春武里府成功逮捕一名涉嫌组织跨国人口贩运的中国籍女子吕若曦，此案揭露了柬埔寨、老挝及泰国东部多个电信网络诈骗窝点背后的犯罪链条。
据了解，此次行动是泰国网络警察与柬埔寨反人口贩卖专案组历经半年联合侦查的突破性成果。警方调查显示，吕若曦长期活跃于社交平台，利用“高薪招聘”、“远程工作”等虚假信息，诱骗中国、越南、印度尼西亚等国求职者。待受害人同意出境后，她便组织非法偷渡，将其转运至隐藏在多国的诈骗窝点，强迫从事非法活动。
据悉，泰国刑事法院早于去年12月便对吕若曦签发了逮捕令。目前，检方正与中国相关部门紧密协调，以依法推进后续司法程序。泰国警方强调，此次抓捕成功切断了跨境电信网络诈骗团伙招募及输送劳工的关键环节，未来将持续开展专项行动，深入追查并打击相关招工中介及跨国犯罪网络。
