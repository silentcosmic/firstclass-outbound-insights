# ️ 中国多地突查手机应用：数字通信合规风险升高

⚠️ 中国多地突查手机应用：数字通信合规风险升高
近期市场消息显示，中国部分地区对个人数字通信工具的审查力度有所加强。据福建一匿名消息人士透露，当地警方近期在夜场包厢进行突击检查，发现手机安装特定境外通讯应用的个人被当场带走调查，且此类检查并非个例，涉及多个场所。
此外，该消息还指出，在写字楼区域也出现了类似的随机抽查。有电商行业从业者反映，警方在写字楼下班时段对路人手机进行抽检，重点核查是否安装了特定境外通讯软件，并对发现安装者采取了进一步调查措施。
这些迹象可能预示着中国境内对数字通信和信息流动的管控正在收紧。对于在华运营的企业和个人而言，这提示了潜在的合规风险和隐私考量。建议密切关注相关政策动向，审慎评估和调整数字通信策略。
