# 习近平会见洪森，深化中柬高层友好访问

🤝 习近平会见洪森，深化中柬高层友好访问
6月26日上午，中共中央总书记、国家主席习近平在北京人民大会堂会见了到访的柬埔寨人民党主席、参议院主席洪森。
洪森此次应中国共产党邀请，于6月25日至27日对中国进行为期三天的正式友好访问。此次高层会晤已获官方确认，双方具体会谈内容及相关合作成果，市场正密切关注后续公布。
