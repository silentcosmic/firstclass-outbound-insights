# ️ 柬埔寨警示：警惕电诈集团高薪招募陷阱

⚠️ 柬埔寨警示：警惕电诈集团高薪招募陷阱
柬埔寨国家警察总署副总监迪伟杰中将（洪森女婿）于7月15日通过个人社交平台发出警告，指出当前有网络诈骗团伙正以每月300至600美元的虚假高薪，诱骗柬埔寨民众参与非法活动。
迪伟杰强调，这些诈骗集团惯用“高薪、轻松工作”等虚假招聘信息吸引当地居民，实则旨在诱导他们从事电信网络诈骗等违法犯罪行为。
他呼吁民众务必提高警惕，切勿被高薪诱惑而触犯法律，并鼓励积极举报相关线索，配合执法部门共同打击电信网络诈骗犯罪，以维护社会秩序和公共安全。
