# 涉150亿美元加密诈骗，太子集团二号人物日本落网！

🚨 涉150亿美元加密诈骗，太子集团二号人物日本落网！
一名被指为“太子集团”二号人物的男子近日在日本被捕，该集团涉嫌通过人口贩运，构建了一个规模高达150亿美元的加密货币诈骗帝国。此案此前因美国史上最大规模的比特币查扣而引起国际关注。
这名嫌疑人以胡晓伟、胡石等多个假身份入境日本，并持有“高度专业人才”签证，同时申请日本永久居留权。他在东京经营一家注册资本从800万日元增至5000万日元的空壳公司，并频繁入住多家顶级豪华酒店。
日本警方已对其监控多时，通过酒店监控摄像头追踪其行踪，并于今年6月在大阪展开抓捕行动。本周，他因涉嫌指使同伙冒充自己办理手续而再次被捕。面对审讯，他声称来日本是为了“寻找一个能够安全生活的地方”。
