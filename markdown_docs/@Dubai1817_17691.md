# 柬埔寨波贝惊现两具肢解男尸，警方介入调查

🚨 柬埔寨波贝惊现两具肢解男尸，警方介入调查
柬埔寨波贝市近日发生一起骇人听闻的凶案。据当地媒体报道，在靠近沙拉干达分区安栋特莫美村的一处荒林旁，当地居民发现两具遭肢解的男性尸体，引发社会高度关注。
事件发生于7月2日，村民因闻到异味并发现野狗从林中拖出可疑黑色塑料袋，靠近查看后赫然发现袋内装有多块人体残肢及两颗头颅。初步判断，受害者为两名皮肤白皙、身上有纹身的男性，疑为外籍人士，死亡时间约3至4天。
案发后，省市两级警方技术部门已连续两天在现场进行勘查取证。目前，警方已全面介入，正全力追查死者身份及凶手下落。
