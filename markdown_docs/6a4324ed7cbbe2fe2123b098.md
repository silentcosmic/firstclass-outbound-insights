# 全球产业瞭望 | 百度OCR模型技术突破；中国教育强国战略布局；美国养老金市场异动

茶饮巨头蜜雪冰城加速中亚及全球市场布局
出海 · 头等舱观察到，中国知名茶饮品牌蜜雪冰城近期正式进驻吉尔吉斯斯坦市场，其首批三家门店已在该国首都比什凯克开业。吉尔吉斯斯坦是蜜雪冰城在中亚地区的第二站，此前其哈萨克斯坦首店已于2025年4月在阿拉木图落地。此后，蜜雪冰城还相继拓展至美国、巴西、墨西哥等重要海外市场，持续深化其全球化战略布局。截至目前，该品牌已成功进入全球16个国家及地区。
中国移动设立Token办公室，深化数字战略布局
最新消息显示，中国移动在其数智事业部与算力办公室之后，近期又增设了Token办公室。此举旨在全面打通“Token的创造、输送与应用”全链条，以期打破此前多个二级部门各自为政的局面。据内部人士证实，中国移动确已成立Token办公室，但关于该办公室级别更高、由集团核心领导直接挂帅等传闻，尚未得到官方进一步确认。
英伟达携手Firmus在印尼打造300亿美元AI数据中心园区
获英伟达支持的数据中心运营商Firmus宣布，将在印度尼西亚启动一项大型人工智能数据中心园区建设项目。该公司预计，此项目在前六年内有望带来高达300亿美元的合作协议。这家澳大利亚企业通过声明确认，已与英伟达签署了一份为期八年的战略合作协议，共同在与新加坡隔海相望的印尼巴淡岛，打造一个装机容量达360兆瓦的专属英伟达DSX AI工厂园区。Firmus进一步披露，该里程碑式的超大规模数据中心项目将与总部位于新加坡的DayOne公司合作开发。
百度开源Unlimited OCR模型，刷新端到端OCR新纪录
国内科技巨头百度近期正式发布并开源了其端到端光学字符识别（OCR）模型——Unlimited OCR。该模型发布后迅速引起业界关注，在多个知名技术社区和模型趋势榜单上均取得亮眼成绩，位居前列。Unlimited OCR专为长文档解析场景设计，其总参数规模达到30亿，而在实际推理时激活参数仅为5.7亿。根据公开评测结果，Unlimited OCR在OmniDocBench v1.6基准测试中获得了93.92%的综合分数，成功刷新了端到端OCR领域的最新纪录。
马斯克宣布Grok 4.5内部测试，SpaceX拟每月推出全新AI模型
当地时间周日（28日），埃隆·马斯克通过其社交媒体平台对外披露，他旗下最新一代大型语言模型Grok 4.5已正式在SpaceX和特斯拉内部启动Beta测试，此后将逐步进行更广泛的推广。马斯克同时透露，SpaceX计划在今年剩余月份中，每月推出一个“完全从零开始训练”的全新人工智能模型。此表态预示，未来的Grok版本或将不仅仅是现有系统的迭代升级，而是通过全新训练范式构建的全新基础模型。
URTOPIA完成逾2亿元B轮融资，发力户外具身智能
全球智能电助力自行车品牌URTOPIA近期宣布完成B轮融资，融资总额突破2亿元人民币。本轮融资由Brizan Ventures和桐乡市政府产业基金联合领投，并获得了高秉强、Kungho Fund、光远和声等机构的持续加注。URTOPIA表示，此次融资所得将主要用于进一步加强核心技术研发、加速全球市场布局、优化供应链体系以及提升组织能力。公司将以此为契机，推动自身从单纯的智能电助力自行车制造商，转型成为一家以户外具身智能与人机耦合智能运动技术为核心的高科技企业。
算力成本推高AI服务价格，业界探讨行业泡沫与基本面
月之暗面Kimi B端负责人黄震昕在最近一次行业交流活动中直言：所有模型服务商今年均面临价格上涨压力，其核心动因在于全球范围内算力成本的攀升，无论是海外还是国内的芯片供应，都难以满足Token需求日益增长的速度。他指出，随着下游企业对AI应用的需求呈爆发式增长，而上游算力供给受限于芯片产能持续紧张，供需两端的双重压力下，API价格的上涨已是市场必然趋势。对于外界普遍担忧的行业泡沫问题，黄震昕坦诚道：“现在肯定存在泡沫。”但他随即强调：“Anthropic已经实现了季度盈利，其收益是实实在在的。”他认为，尽管泡沫存在，但行业的基本面并非空洞，当生产力真正发生质变时，能够计算清楚投入产出账本的企业将持续加大投入，这正是衡量行业健康发展的重要信号。
国务院印发《教育发展“十五五”规划》，擘画教育强国蓝图
国务院近期正式发布了《教育发展“十五五”规划》，为“十五五”时期加速构建教育强国明确了核心目标、战略任务、政策方针与重大项目。该规划系统性地部署了教育资源的前瞻性配置、区域布局强化以及人才培养结构优化等工作，以期适应学龄人口结构变化，有力支撑国家重大战略，并服务于现代化产业体系的建设。规划展望，到2030年，中国教育强国建设将取得显著成效，高质量教育体系基本形成。届时，党对教育事业全面领导的制度体系和工作机制将进一步完善，德智体美劳全面培养体系将更为健全，立德树人成效将更加彰显，人民群众的教育获得感显著提升，教育对科技创新与人才发展的支撑作用将愈发强劲，学习型社会将基本建成，中国教育的国际影响力、竞争力和话语权亦将大幅提升。
三大专业展会联袂亮相深圳，赋能粤港澳大湾区产业
2026深圳食博会、深圳渔博会及深圳春季茶博会定于7月2日在深圳会展中心（福田）同期盛大开幕。据悉，这三大专业展会由华巨臣主承办，旨在充分发挥粤港澳大湾区世界级消费市场与产业枢纽的独特优势，汇聚全球优质食材、尖端科技企业与地方特色产业集群，共同构建食品、渔业、茶业领域的综合性商贸交流平台。此次三展联动将吸引逾1500家参展商，覆盖全球30多个国家和地区，并配套举办20余场行业专题论坛，全力促进产业资源的有效对接与高效落地。
气候变化加剧欧洲极端高温，全球应对挑战迫在眉睫
继5月的酷热之后，欧洲在6月再次遭遇闷热天气，多个地区出现创纪录高温。这种现象并非孤立事件，而是气候变化导致极端天气日益频繁且持续时间更长的直接体现。从联合国官员到气候科学家，各方均已发出警示，强调全球必须加速应对气候挑战的步伐。世界天气归因联盟于26日发布的一项最新研究报告揭示，受气候变化驱动，欧洲出现高温天气的概率已是20年前的200倍。
韩国总统李在明：将斥巨资扩大西南地区芯片产能
韩国总统李在明近日表示，芯片与人工智能领域的全球竞争异常激烈，韩国亟需加速芯片生产基地建设以满足日益增长的需求。他进一步指出，韩国将通过对西南部地区的巨额投资来扩大芯片供应能力，该地区在新项目中的投资规模预计将达到5万亿至20万亿韩元。具体而言，光州和全罗道地区在相关项目中的投资潜力可能高达520万亿韩元。
美国养老金面临300亿美元股票抛售，市场再平衡在即
随着2026年上半年进入尾声，美国各大养老金计划正准备启动其半年度资产再平衡操作。市场分析师预估，本周一和周二两天内，预计将有高达300亿美元规模的美国股票资产面临集中式强制抛售。鉴于美国公共养老金、企业固定收益养老计划以及目标日期基金普遍遵循着60%股票与40%固定收益资产的刚性配置框架，且此比例已明确载入其机构投资政策文件。目前，美国大型养老金的股票仓位已显著超出预设比例，而债券配置则明显不足。因此，在这一半年度结算窗口的最后两天，华尔街分析师普遍认为，一场规模达300亿美元的大型抛售冲击已不可避免。
