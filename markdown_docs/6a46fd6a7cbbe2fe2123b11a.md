# Grab正式完成对美国投资平台Stash Financial的收购

亚洲领先的超级应用Grab已正式完成对美国投资平台Stash Financial的收购。此次交易已于2026年7月1日正式交割，Grab在交割时支付了50.1%的股权对价。
剩余股权的支付将在未来三年内，按照公允市场价值进行。Grab此前已签署协议，计划收购Stash的100%股权，其中首批50.1%的股权企业估值为4.25亿美元。
Stash是一家美国数字金融服务公司，通过其订阅制应用程序，提供投资、银行服务以及金融教育工具。该公司目前管理着超过50亿美元的资产，并拥有超过100万的订阅用户。该平台还提供AI智能理财教练（AI Money Coach）服务，为用户提供个性化的财务指导。
Grab明确表示，收购完成后，Stash将继续作为独立品牌在美国市场运营。同时，Grab计划支持Stash在美国市场的进一步增长，并将在长期内，探索将其投资解决方案引入东南亚市场的可能性。
此次收购标志着Grab的金融服务业务从其主要的东南亚市场向外扩张，此前Grab已在东南亚提供支付、借贷、保险和数字银行服务。
