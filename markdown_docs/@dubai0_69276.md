# 电诈团伙转战斯里兰卡：豪华酒店成新窝点

🚨 电诈团伙转战斯里兰卡：豪华酒店成新窝点
随着缅甸、柬埔寨、老挝等东南亚国家持续加大对电信诈骗园区的打击力度，最新调查显示，部分跨国诈骗团伙正将运营据点转移至斯里兰卡。这些团伙疑似在科伦坡的豪华公寓、康提的别墅以及沿海酒店内设立诈骗中心，从事“杀猪盘”、虚假投资及钓鱼链接等各类诈骗活动。
斯里兰卡媒体的调查揭示，自2024年以来，当地警方已连续捣毁多个诈骗窝点，累计逮捕数百名外籍嫌疑人，其中中国籍公民数量最多。今年4月，警方在一项针对某豪华酒店的突击行动中，一次性抓获150人，包括135名中国公民，并现场查获大量电脑、手机及SIM Box设备。
报道进一步指出，部分团伙利用旅游或商务签证入境，并以高薪招聘为诱饵，将受害者集中安置于酒店或别墅内实施诈骗。诈骗所得随后通过USDT等加密货币进行多层转移，最终流向境外。调查认为，斯里兰卡在签证政策、租房监管及资金流动方面存在的漏洞，为诈骗团伙的隐藏活动提供了便利。斯里兰卡警方也坦承，在追踪加密货币、跨境取证以及追捕幕后主犯方面，仍面临显著挑战。
斯里兰卡媒体警告，若不能进一步加强签证、通信及资金监管，斯里兰卡恐将逐渐沦为跨国电信诈骗团伙的新据点。然而，关于这些团伙与东南亚原有诈骗网络之间是否存在关联，仍有待执法部门的进一步调查确认。
