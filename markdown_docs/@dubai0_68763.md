# ️ 阿联酋警方警告：贩毒网络瞄准青少年线上诱惑

⚠️ 阿联酋警方警告：贩毒网络瞄准青少年线上诱惑
阿联酋警方近日发出警告，指出贩毒网络正日益通过数字平台锁定并诱骗年轻人。当局表示，这些平台包括社交媒体、即时通讯应用乃至在线游戏等多种数字渠道。
为应对此趋势，阿联酋政府已大幅扩大禁毒宣传力度，在学校、大学、购物中心及各类社区活动中，向公众普及毒品对健康、社会及法律层面的严重危害，以提高公众警惕。
