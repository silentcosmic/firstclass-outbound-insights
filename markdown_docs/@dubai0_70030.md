# 迪拜夏日送清凉：逾万户外工人将获冰淇淋慰问

💖 迪拜夏日送清凉：逾万户外工人将获冰淇淋慰问
迪拜近日再次启动年度夏季冰淇淋派发活动，旨在为城中超过1.9万名户外工作者送去清凉与关怀。这项名为“用冰凉甜品打败高温”的社区活动，由在政府社区发展局正式备案的“Spread Kindness（传播善意）”志愿团队主办。
该团队创始人Ayesha表示，活动初衷是为在酷热下辛勤劳作的工人提供简单的降温方式，并为他们带去一份快乐。此举彰显了迪拜社区对基层劳动者的持续关注与人文关怀。
