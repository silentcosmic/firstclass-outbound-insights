# 美国头部风投机构携手新加坡国立大学，启动1.17亿美元深科技投资新篇章

总部位于美国的风险投资机构Playground Global和Matter Venture Partners，已与新加坡国立大学（NUS）的创业核心——国大企业机构（NUS Enterprise）达成战略合作，共同加入其规模达1.17亿美元的国大风投计划（NUS VC Program）。
凭借此次战略携手，这两家风投机构将更深入地接触新加坡的深科技生态系统，及其在人工智能、生物科技、量子技术和先进材料领域开发突破性创新的初创企业储备。新加坡国立大学周四发布声明指出，这一合作将显著提升双方的协同效应。
业界分析指出，对于美国投资者和企业而言，新加坡是通往亚洲创新经济的战略门户。
这座城邦目前已汇聚逾4,500家科技初创企业，并已在其“研究、创新与企业2030计划（RIE 2030）”中投入约290亿美元，以期深度提升国家创新能力。
许多新加坡初创企业自创立之初便着眼于全球扩张，这为美国市场提供了早期接触经亚洲市场动态、区域供应链及多元监管环境打磨的先进技术的机会。
通过这些合作，Playground Global和Matter Venture Partners将获得优先接触新加坡创新生态系统中经过预筛选的高潜力深科技公司的权利。
反之，这些初创企业也将从硅谷的风险投资网络、市场洞察、技术专长及企业合作中获益，从而优化其产品市场契合度，并加速国际化增长。
Playground Global总部位于硅谷，是一家专注于早期投资的风投机构，致力于支持在工程和科学前沿领域最具技术抱负的创始人。
该机构以其将突破性技术转化为全球竞争力的企业而闻名。
其投资组合中约有五分之一的公司已达到独角兽估值，管理资产规模达17亿美元。Playground专注于投资塑造下一代计算、自动化、能源和生物工程未来的早期公司。
“构建具有全球竞争力的科技企业，需要长期的支持、深厚的技术专长以及广泛的国际网络，”Playground Global普通合伙人Bruce Leak表示，“我们期待能协助更多全球创始人顺利走完这段征程。”
Matter Venture Partners则通过一个技术深厚且全球互联的平台来补充这一优势，该平台覆盖硅谷以及亚洲及其他地区的重要工业生态系统，包括中国台湾、日本、韩国、欧洲和阿拉伯联合酋长国。
该机构专注于与“硬科技（HardTech）”领域的创始人合作，这些创始人致力于开发先进半导体、电子产品和物理人工智能（Physical AI）创新。
Matter团队提供亲身指导和战略合作伙伴网络的连接，以帮助创始人驾驭供应链、扩大生产规模并将复杂技术推向市场，其核心能力涵盖机器人技术、先进材料和下一代制造等领域。
“下一代具有全球影响力的硬科技企业将跨生态系统而生，而非局限于单一地理区域，”Matter Venture Partners创始管理合伙人谢文博士表示，“新加坡正是这一网络中的关键节点，人才、资本、技术和雄心在此汇聚。”
他认为此次合作将使该机构能够支持那些从创业伊始便怀抱全球视野的新加坡硬科技创始人，通过将其与合适的合作伙伴、制造网络以及规模化成功所需的市场准入相连接。
声明指出，国大企业机构还将在Playground Global位于硅谷、占地7万平方英尺的孵化设施“The Studio”设立其首个全球海外办事处。
凭借湿式和干式实验室、先进原型制造车间及精密工程工具等资源，获得国大支持的创始人将拥有验证技术、接触美国客户并优化产品市场契合度所需的基础设施和市场曝光机会。
该海外办事处是国大企业机构更广泛战略的一部分，旨在将新加坡的创新生态系统与全球主要市场连接起来。
它将帮助创始人从硅谷获取市场洞察，在这一全球最成熟的风险投资市场中更好地理解客户需求并完善其解决方案。
通过该海外办事处获得的洞察，也将强化国大企业机构在教育、研究和商业化方面的途径。
假以时日，这将构建起一个更为强大的新加坡深科技初创企业储备库，这些企业将具备更清晰的市场验证、更高的商业成熟度以及对全球投资者和行业伙伴更强的吸引力。
“硅谷是深科技领域最具挑战性的试验场之一，投资者和竞争对手都以不懈的速度前行，”新加坡国立大学高级副校长（创新与创业）陈思伟博士评论道，“基于研究的创始人需要的投资者，是真正理解深科技公司如何被构建和扩展，而不仅仅是提供资金的。”
他补充说：“通过我们与Playground Global和Matter Venture Partners的合作，以及我们在硅谷设立的海外办事处，我们的创业公司能够及早发现机遇，根据真实客户需求进行测试，优化产品市场契合度，并最终成长为全球市场中的品类领导者。”
