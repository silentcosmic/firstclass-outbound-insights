# 日本新型诈骗曝光：快递柜成赃款中转站

🚨 日本新型诈骗曝光：快递柜成赃款中转站
日本警方近期成功捣毁一个利用公寓快递柜中转诈骗赃款的犯罪团伙，共逮捕四名涉案人员，其中包括一名中国籍男子。该团伙创新作案手法，不再直接交付赃款，而是利用与案件及受害者均无关联的公寓快递柜进行临时藏匿和中转，以此规避警方追踪。
据日本警视厅通报，嫌疑人津田正人等人此前冒充受害人儿子，谎称因“喉咙治疗急需用钱”，骗取东京都一名八旬女性800万日元现金。调查显示，受害人交付现金后，赃款被放入一栋陌生公寓的快递柜中，随后由津田正人取走，试图通过此方式掩盖资金流向。
警方进一步侦查发现，该快递柜曾出现在其他“特殊诈骗”案件中，表明该团伙可能长期将陌生公寓的快递柜作为赃款存放和转移的固定模式。除津田正人外，被捕人员还包括中国籍男子任柏晓。目前，日本警方正深入调查该团伙的分工、涉案范围以及是否存在专门的犯罪网络。
