# 缅甸妙瓦底持续清剿诈骗窝点，累计拆除63栋

🔥 缅甸妙瓦底持续清剿诈骗窝点，累计拆除63栋
6月23日，缅甸克伦邦妙瓦底水沟谷地区，当局联合行动小组对一栋涉嫌从事网络诈骗及非法网络赌博活动的7层建筑实施了定点爆破拆除。
据悉，此次爆破行动是当地政府计划拆除的20栋爆破目标建筑之一。随着本次拆除完成，水沟谷亚太城区域内累计已有63栋涉案非法建筑被清除，另有14栋建筑也将按计划陆续拆除。
本轮整治行动共锁定77栋涉嫌从事网络诈骗、网络赌博及相关违法活动的建筑设施，其中57栋将采用机械设备拆除，其余20栋则以爆破方式处理，旨在提升效率并确保行动顺利推进。
近期，缅甸当局持续加大对妙瓦底地区电信诈骗和跨境网络犯罪产业链的打击力度。通过清除涉案园区、拆除违法建筑以及加强区域管控，旨在进一步压缩犯罪团伙活动空间，维护边境地区安全秩序。
目前，相关清理行动仍在持续，缅甸当局表示将继续推进剩余目标建筑的拆除工作，并依法打击利用相关场所从事网络诈骗、网络赌博等违法犯罪活动的组织和个人。
