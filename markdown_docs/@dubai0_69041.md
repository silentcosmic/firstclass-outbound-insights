# 阿布扎比自动驾驶：上半年出行突破3万次

🚀 阿布扎比自动驾驶：上半年出行突破3万次
阿布扎比在2026年上半年共记录了31075次自动驾驶汽车出行。阿布扎比交通局（隶属于市政和交通部的综合交通中心）确认，这些行程集中在2026年1月31日至6月期间，标志着阿联酋境内自动驾驶交通服务运营的持续进步与发展。
