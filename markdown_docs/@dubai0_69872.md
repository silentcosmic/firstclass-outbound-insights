# ️ 欧盟升级中东空域警告：避开阿联酋等国领空

⚠️ 欧盟升级中东空域警告：避开阿联酋等国领空
随着美伊军事对抗再度升温，欧盟航空安全局（EASA）于14日再次发布并强化对中东地区航空警告。该机构明确要求航空公司避开巴林、科威特、卡塔尔及阿联酋领空，此禁令有效期至7月29日。针对伊朗、伊拉克和黎巴嫩空域的禁飞警告则延长至8月底。
EASA指出，美国在该区域的大量军事部署，使得相关国家直接面临伊朗导弹和无人机攻击的风险。此外，美方及地区防空系统的高度戒备状态，大幅增加了民航飞机被误识别或误击的可能性，空中军事冲突或将毫无预警地爆发。
