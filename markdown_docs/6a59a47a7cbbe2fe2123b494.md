# 全球商业策略与市场前瞻：AI创新、产业变革与资本新局

中国企业动态与资本风向
存储芯片巨头长鑫科技的市场公告显示，其首次公开发行网上申购表现活跃，吸引了9,428,778户有效申购，总计有效申购股数达8,169.2亿股。初步中签率确定为0.40995452%。
人工智能领域头部企业“深度求索”（DeepSeek）的最新估值意外披露，根据相关投资进展公告推算，该公司在本轮融资后估值已攀升至约3,510亿元人民币。接近公司人士证实，随着本轮融资完成，深度求索已启动第二轮融资计划，但年内是否冲刺科创板上市尚未有定论。
国内知名消费品企业泡泡玛特（POP MART）创始人王宁日前率核心高管团队访问美国硅谷苹果总部Apple Park，并与苹果现任CEO蒂姆·库克以及即将上任的新任CEO约翰·特纳斯等高管举行了会晤。双方围绕创意设计、数字生态建设和全球消费趋势进行了深入交流。
芯片设计与制造企业摩尔线程发布业绩预告，预计2026年上半年营业收入将达到16.5亿元至17.5亿元，同比增长135.12%至149.37%。业绩增长主要得益于人工智能产业的蓬勃发展、市场对全功能GPU的强劲需求，以及公司夸娥智算集群商业化加速、产品性能获得客户高度认可和稳定供货。
另一家芯片企业海光信息同样表现亮眼，预计2026年上半年归属于母公司所有者的净利润将同比增加4.98亿元至6.28亿元，增长幅度达41.50%至52.32%。若扣除股份支付影响，净利润增幅将达到74.27%至84.71%。
互联网巨头百度宣布，其董事会已批准根据香港联交所指引信，寻求将其在香港联交所主板的第二上市地位自愿转换为双重主要上市。预计此项转换将在本年度内生效。转换后，百度将成为在香港联交所主板及纳斯达克全球精选市场双重主要上市的公司，其A类普通股及美国存托股将继续在两个证券交易所内交易，并可互相转换。
中关村科金近期宣布一项重大人事任命，前IBM金融行业总经理范斌正式出任高级副总裁。范斌将负责银行及信贷事业部、财富及保险事业部、联合运营事业部以及海外业务的战略制定与拓展。此举凸显出中关村科金在加固金融科技基本盘的同时，加速全球化业务布局的战略意图。
全球科技前沿与AI浪潮
埃隆·马斯克旗下的SpaceXAI宣布开源Grok Build，并同时重置了所有用户的现有使用限制。针对用户普遍关注的数据隐私问题，该公司声明，自产品上线以来，Grok Build始终秉持零数据保留（ZDR）原则。用户可随时在命令行界面中选择禁用数据上传功能。尽管在早期测试阶段，部分非ZDR用户默认启用了数据保留，但自7月12日起，公司已为所有Grok Build用户禁用了此项默认设置，并正在删除此前保留的所有编码数据，以确保全面尊重每位用户的数据偏好。
日本正积极规划其在人工智能领域的战略布局，计划采购2.75万颗英伟达（NVIDIA）下一代Rubin AI芯片，以构建本土的机器人基础AI模型。为此，新成立的Noetra公司将负责大型数据中心的建设，并预计于2027年3月发布首个AI模型，后续将持续迭代，在未来数年内推出一系列面向机器人应用的人工智能模型。截至2027年3月，Noetra已获得日本政府3,873亿日元的资金支持，并由软银集团、丰田汽车支持的Preferred Networks以及日本电气等数十家企业共同参与运营。此项目旨在降低日本对海外技术的依赖，并致力于推动日本在全球规模约60万亿日元的机器人市场中，到2040年占据30%以上的份额。
关于人工智能市场是否存在“泡沫”的讨论日趋激烈。日本软银集团创始人兼首席执行官孙正义日前在东京出席活动时明确表示，对AI泡沫的担忧毫无依据，他认为今日质疑AI，就如同当年拒绝汽车和飞机一样荒谬。孙正义进一步预测，到2040年前后，全球将有约10亿台搭载AI的人形机器人投入运行，届时，AI机器人市场规模将扩大至约7,000万亿日元（约合人民币292.7万亿元），相当于全球经济总量的20%。
AI手机概念持续活跃，带动了端侧AI（Edge AI）相关概念股整体走强。智度股份实现直线涨停，连续第二个交易日触及涨停板。此前，格林精密、福日电子、国光电器、福蓉科技等公司股价也相继涨停，豪声电子、慧为智能、电声股份、泓禧科技等股票涨幅均超过10%。市场消息指出，某头部互联网公司正联合中兴努比亚打造首款AI智能体手机，代号为“豆包AI智能体手机”，预计今年将有多款机型发布，其中一款将于2026年世界人工智能大会期间亮相，整体备货量约20万台，首批备货量在10万台以内。截至目前，中兴方面尚未对此消息发表回应。
植入式脑机接口技术迎来重大突破。上海市相关部门宣布，植入式脑机接口手部运动功能代偿系统（NEO）在获批上市后，首张处方已于7月13日在上海成功开出，全球首例植入手术在复旦大学附属华山医院完成。该款脑机接口产品于今年3月获得上市批准。受此消息提振，A股脑机接口概念股在7月15日普遍走强。行业专家分析认为，中国脑机接口领域整体发展水平已跻身全球第一梯队，凭借完整的产业链和工程师红利，具备实现“弯道超车”的潜力。然而，在脑神经底层基础研究和专用芯片等领域仍存在短板，业内正从政策引导、产业集聚和技术迭代等多个维度协同发力，以稳步构建完整的商业闭环。
现代汽车集团于7月16日声明，将收购软银集团持有的波士顿动力公司近10%的剩余股份，使其成为现代汽车集团的全资子公司。此举旨在加速集团旗下各项业务中先进机器人技术的部署和应用。此前有媒体报道称，此次交易的价值预计接近5,000亿韩元（约合3.35亿美元）。
全球人工智能合作组织（WAICO）协定签署仪式在上海隆重举行。中共中央政治局委员、外交部长王毅代表中国政府签署了该协定。协定明确指出，世界人工智能合作组织是一个独立的政府间国际组织，将遵循《联合国宪章》宗旨，秉持共商共建共享理念，坚持以人为本原则。其核心目标是促进人工智能领域的国际合作和全球治理，确保人工智能技术朝着有益、安全、公平的方向健康有序发展，最终造福全人类。该组织总部设在中国上海。包括哈萨克斯坦、老挝、巴基斯坦、俄罗斯、印度尼西亚在内的29个国家的代表出席并签署协定，成为创始成员国。联合国秘书长古特雷斯等国际组织代表也出席了仪式。
亚马逊旗下低轨卫星网络服务Leo于当地时间7月15日宣布，已与南非最大的固定互联网服务提供商Herotel达成合作协议，将通过一项名为“evry”的新服务，把卫星互联网引入南非市场。Evry预计将于2027年正式上线，为南非的住宅用户提供稳定的网络连接。这是亚马逊Leo在非洲地区签署的首个此类合作协议。
全球市场与宏观经济聚焦
美国财政部长贝森特于15日宣布，为隆重庆祝美国建国250周年，美国铸币局将开始铸造正面印有前总统特朗普头像的1美元纪念硬币。该硬币正面还刻有“自由”、“我们信仰上帝”、“1776—2026”等字样，背面则以白头海雕图案为设计元素。特朗普对此表示感到“荣幸”。然而，此举也引发了批评人士对该硬币合法性的质疑。一项1866年的美国法律规定，在世人物的肖像不得用于美国货币上，但该法律主要指的是由美国印刷局印制的纸币，而硬币则由美国铸币局铸造。
美国三大股指集体收跌。半导体和存储芯片板块遭遇重挫，费城半导体指数跌幅超过4%。其中，英特尔、美光科技、博通的跌幅均超过5%，SK海力士暴跌逾13%，闪迪也下挫超过12%。与此形成对比的是，中概股逆势走强，纳斯达克中国金龙指数上涨1.79%。大型科技股表现分化，苹果和微软股价均上涨逾1%，特斯拉下跌0.86%，亚马逊跌近2%，英伟达跌超2%，谷歌则大跌逾4%。此外，SpaceX股价持续下行，当日跌幅超过3%。该股自6月上市以来已从高点大幅回落逾31%，市场对其超高估值的质疑持续发酵。
传奇投资者沃伦·巴菲特近日接受采访时透露，伯克希尔·哈撒韦公司近期对谷歌母公司Alphabet的重大投资，是由他本人亲自促成，并非由新任首席执行官格雷格·阿贝尔主导。巴菲特表示，是他发起了这笔交易，并强调他与接班人阿贝尔保持着极其密切的沟通。尽管两人在重大决策上彼此认可，但他才是此项投资的最终决定者。虽然亲自操刀了这笔巨额投资，现年95岁的巴菲特在受访时坦言，他犯了未能及早投资Alphabet的错误，并长期以来一直对错失这家科技巨头的早期崛起表示遗憾。
韩国政府宣布将暂停新的单一股票杠杆交易所交易产品上市，旨在有效抑制市场波动。此前，与三星电子和SK海力士等龙头企业挂钩的杠杆基金需求激增。韩国金融服务委员会周四发表声明指出，该禁令将持续有效，直至市场状况恢复稳定。此外，监管机构还将自8月5日起，将杠杆ETF的最低存款额从1,000万韩元提高至3,000万韩元（约合20,300美元）。这是韩国政府迄今为止为遏制散户交易狂潮而推出的最为全面且果断的举措。
行业专业研究报告指出，医药板块持续展现出刚性需求带来的高确定性，同时伴随着国内生物医药融资数据的持续高增长，预计医药创新企业和全球医药产业链上的相关企业有望在未来3-5年内实现业绩的稳健增长。从今年第三季度开始，国产创新药核心管线即将迎来重磅数据披露的催化，具备全球潜力的资产其国际化价值有望加速兑现。与此同时，政策环境也逐步回归到以临床价值和市场需求为导向的定价体系，为国内医药市场带来稳定且持续的发展环境，确立了行业长期增长的主基调，提供了较高的阿尔法和贝塔收益确定性。建议重点关注医疗健康产业的投资机会。
今年上半年，科技板块无疑是A股市场的“领头羊”，多只重仓科技赛道的权益类基金收益率表现突出。最新披露的公募基金二季报显示，尽管多数基金经理对科技板块保持着极高的配置热情，但在科技股内部的板块轮动以及估值逻辑方面，已出现显著分歧。随着全球AI浪潮的持续推进，科技板块展现出极高的景气度。从近期已披露的基金二季报来看，中欧基金、平安基金、融通基金、红土创新基金和金信基金等多家公募旗下多只绩优基金，仍然维持对科技股的高仓位运作，且多只产品在单季度取得了超高收益。
锂电企业2026年上半年业绩预告的集中披露，进一步印证了行业处于上行周期的逻辑。行业数据显示，截至7月15日收盘，已有35家锂电产业链上市公司发布上半年业绩预告，其中仅有3家业绩同比下滑，其余均为预增、扭亏或减亏。各企业的“成绩单”清晰表明，从上游锂矿、锂盐，到中游锂电材料，再到下游动力电池、储能赛道，产业链各环节的景气度正同步回暖。上游锂盐市场实现量价齐升，相关企业业绩大面积预增；中游隔膜、电解液与下游电池厂商的盈利能力也在同步修复。
印度政府宣布，自7月16日起对石油产品出口暴利税进行调整。其中，柴油出口暴利税从每升8.5卢比上调至15.5卢比，航空涡轮燃料出口税从每升7.5卢比上调至14.5卢比；而汽油出口税则从每升4卢比下调至2.5卢比。
日本立法机构已批准修订《金融工具和交易法》及《支付服务法》，将加密货币重新归类为金融工具。这一结构性变化不仅确立了加密资产单独征税的法律框架，也为未来加密交易所交易基金（ETF）的推出奠定了基础。修订后的法规将加密货币从主要作为支付工具的框架，转变为与其他金融工具一同作为投资对待。新规预计将于2027年生效。
全球半导体代工巨头台积电于7月16日公布了截至6月30日的2026年第二季度合并财务报告。报告显示，合并营收达12,703.8亿新台币，同比增长36%；净利润为7,065.6亿新台币，同比增长77.4%。毛利率达到67.7%，营业利润率为60.3%，净利润率为55.6%。在晶圆总营收中，2纳米、3纳米、5纳米和7纳米制程的出货量分别占比3%、30%、33%和11%，其中7纳米及更先进制程合计占晶圆总营收的77%。
消费市场与社会民生
近期一项调查揭露，市面上部分宣称“鲜榨”、“100%果汁”和“NFC果汁”的菠萝汁、西瓜汁、山竹汁等产品，尽管在电商平台销量惊人，单品动辄数十万件，但其低廉的价格却引发了广泛质疑。记者实地暗访河南多家代工厂后发现，这些生产车间内根本没有用于鲜榨的水果和相应设备。所谓“鲜果冷榨果汁”，实际上全靠水和浓缩原浆勾兑而成。厂家坦承，“低价不可能鲜果现榨”，他们普遍通过调换包装上“100%”、“NFC”等标识的顺序来打擦边球，误导消费者。目前，多家涉事企业已被责令停产整改，然而，调查发现线上仍有部分相关产品在售，且部分瓶身外包装标识尚未作出更改。
与房价相比，房租作为更直接关联居民生活成本的指标，其动态备受关注。随着2026年暑期租赁高峰的全面到来，深圳等一线城市的租房需求呈现快速攀升态势，租赁市场率先走出调整通道。近期对深圳市罗湖区、龙岗区、福田区多个住宅小区与城中村的走访显示，伴随毕业季求职和换房需求的集中释放，深圳租赁市场的活跃度显著提升，这直观反映出城市的强大人口吸引力与经济活力。从全国范围来看，今年上半年一线城市平均租金已实现小幅上涨，正式结束了此前连续两年的调整态势。
奈飞（Netflix）公布第二季度财报，营收达到125.6亿美元，同比增长13%，但略低于市场预期的125.9亿美元；营业利润为42亿美元，同比增长11%。鉴于市场环境变化，公司将全年营收预期下调至510亿至514亿美元，同比增长13%至14%，并预计2026年营业利润率为31.5%。
