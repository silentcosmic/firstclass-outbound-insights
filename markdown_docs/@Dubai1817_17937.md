# 越南公司惊曝内鬼案：中越员工联手诈骗11亿越盾

🚨 越南公司惊曝内鬼案：中越员工联手诈骗11亿越盾
近期，一家在越南运营的企业遭遇重大内部欺诈，一名中国籍员工与当地越南同事内外勾结，长期侵占公司资金，累计金额高达11亿越南盾（约合人民币33万元），导致企业蒙受巨额经济损失。
据调查，该中国籍员工利用其对公司财务流程的熟悉及业务对接中的漏洞，私下与越南同事合谋。二人摸清公司在对账和回款监管方面的短板后，通过伪造业务单据、篡改财务账目、拆分回款、虚假核销及虚构合作项目等多种隐蔽手段，持续侵吞公司流动资金。
在作案过程中，中国员工负责掌握核心财务权限并应对上层审核，而越南本地员工则利用其人脉和当地银行账户协助分流赃款，层层拆分以规避核查，整个欺诈行为持续了数月之久。
直至企业在月度盘点核对资金流水时，才发现巨大的资金缺口。公司随即收集了包括转账记录、虚假单据和聊天证据在内的全部资料报警。目前，当地警方已迅速立案，锁定了两名涉案人员，并正同步追查全部涉案赃款的流向。
