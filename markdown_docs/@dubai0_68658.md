# 迪拜酋长发文祝贺摩洛哥队世界杯晋级

📣 迪拜酋长发文祝贺摩洛哥队世界杯晋级
近日，迪拜酋长谢赫·穆罕默德·本·拉希德·阿勒马克图姆公开发文，祝贺摩洛哥国家足球队成功晋级世界杯16强。他在文中盛赞摩洛哥队的表现，称其为一场展现非凡拼搏精神与决胜意志的较量，更是一次令世界瞩目的精彩演出。
酋长表示，迪拜与摩洛哥同享这份喜悦，并向摩洛哥人民、国王及国家队致以诚挚的祝贺。此举不仅是对体育成就的认可，也体现了海湾地区与北非国家间的深厚情谊与团结。
