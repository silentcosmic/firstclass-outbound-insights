# 柬埔寨Hpay资金冻结：大批华人央行维权

🚨 柬埔寨Hpay资金冻结：大批华人央行维权
近日，大批在柬埔寨经商和务工的中国公民前往柬埔寨国家银行（中央银行）表达诉求，要求监管部门介入协调，解决Huione（Hpay）平台资金冻结问题。
Hpay作为柬埔寨当地普及度极高的线上支付工具，是大量华商门店经营者和普通务工人员日常收付款、发放薪资的主要依赖。此次受影响的资金并非大额投资，而是普通小店的日常经营流水、务工人员的薪资与生活积蓄，对他们的日常生计至关重要。
平台运营出现问题后，众多合规用户账户资金被限制提现，严重冲击了他们的正常生活与商业运作。此前，部分受害者曾寻求中国驻柬埔寨大使馆的帮助，此次直接向支付行业监管方中央银行递交材料，旨在促使主管部门正视并解决普通民众的实际困境。
