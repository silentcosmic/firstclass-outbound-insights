# ️ 东南亚“捞人”骗局曝光：福建男子涉案3.15万U

⚠️ 东南亚“捞人”骗局曝光：福建男子涉案3.15万U
近日，有市场消息指出，一名长期活跃于越南等东南亚地区的福建籍男子，外号“大师兄”，被指控涉嫌以“捞人、办事、关系运作”为名实施诈骗。该男子承诺通过“特殊渠道”解决问题，但已导致多名受害者蒙受经济损失。据悉，此次爆料涉及的金额约为31,500U。
目前，受害者已将包括声纹在内的相关线索提交至福建省公安厅和泉州市公安局进行报案备案。在此提醒在东南亚地区工作生活的华人朋友，务必对任何声称“有人脉”、“能捞人”或“花钱就能解决问题”的说法保持高度警惕，尤其对于“先放人后付款”或“有内部关系”等承诺，更需谨慎辨别，谨防落入二次诈骗陷阱。
请注意，上述情况及涉事人员信息均为网友投稿内容，具体事实尚待警方进一步调查核实。
