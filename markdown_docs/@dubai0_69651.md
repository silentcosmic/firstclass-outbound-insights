# 阿联酋彩票头奖落空：3000万迪拉姆大奖无人认领

⚡ 阿联酋彩票头奖落空：3000万迪拉姆大奖无人认领
阿联酋彩票日前公布了最新一期周六开奖结果。本次抽奖共产生三名幸运中奖者，每人各获得5万迪拉姆的奖金。然而，备受瞩目的3000万迪拉姆头等大奖此次仍无人中得。
据了解，这项名为“幸运日”的抽奖活动每周三和周六晚9:30定期举行。参与者有机会角逐3000万迪拉姆的头奖、500万迪拉姆的二等奖，以及三个每份5万迪拉姆的固定幸运奖。
