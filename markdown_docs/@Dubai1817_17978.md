# 东帝汶警方突袭：98名中国籍诈骗嫌犯落网

🚨 东帝汶警方突袭：98名中国籍诈骗嫌犯落网
据东帝汶媒体7月9日报道，东帝汶刑事科学调查警察部门联合国家情报局近日在首都帝力成功捣毁一处大型跨境网络犯罪窝点，当场抓获99名涉案外籍人员。
据悉，执法机关自6月起便对该位于贝博努克区域的仓库进行为期一个月的秘密监控，确认其长期从事网络诈骗及非法网络赌博活动。7月8日晚间，警方实施突袭，共抓获98名中国籍嫌疑人及1名柬埔寨籍嫌疑人，现场查获电脑、手机等大量作案工具及涉案现金。
目前，所有嫌疑人正接受身份核验与全面取证，后续将移送检察机关依法处理。此次行动显示出东帝汶当局在打击跨国犯罪方面的决心与力度日益增强。
