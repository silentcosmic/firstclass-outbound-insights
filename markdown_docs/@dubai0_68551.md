# 迪拜政府夏季新政：员工可选四天工作制

📣 迪拜政府夏季新政：员工可选四天工作制
迪拜市政府近日宣布，将为政府雇员推出一项夏季灵活工作制度，以在炎炎夏日为员工提供更人性化的工作安排。这项名为“我们的灵活夏季”的计划已于6月29日启动，并将持续至9月10日。
根据新政，各政府机构可根据部门运营需求，选择两种弹性工作模式。第一种模式下，员工每周一至周四每日工作七小时，周五工作四点五小时，实现周五提前下班。第二种模式则允许员工每周一至周四每日工作八小时后，在周五全天休息，享受长周末。
此举旨在平衡员工的夏季需求与政府机构的日常运作，彰显了迪拜在优化公共部门工作环境方面的持续探索与实践。
