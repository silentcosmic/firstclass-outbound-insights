# 柬埔寨女导演坠亡案反转：两名嫌犯获释

🚨 柬埔寨女导演坠亡案反转：两名嫌犯获释
围绕23岁中国女导演赖宇晴在柬埔寨金边坠亡一案，近期出现重大进展。此前被控“故意伤害致死罪”的两名嫌疑人，印度籍导演贾特拉·悉达多和中国籍制片人李芳，已获金边市初级法院裁定不予起诉并解除羁押。
该事件发生于去年12月30日，赖宇晴从金边一出租房高处坠落，于今年1月2日不治身亡。金边警方随后于1月11日逮捕了贾特拉和李芳，指控他们涉嫌与此案有关。
据市场消息，两人已于5月26日获释，结束了逾130天的羁押。不过，截至目前，李芳仍未取回此前被没收的护照。此案的法律走向及其后续影响，正持续引发业界关注。
