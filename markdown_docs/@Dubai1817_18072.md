# 中国储户柬埔寨被拘，百人集会抗议

🚨 中国储户柬埔寨被拘，百人集会抗议
一名参与汇旺维权行动的中国籍储户柯小伟，在柬埔寨被警方拘留后，已移交当地移民局，目前正面临遣返程序。其家属已向中国驻柬埔寨大使馆通报情况，并寻求领事协助。
柯小伟被移交移民部门后，约百名汇旺用户冒雨聚集在柬埔寨国家银行门前表达诉求，并表示将继续前往中国驻柬埔寨大使馆反映问题。此事件也再次引发对在柬中国公民权益的关注。柬埔寨官方数据显示，仅2026年上半年，该国就已驱逐42102名外国人，其中不乏因违反移民法规或涉嫌违法犯罪而被遣返的中国公民。
目前，柯小伟的遣返案件仍在处理中，最终结果尚待柬埔寨有关部门决定。
