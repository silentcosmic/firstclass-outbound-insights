# 阿联酋明日酷热：气温将飙升至47摄氏度

🔥 阿联酋明日酷热：气温将飙升至47摄氏度
阿联酋国家气象中心最新预报显示，本周日（明日）全国天气以晴朗为主，局部地区可能出现多云。
全国最高气温预计将飙升至47摄氏度。其中，阿布扎比和迪拜的最高气温也将达到42摄氏度。
