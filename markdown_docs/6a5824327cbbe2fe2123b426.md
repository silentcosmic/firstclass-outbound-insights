# 越南国家公民商业银行 (NCB) 推出VietQR全球服务，打通跨境二维码支付通道

越南国家公民商业股份银行（NCB）近日正式推出了一项创新的跨境二维码支付服务，专为在越南的国际旅客量身打造。
通过这项集成服务，国际访客只需使用其本国银行应用或数字钱包扫描本地商户的二维码，即可便捷完成支付。
这项名为“VietQR全球”的服务，当前已支持包括支付宝和银联在内的中国主流支付应用。
此外，该服务还覆盖了韩国的GLN、韩亚金融集团（Hana Financial Group）和Naver Pay等支付提供商，并惠及来自泰国、老挝和柬埔寨的用户。
NCB银行计划逐步将支持范围扩展至新加坡、日本、印度、马来西亚以及更多国家和地区的支付应用。
目前，太阳世界（Sun World）旗下的娱乐场所及太阳集团（Sun Group）旗下的酒店，包括河内卡佩拉酒店（Capella Hanoi）、塞雷娜度假村（Serena Resort）和岘港洲际阳光半岛度假村（InterContinental Danang Sun Peninsula Resort），均已开通此支付方式。
该支付网络同时覆盖了太阳富国航空（Sun PhuQuoc Airways）以及众多合作零售商户。
用户只需在结账时打开其常用的银行应用或数字钱包，扫描NCB VietQR二维码，即可完成交易。
系统将立即对用户进行身份验证，并在其现有应用内实时处理支付。
