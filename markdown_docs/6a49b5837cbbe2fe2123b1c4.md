# 电视内容中动态植入广告：颠覆创新抑或干扰体验？

早在十多年前，就曾有关于影视作品中过度品牌植入的讨论。彼时，一些观众会抱怨电影里出现的商业元素过多，例如在某些场景背景中出现的广告牌。然而，当时的观点认为，如果场景本身设定在商业区，这些广告牌的出现便属情理之中。
事后看来，这类早期争论可能有些偏离重点。通常，被视为生硬植入的例子是那些与剧情毫无关联、突兀出现的品牌产品。回溯过去，多数电影和电视制作方并不会刻意替换主要拍摄地点现有的户外广告，除非是为了添加一些彩蛋或进行世界观设定。
然而，在过去十年间，技术已取得了长足进步。对于部分营销人员而言，镜头背景中出现的广告牌，乃至一堵空白的墙面，都被视为绝佳的广告位，蕴藏着更多广告投放的潜力。
这正是一家AI技术公司所致力于实现的商业模式：通过人工智能驱动，将程序化、动态的广告牌插入电视内容中，使其成为内容所有者和广告主可行的选择。其核心理念在于，这种方式相较于传统的商业广告时段，对观众体验的侵入性更低。
该公司的重点在于创作直接融入剧本内容的广告库存。据该公司的首席产品官介绍，这项技术专注于选择那些既不干扰故事叙述，又能确保品牌曝光度足以衡量其影响力的位置。
上述高管指出，通常用于交代场景地点、时间或背景的“建立镜头”（establishing shots），非常适合数字植入广告牌。有时，这甚至意味着在现有节目中插入新的镜头，以创造更多可用的广告库存。
举例来说，如果某电视台正在播出一部设定在芝加哥的电视电影，但现有镜头中没有适合添加广告牌的场景，这家AI公司可以通过AI技术生成一个建筑外景的建立镜头，并预留广告位进行广告内容填充。
然而，这项技术实施起来远非易事。任何添加到场景中的元素都必须与原有内容的色彩和基调相匹配，不能显得过于虚假或与时代背景不符。例如，在一部设定于上世纪60年代的电视剧中，显然不能出现现代手机广告。
所有这些因素都制约了该流程目前尚无法实现大规模的程序化交易。尽管如此，这家AI公司已经启动了试点项目，以测试其程序化投放能力。
迄今为止，该技术提供商鼓励其合作的内容发布方将这些广告库存视为直接赞助机会。这意味着交易双方都清楚地知道广告何时、何地播出。该高管表示：“绝不会出现品牌不愿出现的内容，也不会有内容所有者对此感到意外。”
然而，值得注意的是，尽管内容所有者参与了这些交易，但内容的实际创作者——导演、布景设计师、演员和制作团队等——对于其作品中被添加的广告内容，很可能没有任何话语权。这在某种程度上令人感到不安，但或许这就是娱乐行业的现实。
植入式广告的效果与挑战
那么，最关键的问题是：这种广告模式有效吗？观众真的会对这类隐性广告产生回应吗？
根据该AI公司的说法，当其内容内广告整合与传统电视广告相结合时，在多项指标上表现更优，包括购买影响力、净注意力触达、品牌知名度以及整体好感度。
该AI公司的结果似乎也与联网电视（CTV）广告主中的普遍共识相符，即新兴的内容内广告形式（如压缩画面、叠加广告或主屏幕广告）的整体表现优于传统的商业广告时段。
一些业内专家认为，这种有效性源于新形式本身对观众的干扰性更小，因为广告出现时内容并不会停止播放。
然而，也有观点认为，这种解读并非全面。对于容易分心的观众来说，任何元素都可能造成干扰——甚至有些人无法在观看电视时同时阅读字幕。
更重要的是，内容内广告通常不可跳过，并且它们不像广告时段那样明显地宣告自己的出现。这意味着观众更不可能去做其他事情，比如起身去洗手间或查看手机。
不过，也有很多人对任何品牌信息被植入娱乐内容中，无论其语境如何，都会产生强烈的抵触情绪。
随着这些动态广告形式变得日益普遍，并且其程序化可用性不断提升，品牌和广告主将不得不审慎划定界限。多少广告才算过多？广告何时会因分散观众注意力而产生负面影响？
在此期间，请留意你最喜欢的电视节目背景中的那些广告牌。或许有一天，它们将为你专属定制。
