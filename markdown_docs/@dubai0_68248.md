# 老挝重拳打击网诈：近900人落网

🚨 老挝重拳打击网诈：近900人落网
据《The Laotian Times》6月25日报道，老挝近期在全国范围加大对电信和网络诈骗的打击力度。自6月1日以来，已累计逮捕近900名涉案人员。
最新行动中，6月24日老挝警方通过磨丁国际口岸向中国移交了170名中国籍嫌疑人。这些人员被指控在万象、波里坎赛省和占巴塞省非法居留并运营诈骗网络，将返华受审。
警方报告显示，在几乎每日的突击行动中，执法部门发现多种诈骗窝点，包括“杀猪盘”式网络恋爱诈骗和跨境诈骗团伙。被捕嫌疑人来自多国，包括中国、越南、印尼、菲律宾、马来西亚、新加坡、日本、台湾、缅甸、埃塞俄比亚和乌克兰等。
目前，多数中国籍嫌疑人已被遣返回国；其他国籍嫌疑人仍在老挝接受进一步调查和法律处理。
