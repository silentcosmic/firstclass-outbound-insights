# ️ 东南亚商业警示：一女子被指控涉嫌欺诈吸毒

⚠️ 东南亚商业警示：一女子被指控涉嫌欺诈吸毒
市场消息指出，一名活跃于东南亚商业圈的女性林佳莹（抖音账号“美羊羊”），正面临一系列严重指控。这些指控涉及其在当地商业活动中的不当行为，引发业界广泛关注。
据披露，林佳莹被指涉嫌参与“国盘杀盘”等欺诈性业务，并存在吸食毒品（包括电子烟及冰麻果）的行为。消息人士称，她不仅在七星海长湾地区业务中积累巨额债务，还被指利用不道德手段寻求合作，最终遭合作方解雇。此外，她还被控阻碍员工离职，并在内部清查时推卸责任，甚至诱骗他人为其偿还债务及套取公司现金。
鉴于上述指控的严重性，相关人士呼吁在东南亚地区，尤其是在七星海长湾开展业务的企业及个人务必提高警惕，审慎评估合作风险，以防范潜在的商业欺诈与道德风险。消息人士强调，此事件已在当地业界引发不良反响，并敦促涉事者正视问题，承担相应法律责任。
