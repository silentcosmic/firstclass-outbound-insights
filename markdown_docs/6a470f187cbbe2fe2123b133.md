# 通胀逆风下，AI投资风向生变，加密市场面临严峻挑战

在全球企业财报季表现强劲之后，市场对科技股在今年剩余时间里持续上涨的前景充满信心。观察人士普遍认为，当前的整体市场已经演变为一场围绕人工智能（AI）的宏大交易。客观而言，AI交易主导着科技股的波动，投资者在今年年初几乎对企业盈利表现寄予厚望。得益于美国与伊朗达成协议的乐观预期，投资者继续聚焦盈利和增长，标普500指数在5月创下历史新高。
然而，市场表现并非总能如愿。在高企的估值预期下，AI交易变得脆弱不堪，对任何突发事件都异常敏感，无论是宏观地缘政治的变动，还是美联储官员立场的转变。
AI投资的“完美”与脆弱
5月份远超预期的就业数据以及恶化的消费者物价指数（CPI），再次让投资者对美联储未来的利率决策感到不确定，也使市场对大型科技公司和芯片制造商当前AI支出的可持续性产生了疑虑。AI交易的热度衰退信号在市场抛售中显露无疑，纳斯达克指数创下一年多来最大单日跌幅，暴跌4%，标普500指数也下跌超过1.2%。这在市场对5月就业数据的反应中得到了进一步证实，当时新增就业岗位达17.2万个，远高于预期的8.8万个。美国国债收益率的上行再次加剧了市场对美联储未来政策路径的不确定性，许多人认为，如果当前通胀水平持续不减，加息将不可避免。
尽管近期美联储决定维持利率不变，但其整体立场仍偏向鹰派，并坚定致力于价格稳定。市场普遍预计，交易员已将12月加息完全纳入定价。
通胀与政策：加密市场的双重困境
资金从风险资产中撤离，揭示了加密市场的一些严峻现实。即便是最大的企业级比特币持有者MicroStrategy，也于5月底出售了32枚比特币，价值250万美元，这是该公司自2022年以来的首次出售，旨在支付股息。尽管这笔金额相对于其约84.4万枚比特币的总持有量微不足道，但此举仍动摇了市场情绪，加剧了加密货币价格的下行压力。
比特币长期以来被宣传为对冲货币贬值的工具，但随着通胀数据持续攀升，其叙事吸引力开始动摇。比特币并未成为避险资产，反而再次表现出与美国科技股高度关联的特征——年初市场情绪高涨时它随之上涨，而现在随着市场热情降温，它也陷入横盘整理。其抗通胀的论点并未消失，但变得更加复杂。在实际收益率持续高企、美元因AI驱动的资本流回美国资产而走强的环境中，比特币的避险吸引力正陷入艰难的交火之中。此外，SpaceX的首次公开募股（IPO）不仅使其股价创下上市以来新高，也分走了加密市场相当一部分的关注。
机构化与监管：DeFi的“下一站”迷思
在机构层面，去中心化金融（DeFi）作为大规模资本部署下一个前沿阵地的承诺依然难以捉摸。一项近期投资表明，华尔街青睐那些以机构需求为导向构建解决方案的区块链公司。频发的漏洞攻击，如跨链桥故障和预言机操纵攻击，使得风险规避型配置者坚定地选择观望，而期待已久的加密市场结构法案《清晰法案》（Clarity Act）在华盛顿的进展持续停滞不前。
缺乏明确的监管框架，大型机构的合规团队几乎无从施展。原本可能迎来结构性资金流入的夏季，反而变成了谨慎观望的季节。代币化真实世界资产（RWA）、日益成熟的跨链基础设施、不断增长的链上流动性等DeFi突破的关键要素正在悄然聚合，然而，大规模引入机构资本所需的监管跑道，却依然令人沮丧地遥不可及。
穿越周期：市场的新看点与长远趋势
在美联储最新利率决议维持不变后，投资者正密切关注美国与伊朗之间和平协议的进展，这可能缓解对石油冲击引发的通胀担忧。在AI交易主导的一年里，OpenAI和Anthropic备受瞩目的首次公开募股（IPO）无疑将点燃投资者的热情。一些人认为AI泡沫已现，堪比互联网时代；另一些人则坚信经济基本面依然强劲。
放眼未来，全球央行继续增持黄金储备，这预示着贵金属前景光明。创新从未止步。加密货币市场正再次加速迈入ETF时代。尽管整体市场低迷，Hyperliquid ETF旗下产品，例如Bitwise的BHYP和21Shares的THYP，在5月推出后不久便迅速吸引了约1.6亿美元的资金流入，彰显了其市场潜力。与此同时，预测市场正成为投资者把握快速变化的经济状况的重要工具。加密货币日益融入主流金融体系，这标志着显著的进步，是十年前难以想象的成就。
