# ️ 柬埔寨杀猪盘新变：本地居民高薪入伙

⚠️ 柬埔寨杀猪盘新变：本地居民高薪入伙
近日，柬埔寨金边警方成功捣毁了一个专门针对日本公民的诈骗团伙，该团伙冒充韩国人身份，利用“想和日本人交朋友”等话术实施网络诈骗。据估算，柬埔寨境内的此类诈骗产业规模庞大，其经济总量甚至可能占据该国GDP的30%至60%。
为应对这一严峻局面，柬埔寨政府近年来持续加大打击力度，已关闭逾250个诈骗园区，并驱逐了1.3万余名外国籍涉诈人员。此类诈骗通常被称为“杀猪盘”，即诈骗分子先与受害者建立信任关系，再诱导其进行虚假投资，最终骗取全部资金，过去四年全球因此类诈骗造成的损失高达约750亿美元。
然而，随着当局打击力度的加强，诈骗集团的运作模式正在发生显著变化。过去主要依赖外国人员的诈骗活动，如今正吸引越来越多的柬埔寨本地居民直接参与其中，充当诈骗人员。分析人士指出，高出普通工作2至3倍的薪资是吸引当地人铤而走险的主要原因。
这一转变预示着，未来在打击跨境电信诈骗和网络诈骗方面将面临新的挑战，执法行动需不断调整策略以应对诈骗集团人员结构和运作方式的演变。
