# 万象度假村突袭：电诈团伙覆灭，29名中国籍嫌犯落网

⚡ 万象度假村突袭：电诈团伙覆灭，29名中国籍嫌犯落网
据老挝媒体7月10日报道，万象警方近日展开专项行动，成功捣毁了纳赛通县一处藏匿于度假村内的电信诈骗窝点，现场查获大量作案设备与涉案财物。
6月24日，纳赛通县公安部门联合当地警务人员对该度假村进行突击清查，逐一排查21间客房，最终锁定并控制了整个诈骗团伙，共计抓获29名中国籍嫌疑人。
警方现场依法扣押手机399部、笔记本电脑9台、UPS不间断电源2台等电子设备，并查获人民币、老挝基普、美元、港币现金及黄金首饰等涉案财物。目前，所有涉案人员及物证已移交万象市公安局治安部门，将依照当地法律进行审讯，并深入调查以彻底铲除该犯罪网络。
