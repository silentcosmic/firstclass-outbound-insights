# 迪拜商界痛失英才：加拉达里家族少帅24岁骤逝

⚡ 迪拜商界痛失英才：加拉达里家族少帅24岁骤逝
阿联酋知名商人穆斯塔法·加拉达里之子、加拉达里家族的青年才俊曼苏尔·加拉达里于7月11日不幸离世，年仅24岁，此消息震惊了迪拜商界。
消息传出后，包括加拉达里兄弟集团主席苏海尔·加拉达里和达马克地产创始人侯赛因·萨瓦尼在内的多位商界领袖纷纷表达深切哀悼，凸显了加拉达里家族在中东商业版图中的深远影响力。
曼苏尔自2022年进入家族企业伊利亚斯与穆斯塔法·加拉达里集团工作，以勤奋好学、勇于担当著称。公司首席财务官阿尔萨兰·哈蒂布曾赞其为“一位愿景清晰、以人为本的新生代领导者”，他的骤然离世无疑是业界的一大损失。
