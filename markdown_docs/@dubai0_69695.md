# 迪拜酋长豪掷10万奖学金，学生：精神激励更珍贵

🏆 迪拜酋长豪掷10万奖学金，学生：精神激励更珍贵
阿联酋副总统兼总理、迪拜酋长谢赫·穆罕默德·本·拉希德·阿勒马克图姆近日向全国顶尖中学毕业生送上厚礼。每位优秀毕业生获赠10万迪拉姆奖金，并收到酋长亲笔寄语，其家人也一并受到致意。这份表彰于2026年7月11日正式公布。
礼盒上印刻着“我们为你感到自豪，并对你寄予厚望”的字样。许多学生表示，这份精神鼓励的价值远超奖金本身。
酋长在寄语中不仅祝贺了这些学业有成的年轻人，也对他们背后默默支持的家人表达了敬意。他强调，毕业生的成就是阿联酋持续增长与繁荣的缩影。
此次奖金金额统一为每人10万迪拉姆，覆盖全国范围，不限于特定酋长国。获奖者包括即将升入大学的学生及准备步入职场的年轻人，评选主要依据学业成绩。官方暂未公布具体获奖人数及各校成绩分布。
