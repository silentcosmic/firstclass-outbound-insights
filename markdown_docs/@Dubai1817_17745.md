# 吉隆坡突发大规模执法：多名外籍人士被带走

🚨 吉隆坡突发大规模执法：多名外籍人士被带走
据市场消息，马来西亚吉隆坡Cheras及Maluri地区于2026年7月5日突发大规模联合执法行动。执法人员在多处展开突击检查与盘查，现场气氛一度紧张。
据初步报告，行动中已有包括缅甸籍及中国籍公民在内的多名人员被带走协助调查，具体情况仍在核实中。
目前，本次行动的具体执法部门及所涉案件类型尚未对外公布，有待官方进一步说明。
