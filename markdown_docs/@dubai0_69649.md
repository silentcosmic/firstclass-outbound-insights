# Telegram推出“文章消息”：富文本编辑时代开启

📣 Telegram推出“文章消息”：富文本编辑时代开启
Telegram近日向其Premium订阅用户开放了测试版的“文章消息”功能。这项新特性允许用户通过全屏编辑器发送富文本消息，大幅提升了平台的内容创作能力。该功能支持标题样式、表格、数学公式、代码块、折叠引用及幻灯片等多种高级格式，此前仅在测试服务器运行，现已正式上线主服务器。
目前，用户需使用Android或iOS测试版12.9及桌面端测试版6.9.4方可体验此功能。在手机端，用户可通过附件菜单中的“文章”选项，或点击发送按钮上方的箭头图标进入编辑器。值得注意的是，iOS测试版目前已不再接受新的测试者申请。
