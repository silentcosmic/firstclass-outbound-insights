# 港陆警方联手：捣毁逾2亿跨境网恋投资诈骗团伙

🚨 港陆警方联手：捣毁逾2亿跨境网恋投资诈骗团伙
近日，香港与内地警方联合开展代号“战军”的跨境行动，成功瓦解一个涉及网恋投资“杀猪盘”及洗钱的特大诈骗集团，共拘捕69名涉案人员，涉案赃款累计逾2亿港元。
此案背景下，香港警务处数据显示，今年首季度投资骗案高达1210宗，造成9.2亿港元损失，同比上升17%，占同期全港诈骗总损失的一半。本案源于今年5月一名内地受害者报案，其在网恋圈套中被诱导投资，损失超千万港元，牵扯上百个傀儡账户。
该犯罪团伙分工明确，通过社交平台建立虚假人设，诱骗受害人下载高仿虚拟货币投资APP。为博取信任，团伙甚至先转账97万港元作为虚假盈利分红。受害人随后在数月内向香港一级账户转账千万，并以实物黄金及场外虚拟货币形式交付投资本金。资金流入后，迅速被分流至106个二级傀儡账户进行层层洗白，警方最终锁定116个涉案账户，账户持有人包括内地、香港居民及外籍人士。
目前，香港商业罪案调查科已通过紧急止付机制成功冻结涉案赃款470万港元，并正追查其余未冻结资产。警方还通过资金流水深挖线索，排查出另外173名潜在受害人。此次行动共拘捕54男15女，年龄介乎18至60岁，其中14人在香港落网，其余在内地多地被捕。案件仍在深入侦办中，警方将继续努力为受害人挽回损失。
