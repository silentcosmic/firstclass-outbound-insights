# 柬埔寨波贝持续排查：严打涉外网诈及非法活动

🚨 柬埔寨波贝持续排查：严打涉外网诈及非法活动
7月10日起，柬埔寨波贝市持续开展行政检查行动。根据国家警察总署总监苏泰的指示，国家警察总署专业执法工作组联合波贝市政府，对当地涉嫌聚集外国人、从事网络诈骗及其他违法犯罪活动的场所进行深入排查。
此次行动表明，柬埔寨当局正持续加大力度，打击波贝地区涉及外国人的非法活动，特别是针对日益猖獗的网络诈骗问题，以维护社会秩序和法律尊严。
