# 美国制裁太子集团核心成员，新加坡冻结逾6亿新元资产

🚨 美国制裁太子集团核心成员，新加坡冻结逾6亿新元资产
美国政府继去年制裁陈志后，近日再次宣布对“太子集团跨国犯罪组织”及其关联方实施新一轮制裁。此次行动共涉及9名个人及26个实体，范围涵盖该组织的核心高层、诈骗园区投资者及境外空壳公司，同时将汇旺集团的继承实体“H-Pay”列入限制名单。
本轮制裁的首要目标是太子集团的“二把手”胡小伟，其已于6月14日在日本落网，名下十余家公司已遭查封。此外，陈波、戴安、方智振、罗平华、邱东等多名拥有多国国籍的园区运营及洗钱骨干亦悉数在列。
与此同时，新加坡警方宣布已冻结胡小伟、邱东名下共计1亿新元的资产。至此，该集团在新加坡被冻结的资产累计已超过6亿新元。
