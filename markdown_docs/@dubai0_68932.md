# 日本高管签证疑云：胡石被捕，公司经营受彻查

🚨 日本高管签证疑云：胡石被捕，公司经营受彻查
日本共同社7月2日报道，44岁塞浦路斯籍男子胡石（常用名胡小伟）因涉嫌提交虚假迁入登记资料，已被日本警视厅逮捕。据调查人员掌握，胡石近期入境日本后，曾辗转入住大阪市内多家高档酒店，警方通过监控画面确认其身份，最终在大阪将其抓获。此次逮捕的直接原因，是其涉嫌向东京都中央区政府提交虚假的迁入登记资料，并将不实住址信息登记入官方系统使用。
然而，此案的调查已远超虚假住址登记。日媒披露，胡石持有的是日本“高度专业职1号（ハ）”在留资格，主要面向企业经营和管理人士。日本警方现正重点核查他担任代表董事的公司是否真实经营，以及当初取得签证的过程是否存在问题。公开信息显示，胡石名下公司于2023年4月在东京都足立区成立，经营范围包括贸易，注册资本从800万日元增至5000万日元。但该公司员工此前受访时表示从未见过胡石本人。此外，胡石的个人登记地址近年来也多次变更，从英国伦敦到东京港区、大阪吹田市，直至此次被警方盯上的东京中央区。
与胡石一同被捕的31岁中国籍公司职员李垠宏，持有的是“高度专业职1号（ロ）”在留资格，这类资格通常面向运用专业知识和技术工作的人士。两人均使用高度专业人才类签证，引发日本警方对入管制度可能被滥用的疑虑。目前，日本警方虽然公开的直接案由是虚假住址登记，但调查已深入至公司是否为空壳、签证是否通过包装取得以及在留资格是否存在滥用等核心问题。
