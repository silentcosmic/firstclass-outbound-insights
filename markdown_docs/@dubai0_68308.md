# 中国公民柬埔寨街头困境：只想回国

🔍 中国公民柬埔寨街头困境：只想回国
近日，一名中国公民在柬埔寨一市场内被发现流落街头，其无家可归、身无分文的困境引发关注。据了解，该男子长期栖身于市场角落，生活状况十分艰难。
据目击者称，该男子已连续多日在此区域活动，并常忍饥挨饿。一位当地热心妇女曾主动提供食物及住宿帮助，但该男子仅轻声回应：“我想回中国。”
这句简短的回应，不仅流露出对故土的深切思念，也映射出海外公民可能面临的复杂困境。目前，其流落柬埔寨的具体原因及是否已获得官方或社会组织的进一步援助，尚待厘清。
