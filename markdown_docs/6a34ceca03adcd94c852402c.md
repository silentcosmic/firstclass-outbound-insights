# Wise战略收购Expatica，赋能全球迁徙用户生活与金融服务

全球领先的支付科技公司Wise日前宣布成功收购在线资源平台Expatica。该平台致力于为全球用户提供关于海外住房、医疗、移民、金融以及日常生活等方面的综合指导与信息。
Expatica成立于2000年，长期以来专注于为那些选择迁居或已在海外定居的用户提供实用的生活指南。其内容旨在帮助用户更好地适应异国他乡的环境。
平台信息覆盖广泛，具体包括在新国家的住房选择、医疗保健体系、移民政策解读、工作机会探索、跨境金融管理以及日常生活的方方面面。
据数据显示，Expatica平台在2025年累计访问量已突破700万次，其核心用户群体主要分布在法国、德国、葡萄牙和西班牙等欧洲国家。
Wise目前在全球范围内为约1900万个人用户及企业提供服务，其中大量用户日常业务涉及频繁的跨境资金往来管理。
此次战略收购的背景，正是全球国际移民规模持续保持高位的时期。
Wise援引联合国最新数据指出，2024年全球国际移民人数已达到3.04亿，这一数字相当于全球总人口的3.7%。Expatica多年来通过提供实用且具有本地化意义的指导，持续帮助人们理清海外生活的脉络。这与Wise的使命高度契合，因为我们早已在为数百万跨越国界生活的人们提供支持。当人们规划、迁徙并逐步安顿海外生活时，资金管理是整个体验中不可或缺的一部分——无论是获取薪资、跨境汇款、日常消费还是财务规划。Wise的平台正是为这些关键时刻而构建。
Wise表示，未来将持续加大对Expatica的投资，逐步拓展现有内容深度、扩大本地化覆盖范围、进入更多市场并增加语言支持。
该公司进一步阐释，Expatica的加入将与Wise现有的金融产品形成有效互补，使用户在规划、迁徙和最终安顿海外生活的各个阶段都能获取所需的信息与服务。
