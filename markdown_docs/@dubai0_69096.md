# 印尼破获跨国诈骗案：210外籍嫌犯待遣返

🚨 印尼破获跨国诈骗案：210外籍嫌犯待遣返
印尼近日通报，今年5月在巴淡岛破获的一起跨国网络投资诈骗案取得新进展。210名涉嫌参与此案的外籍人员，其遣返工作已进入最后协调阶段。后续，印尼将与各涉案人员原籍国执法部门合作，进行联合押送与移交，以开展进一步调查与处理。
据了解，该案窝点于今年5月在巴淡岛巴洛伊景观公寓及一处高档住宅被印尼移民总局联合廖内群岛省警方捣毁。此前，执法部门经过近四周的秘密侦查和监控，成功端掉了这一跨国网络投资诈骗集团。
行动中，执法人员共查获涉案外籍人员210名，并扣押了包括电脑、手机在内的大量电子设备。经查，涉案人员包括中国籍84人、越南籍125人、缅甸籍1人。其中209人持短期访问签证入境，涉嫌滥用居留许可，假借股票投资、虚拟金融交易等名义，对境外受害人实施电信网络诈骗。目前，所有涉案人员均已被安置在丹戎槟榔移民拘留中心，等待遣返。
