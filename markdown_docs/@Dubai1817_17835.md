# 新加坡严打非法网赌：世界杯期间17人被捕，查获72万新元现金

🚨 新加坡严打非法网赌：世界杯期间17人被捕，查获72万新元现金
新加坡警方在世界杯期间展开大规模行动，严厉打击非法远程赌博活动。7月6日，刑事侦查局、警察情报局和特别行动指挥部等多部门在全岛多个地点同步突击，目标包括义顺、宏茂桥、大巴窑等13个区域。
此次行动中，警方共逮捕17名涉案人员，包括14名男子和3名女子，年龄介于45岁至76岁之间。现场查获逾72万新元现金，以及手机、电脑和赌博相关工具等大量证物。目前，已有5名嫌疑人被控上法庭，其余人员的调查仍在深入进行中。
警方强调，将持续对非法赌博活动保持高压态势，尤其是在世界杯等大型国际体育赛事期间，以遏制此类犯罪并防止不法分子从中牟利。
