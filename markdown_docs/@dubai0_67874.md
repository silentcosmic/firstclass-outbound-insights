# ️ 阿联酋全国将迎降雨降温，出行需注意

⚠️ 阿联酋全国将迎降雨降温，出行需注意
阿联酋国家气象中心（NCM）发布预警，预计本周二至周四，全国范围将迎来降雨，并伴有雾霾天气。气温预计将在本周晚些时候有所下降。
根据气象预报，对流云团将从周二下午开始形成，为部分地区带来阵雨，尤其是东部区域。同时，潮湿空气可能导致沿海及内陆区域出现雾霾。
NCM还指出，周四气温将普遍下降，西部地区尤为明显。强劲的西北风可能在空旷区域引发扬尘。此外，阿拉伯湾海况将转为中浪，西部部分海域可能出现大浪，海上作业及出行需特别留意。
