# 中美阿联酋联手：柬埔寨9大诈骗窝点被连根拔起！

🚨 中美阿联酋联手：柬埔寨9大诈骗窝点被连根拔起！
近期，由中国、美国、阿联酋三国执法部门首次联合发起的跨境打击行动，直捣柬埔寨境内。此次行动成功捣毁了9个大型电信诈骗窝点，美国联邦调查局（FBI）亦直接参与清查，现场控制并扣押了大批涉案人员及作案设备。
此次三国联手，打破了以往犯罪分子认为躲藏在东南亚便可逍遥法外的幻想，再次彰显了全球执法部门打击跨境犯罪的决心与能力。国际执法合作网络的日益紧密，意味着跨境犯罪分子已无任何避风港。
