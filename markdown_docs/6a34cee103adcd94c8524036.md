# Sumsub推出AI代理工具，助力合规团队智能化构建验证工作流

Sumsub最新推出一项AI代理集成方案，旨在赋能合规团队，将复杂的反洗钱（AML）政策转化为可配置的自动化验证工作流。
这项基于模型上下文协议（Model Context Protocol）的集成，支持团队将反洗钱政策或监管要求上传至各类领先的AI模型中。
随后，AI代理便能依据上传的文档内容，在Sumsub平台内自动配置验证级别、风险问卷及客户入驻流程。
Sumsub方面指出，此举能将设置时间从数天大幅缩短至数分钟，即便面对包含特定国家风险分级、评分表和条件规则的复杂政策亦是如此。
公司强调，此次集成旨在显著减少将合规政策手动转化为平台配置的繁琐工作。
此外，AI代理还能够协助将Sumsub的验证流程无缝嵌入到客户的各类应用程序中。
合规团队可借助这些代理工具，高效审查申请人、执行数据分析、生成验证链接，并快速响应不断变化的监管要求。Sumsub代表表示：“我们的代理驱动体验彻底改变了现状，它将AI代理直接连接到平台的配置层。这意味着团队只需将反洗钱政策提交给AI代理，即可自动构建完整的运行环境。这在该领域内代表着一种与以往截然不同的能力范畴。”
该集成方案与具体AI模型无关，可兼容市面上主流的AI代理。Sumsub同时也在开源社区公布了相关的代理技能。
这项MCP集成是Sumsub现有AI工具的进一步延伸，其中包括专为合规与反欺诈团队设计的AI协处理器“Summy”。
访问权限经过严格分离管控，所有敏感操作均在隔离沙箱环境中运行。同时，任何配置更改仍需人工审核与批准。
此项集成现已全面上线。Sumsub指出，其是首个获官方认证并入驻主流AI应用平台的验证服务商。
相关文档和代理技能可从Sumsub的开发者资源平台获取。
