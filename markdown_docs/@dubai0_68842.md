# 苹果邮件隐私漏洞曝光：真实邮箱仍可被还原

🚨 苹果邮件隐私漏洞曝光：真实邮箱仍可被还原
苹果的“隐藏邮件地址”功能被发现存在一处严重隐私漏洞。该漏洞允许攻击者绕过保护机制，查出与随机生成地址关联的真实用户邮箱。研究人员测试结果显示，所有通过该功能创建的地址均可被还原，且此漏洞目前依然活跃。
据研究人员透露，他们已于2025年6月将此漏洞提交给苹果。尽管苹果曾一度声称已在今年3月完成修复，随后又计划于6月处理，但问题至今仍未得到解决。目前，研究人员尚未公开漏洞的具体细节及利用方法。
