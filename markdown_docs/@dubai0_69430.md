# 美军空袭伊朗，中伊贸易大动脉受损？

🚨 美军空袭伊朗，中伊贸易大动脉受损？
据市场消息，美国对伊朗的一次空袭，击中了连接中伊贸易走廊上的一座重要铁路桥。法尔斯通讯社报道称，遇袭地点位于伊朗北部戈勒斯坦省的阿克塔克汗铁路桥。
该桥梁不仅是德黑兰与伊朗最大贸易伙伴中国之间的关键物流命脉，在美国封锁伊朗港口期间，也曾被用于向俄罗斯输送货物。此次事件的后续影响，值得出海企业密切关注。
