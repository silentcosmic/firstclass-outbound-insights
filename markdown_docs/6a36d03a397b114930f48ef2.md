# 出海深度：中国“基因”如何驱动东南亚多业态扩张，成就本土市场独角兽？

在2026年5月28日于中国澳门举行的BEYOND Expo 2026国际科技博览会上，一场关于东南亚市场深度开拓的专题峰会汇聚了行业领袖，共同探讨了区域扩张的深刻议题。
在高度碎片化的东南亚市场中，一种独特的“打法”正在浮现，助力出海企业实现快速的实体店扩张，并最终蜕变为主导性的本土玩家。
这场题为“从‘中国基因’到‘本土独角兽’：东南亚多业态规模扩张的底层逻辑”的专题讨论，汇集了咖啡、护肤、母婴护理及快餐等多个领域在东南亚市场增长最快的零售帝国的战略操盘手。
这些行业翘楚虽然共享着世界级的供应链效率和高强度分销模式的“中国血统”，但他们的成功更在于其摆脱“外来者”标签，真正作为本土企业深耕市场的能力。与会嘉宾们深入剖析了将实体业务扩展至数千家门店的实际运营机制，以及“中国基因”如何被重新编码，以赢得东南亚消费者的心智和钱包。
本次圆桌论坛由ATM Capital创始合伙人梁民浚主持，特邀Makuku首席财务官白雪峰、Hebe Beauty首席财务官张俊辉、J.Chicken创始人曹锐以及Tomoro Coffee首席执行官杨璐璐出席。
Makuku：深耕母婴护理，四年内跻身市场前三
Makuku专注于母婴护理产品，目前主要销售纸尿裤，并计划在今年新增婴儿配方奶粉业务。公司进入东南亚市场已有四到五年，在印度尼西亚市场表现尤为出色。团队看到了东南亚和中东地区母婴护理行业的巨大机遇。
我们将中国的经验、新产品和创新带到东南亚，服务核心客群是母亲和婴儿。在2021年，当Makuku审视东南亚市场，尤其是印度尼西亚时，发现了一个显著的使用习惯差异。印度尼西亚特定消费群体对母婴产品的需求与主流市场趋势存在差异。
在纸尿裤领域，团队注意到印尼的母亲们仍在广泛使用可能导致婴儿红疹的传统纸尿裤。而同期在中国，母亲们已普遍转向使用技术更先进、更“透气”的纸尿裤。
Makuku看到了提升印尼母亲和婴儿生活品质的巨大机会，这同时也带来了巨大的商业增长潜力。东南亚母婴护理市场价值达180亿美元，而印度尼西亚庞大的人口基数和出生率，为我们提供了广阔的增长空间。
我们始终坚信优秀的产品会不言自明。因此，当我们将创新产品引入印度尼西亚市场，结合本地人才和运营能力，四年内，Makuku成为唯一一家进入当地市场前三的中国品牌。
Hebe Beauty：多品牌矩阵，以效率与渠道深耕美妆市场
Hebe Beauty成立于2018年，专注于健康与美容产业。目前旗下拥有六个品牌，涵盖护肤品、化妆品等多个品类。Hebe Beauty已成功拓展至印度尼西亚、马来西亚、菲律宾、泰国和越南市场，同步布局线上与线下渠道。
回顾2018年，高端美妆市场主要由国际大品牌主导。与此同时，本土品牌凭借其差异化定位和独特优势，在大众及小众市场也表现出色。
彼时，中国化妆品行业正蓬勃发展，拥有丰富的供应链资源。Hebe Beauty团队在线下渠道运营方面经验丰富且具备强大的执行力。
公司在国际品牌与本土品牌之间找到了市场机会。Hebe Beauty在渠道拓展和运营方面的能力，使其能够超越大型国际品牌，实现更高的市场渗透率。同时，依托中国高效的供应链体系，我们能为消费者提供比本土品牌更优质的产品。
Hebe Beauty成功锁定了目标市场。我们认为，在电子商务的迅猛发展、年轻一代消费力提升以及内容电商兴起的共同驱动下，东南亚市场将持续快速增长。因此，东南亚将长期作为我们战略布局的重点区域。
J.Chicken：炸鸡出海，深耕印尼“升级体验”需求
我从大学时代就开始从事炸鸡业务。在中国，我们的品牌名为“叫了个炸鸡”。除了中国本土，我们还在美国和日本设有分店。
2017年，我们开始关注东南亚市场。我们认为东南亚有潜力成为除中国以外最大的炸鸡市场。在2022至2023年期间，我们与ATM Capital一同考察了印度尼西亚。我们认为这是一个未来可以“全力以赴”的市场。
过去两年，我们在印度尼西亚开设了近百家门店。基于在中国积累的经验，我们针对印尼市场进行了相应调整。目前来看，在印尼的扩张之旅进展顺利。
当初团队决定进入印尼市场时，所有人都充满热情。印度尼西亚是一个穆斯林国家，鸡肉是当地居民主要的蛋白质来源。穆斯林不食用猪肉，而牛肉和羊肉相对昂贵。
在印尼市场调研时，我们看到各类炸鸡店铺林立，但其中大部分是路边摊。市场对于提供干净整洁用餐环境的“升级版餐饮体验”供给不足。当时，市场上主要只有麦当劳和肯德基等国际连锁品牌。
因此，我们决定进入这一市场，为当地消费者提供一个升级的品牌体验。J.Chicken由此诞生。
过去两年，我们也遇到了一些挑战。我们希望能提供更美味的炸鸡、更舒适的用餐环境以及更亲民的价格。然而，同时满足这三个条件给我们带来了巨大压力。现在回想，或许我们可以只比本地企业略胜一筹。因为同时满足这三个标准会大幅增加我们的运营成本。
Tomoro Coffee：三年跻身印尼第二，剑指东南亚咖啡连锁榜首
Tomoro Coffee目前已进驻印度尼西亚、菲律宾和新加坡市场，门店数量超过800家。以门店规模计，我们已是印度尼西亚第二大咖啡连锁品牌。
2022年我们进入印尼市场时，当地的咖啡连锁特许经营店数量并不多。与此同时，消费者对咖啡饮品在专业性和创新性方面的需求却在不断增长。
仅仅三年时间，我们已成为印度尼西亚排名第二的咖啡连锁品牌。我们的宏伟目标是成为东南亚地区的第一大咖啡连锁。
我们进行了全面的市场调研，注意到整个东南亚市场存在的明显空白。咖啡特许经营店的渗透率不足5%。而另一方面，消费者对更好的产品创新和品质有着强烈需求。供给侧的不足为我们带来了巨大的发展机遇。
以印度尼西亚为例，人口接近2.8亿，年出生人口约500万至600万，人口结构非常年轻，平均年龄低于30岁。这为消费产品和生活方式服务带来了强劲需求。
