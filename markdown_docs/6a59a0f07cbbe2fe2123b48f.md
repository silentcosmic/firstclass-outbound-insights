# 全球产业前瞻 | 百度智能云领跑金融AI市场；腾讯云具身智能布局深化；半导体与社交媒体监管聚焦

本刊获悉，为深化集团资源整合并优化全球产品策略，realme品牌未来新品将集中发力海外市场，暂停在中国市场的更新。与此同时，一加品牌在中国市场的产品策略将维持不变。两大品牌均致力于在游戏和性能领域持续提供卓越用户体验。
最新行业研究报告显示，中国金融行业生成式AI市场正经历显著的爆发式增长。预计到2025年，市场整体规模将达到17.40亿元人民币，同比增速高达90.4%。在这一快速扩张的领域中，百度智能云凭借其坚实的技术基础和丰富的应用场景落地能力，以16.6%的市场份额位居中国金融行业生成式AI平台及应用解决方案市场榜首。
深圳华强对外披露，公司作为华为计算部件（涵盖昇腾、鲲鹏系列）的总经销商，将继续致力于华为计算技术与产品的推广。公司正积极协助华为构建强大的算力生态，相关合作项目正加速推进。关于第二季度的具体经营业绩，敬请留意公司后续发布的定期报告。
本刊获悉，泡泡玛特创始人王宁近日带领核心高管团队访问美国硅谷苹果总部Apple Park，并与苹果公司现任CEO蒂姆·库克（Tim Cook）以及即将上任的新任CEO约翰·特努斯（John Ternus）等高层管理人员进行了会面。双方就创意设计、数字生态系统及全球消费趋势等议题进行了深入且富有成效的交流。
7月15日，腾讯云与全球领先的通用具身智能公司RoboScience机器科学正式签署战略合作协议。双方计划依托腾讯云强大的基础设施服务能力，共同构建面向具身智能产业的整合底座，旨在打通从数据采集、多地域协同标注到模型训练及商业化落地的全链条路径，共同开创行业首个云端“具身智能即服务”（EaaS - Embodied-AI-as-a-Service）。
百度文库和网盘近期将迎来GenFlow的全面升级，旨在为专业用户和特定场景提供更为专业化的服务。新版本将同步上线独立的网页端和客户端，并被定位为一个专为金融、教育、办公等专业领域从业者量身定制的“多端通用Agent平台”，预计将于7月底启动内部邀请测试。据悉，GenFlow升级后的首个版本将重点聚焦金融领域，为金融从业者、行业爱好者以及金融机构及团队提供高度定制化的金融解决方案。
7月16日，由资管巨头博枫支持的数据中心服务商Csquare宣布，将在美国首次公开募股（IPO）中发行5000万股普通股，发行价定为每股21.00美元。此次发行预计于当地时间7月17日完成，但仍需满足惯例成交条件。
极氪品牌于7月16日正式宣布，旗下9X五座版将于7月28日上午10点直接开启预定，并省略线下发布环节。市场数据显示，极氪9X自上市以来已连续七个月斩获50万级大型SUV销量冠军，平均成交价突破53万元。权威机构的最新用户净推荐值（NPS）报告显示，该车型在50万元以上全系车型中排名第一。此外，迈莎锐（Mansory）、虏曼（Lumma Design）等欧洲知名改装品牌已相继为其推出定制化改装方案，其中迈莎锐版9X的海外售价更是超过40万美元。
京东于7月16日正式发布京麦AI经营中心，并向广大商家全面开放22款核心AI工具。京东京麦相关负责人指出，京麦AI经营中心是京东为商家量身定制的一站式AI经营聚合平台，其首期功能目前已全面投入使用。
7月15日，金山办公在上海成功举办“2026 AI生产力大会”，并同步发布了两款重磅AI办公智能体，分别服务于个人级和组织级两大应用场景：面向个人的“灵犀专业版”，以及面向组织的“WPS Comate”。“灵犀专业版”旨在为每位用户提供专属的AI办公助理，而“WPS Comate”则作为全新组件，助力WPS 365实现全面升级。金山办公CEO章庆元在会上强调，办公软件已迈入第三个发展时代，即从传统的单机办公、协同办公，全面转向智能化的AI办公。
7月17日，世界人工智能大会展览在上海世博展览馆拉开帷幕。在此次盛会上，阿里巴巴集团集中展示了其面向各行各业及前沿科学研究的多元化AI智能体平台，其中包括阿里云百炼推理平台和“达摩灵枢”（DAMO Lingshu）科研智能体平台等。据悉，阿里云百炼推理平台迄今已为全球500万用户提供服务，其中付费用户达120万。此前，科研团队在“达摩灵枢”的辅助下，利用材料基础模型Elements成功预测了6.8万种潜在的超导材料，其中已有4种全新材料被成功合成并验证其超导特性。
中国商务部新闻发言人近日在例行新闻发布会上，就推动安世半导体相关问题进展做出回应。发言人指出，中荷两国在过去十年间建立了开放务实的全面合作伙伴关系，双边经贸合作持续深化。此前，荷兰外贸与发展合作大臣舍尔茨玛访华，与中方商务部长共同主持召开中荷经贸混委会第18次会议，双方就两国及中欧经贸关系坦诚深入地交换了意见。中荷双方一致同意，两国政府应积极营造有利环境，鼓励企业通过协商解决商业纠纷，共同维护全球半导体产业链供应链的安全与稳定。
行业数据显示，“十四五”规划实施以来，我国光热发电产业展现出强劲增长势头，年度复合增长率达到11.7%，远超全球平均4.24%的增速。展望“十五五”时期，中国光热发电产业预计将加速进入发展快车道。预测到2030年，我国光热发电总装机规模有望达到1500万千瓦左右。
韩国监管部门近日表示，鉴于对青少年社交媒体过度使用的担忧日益加剧，正考虑对14岁及以下儿童实施社交媒体使用限制。韩国媒体通信委员会主席金钟哲在政策简报会上强调，青少年过度使用社交媒体已成为一个全球性问题。韩国此项最新提议与全球多国政府采取立法措施限制儿童和青少年使用社交媒体平台的趋势不谋而合。欧盟已表示将在今年夏季后提出一项立法，旨在限制儿童访问此类平台。澳大利亚则已率先成为全球首个禁止16岁以下未成年人拥有社交媒体账号的国家。
