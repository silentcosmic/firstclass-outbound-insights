# 越中联手：30名高科技诈骗嫌犯已移交中方

🚨 越中联手：30名高科技诈骗嫌犯已移交中方
近日，越南广宁省警方在芒街国际口岸，正式将30名涉嫌跨国高科技网络诈骗的外国籍嫌疑人移交给中国警方。此举标志着越中两国在打击跨境电信网络诈骗犯罪方面的执法合作进一步深化。
