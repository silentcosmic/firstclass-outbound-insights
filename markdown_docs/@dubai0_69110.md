# 迪拜RTA：水上交通乘客享69迪拉姆特惠餐

📣 迪拜RTA：水上交通乘客享69迪拉姆特惠餐
迪拜道路与交通管理局（RTA）近日启动一项优惠活动，旨在惠及乘坐水上交通工具的通勤者及游客。凡是当天搭乘渡轮、水上出租车或Abra抵达迪拜码头的乘客，均可在海滨的The Pods餐厅以69迪拉姆的特惠价格享用专属餐点。
乘客只需向餐厅出示当日有效船票即可兑换，每张船票限用一次。此举旨在提升水上交通的吸引力，并为市民及游客提供额外的便利与福利。
