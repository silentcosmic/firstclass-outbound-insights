# 柬埔寨“诈骗帝国”揭秘：高薪诱骗日籍嫌犯被捕

🚨 柬埔寨“诈骗帝国”揭秘：高薪诱骗日籍嫌犯被捕
日本媒体近期将柬埔寨形容为“诈骗帝国”，专题报道当地针对海外受害者的跨国诈骗问题。当前，日本不仅面临国内特殊诈骗金额持续攀升的挑战，更有大量公民被高薪招聘广告诱骗至柬埔寨，抵达后即遭监控、非法拘禁，被迫卷入诈骗活动。
一名受害者案例显示，高薪兼职诱惑往往是陷阱，受害者一旦抵达柬埔寨便被非法拘禁并强迫从事诈骗。今年4月，柬埔寨当局突袭金边一处诈骗窝点，抓获8名涉案人员，其中5名是日本籍。执法人员在现场发现大量伪造的日本警察宣传品、制服及证件，窝点甚至被精心布置成日本警局场景，并利用AI技术实时变脸，冒充警察进行视频诈骗。同时，现场还查获了大量日本居民的个人资料，包括老年人的身份和资产信息，用于精准筛选目标。
目前，这5名日本籍嫌疑人已因参与有组织网络诈骗被起诉并羁押。尽管柬埔寨持续加大对涉日诈骗窝点的查处力度，但被高薪诱骗出境的日本人数量仍在增加。日本驻柬埔寨大使馆几乎每天都接到亲友失联的求助，日本警方也证实已有数十名公民在柬埔寨失踪，其中部分可能已被诈骗集团控制，被迫沦为“话务员”。这一系列事件正引发日本社会对人口贩运和跨国犯罪的高度关注。
