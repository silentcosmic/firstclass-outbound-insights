# 823万诈骗案主犯越南落网，中越联手斩断逃亡路

🚨 823万诈骗案主犯越南落网，中越联手斩断逃亡路
一名涉嫌卷入高达823万元人民币（约115万美元）诈骗案的中国籍主犯，近日在越南成功落网，并已通过中越警方联合执法行动被正式移交中方。此案再次凸显了两国在打击跨境犯罪方面的紧密协作。
据越南警方通报，落网嫌疑人张世兴（1976年出生）曾担任上海多家文化传媒公司负责人。调查显示，其涉嫌以“拥有影视作品版权”为幌子，诱骗自由职业者签订字幕制作合作合同，以此实施诈骗，涉案金额巨大。
中国警方于2026年3月依法对张世兴发布拘留令后，他涉嫌非法入境越南，并计划进一步潜逃至柬埔寨，企图通过跨境流动规避法律追查。
然而，在中越两国执法部门的紧密协作下，其潜逃计划未能得逞。6月3日，越南北宁省公安在越南北部地区成功将其抓获，有效阻断了其继续跨境逃亡的路线。6月26日，双方在越南友谊国际口岸完成了嫌疑人移交手续，张世兴已被正式移交中国警方，将依法接受进一步调查和处理。
随着联合执法机制的日益完善，企图通过跨境潜逃逃避法律责任的犯罪嫌疑人，将面临更为严密的追逃网络，其跨境逃避法律制裁的空间正被进一步压缩。
