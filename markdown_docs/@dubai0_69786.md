# 阿治曼滨海大楼火情速控，民防部门高效处置无伤亡

🚨 阿治曼滨海大楼火情速控，民防部门高效处置无伤亡
近日，阿治曼滨海大道一栋大楼突发火情。阿治曼民防部门迅速响应，成功扑灭火势，确保了居民安全，无人员伤亡报告。
据当地通报，应急救援队接报后立即赶赴现场，迅速控制火情，并对楼内住户进行了安全疏散。官方补充，火势已被完全扑灭，且未蔓延至周边建筑，有效避免了次生灾害。
