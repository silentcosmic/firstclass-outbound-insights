# ️ 阿联酋铁路新规：最高罚款1万迪拉姆，违规者或面临监禁

⚠️ 阿联酋铁路新规：最高罚款1万迪拉姆，违规者或面临监禁
近日，阿联酋阿提哈德铁路公司（Etihad Rail）正式公布了乘客违规行为的详细处罚标准。新规明确，在列车、车站及相关设施内，乘客或访客的违规行为将面临100至10000迪拉姆不等的罚款。工作人员有权当场开出罚单，且缴纳罚款并非终结，有关部门仍可能将违规行为移交检察机关，并依据相关法律采取追加罚款乃至监禁等严厉措施。
此举旨在提升公共交通的安全与秩序，确保乘客的舒适体验。根据公布的细则，最高10000迪拉姆的罚款将针对未经许可的商业摄影、列车运行中上下车、篡改或损坏安全设备、向列车内外投掷物品、擅闯铁轨或禁区、损坏铁路设施等严重行为。此外，如醉酒乘车、在列车上持有或饮用酒精饮料、伪造车票、在禁烟区吸烟、扰乱秩序或骚扰他人等，将面临5000迪拉姆的罚款。
