# 星展银行向ETAFCo提供2.1亿美元融资，加速亚洲能源转型

星展银行（DBS）向ETAFCo提供了一项2.1亿美元的高级融资。ETAFCo是Clifford Capital管理的一项能源转型合作投资载体。
此项交易使得星展银行成为新加坡金融管理局（MAS）“FAST-P”倡议下，唯一支持两项合作的商业银行。
这笔融资是ETAFCo首次获得的贷款，其资金将用于支持能源转型基础设施的债务投资。
具体范围涵盖可再生能源、电网现代化、能源储存，以及其他旨在减少对燃煤发电依赖的清洁能源解决方案。
能源转型加速融资（ETAF）合作计划旨在为亚洲各地的清洁能源项目调动优惠资本和私人资本。
该计划通过资助替代燃煤的清洁能源方案来支持减排。同时，其替代战略也支持煤炭设施的有序淘汰。我们相信可持续发展并非一个独立的议程，而是长期价值的核心驱动力。当以务实的态度推进时，它能增强经济竞争力，改善民生，并为未来建立韧性。
FAST-P倡议汇聚了公共、私人和慈善资本。该计划采用混合融资结构，通过动员绿色投资资金来满足亚洲的基础设施需求。
在第一阶段，ETAFCo将专注于清洁能源转型和电网基础设施项目。通过优化风险分配，该投资载体旨在支持具备可融资性的转型基础设施，并为亚洲地区释放新的资本来源。
此前，星展银行已向“绿色投资合作计划”注资7500万美元。该计划是FAST-P框架下的另一个混合融资工具，由Pentagreen Capital管理。
星展银行在该计划的高级批次中担任牵头协调方。
