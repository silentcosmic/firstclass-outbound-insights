# 越南警方捣毁特大跨境男性卖淫团伙

🚨 越南警方捣毁特大跨境男性卖淫团伙
越南河内市警方近日宣布，成功捣毁一个长期利用网络组织男性卖淫活动的犯罪团伙。此次行动共抓获3名涉嫌组织卖淫的犯罪嫌疑人，并查明约180名男性卖淫人员，涉案非法获利超过10亿越南盾（约合28万元人民币）。
据警方披露，该团伙主谋武光辉（37岁）伙同欧阳礼（43岁）和黄友义（25岁），搭建并运营多个伪装成“模特服务”或“社交交友”的网站，通过线上平台为越南国内外客户提供男性卖淫服务。这些网站详细展示卖淫人员的照片、外貌及活动区域，并根据客户需求安排人员前往不同城市“陪同出游”。
此案由河内市公安局网络安全部门在日常巡查中发现。7月8日下午，警方在河内市一酒店内当场查获一起卖淫交易，并循线在胡志明市抓获主犯武光辉。初步调查显示，该团伙自2023年初开始运作，活动范围覆盖越南多个省市，累计非法获利巨大。目前，3名嫌疑人均已因涉嫌组织卖淫被刑事拘留，案件正在进一步调查中。
