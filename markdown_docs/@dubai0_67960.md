# ️ 阿布扎比航班突发悲剧：印度侨民机上病逝

⚠️ 阿布扎比航班突发悲剧：印度侨民机上病逝
周一凌晨，一名印度籍乘客在从阿布扎比飞往班加罗尔的航班上突发疾病，不幸离世。死者已被证实为51岁的尤萨夫·昆哈姆（Yousafe Kunhammu）。
据了解，该乘客当时正从阿联酋返回印度。其不幸离世，留下妻子及五名年幼子女，其中最小的仅三个月大，家庭状况令人唏嘘。家属与航空公司均已确认此悲剧性事件。
