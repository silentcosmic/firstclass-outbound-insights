# 波贝警方突袭：5名中国籍嫌犯涉诈落网，搜出枪支

🚨 波贝警方突袭：5名中国籍嫌犯涉诈落网，搜出枪支
6月25日上午，柬埔寨波贝市警方在国家警察总署总监苏特指示下，由副总监蒙苏塔现场指挥，联合多部门展开大规模追捕行动。行动中，警方成功拦截两辆企图冲撞执法人员并逃逸的车辆，当场抓获5名中国籍嫌疑人，并查获大量涉嫌网络诈骗的电子设备及一支手枪。
据了解，两辆可疑车辆在奥哲罗分区接受检查时，突然强行冲卡逃逸。联合执法人员随即展开高速追捕，最终在波贝市San Kor经济特区附近将其全部截停。落网的5名中国籍男子身份被确认为LI RUI（30岁）、WANG BO XIN（28岁，绰号“阿勇”）、WANG YUAN（30岁）、ZHANG CHENG SHEN（34岁）和ZHANG MING XUAN（30岁）。
现场查获两辆2025款黑色GAC GN8商务车。其中一辆（金边2CH-2388）内藏有大量疑似用于网络诈骗的作案设备，包括155部手机、1台CPU主机、2台显示器、1台笔记本电脑及5个U盘。另一辆车（柬埔寨D.D.0168）中则搜出一支ZORAKI MOD 914-T手枪（配1个弹匣、7发子弹）及一根伸缩铁棍。此外，警方还扣押了两部涉案人员的iPhone手机作为证物。
目前，所有嫌疑人及涉案物证均已移交波贝市警察局，警方正依法展开进一步调查，并深入追查案件背后是否牵涉跨国电信网络诈骗犯罪团伙。
