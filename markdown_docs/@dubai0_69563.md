# 中国移民局启动“捕蛇”行动，集中缉捕严重逃犯

🚨 中国移民局启动“捕蛇”行动，集中缉捕严重逃犯
中国国家移民管理局于7月10日召开新闻发布会，宣布今年下半年将重磅启动新一轮“捕蛇”追逃行动，集中缉捕罪行严重、影响恶劣并长期藏匿的重点逃犯。国家移民管理局新闻发言人林勇胜指出，此举旨在全面强化追逃举措，并协同深化国际执法合作。
林勇胜介绍，本次行动将聚焦打击妨害国（边）境管理犯罪，尤其针对组织内地居民非法出境从事赌诈，以及组织外籍人员偷渡来华从事违法犯罪活动等重点案件。移民管理机构将集中优势警力，挂牌督办，深挖幕后组织者和运送者，彻底摧毁犯罪网络。
为确保行动成效，移民管理局将一体推进打击与治理工作，加强重点行业日常管理，查处取缔违法违规中介机构，并严查严管涉偷渡违法犯罪重点人员。同时，将充分利用全球公共安全合作论坛等双多边警务合作平台，与重点国家和地区执法部门在情报通报、联合办案及跨境追逃等领域开展深度合作，全力抓捕潜逃境外的“蛇头”骨干。
移民管理局呼吁社会各界积极举报相关违法犯罪活动，共同维护国家安全和正常出入境管理秩序。
