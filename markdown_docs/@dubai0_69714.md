# 泰国严查USDT大额交易：近四成卖家是外国人

🚨 泰国严查USDT大额交易：近四成卖家是外国人
近期，泰国金融监管机构正对大额泰达币（USDT）交易展开联合调查。据市场消息，泰国央行与证券交易委员会重点关注隐藏实际所有人、规避跨境汇款规定以及利用稳定币进行洗钱的潜在风险。
此次调查的导火索是监管部门发现，本地数字资产平台上的USDT卖家中，近四成是外国人或非泰国常住居民。这类非本地居民频繁出售USDT的行为，已引发监管机构对资金来源、交易目的及市场合规性的高度关注。
作为反洗钱措施的一部分，泰国已要求单笔超过500万泰铢（约合14万美元）的现金存款必须提供资金来源证明。监管部门怀疑，部分资金可能通过USDT转移，再兑换成泰铢或购买黄金，以此规避追踪。因此，数字资产平台与传统金融机构之间的信息核查正在加强。
展望未来，泰国计划从2026年第四季度起，进一步收紧稳定币及数字资产交易监管。届时，数字资产平台将面临更严格的客户身份、资金来源和实际受益人审查，并加强对非居民账户和异常交易的监控。此举预计将提高合规成本，并可能导致部分USDT交易量下降。不过，监管部门强调，此次行动主要针对涉嫌高风险交易，并非禁止正常的数字资产投资。
