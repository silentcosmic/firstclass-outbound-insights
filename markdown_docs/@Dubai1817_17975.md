# 甘肃警方破获无人机助电诈取现案，涉案百万

🔍 甘肃警方破获无人机助电诈取现案，涉案百万
甘肃兰州警方近日通报，成功侦破一起利用高科技设备协助电信网络诈骗团伙取现的案件。调查显示，犯罪团伙在6月下旬操控多架未报备无人机，在警方办案区域附近进行违规飞行，其异常轨迹与电诈案件预警及线下取现活动高度吻合。
经查，该团伙利用无人机对警方行动进行实时航拍监控，并将画面传输给取现“车手”，以规避抓捕。警方最终锁定并成功抓获4名犯罪嫌疑人，现场查获无人机、移动摄像头、对讲设备等17套反侦查工具。经审讯，该团伙涉嫌协助境外电诈团伙转移赃款，涉案金额已超过100万元。目前，案件正在进一步侦办中。
