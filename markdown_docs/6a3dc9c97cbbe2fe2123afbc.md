# Airwallex完成3.2亿美元H轮融资，估值突破110亿美元，重塑AI驱动型金融软件未来

全球支付巨头Airwallex近日宣布完成3.2亿美元H轮融资。此轮注资不仅使其估值跃升至110亿美元，更将为其进军人工智能驱动型金融软件领域提供强大资金支持。
本轮融资由原有投资者Addition领投，Baillie Gifford、Hummingbird、QED Investors、T. Rowe Price、Hedosophia、Haun Ventures、Washington University in St. Louis以及Amex Ventures等机构也积极参与。
Airwallex方面表示，这笔新注入的资金将主要用于加速其在自主金融（autonomous finance）领域的产品研发进程。
此外，该企业亦计划拓展其在新市场的技术基础设施建设及监管合规布局。“我们坚信，这是全球金融史上最具决定性意义的时刻，我们正为此积极构建，”Airwallex高层表示。
伴随此次融资的完成，Airwallex同步推出了两款旨在赋能自主金融的新产品。
其中之一是T:0，一个原生于AI技术、旨在全面管理企业财务职能的平台，涵盖了记账、预测、税务以及合规等关键环节。
另一款产品则是面向消费者的数字钱包Airi，该产品在推出之初就将整合Airwallex现有的“一键支付”功能。
Airwallex计划将Airi进一步扩展为一个更广泛的基础设施，以支持授权支付、消费限额及多币种余额管理等多元化功能。
截至2026年3月，Airwallex的年化收入已达到13亿美元，年化交易量更达到了2870亿美元的规模。
此次H轮融资的估值，较其在2025年12月实现的80亿美元估值有了进一步的跃升。
