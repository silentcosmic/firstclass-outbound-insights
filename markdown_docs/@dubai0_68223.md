# 越南平新郡突发：警方地毯式搜查，公司紧急撤离！

🚨 越南平新郡突发：警方地毯式搜查，公司紧急撤离！
据市场消息，越南胡志明市平新郡今晚突发警方大规模搜查行动。此次行动范围广泛，涵盖酒店、街道乃至私人住宅，进行地毯式检查。
受此影响，当地多家公司已紧急通知员工全员撤离，并计划将业务暂时转移至河内，以规避潜在风险。
