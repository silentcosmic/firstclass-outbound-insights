# AI浪潮驱动：韩国、马来西亚、泰国及中国台湾经济增长表现突出

国际货币基金组织（IMF）在其最新发布的《世界经济展望》报告（2026年7月版）中指出，韩国、中国台湾、泰国和马来西亚这四个经济体，作为全球领先的AI相关硬件净出口国，在本轮全球科技周期中表现尤为突出。其增长核心动力源于人工智能工具的进步与广泛应用。
数据显示，2026年第一季度，上述四个经济体的平均经季节性调整后的年化增长率超出预期4.4个百分点。与此同时，全球其他国家同期平均增长率则低于预期0.3个百分点。
国际货币基金组织特别强调，这四个经济体凭借其在全球AI相关硬件出口中的主导地位，成为本轮全球科技浪潮中最引人注目的增长引擎，其背后是人工智能技术迭代与应用的深层驱动。
韩国在2026年第一季度的经季节性调整年化增长率达到了7.5%。国际货币基金组织分析认为，这主要得益于半导体和AI硬件出口的强劲增长。尽管韩国高度依赖从中东进口能源，但其技术出口的蓬勃发展足以抵消能源成本上升带来的负面影响。
针对韩国全年经济增长，2026年预测值已上调0.7个百分点至2.6%，相比4月份发布的报告有所提升。2027年的预测增长率为2.5%，也向上修正了0.4个百分点。
马来西亚方面，2026年增长预测保持在4.7%不变，2027年预测为4.3%。国际货币基金组织将马来西亚的良好表现直接归因于数据中心活动的活跃以及全球科技周期的全面回暖。
泰国的2026年增长预期上调0.4个百分点至1.9%，这反映了其紧急财政措施的有效实施，以及科技相关出口和投资的强劲态势。2027年增长预测为2.2%，亦上调了0.1个百分点。
国际货币基金组织并未披露中国台湾地区的更多细节数据，但提供了其他一些重要经济体的相关信息。
就中国经济而言，根据国际货币基金组织工作人员的估算，2026年第一季度GDP经季节性调整后的年化增长率达到了8.1%，超出市场预期。此次扩张主要得益于前瞻性的公共基础设施投资以及高科技制造业和出口的显著增长。2026年全年增长预计为4.6%，而2027年预测为4.1%，相比此前上调了0.1个百分点。
美国的GDP在2026年第一季度实现了2.1%的年化增长。全年增长预计为2.3%，2027年则预测为2.2%。国际货币基金组织认为，持续的科技相关商业投资和强劲的生产力是支撑其增长的关键因素。
越南的2026年增长预测为7.5%。国际货币基金组织指出，超出预期的科技出口和强劲的国内需求是驱动这一预测上调的两大主要因素。
全球经济产出预计在2026年增长3.0%，2027年增长3.4%，低于2024-2025年平均3.5%的水平。其中，发达经济体预计在2026年增长1.7%，2027年增长1.8%。新兴市场和发展中经济体的增长预测则分别为2026年的3.8%和2027年的4.5%。全球贸易量增长预计将从2025年的5.0%放缓至2026年的3.5%，随后在2027年恢复到4.3%。
