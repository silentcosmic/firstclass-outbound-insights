# 马来西亚政府AI治理前瞻：总理数字分身即将亮相公共视野

马来西亚总理安瓦尔·易卜拉欣即将采取一项前所未有的举措，将其人工智能（AI）数字分身推向公众视野。此举在全球各国政府首脑中实属罕见。
这个数字分身由马来西亚数字基础设施公司Zetrix AI Bhd倾力打造，并被命名为PMX AI，此名称旨在纪念安瓦尔作为马来西亚第十任总理的身份。据了解，该数字分身预计将在数日内首次亮相。
PMX AI的训练数据源自安瓦尔总理的个人著作、公开演讲以及其政府的政策记录。因此，当它发言时，无论从形象还是声音上，都将与安瓦尔本人高度相似，足以达到以假乱真的程度。这一创新项目由安瓦尔所属的政党——人民公正党（PKR）主导发起。Zetrix集团董事总经理黄天送（TS Wong）指出：“人工智能将彻底改变政府治理和政治格局。”
简而言之，AI数字分身是真实人物的数字化替身，能够与用户进行对话互动。而PMX AI的独特之处在于其卓越的自主性。工程师们将其设计为具有“智能代理”（agentic）能力，这意味着它能够自主接收任务、将其分解为多个步骤，并在很大程度上独立完成这些任务，无需人工在每一步都进行干预。
Zetrix公司还开发了一套“个人知识”模型，这是一个AI系统，能够持续不断地获取总理最新的演讲和言论。这一机制有助于实时优化PMX AI的回复，使其表达更接近总理本人的风格和思想。
总理办公室已经制作了一段发布视频，用以正式介绍这一创新成果。
