# 中老联合打击电诈：遣返人员引关注

🚨 中老联合打击电诈：遣返人员引关注
近日，在中老警务联合打击行动背景下，一批涉嫌电信诈骗的女性人员从老挝被移交至云南西双版纳磨憨边境口岸。这批人员的转运画面引发了公众广泛关注。
令人意外的是，许多网友观察到这些被遣返人员普遍体态良好，与外界普遍认为的受困者形象大相径庭。这一现象侧面反映出境外电信诈骗团伙内部的优渥条件，其成员在从事非法活动期间，生活水平甚至远超预期。
据了解，在口岸现场，所有人员身着统一马甲，由移民管理民警全程护送，并经由专门划定的消毒隔离通道进行闭环管理。她们将逐一进行信息登记与身份核验，随后依法接受处理。随着联合整治行动的深入，这些涉案人员的“好日子”已然终结，等待她们的将是法律的严惩。
