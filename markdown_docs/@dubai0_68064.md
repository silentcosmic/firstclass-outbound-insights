# 阿联酋强化反洗钱：携手金融学院升级监管能力

⚡ 阿联酋强化反洗钱：携手金融学院升级监管能力
阿联酋经济与旅游部近日与阿联酋金融学院（EIF）签署谅解备忘录，旨在全面提升国家在金融监管、治理及反洗钱（AML）领域的执法能力。此举标志着阿联酋在打击金融犯罪方面迈出重要一步。
根据协议，双方将共同开发专业培训项目，为指定非金融企业和专业人士（DNFBPs）的合规人员提供资格认证。合作重点将围绕金融行动特别工作组（FATF）的标准，加强风险评估、尽职调查和可疑交易监控，以有效打击洗钱和恐怖主义融资活动。
此次合作还包括共享研究成果和技术报告，并联合举办研讨会，以支持决策制定、提升机构绩效，进而推动“我们阿联酋2031”愿景中建设具有竞争力和可持续发展能力经济的目标。
