# USDT用户突破5亿：Tether加速布局AI与新兴市场

🔥 USDT用户突破5亿：Tether加速布局AI与新兴市场
Tether首席执行官Paolo Ardoino近期披露，全球最大稳定币USDT的网络钱包地址已突破5亿大关，并以每季度超过3000万个新钱包的速度持续增长。最新数据显示，2025年第四季度USDT生态系统新增用户逾3500万，推动其市值达到1873亿美元，稳居全球稳定币首位。链上数据亦显示，USDT持币地址已达1.391亿，月活跃用户创历史新高，达到2480万。
这一显著增长主要得益于USDT广泛的流通渠道、低廉的跨境转账成本，以及在发展中国家日益增长的非交易场景应用。在许多本币贬值压力大、传统金融服务不足的地区，USDT正被视为数字化美元替代工具，用于储蓄、跨境汇款和国际支付。Tether正积极布局非洲市场，通过投资肯尼亚金融科技公司Kotani Pay，推动当地区块链支付与跨境结算发展。
在巩固稳定币业务的同时，Tether正加速向人工智能、能源及金融科技等多元领域扩张。其AI开发工具QVAC近期升级并引入TurboQuant技术，旨在提高AI模型运行效率；同时，Tether支持的数字黄金稳定币XAUT也持续扩张。展望未来，尽管USDT用户规模仍保持快速增长，但日益趋严的全球监管环境将是其持续增长面临的关键挑战。
