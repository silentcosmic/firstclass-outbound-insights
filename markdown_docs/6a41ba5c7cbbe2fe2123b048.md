# 新加坡设立未来金融研究院，加速金融创新规模化应用

新加坡金融管理局 (MAS) 周四宣布，将设立未来金融研究院 (FFI)，旨在加速新兴金融技术的采纳，并促进金融领域的创新发展。
根据MAS的声明，FFI初期将聚焦人工智能 (AI) 和代币化技术，致力于连接金融与科技生态系统，并提供共享资源和专业知识，以推动这些技术在更广泛范围内的普及应用。
该声明指出，FFI的成立，正是基于新加坡金融管理局在人工智能和代币化领域的各项举措中，所积累的深厚公共与私营部门合作基础。
在人工智能领域，MindForge人工智能风险管理工具包由领先的银行、保险公司、资产管理公司及科技企业共同开发，旨在提升行业管理新兴人工智能风险的能力和实践水平。
同样，拥有200多家金融机构参与的PathFin.ai平台，已成为业界开发和验证新兴人工智能应用场景的共享平台。
这一合作模式也推动了代币化技术的发展。例如，守护者计划 (Project Guardian) 汇集了银行、托管机构和市场基础设施运营商，在六种货币环境下，对涵盖外汇、基金和固定收益领域的代币化实际应用场景进行了测试。与此同时，兰花计划 (Project Orchid) 则为用途限定货币的测试以及数字新加坡元基础设施的开发奠定了基础。
FFI将把这些分散的努力整合到一个统一的协调机构之下，这标志着新加坡金融创新发展进入了新阶段。
MAS将继续制定金融创新的政策并完善相应的监管框架，而FFI则将直接与行业合作，加速新技术从实验阶段向大规模部署的转化，从而降低各类规模机构的采纳门槛。
FFI将通过连接金融行业与国内外科技研究者、解决方案提供商以及金融科技生态系统，来统筹和促进生态发展。此举旨在推动深度合作，将现有能力转化为实际解决方案，促进行业范围内的技术应用规模化，并为新加坡国民在金融科技重塑金融业的过程中，把握新兴就业机遇。
MAS强调，FFI将提供四大能力支柱，以支持前沿技术在全行业的应用和部署。
首先是知识中心，它将是一个扩展的资源库，汇集了经过验证的应用案例、部署手册和解决方案提供商，并辅以能力发展路线图，旨在协助各类规模的金融机构采纳前沿技术。
其次是创新工坊，它将围绕特定的人工智能和代币化项目，统筹全行业合作，汇聚研究机构、金融科技公司和金融机构的资源，加速从实验到实际部署的进程。
第三是行业沙盒，它通过受控测试环境，帮助金融机构在安全的环境中测试创新，进行新兴技术（包括可编程货币、代币化资产和人工智能驱动的工作流程）的实验和预部署验证。
第四是实施工具包，它将开发参考工具包，包括一个针对代币化资产的可编程合规工具包，以及一个包含部署智能代理AI（agentic AI）适当安全保障实施指南的更新版AI风险管理工具包，旨在帮助金融机构实现可持续的规模化应用。
MAS表示，FFI将设立一个由新加坡金融管理局、主要金融机构、科技公司和学术界代表组成的董事会。
FFI还将吸纳具备深厚行业和技术经验的实践者，确保其运营具备必要的灵活性，以适应技术发展的快速步伐。
声明指出，FFI的战略和治理细节将于今年晚些时候公布。
