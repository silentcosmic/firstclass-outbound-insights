# ️ 洪森警示世界杯赌球风险：球迷输到“怀疑人生”

⚠️ 洪森警示世界杯赌球风险：球迷输到“怀疑人生”
柬埔寨参议院主席洪森近日就本届世界杯发表看法，指出赛事冷门频出，导致不少赌球者损失惨重，甚至“输到怀疑人生”。
洪森表示，他一直密切关注世界杯，并形容“足球是圆的”，比赛结果充满变数。随着南美和北非球队表现强势，德国、荷兰等欧洲传统强队相继出局，让赛前看好的赌球者大感意外。
此外，洪森还提到，本届世界杯的不少焦点赛事都在深夜和凌晨进行，导致许多球迷连续熬夜，睡眠不足，进而影响白天的正常工作效率。他特别提醒广大驾驶员，熬夜观赛后务必避免疲劳驾驶，确保充足休息后再驾车出行，以保障自身及他人的生命安全。
