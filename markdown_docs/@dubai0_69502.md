# 柬埔寨器官移植案：中方严正驳斥日媒“中国医生”报道

📣 柬埔寨器官移植案：中方严正驳斥日媒“中国医生”报道
中国驻柬埔寨大使馆近日通过官方渠道发布声明，严正驳斥了日本媒体关于“中国医生在柬埔寨参与器官移植”的报道。大使馆在与柬埔寨王家军总医院核实后确认，该院从未有中国医生参与过肾脏移植手术，指责日本《读卖新闻》相关报道纯属虚假信息。
此前，日本媒体曾披露，东京警视厅于7月7日逮捕了三名日本男子，他们涉嫌协助一名居住在东京的70多岁男子在柬埔寨接受肾脏移植手术，并从中收取高达1236万日元（约合7万美元）的报酬。日媒报道中特别提及，此次手术的主刀医生为中国籍，并有协调员全程陪同。
