# 迪拜飙车还拆牌？面临5万迪拉姆重罚

🚨 迪拜飙车还拆牌？面临5万迪拉姆重罚
迪拜近日发生一起严重交通违规事件：一名司机在道路上飙车，时速超过230公里/小时，并为逃避智能监控系统，故意拆除了车辆牌照。
迪拜警方迅速介入并逮捕了该司机，对其处以高达5万迪拉姆（约合9.8万元人民币）的重罚，其车辆也被依法扣押。此案再次凸显迪拜在交通安全及法规执行上的严格立场。
