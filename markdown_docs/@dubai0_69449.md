# ️ 阿联酋航空A380首飞德里：印度第三城迎双层巨无霸

✈️ 阿联酋航空A380首飞德里：印度第三城迎双层巨无霸
阿联酋航空近日宣布，自今年10月25日起，迪拜往返德里的航线将首次由旗舰机型空客A380执飞。这意味着，德里将成为继孟买和班加罗尔之后，印度第三个迎来这款标志性双层客机服务的城市，预示着该航线运力和高端服务体验的显著提升。
