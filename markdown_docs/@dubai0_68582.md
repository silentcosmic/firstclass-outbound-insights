# 东南亚诈骗集团转战斯里兰卡，手法升级！

🚨 东南亚诈骗集团转战斯里兰卡，手法升级！
随着东南亚各国对网络诈骗活动的持续高压打击，此前在该地区盘踞的国际犯罪集团正迅速将斯里兰卡作为新的集结地。据官方报告，仅2026年前五个月，斯里兰卡已逮捕逾千名涉嫌网络诈骗的外国人，其中多数为中国、越南和印度公民，他们利用旅游签证入境后组织并实施诈骗。
斯里兰卡之所以成为新目标，有多重因素：包括对40多个国家（含中印）公民提供30天免费旅游签证且可延期；相对宽松的本地SIM卡和互联网管理；以及对在线博彩的监管漏洞。更关键的是，当地名为“Undiyal”的非正式资金转移系统能规避银行记录，加上斯里兰卡目前对跨国诈骗集团的起诉能力有限，使得该国成为犯罪分子理想的活动基地。
与在柬埔寨、缅甸大规模设立园区不同，这些诈骗团伙在斯里兰卡采取更分散、隐蔽的运作模式，通常以5至10人小组活动，并频繁更换酒店或公寓以逃避追踪。他们不仅针对外国受害者，也日益将斯里兰卡本地居民作为目标。诈骗手段多样，包括利用AI深度伪造技术进行投资及加密货币诈骗，甚至冒充斯里兰卡总统等公众人物背书虚假投资计划；同时，传统的“杀猪盘”式恋爱诈骗、虚假招聘及人口贩运也屡见不鲜。值得警惕的是，他们甚至渗透至政府层面，如2025年底至2026年初，斯里兰卡财政部曾发生250万美元的债务偿还款被黑客转移事件。
目前，斯里兰卡警方、移民局和中央银行正联手打击此类犯罪。然而，专家指出，现有法律漏洞导致大多数外国涉案人员最终仅被驱逐出境而非刑事起诉，这使得许多犯罪分子得以逃避司法制裁并继续作案。
针对日益严峻的诈骗形势，企业和个人应高度警惕任何制造紧迫感的投资机会、来自陌生人的“朋友介绍”式好友请求，以及索要OTP验证码、PIN码等敏感信息的行为。务必独立核实所有公司信息，并直接通过官方渠道进行交易，切勿点击可疑链接。
