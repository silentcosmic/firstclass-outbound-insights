# 迪拜免税店同日双响炮：两印籍侨民各揽百万美元

⚡ 迪拜免税店同日双响炮：两印籍侨民各揽百万美元
本月伊始，迪拜免税店的“千禧百万富翁”抽奖活动爆出惊喜：两位来自印度喀拉拉邦的侨民在同日分别赢得百万美元大奖，为迪拜的财富故事再添传奇。
其中一位是48岁的卡比尔·普瓦廷加尔，他在迪拜生活超过二十载，目前在阿布扎比从事航空运营工作。普瓦廷加尔凭借其6月13日在线购买的彩票，在参与迪拜免税店抽奖活动长达十二年后，终于斩获第547期100万美元大奖。他表示，这是一次“美妙且改变人生的经历”。
另一位幸运儿是30岁的穆罕默德·希比尔·塔伊尔，他近期抵达迪拜，目前在沙迦机场国际自由区担任销售。更令人称奇的是，塔伊尔仅在第二次购买彩票后，便以6月18日在线购得的彩票赢得了第548期百万大奖。得知中奖消息时，塔伊尔激动地称“你们改变了我的生活！”。
这两位来自喀拉拉邦的同乡，分别成为自1999年“千禧百万富翁”活动启动以来，第278位和第279位印度籍百万美元大奖得主。市场观察显示，印度籍参与者长期以来一直是这项抽奖活动最大的购票群体，彰显了迪拜在吸引全球人才和财富方面的独特魅力。
