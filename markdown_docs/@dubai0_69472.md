# 柬埔寨西港论坛：呼吁民间组织共筑数字安全防线

⚡ 柬埔寨西港论坛：呼吁民间组织共筑数字安全防线
柬埔寨西哈努克省（西港）近日聚焦数字安全与治理，由内阁办公厅民间社会组织联盟论坛与省政府联合主办了一场“数字化时代增强民间社会组织能力”对话论坛。会议强调了民间社会组织在打击虚假信息和网络诈骗方面的关键作用，呼吁各方共同构建数字安全防线。
论坛上，民间社会组织联盟论坛常务副主席杜威拉指出，在加速的数字转型浪潮中，基层民间组织应积极投身于社区层面的虚假信息与网络诈骗防治工作。他特别强调，洪玛奈总理领导的柬埔寨第七届政府高度重视民间社会组织的力量，将其视为促进国家各领域发展及提升基层民众福祉的“不可或缺的合作伙伴”。
西港省副省长隆迪蒙则从实践层面提出建议，鼓励各组织持续学习前沿数字技术，强化机构内部的数据安全管理体系，以有效抵御日益复杂的网络犯罪风险。同时，他倡导充分利用数字平台，实现与社区及地方政府的高效沟通，从而显著提升工作的透明度与运作效率。
