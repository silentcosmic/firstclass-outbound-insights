# 阿联酋国家铁路首发：谁将坐上1号座？

📣 阿联酋国家铁路首发：谁将坐上1号座？
阿联酋阿提哈德铁路近日发起一项全国性活动，邀请居民提名那些为社区做出贡献的人，争夺成为国家客运铁路史上首位乘客的殊荣。获选者将坐上2026年6月30日首趟从富查伊拉开往阿布扎比的客运列车，并获安排在具有特殊象征意义的1号车厢1号座，此举标志着阿联酋国家客运铁路网的正式启动。
提名活动将持续至当地时间6月27日（周六）上午10点。公众可通过阿提哈德铁路官方社交媒体平台提交身边默默奉献、热心助人或服务社区人士的故事，推荐亲友、同事或邻居，并说明其值得这份荣誉的理由。
阿提哈德铁路出行部门商务执行总监阿德拉·阿尔曼苏里指出，这份荣誉旨在致敬那些用行动让社区变得更好、推动国家向前发展的普通人。最终获选者将在6月30日首趟列车出发前公布。
