# 吉隆坡酒吧突袭行动，当局警示治安风险

🚨 吉隆坡酒吧突袭行动，当局警示治安风险
马来西亚吉隆坡移民局外勤行动组近日对当地多家酒吧进行了突击检查。行动中，多名涉事人员被捕，其中不乏试图藏匿于桌角、柜子或窗帘后的情况。
此事件再次引发了对吉隆坡治安状况的关注。有市场观察人士指出，近期当地居民对公共安全问题投诉增多，建议出海企业和相关人员提高警惕，避免不必要的夜间外出，以降低潜在风险。
