# ️ 柬埔寨强烈抗议：泰国带外媒入奥斯马，主权争议再起

⚠️ 柬埔寨强烈抗议：泰国带外媒入奥斯马，主权争议再起
柬埔寨新闻部近日发表严正声明，强烈抗议泰方再次未经允许，擅自带领包括澳大利亚广播公司（ABC）在内的外国记者进入奥斯马地区。
柬新闻部副国务秘书兼发言人德阿斯纳里强调，泰军此举严重侵犯柬埔寨主权，违反国际法及联合国宪章。柬方坚决否认并拒绝任何基于此次非法行为、旨在抹黑柬埔寨国际形象的报道。奥斯马地区此前曾被指与电信诈骗园区活动有关，此次事件恐将使两国在敏感边境地带的舆论博弈进一步升级。
