# ️ 缅甸园区亲历者：高薪诱骗，拒绝诈骗即遭电击

⚠️ 缅甸园区亲历者：高薪诱骗，拒绝诈骗即遭电击
乌干达籍受害者吉迪恩近日再度发声，详述其在缅甸诈骗园区的恐怖经历，揭露了犯罪集团以“高薪工作”为饵，实则进行人口贩运、非法拘禁及暴力胁迫的残酷真相，再度引发国际社会对东南亚诈骗问题的警惕。
吉迪恩回忆，他原以为是应聘泰国的高薪职位，不料抵达后却被层层转运，最终被强行带至缅甸克伦邦一处名为“泰昌”的诈骗园区。踏入园区的那一刻，他才惊觉这是一场精心策划的骗局。
在园区内，所有受害者均遭武装看守，人身自由被完全限制，被迫从事电信诈骗并完成每日业绩。吉迪恩因坚决拒绝参与诈骗，随即成为惩罚目标。电击、殴打、威胁和辱骂成为家常便饭，任何不服从、未达标或稍有懈怠者，都将遭受残酷体罚。
园区惩戒手段极其残忍，包括电击、暴晒、关禁闭及毒打。对试图逃跑者，惩罚尤为严酷，吉迪恩本人就曾被手铐锁在露天围栏内五天五夜。更令人发指的是，他亲眼目睹一名埃塞俄比亚籍男子因求救被武装人员当众殴打致死，这一幕成为他挥之不去的噩梦。
最终，吉迪恩冒险藏匿手机发出求救，在家属与多方机构的协同营救下成功脱险，返回乌干达。获救后，他通过公开演讲、媒体采访及音乐创作等多种形式，持续讲述自身遭遇，旨在警醒世人，防止更多人落入陷阱。
