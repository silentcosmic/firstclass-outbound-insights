# 台版KK园区案：竹联帮涉贩运缅甸女性，18人被诉

🚨 台版KK园区案：竹联帮涉贩运缅甸女性，18人被诉
近日，一起被外界称为“台版KK园区”的跨境人口贩运案在台湾引发广泛关注。据台媒报道，一个拥有竹联帮和堂背景的犯罪团伙，涉嫌长期以高薪工作为诱饵，诱骗缅甸女性赴台后实施非法拘禁、强迫卖淫及债务控制。该案已于2026年3月侦破，目前检方已依组织犯罪、人口贩运等多项罪名，对18名涉案人员提起公诉。
调查指出，该犯罪集团自2024年起以“金三角国际公司”为幌子，由成员柯讲韬在缅甸、泰国等地通过社交平台发布虚假高薪招聘信息，如夜店服务员、工厂员工等，诱骗外籍女性。受害者抵达台湾后，团伙立即以垫付机票、签证等名义，为其虚构高达11万至18万元新台币的巨额债务，以此进行人身控制。
受害女性被集中关押在新北市三重区等地出租屋内，房间布满监控，严禁与外界联系。犯罪团伙还强迫她们拍摄裸照，并以其远在缅甸的家人相要挟，迫使她们就范。检方调查显示，受害者被迫每日接待多名嫖客，导致健康问题却被禁止就医，所有卖淫所得均被犯罪集团侵吞。若有反抗，团伙便以违约为由追加高达45万元新台币的所谓“赔偿金”，进一步加重受害者负担。
案件最终因一名化名“圆圆”的缅甸女子趁看守松懈成功逃脱并报警而曝光。接获报案后，台湾移民署新北市专勤队联合检警及调查机关成立专案小组。2026年3月30日，专案组展开收网行动，共逮捕18名涉案嫌疑人，并成功解救9名遭非法拘禁、强迫卖淫的缅甸籍女性。行动中，执法人员查扣了豪华轿车、现金及枪支、刀具等涉案物品。
