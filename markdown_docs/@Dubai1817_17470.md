# 委内瑞拉连发强震：建筑倒塌恐致人员伤亡

⚡ 委内瑞拉连发强震：建筑倒塌恐致人员伤亡
当地时间6月24日傍晚至25日凌晨，委内瑞拉连续发生两次强烈地震，震级分别达到里氏7.1级和7.5级。
据初步报告，地震已造成包括摩天大楼在内的多处建筑物倒塌，恐已造成人员伤亡。目前，灾情及具体损失仍在评估中。
