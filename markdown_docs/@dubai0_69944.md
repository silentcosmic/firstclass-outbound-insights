# 孟加拉国前总理哈西娜超62亿资产遭冻结

🚨 孟加拉国前总理哈西娜超62亿资产遭冻结
孟加拉国当局于7月15日宣布，已冻结并扣押前总理谢赫·哈西娜、其家人及10家关联企业约7600亿塔卡（约合62亿美元）的资产。法新社报道指出，其中超过46亿美元资产位于孟加拉国境内，另有逾15亿美元在海外。目前，当局已针对哈西娜及其关联人员启动了98起案件的立案调查。
哈西娜自两年前下台以来，已因多项指控被缺席审判，包括涉及达卡豪华住宅区土地分配的腐败案。尽管在流亡印度期间被判处死刑，哈西娜此前仍表示计划于今年12月返回孟加拉国。
