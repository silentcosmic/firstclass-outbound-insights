# 缅甸星链服务恐大面积停用：新规致断网风险

🚨 缅甸星链服务恐大面积停用：新规致断网风险
据市场消息，SpaceX旗下星链（Starlink）卫星网络服务在缅甸正面临重大调整。由于SpaceX实施新规，缅甸境内大量未经官方批准的星链设备或将于7月17日后面临停用风险。
新规要求，设备若在非注册国使用超过30天，用户必须更改地区设定并进行强制性护照旅行登记，否则服务将被中断。鉴于缅甸尚未正式批准星链服务，这一新政恐导致下月大批设备断网。此外，服务资费也已有所上调。
过去，星链在实皆省、掸邦等网络受限地区对人道主义通讯至关重要。然而，近期该设备被边境电信诈骗团伙滥用的情况日益增多，迫使星链服务政策不得不收紧。
