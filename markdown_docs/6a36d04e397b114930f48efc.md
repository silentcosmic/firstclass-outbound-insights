# NewGen战略增持K25.ai股权至10%，助推其完成Pre-A轮融资

泰国科技企业NewGenIVF集团有限公司（以下简称“NewGen”）近日宣布，已签署协议，将向K25.ai追加400万美元的战略投资。
此次交易标志着K25.ai以1亿美元估值完成总计1000万美元的Pre-A轮融资。NewGen在一份声明中表示，交易完成后，并满足所有适用交割条件后，NewGen在K25.ai的总持股比例预计将增至10%。
随着Pre-A轮融资的成功落幕，K25.ai已正式启动A轮融资。
此前的投资包括NewGen于2026年5月公布的首笔200万美元投资，以及随后在2026年6月4日追加的400万美元投资。
总计1000万美元的Pre-A轮融资为K25.ai的产品发布提供了战略支持。而新启动的A轮融资，则旨在推动其商业化进程、获取相关监管许可，并加速在选定亚洲市场的业务拓展。
K25.ai是一个专为亚太地区打造的、以AI为核心的直播与“观看即预测”信息市场平台。
该平台将自身定位为“Twitch × Polymarket × ChatGPT”的结合体，致力于开创一种新型互动信息市场。它融合了创作者主导的直播内容、AI辅助的市场创建与结果判定，以及实时社区参与机制。
NewGen认为，此次增资使其得以早期布局一个独特的平台。该平台正处于人工智能、直播以及预测市场基础设施的交汇点。
此笔交易体现了NewGen对AI驱动信息市场长期潜力的坚定信念，以及K25.ai在亚太地区构建定义性平台的能力，尽管其发展仍需考量执行情况、市场条件和相关监管要求。
除了股权投资，NewGen还拥有K25.ai在亚太地区的独家代理权，这为其在许可市场中支持区域合作与分销开辟了额外的商业路径。
预计的10%股权与代理关系相结合，将使NewGen在K25.ai的区域增长中获得重要的经济参与权和强大的战略协同效应。NewGen创始人、董事长兼首席执行官Alfred Siu表示：“完成对K25.ai总计1000万美元的承诺投资并获得10%的预计股权，同时保持1亿美元的Pre-A轮估值，对NewGen而言是一个极具吸引力的战略里程碑。”“我们相信，K25.ai正在人工智能、直播和信息市场的交汇点，构建一个高度差异化的平台。”他补充道，“结合我们在亚太地区的独家代理权，此次增资使得NewGen能够深度参与K25.ai的产品发布、商业化及其下一阶段的区域增长。”K25.ai创始人兼首席执行官Andy Cheung评论道，公司以1亿美元估值成功完成1000万美元Pre-A轮融资，这标志着K25.ai的一个决定性里程碑，并为其构建亚太区下一代AI驱动信息市场的愿景提供了强有力的战略验证。“在NewGen的支持下，我们的Pre-A轮融资现已圆满结束，并已正式启动A轮融资。我们正在以显著提升的估值进行A轮融资，这体现了K25.ai自Pre-A轮以来所展现出的强劲发展势头和投资者的高度关注。”他补充说，“新一轮融资旨在加速产品发布，推进在特定亚洲市场的相关监管许可，深化我们的创作者和战略合作伙伴生态系统，并助力K25.ai迈向其下一阶段的区域增长。”
