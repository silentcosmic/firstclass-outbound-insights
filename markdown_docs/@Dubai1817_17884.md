# 国际刑警破获巨额诈骗：冻结2.93亿美元，逮捕5800人

🚨 国际刑警破获巨额诈骗：冻结2.93亿美元，逮捕5800人
国际刑警组织（INTERPOL）近日公布了“2026年曙光行动”的显著成果。此次全球联合执法行动覆盖97个国家和地区，旨在打击商业电子邮件诈骗（BEC）、投资诈骗、色情敲诈、恋爱诈骗等社会工程类犯罪及其关联的跨国洗钱网络。行动共逮捕超过5800名犯罪嫌疑人，并成功冻结涉案资金高达2.93亿美元。
据国际刑警组织披露，本次行动共识别出全球逾14.2万名受害者，分析调查了152,808起案件，最终侦破23,715起，确认犯罪嫌疑人15,606名，并冻结了31,014个涉案银行账户。此外，行动期间还发布了99份国际通缉令及协查通报。各国执法机构充分利用国际刑警组织的全球快速止付机制（I-GRIP），对涉案法定货币和虚拟资产实施了快速冻结，有效阻止资金转移，为个人和企业挽回了大量经济损失。
国际刑警组织强调，随着网络诈骗犯罪日益呈现跨境化、组织化趋势，国际执法合作的重要性愈发凸显。未来，该组织将持续加强成员国之间的信息共享、资金追踪和联合执法，以维护全球金融安全和网络空间秩序。
