# WhatsApp阿联酋上线用户名：逐步开放预订

📣 WhatsApp阿联酋上线用户名：逐步开放预订
Meta旗下即时通讯应用WhatsApp正在阿联酋逐步推出其新的用户名预订功能。这项更新允许用户为其账户设置一个独特的用户名，简化了联系方式的分享和管理。
目前，该功能正通过服务器端逐步推送，因此并非所有阿联酋用户都能立即看到。用户若已收到该功能，只需将WhatsApp更新至最新版本，即可在“设置”菜单的“账号”选项下进行预订。官方表示，该预订系统将分阶段开放，用户在其所在区域功能上线时会收到应用内通知。
