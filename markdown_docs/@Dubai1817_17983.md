# ️ 新山金海湾警示：华人商家疑举报同胞

⚠️ 新山金海湾警示：华人商家疑举报同胞
近日，市场消息指出，马来西亚新山金海湾区域出现一起针对中国籍人士的警示事件。据称，有中国背景的商家在当地经营民宿租赁和餐饮服务时，涉嫌利用客户信息举报同胞。爆料者称，其团队此前曾因此遭遇执法行动，11人被拘留近三周，并产生巨额费用。
消息来源进一步透露，经内部渠道了解，此次事件的举报方疑为当地一对经营“爱加倍”民宿租赁及“大世界”外卖餐饮服务的华人夫妇。据称，该商家涉嫌通过其业务渠道，掌握中国客户的住所、居住人数等详细信息，并以此为基础进行举报。
多位受害者反映，其遭遇与上述商家存在关联。这一情况引发了在当地中国社群的广泛关注和警惕，提醒在该区域的中国籍人士在租房、外卖等服务中务必审慎，注意个人信息保护及潜在的商业风险。
