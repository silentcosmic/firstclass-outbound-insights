# 吉隆坡捣毁电诈窝点：9名中国籍嫌犯被捕

🚨 吉隆坡捣毁电诈窝点：9名中国籍嫌犯被捕
吉隆坡警方于6月25日上午，在旺沙玛珠一处高档公寓成功捣毁一个网络外汇投资诈骗窝点，当场逮捕9名中国籍男子及1名马来西亚籍男子。
吉隆坡警察总长拿督法迪尔·马苏斯证实，警方在当日上午11时突袭公寓17楼，查获大量作案工具。初步调查显示，该团伙自今年5月初起活跃，专门针对中国民众进行虚假外汇投资诈骗，而涉案的马来西亚籍人员则主要负责为团伙提供后勤支持。
目前，警方正深入调查涉案公寓业主的身份，并追查协助外籍人员入境的中介以及该团伙的幕后组织者。
