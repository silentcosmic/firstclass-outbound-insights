# 大华银行斥资4.5亿美元在胡志明市兴建越南总部，深耕东南亚核心市场

大华银行（UOB）越南公司已在胡志明市启动其价值4.5亿美元新总部的建设工程。
这一开发项目使大华银行成为首家在越南国际金融中心倡议框架内，开发专门建造总部的外资银行。
这座36层高的办公大楼预计于2030年竣工，届时将成为该银行在越南的旗舰所在地。
凭借此项投资，大华银行将在其新加坡、马来西亚、印度尼西亚、泰国和越南这五个核心东南亚市场中，全面建立其总部业务版图。
银行将此项目定位为支持客户跨境贸易雄心发展的关键举措。该项目旨在加强与区域资本及投资流动的互联互通。
大华银行副董事长兼首席执行官黄一宗表示：“我们的新总部反映了我们对越南未来的信心，以及我们对胡志明市成为国际金融中心这一雄心壮志的支持。”
该设施被命名为“大华银行胡志明市大厦”，坐落于西贡河畔的巴逊（Ba Son）区域。
建筑事务所Gensler负责该大楼的设计，旨在达到国际甲级写字楼标准，并争取获得绿色建筑认证。
银行计划在该新兴金融区设立分支机构，以扩展其服务。大华银行越南公司将继续与政府机构和行业利益相关者合作，共同推动该金融中心的建设与发展。
此次实体扩张是大华银行服务其遍布东南亚850万客户的更广泛战略的重要支撑。
