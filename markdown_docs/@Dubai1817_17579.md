# 缅甸妙瓦底打击电诈：77栋违建已拆65栋

🔥 缅甸妙瓦底打击电诈：77栋违建已拆65栋
缅甸政府正持续推进对妙瓦底水沟谷地区违法建筑的整治行动。最新进展显示，6月27日，当地一栋7层违法建筑通过定向爆破方式被成功拆除，这是本轮集中拆除行动中的一个关键项目。
据了解，此次整治行动共将77栋违法建筑列入拆除范围，其中57栋采用机械拆除，20栋通过爆破实施。截至目前，联合工作组已累计拆除65栋违法建筑，表明大部分拆除任务已完成，剩余12栋建筑将按既定方案陆续清理。
缅甸有关部门强调，拆除行动将持续推进，直至全部违法建筑清理完毕。官方指出，此举旨在进一步压缩网络诈骗、网络赌博等跨境违法犯罪活动的生存空间，持续加强对妙瓦底地区的综合整治力度，以净化区域环境。
