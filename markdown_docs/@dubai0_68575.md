# 美国重拳出击：查获柬埔寨诈骗网络12.7万枚比特币

🚨 美国重拳出击：查获柬埔寨诈骗网络12.7万枚比特币
美国财政部与司法部近日联手，对柬埔寨王子集团（Prince Group）及其创始人陈志（Chen Zhi）发起了迄今为止针对东南亚跨国网络犯罪集团规模最大的一次联合打击。此轮行动不仅扩大了制裁范围，更创纪录地没收了127,271枚比特币，按诉讼时价值约150亿美元，成为美国司法部史上最大的加密货币没收案。
据美国执法机构指控，陈志领导的王子集团以房地产、银行、酒店及航空等合法业务为掩护，长期运营涉及网络投资诈骗（俗称“杀猪盘”）、洗钱及人口贩运的跨国犯罪网络。该集团利用虚假加密货币投资平台诱骗受害者投资，并通过复杂的资金流转渠道进行洗钱。据统计，仅2024年，美国民众因此类诈骗造成的损失已超100亿美元，同比激增66%。
此次制裁行动共涉及146名个人和实体，包括陈志、胡晓伟、邱伟仁等关键成员。其中，胡晓伟被认为是陈志的核心助手和“二号人物”，已于2026年6月在日本被捕，目前新加坡、日本及美国正对其资产和商业网络展开调查。邱伟仁则被指为诈骗项目的重要投资人，已离开新加坡。调查还发现，该犯罪网络通过遍布柬埔寨、香港、新加坡、英属维尔京群岛等多个司法辖区的公司架构，借助家族办公室、离岸公司及加密货币进行资金转移。柬埔寨汇旺集团（Huione Group）亦被指为重要洗钱平台。
除了金融诈骗，该网络还被控涉及强迫劳动和人口贩运，大量人员被以高薪招聘名义诱骗至东南亚诈骗园区后，遭到限制人身自由并被迫从事非法活动。截至目前，美国、新加坡、日本、中国、英国及澳大利亚等多国执法机构已参与调查，新加坡方面已累计冻结或查封超6亿新元的相关资产。王子集团案件不仅揭示了现代网络犯罪组织如何利用合法企业、离岸金融体系和国际资本网络进行全球化运作，也预示着各国联合打击力度将持续升级，未来可能有更多相关人员和企业被纳入调查及制裁范围。
