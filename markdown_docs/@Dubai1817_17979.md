# ️ 吉隆坡武吉免登突发移民检查：提醒出海人士注意

⚠️ 吉隆坡武吉免登突发移民检查：提醒出海人士注意
据市场消息，马来西亚吉隆坡核心商圈武吉免登（Bukit Bintang）时代广场附近，近期有移民局执法行动的报告。现场疑似展开突击检查，引发当地居民关注。
有当地观察人士提醒，鉴于该区域人流量大，尤其对于在马出海人士或近期计划前往该区域的商务旅客，务必加强警惕，确保随身携带有效身份证件及护照等合规资料，以避免不必要的延误或麻烦。
