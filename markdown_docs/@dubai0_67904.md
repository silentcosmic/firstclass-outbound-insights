# 妙瓦底打诈升级：七层大楼遭爆破拆除

🚨 妙瓦底打诈升级：七层大楼遭爆破拆除
缅甸妙瓦底水沟谷地区打击网络诈骗与赌博的行动持续升级。当地消息称，一栋七层非法建筑已于今日被成功爆破拆除，这是清理非法建筑行动的一部分。
据统计，在规划拆除的77栋建筑中，目前已有62栋通过机械或爆破方式完成拆除，剩余15栋的清理工作正在进行中。官方强调，打击网络诈骗已上升至国家任务层面，并正联合邻国开展深度治理。
