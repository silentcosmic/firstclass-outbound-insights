# 博睿集团陷劳资纠纷：数十名技术员工工资被拖欠

🚨 博睿集团陷劳资纠纷：数十名技术员工工资被拖欠
市场消息指出，斯里兰卡博睿集团（Borei Group）近日面临严峻的劳资纠纷，数十名技术岗位员工反映工资被长期拖欠。
据员工披露，公司在发放工资条时即出现异常，原定薪资被减半。随后，在约定发薪日，工资迟迟未能到账。博睿集团方面则以“技术负责人冒领”为由，要求所有员工必须进行视频或语音验证，并手持身份证件方可领取工资，此举引发员工普遍不满与质疑。然而，即便有员工按要求完成验证，仍未能成功领到薪资。
员工进一步指出，公司在入职时已收集了员工的加密货币（U地址）信息，并在发放工资条时进行了地址确认，因此对于“冒领”的解释，员工表示难以接受。市场传闻还提及，博睿集团的实际控制人疑为某体育博彩平台（Bty）的股东之一，且有员工抱怨其在项目合作中存在不当行为。
