# 阿联酋迪拉姆稳定币获批：加速融入日常生活

📣 阿联酋迪拉姆稳定币获批：加速融入日常生活
阿联酋的数字支付生态迎来新进展。由阿布扎比国际控股公司（IHC）、阿布扎比第一银行和Sirius国际控股公司联合推出的迪拉姆稳定币DDSC，现已获得虚拟资产监管局（Vara）批准，可在受监管交易平台上架。此举标志着阿联酋在推动数字资产更广泛地走进日常生活方面迈出关键一步，旨在提供便捷的日常支付选择。
DDSC与阿联酋迪拉姆按1比1挂钩，并在ADI Chain上完成结算。普通用户今后可在Vara监管平台便捷访问、购买和兑换DDSC，用于购物、商务付款及朋友间转账等日常场景，所有交易均以迪拉姆计价，在区块链上直接完成。相较传统支付，DDSC的一大优势在于不受银行营业时间限制，且无中间环节延迟，只要有网络即可随时进行资产查看、存储与转移。
该稳定币的推出取得了阿联酋央行的无异议证明，凸显了监管机构对数字资产创新的支持。国际货币基金组织（IMF）曾指出，稳定币在提升支付速度、降低成本，尤其在跨境支付和汇款方面具有显著潜力，同时有助于拓宽金融服务覆盖面。DDSC的落地，正是阿联酋扩大稳定币应用、构建更高效数字金融体系的重要尝试。
