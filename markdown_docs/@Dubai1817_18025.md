# 乌干达：367名中国公民涉非法工作将面临起诉及遣返

🚨 乌干达：367名中国公民涉非法工作将面临起诉及遣返
据乌干达《New Vision》7月12日报道，此前在恩德培联合执法行动中被拘留的367名中国公民，将因涉嫌持旅游签证非法工作而面临刑事指控。乌干达内政部已确认，在司法程序结束后，将启动对这些公民的遣返程序。
乌干达政府重申将持续加强移民执法，以维护国家安全。当局同时呼吁酒店、度假村及房产业主提高警惕，强化对长期入住外国人的身份核查，并及时向警方报告任何异常情况。此案目前仍在进一步调查中。
