# 柬中深化执法合作，联手打击跨境犯罪

🤝 柬中深化执法合作，联手打击跨境犯罪
7月15日，柬埔寨国家警察总署副总监迪伟杰与中国云南省公安厅副厅长张庆星率领的代表团举行工作会谈，旨在加强两国在打击跨国犯罪方面的协作。
双方围绕网络赌博、电信网络诈骗等跨境违法犯罪活动深入交换意见，并就深化执法协作、信息共享及案件协查等议题达成广泛共识。此次会谈强调了双方将持续加强执法合作，共同应对跨境犯罪挑战，以维护两国及区域的安全稳定。
