# 大疆EV50珠峰创纪录：高空物流新突破

⚡ 大疆EV50珠峰创纪录：高空物流新突破
大疆近日展示了其在极端环境无人机技术上的突破性进展。其尚未正式发布的EV50垂直起降运载无人机，在“巅峰使命”珠峰科考任务中，成功飞越珠峰北坡8861米高空，刷新了全球同类无人机公开测试的最高飞行升限记录。
此举不仅获取了8000米以上海拔的珍贵大气剖面数据，更验证了EV50卓越的性能与可靠性。这款复合翼无人机具备原地垂直起降能力，并在起飞后切换至固定翼巡航模式，极大提升了作业效率。在为期12天的科考任务中，EV50累计完成了32架次起降，连续爬升3730米，返程时仍保有30%电量，展现出强大的续航与适应能力。
大疆透露，EV50的研发目标还包括实现百公里级货物运输等低空物流场景，预示着未来物流领域或将迎来一场技术革新。
