# 世界杯竞猜爆冷：广西彩民50元独中182万！

🔥 世界杯竞猜爆冷：广西彩民50元独中182万！
随着2026年世界杯预热赛事持续升温，足球场上的激烈对决不仅吸引了全球目光，也催生了场外竞猜的热潮。近日，在广西桂林，一位幸运的购彩者凭借精准判断，在竞彩足球玩法中斩获巨额奖金。
据了解，该购彩者仅以50元的投入，通过“4串1”玩法成功预测了四场世界杯比赛的精确比分，最终单票独中182万余元。
