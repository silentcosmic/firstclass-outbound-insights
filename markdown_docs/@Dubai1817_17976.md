# ️ 西港夜场负责人被曝诈骗熟人：人品与诚信引关注

⚠️ 西港夜场负责人被曝诈骗熟人：人品与诚信引关注
近日，柬埔寨西哈努克港（西港）一则市场爆料引发关注。一位匿名人士指控当地DV嗨场负责人“老徐”利用熟人关系，在借款2万美元后挪用资金至赌场并拒不归还，导致其商业信誉和个人诚信受到严重质疑。
据爆料者陈述，其初到西港时通过朋友介绍认识了老徐，并出于对熟人的信任，不仅在其经营的场子消费累计达6万美元，更在老徐急需资金周转时借出2万美元，并约定由老徐协助安排住宿。然而，借款后住宿承诺未能兑现，资金被指用于赌博并输光。随后，老徐不仅拒绝还款，甚至试图将其侄子欠款作为抵消，并仅象征性地承认5000美元欠款，且迟迟不予支付。
爆料者进一步指出，老徐一方面声称资金困难，另一方面却被发现购买名贵手表，其言行不一引发强烈不满。此事件不仅暴露了海外商业环境中可能存在的熟人骗局风险，也再次提醒出海企业及个人，在进行商业合作或资金往来时，务必审慎评估合作伙伴的诚信与风控能力，避免因轻信而遭受损失。
