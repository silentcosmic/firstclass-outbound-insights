# 马来西亚端掉假贷款窝点：18名中国人涉案被捕

🚨 马来西亚端掉假贷款窝点：18名中国人涉案被捕
据最新消息，马来西亚森美兰州警方近日在瓜拉庇劳成功捣毁一个专门从事虚假贷款诈骗的犯罪窝点。该团伙主要针对中国籍受害者，以虚假贷款方案诱骗受害者上钩。此次行动共逮捕19名嫌疑人，其中包括18名中国籍人员和1名马来西亚本地男子。
警方的突破口始于6月29日，在双溪杜亚拦截一辆汽车，并控制了车内的一名本地男子和一名中国男子。经检查二人手机，发现其中包含与网络诈骗相关的程序。警方循线追踪，最终锁定并突袭了一处地点，该地点被证实为诈骗团伙的运营窝点。现场另有17名中国籍人员，警方查获了包括笔记本电脑、打印机、手机及大量诈骗话术脚本在内的作案工具。
警方初步研判，该团伙通过虚假贷款模式实施诈骗，受害者多为中国籍人士。目前，19名嫌疑人已被警方拘留并展开深入调查，案件正在进一步处理中。
