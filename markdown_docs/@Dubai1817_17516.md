# 马来西亚捣毁中国电诈窝点：9名中国籍嫌疑人落网

🚨 马来西亚捣毁中国电诈窝点：9名中国籍嫌疑人落网
马来西亚警方持续加强对电信网络诈骗的打击力度。6月25日上午，吉隆坡警方根据公众举报及前期情报，在旺沙玛珠一处豪华公寓成功捣毁一个网络外汇投资诈骗呼叫中心，现场逮捕了9名中国籍男子及1名马来西亚籍男子。
吉隆坡警察总长拿督法迪尔·马苏斯透露，警方于当日上午11时突袭了该公寓17楼的涉案单位，查获多台笔记本电脑、手机等作案设备。初步调查表明，该团伙主要针对中国受害者进行网络外汇投资诈骗，其犯罪活动自今年5月初已在该窝点展开。
警方进一步调查发现，被捕的马来西亚籍男子涉嫌为窝点成员提供日常餐食。目前，警方正深入追查涉案公寓业主的身份，并核实是否存在协助外国人非法入境马来西亚的中介或幕后组织。
