# 霍尔木兹海峡生变：伊阿联手欲征“通行费”

⚡ 霍尔木兹海峡生变：伊阿联手欲征“通行费”
近期，伊朗和阿曼两国正就霍尔木兹海峡的“海上服务费”展开合作，此举可能导致在该关键航道征收实际意义上的通行费。尽管美国方面多次强调，根据国际法，此类通行费的征收行为不被允许，两国仍推进相关计划，引发国际关注。
