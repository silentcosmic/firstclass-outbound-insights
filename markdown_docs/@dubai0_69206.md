# 沙迦酋长新书揭秘：17世纪马斯喀特战役复盘

🔍 沙迦酋长新书揭秘：17世纪马斯喀特战役复盘
沙迦酋长谢赫·苏尔坦·本·穆罕默德·阿勒卡西米博士的最新历史著作《马斯喀特之战》于周日由卡西米出版社正式发行。
该书深入探讨了1649年阿曼人从葡萄牙人手中收复马斯喀特的重大历史事件。酋长在书中不仅详细还原了战役经过，更独辟蹊径地利用葡萄牙方面遗留的珍贵文献和调查记录，剖析了马斯喀特失陷的深层原因，以及战败后追查责任的审判过程。这些第一手资料为理解17世纪中期葡属印度及阿拉伯湾地区在军事、政治和行政层面的变革，提供了不可多得的关键线索。
