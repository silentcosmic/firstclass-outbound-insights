# 阿联酋强烈谴责伊朗袭击，中东局势升级

🚨 阿联酋强烈谴责伊朗袭击，中东局势升级
7月12日星期日，阿联酋外交部发表声明，强烈谴责伊朗近期再次对巴林、科威特、卡塔尔、约旦和阿曼发动的导弹及无人机袭击。阿联酋指出，这些行动公然侵犯了兄弟国家的主权，并对其安全与稳定构成严重威胁。
阿联酋重申，将与上述受攻击国家完全团结一致，并支持它们为维护国家安全与稳定所采取的一切措施。
值得关注的是，当天早些时候，阿联酋防空系统也曾应对新一轮导弹和无人机威胁。随后，阿联酋国家紧急危机和灾害管理局证实，探测到的导弹威胁位于阿联酋境外，并向公众保证当前局势稳定。该机构表示，即使受影响的可能性较低，也会主动发布预防性警报。
