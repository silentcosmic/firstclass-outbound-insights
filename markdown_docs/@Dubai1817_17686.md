# 西港空地惊现AK-47，两名中国男子被捕

🚨 西港空地惊现AK-47，两名中国男子被捕
柬埔寨西哈努克省警方于6月29日接到举报，称在西哈努克市第四分区一处空地内发现一把AK-47步枪。警方迅速赶到现场，当场查获AK-47步枪1支、弹匣1个及子弹7发，并随即展开深入调查。
根据线索，警方先后拘留了5名相关人员，其中包括3名中国籍男子和2名柬埔寨籍女子。行动中，警方还查获了SUR-M9手枪1支、弹匣1个、吸毒工具、手机和摩托车等物品。
经进一步审讯与排查，警方最终确认其中2名中国籍男子涉嫌非法持有并弃置该AK-47步枪。其余3人目前暂未发现与此遗弃枪支案有直接关联。目前，涉案人员已被依法移交处理，案件仍在进一步侦办中。
