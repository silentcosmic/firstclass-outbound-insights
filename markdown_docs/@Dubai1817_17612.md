# ️ 泰国母亲东京贩卖亲女：跨国人伦惨案终审判决

⚖️ 泰国母亲东京贩卖亲女：跨国人伦惨案终审判决
泰国刑事法院近日对一起震惊社会的跨国人口贩卖案作出终审判决。30岁女子拉克萨娜因贩卖人口及协助卖淫罪，被判处7年6个月监禁。此案受害者竟是被告的亲生女儿，案发时年仅12岁。
据悉，2025年夏季，拉克萨娜以旅游为名将女儿带至日本东京，随后强迫其在一家按摩店从事性服务。泰国与日本警方联合调查显示，在短短一个月内，这名未成年少女被迫接待约60名顾客。所幸，受害女孩主动向当地救助部门求助，促使警方介入并将其成功解救。
庭审中，拉克萨娜对其罪行供认不讳，却荒谬辩称带女儿赴日仅为“照看年幼的儿子”。法院最终驳回其辩解，并依法判处重刑，以回应这起严重违背人伦的跨国犯罪。
