# 迪拜逾五万株凤凰木点亮夏日，王储指示扩大种植

🌳 迪拜逾五万株凤凰木点亮夏日，王储指示扩大种植
随着夏季来临，迪拜城市景观被逾五万棵凤凰木的火红橙花点缀，为城市增添了独特的色彩与盎然绿意。这些标志性的树木在气温升高之际进入花期，其宽阔树冠不仅能提供大片遮蔽，更能有效将地面温度降低约5°C，为市民带来丝丝凉意。
据悉，迪拜王储谢赫哈姆丹·本·穆罕默德此前曾指示，要进一步扩大凤凰木在街道、社区及公共空间的种植范围，并鼓励居民参与城市绿化，为此将向希望种植的居民免费发放树苗。凤凰木寿命可达60年，最高可生长至12米，是迪拜城市可持续发展和美化策略的重要组成部分。
