# 柬埔寨电诈新变种：埃尔法车内设流动诈骗站

🚨 柬埔寨电诈新变种：埃尔法车内设流动诈骗站
柬埔寨持续打击电信诈骗，促使诈骗团伙改变传统园区集中模式，转而采取更隐蔽、流动的作案方式。这种新趋势包括利用餐厅、仓库，乃至改装车辆进行诈骗。
6月19日晚，柬埔寨警方在波贝地区查获一辆金边牌照的丰田埃尔法商务车，现场逮捕了6名中国籍嫌疑人。警方在车内搜获了手机、WiFi路由器等用于实施诈骗的工具。据调查，这些嫌疑人将商务车改装成“流动电诈站”，在车辆行驶过程中同步开展诈骗活动。
次日，嫌疑人被带回现场，模拟演示了其利用车辆移动作案的全过程。此案揭示了电信诈骗在面临区域性严打时，其组织形式和技术手段正加速演变，呈现出更高的隐蔽性和适应性，对跨境执法带来新的挑战。
