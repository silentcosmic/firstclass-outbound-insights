# 手机电磁信号泄露隐私：应用识别率高达99%

🔍 手机电磁信号泄露隐私：应用识别率高达99%
一项最新研究揭示，手机在运行过程中会泄漏低频电磁信号，这些信号竟能被精准捕捉并用于识别设备正在使用的应用程序及部分操作。
这项非接触式取证技术无需访问手机系统或存储数据，甚至在设备离线、飞行模式、加密或锁定状态下也能有效工作。研究团队在多款主流机型（包括iPhone 15 Pro、小米 15 Pro、OPPO Reno 13）上进行了测试，针对抖音、微信视频通话、百度地图、短信、浏览器、相机和云存储等常见应用的识别准确率最高达到惊人的99.07%。这意味着用户在不知情的情况下，手机使用习惯乃至部分操作可能面临被外部监测的风险，对个人数据隐私和信息安全构成了新的挑战。
