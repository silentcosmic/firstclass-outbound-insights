# 东南亚区域前沿：越南重金投建AI数据中心，马来西亚擘画数字经济蓝图

过去一周，东南亚科技市场涌现出一系列值得深度剖析的新动向。本期情报聚焦（2026.06.29-2026.07.05）的企业发展、产业脉络以及投融资趋势，为决策者提供独家洞察。
在初创公司领域，本期深度关注了新加坡的自主机器人系统开发商、领先的医疗机器人企业，以及日益壮大的商户服务平台。此外，泰国的AI机器人公司也进入视野，这些创新力量的融资规模介于约613万美元至1.25亿美元之间，展现出区域市场的活力。
区域基础设施建设提速
越南G-Group斥资3亿美元打造AI数据中心： 据当地消息，越南科技集团G-Group已获批投资建设大型AI数据中心园区G-Campus Hoa Lac。该园区选址河内，占地逾3.8万平方米，将涵盖研发、测试、芯片设计以及数据中心等核心功能区。初期IT电力容量规划为20MW，未来可扩容至30MW，旨在提供先进的云计算、人工智能和高性能计算服务，成为越南数字基础设施的关键支柱。
印度至新马海底光缆项目预计2029年竣工： AI网络连接平台Lightstorm、科技巨头微软、新加坡电信以及塔塔通信近期宣布启动I-2SEA海底光缆项目。该项目旨在铺设一条连接印度、马来西亚和新加坡的全新通信干线，总长度约3600公里，预计于2029年第四季度投入使用。建成后，该光缆将主要支撑云计算、GPU算力基础设施，以及日益增长的AI训练与推理工作负载，对区域数据互联互通意义重大。日本电气公司被指定为系统供应商。
核心市场动态与政策导向
蚂蚁国际吉隆坡全球发展中心揭幕： 7月1日，蚂蚁国际在吉隆坡国际金融中心TRX设立的全球发展中心正式启用。该中心计划提供1500个金融科技职位，其中技术岗位占比超过一半。值得关注的是，技术团队中约半数成员毕业于马来西亚本地高校，凸显了对当地人才的重视。蚂蚁国际透露，截至2026年上半年，马来西亚已有超过60万家商户通过Alipay+的合作伙伴接受跨境支付，同比增长逾70%，显示出区域数字支付的强劲增长。
马来西亚半导体孵化计划首批入选十家企业： 马来西亚科技发展公司近日启动“SemiconStart Malaysia”半导体孵化计划，旨在扶持本土在集成电路设计、先进材料、光子技术、传感器、先进封装及半导体自动化等前沿领域的初创企业。首批共十家公司获得入选资格，每家企业最高可获得100万林吉特的资金支持，并参与为期260天的技术验证、融资辅导和商业化指导，以加速其创新成果转化。
马来西亚发布数字行动计划，加速AI国家战略： 马来西亚政府近期公布《2030年数字行动计划》（Malaysia Digital Action Plan 2030），覆盖2026年至2030年。根据该宏伟蓝图，到2030年，数字经济对国家GDP的贡献率将提升至30%，同时创造50万个高价值数字岗位，并通过数字化转型节省45亿林吉特的公共部门开支，并实现95%的公共服务端到端线上化。此计划还深度涉及数据中心、云计算、智能城市建设、人才培养以及AI治理等关键方向，展现出全面拥抱数字化的决心。
新加坡追加25亿新元重磅支持深科技与本地上市： 新加坡近期发布的经济战略评估报告，提出了八项关键发展方向，并规划在未来三年内，支持1万家中小企业 внедрение AI技术，助力10万名劳动者掌握AI技能。针对成长阶段企业融资不足的痛点，新加坡将向“Startup SG Equity”深科技企业股权投资计划追加10亿新元，并设立第二期规模达15亿新元的本地上市基石投资基金“Anchor Fund”，以增强对本土创新企业的支持力度。
新加坡推出金融AI智能体实时安全框架： 鉴于金融机构部署AI智能体可能带来的操作风险，新加坡金融管理局联合多家金融机构及金融科技企业，发布了“金融AI智能体实时安全框架”（SAFR）。该框架要求在AI智能体执行任务前设置治理检查点，对相关操作进行严格验证和记录，并支持实时验证、审计及权限约束。目前，该框架已在支付、财资管理、财富管理、合规审查和客户服务等多个场景进行测试，旨在确保AI在金融领域的安全稳健应用。
Wesco以1.36亿美元收购新加坡数据中心冷却服务商： 据悉，美国电气和通信产品分销巨头Wesco International近期完成了对新加坡数据中心冷却服务商Newark Engineering Group的收购，交易金额达1.75亿新元，约合1.36亿美元。Newark在马来西亚和印度尼西亚均设有分支机构，专注于提供数据中心冷却和热管理服务，2025年收入为6000万美元。Wesco期望通过此次收购，进一步扩大其在东南亚数据中心冷却市场的战略布局。
区域融资与人才趋势
东南亚6月融资总额飙升至38.46亿美元： 数据平台Tracxn与TNGlobal的联合统计显示，东南亚地区在6月共完成18笔融资交易，融资总额达到38.46亿美元。这一数字相较5月增长了约17倍，同比激增1469.8%。后期融资占据总额的86.3%，其中数据中心平台DayOne和云数据库平台Supabase的两笔大额交易，合计贡献了约78%的资金。新加坡企业表现尤为突出，获得约37.8亿美元，占据区域融资总额的近98%。
73%东南亚员工认为管理层数字认知不足： 办公协作平台Lark发布的报告，对新加坡、马来西亚、印度尼西亚、泰国、菲律宾和越南的900名雇主及超过5000名员工进行了深入调查。结果显示，55%的员工每周因数字协作效率低下而损失至少三个小时。尽管62%的受访者担忧AI最终将取代其岗位，但仍有高达88%的员工渴望AI能够承担重复性工作，反映出对技术应用既有戒备又抱有期待的复杂心态。
菲律宾仅17%企业具备充足AI人才： 专业服务机构怡安（Aon）发布的《2026年人力资本趋势研究》揭示，72%的菲律宾机构已经部署或正在试点人工智能技术。然而，仅有17%的机构表示能够招聘并留住足够的AI人才，凸显出该领域严重的人才供需失衡。94%的受访机构预计AI将创造新的工作岗位并改变现有的技能需求。此项研究共调查了2361名企业和人力资源负责人，其中504人来自亚太地区，提供了区域性的人力资本洞察。
聚焦创新初创企业
dConstruct Technologies：自主机器人系统专家
dConstruct Technologies是一家扎根新加坡的机器人技术公司，专注于开发面向复杂及无GPS环境的自主导航、三维扫描和现实捕捉系统。其技术广泛应用于建筑、巡检、物流和娱乐等多个行业，为客户提供高效智能的解决方案。
公司的自主机器人系统d.ASH已被工程建设公司Boustead Projects、公共交通运营商SBS Transit以及软银机器人新加坡公司等行业领先客户采用。dConstruct还规划在新加坡榜鹅数字园区建设一座占地4.2万平方米的全球总部，预计于2026年底竣工，以支撑其未来增长与研发需求。
Biobot Surgical：医疗机器人前沿
Biobot Surgical是一家总部位于新加坡的医疗科技公司，致力于开发用于前列腺癌诊断和治疗的机器人辅助手术系统。其核心产品——前列腺介入机器人平台Mona Lisa，创新性地结合了机器人针头引导技术与磁共振超声图像融合技术，显著提升了手术的精准性与安全性。
在融资方面，Biobot Surgical近日成功获得2000万新元的资金，约合1540万美元。本轮融资由生命科学投资机构ClavystBio领投，ZIG Ventures亦有参与。这笔资金将主要用于扩大Mona Lisa平台在全球范围内的应用，推进其在美国市场的准入计划，并补充临床和商业化所需的关键资源。
Amity Robotics：AI接待机器人创新者
Amity Robotics是一家来自泰国的AI机器人公司，专注于开发面向商场、酒店和大型商业设施的AI接待与引导机器人，并运营着一套自主研发的机器人操作系统，为服务业提供智能化升级方案。
公司的固定式AI接待设备ARC Base已成功进驻新加坡、阿联酋、沙特阿拉伯、中国香港、泰国、马来西亚和印度尼西亚的30多个商业项目。其客户不乏业内知名品牌，包括商业地产运营商Pavilion KL、Siam Piwat，以及洲际、雅高和香格里拉等国际酒店集团。
融资方面，Amity Robotics近日完成700万美元的种子轮融资，该轮资金包含股权和债务两部分。股权部分由东南亚风险投资机构East Ventures领投，500 Global参投；债务部分则由AlteriQ Global提供。此轮资金将用于扩大运营规模、加强系统集成能力，并加速移动机器人ARC Move的商业化进程。
Qashier：商户运营与支付平台
Qashier是一家总部设在新加坡的综合性商户运营及支付平台，为餐饮、美容和零售等行业提供全方位的服务，包括支付处理、订单管理、库存控制、客户关系管理以及嵌入式金融服务，旨在提升中小企业的运营效率。
该平台目前已服务新加坡、马来西亚、泰国和菲律宾超过2万家商户，年化支付交易额达到10亿美元，年化经常性收入同比增长61%，展现出强劲的市场扩张势头。自2025年6月推出商户贷款业务以来，Qashier已向超过100家中小企业发放了逾1000万美元的贷款，有效缓解了中小企业的资金压力。
在融资方面，Qashier近日完成约613万美元的A+轮融资，本轮资金同样由股权和债务组成。该轮融资由新加坡风险投资机构Cocoon Capital领投，IFP Securities和BlackSoil Global等机构参与。所募资金将主要用于支持公司的区域扩张战略和产品研发，以巩固其在东南亚市场的领导地位。
