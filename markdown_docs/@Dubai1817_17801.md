# 警惕：东南亚华人互助酿假表抵押诈骗

🚨 警惕：东南亚华人互助酿假表抵押诈骗
近日，有市场消息指出，在东南亚地区，部分华人商家因过度信任同胞而遭遇诈骗。一名当地餐饮业者便披露了其受骗经历，旨在提醒广大出海华人提高警惕。
据该业者透露，此前曾对一名同胞伸出援手，甚至为其无护照的朋友提供了临时照应。此人后续以资金周转困难为由，向业者借款，并提出以一块手表作为抵押，承诺支付利息。然而，当该表最终被送往专业表店鉴定后，却被证实为赝品，价值微乎其微。
截至目前，借款分文未还，而作为抵押的假表也毫无价值。该业者表示，其手中掌握有诈骗者的护照信息，并决定公开此事，以警示其他在东南亚经商的华人同胞，在提供帮助时务必审慎，避免因善意而蒙受不必要的经济损失。
