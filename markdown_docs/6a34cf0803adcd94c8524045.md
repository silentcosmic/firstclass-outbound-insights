# “中国基因”如何重塑东南亚：多业态连锁巨头的本土化与规模化路径

近期，一场聚焦东南亚市场深度洞察的行业盛会，深入探讨了在区域内实现实体业务快速扩张的“行动指南”。这一策略已成功将诸多出海企业锻造成具有主导地位的本土力量。
在题为“‘中国基因’到‘本土独角兽’：东南亚多业态连锁规模化增长的深层逻辑”的专题讨论中，多个零售帝国的战略掌舵者齐聚一堂。这些快速增长的业务横跨咖啡、护肤、母婴护理及快餐等多个领域。
尽管这些领军者均秉承了源自中国的一流供应链效率与高强度分销模式，但其成功的核心在于，他们能够摆脱“外来者”标签，真正融入并以本土市场先行者的姿态运营。与会嘉宾们深入剖析了实体门店扩展至数千家背后的运营实况，以及“中国基因”如何被重塑编码，以最终赢得东南亚消费者的心智与钱包。
本场圆桌论坛由ATM Capital创始合伙人梁敏俊先生主持，参与嘉宾包括Makuku首席财务官白雪峰、Hebe Beauty首席财务官张俊辉、J.Chicken创始人曹睿以及Tomoro Coffee首席执行官杨璐璐。
Makuku：从市场空白到母婴领导者
Makuku专注于母婴护理产品，主营纸尿裤业务，并计划在今年新增婴幼儿配方奶粉。公司深耕东南亚市场四至五年，尤其在印度尼西亚市场占据了有利地位。我们洞察到东南亚及中东地区母婴护理行业的巨大机遇。
我们致力于将中国的经验、创新产品和技术引入东南亚。我们的核心客户群体是妈妈和宝宝。2021年，在审视东南亚市场，特别是印度尼西亚市场时，我们发现了一个显著的使用习惯差异。印度尼西亚部分消费者群体对母婴产品的使用方式与主流市场趋势存在明显区别。
以纸尿裤为例，我们注意到印尼的妈妈们当时仍在使用传统型纸尿裤，这类产品易导致婴儿红疹。而在中国，妈妈们早已普遍使用技术更先进、更“透气”的新型纸尿裤。
我们看到了提升当地妈妈和宝宝生活品质的商机，这预示着巨大的业务增长潜力。东南亚母婴护理市场价值高达180亿美元，而印度尼西亚庞大的人口基数与出生率，为我们提供了广阔的增长空间。
我们始终坚信好产品会为自己代言。因此，当我们将创新产品带入印尼市场，并结合我们的人才与能力优势，短短四年内，我们成为了当地唯一跻身市场前三名的中国品牌。
Hebe Beauty：供应链赋能美妆新势力
Hebe Beauty成立于2018年，专注于健康与美容产业，旗下拥有六个品牌，涵盖护肤品、彩妆等多个品类。
目前，Hebe Beauty已成功拓展至印度尼西亚、马来西亚、菲律宾、泰国和越南等市场，覆盖线上和线下双渠道。
回溯2018年，高端美妆市场主要由国际大牌主导。与此同时，本土品牌凭借其自身优势和差异化定位，在大众和细分市场也表现出色。
彼时，中国化妆品行业正处于蓬勃发展期，积累了丰富的优质供应链资源。我们的团队在运营线下渠道方面表现卓越，具备强大的执行能力。
我们看到了国际品牌与本土品牌之间的市场机会。我们在渠道拓展和运营上的能力，足以超越部分国际大牌，实现更优的市场渗透。同时，依托中国高效的供应链体系，我们能为消费者提供比本土品牌更优质的产品。
通过精准的市场定位，我们成功立足并持续发展。我们预判东南亚市场将继续保持高速增长，这得益于电子商务的迅猛发展、年轻一代日益增长的消费能力，以及内容电商的快速崛起。长期而言，东南亚将一直是我们的战略重心。
J.Chicken：快餐升级的本土化挑战
我从大学时期便投身炸鸡行业。在中国，我们的品牌是“叫了个炸鸡”。
除了中国市场，我们还在美国和日本设有分店。
2017年，我们开始关注东南亚市场，认为该地区有望成为中国以外最大的炸鸡市场。
在2022至2023年间，我们与ATM Capital合作，对印度尼西亚进行了考察。我们认为这是一个未来值得“all-in”的市场。
在过去两年里，我们在印度尼西亚已开设了近百家门店。基于在中国积累的经验，我们针对印尼市场进行了一系列调整。迄今为止，我们在印尼的扩张之旅进展顺利。
我们团队初次决定进军印尼市场时充满热情。印度尼西亚是一个穆斯林国家，鸡肉是当地居民主要的蛋白质来源。穆斯林不食用猪肉，而牛肉和羊肉价格相对昂贵。
在印尼市场考察时，我们发现各种售卖鸡肉的店铺随处可见，但其中大部分是路边摊。市场缺乏能提供整洁用餐环境的“升级版餐饮体验”，仅有麦当劳和肯德基等国际品牌提供此类服务。
因此，我们决定进入这一市场，为当地消费者提供一个升级的品牌体验，J.Chicken便由此诞生。
在过去两年中，我们也面临了一些挑战。我们希望能提供口感更佳的炸鸡、舒适的用餐环境以及更亲民的价格。然而，同时满足这三个条件给我们带来了巨大压力。现在回想，或许只需要比当地企业略胜一筹即可，因为同时实现这三点会大幅增加我们的运营成本。
Tomoro Coffee：咖啡连锁的蓝海掘金
Tomoro Coffee已在印度尼西亚、菲律宾和新加坡等国布局，门店数量超过800家。以规模而论，我们目前是印度尼西亚第二大咖啡连锁品牌。
2022年我们进入印度尼西亚市场时，当地的咖啡连锁品牌数量并不多。与此同时，消费者对咖啡饮品的专业性和创新性需求却在不断增长。
短短三年，我们已成为印尼第二大咖啡连锁品牌。我们的目标是成为东南亚地区的第一大咖啡连锁品牌。
我们进行了深入的市场调研，并敏锐地捕捉到了整个东南亚市场的空白：咖啡连锁店的渗透率不足5%。而另一方面，消费者对产品创新和品质的要求却越来越高。这种供需缺口为我们带来了巨大的发展机遇。
以印度尼西亚为例，人口接近2.8亿，年出生人口约500至600万，是一个平均年龄不足30岁的非常年轻化的人口结构。这意味着消费品和生活方式领域存在强劲的需求。
