# 菲律宾查获走私中国香烟，一中国公民被捕

🚨 菲律宾查获走私中国香烟，一中国公民被捕
6月24日，菲律宾萨兰加尼省马阿西姆镇警方在检查站拦截一辆皮卡，查获87条未经合法进口及缴税的中国香烟。据菲律宾海关初步估算，这批香烟市值约7.4万比索。
一名43岁的中国籍男子及其19岁的菲律宾籍司机当场被捕，两人均居住在桑托斯将军城。因无法提供合法持有的烟草产品文件，他们将面临非法持有、运输及分销未缴税烟草产品的指控。
当地警方表示，将继续强化检查站及重点路线的执法，严厉打击走私商品及逃避关税、消费税的行为，并正进一步调查涉案香烟的来源。
