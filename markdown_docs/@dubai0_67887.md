# 迪拜租房续约费：合同未载明，可拒付！

🔍 迪拜租房续约费：合同未载明，可拒付！
近期，有出海迪拜的居民反映，其房地产中介每年要求支付一笔“续约费”，但租赁合同中并未提及此项费用。这一情况引发了对迪拜租房市场合规性的关注。
根据迪拜相关法律规定，房东与租客之间的权利义务关系主要由租赁合同明确。除非合同中白纸黑字写明了中介续约费，否则租客并无支付此费用的法律义务。中介无权在合同之外擅自增加收费项目。
然而，需要特别留意的是，如果租客与中介此前曾签署过独立的“经纪协议”或“代理合同”，且其中明确约定了续约费用，则可能需要按照该协议履行支付义务。
