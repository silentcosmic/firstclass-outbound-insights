# ️ 沙迦货机坠毁残骸寻获：导航系统故障

⚠️ 沙迦货机坠毁残骸寻获：导航系统故障
一架从阿联酋沙迦飞往巴基斯坦卡拉奇的波音737-400货机，于周二晚间起飞后不久失联。机组人员曾报告导航系统出现故障。经过巴基斯坦当局长达12小时的搜寻，该失事货机的残骸已被成功寻获。
