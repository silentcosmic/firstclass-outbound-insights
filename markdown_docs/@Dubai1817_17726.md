# 泰柬边境冲突：非法入境者持刀袭警遭击毙

🚨 泰柬边境冲突：非法入境者持刀袭警遭击毙
7月5日凌晨，泰国尖竹汶府边境地区发生一起巡逻冲突事件。当地村卫队在例行巡逻中发现三名可疑男子，要求检查时对方拒不配合并试图逃离。
随后，执法人员与村卫队展开联合搜捕。追捕过程中，其中一名男子突然持刀冲向拦截人员，双方发生冲突。村卫队人员在制止过程中开枪，该男子中弹身亡，另有两人趁乱逃逸。
现场检查发现，死者随身未携带身份证明，其背包内有摩托车钥匙、疑似作案工具及吸食器具等物品。初步判断，这三人为非法入境人员，并涉嫌在当地从事摩托车盗窃等违法活动。
目前，死者遗体已送往医院进行法医检验，两名逃逸人员的追查工作仍在进行中。
