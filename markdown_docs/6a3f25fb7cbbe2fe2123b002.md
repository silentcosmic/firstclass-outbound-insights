# 三星战略注资1.1亿美元予GRAIL，加速亚洲癌症早筛市场布局

总部位于美国的医疗健康公司GRAIL近日完成一轮股权融资，其中三星旗下多家关联公司，包括三星物产（Samsung C&T Corporation），通过以每股70.05美元的价格购入GRAIL普通股，共计投资1.1亿美元。
GRAIL周五发布声明称，三星此次投资旨在支持其业务增长及全球市场拓展目标。
GRAIL首席财务官Aaron Freidin指出，这笔投资将有效强化公司的资产负债表，并延长其现金流储备。此举旨在推进各项重点工作，包括在美国市场为其Galleri多癌早期检测产品争取监管批准与医保报销，并加速其在国际市场的普及。
此前已公布，GRAIL与三星物产将携手合作，在韩国市场推广Galleri检测服务，并可能在获得监管批准后，将业务扩展至日本、新加坡等其他亚洲市场。初期检测工作仍将在GRAIL位于美国北卡罗来纳州三角研究园的临床实验室进行。
三星物产生命科学业务执行副总裁Jaywoo Kim表示，此次投资及战略合作是三星物产推进其“扩大早期癌症检测可及性”愿景的一个重要里程碑。他补充道，这一伙伴关系将使两家公司更接近其共同目标——通过更早的发现来革新癌症护理。
