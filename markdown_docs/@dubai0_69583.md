# 警惕！缅北诈骗势力南下，泰缅边境现新动向

🚨 警惕！缅北诈骗势力南下，泰缅边境现新动向
近期，缅甸掸邦南部勐奈镇出现大量华裔面孔，频繁采购建材及食品等大宗物资，且不议价。这一异常现象正引发当地居民的高度警惕，担忧此前活跃于缅北的诈骗集团（俗称“KK园区”）正将其势力向南迁移。
据市场消息，这些物资装车后并非运往中国方向，而是转向泰缅边境的SSS组织控制区域。此前，“1027行动”后，北部诈骗集团确有大量南迁至掸邦南部及泰缅、老缅边境的迹象。缅甸军方与佤联军近期已在其辖区内逮捕并移交了数百名涉嫌网络诈骗的中国公民，进一步证实了此类犯罪活动的跨区域流动性。
当地民众呼吁武装力量及时清查，以避免该地区重蹈缅北覆辙，形成新的犯罪温床，对区域安全与营商环境带来长期负面影响。
