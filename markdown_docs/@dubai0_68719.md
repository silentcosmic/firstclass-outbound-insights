# 柬埔寨遣返10名越南籍嫌犯：涉网诈贩毒等重罪

🚨 柬埔寨遣返10名越南籍嫌犯：涉网诈贩毒等重罪
柬埔寨内政部移民总局近日通报，10名越南籍男女因涉嫌网络诈骗等多项严重罪行，已被依法驱逐出境。
据悉，这批包括4名女性在内的越南籍人员已于6月25日通过贡布省白查国际口岸被遣返回国。他们不仅涉嫌网络诈骗，还牵涉藏匿、非法贩毒及吸毒、非法持有武器、非法拘禁等一系列犯罪行为，且均无合法护照并在柬埔寨非法居留。此次行动再次彰显了柬埔寨当局打击跨境犯罪的决心。
