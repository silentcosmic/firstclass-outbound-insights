# 柬埔寨严打网络犯罪：涉案人员或遣返或起诉

🚨 柬埔寨严打网络犯罪：涉案人员或遣返或起诉
柬埔寨班迭棉吉省近期持续开展打击非法网络犯罪行动，多名涉嫌从事相关活动的人员已被拘留并接受进一步审查。
官方消息指出，经调查确认身份及违法事实后，符合遣返条件的外国籍人员将依法遣返；而涉嫌触犯柬埔寨法律并达到刑事立案标准的人员，则将依法移送法院审理，追究其法律责任。
