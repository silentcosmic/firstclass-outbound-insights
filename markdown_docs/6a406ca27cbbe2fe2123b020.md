# 新加坡主要银行战略部署：2027年前实现公共住宅区自动取款机服务广覆盖

新加坡三大本地银行及NETS支付网络正积极拓展自动取款机（ATM）与现金服务点，此举是银行业一项更广泛的策略举措，旨在确保为老年群体持续提供实体银行服务。
这项行动是根据一份行业报告《构建长寿社会下的银行服务》（Banking a Longevity Society playbook）所概述的，该报告旨在随着新加坡步入超老龄化社会，提升老年客户获得银行服务的便利性。
星展银行（DBS）、华侨银行（OCBC）、大华银行（UOB）以及NETS已承诺，将在2027年底前实现服务覆盖率达100%，即在每个公共组屋（HDB）街区500米直线距离内，至少设有一台ATM、一家分支机构或一个现金服务点。
这一承诺的核心目标是确保现金服务作为一种可靠的选项持续可用，尤其对于那些不太习惯数字支付的老年人而言，其重要性不言而喻。
作为中期部署，上述三家银行及NETS将确保，在2026年底前，公共交通枢纽、由国家环境局（NEA）管理的熟食中心以及主要超市500米范围内，均有自动取款机、银行分支机构或现金服务点。
该行业报告指出，服务覆盖范围将定期进行审查，以适应新兴城镇的扩张与增长需求。
此外，各银行也在持续优化自动取款机的无障碍设计，以提升老年客户及残障人士的使用体验。
例如，星展银行已对其分支机构和ATM照明进行了升级；而大华银行和渣打银行则提供了具备音频插孔的增强型ATM，能够提供文本转语音指令。
同时，自动取款机的键盘已配备盲文功能，以支持视障客户的使用需求。
大华银行和渣打银行还推出了高度较低的自动取款机，以方便轮椅使用者。
这些现金服务措施构成了银行业整体规划的一部分，旨在帮助老年人更独立地管理财务，同时确保他们能够便捷地获取必要的实体银行服务。
