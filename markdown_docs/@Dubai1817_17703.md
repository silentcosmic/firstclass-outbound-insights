# 欧易OKX澄清：清退大陆用户系假消息

🚨 欧易OKX澄清：清退大陆用户系假消息
近日，市场有传言称加密货币交易平台欧易OKX将退出中国大陆市场，并计划清退现有用户。此消息源于一张声称“欧易调整部分地区业务运营”的公告截图在网络上广泛流传。
欧易OKX官方已迅速对此进行辟谣，明确指出该网传公告系伪造拼接，并非官方发布。平台强调，所有用户应以OKX官方网站、官方公告页面及已验证的官方账号发布的信息为准，谨防虚假信息，切勿轻信来源不明的截图或随意点击陌生链接。
