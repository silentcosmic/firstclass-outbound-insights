# ️ 洪森女婿警告：柬埔寨电诈拉人月薪仅300美元

⚠️ 洪森女婿警告：柬埔寨电诈拉人月薪仅300美元
柬埔寨国家警察总署副总监迪伟杰中将（首相洪森女婿）近日通过社交媒体发出严厉警告，指出当前网络诈骗团伙正以每月300至600美元的低薪，诱骗柬埔寨当地民众加入其非法活动。
迪伟杰中将强调，民众务必提高警惕，切勿参与此类违法犯罪，以免触犯法律并面临严重后果。同时，他呼吁社会各界积极向警方提供线索，共同打击日益猖獗的网络诈骗犯罪。
