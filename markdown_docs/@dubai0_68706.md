# ️ 泰国芭提雅惊现行李箱藏尸案：澳籍嫌犯落网

⚠️ 泰国芭提雅惊现行李箱藏尸案：澳籍嫌犯落网
近日，泰国芭提雅发生一起骇人听闻的凶杀案。一名泰国少女在公寓内遭一名澳大利亚男子杀害，其尸体随后被塞入行李箱，并遗弃在火车站附近。
犯罪嫌疑人西蒙·彼得·卡曼，46岁，在曼谷素万那普机场试图离境时被捕。目前，他已被带回芭提雅接受进一步审讯。
