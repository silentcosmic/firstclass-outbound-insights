# ️ 拉斯海马TikTok骂战升级：一人判监一人罚款

⚖️ 拉斯海马TikTok骂战升级：一人判监一人罚款
近日，拉斯海马初审法院对两名男子作出判决，因其在TikTok直播中互相辱骂、威胁、骚扰及滥用信息技术，一人被判处三个月监禁，另一人则被处以1000迪拉姆罚款。
案件源于两人间的私人纠纷，随后在TikTok直播中演变为公开争执。直播期间，众多用户目睹了双方互相辱骂，其中一人更在镜头前做出了“割喉”手势，以此发出死亡威胁。
法院审理中详细审查了案卷中的视频片段，确认了其中的言语和动作。法院强调，利用社交媒体平台进行辱骂和威胁，违背了这些平台设立的初衷，构成应受法律惩处的犯罪行为。
法院特别指出，做出“割喉”手势的第二被告行为更为冲动且具挑衅性，该手势在传统上明确暗示了剥夺他人生命的意图，因此被认定为利用信息技术手段发出的杀人威胁。法院同时认定，第一被告也实施了犯罪行为，包括辱骂、威胁和骚扰对方。法院重申，即便受到挑衅，也不能作为以辱骂回击辱骂或以威胁回击威胁的理由；每位被告均须对其个人行为承担相应的刑事责任。
最终，法院判处第一被告1000迪拉姆罚款，第二被告三个月监禁，并责令两人各自承担相关的诉讼费用。
