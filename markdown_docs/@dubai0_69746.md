# Telegram 短域名突遭冻结：全球链接大面积失效

🚨 Telegram 短域名突遭冻结：全球链接大面积失效
突发消息，即时通讯巨头 Telegram 的短链接域名 t.me 已被注册局暂停使用。据 WHOIS 记录显示，该域名自7月13日起被更新为 `serverHold` 状态，并同时施加了包括禁止删除、禁止转移、禁止续费及禁止更新等多项限制。
此次冻结导致通过浏览器、网站或社交平台等外部渠道访问的 t.me/ 链接几乎全部失效，无论是频道、群组、机器人，还是个人账号分享的链接，均受到大面积影响。
