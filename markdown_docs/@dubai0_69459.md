# 老挝金三角突发：中方警力介入，多公司被查

🚨 老挝金三角突发：中方警力介入，多公司被查
近期，有市场消息指出，大批中国警力已进入老挝境内，并在首都万象及金三角经济特区（包括特外和特区）展开行动。
受此影响，特区内多家企业已暂停运营，员工被要求限制外出，以避免不必要的风险。据了解，部分企业在收到预警后已提前采取规避措施，但仍有数家公司在昨日（8日）下午的突击检查中被查处，多名相关人员被捕。
消息人士透露，此次行动中，部分被捕人员系中方警方点名通缉的对象，且中方警员亦在现场参与了部分执法行动。目前，特区内部分业务的正常充值和交易功能已受影响，部分“灰色”业务尤其面临严峻冲击。
鉴于当前局势，建议在老挝金三角特区及相关区域的中国企业及人员密切关注动态，审慎评估风险，并采取必要的防范措施。
