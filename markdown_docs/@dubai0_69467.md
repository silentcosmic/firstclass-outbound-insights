# 肯尼亚突袭诈骗窝点：18名中国公民被捕

🚨 肯尼亚突袭诈骗窝点：18名中国公民被捕
肯尼亚警方日前在首都内罗毕基利马尼区展开专项执法行动，成功捣毁一个涉嫌跨国网络诈骗的犯罪团伙。此次行动中，共有18名中国籍嫌疑人被当场逮捕，警方在现场查获了大量多国护照、电子设备及现金。
目前，所有涉案人员已被羁押，相关案件正在进一步深入调查中。
