# 肯尼亚遣返19名涉诈中国公民，查获大量护照

🚨 肯尼亚遣返19名涉诈中国公民，查获大量护照
7月10日，肯尼亚刑事调查局与移民部门联合展开行动，成功遣返19名涉嫌网络诈骗的中国公民（包括18男1女）。此次行动中，现场查获46本中国及马来西亚护照，整个遣返过程安全顺利完成。
