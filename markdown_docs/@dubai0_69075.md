# 迪拜地铁蓝线公布：全球最高地铁站将落成

⚡ 迪拜地铁蓝线公布：全球最高地铁站将落成
迪拜交通管理局（RTA）近期披露了迪拜地铁蓝线的最新规划细节。这条耗资约55.8亿美元（约205亿迪拉姆）的线路全长30公里，共设14座车站，预计于2029年9月9日投入运营。蓝线将连接现有红线的Centrepoint站与绿线的Creek站，预计到2040年将服务约100万居民，并大幅缩短前往迪拜国际机场的通勤时间。
蓝线亮点频出，其中包括位于迪拜云溪港的伊玛尔地产站，其高达74米的建筑将成为全球最高的地铁站。该站采用开放式空间设计，引入自然采光，并融合石材、玻璃与青铜等材质，以“门户”造型融入周边城市景观。此外，蓝线还将建成迪拜地铁网络中最大的地下换乘站，总面积超4.4万平方米，日均客流可达35万人次，并首次跨越迪拜河建设长达1300米的轨道桥梁。
在设计理念上，蓝线车站将以自然界的“四大元素”——空气、水、土地与火焰为主题，打造独特的室内空间。例如，“空气”主题强调开放与通透，“水”主题借鉴海洋文化营造宁静，“土地”主题展现沉稳与厚重，“火焰”主题则通过色彩与灯光彰显速度与动感。
随着蓝线的建成，迪拜轨道交通网络总里程将增至131公里，车站总数达到78座，列车数量增至168列。RTA表示，蓝线也是其首个按照铂金级绿色建筑标准建设的轨道交通项目，将车站设计、运输能力提升与可持续发展理念同步纳入整体建设体系。
