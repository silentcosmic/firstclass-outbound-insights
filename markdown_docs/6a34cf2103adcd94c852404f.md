# 全球数字媒体巨头整合再深化：Penske Media Corp完成对Vox Media剩余资产的收购

业界仍在回味上月James Murdoch旗下Lupa Systems收购Vox Media约半数资产，具体包括《纽约杂志》、Vox及其播客网络的消息。
而今，另一笔重磅交易靴子落地，其深远影响将波及广告商与新闻从业者未来的发展路径。
周四，Penske Media Corporation正式宣布，将收购Vox Media剩余的半数资产。此次交易涉及的知名品牌矩阵包括The Verge、Eater、SB Nation、The Dodo、Popsugar、Punch以及Thrillist等。
此次交易的财务条款并未公开。但相比之下，James Murdoch此前的收购投入高达3亿美元。此外，Penske Media Corporation早在2023年就已向Vox Media注资1亿美元，从而获得了该公司20%的股权，成为最大单一股东。
无论如何，此番交易使得Penske Media Corporation一跃成为全球最大的数字出版商，这一成就已在其公司新闻稿中得到确认。此前，Penske Media Corporation旗下已汇聚了众多行业巨头品牌，包括Variety、Rolling Stone、Billboard、Deadline和The Hollywood Reporter，同时还拥有金球奖等多项知名活动及颁奖典礼。
Penske Media Corporation旗下的所有出版品牌未来将整合至一个名为PMX的新子公司，并由前Vox Media总裁Ryan Pauley负责管理。
对于长期关注Vox Media商业动向的人士而言，此次出售并非意料之外，这不仅仅是因为5月底的James Murdoch交易。
回溯至2025年，该公司已将其旗下广受欢迎的视频游戏网站Polygon出售给Valnet。同年晚些时候，据业界报道，Vox Media当时已在考虑剥离并出售其播客业务。Penske Media Corporation彼时也对Vox Media的数字媒体品牌表达了浓厚兴趣，但对其播客网络兴趣不大，因其估值远高于其他资产。
此外，Vox Media持续裁员也已是公开的秘密，裁员潮几乎每隔数月便会发生：包括2024年12月，2025年1月、2月、5月和8月，以及2026年1月。
与此同时，Vox Media在那段时间将大量精力投入到挽回因Google搜索流量下滑而造成的营收损失上，采取的策略包括与OpenAI达成许可协议、新增订阅付费墙以吸引登录用户，并加大对视频和音频内容的投入。例如，“The Pivot”节目的主持人Kara Swisher和Scott Galloway根据与Vox Media的合同，将获得2000万美元的报酬。
（值得玩味的是，Penske Media Corporation似乎对AI许可协议兴趣寥寥；去年9月，该公司曾起诉Google，指控其未经授权使用其内容生成AI摘要。）
无论如何，即便人工智能对数字媒体领域的影响力持续扩大，此次事件依然标志着Vox Media一个时代的终结。这家公司在2015年的鼎盛时期曾估值高达10亿美元。
尽管两家公司目前仍计划保持独立运营，但媒体行业的进一步整合，无疑将对营销人员产生深远影响，无论好坏。
正如业内分析曾指出，当Penske Media Corporation首次接管Variety、THR和Deadline——这些品牌曾是相互竞争的对手，如今却同属一家并继续覆盖同一行业——要在多个重叠的媒体资产中区分并销售受众变得异常困难。这意味着，如果广告商威胁从多个品牌而非仅仅一个品牌撤回广告投放，他们将拥有更大的议价权。
但从另一个角度看，若所有广告支出最终都流向顶层的同一母公司，广告商在最初选择投放渠道时，其自主选择权反而大大受限。
话说回来，或许广告资金将彻底撤离开放网络，转投至他们眼中更具前景的“沃土”——例如社交媒体、联网电视（CTV）以及AI搜索。这不正也是Vox Media最初陷入困境的症结所在吗？
