# 吉隆坡豪华公寓突查：9名中国籍嫌犯涉诈骗同胞被捕

🚨 吉隆坡豪华公寓突查：9名中国籍嫌犯涉诈骗同胞被捕
7月10日，马来西亚吉隆坡警方根据线报，对满家乐（Mont Kiara）区一处豪华公寓单位进行突击检查，成功捣毁一个已运作约两个月的网络诈骗呼叫中心。
行动中，警方当场逮捕了9名中国籍男子，年龄介于24岁至41岁。据初步调查，其中一人负责管理该诈骗窝点，其余八人则担任“客服”人员，每人月薪约3000令吉。该团伙主要通过网络平台布置股票投资骗局，专以身在中国的同胞为目标，由客服人员负责联系、诱导投资及后续跟进受害者。
警方在现场查获了23部手机、9台笔记本电脑、路由器及2500令吉现金。所有涉案人员及证物已被带回警局，案件目前正在进一步调查中。
