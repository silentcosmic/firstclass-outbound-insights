# 越南富国岛非法居留案：2人被捕，4名中国人涉案

🚨 越南富国岛非法居留案：2人被捕，4名中国人涉案
7月15日，越南安江省公安厅安全调查机关正式对一起“组织他人在越南非法居留”案件立案，并已对两名涉案嫌疑人阮进桑（Nguyễn Tiến Sang）和阮文当（Nguyễn Văn Đang）提出起诉并批准逮捕。此案涉及四名中国籍人员在富国岛的非法居留及出逃未遂。
调查显示，2026年，31岁的阮进桑在富国特别行政区经营机票销售、旅游服务及住宿业务。他将一栋别墅出租给一名中国籍男子，并在明知入住人员缺乏合法证件的情况下，为谋取利益并避免合同解除，仍同意其继续居住，且未依法办理临时居住登记。该别墅随后被转租，四名中国籍人员入住其中。
2026年6月29日，安江省公安厅在富国岛行政检查中发现两名无护照签证的中国籍人员。为躲避进一步检查，这四名中国籍人员在别墅承租人的安排下试图转移藏匿地点。随后，50岁的阮文当明知他们系非法入境和居留，仍出于牟利目的安排其入住公寓并收取费用。阮文当甚至安排四人前往红玉渡口，企图协助他们离开富国岛前往越南本土。然而，该四人一行在渡口被边防人员及时拦截，随后公安机关介入调查，查明了案件全部事实。
目前，阮进桑和阮文当已承认所有犯罪事实，安江省公安厅正持续深入调查此案，并将依法追究相关人员的法律责任。
