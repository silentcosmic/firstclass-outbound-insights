# 出海观察 | 智能穿戴新锐获数百万美元融资：URTOPIA联创团队指环众筹破千万

最近，AI可穿戴领域新兴品牌AIVELA成功完成了数百万美元的首轮融资。此次融资由线性资本领投，锋领资本跟投，并获得了智能电助力自行车品牌URTOPIA等产业投资方的战略加持。
本轮募集资金将主要投向下一代AI可穿戴产品的深度研发、健康数据生态与AI Agent能力的构建、全球市场版图的拓展以及核心团队的规模化扩充。AIVELA规划以智能指环、智能手链等贴身穿戴设备为切入点，致力于在全球范围内打造一个融合健康感知、自然交互与AI健康服务的新一代消费科技品牌。
审视当下，消费者的健康需求已不再局限于基础的运动记录，正逐步转向更全面的长期身体状态监测，涵盖睡眠质量优化、压力管理等维度。在国际市场，Oura和Whoop两家公司已成功崛起，其估值均突破百亿美元大关，它们分别深耕长期医疗级健康监测与专业体能训练指导领域。
根据Precedence Research研究机构的预测，全球智能可穿戴市场规模预计将在2034年超过4300亿美元，期间的复合年增长率（CAGR）将达到约20%。
AIVELA创始人兼CEO李东豪在早年的连续创业生涯中，曾因高强度工作节奏饱受严重睡眠问题困扰，这也促使他成为了智能指环的早期用户。这段切身体验，最终成为了他投身该领域创业的重要契机。
在亲身体验智能戒指后，李东豪深刻认识到，健康的核心不单是运动表现的提升，更在于睡眠质量、身体恢复、压力管理以及长期生活节奏的整体平衡。运动与恢复被视为一枚硬币的两面，唯有将两者有机结合，用户才能真正实现其长期的健康目标。
AIVELA的首款智能指环产品AIVELA Ring Pro于2025年首次亮相国际众筹平台Kickstarter，成功募得超过86万美元。随后在2026年，该产品登陆日本众筹平台Makuake，上线当日即迅速突破3000万日元。截至目前，这款产品在全球范围内的众筹出货量已超过1万只，总金额突破千万元人民币。
智能指环在消费电子领域被公认为工程集成难度极高的品类。其核心挑战在于如何在性能表现、内部空间和电池续航之间寻求平衡，即业界常说的“不可能三角”。它不仅要求在极致小巧的体积内实现精准稳定的健康监测和流畅的交互功能，同时还必须兼顾长效续航、出色的防水性能、优异的佩戴舒适度以及时尚的饰品化外观。
对于AI可穿戴企业而言，真正的行业壁垒并非仅仅停留在产品概念的提出，更在于能否将产品定义、工程研发、供应链管理、质量控制、全球销售与用户服务等关键环节，高效整合并转化为可规模复制的商业化体系。AIVELA已在其首款产品中成功完成了这一商业闭环，充分验证了团队在复杂消费硬件产品化及交付方面的卓越能力。
在硬件技术方面，AIVELA的首款指环产品突破了市面上普遍采用的电容触控方案，创新性地引入了基于光学的指尖交互技术。通过这一先进技术，智能指环能够化身为“指间鼠标”，无缝连接手机、电脑或智能眼镜，从而实现直观且无感的操控体验。
在软件与服务策略上，AIVELA致力于革新传统设备仅“堆砌数据图表”的单一模式。通过深度整合大语言模型技术，AIVELA成功构建了其专属的AI Agent系统。
据AIVELA创始人李东豪介绍，传统的健康指环大多依赖固定算法，例如用户一旦熬夜，系统便直接给出低分警告，这反而可能加剧用户的焦虑感。AIVELA的设计逻辑则截然不同，它不以单纯的评分机制运作，而是结合用户的上下文信息，通过主动对话的形式，智能地提醒用户关注自身的生活节奏，并在恰当的时机提供个性化的睡眠建议。
成功打通智能指环的规模化量产流程后，AIVELA正积极筹划下一阶段的产品线拓宽，旨在进入手环市场。智能指环的优势在于其轻薄与私密性，使其成为夜间睡眠监测的理想选择；然而，其局限性也同样显著——在白天的力量训练或骑行等需要抓握的场景中，指环可能会带来明显的不适感。
鉴于此，AIVELA目前正全力研发其首款无屏智能手链（Bracelet）。该产品特别邀请了全球知名的工业设计师及珠宝设计师深度参与研发，核心聚焦于生理指标的精准监测，并突出外观的可DIY属性。此举旨在让产品不仅能够实现长期的健康感知，更能成为用户乐于每日佩戴、彰显个人风格的生活方式饰品。未来，AIVELA的智能指环与手环产品将在云端实现数据无缝融合，从而全面覆盖用户“日与夜”、“静与动”的完整生活场景。
在团队背景方面，李东豪此前曾担任智能电助力自行车品牌URTOPIA的联合创始人兼CFO，深度参与了该公司从零开始构建全球化消费硬件业务的全过程，涵盖了投融资、市场进入（GTM）、海外渠道建设、售后服务以及跨境运营等多个核心环节。AIVELA的核心团队成员则汇聚了来自香奈儿、百度、阿里、小米、乐心医疗、Totwoo等知名科技公司和品牌的人才，其专业能力覆盖了品牌营销、工业设计、软硬件研发、供应链管理以及全球市场拓展等关键领域。
展望未来，AIVELA计划进一步深化线上DTC（Direct-to-Consumer）渠道的建设，并依托其在欧美市场积累的丰富渠道与服务体系，逐步铺设线下体验、销售及售后服务网络。同时，公司将积极吸纳AI Agent、智能硬件、健康算法及品牌出海等方向的专业人才，共同打造一个全球化的AI可穿戴品牌。
AIVELA创始人李东豪谈智能指环的制造挑战李东豪表示：“智能指环的生产线极为复杂，涉及大约30多个工位，且大部分环节依赖于纯手工组装。由于其独特的形态设计，我们必须在极其微小的空间内集成软板、弧形电池以及各类精密传感器，因此对最大公差的容忍度极低。我们与代工厂共同经历了漫长而艰辛的爬坡期，投入了巨额资源，才最终达到了目前市场领先的良品率水平。”“在算法层面，手指的活动自由度以及所谓的‘负场景’远多于手腕。例如，当用户敲击键盘或使用筷子时，如果直接沿用传统手环的计步算法，很可能产生误判。我们当前的策略是优先推进量产，并结合大量真实用户反馈，持续捕捉和精调边缘场景，这无疑是一个长期迭代和积累的过程。”
未来产品布局：构建“指环+手环”智能饰品矩阵李东豪阐述道：“我们的核心战略规划是构建一套‘指环+手环’的智能饰品矩阵。我注意到一个实际痛点：许多海外用户为了在健身时佩戴戒指，不得不额外购买笨重不便的硅胶保护套，这种体验无疑是反人类的。指环天然适合夜间睡眠监测，但在进行抓握类运动时却会造成不便，而若不佩戴，则无法持续记录数据。因此，我们通过引入白天的手环场景来弥补这一空缺。更进一步，我们未来可能会探索智能耳环。因为耳朵区域血流丰富，且距离嘴巴较近，这使得监测打鼾或梦话等数据变得更为便捷。总体而言，产品研发的方向始终围绕无感、舒适的穿戴体验进行深度延伸。”
投资人洞察：AI可穿戴的价值与AIVELA的潜力线性资本合伙人杨骏强调：“AI可穿戴的真正核心在于运用人工智能重新定义产品体验和长期服务关系，而并非仅仅是接入大型语言模型。AIVELA团队对于健康、技术与人性化融合的理解展现出清晰的判断力，我们对其在AI可穿戴领域的长期发展潜力充满信心。”锋领资本创始人阮飞则指出：“未来的出海品牌不能仅仅依赖流量红利或单一爆品，它们必须构建起从产品定义、供应链交付到品牌建设的全方位综合能力。AI可穿戴市场蕴藏着巨大的增长空间，我们对AIVELA凭借这一优势成长为全球化品牌寄予厚望。”URTOPIA创始人张波表示：“AIVELA的健康监测功能与URTOPIA所涵盖的运动场景形成了天然的互补优势。作为战略投资方，未来我们将在AIoT技术、海外渠道拓展以及用户运营等多个维度，与AIVELA展开深度协同合作。”
