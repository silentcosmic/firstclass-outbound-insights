# 特朗普改弦易辙：霍尔木兹通行费方案紧急叫停

⚡ 特朗普改弦易辙：霍尔木兹通行费方案紧急叫停
美国总统特朗普此前宣布的霍尔木兹海峡通行费征收计划，在海湾国家领导人紧急斡旋下，已于日前被紧急叫停。特朗普通过社交媒体宣布，将以与海湾各国达成贸易和投资协议的形式，取代原有的通行费方案。
据美国媒体披露，为促成这一转变，多位高级助手及中东盟友进行了长达24小时的紧急劝说。国际观察人士指出，此举再次凸显了特朗普政府在伊朗问题上缺乏明确的退出战略。
