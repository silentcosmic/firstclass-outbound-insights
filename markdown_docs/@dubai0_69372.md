# 马来西亚新山批发市场突查，多名外籍人士被带走

🚨 马来西亚新山批发市场突查，多名外籍人士被带走
近日，马来西亚柔佛州新山市班兰地区一处批发市场传出警方及执法部门展开突击检查。行动中，多名外籍人员被现场带走协助调查。
目前，被查人员的身份及其涉嫌的具体违法行为均未获官方证实。外界普遍猜测，此次行动可能指向非法劳工、无证务工或其他非法商业活动。鉴于警方或移民局尚未发布进一步通报，行动是否与近年频发的网络或电信诈骗集团有关，目前尚无法确认。
案件详情有待马来西亚相关执法部门的后续官方通报。
