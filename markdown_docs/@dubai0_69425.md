# 越南6月重拳：144名中国电诈嫌犯落网！

🚨 越南6月重拳：144名中国电诈嫌犯落网！
2026年6月以来，越南针对跨境电信网络诈骗团伙发起了多轮专项打击行动。据公开信息显示，越南公安机关在胡志明市、老街省及富寿省等地，累计查获至少144名中国籍涉诈嫌疑人，并成功侦破多起组织跨境转移诈骗的案件。
此次行动背景复杂，随着东南亚各国持续加大对电信诈骗的打击力度，部分原本活跃于柬埔寨的犯罪团伙正加速向周边国家转移。越南此举，无疑是对这类跨境犯罪活动发出的强烈警告，也标志着其在打击此类犯罪上的决心与力度进一步升级。
