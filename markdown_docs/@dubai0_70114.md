# 洪玛奈：柬中将继续联手清剿网赌电诈

🚨 洪玛奈：柬中将继续联手清剿网赌电诈
在柬埔寨总理洪玛奈近期访问中国后，其助理部长弗朗索瓦在媒体吹风会上通报了访问成果。弗朗索瓦指出，洪玛奈总理在与中国领导人会晤期间，重申了柬埔寨政府的坚定决心，将继续与中方紧密合作，严厉打击并彻底清剿包括网络诈骗在内的各类高科技犯罪活动。
