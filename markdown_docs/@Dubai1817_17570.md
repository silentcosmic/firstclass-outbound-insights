# 行业警示：2亿U跑分盘爆雷，平台信任危机加剧

🚨 行业警示：2亿U跑分盘爆雷，平台信任危机加剧
近日，市场上名为F5678、F1555、K688的多个“跑分”公群项目，以“优乐淘商城跑分”为幌子，并声称获得某知名平台“新币”背书，吸引大量投资者参与。该项目对外宣称拥有700万U的保证金，但实际涉案资金规模据称高达2亿U，其中不乏宝妈群体约130万U的资金，以及上百名单笔投资超30万U的大户。
尽管不少投资者已有所警觉，但由于对“新币”平台规则及其本金保障的信任，仍选择继续参与。然而，项目运营期间，陆续出现加减账不透明、恶意漏报、虚假下发等一系列异常操作。
据反映，数十名受害者曾向F组及巡查部门举报，但未能获得有效处理，反而导致项目规模持续扩大。随后，有内部消息指出，“新币”平台高层与公群负责人之间存在勾结套利行为，最终导致资金盘全面崩盘，相关负责人集体失联。
此次事件引发了市场对“新币”平台信誉的广泛质疑。许多投资者认为，该平台在纠纷处理上采取冷处理态度，未能履行其担保责任，严重损害了其作为“信用背书者”的公信力。有观点直指，此次事件是“新币”利用其平台信誉开设小号公群，诱骗客户资金的欺诈行为。
