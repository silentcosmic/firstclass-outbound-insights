# 柬埔寨副总监亲督：波贝反诈部署再升级

🚨 柬埔寨副总监亲督：波贝反诈部署再升级
今日上午，柬埔寨国家警察总署副总监兼打击网络诈骗工作组组长迪伟杰主持重要会议，部署在卜迭棉芷省（波贝地区）的下一步打击网络诈骗工作。
迪伟杰在会上强调，此次行动旨在总结前期经验，进一步强化专业执法程序，以全面提升预防和打击高科技诈骗犯罪的效率。他同时指出，各部门必须严格落实政府第03号政令，加强对集中居住场所的安全管理，以共同维护社会安全与公共秩序。
