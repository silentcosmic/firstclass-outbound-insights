# 阿联酋ICP服务费支付升级：Tabby引入分期付款

📣 阿联酋ICP服务费支付升级：Tabby引入分期付款
阿联酋联邦身份、公民、关税与港口安全管理局（ICP）近日宣布，其多项政府服务现已支持通过金融科技平台Tabby进行分期支付，进一步提升了居民办理各项服务的便利性。
此项新政允许客户通过Tabby分期支付最高达2万迪拉姆的服务费，分期期限为3至12个月，但需通过信用评估。在此之前，ICP已与十家主要本土银行合作推出“Easy Payment”服务，信用卡持卡人可将500迪拉姆及以上的服务费进行3至12个月的零利率分期。这意味着，在办理身份证、签证、居留许可等服务时，居民将拥有更灵活的支付选择，可根据自身需求在Tabby与银行分期服务间进行切换。
