# 泰缅边境出行警示：中国公民或遭拒入境

🚨 泰缅边境出行警示：中国公民或遭拒入境
中国驻泰国大使馆近日发布领事提醒，指出有中国公民反映，因曾前往泰国达府（Tak Province）等泰缅边境地区，在入境泰国时被移民部门列为重点核查对象，甚至被拒绝入境。
使馆特别提醒中国公民，务必充分了解相关风险，谨慎前往达府、湄索等泰缅边境地区。
