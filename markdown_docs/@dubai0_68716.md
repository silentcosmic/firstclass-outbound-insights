# iPhone 18涨价传闻：阿联酋旧款手机也跟涨！

🔥 iPhone 18涨价传闻：阿联酋旧款手机也跟涨！
近期市场消息显示，即将发布的iPhone 18在阿联酋的售价或将上调。受此预期影响，当地现有型号的iPhone价格已先行出现波动，部分旧款手机价格甚至开始攀升。
据观察，iPhone 17的部分型号价格已上涨200至300迪拉姆（约合人民币400-600元）。消费者为规避未来新机涨价，纷纷转向购买现有型号，导致市场需求激增，部分型号甚至出现供应紧张。一位当地零售商透露，几天前iPhone 17 Pro售价还在4700迪拉姆，目前已涨至4900迪拉姆，若需求持续旺盛，预计很快将突破5100迪拉姆。
受此带动，二手iPhone 17 Pro的交易价格（目前约4200迪拉姆）预计也将上涨。尽管iPhone 16 Pro Max（256GB）目前价格仍维持在4000迪拉姆左右，但业内人士预估其也将面临调价。其中，iPhone 17 Pro Max最为抢手，自iPhone 18涨价风声传出后，咨询量大增，一度出现断货情况。
