# 飞机异常飞行后从雷达消失

🚨 飞机异常飞行后从雷达消失
根据航班追踪网站FL360aero分享的初步数据，一架K2航空公司飞机在经历异常飞行轨迹后，已从雷达屏幕上完全消失，引发广泛关注。
数据显示，该飞机先是出现下降，随后爬升，接着再次急剧下降。当地时间晚上9点18分，当飞机沿G216航线飞行至卡拉奇以南约150海里处时，机组人员报告导航系统出现故障，并请求航向引导。尽管被指示保持当前航线，飞机随后被观察到向右转弯，并以每分钟约15,000英尺的速度迅速下降。
晚上9点21分，即报告故障仅三分钟后，该飞机在卡拉奇以西约155海里处从雷达上完全消失，同时雷达和通信联系全部中断。目前，多次尝试重新建立联系均未成功。
