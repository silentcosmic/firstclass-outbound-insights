# 伊朗宣布关闭霍尔木兹海峡：全球航运面临严峻挑战

🚨 伊朗宣布关闭霍尔木兹海峡：全球航运面临严峻挑战
据最新消息，伊朗革命卫队海军通过无线电通告，宣布霍尔木兹海峡现已对航运关闭。这一关键航道的突然关闭，无疑将对全球能源供应和国际贸易构成重大冲击。
伊朗方面表示，此举是鉴于以色列在黎巴嫩的行动以及美国“违反承诺”。为确保自身安全，伊朗已要求所有船只立即撤离该海域。
