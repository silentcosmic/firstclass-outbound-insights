# 缅甸妙瓦底战事升级：空袭震动泰缅边境

🔥 缅甸妙瓦底战事升级：空袭震动泰缅边境
7月10日至11日，缅甸克伦邦妙瓦底地区的战事持续升级。缅军连续出动战机，对敏拉班村及周边反政府武装目标实施多轮空袭，并配合地面部队展开激烈交火。
据悉，7月10日中午的空袭中，缅军战机向敏拉班村一处区域投掷炸弹，爆炸地点紧邻泰缅边境，与泰国达府湄索县隔河相望。剧烈爆炸声清晰传至泰国境内，引发边境地区高度关注。7月11日清晨，冲突已进入第四天，缅军再次对反政府武装阵地实施多轮空袭，巨大声响持续传至泰国一侧。
目前，妙瓦底及周边地区的交火仍在持续，边境局势高度紧张。此次冲突对在该区域运营的跨国企业及人员构成显著安全风险，伤亡及财产损失情况仍有待进一步确认。
