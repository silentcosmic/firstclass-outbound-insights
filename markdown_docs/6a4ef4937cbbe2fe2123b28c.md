# 马士基资本携手阿亚拉集团，共同赋能AC物流加速拓展菲律宾市场版图

A.P. Moller Capital与阿亚拉集团（Ayala Corporation）近日宣布，A.P. Moller Capital旗下新兴市场基础设施基金II（Emerging Markets Infrastructure Fund II）对AC物流控股公司（AC Logistics Holdings Corporation，简称AC物流）的战略投资已顺利完成。此举标志着菲律宾领先的综合物流供应商在发展历程中迈出了重要一步。
根据双方联合声明，在包括兼并控制审批在内的所有交割条件达成后，A.P. Moller Capital新兴市场基础设施基金II已通过其EMIF II Holding III B.V.实体，完成了对AC物流40%股权的投资。
此次战略合作整合了A.P. Moller Capital在全球运输与物流领域的专业知识和运营能力，以及阿亚拉集团深厚的本土市场洞察、广泛的关系网络和在菲律宾成功构建商业的卓越记录。
A.P. Moller Capital的注资及战略伙伴关系将支持AC物流迈入下一阶段的增长。具体而言，将助力其扩大全国业务覆盖和运力，提升关键物流细分市场的服务产品，强化运营能力与服务质量，并全面采纳全球领先实践与连接技术。
除了资本注入，A.P. Moller Capital还带来了深厚的全球运输与物流领域专业经验，这将显著提升AC物流在其四大战略支柱——冷链物流、航空货运物流、合同物流与全国分销，以及项目货运——的增长潜力。同时，此合作也将使其在食品、农业、医疗保健、制药、消费品和零售等关键领域的服务能力得到进一步加强。
作为一项互补性发展，AC物流此前已完成了对Glacier Megafridge, Inc.（GMI）84%股权的收购，进一步巩固了其在冷链平台的实力。
GMI的加入显著增强了AC物流满足对温控物流需求的能力，尤其是在食品、农业、医疗保健及其他对可靠性和温度完整性要求极高的领域。“这些交易的完成标志着我们在菲律宾建设领先的综合物流平台方面迈出了变革性的一步。通过与A.P. Moller Capital和GMI的携手，我们正在扩大规模、提升能力，并拓展在全国范围内的业务覆盖。”
“这使得我们能够为客户提供更便捷、更全面的菲律宾全国物流节点网络服务。”AC物流总裁兼首席执行官Erry Hardianto表示。
AC物流的当前重心将是把GMI整合到其网络中，并协调运营和团队，同时保持其既有的成功优势。
该公司的首要任务是通过扩大规模来释放商业协同效应，同时采取审慎且分阶段的方法推进更广泛的整合。“AC物流在相对较短的时间内，已在菲律宾物流市场建立了稳固的地位。菲律宾的地理环境和增长前景，明确呼唤更高效、更具韧性且更整合的供应链。”
“我们看到了一个支持AC物流和阿亚拉集团扩大平台的机会，该平台不仅能实现可观且可持续的长期增长，服务关键物流领域的客户，同时也能促进菲律宾物流行业的整体发展。”A.P. Moller Capital高级合伙人Lars Reno Jakobsen指出。
与此同时，阿亚拉集团总裁兼首席执行官Cezar P. Consing表示，此次合作反映了阿亚拉集团与世界一流企业合作的长期战略，旨在扩大业务规模、强化执行力，并更好地服务客户。
