# 吉隆坡警方捣毁电诈窝点，9名中国籍嫌犯被捕

🚨 吉隆坡警方捣毁电诈窝点，9名中国籍嫌犯被捕
6月25日上午，吉隆坡警方根据公众举报与前期侦查线索，对旺沙玛珠一处高档公寓展开突击搜查，成功捣毁一个专门针对中国公民的网络外汇投资诈骗团伙。行动中，警方当场逮捕9名中国籍涉案人员及1名马来西亚籍嫌犯，并查获大量作案工具，包括笔记本电脑和手机。
据警方初步核查，该团伙自今年5月初起便以此为据点，主要针对中国境内民众实施虚假外汇投资诈骗。经查，该马来西亚籍嫌犯主要负责为团伙提供后勤支持，包括伙食配送等。
目前，警方正深入调查涉案公寓业主身份、协助外籍人员入境的中介机构以及该诈骗团伙的幕后主使。
