# 马来西亚捣毁世界杯赌球中心：28名中国籍嫌犯落网

🚨 马来西亚捣毁世界杯赌球中心：28名中国籍嫌犯落网
马来西亚警方日前在吉隆坡冼都区展开突击行动，成功捣毁一个涉嫌利用世界杯赛事进行网络赌球的呼叫中心。此次行动共逮捕34名嫌疑人，其中包括28名中国籍人士，以及来自缅甸、韩国、泰国和马来西亚的公民，年龄介于20至40岁之间。
初步调查显示，该窝点被用作2026年世界杯网络赌球的推广、招揽及客户服务中心。警方在现场查获多部手机、电脑等涉案设备，并正对幕后组织者及资金流向展开进一步侦办，不排除将有更多涉案人员被捕。
