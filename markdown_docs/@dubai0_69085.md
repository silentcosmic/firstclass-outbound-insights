# 霍尔木兹海峡航运渐复，中东产油国增供

🚢 霍尔木兹海峡航运渐复，中东产油国增供
全球能源市场正密切关注霍尔木兹海峡的通航情况。尽管商业航运正逐步恢复，但其运输能力、效率和市场信心尚未完全回到冲突前水平。近期数据显示，海峡通行量达到近一个月高点，但与此前日均百余艘船只的常态仍有差距。
观察人士指出，霍尔木兹海峡仍面临水雷等潜在风险，导致航运风险溢价远高于战事前。高风险运营、保险成本上升以及各方在通航规则上的分歧，持续为海峡通行带来不确定性。
然而，据市场消息，在美伊达成阶段性停火安排后，海峡石油运输已逐步恢复。海湾地区主要产油国，包括沙特阿拉伯、科威特和阿联酋，均已提升了原油发运量，这为全球市场供应带来了积极改善。
