# ️ 英国网红迪拜面临死刑指控：自卫杀人案引关注

⚠️ 英国网红迪拜面临死刑指控：自卫杀人案引关注
近日，英国23岁网红布鲁克·乔治在迪拜被捕，面临故意杀人指控，最高可能被判处死刑。据乔治女士本人陈述，她是在遭受男友威廉·特里比暴力袭击后，出于自卫才持刀反击。
据了解，乔治女士是一名拥有逾10万粉丝的TikTok网红，通过社交媒体与在迪拜工作的英国男子特里比相识。她声称，两人关系在第二次迪拜之行中变得“充满控制欲和虐待”，特里比甚至扣押了她的护照。事件发生前，乔治已订好回国机票。她描述，特里比在公寓内对其施暴，她在恐惧中抓起厨房刀具自卫。
乔治女士于6月22日凌晨在迪拜国际机场准备登机时被捕。人权组织“迪拜被拘留者”已介入此案，为其提供辩护，并指出乔治被捕时“身上有伤痕和瘀青”，且在关押期间被要求脱光衣服接受检查，现场缺乏女性警员。乔治的母亲也证实，女儿在事发前后状态异常，通话时“吓坏了，哭个不停，一只眼睛肿得很厉害”。
英国外交部已表示正向乔治及其家人提供领事协助。此案目前仍在阿联酋司法程序中，相关调查正在进行。
