# 阿曼提议霍尔木兹新航线：伊朗水域需审批

📣 阿曼提议霍尔木兹新航线：伊朗水域需审批
阿曼近日提出了一项关于霍尔木兹海峡航运管理的新方案，旨在将海峡航运划分为两条明确的航线。其中，途经阿曼水域的南部航线将允许船舶自由通行；而穿越伊朗水域的北部航线，则需获得伊朗的预先批准，但船舶无需支付任何过境费用。
目前，这项提案仍在初期磋商阶段。伊朗和阿曼两国官员已在马斯喀特举行了会谈，就具体细节进行深入讨论。此举或将对全球能源运输格局产生潜在影响。
