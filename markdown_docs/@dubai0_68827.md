# 阿联酋彩票再出巨奖：得主身份引印度裔猜测

💰 阿联酋彩票再出巨奖：得主身份引印度裔猜测
阿联酋彩票于7月1日星期三揭晓了第二位3000万迪拉姆（约合817万美元）头奖得主。
彩票运营方公布的获奖者名字为“苏尼尔库S”。此种部分遮盖的公布方式，不禁引发了市场对获奖者身份的广泛猜测。
鉴于“苏尼尔库马尔”是阿联酋印度居民中的常见名字，外界普遍认为此次得主大概率也是一名印度居民。此前，阿联酋彩票曾有多位印度裔大奖得主，包括一位独得1亿迪拉姆巨奖的印度籍IT专业人士。社交媒体上，关于最新得主是否同样来自印度的讨论已十分活跃。
