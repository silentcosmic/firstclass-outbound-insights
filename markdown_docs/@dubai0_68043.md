# 美国务卿鲁比奥将出访中东三国：聚焦海湾外交

📣 美国务卿鲁比奥将出访中东三国：聚焦海湾外交
美国国务院近日发表声明，宣布国务卿鲁比奥将于本月23日至25日，对阿联酋、科威特和巴林进行正式访问。此次行程旨在加强美国与海湾地区盟友的双边关系，预计将就区域安全、经济合作以及共同关注的国际议题进行深入探讨。
