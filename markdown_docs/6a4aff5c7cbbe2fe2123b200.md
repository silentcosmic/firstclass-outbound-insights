# 淡马锡关联机构Azalea重磅推出3.5亿美元私募股权常青基金，开创灵活投资新纪元

Azalea投资管理公司（Azalea Investment Management Pte. Ltd.），这家与淡马锡有着深厚关联的机构，于近日正式宣布，其首支私募股权（PE）常青基金“Azalea All Access”已成功启动。该基金凭借其创新模式，获得了来自一组精选的基石投资者高达3.5亿美元的初始认购承诺。
公司在一份声明中指出，此次基金的推出，标志着Azalea在持续构建创新投资平台、拓宽投资者参与私募股权市场渠道方面，迈出了具有里程碑意义的重要一步。
“Azalea All Access”是在Astrea和Altrium两大平台成功经验之上的一次迭代升级，它代表了Azalea平台化战略演进的最新阶段。
作为Azalea旗下的首支常青基金，“Azalea All Access”的设立，精准契合了当前寻求进入私募股权领域投资者不断演变的需求。
“Azalea All Access”致力于为投资者实现长期的资本增值，并通过其派息份额类别，提供定期分红的灵活选项。
凭借其全额实缴的投资模式，投资者从基金启动首日即可立即享有对高度多元化的私募股权二级市场投资及共同投资组合的参与权，这些组合均由全球顶尖的优质管理人负责运作。
“Azalea All Access”采用常青基金架构，支持按月认购和按季赎回，这极大地提升了参与私募股权投资的可及性，并为投资者长期实现复利增值创造了条件。
“Azalea All Access”获得了亚洲地区多元化投资者的鼎力支持，其中包括机构投资者、私人银行、家族办公室以及高净值个人。这充分印证了市场对兼具机构级准入能力与更高灵活性的私募股权解决方案日益增长的需求。“推出‘Azalea All Access’，我们旨在引入一支常青基金，以拓宽我们的私募股权解决方案体系，并使投资者能够长期持有并维持在该资产类别中的敞口。”
Azalea首席执行官兼首席投资官Chue En Yaw表示：“我们非常感谢投资者给予的信任与支持，这反映出市场对更易于获取的私募股权产品有着日益增长的需求。”
Azalea投资董事总经理兼Azalea All Access平台负责人Mark Hindriks进一步阐释道，“Azalea All Access”的推出，正是基于公司在Astrea和Altrium两大平台积累的丰富经验。
他补充说，新基金巧妙地融合了Astrea在结构设计、收益管理及流动性规划方面的专长，以及Altrium在私募股权二级市场与共同投资领域所建立的深厚投资实力。“通过两大平台长期构建的广泛关系网络，我们能够在全球范围内甄选并评估高质量的投资机会，这为‘Azalea All Access’持续部署资本并实现优异回报奠定了坚实基础。”
作为一个常青基金，“Azalea All Access”为符合条件的投资者提供了按月认购和按季赎回的灵活窗口，使其能够随着时间的推移，有效构建并持续管理其在私募股权领域的投资敞口。
