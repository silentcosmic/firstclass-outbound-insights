# 东南亚诈骗风向变：胡志明市重拳出击，捣毁多起跨国窝点

🚨 东南亚诈骗风向变：胡志明市重拳出击，捣毁多起跨国窝点
越南胡志明市警方近期完成了一项为期45天的大规模专项打击行动，重点清剿跨境网络诈骗团伙及非法居留人员。此次行动中，警方共检查了1616处酒店、公寓及出租住所，查获涉嫌非法居留、非法务工等多项违规的外籍人员311名。
行动期间，多起重大诈骗案件成功侦破。其中，警方突袭一旅馆，发现85名中国籍人员，现场查获近200台电脑及550余部手机等大量涉诈设备，该团伙疑似正筹备针对境外受害人的网络诈骗活动。另一起案件中，警方在团伙正式运营前将其摧毁，发现83名中国籍人员涉嫌组建新的诈骗网络，部分人员疑从柬埔寨非法入境。此外，警方还在一住宅内查获一处已开始实施诈骗的窝点，有42名中国籍人员被封闭式管理，并缴获132部手机、77台平板电脑等设备。
警方调查显示，随着柬埔寨持续加大对电信网络诈骗的打击力度，部分犯罪团伙正试图将活动转移至越南境内，以规避执法风险。截至目前，胡志明市警方已针对此次行动中的相关案件立案8起，并依法起诉了包括组织者、场所提供者及协助非法活动中间人在内的26名犯罪嫌疑人。
