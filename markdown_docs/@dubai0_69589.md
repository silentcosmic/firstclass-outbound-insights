# 越南捣毁跨国诈骗团伙：11名中国人被捕

🚨 越南捣毁跨国诈骗团伙：11名中国人被捕
据越南媒体《青年报》消息，越南北部北宁省警方近日成功抓获一个涉嫌网络诈骗的跨国团伙，共有11名中国籍嫌疑人被捕。
该团伙专门以法国公民为目标，利用社交媒体平台及带有翻译功能的聊天软件与受害者建立联系并获取信任，随后诱骗其在虚假投资平台上投入资金。警方在突击行动中缴获了11台电脑和12部手机等作案工具。目前，越南警方正对嫌疑人的具体分工展开进一步调查，并将依法追究刑事责任。
