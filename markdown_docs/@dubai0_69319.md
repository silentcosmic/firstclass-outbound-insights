# 迪拜空调费大改革：告别乱收费，居民支出将降低

💰 迪拜空调费大改革：告别乱收费，居民支出将降低
迪拜备受诟病的空调收费模式即将迎来重大变革。长期以来，居民普遍反映的费用“乱象”，正随着政府新规的出台而逐步走向规范化与透明化，预计从2025年起，实际支出将显著降低。
此前，迪拜的供冷服务主要分为集中供冷（如通过Empower、Emicool等公司）和独立供冷两种。尤其在集中供冷模式下，“容量费”（Capacity Charge）这一固定费用，以及老旧小区普遍存在的按面积均摊、缺乏独立分表的“大锅饭”现象，导致计费不公。此外，部分物业将空调费打包进服务费中，进一步加剧了透明度不足的问题。
为解决这些痛点，迪拜政府（通过RSB与DSCE）发布了RD10新规，旨在彻底整治供冷市场。根据新规，从2025年起：新建的单体建筑供冷将取消容量费，仅收取实际使用费用，且消耗费设定了0.80 AED/RTH的上限（远低于此前1.3-1.5 AED的普遍水平）。针对老旧建筑，政府也要求逐步安装分表，并对消耗费设定了0.643 AED/RTH的封顶价。
这一系列措施标志着迪拜供冷市场正从过去的“野蛮生长”迈向标准化、透明化管理。未来，居民和企业将享受到更清晰、更合理的空调服务，有效降低运营和生活成本。
