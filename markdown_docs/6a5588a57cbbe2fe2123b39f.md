# 极氪深化马来西亚市场布局，经销商网络增至11家，剑指年内18家战略目标

源自中国的电动汽车品牌极氪（Zeekr）近日宣布，已与Quantum Motors、Wheelcorp Premium和Ghee Hua Co三家新合作伙伴成功签署协议，进一步拓展其在马来西亚的经销商网络，目前零售网点总数已增至11家。
极氪周一发布的声明指出，作为吉利控股集团旗下高端电动汽车品牌，极氪不仅在2026年持续领跑马来西亚高端电动汽车销量榜，更规划在今年年底前，将全国服务网点扩展至18家。
新增的零售网点地理覆盖范围横跨马来西亚半岛及东马区域。目前，极氪在马来西亚的完整经销商合作伙伴阵营已然成形，其中包括位于怡保的BH Premium Auto、马六甲的Unihub Auto Services、安邦的EVA Automobile、八打灵再也的Sentinel Automotive。此外，Preeminent Auto的业务版图延伸至吉隆坡市中心、蒲种、士姑来以及联邦大道；双威集团（Sunway Group）则聚焦双威城；而来自新加坡的Carro则布局孟沙、珍珠白沙罗和Juru Autocity。
此次经销商网络的战略性扩张，是在极氪马来西亚市场一系列重要进展之后顺势推进的。这些进展包括2026款极氪X的推出，以及极氪7X和极氪009车型持续取得的市场成功。在近期落幕的2026年吉隆坡国际移动出行展（Kuala Lumpur International Mobility Show 2026）上，极氪马来西亚公司还向公众展示了极氪9X和极氪009 Grand，这两款车型均已正式开放预订。
极氪智能科技马来西亚公司总经理Eddy Lu先生强调，公司正与经销商伙伴紧密协作，致力于持续优化客户的拥车体验，确保从车辆购置到完善的售后支持，每位客户都能享受到流畅无忧且尊贵的服务旅程。
极氪在全球范围内布局了领先的研发中心与设计工作室，分布于哥德堡、宁波、杭州及上海等地。该品牌于2024年12月正式进入马来西亚市场，并随后设立了国家级销售公司，统筹管理批发业务、经销商网络拓展及全面的售后运营服务。
