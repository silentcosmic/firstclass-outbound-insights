# 朝鲜黑客窃取全球三分之二加密货币

🚨 朝鲜黑客窃取全球三分之二加密货币
区块链分析公司TRM实验室最新报告指出，2026年上半年全球加密货币被盗损失高达9.72亿美元，其中约6.43亿美元，即近三分之二的盗窃案，被指向与朝鲜相关的黑客活动。
报告显示，今年上半年共发生207起加密货币黑客攻击事件，创下TRM实验室统计以来的半年新高。尽管攻击数量激增，但整体损失金额较去年同期有所下降。其中，4月份的两起大型攻击——漂移协议（约2.85亿美元）和凯尔普项目（约2.92亿美元）——合计造成约5.77亿美元损失，占据了上半年被盗总额的大部分。
TRM实验室强调，朝鲜相关黑客仍是全球加密货币盗窃损失的最大来源。即便今年上半年涉案金额低于去年同期，这并不意味着风险降低，而更多是由于其他大型盗窃案相对减少所致，其威胁本质并未改变。
