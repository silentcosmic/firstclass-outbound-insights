# 柔佛州提前投票：3565警力部署2026年州选维安

🚨 柔佛州提前投票：3565警力部署2026年州选维安
柔佛州选举的提前投票已定于2026年7月7日正式举行。为确保投票过程的安全与秩序，柔佛警方已提前部署，将动用共3565名警务人员，全面负责各投票中心的安保及秩序维护工作。
此次执勤警力包括647名警官、2806名普通警员，以及112名文职人员。他们将承担投票中心安保、票箱押运、交通疏导、巡逻防控及现场执法等多项任务。
本次提前投票将在全州64个投票中心进行，涵盖警队与军方人员投票点。警方表示，已针对各投票地点制定了周密的安保安排，以确保选民能够在安全、顺畅的环境下完成投票。同时，警方也提醒相关选民，务必按照指定时间和地点前往投票，并听从现场人员指挥，避免造成拥堵或影响投票秩序。
