# 聚币创始人泰国落网：JU币一度暴跌超90%

🚨 聚币创始人泰国落网：JU币一度暴跌超90%
日前，虚拟货币交易所JuCoin（聚币）创始人刘源在泰国落网，并已被当地有关部门扣留。据市场消息，刘源在入境泰国时因所持某小国护照存在问题被边检拦截，随后其身份被核实，发现已被国际刑警组织（INTERPOL）及国内联合追逃名单列入。
网传信息指出，刘源在被捕后曾出示中国护照，但因其真实身份已被确认，目前已被限制离境，等待遣返回国接受进一步调查。此案被指涉嫌跨境虚拟货币诈骗，涉案金额约达16亿元人民币，引发国际执法部门关注。不过，上述涉案金额及案件具体细节，目前尚未获得官方披露。
受此事件影响，JuCoin平台发行的JU代币当天出现大幅波动，盘中跌幅一度超过90%，市场恐慌情绪迅速蔓延。CoinMarketCap数据显示，JU价格在24小时内曾跌超80%。同时，市场也流传关于该平台提现异常及“跑路”等说法，但这些传闻同样未获官方证实。
