# 缅甸腊戌：屋主反杀劫匪却被捕

🚨 缅甸腊戌：屋主反杀劫匪却被捕
缅甸掸邦北部腊戌市近日发生一起备受关注的案件。一名屋主在自家门前遭遇三名男子意图抢夺摩托车时，持刀反抗，导致其中一名劫匪头部被刺死亡，另外两人逃逸。然而，该屋主却因此被当地警方逮捕，面临杀人罪指控。
这起事件再次凸显了腊戌市严峻的治安形势。当地消息指出，摩托车抢劫案在该地区频发，甚至常伴有暴力伤人事件。例如，6月22日，腊戌市也曾发生一名女性骑手遭两名男子持刀抢劫摩托车的案件。此次屋主自卫反击却被捕，引发了关于当地法律与公民自卫权的广泛讨论。
