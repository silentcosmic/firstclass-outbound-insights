# 芭提雅清晨枪击：轿车连中两枪，司机幸免

🚨 芭提雅清晨枪击：轿车连中两枪，司机幸免
泰国芭提雅近日发生一起枪击事件，一名轿车司机在清晨时分遭遇不明枪手袭击，车辆连中两枪，所幸司机毫发无损。警方已介入调查，全力追查嫌疑人。
据警方通报，这起事件发生在7月10日凌晨6时24分左右，地点位于芭提雅中天二路中天5巷附近。一辆白色本田City轿车车身出现明显弹孔。车主兼司机为一名35岁男子，事件中他幸运地躲过一劫。
司机向警方描述，事发前他曾在中天3巷附近与一名男子发生争执。随后，当他驾车行驶至中天5巷时，一名骑乘黑色本田Wave摩托车的男子靠近其车辆。该嫌疑人当时身穿黄色短袖、牛仔裤，并佩戴黑色头盔，随即向轿车连开两枪后，沿中天5巷方向逃往东滩海滩。
案发后，警方迅速封锁现场，展开调查取证工作，包括收集弹壳、车辆受损证据以及调取周边监控录像。目前，警方正深入调查双方争执的具体原因，并加紧追查嫌疑人的身份及行踪，以尽快将其缉拿归案。
