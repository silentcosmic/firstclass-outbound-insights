# ️ 柬埔寨电诈园区：中国男子逃离西港再陷波贝

⚡️ 柬埔寨电诈园区：中国男子逃离西港再陷波贝
据中国公安部刑侦局披露，一名中国籍受害者近日自述其在柬埔寨电诈园区的悲惨经历。他因高额债务，轻信友人“柬埔寨能快速搞钱”的诱惑，最终前往西哈努克港（西港）从事股票诈骗活动。
该男子在园区内遭受殴打电击，身上伤痕累累。他透露，在柬期间不仅分文未得，逃离后又被朋友诱骗至波贝，囚禁于“中转站”再次遭受暴力，随后被转卖至其他园区。
最终，该男子成功从园区逃脱，但因未随身携带护照原件，在宪兵检查时被发现，最终被驱逐回国。此事件再次凸显了柬埔寨电诈问题的严峻性。
柬埔寨政府近一年来持续加大清查力度，迄今已遣返约36万名涉嫌电诈、非法入境、居留或务工的外国人。然而，尽管西港、波贝、柴桢、金边等重点区域的打击行动已取得显著成效，相关犯罪活动仍可能以更隐蔽的方式流动，甚至向周边国家转移。
中国公安部此前通报，近年来通过国际执法合作已累计移交5.8万名涉诈嫌犯，并成功铲除了缅北“四大家族”等重大跨境赌诈集团。
