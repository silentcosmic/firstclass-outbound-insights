# 优必选U1人形机器人发布，最高定价99万元引关注

🤖 优必选U1人形机器人发布，最高定价99万元引关注
优必选于6月30日在深圳正式发布了其消费级品牌“优世界”的首款全尺寸超仿生人形机器人U1系列。该系列产品定价跨度较大，其中Ultra男款售价高达99万元，女款88万元，而Pro和Lite版本则分别为16.98万元和11.98万元。
U1系列机器人主要面向成年用户，旨在提供家庭情感陪伴。产品配备88个高自由度关节，Ultra版具备自主行动能力，并强调交互数据本地加密存储，不强制上传云端，但明确不承担清洁或烹饪等家务功能。据悉，该产品已于6月2日在京东启动预售，并将持续至7月15日，支付3000元定金即可锁定首批购买资格。优必选方面透露，目前全渠道订单已突破1.1万台，首批产品预计于9月中旬开始交付。
