# 斯里兰卡出台新政：全面打击金融犯罪

⚡ 斯里兰卡出台新政：全面打击金融犯罪
斯里兰卡内阁于7月14日批准了一项重要国家政策——《2026—2030年斯里兰卡防止洗钱、打击恐怖主义融资及防范大规模杀伤性武器扩散融资国家政策》。
该提案由总统兼财政部长阿努拉·库马拉·迪萨纳亚克提交，旨在全面强化该国在金融风险防范领域的监管框架。这项政策依据斯里兰卡中央银行金融情报中心开展的第三次国家金融犯罪风险评估制定，并严格符合金融行动特别工作组（FATF）的国际标准。
斯里兰卡政府指出，洗钱、恐怖主义融资及大规模杀伤性武器扩散融资对国家经济乃至全球金融体系构成严重威胁。面对科技发展带来的金融交易日益复杂化，识别和防范非法活动正变得愈发困难。
新政策的出台，旨在进一步完善斯里兰卡的防范机制，确保其金融体系与国际标准接轨，并增强公众对该国金融稳定的信心。
