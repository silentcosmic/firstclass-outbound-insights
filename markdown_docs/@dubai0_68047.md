# 伊朗宣布：霍尔木兹海峡限时免费开放

📣 伊朗宣布：霍尔木兹海峡限时免费开放
伊朗常驻联合国日内瓦办事处代表阿里·巴赫雷尼近日表示，战略要地霍尔木兹海峡已对国际商船全面开放，并在未来60天内免收一切通行费用。
巴赫雷尼透露，此举是在伊朗与美国谈判的背景下作出的。双方已同意就海峡通行问题建立沟通机制，旨在监控并解决可能出现的任何航运挑战。值得关注的是，该海峡在60天后的具体开放政策，将直接取决于伊美两国后续的谈判进展。
