# 越南公安突查：Topaz Twins公寓发现大型华人犯罪窝点

🚨 越南公安突查：Topaz Twins公寓发现大型华人犯罪窝点
近日，越南镇边坊公安与同奈省公安联合开展外国人居住情况检查行动。行动中，在Topaz Twins公寓附近查获一处大型中国籍人员聚集点。经深入调查，该团伙被确认涉嫌将该地点用作违法犯罪活动的窝点。
