# 摩根大通深化东南亚布局，任命两位联席主管领航区域投行业务

全球金融巨头摩根大通近日宣布，已正式任命 Kelvin Goh 和 Alfons Halim 共同执掌其在东南亚区域的投资银行业务。
此项关键人事任命即刻生效，标志着该行在区域市场战略部署上的重要一步。
两位高管将继续以新加坡为运营基地，履行其拓展后的区域管理职责。
Goh 目前身兼摩根大通亚太区金融机构部门（FIG）主管一职，负责关键客户关系与战略指导。
而 Halim 则执掌该行企业部门的亚太区房地产业务，为客户提供深度的专业咨询。
此次任命的独特之处在于，两位高管在承担区域投行联席主管重任的同时，仍将保留其各自现有的核心职责，共同指引摩根大通在东南亚地区的咨询业务发展方向。
将这一关键的东南亚领导团队本地化，无疑进一步巩固了新加坡作为该地区重要企业融资交易枢纽的战略地位。
