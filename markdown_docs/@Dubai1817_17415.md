# 太子集团高管日本被捕，涉电诈引外交部回应

🚨 太子集团高管日本被捕，涉电诈引外交部回应
据日本媒体报道，一名与太子集团（Prince Group）关联的高管“胡石”（音）近日在东京被捕。太子集团此前被指控运营多处电信诈骗园区。
在中国外交部例行记者会上，路透社记者就此事提问，询问中方是否收到通报及是否推进遣返。外交部发言人郭嘉昆回应称，具体问题建议向中方主管部门询问，并强调打击网赌电诈犯罪是国际社会的共同责任，呼吁周边国家与中方一道加大执法合作力度，以维护地区秩序与人民财产安全。
