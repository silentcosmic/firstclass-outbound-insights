# 苹果产品全线涨价：MacBook、iPad系列售价上调

⚡ 苹果产品全线涨价：MacBook、iPad系列售价上调
科技巨头苹果公司近日对其多款核心产品进行了价格调整。其中，MacBook Neo 的起售价已从599美元上调至699美元。iPad系列也未能幸免，iPad Pro 的价格从999美元升至1,199美元，而iPad Air 的售价则调整为749美元。此外，MacBook Air 和 MacBook Pro 的价格也分别上调至1,299美元和1,999美元。
