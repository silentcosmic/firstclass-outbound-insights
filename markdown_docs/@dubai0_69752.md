# 美国驻阿联酋使领馆取消预约并奉命撤离

🚨 美国驻阿联酋使领馆取消预约并奉命撤离
美国驻阿布扎比大使馆及驻迪拜总领事馆已宣布，因当地安全局势考量，取消原定于7月13日至15日的所有领事预约。使领馆提醒受影响人士无需前往，相关部门将主动联系重新安排。
目前，这两处机构均处于指令性离境状态，意味着非紧急美国政府人员已重新安置至阿联酋境外。受此影响，使领馆仅能为在阿美国公民提供有限的领事协助。
此外，阿联酋境内的所有常规美国签证服务将持续暂停。
