# 扎耶德机场客流破9.3万：夏季出行需求旺盛

📈 扎耶德机场客流破9.3万：夏季出行需求旺盛
据阿布扎比机场公司披露，扎耶德国际机场目前每日旅客吞吐量已突破93,000人次，航班起降接近500架次。随着夏季假期的临近，航空出行需求持续走强。
机场运营方指出，6月26日至30日期间的日均客流量和航班量均显著高于去年同期水平，这充分显示了阿布扎比航空业的持续复苏与强劲增长态势。
