# A+H企业强劲领跑港股IPO募资：龙头力量独占八席

港股IPO：A+H企业强势主导募资榜
2026年6月26日，港股主板迎来三家重量级A股龙头企业——领益智造、圣邦股份及芯碁微装的同步上市，为上半年A+H股的密集上市热潮画上句号。进入2026年以来，A股企业赴港发行H股已蔚然成风，成为资本市场一道亮丽风景线。
权威数据显示，上半年共有24家A股上市公司成功在港发行H股，累计募资规模超过1200亿港元。这一数字不仅刷新了历史同期纪录，更凸显了香港市场对内地优质企业的强大吸引力。对今年港股首次公开发行（IPO）募资额前十榜单的分析显示，A+H企业独占八席。这充分证明了内地行业龙头企业在跨境资本运作上的卓越实力与市场影响力。
这一现象预示着，在当前全球经济格局下，香港作为国际金融中心的独特优势，正持续为寻求全球化布局的A股领军企业提供强劲的融资平台与发展机遇。
市场观察：多地土拍热度回升，核心地块溢价成交
近期，国内房地产市场释放出积极信号。一线及重点二线城市的核心地块土地拍卖呈现显著回暖趋势，多城高溢价成交现象频现，尤其以深圳、杭州、长沙等地表现最为突出。多宗核心片区宅地在激烈竞拍中脱颖而出，以高溢价成交，部分板块地价更是持续刷新当地历史纪录。这背后，房地产企业积极参与竞拍，争夺优质地块，反映出市场信心的逐渐恢复与对未来预期的修正。
