# FPT与微软深化AI战略合作，加速亚洲企业级智能转型

越南领先科技集团FPT公司与全球技术巨头微软公司将携手加速亚洲地区的企业级人工智能（AI）部署及联合创新，重点关注东南亚国家联盟（ASEAN）、日本及韩国市场。
FPT于本周三发布声明，确认已与微软达成一项扩展的战略合作，旨在推动这一目标。此项合作将微软的全球AI平台与FPT在大规模交付及区域市场方面的深厚能力相结合。
作为此次合作的一部分，FPT将以“AI前沿企业”（AI Frontier Company）的身份与微软协作。此类企业以人为本、由智能体驱动，致力于将AI智能体无缝嵌入日常工作流程。这包括早期路线图的协同规划、对新兴AI能力的实验探索，以及为企业级规模化应用开发参考架构。
FPT在全球拥有三万名具备AI增强能力的工程师。该公司将在其内部AI转型中广泛应用微软平台，包括Microsoft 365 Copilot和GitHub Copilot。FPT的目标是在未来三年内，赋能多达两万名开发者掌握智能体开发能力。
此外，两家公司还将携手支持越南实现成为“AI前沿政府”的宏大愿景，合作范围涵盖思想领导力构建、能力提升以及与国家政策相符的解决方案框架搭建。
微软东盟区总裁Mayank Wadhwa指出，越南正迈入AI应用的关键阶段，当地企业已准备好将AI部署从试点项目扩展至更大规模。他强调，结合微软的AI平台与FPT卓越的工程实力，将有力助推越南企业安全且负责任地加速数字化转型。
