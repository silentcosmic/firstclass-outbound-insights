# 埃及绑架案宣判：曾“失联”中国女子获刑15年

🚨 埃及绑架案宣判：曾“失联”中国女子获刑15年
埃及开罗刑事法院近日对一起涉及中国公民的绑架案作出判决，三名中国籍被告邱东升、张龙、池某滢与两名埃及籍同伙均被判处15年监禁。
值得关注的是，21岁被告池某滢此前曾因“在埃及失联”事件在国内引发广泛舆论关注。埃及警方后续证实，她实为犯罪团伙成员，并隐瞒了真实身份。
调查显示，该团伙以高薪工作为诱饵，将中国籍受害人任某骗至开罗，随后在高档公寓内将其囚禁，并采取蒙眼、捆绑及持枪刀威胁等手段，向受害人家属勒索7万美元赎金。当地警方随后采取突袭行动，成功解救人质并抓获所有嫌疑人。
