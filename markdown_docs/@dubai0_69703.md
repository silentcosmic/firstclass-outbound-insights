# 缅甸水沟谷清拆提速：72栋非法建筑已拆除

🚨 缅甸水沟谷清拆提速：72栋非法建筑已拆除
缅甸妙瓦底水沟谷及KK园区非法建筑清拆行动正持续深入，重点针对涉网络诈骗、赌博等违法犯罪活动的设施进行拆除。最新消息显示，一栋9层非法建筑日前被爆破拆除。
截至目前，水沟谷及KK园区累计拆除非法建筑达72栋，清拆行动已取得显著进展。据悉，本轮清拆计划共涉及77栋建筑，目前尚余5栋待处理，相关部门承诺将按计划全面完成清理工作。此外，执法部门同步销毁了查获的各类非法设备，旨在彻底铲除犯罪基础设施，杜绝违法活动死灰复燃。
