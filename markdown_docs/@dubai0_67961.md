# 卡塔尔LNG枢纽爆炸：13人遇难，数十人受伤

🚨 卡塔尔LNG枢纽爆炸：13人遇难，数十人受伤
当地时间6月22日，全球重要液化天然气枢纽之一的卡塔尔莱凡角工业城一家工厂发生爆炸。卡塔尔能源大臣萨阿德·卡阿比确认，此次事故已造成13人死亡、66人受伤，并强调事件为意外，排除侵略或破坏行为。
初步信息显示，遇难者主要为印度和巴基斯坦公民。伤者则涉及卡塔尔、坦桑尼亚、印度、巴基斯坦、几内亚、尼泊尔、孟加拉国、肯尼亚和尼日利亚等多个国家的公民，所幸均无生命危险。阿联酋外交部已就此事件向卡塔尔政府及人民表达慰问，并祝愿伤者早日康复。
