# 出海黑产再爆丑闻：柬埔寨团伙涉毒贩人，主犯东帝汶被捕

🚨 出海黑产再爆丑闻：柬埔寨团伙涉毒贩人，主犯东帝汶被捕
近日，有知情人士向市场爆料称，柬埔寨一名为“华伦”的境外作业团伙涉嫌严重劳务虐待、吸毒及人口贩卖。据称，该团伙骨干成员“阿强”曾多次对员工进行身体体罚、言语威胁，甚至有毒打行为，并涉嫌克扣员工提成。
爆料指出，“阿强”及其团队在柬埔寨期间，曾强制员工进行蛙跳、深蹲、俯卧撑等多项体能惩罚，稍有停顿即遭毒打，甚至因未称呼其“老板”而当众施暴。更令人震惊的是，该团伙被指控参与吸毒、人口贩卖等严重犯罪活动。
尽管“阿强”曾一度逃脱柬埔寨当局的清查，但最新消息称其已在东帝汶遭遇不测。此次事件再次凸显了部分境外非法作业团伙的恶劣行径及其面临的法律制裁风险。
