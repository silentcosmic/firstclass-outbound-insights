# 出海洞察 | 2026年6月香港市场融资概览与深度剖析

2026年6月，香港市场共录得两轮融资交易，总金额达到3117万美元。与2026年5月相比（当月完成6轮融资，总额9530万美元），本月融资总额环比锐减67.3%，表明经过5月份的活跃期后，市场交易活动有所放缓。然而，若与2025年同期（仅录得两轮融资，总额10万美元）相比，本月融资额呈现出超过31000%的惊人增长。需要注意的是，这一显著增长主要源于去年同期的极低基数，而非市场活跃度的实质性激增。
本月资本流向主要集中在融资生命周期的早期阶段。早期融资占据了主导地位，资金总额达2942万美元，占月度总额的94.4%；其余5.6%的资金（约175万美元）则流向了种子轮项目。值得关注的是，本月未录得任何后期融资活动。后期融资通常指C轮及C轮之后的股权投资，以及私募股权（PE）投资。
在资本市场其他交易方面，2026年6月香港市场首次公开募股（IPO）数量为零，与5月份持平。同时，本月也未发生任何并购交易，而5月份和2025年6月则分别录得2起和1起并购事件。
2026年6月重点融资交易聚焦
在仅有的两轮已披露交易中，本月香港的融资格局几乎由一笔交易主导——Urtopia的融资额占据了全港总融资的约94%。地域分布方面，交易活动集中于两个区域，其中湾仔区凭借Urtopia的这一轮融资表现突出。
市场交易活动：放缓与集中
本月香港的交易活动显著放缓。融资总额较5月份骤降三分之二，交易轮次也从6轮锐减至2轮，这表明香港市场正经历一个短暂的交易低谷，而非预示着结构性的深层转变。在有限的交易中，Urtopia高达2942万美元的融资轮次尤其引人注目，其贡献了本月全部募资总额的约94%，使得整个市场的表现高度依赖于这一笔核心交易，凸显了资金集中度极高的特点。
融资结构：早期项目主导
本月香港市场缺乏后期或私募股权投资活动，所有资本均倾斜流向了种子期和早期初创公司。这一现象与周边区域常见的成熟期大型融资轮次形成了鲜明对比，明确揭示了香港市场在当月融资结构上的早期化倾向。
同比数据：低基数效应的警示
虽然同比数据高达31000%的增幅颇为抢眼，但此数据主要反映的是2025年6月（仅10万美元）异常低的融资基数，而非市场动能加速的真实写照。事实上，若将2026年6月的表现置于过去12个月的整体趋势中考量，本月融资水平仍处于相对较低的位置。
