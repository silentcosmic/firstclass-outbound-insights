# 香港汇丰500亿港元巨额骗局曝光，内地男女被捕

🚨 香港汇丰500亿港元巨额骗局曝光，内地男女被捕
香港中环汇丰银行近日揭露一起惊人金融诈骗案。两名持港澳双程证的内地人士，企图通过伪造文书，将高达500亿港元的巨额资金转入私人账户，被银行职员识破并报警。
据港媒报道，6月23日上午银行开门不久，65岁的张姓男子与50岁的孙姓女子进入网点，提交了13页金融文件。银行工作人员在核验过程中，迅速发现其中6份文件存在明显伪造痕迹，包括公章粗糙、数据矛盾及格式不符香港银行规范等问题。工作人员立即稳住二人并报警，中区重案组警员迅速赶到现场。
两人当场被捕，警方同时收缴了所有虚假文书及空白伪造证件。经初步调查，香港警方以涉嫌“行使虚假文书”罪名扣押二人。此罪名在香港一旦定罪，最高可判处14年监禁。目前，案件已由中区警区重案组接手，正深入追查伪造文件的上游来源。
