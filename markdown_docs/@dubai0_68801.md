# 缅甸水沟谷再拆涉诈大楼：已清67栋，仅剩10栋

🚨 缅甸水沟谷再拆涉诈大楼：已清67栋，仅剩10栋
缅甸克伦邦妙瓦底水沟谷地区持续推进打击网络诈骗和网络赌博行动。近日，当地有关部门再次拆除一栋曾用于网络诈骗的非法建筑，进一步加快对涉诈园区基础设施的清理。
此次清理行动中，联合工作组采取了机械拆除与爆破相结合的方式。此前，一栋七层涉诈住宅楼也已通过定向爆破完成清理。官方数据显示，水沟谷地区共计77栋涉诈建筑，原计划以机械拆除57栋、爆破拆除20栋。截至目前，已有67栋建筑完成清理，仅余10栋等待后续处理。
