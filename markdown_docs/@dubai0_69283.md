# 阿布扎比将建全球第二座巨型LED球体地标

⚡ 阿布扎比将建全球第二座巨型LED球体地标
阿布扎比将迎来一座引人注目的新地标——一座巨大的发光球体。这座独特的建筑预计将成为城市天际线的新焦点，其璀璨光芒远至高速公路、桥梁及部分市中心区域均清晰可见，为游客提供独特的视觉体验。
据阿布扎比旅游主管部门证实，这座名为“阿布扎比Sphere”的巨型建筑，其外部LED屏幕——“Exosphere”，将确保在谢赫扎耶德路大段路段及阿布扎比岛的多个观景点都能被清晰观测到。该项目投资高达17亿美元，选址于亚斯购物中心与阿布扎比海洋世界之间。这也标志着，它将成为继美国拉斯维加斯之后，全球第二座、同时也是美国境外首座此类Sphere场馆。
