# ️ FBI提醒：OneCoin受害者赔偿申请截止在即

⚠️ FBI提醒：OneCoin受害者赔偿申请截止在即
美国联邦调查局（FBI）近日发布提醒，全球加密货币投资骗局OneCoin的受害者应在6月30日截止日期前，通过美国司法部（DOJ）设立的特别计划申请赔偿。
此项流程旨在为在2014年至2019年期间购买OneCoin并遭受直接经济损失的个人提供申请途径。受害者可通过邮寄、电子邮件或在线表格提交相关申请。
司法部同时指出，提交申请并不保证一定能获得经济赔偿。据统计，OneCoin骗局给全球投资者造成的损失总计已超过40亿美元。
