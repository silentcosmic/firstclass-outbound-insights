# 迪拜租金新政：弹性支付缓解居民压力

📣 迪拜租金新政：弹性支付缓解居民压力
迪拜土地局（DLD）近日推出“灵活租金”（Flexi Rent）新计划，旨在为居民提供更具弹性的租房支付选项。通过参与该计划的房地产公司，租客可选择按月、按季度或按年支付租金，从而有效减轻一次性大额支付带来的经济负担。
迪拜土地局租赁事务部主任哈立德·阿尔沙伊巴尼表示，此举旨在缓解居民经济压力，并支持社区的长期福祉。该计划首阶段已涵盖12家迪拜本地开发商，未来有望吸引更多企业加入。
