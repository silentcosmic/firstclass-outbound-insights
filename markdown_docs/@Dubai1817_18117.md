# 在柬华人汇旺血泪：从捐款到盼望“铲平”

🚨 在柬华人汇旺血泪：从捐款到盼望“铲平”
近日，在柬埔寨的华人商界因汇旺支付平台崩盘而遭受重创，引发广泛关注。一位受害者投稿，深刻剖析了自身从支持柬埔寨到彻底幻灭的心路历程，揭示了在当地营商的诸多困境与无奈。
据投稿人透露，许多当地商家并非“爱贪小便宜”才使用汇旺，而是由于市场环境所迫，客户坚持使用该平台付款，否则生意根本无法维系。他指出，汇旺在柬埔寨运营多年，不仅有“洪森侄子做靠山”的传闻，当地政府对此也并非不知情。令人愤慨的是，即便汇旺已被宣布违法，仍有具官方背景的公司低价收购用户存款，此举被视为公然的“抢劫”。
投稿人认为，柯小伟被遣返事件，被解读为对所有试图维权者的警告。他绝望地表示，如果血汗钱最终无法追回，离开柬埔寨时将全力劝阻国内同胞切勿前来，并直言柬埔寨并非“诈骗王国”，而是一个“流氓王国”。更令人震惊的是，这位曾为柬埔寨捐款支持其与泰国冲突的商人，如今唯一的盼望竟是“泰国快打过来，铲平柬埔寨”，显示出极度的失望与愤怒。
