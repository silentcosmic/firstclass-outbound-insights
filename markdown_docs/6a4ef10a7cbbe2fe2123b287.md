# 支付基础设施巨头Xendit与菲律宾Dragonpay深化融合，共筑区域数字支付生态

领先的印尼支付基础设施提供商Xendit近日宣布，已成功与菲律宾替代支付平台Dragonpay Corporation完成系统集成。
双方在一份联合声明中指出，此次深度集成是基于2021年启动的战略合作关系进一步发展而来。
这一举措将Dragonpay深受信赖的本地支付网络与Xendit日益扩张的区域基础设施、不断优化的支付平台无缝连接。
该声明显示，Dragonpay的现有商户将能够充分利用Xendit的综合平台能力，这包括超过100种支付方式、便捷的款项支付、跨境支付解决方案以及灵活的金融服务。
对于本地企业而言，这意味着拥有更多元的收款渠道、更高效的款项发放机制，能够有效简化支付运营，并将客户服务范围拓展至菲律宾境外，同时持续受益于Xendit在平台升级和安全保障方面的长期投入。
目前，Dragonpay已为905家商户提供支持，并与44家合作伙伴建立了协作关系，这些伙伴涵盖银行、非银行金融机构、电子钱包以及线下支付渠道服务商。
两家公司将携手合作，共同确保业务连续性、加速产品创新，并在Dragonpay于菲律宾市场奠定的坚实基础上持续发展。
此次合作声明正值菲律宾数字经济加速发展之际，其背后得益于菲律宾中央银行（Bangko Sentral ng Pilipinas）在全国范围内推广支付数字化的积极推动。
对于商户而言，Dragonpay深厚的本地市场根基与Xendit在东南亚地区的强大支付基础设施相结合，将有助于企业满足客户不断变化的支付偏好，并为未来的跨境业务增长做好充分准备。
这项正式发布基于双方始于2021年8月的合作关系，当时Xendit已对Dragonpay进行了战略投资。
Dragonpay成立于2010年，率先在菲律宾开创了替代性数字支付模式，使数百万菲律宾民众（包括那些没有银行账户或信用卡的群体）能够通过网上银行、电子钱包以及全国范围内的银行柜台、便利店和汇款点进行线上交易。
Xendit联合创始人兼首席执行官Moses Lo表示：“Dragonpay书写了菲律宾数字支付的第一个篇章。将Dragonpay融入Xendit，为菲律宾企业提供了两全其美的优势：既拥有Dragonpay值得信赖的本地网络，又可利用Xendit的区域基础设施、先进技术以及日益增强的跨境支付能力。”“我们的核心目标是帮助菲律宾商户把握更多机遇，服务更广泛的客户群体，并实现业务的跨境拓展。”
Dragonpay创始人Robertson Chiang则指出：“我们在2010年创立Dragonpay的初衷，是为了让每一位菲律宾人，无论是否拥有银行账户，都能参与到数字经济中来。”“作为Xendit的一部分，我们的商户和合作伙伴将继续享有同样值得信赖的本地服务，现在更获得了更强大的技术平台、更广泛的区域能力、持续的产品创新以及坚固的安全保障。”“这是我们从合作伊始就运作良好的伙伴关系所迈出的自然而然的下一步。”
Dragonpay的并入，是Xendit持续构建东南亚统一支付网络策略的重要组成部分。此前，Xendit已于2025年全面收购了马来西亚的Payex，并于2024年成功拓展至泰国市场。
如今，Xendit的业务已遍及印度尼西亚、菲律宾、马来西亚、泰国、越南、新加坡和中国香港等多个区域。
