# 警惕：东南亚华人团伙涉嫌多起跨境诈骗与黑产

🚨 警惕：东南亚华人团伙涉嫌多起跨境诈骗与黑产
近期市场消息指出，一名代号“阿三”的华人陈君法及其团伙，涉嫌在东南亚多国从事一系列严重跨境犯罪活动，受害者多为在当地发展的中国同胞。
据披露，陈君法在国内曾涉嫌高利贷、组织犯罪乃至命案。案发后，其携妻子何婉霞潜逃至东南亚。在老挝，该团伙被指控利用人民币换汇业务从事黑钱交易，导致多名同胞的银行卡被冻结。随后在柬埔寨，其团队成员张国龙、曾海宾、黄俊豪等人被控盗取同胞社交媒体账号，并与合伙人合作经商期间，涉嫌飞单（卷款）数百万美元。团队内部曾有员工试图退还款项，但被陈君法私自扣押。
更甚者，陈君法被指控通过谎报财务困境，向合伙人骗取数十万美元借款，同时却私下将高达1500万人民币转移回国。此外，在缅甸，该团伙更涉嫌诱骗同胞至诈骗园区，进行非法拘禁甚至贩卖。
目前，相关证据已提交国内反诈部门，提醒在东南亚的华人社群务必提高警惕，防范此类欺诈行为。
