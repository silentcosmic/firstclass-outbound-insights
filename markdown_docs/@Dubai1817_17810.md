# 印尼重拳打击跨境电诈：92名中国籍嫌犯遭终身禁入

🚨 印尼重拳打击跨境电诈：92名中国籍嫌犯遭终身禁入
据印尼官方证实，92名涉嫌参与跨境网络赌博及投资诈骗活动的中国籍人员，已于7月5日被集体遣返回中国，并被列入印尼终身禁止入境名单。此举是印尼近期持续加大打击跨境电信网络诈骗犯罪力度的重要行动之一。
这批涉案人员于今年5月在印尼巴淡岛一处公寓内被查获，该公寓长期被怀疑用于实施电信诈骗及网络赌博等违法犯罪活动。此次遣返行动在中印两国执法合作机制下进行，中国公安机关专门派出工作组赴印尼执行押解任务。涉案人员已统一搭乘中国南方航空航班返回广州，将接受进一步调查处理。
印尼移民部门强调，将依法严惩任何利用印尼境内从事跨境违法犯罪活动的外国公民，包括遣返并永久禁止其再次入境。事实上，印尼近期已多次开展大规模打击行动。今年5月，印尼警方在全国范围内开展集中收网行动，两天内抓获近580名涉嫌电信诈骗人员；6月底，又在巴厘岛查获103名涉嫌网络犯罪的台湾地区人员，持续彰显其高压执法态势。
更进一步，印尼政府正考虑重新评估部分国家和地区的免签入境政策，旨在从源头上遏制不法分子借旅游或商务之名入境从事跨境违法犯罪活动，以维护国家安全和社会秩序。
