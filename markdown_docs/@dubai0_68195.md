# 美伊原则同意建立军事沟通渠道

🚨 美伊原则同意建立军事沟通渠道
美国参议员JD万斯近日透露，在瑞士举行的谈判中，美伊双方取得了一项重要突破：原则上同意建立直接军事沟通渠道，以有效防止未来局势升级。
万斯指出，此举旨在伊朗方面建立一个冲突降级沟通机制。据他称，伊朗方面已表示愿意派遣革命卫队人员前往多哈，与美国中央司令部代表会晤，共同解决潜在争端。
