# 阿联酋航空A380机翼现裂纹：部分飞机紧急检查

🚨 阿联酋航空A380机翼现裂纹：部分飞机紧急检查
迪拜当地时间6月24日，阿联酋航空宣布，将立即对其部分空客A380客机启动紧急检查。此前，空客公司报告称，在部分A380机型的机翼关键部件上发现了裂纹。
据空客周二发布的消息，全球共有16架A380飞机需进行检查，其中5架被列为需立即处理。这些裂纹主要在阿联酋航空与澳洲航空运营的飞机上被发现。在这批待检飞机中，有15架隶属阿联酋航空机队，另1架由澳洲航空运营。
