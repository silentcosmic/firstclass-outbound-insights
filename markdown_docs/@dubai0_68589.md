# 迪拜十年规划：体育经济规模将翻倍至183亿迪拉姆

⚡ 迪拜十年规划：体育经济规模将翻倍至183亿迪拉姆
迪拜近日宣布一项雄心勃勃的计划，旨在未来十年内，将体育产业对经济的贡献提升近一倍。根据规划，到2033年，该行业每年对国内生产总值的贡献目标将达到183亿迪拉姆。
这项战略目标是《迪拜体育行业2033战略规划》的核心内容。该规划明确将体育产业定位为经济多元化的关键支柱，旨在协同推动旅游、投资、科技、媒体及可持续发展等多个领域。
