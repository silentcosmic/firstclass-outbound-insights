# 全球产业洞察 | 宇树科技获科创板IPO注册，AI算力市场疑云：Meta行动非过剩，乃商业模式成熟信号

企业动态与市场表现
奥飞娱乐近期发布2026年半年度业绩预告，预计上半年归属于上市公司股东的净利润在1.5亿元至1.9亿元之间，较上年同期将实现305.30%至413.38%的显著增长。
雀巢公司表示，面对美国消费者对产品价格敏感度提升，纷纷转向大包装或经济型包装、避开中等规格产品的市场趋势，公司正积极采取措施控制产品定价，以适应市场变化。
资本市场与科技趋势
7月2日，A股科技股出现大幅下挫，半导体、算力硬件以及存储芯片等板块跌幅尤为明显。以科技股集中的“双创板”为例，创业板指和科创综指当日分别下跌5.71%和5.64%。市场普遍将此次下跌归因于Meta对外出售算力的消息，并将其解读为算力市场过剩的信号。然而，多位业内资深人士指出，这实际上是对消息的误读。Meta出售算力并非意味着AI资本投入的终结，而是预示着AI基础设施商业模式正迈向成熟阶段。
本刊获悉，北京君正近日在机构调研中透露，DRAM价格涨势显著，国内客户自一季度开始上调价格，二季度仍在持续，同时海外客户也陆续调整价格。公司预计二季度收入和毛利率将延续增长态势；三季度价格有望继续上调，而四季度的价格政策需待三季度末才能确定，但目前判断四季度供应仍趋紧张，存在进一步调价的可能性。鉴于部分客户以SRAM产品替代DRAM方案，部分SRAM芯片在二季度也出现了价格上涨。NOR Flash在一、二季度亦有涨价，但整体涨幅小于DRAM。尽管NOR Flash产能同样趋紧，但情况远好于DRAM。一季度，公司Flash产品线实现收入大幅增长，其中销量增长贡献显著。三季度价格将根据具体产品和客户情况确定，并非所有产品都会调价。由于公司存储芯片价格持续上涨，预计各季度毛利率将环比提升。
7月2日，行业内首个算力中心SST（固态变压器）智能直流供电商业化项目已在秦淮数据环首都·桑园云计算产业基地正式投运。这标志着算力中心供配电体系实现了从“工频”到“高频”、从“交流”到“直流”的范式突破，对未来数据中心能源效率具有里程碑意义。
消费与零售市场洞察
抖音电商近日在北京举办“侵权仿冒治理开放日”活动，正式推出“肖像保护功能”。该平台将运用模型识别技术，结合授权信息核验及相似内容传播阻断等机制，主动识别并拦截AI仿冒、素材盗用等交易环节中的侵权行为。据抖音电商数据显示，截至2026年6月，平台已累计处置侵权仿冒达人账号逾1.2万个，下架仿冒带货商品超4.8万件，处置仿冒带货内容逾26.1万条。与此同时，抖音电商的“肖像保护功能”已主动拦截侵权账号9.2万个。
就媒体此前报道的一汽奥迪中方一把手换帅传闻，一汽奥迪方面知情人士回应称，该消息不实，郭永锋目前仍担任一汽奥迪销售有限责任公司执行副总经理。自2025年9月1日起，郭永锋全面负责一汽奥迪在华所有国产及进口车的销售、售后服务、网络及品牌营销工作。
小红书近日发布了《世界杯小组赛观赛报告》。报告显示，在小组赛期间，平台直播观看时长达到开赛前的90倍，直播观看人次达到开赛前的40倍，世界杯及足球相关内容曝光量高达282亿。开赛至今，足球内容的日均发布量环比5月增长了10倍。
7月2日，据报道，香奈儿宣布将收购总部位于巴黎的男装衬衫制造商Charvet。具体交易条款未予披露。Charvet是一家成立于1838年的法国高端衬衫制造商，主营高级衬衣与领带定制业务，在巴黎旺多姆广场设有专卖店。
战略调整与融资动向
据多位市场知情人士透露，理想汽车即将围绕产品决策流程进行新一轮组织架构调整，计划将产品部的部分关键职能拆分，并入研发部门。从现有的群组关系来看，理想汽车产品部由范皓宇负责，涵盖电动本体、空间智能、自动驾驶终端产品、交互设计、平台运营以及App与官网等业务。熟悉此次调整的人士表示，调整主要针对电动本体和自动驾驶等核心产品定义团队。其中，产品部的电动本体定义团队已明确拆分并入由刘立国领导的整车研发部门；自动驾驶终端产品团队将整合进由詹锟领导的基座模型研发团队。同时，部分与用户体验相关产品人员亦可能进入刘杰的产品线。
德国国防科技公司Quantum Systems周四宣布成功筹集12亿美元新资金，此轮融资使这家无人机制造商的估值达到约80亿美元。这是迄今为止欧洲国防科技公司获得的最大规模私募融资之一，公司估值较一年前增长了约八倍。这家总部位于慕尼黑的公司表示，本轮融资得到了国际投资者的鼎力支持。
宏观经济数据
据工业和信息化部消息，今年前5个月，我国软件和信息技术服务业运行态势良好，软件业务收入同比增长10.3%。数据显示，今年前5个月，我国软件业务收入达62451亿元，同比增长10.3%；软件业利润总额同比增长2.2%；软件业务出口额达到276.5亿美元，同比增长12.8%。
