# 中国驻阿联酋新大使履新，深化中阿战略伙伴关系

🤝 中国驻阿联酋新大使履新，深化中阿战略伙伴关系
中国新任驻阿联酋大使曾继新近日到任履新，并首次拜会阿联酋联邦最高委员会成员、拉斯海玛酋长苏欧德殿下。双方就中阿两国友好关系及务实交流合作进行了深入探讨。
曾大使指出，在两国元首的战略引领下，中阿关系正处于历史最好时期，各领域交流合作日益紧密，已成为中国与地区国家关系的典范。他强调，拉斯海玛酋长国作为阿联酋的重要组成部分，在中阿经贸合作与人文交流中发挥着积极作用。中方愿持续深化双边交流，为中阿全面战略伙伴关系注入更多务实内涵。
苏欧德酋长对曾大使的到任表示热烈欢迎，并高度赞赏中国的发展成就及其为地区和平稳定与繁荣发展所作出的贡献。他表示，拉斯海玛酋长国高度重视对华关系，期待在经贸、文化、旅游等多个领域与中方深化交流合作，共同造福两国人民。
