# 印尼警方突袭：15名中越籍涉案人员被捕

🚨 印尼警方突袭：15名中越籍涉案人员被捕
7月13日，印度尼西亚泗水一级特别移民检查站联合西多阿乔警察局发布通报，成功破获一起特大跨国信息犯罪案件。此次联合行动中，执法部门累计抓获15名中国及越南籍涉案人员。
调查显示，该犯罪团伙主要通过发布虚假招聘信息，诱骗当地民众提供身份信息。随后，他们利用这些被盗身份信息批量办理银行卡，并将其用于洗钱和网络诈骗的资金周转。
