# 东南亚“仙人跳”升级：专盯出海华人，警惕人身安全

🚨 东南亚“仙人跳”升级：专盯出海华人，警惕人身安全
近期，有市场观察指出，针对中国公民的境外“仙人跳”敲诈陷阱日益猖獗。犯罪团伙常以交友、约会为名，诱骗目标至偏僻或私密场所，伺机作案。
一旦受害者落入圈套，将面临证件扣押、行动受限，并被勒索高额“赔偿费”。若拒不配合，不法分子甚至可能威胁人身安全，乃至将受害者转卖至当地诈骗园区，后果不堪设想。该类犯罪模式在柬埔寨、泰国、老挝等东南亚国家尤为常见，专向不熟悉当地环境的中国公民下手。
对此，提醒出海商务人士及旅居海外的中国公民，务必提高警惕，切勿轻信陌生搭讪，避免私下赴约或前往私密场所，以最大程度保障自身财产及人身安全。
