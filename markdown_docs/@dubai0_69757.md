# 阿联酋船只遭袭：印度籍海员遇难受伤

🚨 阿联酋船只遭袭：印度籍海员遇难受伤
印度外交部近日证实，在阿联酋船只遭遇伊朗袭击的事件中，共有10名印度公民受伤，其中1人不幸遇难。
据印度外交部声明，涉事船只包括MT Al Bahiyah号和MT Mombasa号，两船共载有46名海员，其中30人持有印度国籍。
具体而言，MT Al Bahiyah号货轮上共有12名印度公民，此次袭击导致该船上1名印度籍海员遇难，另有1人受伤。
