# 华为Pura 90s系列全球首发，深度革新移动影像智能体验

周二，华为Pura 90s系列在全球范围内隆重发布，标志着移动美学与科技发展迈入全新篇章。该系列秉持“此刻，为你定格”的品牌理念，将尖端影像创新与独特设计语言深度融合，旨在赋予用户轻松捕捉每个真实瞬间的能力。
华为Pura 90s系列通过硬件、软件、芯片与人工智能的精密协同，超越了传统智能手机摄影的界限，重新定义了移动影像的可能性。
Pura 90s系列的设计哲学核心在于对“色彩韵律”的深刻诠释。全系产品均采用直边显示屏设计，呈现出一种内敛而深邃的标志性优雅美学。
华为Pura 90s Pro Max提供了四款配色：霞光金，寓意光芒初现；丹霞橙，象征追逐余晖；拂晓紫，唤醒绽放的暮光；以及曜石黑，呼应沉静的宇宙。该系列还首创了业界领先的双色渐变金属中框技术，带来独特的层次感和质感体验。Pura 90s Pro则从清新的假日饮品中汲取灵感，推出了更为活泼的配色方案，包括芭乐汽水、橘子汽水、椰风白和桑葚黑，以纯粹精致的工艺捕捉假日的活力色调。
极致影像：重塑移动成像新标杆
华为Pura 90s系列在影像领域实现了三大核心维度的突破性飞跃：长焦微距性能、主摄像头表现以及色彩准确性，全面提升了旗舰级移动影像的用户体验。
华为Pura 90s Pro Max引入了业界领先的2亿像素超大传感器长焦摄像头，以卓越的清晰度和稳定性拓展了远距离成像的边界。同时，该机型首次搭载了芯片级2亿像素RAW实时处理技术，支持清晰的20倍长焦视频录制。作为补充，Pura 90s Pro Max达到了业界顶级的CIPA 7.0防抖等级，即使在严苛条件下也能有效减少手持拍摄模糊。在微距探索方面，Pura 90s Pro支持先进的微距拍摄功能，最近对焦距离可达5厘米，使用户能够以非凡的清晰度捕捉细致入微的纹理。为精准捕捉光影，Pura 90s Pro Max配备了超聚光HDR摄像头，采用大型1/1.28英寸RYYB传感器，确保逆光人像照片依然细节丰富、曝光准确。在“真彩影像引擎2.0”的驱动下，华为Pura 90s系列实现了色彩准确性43%的显著提升，以及色域处理能力34%的扩展，使每一个瞬间都能呈现其真实、生动的色彩。
华为Pura 90s系列的图库系统深度融合了AI能力，将高质量的图像编辑转化为直观便捷的体验。其中，升级版AI构图功能作为智能取景助手，能够即时识别肖像、建筑和风景等主要拍摄对象，并提供实时的拍摄位置指导。
AI去眩光功能有效消除了恼人的窗户反射、玻璃反光和屏幕干扰，无缝还原玻璃表面后的真实肤色与隐藏细节。AI随心抠功能则开启了创新的编辑潜力，用户可以对图像中的元素进行隔离、移动或复制，而系统则会智能地重建背景并协调光线，以实现无缝的编辑效果。
华为Pura 90s Pro Max的前置显示屏搭载了抗反射、防刮的昆仑玻璃，可将屏幕反射率降低高达70%，确保在阳光直射下依然保持清晰可见。同时，它还能有效减少日常刮痕和指纹，使显示屏长久保持 pristine 状态，并提供流畅的触控体验。
华为Pura 90s系列集成了全新的系统级AI助手Celia。
Celia配备了直观的知识问答和内容起草工具，为日常操作提供语音控制，并提供全面的旅行支持，涵盖Petal Map路线规划、本地查询以及打车服务等。
此外，华为Pura 90s系列配备了6000mAh大容量电池，提供持久续航。Pura 90s Pro支持66W有线超级快充，而Pura 90s Pro Max则提供高达100W的有线超级快充。
华为此次在曼谷的发布活动，点燃了全场景智慧生活的愿景，开启了智能生活的新篇章。
