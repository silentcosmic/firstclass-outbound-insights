# 米高梅迪拜度假村：官宣2028年Q3开业

📣 米高梅迪拜度假村：官宣2028年Q3开业
备受瞩目的迪拜米高梅大型度假村项目近日公布了其最新进展及开业时间。据官方消息，该项目正按计划稳步推进，预计将于2028年第三季度正式投入运营。
米高梅方面再次确认了这一时间表，并表示项目的建设工作正朝着既定的开业目标持续推进，未来将为迪拜高端旅游市场注入新活力。
