# 新加坡职场AI深度观察：员工接纳度高，核心应用面临落地挑战

最新数据显示，新加坡职场员工对人工智能（AI）的怀疑态度显著低于全球平均水平，仅有29%的受访者对此表示质疑，远低于全球37%的平均值，更是与美国、英国和法国高达53%的比例形成鲜明对比。然而，这种开放的态度并未完全转化为日常实践：仅有6%的员工将AI视为其核心工作流程的一部分，这一比例仅为全球平均水平11%的一半。
在AI试点项目中，新加坡的落地挑战也浮出水面。31%的受访者坦言曾经历失败案例。其中，高达40%的受访者指出AI生成内容过于泛化，缺乏针对性，此比例高于全球30%的平均水平。此外，38%的员工对AI的输出结果信任度不足，而30%的受访者认为AI生成的内容未能充分融入业务背景，这两个数字也分别高于全球28%和22%的平均值。
由此可见，相较于员工对新技术的尝试意愿，AI工具能否产出准确、可信并与业务需求高度匹配的结果，才是决定其能否在日常工作中获得广泛采纳的关键。特别是在复杂的企业环境中，即使通用型AI工具能高效完成文本摘要、信息搜索或内容生成等基础任务，它们往往难以有效处理内部专属数据、深度行业知识及特有的工作流程。
一项针对全球500余名已从初步试用转向深度日常使用AI的受访者的分析揭示了成功的路径。这些高效用户普遍获得了针对其特定岗位量身定制的AI技能培训，AI工具也已深度融入他们现有的工作流程，同时辅以明确的数据安全协议。企业在引入AI工具后，能否同步优化培训体系、数据管理策略和工作流程，成为提升AI实际使用频率的决定性因素。
新加坡官方数据也印证了这一趋势。新加坡人力部（Ministry of Manpower）于今年4月发布的企业AI采纳调查显示，在雇员规模至少10人的私营企业中，高达71.5%尚未引入AI技术。而在已开始探索AI的28.5%企业中，仅有3.8%将其深度融合至核心业务流程，另有7.4%仍处于规划阶段，6%则停留在试点阶段。
尽管AI的采用范围目前仍相对有限，但已部署AI的企业中，有70.7%明确表示员工生产力得到了显著提升。尤其重要的是，企业能否从AI中获得实实在在的收益，与其自身的规模和数字化成熟度紧密相关。大型企业的AI采采纳率高达76.4%，而员工少于25人的中小型企业，其采纳率仅为23.9%。
在企业层面，成本与内部能力依然是扩大AI应用的主要瓶颈。44.9%的受访企业提及高昂的实施成本，42.4%则面临内部专业人才的匮乏。对于中小企业而言，战略规划的不足和对AI信任度的欠缺构成额外挑战；而大型企业则更多地受限于复杂的系统整合需求和严格的数据安全规范。
综合上述职场调查与新加坡人力部的官方数据，一个清晰的差距浮现：员工对AI的开放态度，并不必然转化为高频的日常使用。企业是否具备健全的数据基础设施、充足的人才储备、完善的安全机制以及适配的业务流程，才是决定AI能否真正成为稳定高效生产工具的核心要素。
因此，对新加坡而言，提升职场AI使用效率的重心将不再仅仅是提供工具，而是转向如何切实改善AI的实际应用效果。企业能否确保AI的输出与特定岗位需求和具体业务流程实现无缝对接，将直接决定员工的积极态度能否最终转化为持久的生产力飞跃。
