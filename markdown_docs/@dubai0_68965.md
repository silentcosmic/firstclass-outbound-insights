# ️ 中国使馆警示：在越电诈多为熟人作案

⚠️ 中国使馆警示：在越电诈多为熟人作案
在越南向中国移交19名跨境电信网络诈骗嫌疑人之后，中国驻越南大使馆近日发布紧急提醒，呼吁在越中国公民高度警惕电信诈骗风险。使馆特别指出，当前在越涉中国公民的电诈案件，多以“熟人作案”形式出现，犯罪分子往往利用信任关系进行诈骗。因此，使馆敦促公民务必防范“天上掉馅饼”的诱惑，并谨慎交友，避免落入精心设计的骗局。
