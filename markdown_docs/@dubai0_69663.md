# 马达加斯加：中国商人遭冒充高官抢劫，总统府急辟谣

🚨 马达加斯加：中国商人遭冒充高官抢劫，总统府急辟谣
近日，马达加斯加首都塔那那利佛发生一起性质恶劣的入室抢劫案，一名当地经商的中国籍公民成为受害者。7月9日，二十余名蒙面持枪劫匪冒充司法部官员，谎称持有国家重建主席的专项搜查指令，闯入该中国商人的住宅，抢走了价值超过2亿阿里亚里（约4.7万美元）的现金、外币及各类珠宝首饰。
据披露，劫匪在未出示任何合法搜查文书的情况下，控制了房屋并大肆搜刮财物。他们一度安抚屋主，声称财物事后会全额返还，并编造前往总统府汇报行动的谎言，同时威胁屋主配合问询，否则将强制拘传。屋主在心存戒备下，暗中拍摄了现场画面与对话录音，成为案件的关键证据。劫匪甚至试图误导受害者，称总统指令权限高于中国大使馆，以切断其求助念头。
案发数小时内，马达加斯加总统府迅速通过官方通讯负责人发布声明，公开否认曾发起此次搜查行动。国家重建主席米夏埃尔·兰德里亚尼纳上校明确表示，从未授权任何个人或团体以其名义进行入户搜查。官方警示，任何假借总统身份实施非法行动者均属冒充公职人员的嫌犯，并要求全国执法机关立即展开抓捕。总统府强调，总统的行政工作均按法定流程推进，不存在私下派遣不明人员入室搜查的情况，此次事件是不法分子精心策划的抢劫犯罪。
媒体指出，此类冒充新政府工作人员上门实施犯罪的模式并非孤例，自新一届政府上台以来，当地已发生多起类似事件，扰乱了民众对正规执法行动的判断。得益于屋主提供的视频证据，两名主要组织者身份已浮出水面，其中一人被指是在职宪兵上尉，另一人自称是国家重建主席的专属顾问。目前，当地警方正深入调查，核实二人公职身份并排查背后是否存在完整犯罪团伙。
