# 腾讯拟成Manus最大股东：中国监管指令引发跨境科技并购新变局

科技巨头腾讯正与人工智能初创公司Manus展开深入谈判，有望成为其最大单一股东。
此项潜在交易背景复杂，此前，北京方面已向Meta发出指令，要求其剥离对这家AI初创公司20亿美元的收购交易。
目前，腾讯正与Manus的原有投资者共同制定回购计划，旨在从Meta手中购回该公司股权。
消息人士指出，由真格基金（ZhenFund）和HSG等机构组成的投资联合体，目标是以至少20亿美元的估值完成对Manus的回购。
Manus的核心业务在于开发智能代理AI工具，这类工具能够以极少的人工干预自主完成复杂任务。这家初创公司已于去年将其核心运营实体从中国迁至新加坡。
Meta曾于去年12月宣布收购Manus，旨在加速其在人工智能技术领域的能力布局。
今年4月，中国监管机构启动了对这项交易的审查，以评估其是否符合现行的投资法规。
此次监管指令，标志着中国在挑战涉及非中国注册公司的跨境科技交易方面的又一备受瞩目案例。
上个月，Meta已完成与Manus的内部运营剥离，并停止了双方之间的数据共享。
此前，国内业界曾一度对Manus寄予厚望，尤其在其推出被业界誉为“全球首个通用AI代理”之后，将其视为中国AI领域继DeepSeek之后的又一重要潜力新星。
