# 日本金主曼谷落网：跨境诈骗曝“AI深伪”新威胁

🚨 日本金主曼谷落网：跨境诈骗曝“AI深伪”新威胁
近日，日本与泰国警方联合侦破一起特大跨国电信诈骗案。日本籍男子佐佐木勇介（38岁），被指控为诈骗集团核心出资人，已在曼谷通罗区一处豪华公寓内被捕，并被遣返回日本接受司法调查。
据警方披露，佐佐木长期以“数字游民”身份在泰国居留，以此规避追查。他被锁定为柬泰边境波贝地区诈骗园区的关键资金提供者，负责园区运营开支，并向招募中介支付高额佣金，诱骗日本公民前往从事诈骗活动，每成功招募一人中介可获数百万日元报酬。
调查发现，该诈骗集团组织严密，内部设有多个业务小组，冒充银行、警察或检察官，通过电话和视频通话实施诈骗。尤其值得关注的是，诈骗园区内设有“AI房间”，利用人工智能换脸技术，将诈骗人员伪装成日本执法人员，对受害人进行恐吓施压，诱导转账汇款。
警方初步估算，该集团已累计骗取受害人资金高达数十亿日元。作为主要投资人，佐佐木可从中获取30%至40%的收益分成，其个人月收入最高可达约1亿日元（约合人民币500万元）。日本警方正持续追查该诈骗网络的其他成员及资金流向，并加强国际执法合作。
