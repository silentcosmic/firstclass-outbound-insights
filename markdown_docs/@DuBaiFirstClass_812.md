# Amity获1亿美元D轮融资，新加坡设AI研发中心

⚡ Amity获1亿美元D轮融资，新加坡设AI研发中心
企业级AI公司Amity近日宣布，成功完成1亿美元D轮融资。在获得这笔重要资金后，该公司随即在新加坡设立了东南亚区域总部及AI研发中心。此举旨在深度布局亚洲市场，并专注于开发企业级代理式AI及各类垂直模型，以期加速实现营收增长目标。
