# Crypto.com 获 Citadel Securities 4亿美元战略投资，估值达200亿美元

出海 · 头等舱 获悉，知名加密货币平台 Crypto.com 近日已获得全球顶级做市商 Citadel Securities （城堡证券）4 亿美元的战略投资。
此轮交易完成后，Crypto.com 的估值已达到 200 亿美元。值得注意的是，这是该平台在十年运营历程中首次引入机构投资者。
这笔资金将主要用于支持 Crypto.com 向更广泛的资产类别拓展，包括代币化证券和衍生品市场。
Crypto.com 方面表示，此次融资不仅将助力数字市场与传统金融体系间的深度融合，还将进一步推动一个全天候不间断金融生态的构建。
此次合作的达成，正值传统金融机构愈发积极地拥抱区块链基础设施和数字资产之际，旨在显著提升资本市场的整体效率。
Crypto.com 联合创始人兼首席执行官 Kris Marszalek 对此表示：“随着加密货币日益成为全球金融体系的核心基础设施，我们面前的市场机遇是巨大的。”
Marszalek 补充指出，经过过去十年在监管合规和技术基础设施方面的持续投入与建设，Crypto.com 已为把握跨资产类别的增长机遇做好了充分准备。
Citadel Securities 总裁 Jim Esposito 则强调了传统金融与数字资产领域日益加深的融合趋势。他指出：“Crypto.com 已奠定了坚实的基础，能够支持数字资产市场的持续机构化进程。我们非常高兴能与 Crypto.com 团队携手合作，共同塑造未来的资本市场。”
