# 周星驰新片《功夫女足》8月6日登陆柬埔寨影院

🎬 周星驰新片《功夫女足》8月6日登陆柬埔寨影院
由中国著名影星周星驰执导的电影《功夫女足》定于8月6日正式在柬埔寨各大影院上映。此举标志着华语电影在东南亚市场的新一轮拓展。
为满足当地多元观众的观影需求，影片将同步提供中文原声版和柬语配音版，旨在覆盖更广泛的观影群体，提升市场渗透率。
