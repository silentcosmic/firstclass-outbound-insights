# 日媒痛批柬埔寨：已成“跨国诈骗帝国”

🚨 日媒痛批柬埔寨：已成“跨国诈骗帝国”
日本媒体近日发出严厉指控，称柬埔寨已从此前被戏称为“诈骗之国（Scambodia）”进一步升级为“跨国诈骗帝国”。
报道指出，大量日本公民被诱骗至柬埔寨，他们往往受到高薪工作广告的吸引，承诺“两天一夜轻松赚取50万日元”。然而，抵达后却发现自己被非法拘禁，并被迫卷入电信诈骗（呼叫中心）犯罪活动，成为跨国犯罪网络的帮凶。
当前，日本正面临史上最严重的电信诈骗问题，而柬埔寨被认为是这些犯罪集团最主要的藏身地之一。这种趋势不仅加剧了日本国内的诈骗危机，也凸显了东南亚地区在打击跨国犯罪方面的严峻挑战。
