# 武汉破获1600万“电费刷单”洗钱案，4人被捕

🚨 武汉破获1600万“电费刷单”洗钱案，4人被捕
湖北武汉警方近日成功破获一起特大跨国“电费刷单”洗钱案，斩断了一条涉案金额超1600万元的黑色产业链，并刑拘了方某、陆某朋等4名核心成员。
这起案件历时数月、横跨四省追踪。据警方调查，该犯罪团伙利用民生缴费渠道，诱骗受害人将资金充值到国内正常电费账户。随后，中介人员以折扣价从户主手中兑换现金，再将赃款转换为虚拟货币转移至境外，以此完成洗钱。
该团伙在2025年1月至9月间，通过此手法清洗赃款逾1600万元，涉及数千个社交群组。此案最初线索来源于2025年8月一名市民遭遇的56万元网络诈骗案，其被骗资金流向异常电费账户，从而牵出整个犯罪网络。目前，案件正在进一步深挖中。
