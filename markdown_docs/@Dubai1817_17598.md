# 上海婚介诈骗案：50元“演员”冒充相亲对象

🚨 上海婚介诈骗案：50元“演员”冒充相亲对象
上海普陀警方近日成功捣毁一个特大婚介诈骗团伙，抓获14名犯罪嫌疑人。该团伙通过开设实体门店并利用网络平台发布相亲广告，诱导客户签订高额服务合同。
警方调查显示，该团伙根据客户的家庭条件和收入水平临时定价，收费从数千元至十几万元不等。签约后，公司会安排大量低质量相亲，快速消耗合同约定次数，并以“结婚返款”或“介绍优质对象”等名义诱导客户持续付费。所谓的“情感辅导课程”也仅是廉价的网络视频。当客户要求退款时，公司则以扣除高额税费为由拖延，甚至让客户自行起诉。
警方查明，该团伙以每次50元的价格雇佣“婚托”冒充相亲对象，专门配合演出，消耗客户的相亲机会。客户花费数万元甚至十几万元，却只见到出场费仅50元的“演员”。该公司经营期间，没有一名客户通过其介绍成功恋爱或获得结婚返款。
初步统计，该团伙一年经营流水超过千万元，目前已核实受害者十余人，涉案金额接近百万元。目前，14名犯罪嫌疑人已因涉嫌诈骗罪被上海普陀警方依法采取刑事强制措施，案件正在进一步侦办中。
