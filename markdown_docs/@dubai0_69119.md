# 上海：2亿虚拟币非法换汇案宣判，团伙获刑

🚨 上海：2亿虚拟币非法换汇案宣判，团伙获刑
上海静安区法院近期宣判了一起涉案金额逾2亿元人民币的非法换汇案。一个打着“私人银行”旗号的犯罪团伙，利用虚拟货币作为中介，将境内人民币非法兑换成境外英镑、欧元等外币。目前，该案已有5名主犯被判处有期徒刑，并处高额罚金。
案件背景正值暑期出境旅游、留学高峰期，部分有大额换汇需求的人士为规避每年5万美元的便利化购汇额度限制，转向此类非法渠道。该团伙操作手法隐蔽：客户将人民币分批转入指定账户，团伙随即用这些资金购买虚拟货币并存入公司钱包，最终通过境外虚拟币承兑商兑换成外币。
法院审理强调，本案的关键在于是否实质完成了人民币与外币之间的非法兑换，虚拟币仅为“搭桥工具”，不能改变其非法买卖外汇的本质。主犯高某因负责对接留学移民中介、招揽客户及统筹整个换汇流程，被判处最重刑罚。
司法机关提醒，参与此类非法换汇风险巨大。客户不仅面临资金被截留或诈骗的风险，银行卡也可能因大额可疑交易被公安机关冻结。此外，即便仅是出借银行卡协助收款，也可能被认定为非法买卖外汇的共犯，面临刑事责任或行政处罚。
