# 迪拜哈利法塔艺术征集回归：你的作品或将点亮世界之巅

📣 迪拜哈利法塔艺术征集回归：你的作品或将点亮世界之巅
迪拜地标哈利法塔的艺术作品征集活动再度启动。地产巨头伊玛尔集团（Emaar）面向全球艺术家、设计师及创意人士发出邀请，征集能够在哈利法塔标志性外墙上展示的投影设计作品。
此次活动为世界各地的创作者提供了一个在迪拜最高建筑上展示艺术的独特机会。有意参与的艺术家需准备好视频和创意说明，并在指定截止日期前将全部材料发送至 opencall@emaar.ae。
