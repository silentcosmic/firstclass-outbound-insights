# ️ 缅甸帕安市酒店疑成诈骗团伙新据点

⚠️ 缅甸帕安市酒店疑成诈骗团伙新据点
近年来，缅甸克伦邦帕安市部分酒店疑成为中国籍网络诈骗团伙的活跃地点，引发了当地社会对治安状况的担忧。有消息指出，自2023年起，部分原在边境诈骗园区的团伙已逐步转移至帕安市，并在当地多家酒店长期活动。
据市场消息，这些人员主要集中在BGF/KNA旗下的瑞各宾酒店（Zwe Ka Bin Hotel）及坎达乌酒店（Kan Thar Yar Hotel）等场所。目前估计，常驻帕安市的中国籍相关人员数量约在300至500人之间。
