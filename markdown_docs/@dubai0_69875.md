# 迪拜推出五年多次入境签：不限国籍，年度可停留半年！

📣 迪拜推出五年多次入境签：不限国籍，年度可停留半年！
迪拜身份与外国人事务总局（GDRFA）近日发布重磅消息，正式推出一项面向全球游客的五年期多次入境签证。此举旨在进一步提升迪拜的国际吸引力，该签证不设国籍限制，且无需本地担保或接待方。
根据新规，持签者每次入境可享受长达90天的停留期，并可申请额外90天延期。值得关注的是，全年累计在阿联酋境内的停留时间上限为180天，这意味着游客每年最多可在迪拜停留半年，且可灵活分多次往返。
申请所需材料包括近期个人照片、有效期不低于六个月的护照复印件、至少4,000美元的银行对账单、有效健康保险证明及往返机票。申请人可通过阿联酋联邦移民局（ICP）官网、GDRFA在线平台，或经授权的旅行社及签证服务机构办理。
