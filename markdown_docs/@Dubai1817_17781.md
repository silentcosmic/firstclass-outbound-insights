# 曼谷车内离奇双亡：引擎运转，死因待查

🔍 曼谷车内离奇双亡：引擎运转，死因待查
泰国曼谷警方近日通报，7月4日，一辆停放在曼谷一处巷内的轿车内发现一对男女死亡。死者分别为一名45岁男子和一名27岁女子，被发现时均已无生命体征。
现场勘查显示，涉事车辆引擎仍处于启动状态，车门紧闭。警方初步判断两人生前曾发生性行为，现场还发现多款壮阳药物，且男子遗体仍戴有安全套。法医初步尸检结果为心肺骤停，警方推测死因可能与车辆密闭空间内引擎长时间运转导致一氧化碳中毒有关，亦不排除服用壮阳药物后的不良反应。
目前，两具遗体已送往医院进行详细尸检，双方家属已完成认领并处理后事。案件仍在进一步调查中，具体死因有待最终确认。
