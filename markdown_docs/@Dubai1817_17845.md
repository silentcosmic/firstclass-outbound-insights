# 斯里兰卡监狱骚乱升级：4人死亡，35人受伤

🚨 斯里兰卡监狱骚乱升级：4人死亡，35人受伤
斯里兰卡尼甘布（Negombo）监狱日前发生严重骚乱，造成4人死亡，其中包括3名监狱工作人员和1名在押囚犯。据医院消息，死者具体为一名监狱狱警、两名监狱看守及一名囚犯。
此外，事件导致至少35人受伤，伤者已送往医院接受治疗。目前，当地相关部门已介入调查骚乱原因，并正加强监狱安全管控措施。
