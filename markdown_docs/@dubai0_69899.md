# Telegram入局Serverless：机器人开发告别服务器

⚡ Telegram入局Serverless：机器人开发告别服务器
即时通讯巨头Telegram近日正式推出了其Serverless服务，为全球开发者带来重大便利。此举意味着开发者现在可以将Telegram机器人和Mini App的后端代码直接部署在其自有基础设施上，从而彻底摆脱了服务器配置、容器维护及扩容等传统运维难题。
据介绍，开发者只需编写标准的JavaScript模块，通过一条简单的命令行工具即可完成部署。这些代码将在紧邻Bot API的隔离V8沙箱环境中运行，并内置了基于SQLite的数据库，极大地简化了开发流程，提升了效率。
