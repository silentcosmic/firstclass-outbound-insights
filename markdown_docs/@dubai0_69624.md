# ️ 中国6月新冠确诊增至7.9万例：疫情上升趋势明显

⚠️ 中国6月新冠确诊增至7.9万例：疫情上升趋势明显
中国疾病预防控制中心（CDC）最新数据显示，2026年6月全国范围内的COVID-19感染病例数呈显著上升趋势。
根据中国疾控中心官网7月8日发布的《全国新型冠状病毒感染疫情情况（2026年6月）》，6月1日至30日期间，全国31个省（自治区、直辖市）及新疆生产建设兵团共报告新增确诊病例7.9万例。其中，重症病例达到130例，并记录到1例死亡病例（系基础疾病合并新冠病毒感染）。
