# 越南严打电诈：四名中国籍嫌犯与房东落网

⚡ 越南严打电诈：四名中国籍嫌犯与房东落网
近期，越南执法部门在全国范围内持续开展针对非法居留及电信诈骗的专项清查行动。日前，警方成功捣毁一处跨国电信诈骗窝点，抓获四名涉嫌非法入境的中国籍男子，同时依法控制了涉嫌知情包庇并违规出租房屋的当地房东。
据警方调查，这四名嫌疑人为逃避周边国家日益严厉的反诈打击，非法进入越南，并租用偏僻民宅搭建诈骗平台。为躲避监管，他们以高价租金租房，却未依法办理外籍人员居住登记。涉案房东明知租客身份异常且不配合登记，仍因谋取租金利益未按规定申报住宿信息，涉嫌为非法活动提供便利。
当地公安机关通过排查流动人口及出租屋登记信息发现异常后展开突击检查，当场控制嫌疑人并查获涉案设备。目前，所有涉案人员已被刑事拘留，案件正在进一步侦办中。越南警方强调，将持续加大对出租屋等重点场所的排查力度，严厉打击非法入境、跨国电诈及相关协助行为，以切断犯罪链条。
