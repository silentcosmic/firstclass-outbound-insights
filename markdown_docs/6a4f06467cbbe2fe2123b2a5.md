# 产业聚焦 | 高端机床赛道再迎资本青睐，星合机电斩获数亿融资，全面提速全球化战略

据本刊独家消息，专注于高端机床研发制造的上海星合机电近日完成数亿元人民币融资。本轮投资由五粮液基金领投，上市公司麦格米特亦参与跟投，深蓝资本则作为长期独家财务顾问，为此次交易提供专业支持。据悉，本轮募集资金将主要投入于高端磨齿机、磨床、铣床等核心产品的技术研发，并加速新产品的市场导入及海外商业化布局。
星合机电成立于2009年，长期深耕于高端机床的研发与制造。目前，其产品线已涵盖高端磨齿机、磨床、铣床等一系列精密加工设备，广泛应用于汽车制造、机器人、航空航天、精密减速机等前沿领域。
公司早期业务重心在于齿轮刀具的研发及技术服务，并成功开发出滚刀、剃齿刀设计软件及倒棱刀等产品，在全齿廓倒棱、大角度倒棱等加工工艺方面积累了深厚技术。自2012年起，星合机电战略性地将发展方向转向高端齿轮加工设备的研发，并逐步构建起完善的技术中心和自主研发体系。
近年来，上海星合持续拓宽产品边界。2024年，公司业务范畴从齿轮精密磨削设备进一步拓展至金属切削领域，形成了磨齿机与加工中心双轮驱动的产品发展格局。目前，公司已建成上海宝山、南通通州湾和南通海门三大生产基地，总占地面积超过3万平方米，同时，约有6万平方米的产能正处于建设进程中，为未来发展奠定坚实基础。
在技术路径上，上海星合坚持整机正向设计理念，围绕机械、电气、控制与软件一体化进行协同研发。公司已成功布局直驱重切、微米级磨削、高精度电主轴、低应力制造等多项核心技术，并在高速主轴、高精度分度头、数控系统等关键核心部件上具备自主研发能力。
公司的核心产品之一是磨齿机。其成形磨齿机产品精度已达到国内唯一的GB一级水平，能够充分满足新能源汽车高速电机齿轮（转速高达20000-30000转/分）的严苛加工需求。当前，星合机电的成形磨齿机在国内市场占有率稳居榜首，累计服务客户逾千家，覆盖比亚迪、上汽、双环传动等众多行业领军企业。
除了传统优势产品，公司亦积极拓展磨床、五轴加工中心等新兴品类。以直驱重切五轴加工中心为例，其采用直驱技术方案，相比传统丝杠传动方式，在动态响应、定位精度及机械磨损控制等方面展现出显著优势。该产品正面向航空航天、人形机器人、新能源汽车零部件等精密制造场景进行战略布局。
星合机电创始人蒋能表示，此次融资是公司成立以来首次引入外部资本。此前，企业主要依靠自有资金和经营现金流实现发展。他希望通过此次融资，借助资本力量，显著加快产品研发迭代与市场拓展步伐。
蒋能指出，公司经过多年的研发积累，相关产品已进入规模化生产阶段，这为加速发展奠定了坚实的内部条件。同时，在制造业转型升级、国产替代需求日益增长，以及五轴加工、直驱技术等行业技术持续迭代的背景下，高端机床市场正迎来一个全新的发展机遇期。此次融资之后，公司将进一步推动高端工业母机产品的研发与商业化进程，并全面提速海外市场的战略布局。
核心竞争力与战略优势
在谈及星合机电的核心竞争力时，蒋能强调了四个关键支柱。
他首先提及的是公司的研发体系：“我们始终秉持‘机、电、控、软’一体化理念，并非各部门各自为政，而是从产品定义阶段就共同参与研发。我本人技术出身，因此公司一直坚持技术驱动，并鼓励工程师们跨领域成长，而非局限于单一专业。”
其次，是“坚持正向设计”的原则：“许多企业可能会参考国外产品，但我们更致力于从底层原理出发，深入理解机床为何能实现高精度、高可靠性，而非简单复制方案。尽管这会延长研发周期，但由此积累的能力可不断复用，也更容易拓展至新的产品线。”
再者，蒋能强调了核心零部件的自主研发：“我们坚持自主研发高速主轴、高精度分度头、伺服尾座等关键部件。自主研发不仅有助于降低成本，更重要的是能够全面掌握产品性能、精度，并拥有后续迭代的主动权。”
最后一点，他认为研发、生产与应用验证的闭环至关重要：“很多产品研发出来后，我们会先进行内部加工验证，再根据客户反馈持续优化。这种研发、制造、验证形成闭环的模式，是我们能够持续提升产品性能的重要原因。”
战略机遇期与市场增量
对于当前高端机床市场所处的战略机遇期，蒋能阐述了多重利好因素的叠加效应。
他首先指出，公司自身的技术体系已趋于成熟，各项核心指标均达到可规模化放量的标准，这构成了加快发展的内部条件。其次，从宏观政策层面考量，国家在“十五五”规划中明确提出了推动工业母机发展的战略部署，这无疑是重大的政策利好。其三，在行业格局方面，国内机床行业正处于加速进口替代的关键时期，加之地缘政治变化推动供应链重构，为国产机床拓展市场空间提供了有利契机。第四，从技术演进角度看，直驱技术对丝杆传动的替代趋势日益加快，五轴联动技术对三轴、四轴机床的升级换代也在持续推进，这些技术路线的迭代本身就带来了结构性增长机会。蒋能判断，这几重机遇期的叠加，共同构成了当前一个难得的战略窗口期。
目前，新能源汽车被视为最大的市场增量。新能源汽车高速电机对齿轮的精度要求日趋严苛，某些应用场景甚至要求达到GB一级或更高的加工精度。星合机电以其卓越的超高磨齿精度著称，磨齿机需求直接受益于新能源车产销的强劲增长。公司已与上汽、吉利、长城等知名车企建立了深度合作关系，成为该领域国产替代的核心受益者。
人形机器人是另一个值得关注的领域。无论是行星滚柱丝杠还是RV减速器，其本质都对加工设备的精度提出了更高要求，这将为螺纹磨床、五轴加工中心等产品带来新的需求。星合机电已提前对相关产品进行了布局，并已开始有客户导入。
此外，工业齿轮、工程机械等传统制造行业也在经历升级。过去许多产品采用普通加工工艺，但随着对效率和精度要求的提升，越来越多的企业开始转向磨齿等高精度加工方式，这也催生了一轮设备更新换代的需求。
