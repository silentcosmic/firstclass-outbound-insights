# 泰北警方开枪拦截：查获210万颗“疯药”

🚨 泰北警方开枪拦截：查获210万颗“疯药”
7月16日，泰国北部普莱府宋县的南法检查站发生一起惊险追捕事件。一辆涉嫌运输毒品的白色轿车拒绝停车闯卡，警方随即展开追击，并开枪射击车辆轮胎，最终成功将其拦截。
警方在车内查获约210万颗俗称“疯药”的甲基安非他命。据悉，该嫌疑车辆此前已从附近检查站高速逃逸，警方随后设卡围堵。目前，此案正在深入调查中。
