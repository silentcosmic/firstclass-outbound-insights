# 阿联酋夜空奇观：小行星日邂逅草莓微月

🌌 阿联酋夜空奇观：小行星日邂逅草莓微月
6月30日，在全球共同关注“小行星日”之际，阿联酋的居民将迎来一场独特的夜空盛宴——“草莓微月”。这一天文事件不仅为观月提供了绝佳时机，也同步提升了公众对小行星科学的关注。
“小行星日”旨在普及小行星知识，探讨其形成机制以及地球防御潜在撞击的策略。该日期选定于1908年6月30日，以纪念当年西伯利亚地区因一颗小行星（或彗星碎片）撞击而引发的大规模森林破坏事件。
