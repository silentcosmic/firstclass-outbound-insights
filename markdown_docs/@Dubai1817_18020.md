# iPhone 20重塑外观：玻璃回归，屏下Face ID终结刘海

📱 iPhone 20重塑外观：玻璃回归，屏下Face ID终结刘海
据市场消息，为纪念iPhone问世20周年，苹果计划于2027年跳过iPhone 19，直接推出代号为V73和V74的iPhone 20系列。新机预计将配备6.3英寸和6.9英寸屏幕，尺寸与未来的iPhone 18 Pro系列相近。目前，相关生产场地已完成翻新，等待设备进场。
在外观设计上，iPhone 20系列将回归玻璃材质，有望采用与iPhone Air类似的一体化玻璃工艺，取代沿用两代的铝合金边框。更引人注目的是，新机将搭载四微曲屏设计和屏下Face ID技术，实现正面近乎无边框的真全面屏效果，彻底告别此前的刘海和挖孔。核心配置方面，iPhone 20系列将首发台积电2nm制程的A21 Pro芯片，并预装iOS 28系统。供应链方面，玻璃素材预计将以伯恩光学为主，比亚迪和蓝思科技为辅。
