# 洪森足球论引中方共鸣：小国亦平等

📣 洪森足球论引中方共鸣：小国亦平等
近日，中国驻柬埔寨大使汪文斌转发了柬埔寨参议院主席洪森关于“足球地理”的社交媒体帖文，并附言强调，国家无论大小、贫富、强弱，都应在国际关系的方方面面体现平等互尊，而非仅限于足球场上。他指出，中柬两国间的铁杆友谊正是国家间相互尊重、平等相待的典范。
此前，洪森在帖文中提出了“足球地理”概念，认为体育领域是世界上唯一存在真正国际标准的地方。他表示，足球运动不分国家贫富或人口多寡，这与当前世界陷入混乱、失去秩序的政治地理截然不同。洪森断言，基于这种足球地理的变化，未来总有一天，一个人口稀少的穷国也必然能够捧起世界杯冠军奖杯。
