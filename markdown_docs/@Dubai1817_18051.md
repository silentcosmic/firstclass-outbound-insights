# 老挝向中国移交95名嫌犯：跨境打击电信诈骗再升级

🚨 老挝向中国移交95名嫌犯：跨境打击电信诈骗再升级
7月12日，老挝警方在磨丁—磨憨国际口岸举行仪式，正式向中国警方移交95名中国籍嫌疑人。此次移交是两国执法部门跨境合作的最新成果，所有嫌疑人已由中方依法带回处理。
据了解，在被移交的95人中，有92人涉嫌参与电信网络诈骗活动，另外3人则涉嫌非法入境。这些嫌疑人均因违反老挝当地法律，被当地执法部门依法查获。
此次联合行动由老挝博胶省公安厅与中国公安机关共同开展，旨在进一步深化两国在打击电信网络诈骗、非法出入境等跨国违法犯罪活动方面的执法合作。此举彰显了区域内对跨境犯罪活动的零容忍态度。
