# 西港突查三场所：双狮酒店惊现毒品与赌博机

🚨 西港突查三场所：双狮酒店惊现毒品与赌博机
6月26日下午，在检察机关协调下，柬埔寨西哈努克省联合执法部门对西港第四分区的三处场所展开突击检查。此次行动旨在打击非法经营活动。
检查行动中，执法人员在一家持牌赌场内发现捕鱼及棋牌游戏设备。另一处此前被举报涉嫌非法经营捕鱼游戏的场所，设备虽已被移走，但现场负责人承认设备在检查前被运离。更引人关注的是，在双狮国际酒店内，执法人员查获一台捕鱼游戏机，并在其中一间客房内搜出疑似毒品及吸食工具。此外，部分外籍人员未能提供合法居留或住宿证件。
目前，所有涉案人员及查获物品已被依法带回，正接受进一步调查处理。
