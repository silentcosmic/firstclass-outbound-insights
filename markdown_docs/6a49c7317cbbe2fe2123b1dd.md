# 智能外骨骼：重塑户外探索与人机协作的新纪元

近日，在阳澄湖半岛度假区，一群身着创新“智能机甲”的徒步者沿湖畔前行。他们所佩戴的正是傲鲨VIATRIX增程动力外骨骼机器人。穿戴者仅需数步，便能感受到步伐变得格外轻盈，这得益于外骨骼系统通过实时学习其行走姿态，在抬腿瞬间精准施加推力，使得长达六公里的徒步旅程，转化成如同漫步城市街区般的轻松惬意体验。
此项“机甲徒步”活动由傲鲨科技与一家知名户外生活平台共同策划。其精心设计的路线串联起沿湖步道、休闲度假空间与自然生态公园，并沿途设置了多个观景与装备体验点。活动将前沿科技、户外美学与湖岸风光巧妙融合，旨在让参与者沉浸式体验人机协同的徒步模式，深切感受由科技赋能所带来的身心放松与探索乐趣。
作为人类体能的有效延伸，以傲鲨科技为代表的智能外骨骼机器人正经历显著的应用场景拓宽，其影响力正从传统的工厂生产线和医疗康复领域，逐步拓展至广阔的户外山野。这不仅突破了人体固有的移动与体能限制，更预示着一种全新的、基于人机深度协作的出行与运动范式的兴起。
战略布局：从工业深耕到消费级蓝海
傲鲨科技于2018年在上海创立。其团队早期专注于医疗康复外骨骼的研发，并迅速将技术拓展至工业应用领域。目前，其产品已在应急消防、电力巡检等众多工业物流场景实现落地应用，其海外业务版图已覆盖美国、日本等25个国家及地区。随着轻量化等核心技术的日趋成熟，傲鲨科技在2025年正式进军消费级外骨骼市场。其旗下的消费级品牌VIATRIX，凭借独创的Float360浮动式髋关节架构与先进的AI运动部署系统，荣获CES 2026创新奖及第十届中国设计智造大奖银奖，充分彰显其技术实力与设计创新。“将消费级外骨骼视为我们的核心发展目标，与此同时，工业场景的研发投入亦将持续推进。我们致力于形成‘工业奠定基础，消费驱动增长’的双轮驱动发展格局。”傲鲨科技创始人徐振华深入阐释其战略愿景。
市场洞察：用户需求与产品迭代
令人意外的是，早期对傲鲨消费级产品表现出浓厚兴趣的并非极限运动员或专业越野跑者，而是那些“久坐办公室、体能状况普通，但渴望走出户外、探索世界”的普通大众。徐振华观察到，在这一群体中，30至50岁之间的男性用户占据了显著多数。
至于女性用户数量相对较少的原因，徐振华分析指出，这主要在于当前外骨骼产品在整体外观上仍显得较为粗犷，其穿戴协调性及视觉呈现效果相对受限。他强调：“因此，我们正持续投入于材料与系统设计研发，力求使外骨骼产品实现更轻盈、更美观的突破。”在他看来，轻量化与智能化是消费级外骨骼产品发展的必然趋势。轻量化不仅能显著提升用户的穿戴舒适度，还能有效降低动能损耗，从而使设备更为节能，进而支撑电池尺寸向更小、更轻方向优化演进。
智能化的深度融合，是外骨骼技术真正在复杂户外场景实现有效应用的关键基石。徐振华回顾道，数年前曾有部分外骨骼品牌尝试进行市场推广，但最终未能取得预期的成功。他对此深入剖析，指出症结在于当时的外骨骼技术缺乏人工智能的驱动能力，仅能实现预设的运动规划。然而，每个个体的运动习惯与行走步态均存在差异，在应对爬坡、崎岖山路等不确定性环境时，这种缺乏智能适应性的设备难以与用户的真实运动状态实现高度融合。
如今，在人工智能能力的加持下，外骨骼技术实现了从僵化的预设程序到动态实时学习的根本性跨越。徐振华进一步介绍：“设备启动后，用户只需迈出几步，AI系统便能自动识别并根据其步态提供精准助力。在整个运动过程中，系统会持续采集数据并进行修正，从而实现设备与用户之间日益深度的融合。”他强调，VIATRIX所搭载的自研车规级响应电机，具备毫秒级的精准动力输出能力，双腿合计可提供46牛米的强大助力，结合先进的AI步态算法，显著提升了使用者腿部的力量与耐力。
生态构建：体验式营销与行业协同
傲鲨科技将外骨骼技术引入广阔山野的实践，并非始于阳澄湖。此前，在千岛湖成功策划的“机甲徒步”活动结束后，该路线已常态化运营。傲鲨科技与户外生活伙伴在景区内合作设立了户外场景租售体验点，为后续游客提供外骨骼租赁服务，从而打造了一个集户外场景、主题活动与深度体验于一体的特色空间。
从千岛湖到阳澄湖，傲鲨科技已逐步在全国范围内构建起20个文旅生态合作网点。其策略是先行通过系列活动验证路线可行性、积累内容素材、精准渗透核心用户群体，随后将成功经验产品化，转化为可反复运营的租售体验点。这一模式旨在让潜在用户能在真实的户外环境中亲身体验外骨骼的效能与乐趣，进而作出是否购买的决策。徐振华指出，这种“科技+文旅”的生态运营模式，不仅能有效消除潜在用户在购买决策上的疑虑，更可通过用户实际体验数据，为产品迭代提供宝贵反馈，从而形成一个“体验-验证-转化-迭代”的良性循环。
除了将业务拓展至山野户外，傲鲨科技也积极进驻城市内部的知名户外品牌门店，例如Haglöfs。徐振华坦言：“国内多数外骨骼科技公司普遍缺乏十年以上的专业户外经验。”他表示，傲鲨科技选择与那些深谙户外精髓的品牌进行合作，旨在“补齐短板”，通过“向资深的百年户外品牌企业学习，能够更精准地洞察客户需求与应用场景，进而优化调整产品设计。”同时，作为一家以产品研发为核心竞争力的外骨骼科技公司，傲鲨VIATRIX的愿景是与户外行业的生态伙伴们携手，通过技术赋能，共同定义并打造下一代户外科技装备。
傲鲨科技正通过其可复制、可扩展的运营模式，系统性地构建从单一尝鲜体验向可持续户外生活方式的转变，以及从传统零售终端向深度体验网络节点的升级，从而实现品类心智的全面占领与商业闭环的有效构建。
人机协作：定义边界与未来愿景
对于外骨骼功能原理的疑问，徐振华对此已是驾轻就熟。他解释道：“外骨骼的核心功能在于根据人体动作与指令，实现毫秒级的响应并进行力学放大，其目标是确保用户在体能即将达到极限时，仍能维持理想的技术动作。”
换言之，外骨骼并非取代人类进行决策，而是通过提供“增程动力”，从宏观层面节约人体消耗，使得正确动作能够以更省力的方式完成，进而延长用户的极限耐力持续时间。这意味着，在同等体力极限条件下，使用者能够行进更远的距离，同时身体机能也能获得充分的锻炼。
徐振华清晰界定了消费级外骨骼与医疗级外骨骼之间的边界。他认为，两者之间的区分并非技术能力的简单递减，而是产品设计逻辑的根本性差异。医疗级外骨骼需与医生的临床诊断深度结合，并涉及复杂的康复评估与严格的风险控制。
然而，在消费市场层面，这两类产品的边界对于普通用户而言往往模糊不清。徐振华坦言：“许多人难以准确辨别外骨骼的实际功能，不清楚何为医疗用途，何为日常穿戴。”鉴于此，傲鲨科技高度重视售前的主动沟通与说明，在用户咨询阶段便提供明确的指导。他表示：“如果用户存在偏瘫、脑卒等健康问题，我们会明确告知不推荐使用消费级外骨骼。对于年长用户，我们还会要求其提供一段行走视频，以便根据行走状况进行专业的售前预判。”
这种审慎的“有所不为”策略，体现了傲鲨科技对自身技术边界的深刻理解。据市场分析机构预测，2025年全球可穿戴机器人外骨骼市场规模将达到24.9亿美元，这一赛道热度持续升温，包括傲鲨科技在内的多个品牌已完成多轮融资。但徐振华对于行业当前所谓的“爆发式增长”保持着一份理性。他认为：“外骨骼技术的发展不会出现类似‘iPhone 4时刻’那样的突变式爆发，而将是一个在更长时间内持续爬坡、稳步上升的过程，最终依靠卓越的产品能力来逐步培养用户的使用习惯。”
外骨骼技术的快速演进，也引发了更深层次的思考：当科技能够显著拓宽人类的行动疆界时，人类的物理“边界”究竟将由何来界定？在徐振华看来，与以替代人类为主要目标的人形机器人不同，外骨骼技术所追求的，是与人类的深度协作。它旨在保持人类主体性的前提下，赋予个体更强大的力量与更广阔的自由。
