# ️ 洪森率团访华：深化柬中战略合作

✈️ 洪森率团访华：深化柬中战略合作
柬埔寨参议院主席、人民党主席洪森于6月25日上午率领高级代表团启程，对中国展开为期三天的正式友好访问。此次访问旨在进一步巩固两国传统友谊，并加强战略合作。
访问期间，洪森将与中国国家主席习近平举行会见，并分别与国务院总理李强、全国人大常委会委员长赵乐际举行会谈。双方将就深化双边关系及共同关心的地区与国际问题深入交换意见。
此外，洪森此行还将探望正在中国疗养的柬埔寨国王西哈莫尼和太后莫尼列。
