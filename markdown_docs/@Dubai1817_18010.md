# 阿联酋严厉谴责伊朗袭击，中东多国安全承压

🚨 阿联酋严厉谴责伊朗袭击，中东多国安全承压
上周日（7月12日），阿联酋发表声明，强烈谴责伊朗对巴林、科威特、卡塔尔、约旦和阿曼发动的最新一轮导弹及无人机袭击。
阿联酋外交部强调，此类袭击公然侵犯了兄弟国家主权，对其地区安全与稳定构成严重威胁。阿联酋重申与受影响国家全面团结，并支持为维护其国家安全所采取的一切措施。
与此同时，阿联酋防空系统当日早些时候也曾应对新一轮导弹及无人机威胁。阿联酋国家紧急危机和灾害管理局随后确认，所探测到的威胁位于阿联酋境外，并向居民保证当前局势稳定，同时指出即使影响可能性较低，仍会主动发布预防性警报。
