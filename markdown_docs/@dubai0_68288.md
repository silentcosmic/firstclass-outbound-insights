# 浦发银行迪拉姆跨境服务落地：中企出海再添便利

📣 浦发银行迪拉姆跨境服务落地：中企出海再添便利
随着跨境电商等新业态的蓬勃发展，中资企业在阿联酋的业务拓展对本地化金融服务需求日益迫切。此前，一家深耕中东市场的头部跨境电商物流企业，在服务海量国内跨境商家时，面临迪拉姆本地回款的痛点，期望能直接接收阿联酋合作方款项，以降低汇兑成本、缩短回款周期并规避汇率波动风险。
浦发银行西安分行迅速响应，为该企业开立账户并完成代理行报备，确保迪拉姆款项可直接入账。账户启用首日即成功实现原币入账，显著提升了客户资金周转效率。该服务还配套提供原币汇入、套汇汇出等综合方案，在强化跨境风险管理的同时，显著提升了业务便利性。
此次迪拉姆跨境业务的落地，标志着浦发银行在完善“一带一路”沿线小币种结算体系方面迈出重要一步，并有力助推中阿两国本币结算进程。浦发银行表示，未来将持续深化中东区域特色币种服务，为中资企业出海提供更高效便捷的跨境金融方案。
