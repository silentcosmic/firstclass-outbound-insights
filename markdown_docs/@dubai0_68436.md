# ️ 港女美容师涉人口贩卖，诱骗赴泰转卖缅甸诈骗园区

⚠️ 港女美容师涉人口贩卖，诱骗赴泰转卖缅甸诈骗园区
一名香港美容师因诱骗两名女性前往泰国，并将其转卖至缅甸诈骗园区，已在区域法院承认欺诈罪行。33岁的潘心怡于6月26日承认一项欺诈罪，此案揭露了跨境人口贩卖与诈骗的复杂链条。
案情显示，潘心怡于2024年11月以免费机票和住宿为诱饵，邀请两名女子旅行。随后，她突然安排两人于同年12月27日前往泰国，声称需在当地运送泰铢以获取报酬。抵达曼谷后，两名受害者随即被当地男子接走，并经过多次转运，最终被武装人员押送至缅甸的诈骗园区。
在园区内，受害者被告知以每人2.7万美元的价格被转卖，并被迫学习网络诈骗话术，针对海外华人实施诈骗。直至2025年1月初，园区方面提出每人支付2.8万美元赎金方可获释。两名受害者的家属随后支付了高额赎金，最终她们于2025年1月11日安全返回香港。
法官指出，此案性质极其严重，涉及人身自由、人口贩卖、跨境勒索及有组织犯罪集团的精密安排。潘心怡的判刑已押后至2026年9月29日，以等待同类案件的上诉结果，她将继续还押。
