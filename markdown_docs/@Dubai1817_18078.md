# 汇旺案：维权者柬埔寨被捕，或引发更大抗议潮

⚡ 汇旺案：维权者柬埔寨被捕，或引发更大抗议潮
近日，在柬埔寨，多名汇旺平台受害储户再次集结，前往中国驻柬埔寨大使馆前请愿。他们的核心诉求是希望使馆关注此前被捕的维权代表柯小伟，并协助推动其尽快获释。
据悉，柯小伟于7月13日在柬埔寨国家银行前参与相关维权活动后，被当地警方带走。此事件迅速在受害者群体中引发强烈反弹，他们普遍认为，此次逮捕旨在震慑维权行动，但反而可能激化矛盾，促使更多受害者继续追讨被冻结的资金。
有消息指出，受害者群体正酝酿发起新一轮、更大规模的维权行动，要求当局释放柯小伟，并追回他们的血汗钱。有维权人士表示，若不采取行动，今日之捕可为明日之鉴。
