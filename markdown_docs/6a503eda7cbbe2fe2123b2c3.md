# SWIFT区块链账本启动试点，17家全球银行率先接入

环球银行金融电信协会（SWIFT）已正式开放其基于区块链技术的分布式账本的初始阶段。多家全球主要银行正积极准备，即将试点真实交易。
这家全球金融信息网络巨头正赋能17家国际银行，通过这一新型账本，试点基于代币化存款的24/7全天候跨境支付服务。
该共享账本为参与的金融机构提供了一个安全、高效的统筹协调层。
银行能够在其内部系统发行代币化存款，并随时为客户进行资金转移，即便是在夜间或周末亦不受限。最终的清算环节则通过现有支付基础设施完成。
准备参与真实交易试点的银行阵容强大，包括澳新银行（ANZ）、法国巴黎银行（BNP Paribas）、纽约梅隆银行（BNY）、花旗银行（Citi）、星展银行（DBS）、阿布扎比第一银行（First Abu Dhabi Bank）、第一兰德银行（FirstRand Bank）、汇丰银行（HSBC）以及伊塔乌联合银行（Itaú Unibanco）。此外，劳埃德银行（Lloyds Bank）、马士礼格银行（Mashreq）、三菱日联银行（MUFG Bank）、华侨银行（OCBC）、渣打银行（Standard Chartered）、瑞银集团（UBS）、大华银行（UOB）和富国银行（Wells Fargo）等也在首批参与机构之列。SWIFT方面表示：“凭借我们新型账本功能，我们正将传统金融的信任与稳定性，延伸至数字货币的前沿领域。它将赋能代币化价值在跨境流通中，实现现代商业所期待的速度与灵活性。”
SWIFT在收到国际金融机构的反馈后，仅用九个月时间便完成了此账本的设计与构建。
该合作组织将此次推出视为其技术堆栈的首个应用案例，未来有望支持可编程货币和智能代理商务等前瞻性应用。
经过初始的受控部署后，该账本的功能将进一步拓展。这亦是基于SWIFT现有基础设施近期升级的成果，目前该网络上75%的支付款项可在10分钟内抵达受益银行。
此外，SWIFT还在积极部署一套零售支付框架，旨在提升消费者交易的费用透明度并加速交付速度。
