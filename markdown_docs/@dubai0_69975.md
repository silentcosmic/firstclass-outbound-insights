# 柬埔寨勋爵旗下赌场涉网络诈骗：路透社披露

🚨 柬埔寨勋爵旗下赌场涉网络诈骗：路透社披露
据路透社7月15日报道，位于柬埔寨与泰国边境奥斯玛（O'Smach）口岸的皇家山赌场（The Royal Hill Casino），被指控为大规模网络诈骗活动的据点，并与一位极具影响力的柬埔寨勋爵存在关联。
报道指出，该赌场由柬埔寨勋爵林兴（Lim Heng）旗下Lim Heng Group所有，据称其部分设施被出租给中国籍人员，用于从事网络诈骗等非法犯罪活动。
路透社进一步披露，Lim Heng Group作为皇家山赌场的业主，其物业被用于跨境网络诈骗，凸显了该地区非法活动与当地权势人物之间的复杂联系。
