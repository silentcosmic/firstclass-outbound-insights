# 出海头等舱深度：长鑫存储IPO承销团阵容揭秘，SK海力士强势登陆纳斯达克，OpenAI推出智能体平台

“国产存储第一股”长鑫科技的首次公开募股（IPO）已进入发行倒计时阶段，其背后的承销团阵容也随之浮出水面。根据最新披露的招股意向书，长鑫科技预计于7月16日正式发行。本次IPO，中金公司与中信建投将共同担任联席保荐机构及主承销商，同时，国泰海通、国元证券、华泰联合以及招商证券亦将作为联席主承销商参与，共同构成了多达六家券商的承销联盟。
这样的承销团被业界视为一支罕见的“豪华阵容”。一方面，中金、中信建投、国泰海通、华泰联合、招商证券等国内顶尖投行的密集参与，彰显了长鑫科技在资本市场的吸引力；另一方面，安徽本土券商国元证券也成功跻身主承销商之列。更值得关注的是，承销团中多家主要券商同时也是长鑫科技的现有股东。这意味着除了传统的投行承销收入外，这些机构还可能从股权投资中获得更为可观的收益，为此次IPO增添了更大的想象空间。
SK海力士强势登陆纳斯达克，募资规模超260亿美元
韩国芯片巨头SK海力士于美东时间7月10日正式登陆以科技股闻名的纳斯达克交易所。此举为其开辟了强劲的全球融资新渠道，也为全球投资者提供了布局高速增长的人工智能基础设施赛道的绝佳机会。这家全球第二大存储芯片制造商计划通过在纳斯达克发行美国存托凭证（ADR），募集约40万亿韩元（折合265亿美元）资金。本次发行共计1.779亿份美国存托凭证，发行价最终定为每份149美元，每1份ADR对应其在韩国首尔交易所上市普通股的十分之一。
上市首日，SK海力士ADR开盘价报170美元，较发行价149美元上涨约14.1%。盘中一度触及177美元高点，涨幅扩大至约18.8%，最终收盘报168.01美元，当日涨幅约12.8%，对应市值达到了约1.22万亿美元。
OpenAI推出ChatGPT Work智能体，深化AI应用版图
OpenAI近日发布了由其最新前沿模型系列GPT-5.6驱动的ChatGPT Work。这款全新的ChatGPT桌面应用巧妙地将聊天、工作与Codex功能深度整合，旨在为用户提供更高效、智能的体验。ChatGPT Work被设计成一款能够跨各类应用与文件执行操作的智能体，它能够根据需要持续跟进项目数小时，直接将既定目标转化为已完成的工作成果。其能力涵盖了在不同应用和工作流中收集信息，并能生成表格、幻灯片、文档、网页应用等多样化的成品内容，同时还能将复杂任务拆解为独立执行的小步骤，从而持续推进长时间项目。

资本市场聚焦AI与新能源：大额融资与并购频现
7月10日早间消息，MiniMax已完成新一轮160亿港元的融资，吸引了多家国际主权基金、长线基金、中资机构和多策略基金的参与，覆盖亚太、欧洲和美国市场。本次交易共吸引逾百家机构参与，实现了7倍的认购覆盖。最初发行规模约18亿美元，在机构需求的强劲推动下，最终扩大至20亿美元以上。
近日，北京新奥聚变能源科技有限公司完成首轮（Pre-A）融资，投后估值高达106亿元。本轮融资由龙芯创投领投，中科创星、经纬创投联合领投，鼎晖投资、凯辉基金、上汽恒旭资本、联想之星、招商致远、亦庄产投等多家机构跟投。新奥方面表示，此轮融资将加速推进聚变商业化落地进程。
近日，可信物理智能企业“Xspark AI”（无界智航）宣布完成首轮近亿元天使轮融资。本轮融资由鼎晖VGC、初心资本、SEE Fund无限基金联合领投，深圳天使母基金、水木校友种子基金等多家财务投资机构跟投。资金将主要用于核心技术研发、产品迭代以及Physical AI的规模化落地。
近日，端侧AI推理芯片企业“聆思科技”宣布完成近5亿元B轮融资。本轮融资由安徽省与合肥市多家国资平台联合战略领投，深报一本、天智投资、科讯创投、盈科投资、东瑞投资、永鑫方舟等机构跟投，泰合资本担任聆思科技长期财务顾问。本轮资金将重点投入新一代端侧大模型AI推理芯片的研发，推动聆思的产品体系从感知模型AI推理芯片向认知大模型AI推理芯片进一步升级。
近日，“灵睿智芯”宣布完成数亿元融资。本轮融资由张江高科、中科创星、东方富海、成为资本、石溪资本、壁仞科技以及多个行业合作伙伴等产业投资机构共同参与，元禾璞华、上国投孚腾资本、华山资本等老股东也持续加码。资金将重点用于公司高性能RISC-V内核的应用和产业落地，加速公司智能体CPU产品研发，并深化AI原生融合计算架构的创新和布局。与此同时，公司后续融资工作也已展开。
“他山科技”已于近期完成B轮融资，金额达到数亿元。本轮投资方包括均胜电子、太平创新、奥克斯、鹏翎股份、Lavender Hill Capital Partners（LHCP）、洪山资本等产业方，老股东道氏技术、彬复资本也追加了投资。资金将主要用于触觉传感器及芯片迭代、场景化方案落地以及触觉训练平台建设。
