# 泰国严打挂名持股：芭提雅别墅项目遭突查

🚨 泰国严打挂名持股：芭提雅别墅项目遭突查
7月3日，泰国春武里府府尹纳里·尼拉迈翁联合商务、土地、移民、警方、税务及芭提雅市政府等多部门，对芭提雅和中天地区的五处大型泳池别墅项目展开突击检查。此次行动旨在重点打击外国资本利用泰国人“挂名持股”，非法经营房地产及旅游业务的行为。
行动中，有4名中国公民因涉嫌吸食笑气被带走调查，显示出执法部门在打击非法行为上的决心。泰国政府明确表示，将持续跟进相关调查，并进一步加强对外国资本规避法律、非法经营房地产及旅游产业的打击力度。
