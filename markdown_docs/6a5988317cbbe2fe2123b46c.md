# 中东AI雄心遇阻：巨额投入下仍难摆脱英伟达芯片制约

坐拥丰厚石油财富的海湾国家，正投入数百亿美元押注人工智能，旨在将自身重塑为全球AI中心。然而，他们逐渐发现，金钱几乎能买到一切，唯独无法摆脱对英伟达的依赖。
沙特阿拉伯和阿拉伯联合酋长国已承诺投入数百亿美元，力求在石油财富耗尽前成为AI强国。过去一年中，两国积极尝试从多家公司采购芯片，以避免单一供应商在成本和建设速度上形成垄断。
然而，这一策略面临巨大阻碍。英伟达的少数替代方案要么性能远不及，要么像中国产芯片那样，受到美国出口管制的限制。卡内基国际和平基金会智库研究芯片地缘政治的学者萨姆·温特-利维指出：“摆脱英伟达的多元化举措，确实降低了海湾地区对某个商业供应商的依赖，但这丝毫未改变其对美国政治依赖的风险。真正能改变这种状况的是中国的供应，但中国目前尚无法大规模出口具有竞争力的芯片，而且美国已明确，海湾国家获得美国AI技术的条件是排除中国硬件。”
6月1日，沙特阿拉伯签署协议，将在自动驾驶出租车项目中使用英伟达技术，涵盖从车载计算机到操控软件的全面解决方案。这项与美国芯片制造商的协议由Humain达成，该公司是沙特公共投资基金去年成立的合资企业，旨在将沙特王国打造为全球AI强国。Humain同时正在利雅得和达曼建设数据中心，这些中心将由数十万枚英伟达芯片驱动。
阿联酋也通过其国家支持的AI公司G42进行同等规模的投资，该公司正在阿布扎比建设名为“星门”（Stargate）的数据中心。其第一阶段将部署40万枚英伟达芯片。G42数月来一直在尝试将订单分散给多家供应商，但公司首席执行官彭晓在1月的一次采访中坦言，“星门”内部的设备“大部分”仍将是英伟达产品。
英伟达市场主导地位的广度在截至去年10月的财季中得到充分体现，其数据中心业务营收达到创纪录的512亿美元，同比增长66%。英伟达在财报中表示，其最新的Blackwell计算架构系列产品已“销售一空”。
英伟达的这种主导力，一部分源于其芯片本身的卓越性能，另一部分则在于其专有的并行计算平台——统一计算设备架构（CUDA）。英伟达耗时近二十年构建了工程师在其芯片上运行AI所需的各项工具，如今超过400万开发者活跃在CUDA生态系统中。转向竞争对手意味着要从头开始重建多年的工作积累。
Humain一直在努力拓宽其供应商名单。去年8月，该公司与AMD达成了一项价值100亿美元的协议，旨在获取500兆瓦的计算能力；与Groq签署了一项20亿美元的协议，用于运行其聊天机器人；并与高通合作，为沙特的一个数据中心配置200兆瓦的高通芯片。
新兴市场投资公司North of South Capital的合伙人卡米尔·迪米奇表示，这些交易确实为海湾地区带来了能够运行AI的芯片，但其任务范围比英伟达芯片更为狭窄。高通和Groq的芯片旨在经济高效地运行已完成的AI模型，适用于回答用户问题等日常任务，而训练强大的AI模型仍然需要英伟达的芯片。
迪米奇透露，Humain从英伟达订购的首批产品是18,000枚最新的Blackwell芯片——正是这类进行“重型计算”的硬件，目前尚无竞争对手能够匹敌。
温特-利维指出，无论是AMD还是高通的芯片，都与英伟达的产品面临相同的附加条件，因为它们都是美国产品，在销售给海湾国家之前，都需要获得华盛顿的批准。
中国最先进的芯片至少落后英伟达一代，且其大部分产能都被国内企业消化。温特-利维还补充说，美国已告知海湾国家，能否获得其顶级芯片，取决于是否将中国设备排除在外。
迪米奇分析称，即使从竞争对手那里采购，也无法完全绕开英伟达，因为其竞争对手在供应链的更下游环节，同样依赖相同的供应商。目前最难获得的两种资源是为芯片提供数据的高速专用存储器，以及台积电（TSMC）代工厂的产能——台积电同时为英伟达及其竞争对手生产芯片。迪米奇强调：“英伟达芯片对于AI训练至关重要。对于推断计算这类任务，虽然应用专用集成电路（ASIC）设计正变得越来越普遍且比英伟达更经济，但它们依然需要台积电的代工能力。”
ASIC设计是专为特定任务定制的芯片，其成本低于英伟达的通用处理器。
迪米奇指出，即使海湾国家拥有巨额财富，其在芯片采购方面的优势也远不如预期，因为与他们竞争芯片的美国科技巨头同样资金雄厚。他表示，现金并非决定谁能优先获得供应的关键因素。迪米奇进一步阐述：“我认为目前‘无限资金’并非一个显著优势。所有美国设备采购商似乎都能以极高的估值在资本市场获得无限资金，因此这尚未构成限制，尽管未来可能会如此。”
华盛顿中东研究所高级研究员穆罕默德·索利曼指出，建立一个不依赖任何外部力量的AI产业，其成本已高昂到连全球最富有的经济体都难以承受。索利曼表示：“人工智能领域的真正技术主权极其昂贵，对于美国和中国以外的几乎所有国家来说，实际上都无法实现。即使是欧洲，尽管拥有强大的工业能力和资本市场，也缺乏在任何有意义的时间框架内建立完全独立AI技术栈的现实前景。”
温特-利维认为，海湾地区的资金、对美国科技公司的持股以及数据中心建设的速度，赋予了它真正的议价能力。然而，它正在选择依靠英伟达来采购整个行业赖以构建的最佳芯片，而非等待那些可能永远无法匹敌的替代方案。索利曼指出，海湾国家的领导者们非常清楚他们正在做出何种选择。索利曼总结道：“海湾国家当前的道路与多元化背道而驰。他们正通过大规模数据中心和直接的美国合作关系来加速整合，而不是追求昂贵且不确定的独立性。海湾国家是AI领域的现实主义者。”
