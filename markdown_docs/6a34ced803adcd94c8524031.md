# 早期投资者拟斥资20亿美元从Meta回购新加坡AI初创公司Manus股权

据本刊独家消息，新加坡人工智能初创公司Manus的早期中国投资者正计划从Meta手中回购该公司，交易金额预计为20亿美元，与Meta去年12月收购该公司的价格持平。
此次拟议的交易吸引了早期支持者HSG、真格基金和腾讯的参与。其中，HSG和真格基金据称正在考虑动用新的资本来接手Meta在该初创公司中的持股。
而另一家早期投资方Benchmark则未计划参与此次回购。
此举的背景是，北京方面在今年4月发布了一项监管指令，导致该交易受到更严格的审查，并引发了关于解除Meta所有权的讨论。这反映出中国监管机构正收紧对美国在先进AI初创公司投资的监管力度。
根据掌握的信息，在该指令发布后，Meta已与Manus进行了内部运营剥离，并停止了两公司之间的数据共享。
Manus专注于开发能够以极少人工干预自主执行任务的AI代理。自被Meta收购以来，该公司的营收实现了显著增长。
其近期年化收入运行率已从交易完成时的1亿美元飙升至4亿至5亿美元之间。
此外，该初创公司据称还在探讨重组为一家中国注册的合资企业，此举可能为其未来在香港上市铺平道路。
