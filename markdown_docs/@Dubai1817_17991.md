# 阿联酋确认境外导弹威胁，全国高度戒备

🚨 阿联酋确认境外导弹威胁，全国高度戒备
阿联酋国家紧急危机与灾害管理局（NCEMA）近日表示，该国已确认今天上午探测到的导弹威胁位于阿联酋境外。目前，阿联酋局势保持稳定，国家各系统正维持最高级别的准备状态。
NCEMA强调，在与战略伙伴协调后，已确认该威胁未对阿联酋境内构成直接影响，目前没有令人担忧的迹象。国家监测系统将继续全天候运行，密切追踪事态发展。
