# ️ 柬埔寨前诈骗园区转型：疑涉黄赌毒与警方高层

⚠️ 柬埔寨前诈骗园区转型：疑涉黄赌毒与警方高层
据市场消息，位于柬埔寨干拉省6号路的湄湾度假村，曾是当地知名的“名门园区”，一度是上万名诈骗从业人员的聚集地。去年起，园区内大部分人员已陆续撤离，至今年初已全部搬空。
然而，该场所并未回归正常运营，反而据称已转型为一处涉及黄赌毒的“嗨场”。更令人震惊的是，有爆料指出，该场所的实际控制人疑为当地一名被称为“华哥”的公安局长，并声称其本人常在其中吸食毒品并与未成年人发生关系。现场据称发现大量K粉等毒品，但爆料人出于安全考量，仅提供了部分可公开的影像资料。
此情况引发了对当地治安环境的深切担忧。市场观察人士呼吁柬埔寨有关部门对此类违法行为展开彻底调查，以维护社会秩序和公众安全。
