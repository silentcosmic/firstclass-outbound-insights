# 伊朗阿联酋贸易航线全面恢复

⚡ 伊朗阿联酋贸易航线全面恢复
伊朗官方媒体近日援引贸易部门官员消息称，伊朗与邻国阿联酋已正式恢复双边贸易关系。据悉，德黑兰至迪拜的商业航线也将于7月1日起重新投入运营。此举标志着两国经贸往来的重要进展，预计双边贸易额将逐步回升至此前水平。
