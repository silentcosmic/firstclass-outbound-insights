# 37亿美元医保诈骗主犯落网：美国史上最大案之一

🚨 37亿美元医保诈骗主犯落网：美国史上最大案之一
美国联邦调查局（FBI）近日在土耳其成功逮捕并引渡了易卜拉欣·希尔米，此人被指控策划了美国历史上最大规模的医保欺诈案之一，涉案金额高达37亿美元。
据调查，希尔米及其同伙利用空壳公司向联邦医疗保险计划（Medicare）提交虚假的医疗设备账单。这些公司表面上经营医疗用品，实则通过批量盗用真实患者的医保号码，开具从未提供或需要的数千美元医疗设备账单，受害者遍布全美各地。
此次逮捕是司法部今年6月全国医疗保健欺诈打击行动的重大进展之一。该行动共起诉455名被告，涉及虚假索赔总额超过65亿美元，是司法部历史上执法机构参与范围最广的一次。美国副总统万斯已在社交平台确认希尔米落网，并强调：“如果你从美国人民那里偷窃，世界上没有任何地方能让你安全藏身。”
