# ️ 汇旺维权新进展：巨额债务二折收购遭拒

⚠️ 汇旺维权新进展：巨额债务二折收购遭拒
今天上午，汇旺的用户再次聚集在柬埔寨大使馆门口，就其所欠债务进行维权。
据现场抗议者透露，有三家在柬的大型企业曾主动接触他们，提出以债务总额20%的价格进行收购。然而，这一收购提议最终被抗议者拒绝。
