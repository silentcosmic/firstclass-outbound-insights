# 柬埔寨警方与FBI：深化跨国协作打击网诈

🤝 柬埔寨警方与FBI：深化跨国协作打击网诈
柬埔寨国家警察总署于6月23日上午在内政部举行高级别会议，与美国联邦调查局（FBI）驻柬代表处共同审查执法合作进展。此次会议由副总理兼内政部长苏速卡主持，国家警察总署总监苏通及各省警察局局长均出席。
会议重点评估了近期在打击网络诈骗犯罪方面取得的成果。此外，双方代表还就如何进一步织密跨国协作网络、共享关键情报资源以及协同攻克案件难点等议题进行了深入讨论，旨在全面深化双边执法合作。
