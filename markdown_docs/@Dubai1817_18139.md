# 太子集团“二号人物”日本落网，多重身份曝光

🚨 太子集团“二号人物”日本落网，多重身份曝光
近日，柬埔寨太子集团核心成员胡小伟在日本大阪被捕，引发国际社会广泛关注。这位44岁的关键人物，曾使用“胡晏铭”、“陈小二”等多个化名，被指持有中国、柬埔寨、塞浦路斯以及圣基茨和尼维斯等多重国籍身份，长期活跃于海外商业网络。他被认为是太子集团创始人陈志的重要助手，在集团的海外布局、技术运营及资金体系中扮演核心角色。
日本警方于2026年6月14日在大阪一家高级酒店内首次逮捕胡小伟，指控其涉嫌通过伪造居民迁移申报、提交虚假电子公证文件等方式，试图获取长期居留资格。调查显示，他曾申报迁入东京中央区，但实际长期居住在大阪。同年7月5日，胡小伟再次被捕，罪名是涉嫌非法出借在留卡并利用他人身份办理相关手续。
此外，胡小伟的过往记录也浮出水面。他曾涉及2010年“骑士攻击小组”案件并被中国公安部挂牌督办。2025年，其化名“陈小二”还出现在美国财政部的制裁名单中。
值得注意的是，胡小伟在日本期间的居住安排与一名柬埔寨籍人士苏江波（Su Jimmy）有关。苏江波于2024年前后移居日本，并在大阪经营天智富加有限公司，业务涉及旅游及房地产。国际调查记者组织曾指出，苏江波可能涉嫌协助太子集团部分高层人员在日本进行安置。目前，苏江波尚未被日本警方正式起诉。
太子集团长期以来受到国际执法机构的密切关注，其业务涵盖房地产、金融投资等多个领域，但也被指控涉嫌参与跨境电信诈骗、网络赌博、洗钱等违法犯罪活动。美国方面曾将太子集团描述为亚洲大型跨国犯罪组织之一。胡小伟的落网，无疑为围绕太子集团海外网络、资金流动及相关人员去向的调查增添了新的进展。
目前，胡小伟仍处于日本司法程序之中，警方正对其电子设备、资金流向以及海外关系网络展开深入调查。
