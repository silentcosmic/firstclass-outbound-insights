# ️ 超强台风“巴威”来袭：关岛、北马里亚纳群岛及东南沿海需警惕

⚠️ 超强台风“巴威”来袭：关岛、北马里亚纳群岛及东南沿海需警惕
今年第9号超强台风“巴威”正持续增强，目前中心附近最大风力已达17级以上，风速约62米/秒，中心气压低至915百帕。受其影响，关岛及北马里亚纳群岛附近海域已遭遇严重的风雨天气。
鉴于“巴威”的强大势头，台湾、福建、浙江、上海、江苏南部等中国东南沿海地区应密切关注台风路径动态，提前做好防范措施，确保人员及财产安全。
