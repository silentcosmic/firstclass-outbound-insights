# 乌干达突袭千人电诈窝点：多数为中国公民

🚨 乌干达突袭千人电诈窝点：多数为中国公民
乌干达近日开展了一场史无前例的跨国网络诈骗清查行动。据7月10日多家外媒报道，乌干达刑事调查局联合多支安全力量，在首都坎帕拉及瓦基索区同步突袭，一举拘留逾千名外籍人员，其中多数为中国公民。此次抓捕规模刷新了乌干达打击网络犯罪的历史纪录。
执法部门透露，此次多区域同步清剿行动旨在打击境内多处隐蔽的非法涉诈窝点，特别是无证离岸话务中心。其中，与乌干达前副总统吉尔伯特·布肯亚有关联的瓦基索区卡托米王国度假村是核心目标，仅此一处便抓获约430名涉案人员。其余数百名嫌疑人则在坎帕拉纳古鲁、坎桑加一带的城郊公寓与私人园区内被捕。
突袭现场，执法人员查获了大量作案设备，包括电脑、手机、服务器及成套诈骗话术脚本。初步核查显示，这些窝点均不具备合法经营资质，长期组织人员针对全球民众实施投资诈骗、虚拟货币骗局，并非法窃取他国居民身份信息，已形成完整的跨境电诈犯罪链条。
目前，所有被拘外籍人员已被统一安置在纳曼维封闭式安保专区，办案人员正分批核验身份及签证状态，并根据涉案程度分类处置。其中，被虚假招工诱骗入境、遭受人口贩运的受害者将启动人道主义遣返；仅签证过期且未参与诈骗活动的外籍人员将被驱逐出境；而诈骗团伙组织者、财务管理人员及核心操盘骨干，则将依据乌干达本土法律面临刑事诉讼，从严追究责任。乌干达警方强调，此次行动是全国常态化打击跨境网络诈骗工作的一环，未来将持续深化与亚洲各国的跨境警务协作，以斩断人口偷渡输送链条，彻底瓦解跨国网络犯罪利益产业链。
