# 湖南破获裸聊诈骗案：千余推特账号引流境外

🚨 湖南破获裸聊诈骗案：千余推特账号引流境外
湖南常德汉寿县警方近日成功捣毁一处裸聊引流电信诈骗窝点，抓获3名嫌疑人。该窝点依托线上平台，为境外裸聊敲诈团伙精准输送受害目标。
警方通报显示，该团伙自6月26日起搭建引流窝点，累计操控逾千个Twitter账号，发布千余条色情图片帖文，诱导境外男性用户添加指定Line账号。随后，受害人被引导至境外团伙进行裸聊，其通讯录及不雅视频则被同步窃取，用于后续的敲诈勒索，该团伙从中赚取引流佣金。
7月12日，汉寿县公安局反诈、刑侦等多部门联合收网，现场抓获帅某朋等3名嫌疑人，查获电脑5台及带VPN功能的定制U盘等全部作案工具。目前，帅某朋等三人已因涉嫌非法利用信息网络罪被依法刑事拘留。警方正持续深挖案件涉案流水、上下游犯罪链条，并全力推进追赃挽损工作。
