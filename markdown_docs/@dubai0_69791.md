# 阿联酋释放55名伊朗渔民，结束数月拘留

📣 阿联酋释放55名伊朗渔民，结束数月拘留
伊朗驻阿联酋大使馆近日证实，此前被阿联酋海岸警卫队拘留的55名伊朗渔民已于当地时间7月14日获释。据大使馆披露，这些渔民在过去数月内因该地区“特殊情况”被扣押。目前，使馆正积极协调，安排这些获释渔民返回伊朗。
