# 马国否认刘特佐曾入境谈判1MDB案

🚨 马国否认刘特佐曾入境谈判1MDB案
马来西亚财政部副部长刘镇东今日在国会下议院明确否认了在逃富商刘特佐曾秘密入境马国，并驳斥了其与政府就一马公司（1MDB）贪腐案还款事宜进行谈判的市场传闻。
今年5月，曾有媒体报道称，一马公司案核心人物刘特佐于去年底在一个中国代表团的严密保护下秘密入境马国，并与财政部举行秘密会议，商讨通过特定合作来清算和撤销一马公司的债务。对此，大马移民局总监当时也表示毫不知情。
刘特佐涉嫌挪用一马公司至少183亿令吉资金，目前在美马两国面临多项指控，自2015年案发后行踪成谜。
