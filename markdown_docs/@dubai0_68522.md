# 老挝重拳捣毁跨国电诈：百余人被捕，90名中国人涉案

🚨 老挝重拳捣毁跨国电诈：百余人被捕，90名中国人涉案
据老挝塞公省公安厅最新通报，近日当地警方联合多部门展开专项行动，成功捣毁了藏匿于拉曼县和塔登县民居内的9处特大外籍电信诈骗及非法偷渡窝点。
此次行动共抓获外籍嫌疑人101名（含6名女性），其中中国籍90人、日本籍7人、柬埔寨籍4人。警方将涉案人员分为两类处理：其中电诈团伙63人，现场缴获智能手机1331部、电脑9台、银行卡43张，并发现手枪1支及部分毒品；另有38人为非法入境人员，查获随身手机41部及大量现金。
此次打击行动不仅揭示了跨境电信诈骗活动的复杂性与蔓延趋势，也反映出东南亚地区对非法活动的持续高压态势。据了解，部分团伙成员此前或从周边国家转移至老挝，试图重新设立据点，但最终仍被警方查获。目前，所有涉案人员及大量作案工具已被控制，案件正在进一步审理中。
