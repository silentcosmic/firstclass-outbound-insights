# 柬埔寨雷霆行动：170名涉诈涉毒中国籍人员遭驱逐

🚨 柬埔寨雷霆行动：170名涉诈涉毒中国籍人员遭驱逐
柬埔寨移民总局与中国驻柬埔寨大使馆于7月2日联合采取行动，在德崇国际机场依法驱逐了170名中国籍人员，其中包括8名女性。
这些被驱逐者被指控涉嫌多项严重违法犯罪活动，包括高科技网络诈骗、非法贩毒吸毒，以及无护照和非法居留等问题。
柬方重申，将对涉诈犯罪的幕后主使展开无例外严厉打击，并承诺全面营救受害者，协助他们安全回国。
