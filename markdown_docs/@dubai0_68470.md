# 波贝突袭清查：逾80名印尼籍人员被盘查，涉诈手机被缴

🚨 波贝突袭清查：逾80名印尼籍人员被盘查，涉诈手机被缴
近日，柬埔寨波贝市及马莱县等地警方展开了针对多处嫌疑据点的清查行动，重点打击非法活动及潜在的网络诈骗。
在乌茨劳分区的一栋五层出租楼内，执法人员盘查了81名集体居住的印尼籍男子。经初步调查，这些人员均受雇于当地一家赌场。尽管现场未查获明确的电诈工具，但因一名印尼籍主管集中扣留了所有人员的护照，警方已将其带回警局协助调查。
随后，执法人员在马莱县的另一处出租屋中，成功抓获一名印尼籍男子，并当场缴获11部涉嫌用于网络诈骗的手机。目前，该涉案嫌犯及相关证物已移送至省警局，将接受进一步的法律审理。此次行动再次凸显了当地政府对打击非法网络活动，尤其是在边境地区加强监管的决心。
