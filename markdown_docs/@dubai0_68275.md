# 柬埔寨打击电诈：173名中国嫌犯被遣返

🚨 柬埔寨打击电诈：173名中国嫌犯被遣返
6月24日，柬埔寨移民局调查与程序执行局与中国驻柬埔寨大使馆合作，通过德崇国际机场的包机，成功将173名中国籍嫌疑人（其中包括2名女性）遣返回国。
官方通报指出，这些被遣返人员均涉嫌参与网络科技诈骗，并存在非法居留及非法务工行为。此次联合行动，再次彰显了柬中两国在打击跨境犯罪、维护区域秩序方面的紧密协作与坚定立场。
