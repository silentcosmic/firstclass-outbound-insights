# 美国将投放25万枚“7·4”纪念硬币入市

📣 美国将投放25万枚“7·4”纪念硬币入市
美国铸币局宣布，为纪念国家独立250周年，将发行25万枚带有特殊“7月4日”标记的25美分硬币。
这些不带铸币厂标记的纪念币将随机混入日常流通货币中，供民众在交易中惊喜发现。铸币局计划在7月4日前将这批硬币分发至全国各银行及金融机构。
铸币局局长保罗·霍利斯表示，此举旨在鼓励人们通过寻找和分享这些特殊硬币，共同铭记这一历史性的独立时刻。
