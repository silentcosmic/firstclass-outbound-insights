# 香港机场700万黄金劫案：警方速破，7人落网或涉内鬼

🚨 香港机场700万黄金劫案：警方速破，7人落网或涉内鬼
香港国际机场近日发生一起价值700万港元的黄金劫案。一批从海外运抵的黄金在机场停车场遭精准抢劫，引发各界关注。
据悉，一名36岁男子受委托从印尼巴厘岛携带约6公斤黄金返港。6月18日凌晨，当其抵达机场并前往停车场取车时，突然遭到多名蒙面持刀男子袭击，腿部和手臂受伤，背包内6条金条被抢走。
案件发生后，香港警方迅速展开调查。通过监控录像和情报分析，警方在约12小时内成功锁定涉案团伙成员，并已拘捕7名嫌疑人，包括4男3女，年龄介于20至39岁，其中部分人员有黑社会背景。
警方调查显示，嫌犯对受害人的行程、抵港时间及停车地点掌握精准，怀疑案件背后存在熟人泄密或内部信息外流。该团伙分工明确，从策划、跟踪到实施抢劫和逃离，整个过程均经过事前周密安排，显示出高度组织性和预谋性。
目前，被抢黄金尚未追回。警方同时确认，部分涉案人员已在案发后逃往内地，香港警方正通过跨境执法协作机制展开追捕。调查工作仍在持续，不排除有更多人员参与策划或协助藏匿赃物。
