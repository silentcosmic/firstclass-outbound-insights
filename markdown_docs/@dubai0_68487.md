# 梅西破纪录再进一球：阿根廷世界杯三连胜晋级

⚽ 梅西破纪录再进一球：阿根廷世界杯三连胜晋级
在近期举行的美加墨世界杯J组第三轮比赛中，卫冕冠军阿根廷队凭借出色的定位球战术，以3:1力克约旦队，成功锁定小组三连胜，强势晋级淘汰赛。
本场比赛中，巨星梅西替补登场并贡献一粒进球，将其个人在世界杯赛场上的总进球数改写为19个，再次刷新了由自己保持的纪录。
