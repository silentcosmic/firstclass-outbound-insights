# 区域局势缓和驱动：海湾地区6月原油出口量显著反弹

中东海湾地区的石油出口量在六月份实现了显著反弹。得益于区域紧张局势的缓和以及霍尔木兹海峡航运秩序的逐步恢复，该地区原油的日均出口量已突破1000万桶大关。然而，这一数字与冲突前的历史高位相比，仍存在一定差距。
根据最新市场数据，沙特阿拉伯、阿拉伯联合酋长国、科威特、伊拉克和伊朗这五大产油国，在六月份的原油及凝析油日均出口总量攀升至1007万桶。与五月份相比，这一数字实现了超过350万桶的强劲增长，显示出市场信心的迅速恢复。
另一项独立能源分析机构的评估则指出，六月份该地区的日均出口量约为1020万桶，远高于五月份的约700万桶。即便如此，与去年同期约1650万桶的峰值水平相比，当前出口量依然存在40%的下降空间，揭示了全面复苏之路的漫长性。
综合多家权威机构的统计，阿拉伯联合酋长国在本轮出口复苏中表现尤为突出，成为增长速度最快的国家。六月份，阿联酋的日均原油出口量飙升至370万至380万桶之间，创下了历史新高，较五月份环比增长逾100万桶。
业内专家分析指出，自美国与伊朗达成谅解备忘录并推动霍尔木兹海峡的通航逐步常态化以来，此前滞留在海湾水域的大批原油库存得到了快速的疏解。当前，约有2300万桶原油仍在等待通过这一战略要道，而回顾四月底，这一积压量曾一度高达9600万桶，足见近期疏通效率之显著。
