# 柬埔寨神秘“科技园”涉命案：资料已报中国警方

🚨 柬埔寨神秘“科技园”涉命案：资料已报中国警方
市场消息披露，柬埔寨磅湛省72号公路附近新出现一个名为“科技园区”的神秘运营实体。据称，该园区此前在东南亚地区从未有过公开曝光，其背后运作模式及具体业务内容尚不明确。
爆料人指出，该园区主要负责人包括被称为“树总”的老板，以及来自福建龙岩的二老板“泰哥”，物业管理由“静姐”负责。其中，“泰哥”被指主要负责园区日常管理，并有消息称其在柬埔寨境内已涉嫌多起命案。更骇人听闻的是，泰哥的司机“狼头”据称专门负责处理埋尸事宜。
据称，一名与该园区相关的人员在死里逃生后，已将所有相关资料提交给中国警方。此事引发了对该神秘园区及其运营者的高度关注，后续发展值得密切追踪。
