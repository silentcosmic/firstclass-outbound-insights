# 越南Techcombank高管变动：首席信息官阮端卸任，六年科技转型成绩斐然

越南领先银行Techcombank的首席信息官阮端（Tuan Nguyen）已正式卸任，结束了其长达六年的任期。
阮端先生通过职业社交平台LinkedIn对外公布了这一消息，明确表示他的任期已于六月底画上句号。
阮端先生于2020年7月加入Techcombank，此后一直主导并推动了该行的全面技术转型，其中包括数字银行解决方案的构建以及人工智能能力的深度开发。
在其领导下，Techcombank不仅成功打造了多个新型生态系统业务，更在技术与人工智能领域实现了显著的能力拓展。
尽管阮端先生尚未透露其下一站职业规划，但他明确表达了与美国及亚洲地区董事会、投资者及高管团队合作的浓厚兴趣。
阮端先生拥有逾二十年的丰富经验，其职业足迹遍布银行业、金融服务、保险以及科技领域。
在加盟Techcombank之前，阮端先生曾在花旗银行（Citi）担任拉丁美洲支付与应收账款技术主管一职。
此前，他还曾在Travelers、Finastra、拉斯维加斯金沙集团（Las Vegas Sands）、威科集团（Wolters Kluwer）以及通用电气能源（GE Energy）等知名企业担任高级技术管理职务。
