# 泰国突发惨剧：11岁男童驾车致8名僧侣身亡

🚨 泰国突发惨剧：11岁男童驾车致8名僧侣身亡
泰国穆达汉府于7月2日上午发生一起严重交通事故。一名11岁男童在未经家人允许的情况下，私自驾驶家中皮卡外出，不幸在行驶途中车辆失控，冲入一支正在徒步修行的僧侣队伍，导致至少8名僧侣遇难，多人受伤，事件震惊了当地社会。
据悉，遇难僧侣来自穆达汉府佛足印山寺，当时正与信众进行年度徒步修行，计划从穆达汉府前往乌汶府。警方通报指出，事故发生时有5名僧侣当场身亡，另有3名伤者在送医抢救后不治，使遇难人数增至至少8人。此外，多名僧侣及信众受伤，部分伤势严重，已送往穆达汉医院救治。
警方初步调查确认，肇事者为一名年仅11岁的男童。他趁家人不备私自驾车外出，行驶约10公里后车辆失控酿成惨剧。鉴于其未达刑事责任年龄，警方将与儿童保护部门联合处理此案，并深入调查监护人责任及车辆管理是否存在疏漏。
截至目前，事故具体原因仍在进一步调查中。部分国际媒体指出，最终伤亡数据仍需等待泰国官方的进一步确认。
