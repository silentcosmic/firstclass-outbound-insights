# 阿布扎比地毯市场火灾：无人员伤亡

🔥 阿布扎比地毯市场火灾：无人员伤亡
周五上午，阿布扎比扎耶德港附近的地毯市场突发火灾。由阿布扎比警方与民防局组成的联合应急小组迅速响应，已成功将火势扑灭。
据官方通报，此次火灾未造成任何人员伤亡。警方表示，周边道路交通运行正常，未受到明显影响。相关部门呼吁公众关注官方发布的核实信息，避免传播未经证实的消息。
