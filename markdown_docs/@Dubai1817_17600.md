# 泰国警局突发枪击：副督察中弹身亡

🚨 泰国警局突发枪击：副督察中弹身亡
泰国春蓬府帕提奥警察局后方警员宿舍于6月29日凌晨3时46分发生一起枪击事件。一名58岁副督察在住宅内中弹重伤，经紧急送医抢救无效后不幸身亡。
据现场勘查，受害者头部被9毫米子弹击中，伤势严重。目前，当地警方已全面介入调查，正从多角度排查案件具体原因，包括是否涉及他杀或其它因素，相关证据正在积极收集中。
