# ️ Polymarket涉嫌社媒“假赢”骗局：诱导美国用户

⚠️ Polymarket涉嫌社媒“假赢”骗局：诱导美国用户
华尔街日报的一项最新调查揭露，预测市场平台Polymarket（通过其关联公司Virality）涉嫌秘密资助社交媒体创作者，通过TikTok、Instagram和YouTube大量发布虚假的“赌赢”视频内容。这些内容旨在引诱美国用户访问其境外运营平台。
调查发现，这些创作者利用虚假的镜像网站模拟从未真实发生的“获胜”场景。在分析的10位受邀创作者发布的1,105个视频中，约七成视频展示了虚假的赌局，涉及金额高达约190万美元，而实际关联账户的真实损失仅约16.6万美元。知名主播Adin Ross也曾签下数百万美元的推广协议为该平台造势。
据统计，相关推广活动累计获得约1.4亿次观看，其中超过60%的目标受众为美国用户。调查还指出，创作者们通常被要求对收取报酬一事保密，进一步加剧了信息的不透明性。
