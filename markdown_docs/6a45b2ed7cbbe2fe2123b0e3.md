# 出海观察：越南信贷评分金融科技Trusting Social母公司2025年净亏8750万美元

2025财年，尽管营收持续增长，越南信贷评分金融科技公司Trusting Social却再度陷入亏损境地。
据其新加坡注册母公司Trust IQ的财务披露显示，该公司当年录得8750万美元的净亏损。
Trusting Social主要为金融机构提供信贷评分和数据分析服务，旨在帮助它们评估那些传统信用记录有限的消费者。
这一业绩与2024年的3380万美元盈利以及2023年的3190万美元亏损形成对比。
2025年，公司营收达到2310万美元，高于前一年的2120万美元和2023年的1810万美元。
金融负债的重估对公司业绩造成了沉重压力。
2025年，Trust IQ因此项负债计提了7760万美元的公允价值损失，而前一年同期该项目曾带来4180万美元的收益。
截至2025年末，Trust IQ的总资产从一年前的5480万美元降至4480万美元。
负债总额从1.23亿美元攀升至2.016亿美元，同期现金储备为2340万美元。
结合2022年的融资以及约2700万美元的净流动资产，公司预计自财务报表发布之日起，其资金足以维持至少12个月的运营。
2022年的融资是指来自越南综合企业Masan Group计划进行的1.05亿美元投资的首批款项，即6500万美元的C轮融资。
该交易预计将使Masan获得Trusting Social 25%的股权。
在Masan投资之前，Trusting Social已累计筹集约4400万美元，早期投资者包括红杉资本（现为Peak XV）、BEENEXT、500 Global和Tanglin Ventures。
2025年，分销和营销成本从1030万美元降至860万美元，而行政开支则增长20%，达到2410万美元。
在Trusting Social的营收构成中，越南市场贡献最大，领先于菲律宾和印度尼西亚。
该公司宣称已为超过10亿消费者提供信用评分服务，并与130多家金融机构建立了合作关系。
