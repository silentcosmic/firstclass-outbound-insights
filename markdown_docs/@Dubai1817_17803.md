# 越南重拳打击赌球：捣毁超1.3亿美元网络

🚨 越南重拳打击赌球：捣毁超1.3亿美元网络
据法新社报道，越南警方近日宣布，在全国范围内的世界杯非法赌球专项行动中，成功捣毁了两个涉案金额高达1.33亿美元的特大地下网络赌博团伙，并逮捕了85名嫌疑人。
胡志明市警方透露，此次行动于六月底启动。经初步调查，这两个网络组织严密，累计非法交易额已达1.33亿美元。
越南公安部上周曾表示，在世界杯开赛前20天内，警方已在全国范围内取缔73个非法赌博窝点，逮捕346名涉嫌参与赌球的人员，涉案总金额高达数亿美元。此次行动是越南持续强化对非法赌博活动打击力度的最新体现。
