# 中越边境合作升级：严打跨国犯罪

🤝 中越边境合作升级：严打跨国犯罪
2026年7月1日消息，为有效遏制跨国犯罪活动，中国与越南两国边境合作已进一步升级。双方已达成一致，将继续严格落实此前签署的三项法律文件，旨在深化执法协作，显著提升联合打击犯罪的效能。
此举标志着两国在共同维护边境地区安全与稳定方面迈出了关键一步，对区域治理具有重要意义。
