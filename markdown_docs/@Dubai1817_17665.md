# 新加坡警方：警惕视频会议与世界杯加密诈骗新陷阱

🚨 新加坡警方：警惕视频会议与世界杯加密诈骗新陷阱
新加坡警方近日发出紧急警告，指出不法分子正利用视频会议和世界杯等热门话题，设计复杂圈套实施加密货币诈骗，已导致部分受害者蒙受巨大经济损失。
其中一种手法是，诈骗者冒充商业伙伴或熟人，通过聊天软件发送伪装成视频会议的恶意链接。一旦点击，受害者会被诱导下载“过期软件组件”或执行指令，实则安装了窃取个人信息的恶意软件。随后，诈骗者能远程控制受害者设备，监控加密货币操作，窃取钱包资料，并诱导批准看似无害的“授权签名”。这种授权允许诈骗者在未来无需再次确认即可转移资产，甚至连硬件钱包也可能被攻破，受害者往往难以立即察觉。
此外，警方还发现有诈骗网站以世界杯门票、官方纪念品、免费直播及“世界杯官方代币”为诱饵，要求受害者使用比特币或泰达币付款。这些钓鱼网站会进一步诱导用户连接加密货币钱包、安装不明软件或批准交易授权。为增强可信度，诈骗者甚至会利用运动员和明星的AI换脸视频。
警方特别提醒公众，务必提高警惕，切勿点击陌生来源的视频会议或世界杯相关链接，不下载不明文件或执行陌生指令，更不要随意批准任何钱包签名或授权请求。一旦发现钱包出现异常，应立即联系加密货币交易平台停止交易，撤销可疑授权，并将剩余资产转移至安全的新钱包，同时尽快向警方报案。
