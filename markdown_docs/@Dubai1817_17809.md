# 泰国大麻出境警示：跨境走私案频发，企业与个人面临严刑峻法

🚨 泰国大麻出境警示：跨境走私案频发，企业与个人面临严刑峻法
近期，泰国警方连同国际执法机构密集通报多起重大跨境大麻走私案件，并再次向公众发出严厉警示：尽管泰国已部分合法化医用大麻，但这绝不意味着可以将其携带出境。任何未经许可的跨境运输行为，都将严重触犯泰国及目的地国家的法律法规。
据警方披露，近期查获的案件规模惊人，包括香港海关截获约23公斤泰国来源大麻并拘捕涉案人员；印度尼西亚查获约3.37吨泰国来源大麻，涉及多国犯罪网络；以及波兰警方在泰国出口集装箱内发现约1.2吨大麻，多人被捕。这些案例明确指出，国际社会对大麻走私的打击力度正在持续升级。
泰国警方强调，合法医用大麻的使用范围仅限于境内，严禁非法携带大麻产品离境，更不可替他人“带货”出境，或通过行李、包裹等方式运输。一旦涉及跨境走私，行为人将面临严重的刑事处罚。当局提醒所有出入境人员，切勿接受陌生人携带物品的要求，警惕所谓的“兼职带货”或“免费托运”陷阱，并务必在出境前仔细检查个人行李，以防误带含大麻成分的食品、药品或护肤品。
许多游客和商务人士误以为“泰国可以购买，便可携带离境”，这是一个严重的法律认知误区。各国和地区对大麻的管控标准差异巨大，跨境携带可能导致高额罚款、长期监禁，乃至更严重的法律后果。
