# 埃及足球晋级世界杯16强：迪拜深夜万人空巷庆祝！

🎉 埃及足球晋级世界杯16强：迪拜深夜万人空巷庆祝！
上周五深夜，阿联酋各地洋溢着浓厚的节日气氛。埃及国家足球队在世界杯点球大战中以3比2力克澳大利亚，历史性地首次晋级16强，引发了居住在阿联酋的埃及社群的巨大欢腾。
凌晨1点半，迪拜和阿治曼的街头化作一片红色海洋，无数身着国家队球衣的球迷，包括众多家庭与儿童，涌上街头载歌载舞，共同庆祝这一里程碑式的胜利。据悉，迪拜酋长也深夜发文，向埃及队的出色表现表示祝贺。
