# 马来西亚沙巴遣返再升级：年内累计84名中国公民被驱逐

🚨 马来西亚沙巴遣返再升级：年内累计84名中国公民被驱逐
马来西亚沙巴州移民局于2026年7月9日通过亚庇国际机场，将12名中国籍非法移民遣返回广州。此次被遣返人员均为成年男性，他们此前因涉嫌逾期逗留或非法入境等违反移民法行为，在完成法律程序并经中国驻马来西亚大使馆核实身份后被扣押并最终遣返。
沙巴州移民局强调，截至本次行动，今年已累计遣返84名中国公民。当局表示将持续严厉打击非法移民及滥用护照、入境便利等违法行为，以维护边境管理秩序。
