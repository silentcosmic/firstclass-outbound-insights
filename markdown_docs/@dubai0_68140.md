# ️ 使馆警告：泰缅边境诈骗猖獗，赴达府需警惕

⚠️ 使馆警告：泰缅边境诈骗猖獗，赴达府需警惕
中国驻泰国大使馆近日发出安全警告，提醒赴泰中国公民高度警惕泰缅边境地区，特别是达府（Tak Province）及其湄索（Mae Sot）周边。鉴于跨境电信诈骗及人口贩运案件持续高发，使馆特别指出，妙瓦底（Myawaddy）等区域已成为此类犯罪的温床。
许多中国公民因轻信网络高薪招聘或虚假旅游考察信息，被诱骗至这些区域，随后遭受人身自由限制，甚至被迫参与违法活动。对此，大使馆强调，务必通过正规渠道获取信息，切勿轻信陌生人的诱惑，并避免前往上述高风险敏感区域。
此外，出行前应提前告知家人行程，妥善保管个人证件，确保通讯畅通。如遇紧急情况，应立即向当地警方报案，并及时联系中国驻泰国使领馆寻求协助。
