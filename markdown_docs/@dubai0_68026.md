# 柬埔寨RPR：太子地产客户须限期更新产权信息

🚨 柬埔寨RPR：太子地产客户须限期更新产权信息
柬埔寨房地产和抵押贷款业务监管局（RPR）近日发布重要通知，要求太子控股集团（Prince Holding Group）旗下九个房地产项目的业主，在2026年7月特定时段内，完成产权信息的提交与更新工作。
RPR已公布详细的办理时间表，指定2026年7月6日至31日期间，对太子·幸福广场、太子·国际广场、钻石一号、太子·现代广场、太子·中央广场、太子·金海湾、太子·阳光壹号、太子·寰宇中心以及Samraong住宅区等九大项目分批进行。客户需密切关注其所属项目的具体办理日期与时间。
办理地点设在金边市雷西郊区雷西郊分区4村的房地产和抵押贷款业务监管局文件接待处，工作时间为工作日的上午8:30-11:30及下午1:30-4:30。业主前往办理时，须携带房屋买卖合同、身份证明、付款凭证（包括发票、收据、转账记录等）、交房确认书，以及开发商名下的不动产权证原件及复印件。对于尚未全额付清房款的客户，则需提供已付部分的款项证明。
RPR特别强调，若客户未能按期提交所需文件，由此产生的任何个人利益损失，监管局将不承担任何责任。此举旨在规范当地房地产市场产权管理，保障业主合法权益，同时也提醒相关投资者务必重视此类官方通知。
