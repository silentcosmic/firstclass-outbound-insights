# 波贝突袭：近40名跨国诈骗嫌犯被捕

🚨 波贝突袭：近40名跨国诈骗嫌犯被捕
柬埔寨波贝市近日发生一起重大执法行动，警方突击检查了当地五处可疑地点，成功捣毁一个跨国网络诈骗窝点。
行动中，近40名涉嫌诈骗的外籍人员被捕，其国籍涵盖印度、泰国、尼日利亚及中国等多个国家。警方在其中一处窝点查获大量作案工具，包括电脑、手机，并惊人地发现了假冒中国警用装备及标志。随后，警方又在其他地点追查并扣留了多名涉案的中国籍无护照人员。
所有涉案人员已被移送至省警察局接受进一步审讯，相关车辆及大量作案工具也已被依法暂扣。此案再次凸显了东南亚地区跨境电信诈骗活动的复杂性与风险。
