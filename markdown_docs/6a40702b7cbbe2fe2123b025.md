# 瑞典发展基金注资1500万美元，赋能菲律宾中型企业加速增长

瑞典发展基金（Swedfund）日前宣布，已承诺向Navegar三期基金（Navegar Fund III）注资1500万美元。该基金专注于投资菲律宾的中型企业。
瑞典发展基金在一份声明中指出，这项投资旨在帮助当地企业实现增长，强化其运营能力，并创造更多高生产率且优质的就业机会。
这笔资金将为消费服务及商业服务领域的中型企业提供资本支持，助力其业务扩张并实现运营正规化，覆盖行业包括医疗保健、食品分销和物流等。
此举将有助于构建更稳健的本地价值链，并有望改善菲律宾全国范围内对基本商品和服务的获取。
瑞典发展基金通过提供长期资本和积极参与所有权管理，致力于支持企业发展韧性，提升就业质量，并促进更具包容性的经济增长。
这项投资亦与联合国可持续发展目标中的“体面工作与经济增长”子目标高度契合。
这项战略性投资符合瑞典发展基金的主题基金投资策略，强化了其在东南亚地区的市场布局，尤其支持那些具备强劲包容性增长和就业创造潜力的市场。
Navegar三期基金是一支私募股权基金，专注于投资菲律宾中型企业，其目标募资规模为2.5亿美元。
该基金由总部位于马尼拉的Navegar公司管理，通过提供长期资本和积极的所有权管理，支持菲律宾国内经济关键领域的业务增长。
声明显示，菲律宾经济近年来保持了稳健增长态势，然而，诸多中小型企业在获取长期资本方面仍面临挑战。
目前，约有70%的劳动力处于非正式就业状态，这些岗位普遍缺乏稳定性、社会保障以及技能提升的机会。
因此，中型企业作为重要的雇主和服务提供者，其发展面临资源短缺，难以有效提升生产力并强化公司治理。“创造更多高生产率和正式的就业岗位，对于实现包容性经济发展至关重要。”瑞典发展基金食品系统与战略投资总监海伦·哈格斯（Helen Hagos）表示，“通过帮助成长中的企业获得扩张所需的资本，这项投资旨在增强菲律宾的私营部门实力，并为当地可持续的就业创造贡献力量。”
