# 非洲最大炼油厂首购阿联酋原油，全球油市变局显现

🔥 非洲最大炼油厂首购阿联酋原油，全球油市变局显现
非洲最大的尼日利亚丹戈特炼油厂近期首次向阿联酋国家石油公司（ADNOC）采购了200万桶原油，此举标志着该巨型炼油厂原油供应策略的关键转变。
这笔交易恰逢全球原油市场显著走软。随着美伊紧张局势缓和、霍尔穆兹海峡石油运输量回升，以及亚洲需求增长乏力，中东地区的原油供应正寻求更广阔的出口渠道。
市场消息人士向路透社透露，丹戈特炼油厂在6月份进口了包括Umm Lulu以及Das或Murban在内的阿联酋原油。对此，丹戈特炼油厂和阿联酋国家石油公司均未立即置评。
尽管丹戈特炼油厂通常每月可从尼日利亚国家石油公司获得5至7船原油，并享有运输成本优势，但其工厂满负荷运转每月约需13至15船。此前，该厂已成为欧洲中质馏分油的主要出口商，并曾于5月份采购利比亚原油，显示其在全球原油供应链中的积极布局。
