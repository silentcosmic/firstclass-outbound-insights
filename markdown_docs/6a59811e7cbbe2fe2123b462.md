# 乔治亚-太平洋：构建自有程序化媒介团队，解锁营销新效能

了解消费者购买卫生纸，或许无需复杂的内部营销组合模型。然而，若要精准洞察媒介投放方案中哪些环节卓有成效，哪些亟需优化调整，则势必需要更深入的掌控与分析。
在全球领先的纸巾、纸浆、卫生纸及包装产品制造商乔治亚-太平洋（Georgia-Pacific），哈维尔·布斯蒂略斯（Javier Bustillos）肩负着解答这一关键问题的重任。他负责旗下Angel Soft、Brawny、Dixie、Quilted Northern和Vanity Fair等多个知名品牌的全国性媒介及营销分析业务。
布斯蒂略斯的职责拓展，源于一个普遍存在的困境：数字营销成本不断攀升，而资金流向的透明度却未能同步提升。
布斯蒂略斯在一次行业交流中透露：“早在2019年，我们与一家媒介代理机构合作，负责数字媒介投放。当时我们正大幅增加在数字媒介上的投入，但对于所支付的全部费用，缺乏足够的透明度。”
这一情况预示着乔治亚-太平洋的媒介采购模式亟待变革。若要真正掌控数字支出，核心业务环节便不能再一味依赖外部机构。
首要举措是探索内部团队能独立运营哪些业务，以及亲力亲为的操作模式能否显著提升媒介效能。
布斯蒂略斯及其团队选择了一个品牌和一个单一渠道作为试点，进行重点攻坚。
他表示：“我们起步很谨慎。核心理念是，如果能证明其价值——不仅是费用上的节省，更包括媒介投放的实际效果和成本效率——我们便会逐步扩大范围。”
这并非一蹴而就。乔治亚-太平洋耗时近四年，才将最初的试点逐步发展成为成熟的内部媒介运营体系。其中涉及大量的技术配置工作，然而， 상당部分任务实际上与数据流和仪表盘的设置并无直接关联。
组建内部团队，意味着需要说服那些程序化交易员和数据分析专家——通常他们可以选择在代理机构、平台或硅谷大型科技公司发展事业——来到一家纸巾和纸张制造商开启职业生涯。
布斯蒂略斯指出：“人们普遍认为，将媒介业务内包能够节省大量人才成本，因为无需支付代理机构费用。但实际情况是，招募和留住这些专业人才充满挑战。”
接下来是选择合适的广告技术供应商，并在没有代理控股公司作为谈判筹码的情况下进行议价。
布斯蒂略斯表示：“在缺乏规模效应的情况下，争取到有利的合作费率确实异常困难。”
为实现资金利用率最大化，乔治亚-太平洋将其程序化媒介支出集中于少数核心合作伙伴。目前，公司将雅虎（Yahoo）作为全国性采购的主要DSP（需求方平台），同时根据不同营销活动的需求，灵活结合谷歌DV360（Google DV360）、The Trade Desk以及亚马逊（Amazon）的资源。
随着更多采购业务转向内部，乔治亚-太平洋渴望获得更深层次的透明度，具体而言，即对日志级数据的访问权限。
许多广告技术供应商并不愿提供如此高层级的透明度。然而，布斯蒂略斯表示，乔治亚-太平洋并未遭遇此难题，因为公司在“合同谈判初期就明确提出了这一要求”。
乔治亚-太平洋坚持要求合作伙伴提供日志级数据，并在合同中明确纳入定期媒介审计条款，以确保实际投放和绩效与既定计划保持一致。
这一切自然引出了一个问题：媒介代理机构的角色还剩下什么？
布斯蒂略斯透露，自2020年起担任乔治亚-太平洋媒介代理机构的OMD，仍负责整体策略规划，并处理那些规模效应和人际关系至关重要的工作，例如电视广告投放。
与此同时，乔治亚-太平洋则专注于那些需要直接掌控和快速反馈的数字渠道，业务范围涵盖程序化购买、搜索营销、社交媒体、营销组合模型（MMM）、营销分析、A/B测试以及销售增长分析。
如今，内部团队每年进行两次营销组合模型分析，并计划最终过渡到季度分析，并根据设备、受众和广告类型进行更细致的切分。
尽管已部署了诸多机制，布斯蒂略斯坦率承认，并非所有内部化举措都取得了压倒性胜利。有些渠道的突破难度高于预期，少数早期测试的成效也未能达到乔治亚-太平洋在展示广告和在线视频领域所见证的提升。
乔治亚-太平洋已尝试了不同的联网电视（CTV）采购方法，包括程序化保量和私有市场交易。但正如布斯蒂略斯所言，团队“尚未达到能将所有CTV采购完全内部化的阶段”，因此在前期集中采购时仍需依赖OMD的规模优势。
布斯蒂略斯表示，这也是为何混合模式对乔治亚-太平洋而言依然具有重要意义的原因之一。
他强调：“我们并没有将所有业务都内部化的设想，也无需代理机构来执行我们自己就能完成的工作。我们所需要的是，代理机构能提供互补性的能力，从而使双方都能贡献出独特的价值。”
