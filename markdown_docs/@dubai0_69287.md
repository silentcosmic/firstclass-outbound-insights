# ️ 迪拜居民家中惊现黑寡妇，官方紧急提醒

⚠️ 迪拜居民家中惊现黑寡妇，官方紧急提醒
近日，迪拜一居民在家中发现罕见的黑寡妇蜘蛛群，引发了社区关注和有关部门的介入。该居民在Emaar South Parkside 2住所清理庭院时，先发现一只，随后在自行车防尘罩下又找到约八只。黑寡妇蜘蛛并非阿联酋本土物种，其在迪拜出现极为罕见，毒液虽能引起剧烈疼痛但极少致命。
事件在社区群组中引发讨论，有居民质疑为何不直接消灭。对此，迪拜市政府已发布明确指引：公众若发现疑似危险动物，应立即致电相关部门。市政府服务中心表示，居民只需提供Makani编号，专业团队将迅速上门进行安全处置。
