# 暹罗商业银行率先采用花旗全天候美元清算方案，实现跨境支付新突破

暹罗商业银行（SCB）已正式启用花旗银行集成的全天候（24/7）美元清算与代币化服务解决方案，使其成为花旗在全球范围内的首家金融机构客户。
这项创新服务旨在赋能企业及机构客户，使其能够在常规银行营业时间之外，包括周末和法定节假日，实现近乎实时的跨境美元支付。
花旗银行的代币化服务（Citi Token Services）采用了私有许可区块链技术，能够将花旗全球网络内持有的存款进行代币化处理，并严格遵循既定的监管银行体系运作。
该代币化服务已与花旗的全天候美元清算解决方案深度整合，后者目前连接了全球50多个市场中的300多家金融机构。
在双方合作框架下的首笔交易，由华侨证券（泰国）（Phillip Securities Thailand）成功完成。华侨证券是华侨金融（PhillipCapital）的子公司，同时也是暹罗商业银行的重要客户。
这笔美元资金的转移发生在某美国公共假期周末期间，具体流程为：从华侨金融旗下的另一子公司华侨资本公司（Phillip Capital Inc）在花旗伦敦开设的账户，转账至暹罗商业银行位于泰国的收款人账户。
暹罗商业银行对此项合作的战略意义强调指出：“通过与花旗银行的深度协作，我们成为泰国首家运用代币化技术提升跨境美元处理能力的银行。此举显著帮助我们的客户简化操作限制，并以更大的灵活性开展其国际业务活动。”
花旗银行方面，对该方案的行业创新性与未来愿景阐释道：“我们与花旗代币化服务深度整合的增强型全天候美元清算解决方案，是当前行业内的里程碑式创新，它有效地连接了传统与数字金融通道，从而扩展了我们为客户提供常态化服务的整体产品组合。这一进展进一步巩固了我们‘网络之网络’的发展策略，助力我们构建起具有互操作性、能够支持多银行、多市场和多网络的强大功能体系。”
