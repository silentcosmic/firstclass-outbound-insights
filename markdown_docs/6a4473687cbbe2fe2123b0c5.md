# 全球产业洞察 | Kimi估值跃升至315亿美元，ARR突破3亿美元；优艾智合全球首发具身智能新品，目标赋能万个工业现场

豪华汽车品牌保时捷近期对其中国市场的经销商网络进行策略性调整。据悉，山东济宁、江苏淮安以及广西南宁兴宁保时捷中心已于6月30日终止其经销业务，而安徽芜湖保时捷中心也将于7月31日停止销售业务。尽管销售授权有所变动，上述门店的售后服务仍将正常运行，持续为车主提供符合品牌标准的专业服务。
头部休闲零食企业三只松鼠近日发布公告，其全资子公司三只松鼠（芜湖）坚果智造有限公司已于2026年6月30日获得一笔与收益相关的政府补助资金，金额达2140.36万元人民币。此笔资金占公司最近一个会计年度经审计的归属于上市公司股东净利润绝对值的13.78%，显示出政策对特定产业发展的支持。
英国竞争与市场管理局（CMA）于6月30日启动了针对苹果和谷歌移动平台新一轮监管措施的公开咨询。根据CMA的提案，这两家科技巨头将被禁止限制英国应用开发者引导用户至应用商店外部的网站或平台完成支付、订阅或交易，此举旨在提升市场竞争活力，并减轻开发者对官方应用商店的过度依赖。CMA进一步强调，任何引流相关的收费都应遵循“公平、合理”原则，并预计其费用将低于当前应用商店的佣金水平。同期，CMA也正针对苹果iOS近场通信（NFC）功能的开放技术实现路径及收费机制征求意见，以期在移动支付等服务领域激发更多竞争与创新。
三星电机公司周二披露，其已与一家美国客户达成一份价值4540亿韩元（约合2.93亿美元）的供应合同，将为该客户的人工智能（AI）服务器提供多层陶瓷电容器（MLCC）。三星电机方面透露，此份合同的有效期将从2027年1月延续至2027年12月。尽管基于保密协议未能公布具体客户身份，但业界普遍推测，该美国客户可能是一家大型云服务提供商（CSP）。
英国高端折叠自行车品牌BROMPTON今日宣布，其已与迪卡侬旗下的投资与创新部门Decathlon PULSE以及知名风险投资机构黑蚁资本建立战略合作伙伴关系。在一轮近期完成的股份转让交易中，Decathlon PULSE与黑蚁资本分别对BROMPTON进行了股权投资，获得了少量股份。
爱芯元智开发者生态沙龙于6月28日在深圳成功举办。爱芯元智开源社区解决方案负责人唐琦在活动中指出，截至2026年6月，公司平台模型库已实现突破性进展，模型数量超过200个，全面覆盖大型语言模型（LLM）、视觉语言模型（VLM）、多模态以及垂类检测等多个领域，并已同步入驻Hugging Face和ModelScope两大模型社区。此外，全新的Pulsar2 6.0工具链已完成迭代升级，不仅能适配公司全系列芯片，还针对Qwen、MiniCPM等主流端侧大模型推理进行了优化。据透露，公司下一代大模型专用NPU架构在性能上取得了显著跃升，并能原生兼容PyTorch、Sglang、vLLM等主流框架，为开发者提供更高效便捷的工具。
专注于生物智能领域的“耀速科技”近日宣布完成近2亿元人民币的A+轮融资。本轮融资由国方创投、清松资本、广发信德、东方富海及沄柏资本共同投资，原有股东雅亿资本也继续追加支持。据公司介绍，此轮募集资金将主要用于持续推进其“3D Bio Intelligence（3D生物智能）”平台建设，具体涵盖垂类AI模型与生物机制解析能力的深度提升、高通量自动化实验体系的全面落地、多模态生物数据的持续积累，以及加速虚拟细胞（Virtual Cell）和生物世界模型（Biological World Model）的构建，旨在全面构建新一代药物研发的人源化数据基础设施。
大模型公司月之暗面旗下的Kimi，其上一轮200亿美元估值融资近期已完成交割，而新一轮融资也已启动，投前估值显著上升至315亿美元。据接近Kimi的机构人士透露，该公司在本轮融资洽谈中披露了最新的运营数据：截至6月中旬，其年度经常性收入（ARR）已突破3亿美元大关。Kimi本轮收入增长主要得益于模型迭代所驱动的开发者使用量和API收入的显著提升。当前，API收入已占据Kimi总收入的七成以上，并且这一比例仍在持续增长。Kimi的收入增长轨迹初步呈现出类似Anthropic早期商业化阶段的特点：开发者调用量大幅增加、API收入占比提升、海外付费用户数量增长，以及由模型能力增强所推动的价格体系上调。
6月29日，优艾智合机器人正式发布其具身智能系列新品，其中包括全球首款可规模化应用的工业具身智能大模型“智合”（FabriX），以及工业原生人形机器人“隙锋”。同时，优艾智合宣布启动FabriX生态伙伴计划，并立下“3年内赋能10000个工业现场”的宏伟目标。凭借“一脑多态”的创新架构，优艾智合的机器人解决方案已广泛服务于半导体、锂电、3C、能源等关键工业领域，累计完成超过800个项目落地。根据弗若斯特沙利文的报告显示，若以2024年收入计算，优艾智合在工业移动操作机器人市场的全球占有率位居首位。
高端电动摩托车品牌HORWIN号外近日宣布，全球首款400V高压超级电摩SENMENTI ONE已正式启动交付，起售价为9.98万元人民币。这款创新产品集成了超过200项核心专利，搭载品牌自主研发的车规级400V高压平台，实现了三电系统与底盘的高度集成。其配备16.9kWh三元锂电池，城市综合续航可达300公里，并支持汽车直流快充，仅需充电10分钟即可行驶100公里，同时具备3kW的外放电功能。在安全性与智能化方面，该车型配备了博世ABS以及机器人架构域控系统；性能上，其零百加速仅需3.3秒，最高时速可达200公里/小时，展现出电动出行领域的技术前沿。
在国际小行星日之际，国家航天局宣布，中国将着手建立一套天地协同的近地小行星监测体系，旨在为小行星撞击防御提供关键的预警与监测支持。专家指出，近地小行星撞击是全人类共同面临的潜在威胁。中国作为负责任的航天大国，目前正在积极论证中国小行星防御系统的建设。其中，构建天地一体化监测网络是实现早期预警的核心环节。我国计划通过在地基部署多台望远镜并建设天基星座等方式，打造一个全天候无盲区的监测系统。
美国教育部周一宣布，正在最终确定一套新的联邦学生贷款规定，该规定将高校参与联邦贷款计划的资格与其毕业生的收入水平直接挂钩。此举被视为政府向高等教育机构施压的最新策略。根据教育部声明，在新的“学生学费与透明度体系”（STATS）及“收入问责规则”下，本科项目需证明其毕业生的收入高于普通高中毕业生，而研究生项目则必须证明其毕业生的收入高于普通学士学位持有者，以此确保教育投资的实际回报。
