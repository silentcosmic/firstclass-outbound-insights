# 菲律宾上半年7起绑架案：施害者多为同胞

🚨 菲律宾上半年7起绑架案：施害者多为同胞
中国驻菲律宾大使馆近日通报，2026年上半年菲律宾共发生7起涉及中国公民的绑架案件，主要集中在马尼拉及周边地区。尽管案件数量较去年同期下降约36%，但安全形势仍不容忽视。
使馆披露的案件详情显示，受害者多因社交媒体交友、学车、求职等日常活动，或涉及网络赌博、电信诈骗及债务纠纷而遭遇不测。绑架手段包括强行拖拽上车、通过社交媒体诱骗、非法控制并转移资金，甚至强迫网络借贷。部分受害者在家属支付赎金后获释，也有成功逃脱者。
值得注意的是，这些案件呈现出共同特点：受害者和涉案绑匪多为中国籍人员。中国驻菲律宾使馆对此高度重视，正持续协助两国执法部门打击跨境犯罪。今年上半年，中菲双方已联合抓获并遣返56名中国籍犯罪嫌疑人，其中包含4名涉嫌绑架人员。
