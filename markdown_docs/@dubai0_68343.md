# ️ 北京中信大厦遭飞机撞击：1死13伤

⚠️ 北京中信大厦遭飞机撞击：1死13伤
6月26日傍晚，北京朝阳区东三环附近发生一起罕见事故，一架单发双座轻型运动航空器在飞行中意外碰撞北京中信大厦。
事故导致机上驾驶员当场死亡，另有13人受伤。该栋中信大厦，又称“中国尊”，高528米，是目前北京市最高的摩天大楼，亦为中信集团总部所在地。受撞击影响，大厦部分玻璃受损，机身残骸散落路面，底层一度冒出浓烟，楼内人员紧急疏散。
据了解，相关部门已介入调查事故原因，而现场部分目击者拍摄的视频已在网络平台下架。
