# 泰国突袭豪宅：中国籍赌博头目落网，涉案4亿泰铢

🚨 泰国突袭豪宅：中国籍赌博头目落网，涉案4亿泰铢
6月30日，泰国移民局联合国际刑警组织北京国家中心局，在曼谷成功抓捕中国籍男子黄浩义。黄浩义系中国公安机关通缉的嫌疑人，涉嫌组织运营大型跨国网络赌博集团，包括“TIANHONG INTERNATIONAL”和“PHOENIX INTERNATIONAL”等多个赌博网站。
警方调查显示，该赌博网络在全球拥有逾5万名赌客，平台资金流水累计超过8000万元人民币，约合4亿泰铢。黄浩义案发后潜逃至泰国居住，泰国警方在其外出领取外卖时，于一处豪华住宅将其控制，并撤销其在泰居留许可。
目前，黄浩义已被泰国警方拘留，后续将依法办理遣返手续，移交中国有关部门接受进一步调查和法律处理。
