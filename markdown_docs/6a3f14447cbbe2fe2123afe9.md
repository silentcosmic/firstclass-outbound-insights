# Jumio携手Trinsic拓展数字身份验证版图，网络触达全球60余国

出海 · 头等舱获悉，身份验证技术巨头Jumio近日宣布，通过与领先的数字身份网关Trinsic达成深度技术整合，其数字身份验证网络已成功拓展至全球60多个国家和地区。
此番扩展赋能企业直接运用各国政府发布的数字身份凭证进行用户验证，其中包括新加坡的Singpass和欧洲的eID等，显著简化了过去需逐一处理特定国家认证的复杂流程。
这项创新整合方案主要面向全球金融机构与新兴金融科技企业，旨在帮助他们优化客户入驻体验，并高效满足当地的“了解您的客户”（KYC）合规要求。
随着数字身份凭证与传统实体证件并存的趋势日益显著，此次战略合作彻底消除了跨国企业为适应各地差异而构建多套独立验证体系的冗余需求。
经过升级的平台，在基础凭证核验功能之上，进一步融入了先进的生物识别认证与活体检测技术，大幅提升了验证过程的准确性与安全性。
Jumio凭借其全球数据网络，能够在客户入驻流程中实时洞察并评估潜在风险信号，并将整个身份验证工作流整合至一个统一高效的渠道之中。
据了解，这家身份验证服务商早在2025年初便已开始支持巴西数字驾照的验证。彼时，通过结合QR码扫描和生物识别技术，与政府数据库进行交叉比对，已成功处理了逾百万次验证请求。
Jumio计划将这一成功的验证策略推广至由欧盟eIDAS 2.0框架等新规所催生的各类新兴数字身份格式。Jumio高层指出：“我们的核心职责，是确保客户能够在新出现的政府法规、数字钱包标准或欺诈威胁面前，始终保持领先地位，从容应对。”
同时，此次整合还全面兼容Apple和Google等主流数字钱包提供商，用户可以直接利用其移动设备中已储存的身份凭证，便捷地完成身份核验。一位项目负责人补充道：“通过接入Trinsic的全球验证网络，Jumio的客户现在能够即时完成基于移动驾照、eID以及可复用数字凭证的身份核验。”
这一统一化的平台致力于为国际化运营企业整合身份验证工作流，显著减轻了它们在应对全球碎片化数字凭证标准时的技术挑战与负担。
