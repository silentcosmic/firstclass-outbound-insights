# ️ 阿联酋TikTok直播惹祸：辱骂威胁将面临监禁与罚款

⚠️ 阿联酋TikTok直播惹祸：辱骂威胁将面临监禁与罚款
近日，阿联酋拉斯海马初审法院对两名男子作出判决，因其在TikTok直播中互相辱骂、威胁并滥用信息技术。其中一人被判处三个月监禁，另一人被处以1000迪拉姆罚款，并均需承担诉讼费用。
案件源于一场私人纠纷，随后在TikTok直播中升级为公开争执。庭审中，法院审查了视频证据，确认了双方的言语和行为。尤其值得注意的是，第二被告在直播中做出了“割喉”手势，被法院认定为利用信息技术发出的杀人威胁，行为更为冲动挑衅。
法院强调，利用社交媒体平台进行辱骂和威胁，严重违背了平台的初衷，构成犯罪。同时指出，即便受到挑衅，也无法作为以辱骂或威胁回击的理由，每位被告都需对其个人行为承担相应的刑事责任。
