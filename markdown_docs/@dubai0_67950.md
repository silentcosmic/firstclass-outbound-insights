# 波贝赌场突袭：数千外籍人士聚集，警方却“一无所获”？

🔍 波贝赌场突袭：数千外籍人士聚集，警方却“一无所获”？
柬埔寨媒体报道，6月9日，柬埔寨国家警察总署联合卜迭棉芷省多个执法部门，对位于波贝的一家名为“Holiday”的赌场进行了突袭搜查。该赌场据称聚集居住了数千名外籍人员。
然而，此次针对网络诈骗的清查行动，官方却声称并未发现任何违法犯罪行为。这一结果引发了当地媒体的普遍质疑，但截至目前，官方尚未对此作出正面回应。
