# 洪玛奈：柬埔寨打诈获国际认可

📣 洪玛奈：柬埔寨打诈获国际认可
柬埔寨总理洪玛奈近日公开表示，在过去五个月中，柬埔寨在打击网络诈骗犯罪方面展现出坚定决心，并已获得国际社会的普遍认可。他指出，国际媒体对柬埔寨采取的雷霆行动给予了正面报道和肯定。
洪玛奈强调，此前国际社会对柬埔寨的安全性存有疑虑，甚至将柬埔寨视为诈骗集团的温床。然而，随着当局持续高压打击，国际舆论已发生显著转变，多数媒体开始报道柬埔寨已成功捣毁诈骗窝点，迫使犯罪分子流窜至其他国家。
