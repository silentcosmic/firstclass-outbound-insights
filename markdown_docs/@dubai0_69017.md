# 泰国重拳出击：SHIELD平台剑指跨国犯罪

🚨 泰国重拳出击：SHIELD平台剑指跨国犯罪
在曼谷闭幕的打击网络诈骗与强迫犯罪国际行动对话会上，泰国宣布推出一项名为“SHIELD”的开创性信息共享平台，旨在全面瓦解跨国电信诈骗及人口贩运网络。
SHIELD，全称为“诈骗和人口贩运信息交换及关联数据库”，其核心目标是协助参与国安全高效地共享有关诈骗团伙、受害者、金融交易和跨境流动等关键情报。
泰国国家警察总署副总署长塔猜指出，以往跨国信息共享的滞后严重阻碍了调查与救援效率。为此，泰国政府开发了这一中央网络应用程序，通过整合跨境通信、银行交易及流动人口数据，可直接追踪并瓦解复杂的国际诈骗集团。
