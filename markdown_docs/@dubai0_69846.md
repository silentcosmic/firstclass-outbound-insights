# AI情人应用全球吸金逾4亿，近七成用户选择隐瞒伴侣

🔍 AI情人应用全球吸金逾4亿，近七成用户选择隐瞒伴侣
自2022年底ChatGPT问世以来，全球AI情人应用市场呈现爆发式增长，累计创收已达4.273亿美元，总下载量突破1.653亿次。
据Appfigures数据，仅2026年上半年，214款此类应用便吸金1.628亿美元。其中，Zeta以3300万美元的收入拔得头筹，Emochi则以790万次下载量领跑。同期观察，普通AI陪伴应用的下载量虽是AI情人应用的两倍，但收入水平却几乎持平，这凸显了AI情人应用更强的用户付费意愿和商业变现能力。
深入研究揭示了其潜在的社会影响：在美国18至30岁有伴侣的年轻人中，有15%定期使用AI情人伴侣，而其中高达69%的用户选择向其人类伴侣隐瞒真实使用情况。
