# 马来西亚PJ移民局突查：娱乐业多名人员被带走

🚨 马来西亚PJ移民局突查：娱乐业多名人员被带走
马来西亚雪兰莪州八打灵再也（PJ）移民局近期深夜对当地娱乐场所展开突击清查。此次专项执法行动旨在打击非法劳工及潜在的违规经营活动。
据现场消息，大量涉事场所的服务人员及女性从业者被逐一登记管控，并统一押送上车，移送至相关部门进行进一步核查。此次行动可能对当地夜间经济及外籍劳工管理带来影响。
