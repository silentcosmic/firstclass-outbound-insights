# ️ 沙迦大学城路段将改道两月，配合阿提哈德铁路

⚠️ 沙迦大学城路段将改道两月，配合阿提哈德铁路
沙迦道路与交通管理局（SRTA）近日宣布，为配合阿提哈德火车站连接道路项目的施工，沙迦大学城（University City）与Ard Al Maahid之间路段将于2026年7月4日起实施为期近两个月的临时交通改道，预计持续至8月23日。
此举旨在优化施工流程，并最终提升区域交通连接性及通行能力，是沙迦整体基础设施升级计划的重要组成部分。项目全面完工后，该区域的交通效率将显著提升。
当局提醒，鉴于该区域施工期间交通流量可能发生变化，驾驶员应提前规划路线，尤其在高峰时段，预计可能出现一定程度的延误。
