# 阿联酋租户信用审查上线：房东可提前规避风险

🔍 阿联酋租户信用审查上线：房东可提前规避风险
阿联酋征信局（Etihad Credit Bureau，ECB）已正式推出一项名为**租户背景审查（Tenant Screening）**的全新服务。这项服务旨在帮助阿联酋的房东在签署租赁合同前，查询潜在租户的信用评分，从而有效筛选优质租户，并减少支票跳票等财务风险。
该服务最早于今年5月对外公布，目前已可通过ECB手机应用程序使用。房东现在可以利用这一工具，在决定是否出租房屋之前，更全面地评估租户的财务可靠性，进一步保障自身权益。
