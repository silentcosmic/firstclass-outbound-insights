# 越南金融科技巨头MoMo寻求新一轮融资，估值有望突破20亿美元大关

越南移动支付和金融服务平台MoMo正积极寻求新一轮资金注入，其潜在估值预计将超过20亿美元。这一举动已吸引了全球主要投资机构的广泛关注。
据悉，包括全球私募股权巨头黑石（Blackstone）、欧洲私募股权公司CVC Capital Partners以及日本三菱日联金融集团（MUFG）在内的多家知名机构，都在积极争取入股这家越南金融科技企业。
有消息指出，具约束力的要约预计将在9月份提交。目前，最终的股权出售比例尚未确定，但知情人士透露，此次出售的股份可能高达50%。
MoMo与黑石方面均拒绝对此置评。CVC和MUFG也未就相关询问作出回应。
MoMo于2010年创立之初，专注于移动支付业务。经过多年的发展，其服务范围已显著拓宽，如今涵盖了贷款、保险、储蓄、投资及商家服务等多元化金融产品。
此次潜在的交易正值越南数字金融市场蓬勃发展之际。在该国，无现金支付的普及率持续提升，在线金融服务的使用也日益广泛，为数字金融的增长提供了强劲支撑。
据此前了解，MoMo自今年4月以来便一直在评估引入新投资者的可能性。
该公司自2024年起实现盈利，并在吸引了战略及金融投资者兴趣后，开始与专业顾问团队合作，推进融资进程。
MoMo声称其服务用户已超过3000万，并已在全国范围内建立了广泛的数字交易网络。
该公司上一次大规模融资发生在2021年，当时由日本瑞穗银行（Mizuho Bank）领投，成功募集了2亿美元资金。
去年，MoMo还公布了进一步拓展面向消费者和中小企业服务的计划，展现了其持续增长的雄心。
