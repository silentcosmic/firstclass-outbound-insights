# Tabby推出免费消费账户：最高3%返现抢占市场

📣 Tabby推出免费消费账户：最高3%返现抢占市场
近期，中东“先买后付”巨头Tabby在阿联酋市场推出了一款名为“Tabby Cash”的免费消费账户，旨在简化日常支付、提供现金返还并实现便捷转账，标志着其在消费者金融领域迈出了战略性一步。
据介绍，该账户开户完全免费，不收取任何账户或卡片费用，用户可在数分钟内完成申请。在现金返还方面，Tabby Plus用户可在指定类别和所有国际消费中获得3%的返还，普通用户则可享1%返现。值得关注的是，作为首发福利，所有用户在2026年11月1日前均可享受3%的统一返现比例。
此外，Tabby Cash支持本地无限次免费转账，并计划未来开通全球转账服务。目前，该服务已吸引超过15万用户，并即将面向阿联酋所有年满18岁的居民全面开放。此举有望进一步巩固Tabby在中东数字支付市场的地位。
