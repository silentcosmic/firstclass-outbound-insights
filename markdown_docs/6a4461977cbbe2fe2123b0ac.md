# 东南亚商户操作系统Qashier完成612.5万美元A+轮融资，年交易额突破10亿美元并实现盈利

专注服务东南亚市场的商户操作系统提供商Qashier近日宣布，成功完成一笔612.5万美元的A+轮融资，本轮资金由股权与债务两部分构成。此次A+轮融资由Cocoon Capital、IFP Securities和Blacksoil Global联合领投，并获得了多位天使投资人的参与支持。
Qashier表示，新募集的资金将主要用于推动公司在区域市场的进一步扩张，并加速其产品线的深度研发。Qashier联合创始人兼首席执行官Christopher Choo对此阐述道：“商户不应被迫整合来自多家供应商的服务来支撑一项业务的运营。我们致力于通过将支付、软件、金融服务和客户互动整合至一个统一的生态系统，为他们提供清晰的业务洞察、有效降低运营成本，并赋予其在不同市场规模化扩张的信心。本轮融资将助力我们将这一整合优势转化为下一阶段的强劲增长动力。”“Qashier联合创始人Christopher Choo和Franklin Zhao对Qashier的未来发展有着异常清晰且引人入胜的愿景，即成为‘东南亚商业领域的默认运营基础设施’。我们作为投资方，将坚定不移地支持他们实现这一宏大目标。”
根据Qashier的规划，新一轮资金将重点投入到强化全渠道支付能力、拓展更广泛的嵌入式金融服务套件，以及开发由人工智能驱动的业务洞察与工作流程自动化解决方案。
此外，公司还将拓宽其服务范围，旨在更好地满足大型连锁企业和多门店商户的需求，尤其是在蓬勃发展的餐饮业以及美容与健康领域。
目前，Qashier已在新加坡、马来西亚、泰国和菲律宾等市场为超过2万家商户提供服务，年化支付处理量已达到10亿美元。公司透露，自2025年12月起已实现盈利。
Qashier目前也在积极筹备下一轮B轮融资，以进一步支持其未来的增长阶段。此轮融资将围绕循环收入增长、支付牌照获取以及贷款发放等关键业务里程碑展开。
