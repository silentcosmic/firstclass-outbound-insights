# 涉太子集团“陈小二”日本落网：多重身份迷局曝光

🚨 涉太子集团“陈小二”日本落网：多重身份迷局曝光
日本警方近日逮捕了44岁的塞浦路斯籍男子胡石，同时还有两名中国籍人士被捕。胡石涉嫌通过代理人提交虚假住址登记，初步调查显示这可能与申请日本永久居留资格有关。尽管目前公开案由仅涉及虚假登记，日本警方怀疑胡石正是美国制裁名单中的“陈小二”，引发国际社会高度关注。
胡石（又名胡小伟、陈小二、胡晏铭）的背景复杂。他早年曾被誉为“超级黑客”，后被指控参与“骑士攻击小组”等网络灰色产业，涉案金额巨大。近年来，随着陈志及其太子集团案件受到国际关注，胡石被多方指认为陈志早期的“大哥”和重要合作伙伴，参与了帕劳度假村项目以及太子集团旗下的投资和房地产架构。
在“骑士攻击小组”案后，胡石曾一度消失，随后以塞浦路斯、圣基茨和尼维斯、柬埔寨等多重国籍及不同姓名交替出现。2025年10月，美国财政部对与太子集团有关的“陈小二”实施制裁，英国也因其涉嫌与柬埔寨网络诈骗中心的强迫劳动和不人道待遇相关联，将其列入全球人权制裁名单。
此次在日本的逮捕，虽然起因是虚假住址登记，却意外揭开了这名“超级黑客”长达16年的身份迷局，也为国际社会追查其在跨国网络犯罪和制裁规避中的角色提供了重要线索。
