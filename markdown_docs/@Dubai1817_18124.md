# 特朗普再抛重磅言论：中国欲阻其连任

⚡ 特朗普再抛重磅言论：中国欲阻其连任
美国前总统特朗普近日在公开场合重申，他认为中国政府正致力于阻碍其在即将到来大选中获胜，理由是中方深知他已“看穿了他们”。
特朗普在讲话中强调，其执政期间曾对中国商品征收巨额关税，并声称打造了“世界上最强大的军队”。他指出，这些举措体现了他对华政策的强硬立场。
此番言论再次将中美关系、贸易策略及国家安全等核心议题推至美国大选辩论的中心，持续引发国际社会关注。
