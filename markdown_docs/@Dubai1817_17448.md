# ️ 高薪诱惑下，大马青年明知故犯赴海外诈骗

⚠️ 高薪诱惑下，大马青年明知故犯赴海外诈骗
近期，三名前马来西亚诈骗人员揭露了海外诈骗基地的内幕，指出不少马来西亚青年为追求每月高达1万令吉的收入，明知工作违法仍自愿前往柬埔寨、泰国及缅甸，加入犯罪集团。
据披露，许多受雇者通过非正式渠道获得工作，抵达后才发现所谓高薪职位实为有组织的网络诈骗。尽管如此，面对高额报酬的诱惑，他们仍选择继续从事非法活动。
值得警惕的是，受访者坦言，不少人在出境前已清楚工作性质，甚至有部分人是受朋友主动招揽而加入。这种“短期致富”的侥幸心理，不仅导致同胞受害，更可能让参与者付出失去自由乃至生命安全的沉重代价。
