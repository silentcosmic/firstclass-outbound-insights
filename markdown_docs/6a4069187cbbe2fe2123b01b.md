# 新加坡PayNow支付系统迈向“二代”升级：重塑交易格局，扩展多元应用场景

新加坡的PayNow即时支付系统正准备超越其基础转账功能，目前正积极研究一系列新功能，以服务消费者、商家、企业乃至政府支付场景。
这项名为“PayNow二代”（PayNow Gen2）的研究工作，由新加坡金融管理局（MAS）与新加坡银行公会（ABS）联合主导。
该研究将聚焦新加坡即时支付基础设施的四大潜在升级领域，具体涵盖二维码支付互操作性、在线结账流程优化、面向政府机构的大额支付，以及针对企业用户的扩展功能。
这些核心升级方向，是在与37家机构进行深入咨询，并对全球11个司法管辖区的领先实践进行对标分析后确定的。
其中一项重要的拟议变革，是提升PayNow与NETS QR之间的互操作性。
这将使消费者能够无论商家采用何种支付方案，都能通过扫码完成支付。MAS和ABS计划在2026年底前启动这项功能的试点。
另一个研究重点是PayNow二维码内部的深度链接技术，旨在简化在线结账流程。
引入此功能后，完成支付所需的步骤将显著减少，有望帮助在线商家有效降低因支付流程繁琐导致的交易流失。MAS和ABS设定的目标是在一年内实现该功能的上线。
此外，该研究还将通过一项设有交易限额与安全保障措施的“沙盒”试点，探讨PayNow系统处理政府机构大额交易的可行性。
目前，向政府机构支付大额款项主要通过GIRO系统完成，该方式需提前注册，且处理周期长达三个工作日。
针对企业用户，PayNow未来可能扩展的功能包括“请求支付”（request-to-pay）、用于自动化对账的结构化数据字段、小额支付、更广阔的跨境连接能力、离线支付功能，以及支持“代理商业”（agentic commerce）模式的特性。
MAS和ABS计划于今年启动这些企业级功能的基础性开发工作。
本次研究第一阶段的成果，已在一份题为《PayNow第二代：定义新加坡即时支付系统的下一波创新浪潮》（PayNow Generation 2: Defining the Next Wave of Innovation for Singapore’s Instant Payments System）的报告中详细阐述。
随后的阶段，MAS和ABS将与行业参与者紧密协作，共同推进选定增强功能的实施细则与试点项目。
