# 老挝KTV深夜突袭：跨国犯罪团伙涉枪落网

🚨 老挝KTV深夜突袭：跨国犯罪团伙涉枪落网
老挝甘蒙省警方近日深夜对辖区内一家卡拉OK厅展开突击检查，成功捣毁一处隐藏其中的跨境违法窝点，现场共逮捕涉案人员65名。这些人员国籍复杂，包括54名中国籍公民、7名缅甸籍（其中3名女性）、3名埃塞俄比亚籍以及1名乌克兰籍女性。
警方在现场查获大量作案工具和财物，包括65台台式电脑、4台笔记本电脑、420部涉案手机、26张办公桌椅，以及多国货币现金，总额折合约数十万元人民币。此外，现场还扣押了1辆丰田皮卡车、1辆面包车，并发现1支散弹枪及5发子弹。
据了解，该团伙利用娱乐场所作掩护，长期从事跨境网络犯罪活动，并私藏枪支，性质极其恶劣。目前，所有涉案人员及扣押财物均已被扣留，案件正处于深度侦办阶段，外籍涉案人员将根据当地法律进行依法处置。
