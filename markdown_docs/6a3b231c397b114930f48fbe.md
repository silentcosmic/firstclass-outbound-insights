# 国际足联以区块链技术筑牢防线，加密诈骗却趁势侵袭：数字票务攻防战

2026年国际足联世界杯虽未正式开幕，但针对赛事相关的加密货币诈骗网络已然成形，其基础设施早已悄然搭建，伺机盗取球迷钱财。
一家领先的区块链情报公司已识别出与三项活跃诈骗活动相关的四个加密货币地址，这些活动均以足球支持者为目标。其中包含两个虚假票务网站和一个操控比赛结果的赌博平台。相关信息已被报告至行业警示平台。
一位资深行业政策专家指出，犯罪分子惯于利用重大赛事和文化热点，并且从不等待活动正式开启。他强调：“诈骗者往往提前数周构建并部署其作案基础设施，然后在公众关注度达到顶峰时迅速扩大规模。然而，当前的一大优势在于，每一笔此类支付都在链上留下记录。这意味着合规团队和执法机构可以在损失扩大之前，立即对这些地址和交易路径进行筛查。”
历史为我们提供了警示。以往的研究表明，奥运会同样未能幸免于网络攻击的威胁。
2020/2021年的东京夏季奥运会便曾遭遇网络钓鱼和社会工程攻击，运动员和持票人成为主要目标。
而在此两年前，2018年平昌冬季奥运会遭遇了更为严重的打击：一场网络攻击在开幕式期间爆发，导致互联网网络瘫痪，局域网和无线网络通信均告中断。超过300个系统受到入侵，耗时约12小时才完全恢复运作。
迄今为止，票务欺诈活动留下的链上证据最为清晰。一家市场情报机构已将两个虚假票务操作关联到活跃的加密地址。两者均模仿官方或转售平台，宣传热门比赛的票务供应，并引导买家使用加密货币支付。
一个部署在Polygon（同时也在以太坊上运行）的地址，在2026年4月的一天内收到了约1,562美元，几乎是其全部收款。另一项与比特币地址相关的操作，其钓鱼页面仍处于活跃状态，但尚未收到受害者的付款。
这两项操作都显露出活跃诈骗基础设施的迹象：它们构建了具有高度迷惑性的“店面”，旨在将球迷的急迫情绪转化为不可逆转的加密货币转账。
这些诈骗网站利用了票务市场中存在的真实矛盾。就在比赛开始前两天，官方转售平台上仍有超过18万张小组赛门票未售出。然而，通过非官方渠道购票的球迷依然面临被欺诈卖家利用稀缺性和紧迫感进行诈骗的风险。
赛事启动前，美国执法部门已对此表示担忧。
美国某联邦调查机构在5月警示，网络威胁行为者正通过“域名抢注”（typosquatting）——即创建与官方URL仅有微小拼写差异的仿冒域名——来伪造国际足联的官方网站，以此窃取个人身份信息，包括姓名、地址、电话号码和银行详情。
某地方警务部门也单独向居民发出警告，指出网络犯罪分子正在构建虚假的国际足联网站和社交媒体广告，旨在模仿合法的票务页面。
该警报敦促球迷警惕要求通过加密货币和点对点支付应用等方式进行付款的卖家，因为通过这些渠道进行的交易往往难以撤销。
鉴于2026年世界杯的门票、旅行和住宿成本给狂热球迷带来了经济压力，虚假流媒体平台也成为另一种诈骗途径。
某些案例中，诈骗分子利用加密货币支付方式，因为它能提供更快的结算途径，同时削弱受害者撤销交易的能力。
一家领先的网络安全研究机构已识别出2026年国际足联世界杯期间网络犯罪分子可能采取的其他潜在手段。
有趣的是，当诈骗者利用加密技术进行欺诈活动之际，国际足联（FIFA）却正在部署区块链基础设施，旨在整顿体育赛事的票务环节。
据悉，国际足联正采用Avalanche和Modex技术，推出一种全新的票务模式，以有效遏制机器人抢票、票务欺诈和黄牛囤票等挑战。
该系统被命名为“国际足联区块链”，运行在Avalanche的Layer-1区块链上，并引入了“购票权”（Right-to-Buy, RTB）和“持票权”（Right-to-Ticket, RTT）的概念。
“购票权”（RTB）是指国际足联为“FIFA Collect”用户提供的官方配额，赋予用户购买2026年世界杯等国际足联赛事门票的专属权限。
它明确规定了用户可以购买的门票数量，以及适用于哪些比赛或活动。这实质上允许用户在特定窗口期内购买门票。一旦该窗口期关闭，“购票权”便会失效。
而“持票权”（RTT）则赋予“FIFA Collect”用户在特定比赛和类别中申领一张2026年国际足联世界杯官方门票的权利，并以数字藏品的形式记录。
“持票权”可在“FIFA Collect交易市场”上进行交易，但需支付15%的转售手续费。墨西哥居民的交易则受到限制。
国际足联通过这些策略，可能旨在重新掌握球迷数据和二级票务市场的控制权。此举与国际足联更宏大的目标紧密相连，即优化用户与其平台的互动体验，并推出能吸引其庞大球迷基础（据称超过50亿人）的数字产品。
现有报告显示，已发放超过10万个“购票权”（RTB）。二级市场上的“持票权”（RTT）交易量已突破1500万美元，而“购票权”和“持票权”的总交易量则已超过2500万美元。
关键在于，诈骗分子与体育赛事本身正在进行一场对立的博弈，而最终的胜利果实，只能归属于其中一方。
