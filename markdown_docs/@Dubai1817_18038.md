# 柬埔寨上半年驱逐逾4.2万外国人，电诈犯超1.5万

🚨 柬埔寨上半年驱逐逾4.2万外国人，电诈犯超1.5万
根据柬埔寨移民总局最新报告，今年上半年该国共驱逐出境42,102名外国人。其中，有15,439人涉嫌参与电信及网络诈骗活动，这一数字较去年同期显著上升，凸显了柬埔寨在打击跨国犯罪方面的力度。
报告进一步指出，在此期间，移民总局联合国家警察及相关执法部门，共调查处理了3,917起涉及外国人的案件，牵涉来自78个国家和地区的56,048人。这些案件主要围绕非法居留、违反签证规定以及各类刑事犯罪展开。具体而言，违反《移民法》的案件有3,475起，涉及53,089人；刑事案件则有442起，涉及2,959人。
此外，上半年柬埔寨还拒绝了来自76个国家和地区的4,180名外国人入境，原因包括不符合入境条件或被列入黑名单。同期，另有386名外国人因涉嫌犯罪或被列为重点监控对象而被禁止离境。
