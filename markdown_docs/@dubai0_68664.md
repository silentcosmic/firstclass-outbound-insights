# 马来西亚三年逾16万宗网络诈骗案，近4.5万嫌犯落网

🚨 马来西亚三年逾16万宗网络诈骗案，近4.5万嫌犯落网
马来西亚内政部长赛夫丁纳苏迪安近日披露，自2023年以来，警方共受理调查网络诈骗案件达16万5501宗。在此期间，执法部门已逮捕近4万4990名嫌犯，并成功追缴超过870万令吉的涉案资金。这一数据凸显了马来西亚在打击网络犯罪方面所面临的严峻挑战及持续努力。
