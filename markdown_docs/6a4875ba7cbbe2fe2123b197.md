# 欧洲生物医药前沿：巴塞罗那如何成为创新药企出海与巨头重注的核心门户？

2026年6月底，瓦尔德希伯伦大学医院成功完成了欧洲首例单孔机器人儿科肾输尿管切除术，标志着医疗技术应用的一项重大突破。
作为西班牙规模最大的综合医院之一，瓦尔德希伯伦大学医院年接诊量超百万人次，其对前沿医疗技术和产品的探索，在欧洲医疗体系中处于领先地位。该院此前曾完成世界首例供体来自安乐死患者的面部移植手术，彰显其在复杂外科手术领域的卓越实力。
院方近日向本刊透露，此次里程碑式的儿科手术，其核心设备来自中国手术机器人企业——术锐机器人。这标志着瓦尔德希伯伦大学医院首次以直销模式采购中国高端医疗器械。该设备的引入，预计每年将为医院新增200至300台微创机器人手术，从而为中国高端医疗器械进入欧洲临床应用场景开辟了一条重要通道。
近期，在西班牙加泰罗尼亚贸易投资局的邀请下，本刊记者团队深入走访了加泰罗尼亚首府巴塞罗那地区的生物医药产业园区及头部药企研发中心，旨在探究这片地中海沿岸土地上涌动的产业变革。
加泰罗尼亚政府企业与竞争力秘书长兼贸易投资局局长Jaume Baró提供的数据显示，该区域目前汇聚了1523家生命科学相关企业，其吸纳的就业人口占当地总人口的7.3%，并贡献了4.1%的GDP。在对外出口方面，加泰罗尼亚稳居西班牙榜首，全国46%的医药产品出口源自该地区。尤其在2024年，医药品类贡献了近九成的本地出口总额，凸显其经济支柱地位。
从外商投资视角审视，2020年至2024年的五年间，加泰罗尼亚生命健康产业累计吸引了高达22.8本站元的外商直接投资，成功落地46个外资项目，并创造了6050个优质就业岗位。事实上，全西班牙超过半数的生命科学领域外资均流向了加泰罗尼亚，印证了其强大的吸引力。
深入探究，加泰罗尼亚生命科学产业之所以能在南欧地区独树一帜，并持续吸引全球制药巨头重金投入，其核心在于以巴塞罗那生命科学园（Parc Científic de Barcelona，PCB）为核心枢纽，构建起了一套产、学、研、医、资一体化的闭环生态系统。这套成熟的资源整合机制，不仅源源不断地孵化着本土创新企业，更以外来投资者提供了难以快速复制的独特发展土壤。
巴塞罗那生命科学园：产学研医资一体化的创新引擎
成立于1997年的PCB，是西班牙首个专业的生命科学科技园。目前，园区已成功打通医疗实验室与临床终端的资源通道，实现了基础科研、临床验证与产业转化的无缝衔接，从而形成了一套可持续的创新循环：科研人员依托高校实验室产出前沿技术，借助园区共享平台完成早期验证，再利用本地丰富的多中心临床资源进行人体试验，最终将科研成果转化为成功的商业化企业。
一组数据为这一模式提供了有力佐证：由加泰罗尼亚政府与巴塞罗那市政府联合创建的公私合营基金会Biocat统计表明，在2015年至2025年的十年间，加泰罗尼亚的生命健康领域已孕育出超过400家初创公司。此外，据业内人士透露，全西班牙九成以上的临床试验均在此地完成，再次凸显其在生物医药研发领域的关键地位。
PCB市场营销与传播总监Grisha Domakowski介绍道，这套生态系统已培育出多个成熟典范。例如，专注于基因疗法开发的SpliceBio曾入驻PCB园区，其核心研发管线SB-007是一种针对遗传性视网膜退行性疾病的基因疗法，目前已进入临床研究阶段。2025年6月，SpliceBio成功完成1.35亿美元的B轮融资，刷新了加泰罗尼亚生命科学领域单轮融资的纪录。
与此同时，PCB卓越的资源整合能力，同样是海外制药企业选择在此进行战略布局的关键考量。Jaume Baró指出，海外生命科学企业选择落户加泰罗尼亚的核心动因中，产业集群和科创资源各占据40%的权重。PCB作为这些集群和科创资源的核心载体，使得跨国企业无需从零开始构建完整的研发配套。它们入驻后即可直接对接高校博士人才库、第三方合同研究组织（CRO），甚至产业资本和知识产权服务机构等，从而显著缩短研发项目的落地周期。
以英国制药巨头阿斯利康在当地的战略部署为例。2020年，阿斯利康选择巴塞罗那作为其全球生物医药创新中心的所在地，并计划到2027年追加投资总计13本站元。自2023年投入运营以来，该创新中心已启动近400个研发项目和190项临床试验，涵盖肿瘤、心血管疾病、罕见病等多个关键治疗领域。
此外，日本AGC Pharma Cheminaik、东和制药以及美国远程医疗企业Teladoc Health等国际知名公司，也已相继在加泰罗尼亚落地，累计投资数本站元，创造了数千个高端产业就业岗位。
Jaume Baró强调，这种“临床、科研、产业、资本”深度融合的综合竞争力，促使顶尖医疗机构、科研中心、活跃的初创企业生态与成熟制药企业之间形成强大的协同效应，使其天然成为了全球创新企业进军欧洲市场的首选门户。
中欧合作新范式：从优势互补到深度协同
当一个区域的产业生态步入成熟阶段，其视野必然会向外拓展，寻求新的技术源泉并开发更广阔的市场腹地。加泰罗尼亚生命科学产业的演进轨迹亦印证了这一规律。在过去十年间，这个地中海沿岸的产业集群成功实现了从内部资源积累到外部深度联结的战略跃迁。
这种互补性根植于中欧两个医疗体系各自禀赋的深层差异。瓦尔德希伯伦大学医院院长Albert Salazar指出，尽管中欧医疗体系架构存在不同，但“双方都拥有值得互相借鉴的成熟经验”。中国在人工智能技术应用、数字健康平台、远程医疗以及超大规模人口医疗服务管理等领域展现出强大的实践能力，欧洲医疗机构可以从中学习数字化转型推进速度和医疗技术规模化普及的经验。而欧洲医疗体系的优势则体现在一体化公立医疗体系、循证医学体系和临床治理规范等方面。因此，双方合作的最大机遇在于优势互补：欧洲可以输出临床验证和医疗质控的相关经验，而中国则凭借其技术规模化落地、创新产业生态和工业化发展等突出能力。
随着中国本土创新药产业逐步实现从“跟随式仿制”到“源头式创新”的战略跨越，为自主研发的新药寻找具备全球市场开发能力的欧美制药伙伴，已成为中国创新药企近两年来的核心任务。预计到2025年，中国创新药的海外业务拓展（BD）总金额将达到1300亿美元的基数。在某种程度上，这一角色转换正从根本上重塑着中国与欧美企业间的合作格局。
一位长期关注中国创新药企出海的财务顾问（FA）人士观察到，此前，欧洲诸多药企虽早已布局中国市场，但其策略较为保守，与中美药企的合作频率相对较低。然而，自2025年起，部分欧洲制药公司的高层（C-level）对“了解中国市场的需求变得异常强烈，即便仅从个人职业发展角度考量亦是如此”。
在加泰罗尼亚，本土企业也在积极调整其全球策略。今年3月，老牌皮肤病领域领军制药公司阿尔米拉（Almirall,ALM）正式在上海设立办事处，专门负责在中国发掘合适的创新药项目并构建合作关系。阿尔米拉首席科学官Karl Ziegelbauer向本刊表示，中国市场是公司一项重要的“补充性布局”，旨在通过多元化的合作模式拓宽其创新研发的边界。
早在2022年，阿尔米拉便与中国一家知名药企达成合作，共同开发一款治疗斑秃和特应性皮炎的新药，其里程碑付款最高达4.92亿美元。今年3月，该公司再次与另一家中国药企的子公司签署了全球研究合作及许可协议，共同开发一款早期单抗候选药物，累计付款最高可达3.4亿美元。
Ziegelbauer指出，中国创新企业普遍展现出“执行效率高、运营灵活敏捷、创新产业生态高速迭代”的显著优势。他强调：在上海设立办公室后，我们不会设定年度交易规模指标。我们的优先考核标准是科研质量与战略匹配度，以确保所筛选的研发资产能为皮肤病患者带来真正且具有颠覆性的治疗价值。
Jaume Baró对当前的合作态势评价道：双方的合作已远不止于商品贸易。联合创新项目、产业合作、技术转让、研发协同以及双向投资的案例正在持续增多。
Jaume Baró坦言：当前，双方在生命科学领域的合作仍面临诸多挑战，例如各国监管规则的复杂性、地缘政治的不确定性以及文化差异。这要求所有合作方必须建立深厚的互信基础，保持合作的稳定性，并树立长远的战略发展视野。
造成这些挑战的原因在于多方面：首先，欧洲市场与中国的药品器械监管体系存在显著差异，跨国临床试验和产品注册流程复杂且耗时，不仅延长了项目落地周期，也增加了合作成本。其次，生命科学行业的准入门槛远高于传统制造业，难以简单复制汽车或化工行业独立建厂的模式。因此，企业合资或开展科研合作成为主流，但也容易在合作权责和知识产权分配上引发分歧。
PCB市场、传播与竞争情报部门主管Silvia Labé也指出，园区曾接待过众多来自中国的医药代表团，涵盖政府机构、企业、医疗机构及科研单位。他们来访的核心诉求多为学习当地生物医药产业模式并探索合作机会。然而，她也坦承：有些时候，代表团参访结束后，难以进行持续的对接和跟进。由于资源有限，我们在选择合作对象时会更加谨慎。对我们而言，身处欧洲，要全面了解中国当前生物医药产业的快速变化并非易事。
针对如何进一步深化合作，加泰罗尼亚政府已制定了清晰的战略。Jaume Baró介绍说，加泰罗尼亚贸易投资局在中国上海、北京、香港均设有办事处，旨在为有意赴加投资的中国企业提供全流程服务，范围涵盖协助准备投资申报材料、选址、签证办理、人才招聘，直至投资落地后的持续跟踪服务。此外，赴加投资的中国企业还有资格申请专项基金的资金扶持。
Jaume Baró表示：我们的最终目标，是让加泰罗尼亚成为中国创新技术进入欧洲的核心门户之一，并成为双方企业在技术和产业协同上的可靠合作方。
