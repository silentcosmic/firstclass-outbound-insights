# 8个月蒸发2.3万亿！加密市场创史上最大回撤

🚨 8个月蒸发2.3万亿！加密市场创史上最大回撤
全球加密货币市场近期遭遇剧烈动荡，数据显示，自2025年10月创下约4.3万亿至4.4万亿美元的历史高位后，总市值一度跌至约2万亿美元，短短8个月内蒸发约2.3万亿美元，规模超过市场最高市值的一半。此轮累计缩水幅度高达约53%，被市场研究机构形容为加密市场有记录以来规模最大的同期回撤之一。
市场自2025年10月见顶后便持续下行。2025年第四季度，加密市场总市值下降约9460亿美元，以约3万亿美元收官，全年总市值下跌10.4%。进入2026年，跌势未止，第一季度市值再减6220亿美元，季度跌幅20.4%，至3月底降至约2.4万亿美元，较2025年10月高点已低约45%。此后，随着比特币及多数主流代币持续下挫，市场总市值曾进一步逼近2万亿美元。
作为加密市场市值最大的资产，比特币在本轮下跌中首当其冲，价格一度跌破6万美元，降至约5.9万美元附近，创下约20个月新低，较2025年10月历史高位下跌超过一半。以太坊、Solana等主流加密货币同样出现明显回撤，进一步拖累整体市场表现。
市场分析指出，此轮下跌并非单一事件引发，而是多重因素共同作用的结果。宏观层面，美国利率政策预期的变化，投资者对降息期待减弱，美元及融资成本走强，导致资金减少对加密货币等高风险资产的配置。同时，部分投机资金正从加密市场转向人工智能、半导体及即将上市的科技公司，加之现货比特币ETF资金流出、全球地缘政治风险以及市场缺乏新的上涨催化因素，均进一步打击了投资者信心。
尽管最新数据显示全球加密货币总市值已回升至约2.3万亿美元，但与2025年10月历史高点相比仍缩水近半，市场整体仍处于大幅回撤区间。在价格波动加剧的背景下，投资者仍需警惕高杠杆合约、强制平仓以及流动性突然下降带来的潜在风险。
