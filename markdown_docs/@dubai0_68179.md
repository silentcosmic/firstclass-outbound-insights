# 越南胡志明市重拳出击：捣毁跨国诈骗窝点，多名中国公民涉案

🚨 越南胡志明市重拳出击：捣毁跨国诈骗窝点，多名中国公民涉案
越南胡志明市公安局近期持续加大对跨国犯罪的打击力度，连续捣毁多个外国犯罪团伙藏匿窝点，查获大批涉嫌从事网络诈骗活动的外国公民，其中包括多名国际通缉人员。此轮行动揭示了当地酒店、公寓及民宅被犯罪团伙利用的严峻态势。
据警方6月24日通报，胡志明市公安局已对阮红草妮（Nguyễn Hồng Thảo Nhi）、范青山（Phạm Thanh Sơn）、张氏碧簪（Trương Thị Bích Trâm）等20余名涉案人员提起诉讼并依法拘留，他们涉嫌“组织他人在越南非法居留”。调查显示，这些团伙多通过非法入境或非法居留方式，在当地建立窝点，准备或实施跨国网络诈骗。
值得注意的是，部分越南本地人员受利益驱使，与境外犯罪组织相互勾结，负责租赁整栋住宅、公寓及酒店，为外国犯罪团伙提供住宿、网络及生活保障等后勤支持，并故意规避住宿登记以逃避监管。例如，阮红草妮以每月4500万越南盾的租金为42名中国公民（其中4人为非法入境）提供住所，并采购大量诈骗设备；范青山则租赁并改造酒店，为82名中国公民提供住宿，即便明知对方居留手续存在问题仍持续协助，每月非法获利数千万越南盾。张氏碧簪也通过类似手段，为大批外国人（主要是中国公民，包括非法入境者）租赁公寓。
警方表示，自今年5月启动为期45天的专项打击行动以来，已立案侦办8起案件，起诉26名涉嫌“组织、协助他人在越南非法居留”的犯罪嫌疑人，其中不乏酒店经营者、住宿场所管理人员及房屋租赁中介。越南警方提醒，任何代他人租赁房屋、隐瞒入住人员身份信息，或明知对方非法入境、非法居留仍提供协助的行为，均将面临刑事责任。
