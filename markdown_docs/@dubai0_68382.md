# 迪拜外籍人士留任动机巨变：免税收入不再是首因

⚡ 迪拜外籍人士留任动机巨变：免税收入不再是首因
一份最新报告揭示，高达94%的阿联酋外籍人士计划长期留居当地，这一数据有力回应了近期市场对外籍劳工吸引力的猜测。该报告由ARCET Global与QuestionPro合作发布，名为《阿联酋外籍人士监测报告》。
报告中最引人瞩目之处在于，免税收入已不再是外籍人士首要的留任考量。受访者普遍将生活质量和工作机会（均为88%）列为选择留在阿联酋的首要因素，紧随其后的是安全与低犯罪率（87%）。免税收入则降至第五位，但仍有81%的受访者认为其是重要原因之一。此外，有87%的受访者表示，阿联酋提供的机会多于他们的祖国。
