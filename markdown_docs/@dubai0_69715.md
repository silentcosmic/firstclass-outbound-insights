# 迪拜女乘客遭出租车骚扰：RTA快速响应获赞

📣 迪拜女乘客遭出租车骚扰：RTA快速响应获赞
近日，一位女性乘客在迪拜乘坐出租车时遭遇不当搭讪及骚扰，引发关注。据当事人描述，司机在途中索要联系方式并发表对乘客国籍的不当评论。
尽管此次事件令该乘客感到不适，但她选择直接向迪拜道路与运输管理局（RTA）投诉。她表示，过去十年在迪拜的出行体验几乎无可挑剔，此次事件属个例，但为避免类似情况再次发生，她认为有必要将其记录。
RTA在接到投诉后迅速响应，展开调查并证实了事件属实。官方不仅就乘客的遭遇表达歉意，更承诺将采取必要措施。RTA的快速且认真处理，让该乘客对迪拜的城市安全和治理机制重拾信心。
