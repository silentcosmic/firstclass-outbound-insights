# 东盟与伙伴国强化合作：剑指跨国网络诈骗

⚡ 东盟与伙伴国强化合作：剑指跨国网络诈骗
2026年6月25日，第29届东盟移民及领事高级别会谈在柬埔寨暹粒圆满结束。会议期间，东盟成员国与日本、中国、韩国、澳大利亚等对话伙伴达成广泛共识，将进一步深化联合行动，共同打击日益猖獗的跨国犯罪，其中网络诈骗团伙被列为此次合作的重点打击目标。
各方一致同意，将通过加强执法人员培训、扩大成员国间的情报与信息共享等措施，提升区域内执法协同能力。同时，将在打击人口贩运、毒品走私以及网络诈骗等领域展开更紧密的相互配合。
值得一提的是，会议还批准了柬埔寨关于设立东盟移民培训中心的提议。该中心旨在全面提升各国移民和边境执法人员的专业能力，从而有效瓦解东南亚地区的跨国网络诈骗犯罪网络，维护区域安全稳定。
