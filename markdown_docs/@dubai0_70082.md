# 5亿跨境传销大案主犯，潜逃泰国终落网

🚨 5亿跨境传销大案主犯，潜逃泰国终落网
一起涉案金额高达5亿元人民币、受害者逾4000人的特大跨境传销案，其幕后主脑在潜逃泰国期间终被捕获。
泰国移民局日前宣布，成功抓获63岁的中国籍男子李某。他不仅是中国法院签发逮捕令的在逃嫌疑人，更是国际刑警组织红色通缉令的重点追捕对象。调查显示，李某涉嫌组织运营“KP集团”投资平台，以“高收益、稳赚不赔”为噱头，诱骗大量民众投资。
案发后，李某潜逃至泰国。中国驻泰国大使馆随即向泰方通报情报，请求协助追捕。泰国移民局迅速响应，撤销了李某在泰国的居留资格，并经过周密布控，最终在春武里府邦拉蒙县一处住所将其成功控制。
目前，泰方已依法完成相关法律程序，并将在近期将嫌犯遣返回中国，接受进一步司法审理。
