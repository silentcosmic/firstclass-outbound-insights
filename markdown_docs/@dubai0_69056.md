# 亚太电诈集团换帅续命：全球围剿升级

🚨 亚太电诈集团换帅续命：全球围剿升级
备受关注的跨境赌诈“十大逃犯”之一、缅甸妙瓦底“亚太新城”主犯佘智江，已于2025年11月12日被泰国正式引渡回中国，目前正在南京接受法律审判。然而，令人震惊的是，尽管佘智江已落网，其在缅甸克伦邦水沟谷的亚太电诈集团（亚太新城）却并未因此停摆，反而由其泰国狱友王富贵及其亲信徒弟何英雄接手，继续疯狂从事人口贩卖、电信诈骗及网络赌博等非法活动，无数中国家庭仍在遭受侵害。
据市场消息披露，王富贵自称曾是中国广西前警察，后因涉嫌诈骗在泰国入狱。正是在曼谷中央监狱服刑期间，他与同样因跨境赌诈被通缉的佘智江同监结识，并成为后者在狱中与外界沟通的关键“中间人”及“最信任的干部之一”。佘智江被捕并引渡后，王富贵顺利出狱，随即与31岁的何英雄共同接管亚太集团的核心运营，成为该集团实际掌权的主要领导人物。
在王富贵、何英雄二人掌控下，亚太国际控股集团以更为隐匿的方式持续运作，犯罪活动非但没有停止，反而呈现出更强的隐蔽性、高强度和规模化趋势。最新情报显示，两人正秘密将部分核心产业链转移至缅甸东部三佛塔地区、苏奇督将军家乡等偏远地带，建立地下指挥中心，并利用Starlink卫星通讯、加密工具及深假AI等高新技术，试图彻底规避公众和执法部门的视线，继续大规模进行电信诈骗、网络赌博和人口贩卖。大量诈骗人员回流，每天仍有数以千计的中国受害者被诱骗至园区。
面对日益猖獗的跨境电诈犯罪，全球执法机构，特别是以美国FBI为首的国际力量已高度重视。据悉，美方已成立专门的“诈骗中心打击特遣队”（Scam Center Strike Force），并通过美国财政部对亚太集团（Yatai）及Shwe Kokko相关目标实施严厉制裁。同时，多国联合开展跨国情报共享、域名查封、资产冻结和抓捕行动，并悬赏高达1000万美元征集情报，明确将东南亚诈骗园区视为美国“新战场”，表明跨境犯罪的生存空间正被急速压缩。
尽管天网恢恢，王富贵、何英雄二人至今仍逍遥法外，牢牢掌控集团核心权力。然而，随着国际合作的深入和执法打击力度的加大，任何企图通过转移阵地、技术升级来逃避法律制裁的行为都将是徒劳。全球执法部门正编织一张密不透风的法网，所有涉案人员终将难逃法律的严厉审判和全球执法铁拳的彻底清算。
