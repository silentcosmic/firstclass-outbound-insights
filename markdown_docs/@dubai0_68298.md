# 阿联酋伊朗外长通话，促全面履行美伊备忘录

📣 阿联酋伊朗外长通话，促全面履行美伊备忘录
阿联酋副总理兼外交部长谢赫·阿卜杜拉·本·扎耶德·阿勒纳哈扬于6月26日与伊朗外交部长阿拉格奇通电话，双方就地区局势及美国与伊朗谅解备忘录签署后的最新进展进行了深入交流。
通话中，谢赫·阿卜杜拉强调全面履行该协议条款至关重要。此举旨在确保地区敌对行动立即停止，尊重各国主权及睦邻友好原则，并严格遵守国际法。他特别指出，这包括保障海上通道及霍尔木兹海峡的国际航行自由。
谢赫·阿卜杜拉表达了对当前谈判能取得积极成果的期望，以实现地区持久安全与稳定。他重申，务实外交与负责任对话是应对地区及国际危机的最佳途径，有助于满足各国人民对繁荣发展的共同愿望。
