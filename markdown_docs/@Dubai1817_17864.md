# 阿根廷足协遭FBI调查：数亿美元资金流向引国际关注

🚨 阿根廷足协遭FBI调查：数亿美元资金流向引国际关注
就在阿根廷队在世界杯淘汰赛中逆转晋级八强之际，国际足坛突传重磅消息：美国联邦调查局（FBI）已对阿根廷足球协会（AFA）在美国的商业及财务活动启动调查。此次调查聚焦数亿美元资金流向及多笔用途不明的转账。
据阿根廷媒体《La Nación》披露，调查核心是AFA在佛罗里达州设立的商业实体TourProdEnter LLC。执法部门正核查该公司近年来管理的商业收入、赞助费及赛事版权收入等资金往来，以确保其符合美国法律规定。
据悉，此案涉及约2.6亿美元商业收入，其中约5700万美元资金因缺乏明确用途说明，已成为FBI的重点核查对象。美国执法部门正调取相关银行交易记录，深入追溯资金流向及最终受益人。
AFA主席克劳迪奥·塔皮亚因此再次成为舆论焦点。近年来，塔皮亚曾多次因足协管理、商业合同及税务问题受到质疑。此次美国方面的介入，无疑使AFA的财务透明度再次面临国际社会的审视。
尽管调查曝光时机敏感，恰逢阿根廷队在世界杯赛场上备受瞩目，但美国司法部门目前尚未公布正式起诉文件，也无证据表明此次财务调查与世界杯比赛结果存在直接关联。FBI当前的调查重心仍集中于AFA在美国境内的商业资金运作，而非赛事本身。后续进展有待美国司法部门进一步发布。
