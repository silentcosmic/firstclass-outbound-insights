# 拼多多打破轻资产策略：重金购入雄安办公楼，深拓数字经济新版图

自2015年9月正式上线以来，拼多多集团已发展成为拥有约2.55万员工、年度营收逾4300亿元的行业巨头。然而，在办公场所策略上，该公司长期坚持轻资产模式，奉行不购置土地、不自建楼宇的原则。目前，拼多多在上海的办公地点，包括金虹桥国际中心、艺丰中心、晶耀虹桥、星展银行大厦等，其楼宇均为租赁。
不建设总部、分散租赁办公楼宇，乃至服务器、仓库等核心运营资产也普遍采用租赁模式，这已成为拼多多一项行之有效的经营准则。此策略旨在最大程度减少资金沉淀，将资金集中投入到供应链优化、用户补贴及其他核心运营环节，以维持其高效的运营效率。
近期，拼多多在雄安新区设立的新公司引发业界关注。据公开信息，该公司的法人代表为拼多多集团Temu高级副总裁杨清勇，监事则由拼多多集团执行总裁兼新拼姆总裁朱政担任，而实际控制人为拼多多集团联合创始人、联席董事长兼联席CEO陈磊。
杨清勇作为拼多多集团内部的中高层核心管理人员，负责统筹国内跨境招商与政企合作业务。此次由其主导成立的雄安新公司，其战略定位明确：不涉足传统电商交易业务，而是专注于大数据处理、云平台建设、数字化运维、平台合规审核以及技术外包服务。这表明该实体是拼多多集团布局国家级数字新区的关键战略主体。
拼多多集团执行总裁朱政表示：“拼多多集团将扎根雄安、服务雄安，持续加大资源投入与产业布局力度，积极探索更多合作潜力，落地更多业务场景。我们希望以数字技术赋能产业链上下游，搭建现代化、一体化的供应链体系，以实际行动为深入推进雄安新区高质量建设和发展贡献力量。”
根据掌握的信息，拼多多在雄安购置的新办公楼总高12层，是雄安·电建智汇城重点开发的主题楼宇之一。大楼内部配备了大型会议中心和培训中心，并为员工提供了母婴室、休闲区等完善的福利设施，预计将在7月底正式投入使用。目前，园区食堂亦在积极筹备中。首批雄安公司员工目前在临时办公场所过渡，届时将全部迁入新购置的办公楼。
拼多多方面透露，未来雄安公司预计将为当地创造超过5000个工作岗位。首批入职的150名新员工中，超过八成来自京津冀地区，其中不乏河北高校的应届毕业生。新办公楼周边便捷的立体化交通网络，连接京津冀各主要城市，无疑为新公司吸引各类优质人才提供了有利条件。
中国电建雄安·电建智汇城项目总建筑面积约为166万平方米，规模宏大。拼多多此次购入的是铭智港西区1-4号楼中的一栋，市场单价估测在每平方米1.3万元至2.1万元之间。该园区本身的战略定位清晰，旨在“打造河北省电商产业示范基地、京津冀数字电商协同创新总部，并成为国家级数字经济发展标杆园区”。
战略考量与未解之谜
此举也引发了业界一系列深层疑问：拼多多为何选择在此时改变其长期坚持的轻资产策略，斥资购置办公楼？为何又将首个重资产布局选择在了雄安新区？在此之前，拼多多并未在北方设立大型子公司或分支机构，此次行动在公司整体战略层面有何深远考量？雄安新区又提供了哪些具有吸引力的税收、补贴或其他优惠政策？本刊就相关疑问向拼多多方面进行问询，对方表示，除已对外公布的信息外，暂无更多官方回应。
此外，市场有观点猜测，拼多多此次策略转变、斥资购楼并设立新公司，可能与今年4月其因“幽灵外卖”乱象被处以15.22亿元巨额罚款有关。当时，多家大型平台均因此类问题受罚，拼多多是涉事七家平台中罚款金额最高的。对于此类市场推测，拼多多方面并未予以置评。
