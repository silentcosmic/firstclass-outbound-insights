# 柬泰边境电诈园区曝光：伪造多国警局专骗海外公民

🚨 柬泰边境电诈园区曝光：伪造多国警局专骗海外公民
澳大利亚广播公司（ABC）近期获准进入柬埔寨奥多棉芷省奥斯玛（O'Smach）地区一处大型电信诈骗园区，首次揭露了其针对海外受害者的犯罪模式。该园区位于柬泰边境，内部设施包括伪造的“澳大利亚联邦警察办公室”、标准化诈骗话术及隔音办公区，显示出高度组织化的诈骗体系。
更令人担忧的是，园区内不仅搭建了仿冒澳大利亚联邦警察的场景，还设有疑似伪装成中国、新加坡、巴西、印度、印度尼西亚及越南等多个国家执法机构的布景。这些场景专用于视频通话中冒充官方人员，以骗取不同国籍受害者的信任，暴露出其全球化的诈骗网络布局。
针对澳大利亚受害者，诈骗人员通常冒充联邦警察，谎称受害者身份信息被用于洗钱，并要求配合调查。为增强可信度，他们会通过加密视频通话“录取口供”，展示虚假涉案照片，并要求签署“保密协议”。随后，诈骗团伙利用受害者对执法机关的恐惧，以账户冻结为由，持续施压索要银行资料、账户截图，甚至要求汇报行踪，最终诱导资金转移。
调查还发现，澳大利亚国家反诈骗中心此前警告的约300名公民的个人资料（姓名、电话、住址）已在该园区文件中泄露，表明诈骗集团会从黑客或其他犯罪组织批量购买个人信息，并根据受害者背景量身定制剧本，大幅提升诈骗成功率。这种诈骗不依赖复杂技术，而是通过统一剧本、角色分工及专业培训，进行持续的心理操控。
澳大利亚民众2025年因各类诈骗预计损失约21.8亿澳元。尽管奥斯玛园区被证实长期针对澳大利亚受害者，但官方强调并非所有损失均源于此。澳大利亚联邦警察再次提醒公众，真正的警方绝不会通过电话或视频要求核实银行账户信息或要求转账。
