# 金边警方重拳出击：打击网诈与人口贩卖

🚨 金边警方重拳出击：打击网诈与人口贩卖
柬埔寨国家警察总署副总监兼金边市警察局长尊纳林将军近日主持会议，部署深化警务与专业执法效能。会议强调，将持续坚决打击网络科技诈骗犯罪，并强力推进人口贩卖犯罪的预防与打击工作，同时确保各类刑事犯罪的高效打击。
尊纳林将军指示全体执法人员务必严格依法、依专业行事。此次行动旨在全面提升金边市的社会治安水平。市警察局各级官员及14个区警察局长均出席了此次重要会议。
