# 涉42亿航空级庞氏骗局主犯在泰落网

🚨 涉42亿航空级庞氏骗局主犯在泰落网
一名涉及金额高达42.43亿元人民币的特大非法集资案主犯，中国籍女子郑某，已于6月28日在泰国清迈被捕。此案此前已引起中国驻泰大使馆的关注，并发出通报。
据查，郑某在担任深圳某科技公司总经理期间，伙同法人白某某，将云存储服务器采购项目包装成高回报投资计划。她以每月4.75%至5%的诱人回报为饵，成功诱骗逾4500名中国投资者，非法集资总额超过42.43亿元人民币，被业界形容为“航空级庞氏骗局”。
目前，泰方已依法撤销郑某在泰的停留许可，并将其移送至移民局相关部门，等待后续的驱逐出境程序。
