# 泰国签证新政：93国60天免签取消，重点平衡安全

🚨 泰国签证新政：93国60天免签取消，重点平衡安全
近日，泰国政府内阁会议审议并批准了对现有免签政策及其他签证便利措施的重大调整方案。据总理府副发言人普洛塔莱透露，此举旨在确保政策与时俱进，有效堵塞安全漏洞，防范签证权益被滥用，并减少政策重叠，坚持“一国一权益”原则。
此次调整的核心内容是，此前93个国家和地区享有的旅游、短期工作或短期商务免签入境待遇（可停留不超过60天）将被取消。泰方解释称，较长的60天停留期限可能被部分人员利用，从事非法活动或不符合旅游目的的行为。未来，政府将综合考量各国经济因素、国家安全、国际关系及互惠原则，重新授予相应的签证权益。
根据新措施，共有59个国家和地区将获得30天免签待遇，其中包括新增的印度、克罗地亚、保加利亚、塞浦路斯、马耳他和马尔代夫。调整后，欧盟27个成员国将全部统一享有30天免签待遇，此举亦被视为推动泰国与欧盟在申根免签及经济合作协议谈判方面的重要一步。
此外，毛里求斯和塞舌尔两国将获得不超过15天的旅游免签待遇，未来会根据游客数量和消费金额等因素进行评估。在落地签政策方面，阿塞拜疆、白俄罗斯和塞尔维亚三国将继续享有此待遇。值得注意的是，印度游客的落地签资格已被取消，以避免签证权益重复。
普洛塔莱强调，此次调整完成后，享有各类签证便利措施的国家和地区总数将为65个。相关内政部公告草案将在《政宪公报网站》公布满15天后正式生效。同时，泰国安全部门将加强“泰国数字入境卡”系统的效率，强化背景资料筛查及出入境记录追踪，并推动各部门间数据共享，以便在游客入境前进行风险评估，并监督外国人是否按照申请目的使用签证权益。
泰方重申，重新审查签证措施并非为了降低泰国旅游吸引力，而是为了构建一个更符合当前情况、更清晰、更易于监管的制度，以在促进经济发展、便利外国游客、维护国际关系及保障国家安全之间取得平衡，并防止签证政策被不法分子利用。
