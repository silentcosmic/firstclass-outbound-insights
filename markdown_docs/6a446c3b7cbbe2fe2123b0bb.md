# Tata Communications斥资1.52亿美元拓展印新数字走廊，赋能AI与数据互联

印度Tata Communications宣布进行一项1.52亿美元的战略性投资，旨在扩建印度与新加坡之间的数字通信走廊，预计将新增98太比特每秒（Tbps）的AI就绪容量，以大幅提升该区域的数字基础设施承载力。
业界观察认为，此项投资举措旨在积极响应亚洲乃至全球范围内，对带宽和由人工智能驱动的数据传输日益增长的强劲需求。
印度-新加坡海底光缆线路正日益成为全球数字基础设施中的关键枢纽。目前，全球超过35%的互联网流量通过Tata Communications的全球网络进行路由。此次投资将显著提升网络的多元性、韧性及低延迟连接能力，从而有力支撑企业级、云计算和超大规模数据中心流量，全面推动亚洲数字经济、人工智能、云计算及各类数据密集型应用的发展。
具体而言，这笔投资将用于整合孟买与新加坡之间的新建海底光缆系统，并作为联合体成员参与建设连接金奈与新加坡的新海底光缆系统。预计这些先进的光缆系统将于2029年第四季度投入运营。
这些新建的光缆系统将与Tata Communications在印度的地面光纤网络实现无缝对接，进一步延伸至全国超过100个数据中心。结合该公司全球TGN海底网络，此次投资将显著优化其IZO连接解决方案，包括IZO DC动态连接和IZO多云连接。这些升级后的解决方案将为数据中心和云生态系统提供自愈、永续在线及自助配置的强大能力。
Tata Communications核心及下一代连接服务执行副总裁兼首席技术官Genius Wong强调，这笔投资彰显了公司致力于大规模建设面向未来数字基础设施的坚定承诺。此举不仅能增强全球最繁忙数字走廊之一的连接解决方案的可靠性、可扩展性和性能，更将进一步巩固印度作为区域数字枢纽的战略地位。
