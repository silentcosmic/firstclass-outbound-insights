# 智能代理商务市场潜力洞察：2031年用户规模将达13亿，交易额或突破3.5万亿美元

一份最新发布的研究报告指出，智能代理商务用户数量预计将在2031年达到13亿，远超今年不足3亿的水平，预示着该领域未来巨大的增长空间。
该研究进一步预测，智能代理商务的整体交易额将从2026年的80亿美元，飙升至2031年的3.5万亿美元，实现高达43,240%的惊人增长。
报告将此轮显著增长归因于三大关键要素：主流零售商直接支持的日益普及、消费者对人工智能系统接受度的提升，以及智能代理支付基础设施可用性的增强。尽管当前用户对智能代理商务的熟悉度和信任度仍处于初级阶段，但研究发现，随着人工智能技术深度融入日常生活，以及零售商逐步部署更多智能代理功能，这一局面预计将迅速扭转。
在支付基础设施方面，报告观察到银行卡支付在早期已成为智能代理商务交易的基础支柱，各大卡组织积极参与了最初的试点项目和框架构建。然而，报告也强调，鉴于支付偏好对用户行为的重要性，智能代理商务平台应优先整合本地流行的支付方式。
行业专家指出，尽管银行卡支付正通过代币化等技术持续增强对代理支付的支持，但银行卡在智能代理商务领域的过度主导，并非市场健康发展的最佳路径。这位高管强调，如果未能有效支持数字钱包和账户间转账等本地支付方式，将严重限制整个市场的整体增长潜力。
这份综合性研究套件，提供了涵盖五年期内38,000个数据点的市场分析与预测，并附带两份针对支付基础设施和人工智能开发商的竞争格局排行榜。
