# 缅泰边境诈骗园区：超5300人受困，多国公民卷入

🚨 缅泰边境诈骗园区：超5300人受困，多国公民卷入
据路透社援引人权组织消息，缅甸靠近泰国边境的诈骗园区内，目前仍有超过5300人被困。这些受害者多为外国人，被强制从事网络诈骗活动。
反人口贩运民间社会组织（CSNHTV）已致信泰国警方，敦促其介入解救。据该组织估算，受害者包括约1600名中国公民、200名缅甸公民、20名泰国公民，以及来自菲律宾、台湾、巴西和肯尼亚等地的公民。他们被困在由缅甸武装组织（DKBA）控制的至少四处地点。
CSNHTV指出，由于相关区域尚未得到全面清剿，犯罪活动依然猖獗，受害者范围已遍及全球。目前，缅甸武装组织DKBA及缅甸军政府均未对此事件作出回应。
