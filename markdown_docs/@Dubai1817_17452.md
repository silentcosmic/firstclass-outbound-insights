# 港陆联手捣毁跨境诈骗洗钱集团：69人落网，涉案超2亿

🚨 港陆联手捣毁跨境诈骗洗钱集团：69人落网，涉案超2亿
香港与内地警方近日联合开展代号为“战军”的跨境行动，成功侦破一个庞大的诈骗及洗钱犯罪集团，共拘捕69名涉案人员，其中包括54名男子和15名女子，年龄介于18至60岁之间。该集团涉案总金额高达逾2亿港元。
此次行动源于一名内地女子遭遇的网恋投资骗局。骗子诱导她下载虚假投资软件，谎称投资虚拟货币可获高额回报。今年3月至5月间，该女子按指示将超过1000万港元转入10个香港银行账户，并交付黄金及虚拟货币。为进一步诱骗受害人，骗子甚至曾返还约97万港元作为“投资收益”。
香港警方介入调查后，成功追查并拦截了约470万港元的涉案资金，同时发现骗款被迅速转入106个第二层银行账户。警方顺藤摸瓜，进一步锁定了118个相关傀儡账户，并发现另外173名受害者，揭示了该犯罪集团涉及的庞大诈骗网络。
