# 斯里兰卡机场：饼干盒藏匿香烟走私被查

🚨 斯里兰卡机场：饼干盒藏匿香烟走私被查
斯里兰卡卡图纳亚克国际机场海关近日查获一起香烟走私案，三名自称商人的亚洲籍旅客因试图利用饼干盒藏匿大量香烟入境而被当场截获。
这三名约25岁的旅客，包括两名中国籍公民和一名香港籍公民，于6月30日凌晨搭乘亚洲航空AK-047航班自马来西亚吉隆坡抵达。海关人员在例行检查中发现其行李中的饼干包装箱存在异常，开箱后证实盒内饼干已被香烟取代。
此次查获的中国制造香烟共计42,720支（2,136包），市场价值估算高达640.8万斯里兰卡卢比。斯里兰卡海关已依法没收全部走私香烟，并对涉案旅客处以合计22万斯里兰卡卢比的罚款。目前，案件仍在进一步调查中。
