# 菲律宾华商遭绑索300万美元赎金，两名中国籍嫌犯落网！

🚨 菲律宾华商遭绑索300万美元赎金，两名中国籍嫌犯落网！
近期，菲律宾发生一起华人女商人绑架案，绑匪向家属勒索高达300万美元赎金。菲律宾反绑架小组迅速响应，在两天内成功解救人质。
行动中，警方逮捕了包括两名中国籍嫌犯和一名菲律宾籍嫌犯，另有一名嫌犯在交火后逃脱。警方现场查获涉案车辆、枪支及疑似毒品，并指出该团伙可能长期活跃于马尼拉地区，目前正就其是否涉及更多绑架案进行深入调查。
