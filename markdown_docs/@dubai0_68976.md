# 上海警方捣毁“开店创业”诈骗团伙：涉案超1500万，8000余人受骗

🚨 上海警方捣毁“开店创业”诈骗团伙：涉案超1500万，8000余人受骗
上海警方近期成功捣毁一个以“开店创业”为名实施诈骗的犯罪团伙，共抓获犯罪嫌疑人27名。此案涉案金额逾1500万元，受害者人数多达8000余名，遍布全国各地。
该团伙通过线上广告诱骗受害者，谎称可“宅家创业、月入过万”，并收取数千元不等的“合作费”参与虚假线上“开店培训班”。他们伪造成功案例，利用虚假交易制造“开店即盈利”的假象，以此推销2680元至3980元不等的“运营套餐”。一旦受害者要求退款，团伙便以各种理由推诿拒绝。
上海警方在接到市民肖女士报案后展开调查，发现受害者网店订单多为虚假交易，买家账号存在明显关联。警方随后锁定并联合江西赣州警方，成功展开收网行动。目前，包括李某、朱某在内的4名主犯已因涉嫌诈骗罪被依法批准逮捕，其余23名犯罪嫌疑人亦被采取刑事强制措施。
