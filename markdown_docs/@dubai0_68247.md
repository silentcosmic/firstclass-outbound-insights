# 西港诈骗团伙覆灭：50名中国人被查

🚨 西港诈骗团伙覆灭：50名中国人被查
柬埔寨西哈努克港（西港）近日破获一起重大网络诈骗案件。一个由50名中国公民组成的诈骗团伙被成功查处。
据了解，该团伙长期藏匿在奥彻迭海滩（Otres Beach）附近的一处V度假村内，从事非法网络活动。
