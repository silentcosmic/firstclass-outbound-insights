# 沙迦入室盗窃未遂：屋主意外擒贼判两月

🚨 沙迦入室盗窃未遂：屋主意外擒贼判两月
沙迦轻罪法庭近日对一起入室盗窃未遂案作出判决，一名男子因私闯民宅并企图盗窃被判处两个月监禁，刑满后将被驱逐出境。法庭上的证据链完整，被告本人也当庭认罪。
据了解，事件发生在一居民家中。当时房主与家人外出，因疏忽未锁大门。不料，房主中途返家如厕时，意外发现屋内正有人企图行窃，当场将其制服。经搜身确认，嫌疑人尚未盗走任何物品。
调查人员调取监控录像后还原了事发经过：嫌疑人趁门未锁潜入公寓，曾试图拿走一部手机，但察觉到房主返回后又将其放回。这段监控录像成为案件的关键证据。庭审中，被告对所控事实供认不讳。
