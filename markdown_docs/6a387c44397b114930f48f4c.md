# 越南企业技术服务商Luvina进驻新加坡，赋能亚太数字化转型新浪潮

总部位于越南的软件公司Luvina Software旗下的新加坡实体——Luvina Software Singapore，近日正式启动在新加坡的运营。此举不仅标志着Luvina全球增长战略的一个重要里程碑，更进一步巩固了新加坡作为区域先进技术、人工智能及数字创新枢纽的领导地位。
据Luvina周四发布的一份声明，Luvina Software Singapore是由Luvina Software越南与新加坡咨询公司GleamCore共同成立的战略合资企业。其核心目标是成为亚太地区乃至全球寻求高质量、可扩展且具成本效益的数字化转型解决方案的企业值得信赖的门户。
此次在新加坡设立运营中心，是Luvina弥合数字经济领域日益扩大的鸿沟的关键一步。当前，全球企业正面临在成本攀升和人才短缺的双重压力下，加速现代化改造遗留系统、拥抱人工智能并拓展技术能力的迫切需求。
随着全球企业加速数字化进程，新加坡已然崛起为全球最受信任的技术生态系统之一，并凭借其强大的全球互联互通、高度完善的监管环境以及在人工智能和“智慧国家”倡议方面的日益领先地位，成为连接东西方的战略门户。
Luvina Software选择在新加坡设立业务，正是顺应了市场对能够提供可衡量成果的可靠技术合作伙伴日益增长的需求。
尽管众多企业已将数字化转型置于优先地位，但在实际推进过程中，仍普遍面临高昂的实施成本、交付质量难以保持一致、技术生态系统碎片化以及专业工程人才短缺等障碍。
新加坡对前沿科技的持续投入，包括承诺在2025年至2030年间向国家人工智能研发领域注入超过10亿新元（约合7.8亿美元），进一步巩固了其作为全球可信赖AI枢纽的地位，并为区域及国际拓展提供了理想的启动平台。
Luvina Software于2004年在越南河内创立，现已发展成为一家全球信赖的软件工程公司。其团队规模逾750名员工，累计交付项目超过1000个，在全球范围内服务了超过110家企业客户，业务横跨保险、银行、医疗健康、制造和零售等多个行业。
除在河内、岘港和胡志明市设有交付中心外，Luvina的国际业务足迹已遍布日本、美国，如今又新增新加坡。
Luvina Software Singapore所提供的独特运营模式，巧妙地融合了新加坡的战略咨询与治理优势，以及越南大规模的工程交付能力。这种模式旨在帮助企业在不牺牲质量与管控的前提下，实现系统现代化并加速创新。
作为其在新加坡增长战略的一部分，Luvina Software Singapore计划与解决方案提供商、系统集成商、专业服务公司、风险投资和私募股权基金，以及寻求共同开发面向区域及国际市场可扩展数字产品的企业建立紧密合作。
目前，该公司已在新加坡市场取得初步进展，成功承接了十余个本地项目。根据其后续增长阶段规划，Luvina将进一步将其业务版图拓展至东南亚、澳大利亚及欧洲市场。Luvina Software联合创始人兼首席执行官Le Quang Luong指出：“对我们而言，新加坡不仅是一个全新的市场，更是一个战略性的门户。我们期望通过新加坡，与区域伙伴共同擘画企业技术的未来。”他进一步强调：“随着企业面临现代化改造、采纳人工智能并信心十足地拓展规模的日益增长的压力，它们需要的不仅仅是更多的技术供应商，而是能够连接战略、执行与可衡量成果的可靠合作伙伴。”Luvina Software Singapore首席执行官Ethan Lim表示，通过Luvina Software Singapore，他们将新加坡的全球互联互通及治理优势、越南的卓越工程能力，以及日本根深蒂固的质量与可靠性标准融为一体，旨在助力企业以更快的速度、更大的信心和更强的责任感实现转型。他补充道：“归根结底，我们的使命是将复杂的任务转化为企业真正依赖并为之买单的切实成果。”
展望未来，Luvina Software Singapore计划深化其作用，持续助力企业实现遗留系统的现代化，加速人工智能技术的采纳，并共同开发可在全球商业化的可扩展数字解决方案，以此进一步巩固新加坡作为区域战略创新中心的地位。
