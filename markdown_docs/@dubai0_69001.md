# 迪拜杰贝阿里路多车相撞：25人伤，警方再发安全提醒

🚨 迪拜杰贝阿里路多车相撞：25人伤，警方再发安全提醒
迪拜杰贝阿里路近日发生一起严重多车连环相撞事故，导致25人不同程度受伤，并已全部送往医院接受治疗。
据警方通报，事故涉及一辆轿车和三辆巴士。初步调查显示，其中一辆巴士因技术故障突然停在行驶车道上，后方车辆未能及时避让，引发追尾。迪拜警方借此再次向公众发出安全提醒，强调车辆在行驶中若遇故障、燃油耗尽或爆胎等突发情况，切勿将车辆停留在路中间，以免造成二次事故。
