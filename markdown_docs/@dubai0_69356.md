# ️ 阿布扎比法院重罚：WhatsApp侮辱言论代价高昂

⚠️ 阿布扎比法院重罚：WhatsApp侮辱言论代价高昂
阿布扎比刑事法院近日对一名女子作出判决，因其通过WhatsApp语音信息侮辱前同事，被判处3万迪拉姆（约合5.9万元人民币）罚款，并须额外支付2.1万迪拉姆（约合4.1万元人民币）的临时赔偿金。
此案再次提醒在阿联酋，数字通讯行为同样受到严格的法律约束。对于出海企业及个人而言，职场内外通过社交媒体或通讯工具进行的言论交流，尤其需谨慎，避免不当言辞可能带来的严重法律后果及经济损失。
