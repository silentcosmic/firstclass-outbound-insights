# 阿布扎比卢浮宫今夏推新展：深度探索全球艺术瑰宝

🔍 阿布扎比卢浮宫今夏推新展：深度探索全球艺术瑰宝
阿布扎比卢浮宫近日宣布，今夏将推出“全新亮点与新发现之旅”特别参观行程。这一45分钟的导览将带领访客穿越博物馆四大展翼，深入探索一系列精心挑选的新入藏品及本年度引入的重要借展作品。
此次旅程旨在呈现博物馆的持续发展及其致力于多元艺术叙事的承诺，通过新增展品进一步丰富其常设展厅内容。
新增亮点包括多件珍贵展品：如来自加蓬、与祖先传统和仪式活动紧密相关的科塔—奥班巴遗骨匣守护雕像；一枚展现罗马帝国肖像艺术的浮雕宝石；以及一盏象征光在伊斯兰文化中深远意义的马穆鲁克清真寺灯。
此外，来自地区及国际机构的借展作品同样引人注目，例如来自巴林的早期迪尔蒙时期陶罐，为理解古代海湾地区的贸易网络与仪式生活提供了独特视角；以及一尊来自西欧、反映人类早期形象表现形式的新石器时代人形石碑。
整个参观行程横跨多个展厅，从海湾地区的早期文明追溯至全球的古典、现代与当代艺术传统，旨在全面展现艺术表达随时代演变的轨迹，并揭示不同文化之间共通的深刻主题。
