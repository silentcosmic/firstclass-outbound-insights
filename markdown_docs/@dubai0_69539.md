# 柬埔寨西港星汇物业突遭逮捕，行业担忧加剧

🚨 柬埔寨西港星汇物业突遭逮捕，行业担忧加剧
市场消息指出，柬埔寨西哈努克港（西港）星汇物业近日遭遇紧急情况。据内部通知，该公司多名相关人员在过去两天内已被当地执法部门采取行动。
此次事件在当地地产及相关行业中引发广泛关注与不安。有行业人士担忧，在当前的监管环境下，未来是否会有更多物业公司面临类似调查或行动，行业前景的不确定性进一步升高。
