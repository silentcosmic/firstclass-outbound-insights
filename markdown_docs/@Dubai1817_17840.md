🔍 柬埔寨‘嘎腰子’传闻再起：揭秘地下器官交易真相
近日，有消息指出一名日本公民在柬埔寨完成了肾脏移植手术，此事迅速引发了公众对当地“嘎腰子”传闻的再次关注和担忧。
然而，市场观察人士指出，尽管非法器官交易在全球范围内普遍存在，其性质与网络上流传的暴力强制“嘎腰子”说法存在本质区别。多数此类交易，包括此次事件，通常涉及供体和受体双方在地下市场中的自愿行为，即一方出于经济需求出售器官，另一方支付费用获取。
这与强制性、非自愿的器官摘取有着显著差异。因此，仅凭此类自愿性质的非法交易，将柬埔寨与普遍的“嘎腰子”犯罪划等号，可能存在过度简化和误读的风险。
