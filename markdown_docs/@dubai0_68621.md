# 柬越老三国联手：常驻机制剑指网络诈骗与跨境犯罪

🤝 柬越老三国联手：常驻机制剑指网络诈骗与跨境犯罪
东南亚区域合作再升级。日前，柬埔寨副总理兼内政部部长苏速卡倡议，柬越老三国将建立官员常驻协作机制，以深化对网络诈骗及跨境犯罪的联合打击。
此项提议是在苏速卡与来访的越南公安部部长梁三光、老挝公安部部长万通·贡马尼会晤时提出的。该机制旨在实现高效情报共享，及时交换外籍犯罪分子“黑名单”数据，并简化涉案人员的遣返出入境流程。
越老两国部长对该倡议表示高度赞同与支持，三方一致认为，此举将显著提升区域执法合作效能，并有望尽快落地实施。
