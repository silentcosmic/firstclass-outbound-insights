# 新加坡职场AI应用深度洞察：青年劳动力使用率缘何不及资深群体？

在新加坡的职场中，一个出人意料的趋势正在浮现：年龄介于18至26岁的青年员工，在所有年龄段中，对人工智能（AI）的日常使用频率最低，仅占20%。这一比例不仅低于27至39岁年龄段的25%，也落后于40至54岁群体的23%。这项数据揭示了AI在劳动力中渗透的复杂性，并非简单地与年龄成反比。
行业分析指出，尽管人工智能在不同群体中的采纳速度不一，但新加坡职场对AI未来影响的担忧已显著下降。市场普遍认为，随着技术演进，对AI可能取代岗位的焦虑正逐渐被对其赋能潜力的理解所取代。
具体来看，目前仅有10%的新加坡员工对AI如何改变自身工作感到不确定，这一数字相较于去年19%的比例已大幅下降。伴随这种信心增长的是AI使用率的提升：56%的员工每周多次使用AI工具，而整体而言，每天使用AI的员工比例已达到23%。
AI技术的采纳程度同样展现出与企业规模的显著关联。拥有250至999名员工的中型组织在AI日常使用方面表现领先，其日常使用率高达34%，超越了区域平均水平。相比之下，员工人数在1至249人的小型企业与员工超过1000人的大型企业，其AI日常使用率均为19%。
有资深人力资源专家观察到，新加坡的劳动力对人工智能的信心日益增强。小型企业或能从更容易获取的实用且高性价比AI工具中获益。然而，大型组织可能需要进一步简化其内部流程，以有效扩大AI应用规模。值得关注的是，中型企业常能通过资源与敏捷性的恰当平衡，展现出AI大规模应用的巨大潜力。
尽管AI使用率持续攀升，但仍有相当一部分员工尚未完全确信其对自身工作的实际价值。例如，在新加坡，仅有15%的员工在2025年明确表示，AI将在未来一年内对其工作职责产生积极影响。这种认知上的落差并非个例，全球范围内也呈现出类似趋势：30%的日常AI用户表示工作投入度很高，然而，这些日常用户感到生产力下降的可能性却是非用户的四倍。
在新加坡，工作类型也显著影响着员工对AI的看法。知识型工作者对AI影响其工作职责的乐观程度最高，达到22%。这一比例大约是技能型任务工作者（11%）的两倍，更是重复性任务工作者（7%）的三倍。
经济学界专家强调，AI不仅正在重塑工作方式，也在深刻改变人们在工作中的感受。频繁使用AI工具的员工普遍报告有更高的投入度和更低的压力，但与此同时，他们也感到生产力有所下降。这凸显了引导员工适应并过渡到与AI技术协同工作新模式的重要性。
全球视野下的AI采纳动向
此外，有国际趋势报告显示，新加坡在AI采纳方面领先于全球许多地区，这进一步印证了其在技术创新和劳动力转型方面的积极姿态。这些数据共同描绘了一幅复杂而动态的AI应用图景，对企业战略规划具有重要的参考价值。
