# 马来西亚：吉隆坡雪兰莪今晚启动联合执法行动

🚨 马来西亚：吉隆坡雪兰莪今晚启动联合执法行动
据市场消息，马来西亚雪兰莪和吉隆坡多地今晚将展开联合执法行动。此次行动由交通警、移民厅及皇家警察局共同执行，主要针对来往车辆进行排查，重点检查毒驾、酒驾、证件及其他相关违法行为。
行动预计从晚上8点55分持续至午夜12点。具体设卡地点包括巴生地区联邦100号环岛后方、旧拉什纳餐厅门前，涉及通往伯克利桥并朝莎亚南i-City环岛方向的联邦公路主干道及辅路。此外，今晚8点45分至午夜，移民厅和交通警察也将在KESAS高速公路哥打甘文宁收费站后方，往莎亚南及吉隆坡市区方向设置检查点，对所有车辆进行检查。
建议今晚需途经巴生、莎亚南、吉隆坡方向的商务人士和驾驶员，务必随身携带齐全的证件、驾照和车辆资料，并严格遵守交通法规，切勿酒后驾车。非必要情况下，建议避免深夜出行，以确保行程顺畅。
