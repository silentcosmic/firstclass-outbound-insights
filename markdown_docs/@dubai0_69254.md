# 金边大规模清查：跨国‘杀猪盘’电诈窝点被捣毁

🚨 金边大规模清查：跨国‘杀猪盘’电诈窝点被捣毁
柬埔寨金边市政府于7月6日晚，由市长坤盛亲自指挥，对市内94处公寓、酒店等可疑场所进行了大规模突击清查，共盘查外籍人员286人。
金边市政府发言人道桑帕昨日证实，此次行动在朗哥区一处住宅内成功捣毁一个跨国网络诈骗窝点。该窝点涉嫌利用“网络恋爱”（俗称“杀猪盘”）诱骗受害者进行虚假投资，主要目标指向美国、巴西、墨西哥和哥伦比亚等国。现场抓获8名外籍嫌疑人（包括7名孟加拉国男子和1名缅甸女子，其中3人无有效护照），并查获了12台笔记本电脑、17部手机、大量网络设备及49张伪造美元。
目前，所有涉案人员及相关证物已移交金边市警察局，将依法进行后续处理。
