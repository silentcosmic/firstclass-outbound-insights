# 香港警示：63岁阿伯网恋被骗，损失超468万令吉

🚨 香港警示：63岁阿伯网恋被骗，损失超468万令吉
香港近日发生一起令人震惊的网上情缘诈骗案。一名63岁长者因轻信自称“楼下住户”的陌生人，从最初的噪音投诉问题，逐渐发展成“网上恋人”关系。骗徒取得信任后，随即诱导他投资虚假的加密货币项目。
受害长者先后分17次转账逾700万港元（约363万令吉）至骗徒指定账户。当他试图提取“利润”时，又被要求亲手支付200万港元（约104万令吉）现金作为“手续费”。直至对方突然失联，长者才意识到自己被骗，总计损失高达900万港元（约468万令吉）。
香港警方透露，过去一周已接获超过20宗同类网上情缘诈骗报案，累计损失金额逾1770万港元（约920万令吉）。此案再次提醒公众对网络交友及投资陷阱保持高度警惕，务必核实对方身份，切勿轻信高额回报承诺。
