# 马来西亚大规模清查：逾1.7万无证外籍人士被捕

🚨 马来西亚大规模清查：逾1.7万无证外籍人士被捕
马来西亚移民局近期披露，该国已完成一轮全国性的非法外籍劳工大规模清查行动。截至5月25日，共计逮捕超过1.7万名无证外籍人员。
此次行动共开展5010次清查，核查71734人。此外，有789名涉嫌违规雇佣无证劳工的雇主也一并被捕。此次整治行动彰显了马来西亚政府规范劳务市场、打击非法雇佣的决心。
