# 美国H-1B签证政策不确定性加剧，科技人才加速外流

美国联邦法院日前宣布，废止了前总统唐纳德·特朗普时期推出的高达10万美元的H-1B签证附加费用。尽管这一裁决为部分外籍科技工作者带来一丝喘息，但长期以来美国移民政策的反复与不确定性，已深刻影响了他们的职业规划。许多在美工作的外籍专业人士指出，政策的摇摆不定正促使他们重新审视职业发展路径，并积极考虑转向加拿大、英国乃至海湾国家等其他更具稳定性的目的地。
