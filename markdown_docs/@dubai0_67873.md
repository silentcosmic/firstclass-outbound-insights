# 京东刘强东：70万快递员将转型技术岗，应对AI冲击

⚡ 京东刘强东：70万快递员将转型技术岗，应对AI冲击
在近日举行的2026 APEC工商领导人中国论坛上，京东集团创始人、董事局主席刘强东透露了一项名为“涅槃计划”的内部战略。此举旨在应对人工智能和技术发展对传统蓝领岗位的冲击，计划将京东70万快递员等基层员工送入学校进行技术培训，实现职业转型。
刘强东表示，未来物流配送将完全由机器人承担，但京东承诺不会因此让员工失业。他构想，通过专业培训，这些蓝领员工将转型为技术型白领，负责机器人的维护与管理，工作环境也将从风雨兼程的户外转变为更稳定、更舒适的室内办公。这一计划体现了京东在科技变革中对员工发展的长远考量。
