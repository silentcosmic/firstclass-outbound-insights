# 越南警方重拳出击：百人诈骗团伙疑涉中国资金

⚡ 越南警方重拳出击：百人诈骗团伙疑涉中国资金
越南胡志明市警方近日在第3郡一处写字楼内发起突袭行动，成功捣毁一个大型电信网络诈骗团伙，现场控制逾120名涉案嫌疑人。
据初步调查，该团伙主要利用各类社交和交友平台，通过“网恋”等情感培养方式接触受害者，进而诱骗其进行转账或投资。警方指出，该犯罪网络组织严密，分工明确，长期从事跨境诈骗活动，现场已查获大量作案工具。
值得注意的是，警方初步掌握线索表明，该团伙的幕后资金支持方可能涉及中国籍人员。目前，当地警方正围绕资金流向、组织架构及涉案人员身份展开深入调查，誓将依法追查所有组织者及主犯。
