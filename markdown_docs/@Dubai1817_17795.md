# 斯里兰卡突击检查：中资商铺面临合规挑战

🚨 斯里兰卡突击检查：中资商铺面临合规挑战
近日，斯里兰卡科伦坡的营商环境趋严。一位在当地经营中国粮油副食品杂货铺的华商透露，自七月初以来，斯里兰卡政府监管部门对外国人集中的综合市场进行了密集的食品安全与经营合规检查。
该华商的店铺即是受检目标之一。监管人员主要关注商品外包装标识、介绍及日期等信息，发现部分纯中文外包装商品不符合当地规定，要求立即整改。尽管此次仅给予口头警告，但明确指出若再犯将面临商品没收、罚款，甚至吊销营业执照的严厉处罚。
据悉，该市场内其他四家外籍商铺也无一幸免，其中一家已因违规被处以数万卢比罚款。此次集中检查被视为对当地外籍商户营商环境的一次“警钟”。分析认为，随着在科伦坡工作和消费的外国人数量激增，当地商业活动日益活跃，监管部门对市场规范和食品安全的要求也随之升级。
