# 马斯克重磅宣布：X平台代码将无条件开源并接受三方审查

📣 马斯克重磅宣布：X平台代码将无条件开源并接受三方审查
埃隆·马斯克近日宣布，X平台将在完成安全漏洞审查后，其全部代码库将无条件向公众开源。此举旨在全面提升平台透明度与用户信任。
马斯克进一步表示，X还将邀请独立的第三方机构审查正在运行的系统，以核实开源代码与实际运行代码的一致性。他强调，完全透明所带来的信任，是平台最核心的价值。
