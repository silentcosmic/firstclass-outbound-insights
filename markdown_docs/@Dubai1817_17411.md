# 柬埔寨高官身份遭冒用，嫌犯诈骗超11万美元落网

🚨 柬埔寨高官身份遭冒用，嫌犯诈骗超11万美元落网
柬埔寨警方近日成功破获一起长期冒充政府高层身份的诈骗勒索案。一名嫌犯因盗用柬埔寨国家警察总署副总监迪伟杰中将的身份信息，累计实施约50次诈骗，涉案金额超过11万美元后被捕归案。
此案最初因迪伟杰中将本人在官方社交媒体上公开悬赏，征集盗用其照片和姓名进行诈骗的线索而引发关注。警方调查发现，嫌犯主要通过伪装成电商卖家，以“系统故障”等借口诱骗受害者转账。一旦受害者产生怀疑，嫌犯便会切换至伪造的社交媒体账号，冒充迪伟杰中将，利用其职务身份对受害者进行威胁恐吓，迫使其支付钱财。
警方指出，嫌犯利用公众对执法部门的信任实施犯罪，不仅给受害者造成经济损失，也严重损害了国家机关的形象。目前，此案已移送金边法院依法处理。迪伟杰中将也再次呼吁公众提高警惕，切勿轻信社交媒体上自称政府官员或执法人员的账号，并及时向警方核实举报可疑情况。
