# AI信息失准扩散：品牌声誉管理与智能校准工具新趋势

传统观点认为“所有曝光都是好的曝光”，然而，对于任何一位资深营销人士而言，这并非总能成立。
随着越来越多的消费者转向人工智能进行产品研究，品牌方必须关注的不仅是何时出现在搜索结果中，更重要的是以何种方式呈现。
Profound公司产品营销经理Garrett Gomez强调，营销人员必须将消费者情绪和信息准确性置于首位。
他指出，仅仅关注品牌是否出现，是一种“典型的搜索引擎优化（SEO）思维”。在人工智能搜索的语境下，品牌方还需自问：“人工智能所提及的我的产品和服务，其信息是否准确无误？”
Profound公司推出了一款面向品牌方的全新工具，名为FactCheck。该工具通过比对大型语言模型（LLM）的输出与品牌自身的知识库，并标记出其中的不一致之处，从而解答了上述疑问。
人工智能搜索结果的准确性，对品牌声誉具有举足轻重的影响。
即使不准确的信息源于大型语言模型，品牌方也可能为此承担后果。一家领先的内容管理系统供应商近期数据显示，若人工智能系统生成了关于某品牌的不实信息，即便这些数据并非源自品牌自有渠道，仍有21%的消费者会将责任归咎于品牌本身。
FactCheck工具将大型语言模型的输出与品牌方的知识库进行比对，该知识库由品牌方上传至Profound平台的内容组成。Profound公司数据科学家Théophile Langanay表示，这些内容通常涵盖了网站信息、公开文档以及任何包含“品牌希望确保在各类平台上得到准确呈现的重要事实”的内部文件。为简化操作流程，Profound已与市场上广受客户欢迎的主流知识库技术平台，如某知名云存储和某协作平台，建立了集成。
Gomez解释，该工具能够识别不准确的表述，例如已停用的功能或过时的价格，随后列出知识库中的“基本事实”，以“证明引用的信息有误”。在Profound平台内部的FactCheck标签页中，用户可以一站式查阅不准确的表述、对应的“基本事实”以及影响该表述的引用来源，从而获取所有相关信息。
除了发现不准确信息，FactCheck更进一步，提供纠正这些信息的指导方案。
一家知名健身可穿戴品牌（WHOOP）在使用FactCheck后发现，引用该品牌的AI回复中有7.9%因准确性问题被标记。该品牌高级业务分析师Sophia Scaglioni Melegari指出，其中一些问题是将其他品牌最新一代的可穿戴设备与该品牌旧款产品进行比较，这无异于“将苹果和橘子作比较”。
然而，该品牌通过FactCheck工具得以纠正这些偏差，该工具将不准确的信息标记并提交给其团队进行审核。
Gomez表示，如果信息不准确源于品牌自有网站，例如引用了过时的URL，品牌可以部署Profound的代理程序，重写内容以纳入经批准知识库中的最新数据。
但有时，不准确信息则来自第三方网站，例如竞争对手的平台。
Gomez指出：“显然，您无法直接更新竞争对手的页面。”在这种情况下，Profound将启用其内部专门用于内容创作的代理程序，为品牌网站开发新的内容，以“抵消”竞争对手网站上的错误信息。
当然，这也带来了一个新的挑战：如何确保品牌的优质内容能够优先呈现，而非竞争对手的错误信息。
Profound公司自然无法保证谁的内容会优先展示，但它可以向品牌方提供最佳实践建议，帮助其内容在AI搜索中获得比竞争对手更高的提及率——这正是Profound的核心服务所在。Profound的洞察工具每天都会向大型语言模型发出提示，追踪回复中出现的来源类型，并据此推荐相应的内容策略。
即使新内容在AI查询中未能如品牌预期般频繁出现，希望也并未完全泯灭。Langanay表示，Profound作为一个具备“闭环自动化反馈机制”的衡量工具，“我们能够了解其效果如何，并进行迭代和重新尝试。”
然而，有时错误的来源既非品牌方，也非其竞争对手，而是一位独立出版商，例如某个博客。Gomez表示，在这种情况下，内容通常会关联到作者姓名，Profound会部署一个外联代理程序，起草邮件给作者，请求其进行修正。
Profound的客户拥有对工作流程的“完全自主权”，因此校正过程可以从完全自动化到每一步都需人工干预。例如，上述健身可穿戴品牌在整个过程中保持人工审核，但利用Profound的代理程序来建议更新和内容刷新。
该品牌的产品营销专家Kieran Riley表示：“如果人工智能对我们产品（WHOOP）提供错误信息，我能够捕捉到它，找出其来源，并启动修正程序。这在过去对我而言是完全不可见的。”
