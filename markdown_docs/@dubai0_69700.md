# 乌干达突袭：367名中国公民因非法工作被捕，面临起诉与遣返

🚨 乌干达突袭：367名中国公民因非法工作被捕，面临起诉与遣返
乌干达内政部近日宣布，此前在恩德培联合执法行动中被拘留的367名中国公民将面临刑事指控，随后依法启动遣返程序。调查显示，这批公民均持旅游签证入境，但实际从事工作活动，涉嫌违反乌干达移民法律关于外国人在境内就业的规定。
据乌干达《New Vision》报道，这批中国公民是在恩德培地区被查获的，警方最初公布的被捕人数为365人，经核实后修正为367人。目前，所有涉案人员已被安置在安全羁押地点，案件材料正在加紧整理，将尽快移交法院审理。司法程序完成后，确认属于非法居留或非法工作的人员将被依法遣返回原籍国。
此次行动中，恩德培的Country Lake Resort Hotel和Katomi Kingdom Resort Hotel两家涉事酒店已被安全部门封锁，调查人员正现场搜集证据。执法部门还查获了一批高科技电子设备，正进行司法鉴定以确认其用途及相关互联网设备如何进入乌干达境内。
乌干达政府强调，此次行动旨在保护本国公民、维护国家安全并严格执行移民法律。政府同时呼吁酒店、度假村及房产业主加强对长期入住外国人的身份核查，及时举报异常情况。
