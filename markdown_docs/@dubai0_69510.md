# ️ 阿联酋e&amp;集团退出沃达丰：溢价13%套现59.5亿美元

⚡️ 阿联酋e&集团退出沃达丰：溢价13%套现59.5亿美元
阿联酋电信集团公司（e& Group）近日宣布，已签署一项具有约束力的协议，将出售其持有的沃达丰集团有限公司的全部股份。此次交易总对价为每股112.5英镑，总计约59.5亿美元，相较市场价格溢价13%。此举标志着e&集团将完全退出对沃达丰的投资。
