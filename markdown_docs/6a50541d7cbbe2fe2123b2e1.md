# 行业风向 | 丨老乡鸡五年五次冲击上市失败；两款燃油车重回销量榜前十；长鑫科技披露招股意向书，预计7月16日发行

LXJ International Holdings Limited（简称“老乡鸡”）于2026年1月8日递交港交所的招股书于7月8日失效。这家被称为“中式快餐第一”的企业，五年间已第五次冲击资本市场失败。（界面新闻）
在短暂告别销量榜前十后，燃油车又杀回来了。在新能源车渗透率连续三个月维持在60%以上的市场趋势下，两款燃油车——丰田凯美瑞与大众朗逸——于2026年6月重回乘用车零售销量前十，打破了5月销量榜没有燃油车的局面。
懂车帝发布的5月乘用车零售销量排行榜显示，丰田凯美瑞以17114辆位列第九，大众朗逸以15444辆排名第十。而此前的5月份，销量前十榜单已历史性地被新能源车型包揽。彼时朗逸已跌至第18位，轩逸第20位，凯美瑞更退居第24位。零售销量TOP 10中，特斯拉Model Y以38654辆居首，吉利星愿33359辆位列第二，零跑A10以24865辆位列第三。
燃油车此番回温，受到油价相对下行的影响。6月国内成品油价迎来两轮下调，92号汽油两个月累计合计每升下调0.81元。用车成本的实质性下降，在一定程度上修复了燃油车的使用经济性。另一方面，大宗材料及芯片价格的持续上涨，新能源汽车在成本端压力增大，部分车型优惠力度有所收缩。此消彼长之下，燃油车的性价比优势阶段性回升。（第一财经）
