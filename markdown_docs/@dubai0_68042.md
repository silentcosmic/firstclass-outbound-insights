# 洪森访华：中柬“铁杆友谊”再升级，或谈打诈与投资

📣 洪森访华：中柬“铁杆友谊”再升级，或谈打诈与投资
据官方消息，柬埔寨参议院主席、人民党主席洪森亲王将于2026年6月25日至27日率高级别代表团对中国进行正式友好访问。此次随行人员阵容强大，包括国王高级顾问兼人民党副主席狄班等重要领导人。
访问期间，洪森亲王将与中国国家主席习近平举行双边会谈，并分别会见国务院总理李强和全国人大常委会委员长赵乐际。此外，洪森亲王还将拜会目前正在中国疗养的柬埔寨国王西哈莫尼及太后。
此次访问被视为两国深化“铁杆友谊”的坚定信号，旨在全面推进新时代柬中命运共同体建设，推动双边关系迈上新台阶。市场观察人士分析，除了官方议程，此次访问也可能涉及阶段性反诈成果的汇报，以及探讨两国间的投资与经贸合作新机遇。
