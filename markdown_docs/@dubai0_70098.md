# 迪拜出台新规：航班延误取消，乘客权益获全面保障

📣 迪拜出台新规：航班延误取消，乘客权益获全面保障
随着夏季出行高峰期的到来，全球各地航班运营仍面临诸多不确定性。在此背景下，迪拜民航局（DCAA）发布了《航空消费者福利指令》，为往返阿联酋的乘客在航班延误、取消或受其他干扰时，提供了更明确的权益保障和航空公司责任指引。
这项新指令旨在提升航空出行的透明度和问责机制，适用于总部设在迪拜的国家航空公司及其合作伙伴、从迪拜各机场运营的外国航空公司，以及持牌旅行社。它详细规定了航空公司在订票前后必须提供的信息，乘客在航班受影响时应获得的协助，以及问题未能解决时的投诉与调解渠道。
具体而言，航空公司在售票时必须清晰列明机票总价（含所有税费）、预订条件（如取消和退款政策）、实际承运航空公司信息、完整航班行程、行李额度及相关费用，以及票价类型的所有限制。
当航班发生延误或取消时，乘客有权及时获得航空公司或旅行社的最新通知，了解受影响程度、原因和预计延误时间。根据《民航条例——乘客福利计划》（CAR-PWP），航空公司必须提供改签至下一班可用航班、在可行情况下改乘其他航空公司航班，或获得未使用部分机票全额退款或代金券等选择。
对于长时间延误（通常为3至8小时），乘客应获得合理的照顾与协助，包括餐食饮料、通讯服务，以及必要的酒店住宿和机场与酒店间的交通。指令还强调，航空公司必须优先为残障人士、无人陪伴儿童和老年乘客等弱势旅客提供额外支持。
值得注意的是，乘客应首先给予航空公司合理机会安排替代方案。若航空公司明确拒绝、无法联系或仅提供不合理的替代航班，乘客可自行预订替代航班，但需保留所有收据，且报销通常限于合理且与原行程相当的费用。此外，即使航班状态仍显示“准时”，乘客也应避免在未收到官方取消通知前自行取消机票，以避免影响自身权益。
若乘客认为自身权利未得到尊重，可直接向航空公司或持牌旅行社反映，指令中也明确了争议解决和调解程序。
