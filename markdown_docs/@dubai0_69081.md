# 以色列首认：铁穹系统曾部署阿联酋

🚨 以色列首认：铁穹系统曾部署阿联酋
以色列交通部长米里·雷格夫近日证实，以色列曾于美以伊冲突初期向阿联酋部署“铁穹”防空系统。这是该系统首次被部署到以色列和美国以外的国家，也是以色列政府官员首次公开承认这一重要军事合作。
此前，美国官员已多次透露相关消息，并指出“铁穹”在近期冲突中曾协助阿联酋成功拦截伊朗导弹。此举进一步凸显了自2020年签署关系正常化协议以来，以色列与阿联酋之间不断深化的经济和军事联系，也标志着中东地区安全合作的新动向。
