# 美国制裁升级：迪拜伊朗银行家被盯上

🚨 美国制裁升级：迪拜伊朗银行家被盯上
美国财政部日前宣布，针对伊朗实施新一轮制裁，此举正值伊朗涉嫌袭击通过霍尔木兹海峡的商船之后。本次制裁重点针对常驻阿联酋迪拜的伊朗银行家兼商人阿里·安萨里，以及其他13个个人和实体。
美方指出，安萨里被视为伊朗最高领袖穆杰塔巴·哈梅内伊的“关键资助者”，此前已因向伊朗伊斯兰革命卫队等机构提供资金支持而受到英国制裁。美国财政部外国资产控制办公室（OFAC）将三家伊朗货币兑换所、数家外国公司及相关人员列为制裁对象，指控其每年通过“空壳公司”为受制裁的伊朗银行转移数十亿美元资金。
与此同时，伊朗伊斯兰议会议长卡利巴夫警告，若美方背弃上月达成的谅解备忘录，伊方已做好“全面防御”准备。巴基斯坦、卡塔尔、埃及等多国正加紧斡旋，力促美伊双方重返谈判。
