# ️ 迪拜华人圈曝诈骗与剥削指控：三名涉事者遭点名

⚠️ 迪拜华人圈曝诈骗与剥削指控：三名涉事者遭点名
近日，有匿名爆料在迪拜华人社群中流传，指控一个由三名华人组成的团队涉嫌对在当地工作的同胞进行诈骗与剥削。该爆料点名“小天”、“天一”及“菲菲”三人，并详细列举了他们的不法行为。
其中，被称为“妈咪”的菲菲被指控利用职务之便，克扣女性从业者的收入，并涉嫌欺诈小费。爆料指出，菲菲此前在柬埔寨等地已有不良记录，并因声誉受损而转移至迪拜。
另一名核心人物“天一”（据称曾用名“王建”）则被指控在全球多国从事欺诈、勒索等非法活动，并因此成为多地通缉对象，包括在中国和泰国。爆料称，其惯用手法是利用新身份继续欺骗同胞。
而“小天”则被指控以各种理由拖欠并侵吞女性从业者的工资，甚至对要求结清款项的受害者进行威胁恐吓，声称其在迪拜及国内均有“关系”，可对受害者回国后实施报复。爆料称，这三人通过相互勾结，已对多名女性同胞造成经济损失和精神压力。
此事件再次提醒海外华人社群，在异国他乡工作和生活中需提高警惕，防范各类潜在的欺诈与剥削风险。相关受害者呼吁更多人站出来，共同揭露这些不法行为。
