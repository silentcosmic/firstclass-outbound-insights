# 日本收紧三国籍及一省份签证：严打诈骗

🚨 日本收紧三国籍及一省份签证：严打诈骗
日本政府近日宣布，为有效遏制日益猖獗的跨境电话和电商诈骗活动，将对部分国家和地区的人员加强入境审查。这些诈骗活动主要源自东南亚及南太平洋地区，涉及大量中国籍犯罪分子。
自2026年7月15日起，日本外务省将对来自柬埔寨、瓦努阿图以及中国福建省的人员实施更严格的签证签发和入境审查程序。对于柬埔寨公民，主要影响特定技能工人、技能实习生和商务签证持有者。瓦努阿图公民则涉及旅游、商务、特定技能工人及家庭团聚等多种居留身份。值得注意的是，针对来自中国福建省的人员，特别是闽南地区出身者，无论其当前居住地，都将面临更为严格的审查。此举旨在从源头上进一步打击日本境内的诈骗犯罪。
