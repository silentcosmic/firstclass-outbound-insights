# 柬埔寨重拳反诈：捣毁超500窝点，起诉近2000人

⚡ 柬埔寨重拳反诈：捣毁超500窝点，起诉近2000人
柬埔寨国家打击网络诈骗委员会秘书长蔡西那烈近日接受日本TBS电视台采访，重申柬埔寨政府打击网络诈骗的坚定决心。他指出，跨境诈骗已成为全球性危机，不法分子利用柬埔寨领土对海外行骗，严重损害了国家声誉。为此，洪玛奈总理已下令集结各方力量重拳出击，并出台了前瞻性的《反电信网络诈骗法》。
蔡西那烈介绍，自2025年初至2026年6月22日，柬埔寨警方已捣毁超过500处涉诈窝点，查封400余处，并吊销或暂停了25家赌场牌照。目前，已有184起案件被移交法院，共起诉1761人，其中包括涉及33个国籍的2000多名外籍嫌疑人。他强调，柬埔寨将持续收紧外国人管理，对任何电信诈骗行为绝不姑息。
