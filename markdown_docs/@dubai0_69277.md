# 伊朗恢复霍尔木兹海峡袭击：商船遭导弹攻击

🚨 伊朗恢复霍尔木兹海峡袭击：商船遭导弹攻击
据美国官员透露，伊朗于周一晚间再次向霍尔木兹海峡的商业船只发射了至少两枚导弹，标志着其在该区域袭击行动的恢复。
此次袭击导致两艘船只受损，但目前尚未有人员伤亡报告。此事件无疑将再次加剧全球航运市场的紧张情绪。
