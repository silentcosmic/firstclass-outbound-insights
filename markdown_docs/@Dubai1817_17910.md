# 吉隆坡蕉赖深夜突查：多人被带走，详情待公布

🚨 吉隆坡蕉赖深夜突查：多人被带走，详情待公布
7月11日凌晨约3时，马来西亚吉隆坡蕉赖Memi ZoooZooo一带突发执法行动。大批执法人员进入相关场所，对现场人员进行身份核查与登记，迅速控制了现场秩序。
