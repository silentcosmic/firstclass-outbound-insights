# 全球产业风向 | 智谱科技完成巨额H股配售；荣耀与阿里巴巴深化AI终端合作；小米机器人在汽车制造实现突破

中国神华发布业绩展望，预计2026年上半年归属于上市公司股东的净利润将达到263亿元至298亿元，实现6.9%至21.1%的同比增长。公司盈利增长主要得益于煤化工业务量，以及自有铁路、港口、航运业务量的显著提升，有效带动了相关业务利润的增长。
中国人寿的半年报预告显示，其2026年上半年归属于母公司股东的净利润预计在1289.33亿元至1371.19亿元之间，同比增长高达215%至235%。业绩表现强劲的原因在于公司持续加强资产负债联动管理，优化资产配置策略，并显著提升了投资专业能力，从而取得了出色的投资回报。
光迅科技发布业绩预告，预计2026年上半年归属于上市公司股东的净利润将达到5.59亿元至6.15亿元，同比增幅介于50%至65.15%。本报告期内，全球对人工智能算力投资的热潮持续，显著推动了数通光模块需求的快速增长。同时，国内云服务提供商也大幅增加了数据中心建设投入。光迅科技凭借其全面的产品体系，覆盖了数据中心和通信网络的全场景应用，有力支撑了经营业绩的持续攀升，预计净利润实现显著增长。
Meta于近期公布，将大幅扩展其位于美国路易斯安那州里奇兰县的“许珀里翁”（Hyperion）数据中心超级集群项目。该项目的规划容量将跃升至5吉瓦（GW），预计总投资额将突破500亿美元。行业分析师认为，在当前人工智能（AI）投资热潮的驱动下，包括Meta、微软、谷歌、亚马逊在内的科技巨头企业正加快其全球算力基础设施的部署步伐。
赣锋锂业发布业绩预告，预计2026年上半年净利润将达到36.5亿元至46亿元，与去年同期亏损5.31亿元形成鲜明对比。报告期内，全球新能源产业的快速发展推动了下游客户对锂盐需求的增长，公司锂盐产品的销售价格同比显著上涨。同时，随着锂资源项目的产能逐步释放，公司的成本结构得到有效优化。加之储能市场需求的持续增长以及锂电池板块产销量的显著提升，这些因素共同推动了公司经营业绩的强劲增长。
腾讯云宣布，将于2026年7月27日对其数据万象MetaInsight智能检索服务的相关计费项进行命名规范调整及价格优化。具体而言，标量检索价格将从2元/千次下调至0.04元/千次，混合检索从5元/千次降至0.08元/千次，而标量存储费用则由0.06元/千个/月调整为0.04元/千个/月等，旨在提升服务性价比。
国际钻石产业巨头戴比尔斯集团近日宣布，计划暂停其位于南非林波波省的维尼蒂娅钻石矿的生产两年，以期进一步降低运营成本。戴比尔斯在声明中强调，公司其他矿区的产量将维持不变，整体产量指引亦未调整。集团首席执行官阿尔·库克指出，此次战略性调整旨在提升公司的经营韧性。此次停产是戴比尔斯全球成本削减和业务重组计划的关键环节。自2024年以来，戴比尔斯已累计削减年度管理费用超过1亿美元，并已出售或关闭了部分非核心资产。维尼蒂娅钻石矿是南非产量最大的钻石矿之一，其产量约占全国年产量的40%，该矿毗邻博茨瓦纳和津巴巴韦边境，已运营逾三十载。
荣耀宣布，将于2026年7月18日在世界人工智能大会期间举办一场题为“从数字屏幕到具身智能——物理世界新范式”的分论坛。此次论坛除了此前已确认出席的世界著名科技未来学家凯文·凯利外，阿里巴巴集团副总裁、ATH事业群Token Foundry多模态交互负责人许主洪的加入，尤其引发了业界对双方未来合作前景的广泛关注。据内部渠道消息显示，荣耀与阿里巴巴即将正式公布一项重大合作，其合作方向极有可能聚焦于下一代终端操作系统 Agentic OS 的落地实践。此前公开信息表明，本次论坛将深入探讨人工智能的智能水平与生命感，并分析 Agentic OS 从“工具”向“伙伴”演进的趋势。
智谱科技近日通过港交所发布公告，此前公司披露的拟配售最多1978万股新H股已于2026年7月13日完成配售。此次配售以每股1588港元的价格，向不少于6名承配人配发及发行了合计1978万股新H股，约占公司扩大后已发行H股总数的4.25%。此次配售事项所得款项总额约为314.11亿港元，所得款项净额约为313.75亿港元。
桌面级CNC（数控机床）解决方案提供商奇塑科技近期宣布完成近亿元人民币天使轮融资。本轮融资获得了商汤国香、首形科技、全新世以及奇绩创坛的联合投资。据透露，所募资金将主要用于桌面级CNC软硬件产品的深度研发、规模化量产以及市场营销推广活动。
新奇机器人公司近日完成A+轮融资。这是自2025年底启动首轮市场化融资以来，新奇机器人在半年内获得的第三轮融资。完成本轮融资后，新奇机器人的投资方阵容进一步扩大，包括经纬创投、沃赋创投、云晖资本、厚雪资本、Synstellation Capital等市场化投资机构，智元机器人、福然德等产业投资方，以及建发新兴投资、吴中金控、雉荟投资、宁波天使等国有背景资本。
空间智能领域企业大衍科技近期宣布完成数千万元天使轮融资。本轮融资由松禾资本领投，并吸引了浙江省金控和广州番禺创新基金等国有背景机构的参与。据悉，募集资金将主要投入于触觉大模型的研发、机器人数据产线的构建以及团队的进一步扩张。
小米公司于7月14日公布了其人形机器人在汽车工厂“实习”的最新进展。经过长达四个月的持续迭代优化，小米机器人在自攻螺母上件工站的双侧作业成功率已从90.2%显著提升至98%，与人工操作的合格率仅相差一个百分点。此外，该机器人还成功拓展了两项全新作业任务：中控台侧盖板排序以及料箱折叠回收，这两项任务的成功率均已达到90%。值得关注的是，小米此次演示的中控台侧盖板排序工站场景，标志着其机器人在汽车工厂首次实现了对柔性工件的长时间连续作业。
哔哩哔哩（简称“B站”）于7月14日正式推出了“build in bilibili · AI创造公开赛”的作品投币入围榜。数据显示，大赛启动仅一个月，投稿创作者数量已逼近5000人，站内“vibe coding”等相关关键词的日均搜索量激增六倍。值得注意的是，在当前投币榜前十名中，约半数是粉丝量不足一万且缺乏专业开发经验的UP主，另有两到三个席位被创业团队所占据。
韩国高丽大学医学院团队成功研发出一款新型脑部植入物。该设备能够通过温度调控机制，实现对神经活动的双向调节，这为下一代脑机接口技术和神经系统疾病的治疗方案开辟了全新的可能性。相关研究成果已刊载于最新一期的《科学进展》杂志。
在近期13日收盘后，市场中流传着一则关于“A股量化策略已将韩国科技股走势纳入因子模型，并由此引发A股不同科技板块连锁反应”的传闻，在业内引起广泛关注。针对此，一位管理规模逾200亿元的量化私募创始人明确指出：“从行业实践来看，国内顶尖量化机构极少会采用韩国市场数据来构建交易因子，此类传闻实为‘外行论调’。实际上，近期量化行业普遍遭遇亏损，据我盘后了解，多数同行的超额收益均为负值。”谈及当前市场回调的原因，这位私募人士认为，市场遵循“涨跌循环”的自然规律，不应过多解读为“阴谋论”。另一位百亿级量化私募总经理回应称，上述传闻并不客观，“如果量化机构完全依赖海外科技股的波动进行交易，从长期看必然导致负超额收益，这并非主流量化机构的做法。量化行业的核心策略仍是通过甄选并持有优质企业来获取超额回报。”此外，一家以量化中性策略著称的知名量化私募创始人亦表示，“这种观点毫无依据”。他进一步分析，近几日A股市场的调整更多体现了与全球市场的联动效应，以及前期科技资产集中的拥挤度过高所致。该人士强调，目前A股主要权重指数的估值水平仍属合理区间，后续市场进一步调整空间有限。
美国加利福尼亚州州长加文·纽森（Gavin Newsom）于13日签署了SB 168号法案，此举标志着加州新一轮零排放汽车（ZEV）即时补贴计划的正式启动。依据SB 168号法案，加州政府将划拨1.355亿美元的财政资金，并强制要求参与该计划的汽车制造商进行等额配套投资，从而共同设立一个总额约为2.7亿美元的“首次购买电动汽车”补贴基金。该法案明确规定，首次购买电动汽车的加州家庭在购车时即可直接享受“即时返现”优惠，无需再等待后续的退税流程。
