# 沙迦青少年毒瘾问题引关注：330家庭求助免罚

🚨 沙迦青少年毒瘾问题引关注：330家庭求助免罚
过去三年间，阿联酋沙迦警方禁毒部门透露，已有近330个家庭主动向警方报告了子女的毒瘾问题，其中最小的受害者仅14岁。这一数据由禁毒预防与控制部门负责人马吉德·阿勒·阿萨姆对外公布。
报告数量显著增长，主要得益于警方发起的一项鼓励家庭寻求帮助的行动。阿勒·阿萨姆指出，父母态度的转变，源于他们确信子女将获得康复治疗而非法律惩罚。根据阿联酋法律，若成瘾者本人或其家庭成员主动寻求帮助，可完全免于刑事起诉，转而通过康复中心接受治疗。
这一政策转变有效消除了家庭的顾虑，促使更多父母为子女的福祉挺身而出。
