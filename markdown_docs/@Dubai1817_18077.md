# 马来西亚惊人劫案：两名现役警员涉案被捕

🚨 马来西亚惊人劫案：两名现役警员涉案被捕
马来西亚吉打州近日破获一起冒充警察实施的抢劫案，警方一举逮捕了五名嫌疑人。令人震惊的是，其中两名落网者竟是现役警员，这一事件严重冲击了公众对执法部门的信任。
据警方调查，这五名嫌犯身穿印有马来西亚皇家警察（PDRM）标志的反光背心，伪装成执法人员闯入一名外籍人士住所，以“搜查”为名抢走了1750令吉现金。
作案后，嫌犯一行人的异常举动引起了当地志愿巡逻队的注意并被拦下。随后赶到的警方展开调查，最终揭露了其中两名涉案人员的现役警员身份。行动中，警方查获了包括PDRM反光背心、警靴、手铐、多部手机及现金等大量涉案证物。
目前，所有嫌疑人均已归案，案件仍在深入调查中。警方强调，执法身份绝非违法犯罪的“保护伞”，任何触犯法律者都将依法受到制裁。
