# 新加坡重磅推出3700万美元计划，赋能媒体专业人士加速数字内容创新与能力建设

新加坡资讯通信媒体发展局（IMDA）于本周四正式启动了一项总额高达4800万新元（约合3700万美元）的专项计划，旨在全面支持媒体专业人士进行数字内容创作。
IMDA在官方声明中指出，这项名为“数字内容与能力发展计划”（Digital Content and Capability Development, 简称DCCD）将为媒体专业人士提供全方位支持，以推动数字内容产出。
该计划为期四年，旨在帮助新加坡的媒体专业人士及相关企业在日益由数字技术主导的媒体格局中保持并提升竞争力。
与此同时，媒体专业人士将有机会学习如何将人工智能（AI）作为内容创作工具，以确保自身在行业发展中始终保持领先地位。
观察发现，本地观众的行为习惯正持续向多样化的数字平台、设备和内容格式转移。目前，87%的15岁及以上人群每周都会访问TikTok、Instagram、YouTube和Facebook等主流社交媒体视频网站。
短视频、微短剧、AI辅助制作流程以及AI生成内容正日益受到关注，伴随而起的是新一代数字原生内容创作者，他们正在重塑内容的生产、分发与消费模式。
IMDA强调，新加坡不乏引人入胜的故事可向世界分享。媒体专业人士可以运用技术和人工智能，辅助其叙事创作技艺，以新颖且引人入胜的方式将这些故事呈现给广大受众。
DCCD计划旨在通过两大核心支柱，协助媒体专业人士积极应对这些行业变迁。
其一，数字内容开发。此部分旨在支持内容的创作与分发，以反映当前本地观众通过社交平台、数字渠道及移动优先格式发现并连接故事的方式。这包括支持能够通过相关格式和叙事，与青年、成人及老年群体产生共鸣的“新加坡故事”。
其二，能力发展。此支柱侧重于支持数字叙事、新兴内容格式与技术以及制作方法方面的创新，其中包括AI生成内容（AIGC）以及涵盖真人表演和动画的短篇剧集内容。
该计划还将鼓励对AI辅助工作流程的实验和采纳，应用于内容开发、制作及本地化环节。此举将使媒体专业人士能够将更多时间投入到创意核心工作上，例如创意指导、故事构思和叙事打磨，以及业务拓展和提案演示。
为了参与DCCD计划，媒体公司需通过一项资质认证，该认证将评估其当前内容开发能力和过往业绩。通过认证的公司将有资格参与“提案征集”（CFPs）。
资质认证每年举行两次。首次认证已有117家公司获得批准，首轮提案征集活动将于2026年6月18日启动，并持续至2026年7月31日。
IMDA亦指出，在快速变化的媒体环境中，媒体专业人士还可以作为“国家AI影响力计划”（NAIIP）的一部分，发展其人工智能技能，以提升内容创作效率并简化制作过程中重复性环节。该计划旨在未来三年内，培训各行业10万名非技术专业人士掌握AI“双语”能力。
DCCD计划也建立在IMDA总额2亿新元（约合1.55亿美元）的“人才加速计划”（TAP）基础之上。TAP计划于2025年12月启动，旨在全面提升新加坡媒体人才（特别是创意制片人、编剧和导演）在电影和电视领域的专业能力，从而持续产出高质量、全球市场认可的“新加坡制造”故事。
IMDA重申，致力于支持新加坡的媒体生态系统发展新型数字内容格式，并构建数字能力，以保持创新和竞争力。声明指出，这些共同的努力也将有助于创造更多机会，更好地连接新加坡各平台观众，同时部分精选的故事和内容形式有望走向国际，超越新加坡本土市场。
