# 西港警方发布电诈特征，呼吁酒店协助排查

🔍 西港警方发布电诈特征，呼吁酒店协助排查
柬埔寨西哈努克省警方近日向当地酒店和民宿业者发布重要安全提示，旨在加强对电信诈骗嫌疑人员的识别与排查。警方呼吁各住宿场所积极配合，共同防范潜在风险。
警方指出，电诈团伙成员的作息与普通游客显著不同，存在一系列异常行为模式。例如，他们常拒绝客房清洁服务，少量住客却持有大量手机或电脑等电子设备，房间可能整夜灯火通明并持续传出深夜交谈声。此外，这类人员极少外出，高度依赖外卖服务输送日常物资，且房间常有陌生人频繁进出，疑似进行人员轮班。房间门外的垃圾桶内若发现大量功能性饮料罐及零食包装，或未经允许私自增加桌椅，均是诈骗活动在房间内进行的明显迹象。
警方强调，酒店前台、客房保洁及安保人员需严格执行入住登记制度，一旦发现上述异常情况或任何可疑人员，应立即上报或联系当地警方进行处置。
