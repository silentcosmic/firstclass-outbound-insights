# 泰女海外习诈术，回国操控亿元骗局被捕

⚡ 泰女海外习诈术，回国操控亿元骗局被捕
近日，泰国中央调查局成功侦破一起涉及金额逾1亿泰铢的大型网络投资诈骗案，逮捕一名46岁泰国女子。该嫌犯面临共同诈骗、输入虚假计算机数据、洗钱及参与有组织犯罪等多项指控。
调查显示，该犯罪团伙长期以“高回报投资”为诱饵，通过网络平台发布虚假信息，诱导受害者持续转账。被骗资金随后流入犯罪集团控制的法人银行账户，其中涉及50多家充当资金通道的壳公司及有限合伙企业。
警方进一步查明，这名被捕女子曾赴邻国诈骗园区工作，期间参与电信诈骗及网络赌博集团运作。返回泰国后，她与昔日同伙重操旧业，专门负责成立代持公司、开设法人银行账户，为各类诈骗及网络赌博组织提供资金流转渠道。她还以董事或合伙人身份控制4家企业，不仅协助开设多个法人账户，还负责招募他人成立空壳公司，从中获取分红或佣金。此外，她也涉嫌协助犯罪集团办理大量“人头SIM卡”，用于注册诈骗账号、拨打诈骗电话及隐藏身份。
嫌犯在初步审讯中已对相关犯罪事实供认不讳。目前案件已移交曼谷空丹警察局继续深入调查，警方将持续追查幕后主谋及其他涉案人员，以彻底瓦解整个犯罪网络。
