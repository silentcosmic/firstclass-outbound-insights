# ️ Gate.io承认人脸识别系统被攻破，不赔偿用户损失

⚠️ Gate.io承认人脸识别系统被攻破，不赔偿用户损失
近日，加密货币交易所Gate.io（芝麻开门）公开承认其人脸识别系统遭到攻破。此消息引发了市场对其安全协议的广泛关注。
据了解，Gate.io方面表示，仅就此前在处理此事件时沟通态度生硬致歉，但明确指出不承担因此次安全事件造成的用户资金损失。尽管交易所承诺将配合用户进行资产找回，但拒绝提供任何赔偿。
