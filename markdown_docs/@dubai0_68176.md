# ️ 迪拜新地标招标推迟：成本与供应链承压

⚠️ 迪拜新地标招标推迟：成本与供应链承压
全球地产巨头伊玛尔地产（Emaar Properties）宣布，其备受瞩目的迪拜云溪塔项目（Dubai Creek Tower）招标流程将延期约三至四个月。该项目旨在超越哈利法塔，成为新的世界最高建筑。
伊玛尔地产创始人Mohamed Alabbar解释称，此次延期主要由于成本和供应链问题。受当地港口关闭影响，建筑材料价格上涨，导致项目整体定价面临调整。公司将利用这段时间重新评估成本结构，以应对当前市场挑战。
