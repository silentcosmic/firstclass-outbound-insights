# 柬埔寨波贝反诈清查：酒店查获大量涉诈设备

🚨 柬埔寨波贝反诈清查：酒店查获大量涉诈设备
柬埔寨波贝市近日展开大规模反诈清查行动。国家警察总署联合班迭棉吉省多部门，对当地3处重点场所进行突击检查，并在一处酒店内查获大量涉嫌用于网络诈骗的电脑及手机设备。
此次行动由国家警察副总监芒·索提亚中将指挥，重点排查了3处住宅及酒店。尽管其中两处住宅未发现明确诈骗证据，但现场发现的数十名外籍人员已移交移民部门进行身份核查。
在位于奥克罗夫分区的温特莫塔奇夫天酒店，执法人员查获1台笔记本电脑和62部手机，这些设备被怀疑用于实施网络诈骗。该酒店共有11层63个房间，居住人员来自10个不同国家，共计66人。目前，所有涉案物证及相关人员已移交国家警察总署专业部门，案件仍在深入侦办中。
