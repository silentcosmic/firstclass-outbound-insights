# 长征十号乙火箭海上回收：中国首次，全球首创网系技术

🚀 长征十号乙火箭海上回收：中国首次，全球首创网系技术
7月10日，长征十号乙运载火箭在海南商业航天发射场成功升空。其一子级在与二级分离约6分钟后，实现垂直返回并精准降落在海上回收平台，标志着中国首次成功实施运载火箭一子级可控回收。
此次任务在全球范围内创造了历史，首次完成了运载火箭的网系回收技术验证，展现了中国在航天回收领域的前沿创新实力。
