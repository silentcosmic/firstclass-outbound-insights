# 马来西亚“超级行动”：503名外籍人员被扣

🚨 马来西亚“超级行动”：503名外籍人员被扣
马来西亚移民局近日宣布，在全国范围内开展了为期两天的“超级行动”（Op Mega），旨在打击由外籍人员经营或大量雇用外籍劳工的非法商业活动。此次行动同步突击检查了16处目标地点，共检查2260人，最终扣留了503名涉嫌违反移民法规的外籍人员。
此次联合行动从7月12日启动，共出动876名来自移民局、国民登记局、皇家警察等多个部门的执法人员。被扣留的外籍人员年龄介于21岁至52岁，主要来自孟加拉国、缅甸、印度尼西亚、尼泊尔和印度等地。他们普遍涉嫌无法出示有效证件、逾期停留或违反签证条件。
值得关注的是，移民局还向120名马来西亚公民发出传票，要求他们协助调查，以查明是否存在雇用、容留或协助非法外籍人员的行为。马来西亚移民局强调，未来将持续展开此类执法行动，严厉查处并遣返违规外籍人员，同时追究非法雇主及相关责任人的法律责任。
