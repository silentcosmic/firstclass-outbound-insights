# 亚洲在全球资本市场中地位日隆：新加坡一领先学府构筑前沿研究高地

随着亚洲在全球企业融资版图中的影响力日益增强，新加坡管理大学正积极拓展其在资本市场领域的研究深度与广度。
这所知名学府在其永顺豪法学院亚洲商法中心旗下，正式启动了“新加坡资本市场倡议”（Singapore Capital Markets Initiative, SCMI）。
这项倡议的核心在于法律与金融的交叉领域，致力于推动研究、政策对话以及人才培养，并特别聚焦于新加坡乃至整个亚洲区域。
其研究范畴将极为广阔，涵盖股权与债权资本市场、风险投资、私募股权、私募信贷、公司治理、投资者保护、金融监管、市场基础设施以及新兴技术等多个关键领域。
此次倡议的推出，正值全球资本市场经历深刻变革之际。
众多经济体的首次公开募股（IPO）活动普遍放缓，而与此同时，私募股权、风险投资和私募信贷在企业融资中的作用却日益凸显。
亚洲已然成为这一结构性转变的核心。据该学府统计，目前全球超过55%的上市公司以及逾半数的全球风险投资活动都集中在该区域。
该大学永顺豪法学院院长李佩婉教授对此表示：“‘新加坡资本市场倡议’将作为一个跨学科研究与政策参与的平台，旨在将新加坡管理大学打造成为资本市场研究领域的国际领先机构，并进一步巩固新加坡在法律与金融交叉领域的资本市场研究、思想领导力及政策影响力中心地位。”
这项重要倡议将由奥雷利奥·古雷亚-马丁内斯教授牵头，并由丹·普奇尼亚克教授及妮迪亚·雷莫利娜助理教授共同领导。
