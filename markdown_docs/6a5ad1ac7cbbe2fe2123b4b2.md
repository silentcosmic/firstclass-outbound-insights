# 人形机器人ODM方案商半醒具身斩获千万级融资，飞荣达与索辰科技接力布局具身智能赛道

据《出海 · 头等舱》最新消息，专注于人形机器人ODM解决方案的半醒具身（BXI Robotics）近日成功完成新一轮千万元级融资，本轮投资由A股上市公司索辰科技领投。这笔资金将主要投入到下一代人形机器人的研发迭代以及其海外市场的战略拓展。值得注意的是，半醒具身此前已于2025年获得首轮机构融资，当时即获得了另一家上市公司飞荣达的青睐。
创始团队与技术基石
半醒具身正式成立于2022年，然而其核心团队早在2020年便已深入人形机器人的前沿研发。公司联合创始人兼首席执行官陈彦，毕业于东南大学计算机专业，在投身机器人领域之前，积累了多年的二级市场量化交易经验，为公司的战略决策带来了独特的商业视角。另一位联合创始人刘福强，本科毕业于西南科技大学，在校期间连续四年参与全国大学生机器人大赛并屡获殊荣，展现出深厚的机器人技术底蕴。
该公司致力于高动态双足人形机器人的全栈自主研发，在国内业界堪称稀缺，是少数能够同步掌握整机设计、核心电机、高级运动控制算法及定制化场景软件交付能力的团队。当前，半醒具身主要面向品牌客户提供全面的通用人形机器人ODM服务，其解决方案覆盖从白牌整机、精密关节电机模组、前沿运动控制算法到定制化应用软件开发等多个核心环节。
务实的选择：ODM市场策略
半醒具身选择ODM路线，源于其团队对当前市场需求的深刻洞察与务实判断。公司创始人陈彦向《出海 · 头等舱》分析指出，随着人形机器人赛道的迅速升温，众多企业渴望快速推出自有品牌的人形机器人产品。然而，自主研发往往面临周期漫长（通常需要2至3年）、高难度团队组建以及结果不确定性等诸多现实挑战。这些痛点叠加，共同催生了对专业ODM服务的强大需求。
半醒具身的核心竞争力在于其独特的“全栈自主研发”能力。与市场上众多依赖方案集成的供应商不同，半醒具身从最底层的电机技术开始深耕，逐步构建并完善了包括电控系统、精密结构设计、先进运动控制算法在内的全链条技术实力。
以其关键技术——关节电机为例，团队在2020年项目启动之初便发现，市场上尚无法获得能够满足人形机器人高难度动作，例如后空翻所需的大扭矩电机与高频率电控系统。面对这一技术空白，半醒具身毅然从关节电机模组着手自主研发，先后尝试了摆线与行星齿轮方案，最终成功开发出峰值扭矩高达800牛米的行星关节电机。为进一步提升性能，团队还创新性地引入了电机内水冷方案，并配合自主研发的1000Hz电控系统，成功打造出首代样机。经过持续的优化迭代，目前公司已形成BXI-50/70/85三款标准化行星中空关节电机模组，全面覆盖了人形机器人全身31个自由度的控制需求。
赛场验证与产能布局
《出海 · 头等舱》获悉，半醒具身是业界极少数曾成功参与并完成机器人马拉松挑战的ODM方案商之一，这充分验证了其技术的成熟度与可靠性。在2025年4月举行的首届北京人形机器人半程马拉松中，其“精灵2”机器人以未经改装的量产版本参赛，以4小时2分19秒的成绩完赛，位列第六。次年，即2026年4月，“精灵3”机器人再度出征，并以150分钟的净时间完成比赛，排名显著靠前。
在产能布局方面，半醒具身依托其战略股东飞荣达位于深圳和常州的生产基地支持，目前已具备每月数十台的生产能力，并有望在短期内迅速扩充至每月百台以上。公司展望，在2026年7月至2027年7月期间，单月订单金额有望突破千万元大关，全年营收预计将超过一亿元人民币。
市场版图与全球视野
半醒具身透露，其客户群体广泛而多元，不仅包括顶尖的科技互联网企业、国内多家主流汽车制造商及知名机器人公司，还涵盖了主机厂、二次开发商、具身智能“大脑”的研发机构，以及纺织、光伏清洁、无人洗车等多个垂直应用场景的先行者。
海外市场同样是半醒具身战略布局的重中之重。目前，公司已成功服务一家美国领先的科技企业，并正积极拓展韩国、日本、欧洲等关键市场。陈彦表示：“在全球范围内，没有任何一个国家希望在人形机器人技术上完全依赖他国。因此，每个本地市场都蕴藏着巨大的ODM服务需求。”
创始人洞察：对行业与模式的深度思考
半醒具身联合创始人陈彦、刘福强表示：“我们真正做到了提供从底层电机、电控系统、传感器、算力定制，直至上层运动算法和操作算法的全栈式解决方案，并且这些方案都经过了严苛的量产实践与赛场检验。客户选择与我们合作，便无需再寻找其他供应商，项目失败的风险也因此大大降低。我们就像一个‘六边形’战士，任何一环的缺失都无法成就完整的系统。表面上看市场竞争者众多，但客户真正能信赖的选择其实并不多。”
“我们看到了ODM模式在人形机器人领域蕴含的巨大机遇，现在要做的就是深耕这片‘金矿’，将其潜力充分挖掘。”陈彦与刘福强进一步指出，“目前人形机器人的终端市场主要集中在科研机构和实验室，其购买主体属于‘资本开支’，而非普通消费者。大量企业渴望快速构建自有品牌的人形机器人产品线，但自主研发的周期漫长且风险高昂。我们团队历经五年，从零开始将技术成熟度提升到及格线以上，客户与我们合作，本质上是在‘购买时间’，以更高效的方式进入市场。”
对于行业发展，两位创始人分享道：“我们常犯的错误是高估短期趋势，却低估了长期的发展潜力。2020年我们刚起步时，最初的愿景仅仅是让机器人能够自主行走。而今，行业已成功攻克‘自由移动’的技术难关，接下来，我们将聚焦于解决‘自由执行任务’这一核心挑战。”
他们判断：“到2027年，人形机器人将真正融入日常生活——届时我们不再是隔着屏幕观看，而是能在身边真切地感受到它们的存在。这个行业的发展并非线性递增，而是呈现出阶梯式跃进的特征，一旦跨越某个关键临界点，便会迎来集中式的爆发增长。”
战略投资方视角：索辰科技的产业协同愿景
索辰科技（688507）董秘吴味子对本次战略投资表达了高度认可：“索辰科技对半醒具身的底层硬件创新能力、硬核研发团队以及其标准化量产的商业模式高度看好。半醒具身自主研发的机器人核心关节及开放式整机底座，实现了高精度动态运动调控，其技术不仅稀缺，且具备极强的落地性。同时，其轻量化ODM交付模式大幅降低了具身智能产业化的进入门槛，与索辰的物理AI仿真体系实现了高度协同，成为连接仿真模拟与实体机器人落地的关键环节。”
吴味子进一步阐释道：“此次战略入股将有力推动双方在技术与产品层面的深度融合，我们将围绕硬件适配、数据迭代以及运动控制重构，共同完善具身智能的完整技术链条。双方的目标是打破传统机器人封闭的硬件壁垒，共同构建标准化、开放且轻量化的硬件底座，为物理AI算法的快速迭代提供专属载体。同时，通过建立全域数据闭环，实现真机运行数据实时回流至索辰的物理AI平台，从而形成‘仿真训练—真机落地—数据回传—模型优化—迭代重训’的良性循环，持续提升物理模型的仿真精度。展望未来，我们将依托营造·万象大模型重构机器人的运动控制逻辑，实现高动态自主运动调控，让硬件真正成为物理AI的实体执行终端，完成从物理仿真到实体具身智能落地的关键突破，最终实现具备高阶作业能力的人形机器人的规模化部署。”
