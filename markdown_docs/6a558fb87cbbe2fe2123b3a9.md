# 欧佩克基金5000万美元贷款注入越南SeABank，赋能中小企业与气候投资

欧佩克国际发展基金（OPEC Fund for International Development）近日宣布，已向越南东南亚商业股份银行（SeABank）提供一笔5000万美元的贷款。此举旨在全面扩大对越南境内中小企业（MSMEs）及气候相关投资的融资支持。
SeABank周一发布声明指出，这笔贷款将显著增强其对微型、小型和中型企业的信贷能力。这些企业在越南贡献了约一半的就业岗位，但长期面临显著的融资缺口。此外，部分资金还将专项用于支持能源效率提升、绿色建筑、屋顶光伏以及电动出行等关键气候友好型投资。
欧佩克基金总裁阿卜杜勒哈米德·阿尔哈利法（Abdulhamid Alkhalifa）表示：“中小企业是越南经济的核心驱动力，也是就业和增长的重要来源。此次融资将帮助更多企业获得发展所需的资本，进行投资和业务扩张，同时支持实用的气候解决方案，以提升整体竞争力和韧性。”
SeABank董事会副主席黎秋水（Le Thu Thuy）强调，这笔融资标志着该行与欧佩克基金合作关系中的一个重要里程碑。它不仅将进一步巩固SeABank对普惠性和可持续增长的承诺，也凸显了其在动员国际资本、支持越南可持续发展重点方面的关键作用。
此次贷款是SeABank与欧佩克基金深化合作的最新成果。双方的合作始于2022年，当时欧佩克基金首次向SeABank提供贷款，主要用于支持当地中小企业（包括女性企业）以及气候项目。
