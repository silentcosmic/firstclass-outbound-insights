# 洪都撤诉半岛电视台：柬埔寨权贵止损止战

🚨 洪都撤诉半岛电视台：柬埔寨权贵止损止战
近日，柬埔寨前总理洪森的侄子洪都在澳大利亚主动撤回了针对半岛电视台的诽谤诉讼。
这起诉讼源于2022年半岛电视台发布的一部调查纪录片，内容深入揭露了柬埔寨境内的诈骗园区、人口贩运及跨境犯罪网络。片中提及洪都与部分商业实体存在关联，并引用了相关执法调查信息，随后洪都向澳大利亚法院提起诉讼。
今年3月，澳大利亚上诉法院推翻了一项限制性裁定，允许半岛电视台在诉讼中推进其“真实性抗辩”。这意味着，半岛电视台被授权调取更多材料，包括与澳大利亚联邦警方（AFP）相关的文件，以支持其在纪录片中的部分陈述。这一裁决使案件焦点从“是否构成诽谤”转向“报道内容是否具备事实依据”，对洪都而言极为不利。
市场观察人士普遍认为，此次撤诉是洪都方面在面临深入调查可能揭露更多细节的压力下，采取的止损策略。半岛电视台以其深入的调查报道和强大的资源背景闻名，与这类媒体机构持续对抗，往往意味着可能面临更多潜在的负面曝光。
