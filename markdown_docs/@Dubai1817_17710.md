# 印度监管风暴：Telegram、Signal反诈措施遭问询

🔍 印度监管风暴：Telegram、Signal反诈措施遭问询
7月3日，据外媒报道，印度电子和信息技术部（MeitY）已正式致函Telegram和Signal，要求这两大即时通讯平台详细说明其“用户名功能”所采取的反诈骗安全措施。
截至目前，印度电子和信息技术部、Telegram及Signal三方均未就此通知作出公开回应。此举凸显了印度政府在数字安全和用户保护方面日益强硬的监管立场，尤其是在平台用户身份识别及潜在滥用风险上。
