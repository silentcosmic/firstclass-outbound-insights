# 波贝电诈窝点被查：9名中国籍男子落网

🚨 波贝电诈窝点被查：9名中国籍男子落网
近日，柬埔寨卜迭棉芷省联合工作组对波贝市三处地点展开突击检查，成功捣毁一处跨境电信诈骗窝点。
在其中一处伪装成商贸公司的据点，警方当场抓获9名涉嫌搬运设备的中国籍男子及4名柬籍人员。经查，该窝点主要针对中国和新加坡进行网络诈骗活动，现场查扣了涉案车辆及大量设备，嫌疑人已被移交边防警察营处理。此外，在另一处地点，警方查获一辆藏有电诈设备的可疑车辆。而在第三处检查点，25名印尼籍人员因证件齐全且确认为合法赌场员工，已获准暂时留宿等待回国。
