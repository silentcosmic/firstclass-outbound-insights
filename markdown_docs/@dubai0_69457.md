# ️ 迪拜传奇裁缝谢赫·马达尼逝世，匠心铸就城市记忆

🕯️ 迪拜传奇裁缝谢赫·马达尼逝世，匠心铸就城市记忆
迪拜首位裁缝、被誉为“裁缝酋长”的谢赫·阿卜杜勒拉赫曼·阿尔·沙菲·阿尔·马达尼于7月7日安详离世，享年97岁。他用近八十年的时间，一针一线地缝制出迪拜的故事，其逝世引发各界广泛悼念，人们称颂他为迪拜最早的奠基者之一。
阿尔·马达尼的创业始于迪拜早期，彼时德拉一带仍是荒芜。他最初开设小店，为采珠人缝制潜水服，随后其精湛技艺与阿联酋传统服饰坎杜拉紧密相连。从1947年在阿尔·拉斯开设“国家裁缝店”起，他不仅将裁缝手艺发展成遍布阿联酋的家族企业集团，更在六十年代涉足地产，七十年代创立建筑公司，并在1992年开创了全国首家入驻购物中心的裁缝店，其商业远见与坚韧不拔为迪拜的早期发展贡献卓著。
即便年逾九旬，阿尔·马达尼仍对缝纫充满热情，每日前往店中。他的孙子、电影制作人阿卜杜勒拉赫曼·阿尔·马达尼回忆，祖父将一门看似平凡的手艺，升华成一份伟大的传承，其白手起家的奋斗故事至今仍是后人的不竭灵感。他所代表的稳重、尊严、坚韧与温和，已成为老社区记忆中不可磨灭的一部分。
