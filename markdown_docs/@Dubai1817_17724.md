# 缅甸交克园区疑涉虐待，求助曝光

🚨 缅甸交克园区疑涉虐待，求助曝光
近日有市场消息指出，缅甸交克（Kawkareik）地区一家位于山上的非法运营园区，距离城镇仅半小时车程，被指控从事“国盘军包”业务，即涉嫌线上博彩或诈骗，并疑有当地武装势力背景。
爆料人称，该园区存在严重的员工虐待行为，有被困人员遭受非人折磨，甚至被剥夺基本生活用品。消息指，该园区负责人为“二老板宝哥”，管理人员包括阿成、老四（均来自贵阳），后勤人员小飞、螃蟹，以及代理小寇、李响（贵阳人）。
事件源于爆料人曾试图营救两名中国籍员工小勇和阿鑫，但在转账后遭遇推诿，未能成功。爆料人强烈呼吁中国警方和缅甸政府军介入调查，并要求“二老板宝哥”出面解决此事，否则将持续曝光涉事人员的护照及证件信息，并警告公众切勿轻信此类“兄弟”间的承诺。
