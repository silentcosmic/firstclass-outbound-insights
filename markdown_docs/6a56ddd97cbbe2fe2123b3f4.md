# 赋能实时互动世界与游戏引擎：AI视频生成平台PixVerse获4.39亿美元C轮追加投资

作为一家全球性的AI视频生成平台，PixVerse凭借其在177个国家积累的逾1.5亿用户基础，近期完成了C轮追加融资。此轮融资获得了国际媒体、娱乐与科技领域投资者的广泛青睐，使得PixVerse的C轮融资总额达到了4.39亿美元。
PixVerse方面近日发布声明指出，本轮追加融资的新进投资者包括阿里巴巴、Lollapalooza Capital、Ivy Capital、Grand Mount Capital、Eastern Bell Capital、Mirae Asset、蓝色光标和CloudAlpha。同时，现有投资者iGlobe Partners和华侨银行旗下的LionX Ventures也继续加持。
PixVerse最新研发成果之一是于2026年1月推出的R1模型，其被誉为全球首个实时世界模型。该产品能够将视频生成模式从传统的固定输出，革新为可即时响应用户输入的连续互动流。R1模型推出三个月后，PixVerse进一步对其进行了升级，增加了共享世界和个性化形象功能，从而实现了多用户实时进入并共同塑造同一个AI生成世界的能力。
目前，PixVerse正积极将R1模型的应用范围拓展至互动游戏和娱乐领域。其全新的PixVerse游戏引擎将R1技术融入游戏创作之中，允许玩家通过自然语言进行互动，而游戏世界则能实时在视觉和机制上作出响应。PixVerse联合创始人兼首席执行官Changhu Wang表示，公司正在构建一个系统，玩家所身处的世界并非预先渲染，而是根据他们的行为实时、连续生成。这为游戏的未来形态奠定了截然不同的基础。PixVerse联合创始人兼总裁Jaden Xie指出，当初驱动PixVerse致力于让专业视频创作普惠大众的信念，如今正驱动着公司向互动世界领域的一切构建。
