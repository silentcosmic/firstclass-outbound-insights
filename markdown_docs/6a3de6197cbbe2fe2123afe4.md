# 权威内容生成挑战：维基百科联合创始人警示AI“幻觉”风险，审慎评估直接编辑能力

维基百科联合创始人吉米·威尔士近日明确指出，鉴于当前人工智能（AI）技术的局限性，维基百科尚未对其建立足够的信任，以允许AI直接参与平台上的词条编辑工作。
威尔士进一步阐释，尽管新一代AI模型在一定程度上已缓解了其“幻觉”问题——即AI系统看似确凿实则捏造信息的情况——然而，他强调这一挑战目前仍“异常严峻”。
与此同时，观察到各个人工智能平台普遍依赖维基百科内容以响应用户查询。这种趋势直接导致了来自AI机器人的网站访问量整体攀升，而与此同时，人类用户的直接访问量则下降了8%。
作为维基百科运营机构——维基媒体基金会的董事会成员，威尔士对这一现象进行了评估。他认为，尽管对于这个稳居全球访问量前十的在线百科全书而言，人类用户流量的下降“值得审视”，但这“并非一场灾难”。
这位联合创始人还透露，维基百科在与多家科技巨头达成合作协议方面已取得“显著进展”。他表示：“我们正在着手屏蔽那些未能遵守规范的平台，但其后续成效尚需时间观察。”
