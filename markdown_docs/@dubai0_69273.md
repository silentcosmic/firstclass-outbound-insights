# ️ 迪拜警方紧急警告：警惕假冒“消保”网站窃取银行信息

⚠️ 迪拜警方紧急警告：警惕假冒“消保”网站窃取银行信息
迪拜警方近日发出紧急警告，提醒在阿联酋的居民和企业高度警惕假冒的“消费者保护”网站，这类欺诈网站正被不法分子用来窃取银行账户信息及资金。
据警方披露，诈骗分子通过设置虚假的消费者投诉平台，诱导用户在网上搜索并提交投诉。一旦用户点击进入假网站并填写个人信息（如姓名、邮箱、电话和投诉详情），不久后便会接到自称“消费者保护部门工作人员”的电话。这些骗子会准确复述用户提交的投诉内容，以此获取受害者的信任。
随后，诈骗者会诱导受害者下载远程控制软件，并要求其登录网上银行。一旦受害者依指示操作，骗子便能远程监控手机或电脑屏幕，直接窃取银行账号、密码，并迅速转移账户内的资金。当局再次强调，公众应提高警惕，在处理涉及个人敏感信息和金融操作时务必核实官方渠道，谨防此类新型网络诈骗。
