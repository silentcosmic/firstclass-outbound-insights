# 斯里兰卡突查外国商铺：中国商品包装引监管警示

🚨 斯里兰卡突查外国商铺：中国商品包装引监管警示
近期，斯里兰卡科伦坡的外国商铺面临日益严格的市场监管。据当地一名中国粮油副食品杂货店铺经营者透露，自七月初起，斯里兰卡政府监管部门对外国人集中的综合市场进行了突击检查，重点涵盖食品安全与经营合规性。
该经营者反映，其店铺在检查中因部分商品外包装仅为纯中文标识，被监管人员指出不符合当地规定。尽管首次仅获口头警告，但被明确告知若不立即整改，后续将面临商品没收、罚款，甚至营业执照被吊销的风险。据悉，同市场内已有其他外国商铺因此被处以数万卢比的罚款，预示着当地营商环境趋向收紧。
此次监管收紧的背景，是科伦坡外国人口及消费的显著增长，带动了外国商铺业务的蓬勃发展。上半年，该中国店铺的业务量相比去年有明显提升，新客户不断增加。伴随市场活力的提升，政府对商家安全规范经营的要求也随之升级，出海企业需密切关注并提前做好合规准备。
