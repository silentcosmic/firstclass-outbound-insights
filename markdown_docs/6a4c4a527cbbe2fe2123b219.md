# dtcpay携手TP，构建亚太区全天候客户服务新体系

随着新加坡支付巨头dtcpay在亚太地区拓展其稳定币赋能的支付服务，TP将为其提供全天候的客户支持。这标志着dtcpay在区域扩张战略中的一项关键举措。
此次合作涵盖了针对客户和商家的支持服务，支持语种包括英语和普通话，并通过数字渠道及语音热线全面提供。
dtcpay是新加坡一家受监管的数字支付服务提供商。公司一直在积极拓展其面向企业和商家的支付服务，其中包含创新的稳定币支付解决方案。
TP将凭借其卓越的客户体验能力和内部AI驱动工具，全面负责dtcpay的客户互动运营。
这一举措将助力dtcpay有效管理日益增长的服务量，优化工作流程，并在其业务扩张过程中提供更加稳定和高质量的支持。“亚太地区的支付格局正经历快速演变，稳定币作为企业跨境价值转移的更快、更高效、更透明方式，正展现出强劲的发展势头。随着稳定币的广泛应用，提供值得信赖、响应迅速且高质量的客户体验，对于在用户和商家中建立信心而言，显得至关重要。”“亚太区数字支付的普及速度正在加快，领军企业通常都具备强大的技术基础设施，并辅以同样出色的客户体验运营能力。dtcpay已经构建了一个极具吸引力且受严格监管的支付平台，我们很荣幸能提供与之雄心相匹配的客户体验能力和规模化支持。”
未来，双方的合作有望进一步深化，涵盖更广泛的区域支持、多语言互动以及更多运营规模化方案的探索。
