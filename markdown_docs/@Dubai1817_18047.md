# 越南巨额赌球案告破：流水超3.5万亿盾

🚨 越南巨额赌球案告破：流水超3.5万亿盾
胡志明市警方日前披露了两起特大网络足球赌博案的最新进展，目前已累计起诉97名涉案人员，并传唤另150名相关人员接受调查。此案涉案金额巨大，影响力波及越南多个省市。
警方调查显示，黄黎清龙、阮越保龙等犯罪嫌疑人长期利用网络赌球账号组织他人进行足球比赛下注。专案组自2026年6月初成立，经过近一个月的证据收集，于6月29日联合多地执法部门展开收网行动。
据多名团伙成员交代，该组织从柬埔寨方面获取Master级总控账号，随后层层拆分并分发给各级代理及赌客，主要通过Bong88赌博网站进行投注。其代理网络遍布胡志明市及越南多地，通过抽佣获利。
警方初步查明，自2025年10月至案发，这两个赌球团伙的交易流水已累计超过3.5万亿越南盾。该团伙组织严密，分工明确，活动范围广泛。
