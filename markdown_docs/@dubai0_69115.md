# ️ 泰国“担保入境”骗局曝光：中介收费飙升近10倍

⚠️ 泰国“担保入境”骗局曝光：中介收费飙升近10倍
泰国移民系统审查日趋严格之际，泰国当局正对一个涉嫌向外国人收取高达1.2万泰铢“担保入境费”的中介网络展开调查。这些不法中介利用虚假宣传，声称拥有“内部渠道”可保障旅客无需接受移民官询问或检查即可顺利入境，收费较以往上涨了近十倍。
中国驻泰国大使馆已发布紧急提醒，警告赴泰中国公民警惕此类“付费担保入境”骗局。大使馆强调，泰国不存在任何合法的此类服务，并指出只要持有合法有效证件、旅行目的明确并配合移民部门依法检查，正常旅客入境通常不会遇到问题。
此次执法行动与泰国自2025年起加强边境和移民审查的政策紧密相关，旨在有效防止跨境犯罪集团将活动转移至泰国境内。数据显示，今年以来已有超过1.3万名外国人被拒绝入境，重点针对存在可疑签证记录、涉嫌与跨境犯罪有关联，或存在被诱骗前往其他国家风险的旅客。
