# 特朗普加密收入破10亿：政策与利益冲突引关注

💰 特朗普加密收入破10亿：政策与利益冲突引关注
根据最新披露的财务文件，美国前总统特朗普预计在2025年从加密货币相关业务中获得超过10亿美元的收入。这笔巨额收入主要源于World Liberty Financial项目的代币销售、钱包分配及股权转让，贡献逾5亿美元；此外，特朗普品牌Meme币的授权版税也带来了超过6.35亿美元的收益。文件同时显示，特朗普还持有价值逾5000万美元的World Liberty治理代币，以及包括比特币在内的其他加密资产。
值得注意的是，此番财务披露正值其政府大力推动加密行业发展之际。白宫已着手设立战略比特币储备、召开加密峰会、放宽对加密企业的执法力度，并积极推动如《GENIUS法案》等相关立法。尽管外界对总统及其家人可能存在的利益冲突提出质疑，白宫发言人坚称，相关政策旨在促进创新与经济发展，否认存在不当利益关联。
