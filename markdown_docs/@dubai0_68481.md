# 苹果阿联酋大涨价：iPhone 18 Pro或创历史新高

🔥 苹果阿联酋大涨价：iPhone 18 Pro或创历史新高
苹果公司近期在阿联酋市场对其多款产品进行了价格上调，这一调整被视为即将发布的iPhone 18 Pro系列售价可能大幅上涨的明确信号。
此次涨价幅度显著，例如13英寸M5 MacBook Air从4599迪拉姆升至5499迪拉姆，15英寸MacBook Air从5499迪拉姆涨至6299迪拉姆，而14英寸MacBook Pro的涨幅最为惊人，从约7199迪拉姆飙升至8499迪拉姆，部分产品涨价超过1000迪拉姆。
对于此次全球性的价格调整，苹果CEO库克曾公开表示，内存和存储芯片成本高企以及供应链紧张是主要原因。公司已在全球范围内对MacBook和iPad等产品实施了价格上调。
市场分析师预测，考虑到为支持先进AI功能所需的高内存配置，iPhone 18 Pro型号此次可能迎来显著涨价。在美国，Pro版起售价或从目前的1200多美元升至1249-1299美元，Pro Max则可能达到1349-1399美元。折算至阿联酋市场，iPhone 18 Pro预计起售价在5400-5500迪拉姆区间，Pro Max则可能达到5800-5900迪拉姆。
若预测属实，这将使其成为阿联酋历史上售价最高的iPhone系列。
