# 阿提哈德铁路乘客守则公布：行为规范与违规后果明确

📣 阿提哈德铁路乘客守则公布：行为规范与违规后果明确
阿提哈德铁路近日正式发布了乘客行为守则，旨在确保所有搭乘其列车的乘客享有安全、舒适且有序的出行体验。此举明确了旅客在乘坐期间所需遵守的各项规定。
守则对乘客的着装、行为及禁烟区域做出了明确规定。所有乘客必须穿着得体，避免过于暴露或带有冒犯性图案的服饰。在列车和车站内，严禁吸烟（包括电子烟），仅限指定区域除外。同时，守则强调文明出行，禁止辱骂、骚扰他人、插队或任何干扰公共秩序的行为。
此外，乘客严禁携带毒品、危险品或任何违法物品上车，也不得干扰列车正常运行或滥用紧急设备。为保障公共卫生，患有传染病的乘客被禁止乘车。未经许可的商业活动、乞讨及散发广告等行为同样被明令禁止。关于电子设备使用，乘客被允许使用，但必须佩戴耳机，确保不打扰其他乘客。
违反上述规定的乘客，工作人员有权要求其离车，并可能面临法律处罚及相应的赔偿责任。
