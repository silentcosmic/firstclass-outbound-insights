# ️ 境外返国人员注意：手机数据成执法焦点！

⚠️ 境外返国人员注意：手机数据成执法焦点！
近期市场消息指出，部分从境外（尤其是特定区域）返国的相关人员，在接受国内执法部门调查时，面临手机数据被深度取证的风险。这引发了对个人隐私权与法律配合义务之间平衡的广泛讨论。
据了解，执法部门在传唤过程中，对手机等数字设备的处理方式成为关注焦点。尽管刑事法规定公民有权不自证其罪并享有隐私权，但在实际操作中，执法人员会根据案件性质（如询问笔录中的证人身份或讯问笔录中的嫌疑人身份）采取不同策略。作为嫌疑人，不配合可能面临刑事拘留等后果。
值得注意的是，技术分析表明，执法部门的数字取证能力不容小觑。即使手机记录被删除、设备被格式化，甚至更换新手机，物证科仍能通过专业技术手段，迅速恢复并导出包括境外消费记录和行动轨迹在内的关键数据。因此，有观点认为，对于极力规避数据泄露风险的人员，物理销毁手机并声称遗失，被视为一种“终极”防范策略。
