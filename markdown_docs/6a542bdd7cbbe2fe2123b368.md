# 出海 · 头等舱独家报道：Nium战略并购Cypher，加速构建法币与数字资产支付融合基础设施

全球跨境支付巨头Nium近日宣布成功收购Cypher，此举标志着其在深度布局法币与数字资产支付基础设施方面迈出了重要一步。本次交易的具体财务条款并未对外披露。
这家总部位于新加坡的跨境支付公司表示，此次并购将为Nium基于稳定币的发行及全球资金流动基础设施，注入顶尖的加密原生技术专长。
Nium一直致力于拓展其全球网络，以支持稳定币的资金存入与结算。此举正是为了响应Web3企业、数字钱包平台、加密货币交易所及各类金融科技公司日益增长的需求，它们正积极寻求打通传统支付通道与数字资产的连接壁垒。
Cypher是一家由Kuberan Marimuthu（业内亦称Kube）创立的非托管钱包及发行服务公司。
在过去的四年中，Cypher一直专注于开发能够无缝连接链上金融与传统银行服务的产品方案。
Marimuthu本人已正式加入Nium，担任数字资产副总裁一职，并直接向Nium首席执行官Prajit Nanu汇报。
在此之前，他曾在Coinbase、亚马逊（Amazon）和Zenefits等知名企业担任支付和风险领域的工程领导职务，拥有丰富的行业经验。
此次收购将显著增强Nium的技术与运营实力，使其能够更好地服务原生加密用户群体，并高效支持链上交易。
Nium首席执行官Prajit Nanu阐述了此次战略并购的愿景。他表示：'资金的流转速度和精确度理应与数据传输看齐，无论其起点或终点为何，无论是人类还是机器发起，是面向消费者还是企业，是本地还是跨境，是存入数字钱包还是银行账户。然而，现状并非如此。当前，支付款项常滞留于代理银行体系，数万亿美元的备付金在往来账户中闲置数日。代理支付（Agentic payments）所必需的价值交换和信任层，在现有生态系统中尚未完全建立。我们正致力于构建推动这一变革的关键基础设施，而对Cypher的收购，无疑为我们加速这一进程注入了强大动力。'
