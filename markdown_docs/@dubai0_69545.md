# ️ 新加坡女商旅迪拜失联：离奇信用卡消费引关注

⚠️ 新加坡女商旅迪拜失联：离奇信用卡消费引关注
近日，一名新加坡籍女性陈诗敏在阿联酋迪拜失联，引发广泛关注。据家属透露，陈诗敏于6月16日因业务出差入境迪拜，并在凌晨0时36分向家人发送了最后一条消息后便失去联系。由于失联发生在她抵达后不久，目前尚无她在当地的酒店入住记录。
事件最新进展显示，6月25日，家属发现陈诗敏的母亲信用卡在迪拜的EPPO站点发生一笔90美元的消费记录。然而，自该笔交易后，陈诗敏再无任何消息。目前，相关领事馆已介入协助，但尚未获得任何有效线索。家属正悬赏寻求知情人士提供线索。
