# 迪拜NOL卡全面升级：交通零售一卡通行！

⚡ 迪拜NOL卡全面升级：交通零售一卡通行！
迪拜的公共交通支付系统正迎来一场深远变革。迪拜道路与交通管理局（RTA）宣布，其NOL系统升级改造已完成72%，核心是将现有卡片式票务系统全面转向基于账户的数字支付体系，极大提升支付灵活性。
此次升级后，公共交通将支持二维码车票、新一代NOL卡、银行卡及各类电子钱包支付。RTA强调，新系统将紧随全球数字支付和金融科技前沿趋势，致力于打造同类系统中的领先典范，预计整个项目将于2027年第一季度末全面竣工。
值得关注的是，新一代NOL卡功能将大幅拓展。除了传统的地铁、公交、水上交通，它还将具备普通银行卡的消费能力，可直接用于阿联酋境内的零售门店及线上交易。
RTA局长兼执行董事会主席马塔尔·阿尔·塔耶表示，NOL系统升级是RTA数字化转型战略的关键一步，代表着一项面向未来的数字服务投资。RTA正致力于构建一个统一、一体化的支付生态系统，旨在连接不同交通方式和各类服务，同时追求更高的运营效率和更便捷的出行体验。
