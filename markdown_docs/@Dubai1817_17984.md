# 柬埔寨中国女医遇害案：疑犯曾弑妹，警方设局诱捕

🚨 柬埔寨中国女医遇害案：疑犯曾弑妹，警方设局诱捕
近日，柬埔寨金边市发生一起中国籍女医生遇害案，警方最新披露的案情细节令人震惊。一名47岁的中国籍女医生LIN ZHA在当地诊所内被发现赤裸身亡，现场血迹斑斑。
警方迅速锁定一名37岁的柬籍男子英强为重大嫌疑人。据调查，嫌疑人涉嫌企图强奸受害人未遂后将其残忍杀害，并用诊所内医疗器械多次刺伤受害者致其死亡。案发后，嫌疑人搭乘其姐姐的车辆潜逃至柴桢省老家。警方通过巧妙设局，谎称嫌疑人有事需返回处理，成功诱使嫌疑人姐姐驾车折返金边，并在其住处将其抓获。
更令人不安的是，调查过程中嫌疑人家属透露，英强患有精神疾病近十年，并曾在家乡持木棍杀害亲妹妹。目前，嫌疑人已被依法拘押，案件仍在进一步调查中。警方正协调中国驻柬埔寨使馆及死者家属处理善后事宜。
