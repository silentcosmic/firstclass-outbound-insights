# 马来西亚毒品泛滥：青年群体与合成毒品成最大挑战

🚨 马来西亚毒品泛滥：青年群体与合成毒品成最大挑战
马来西亚内政部长赛夫丁纳苏申近日在国会下议院揭示，该国毒品问题持续恶化，尤其令人担忧的是，15至39岁的青年群体在所有涉毒案件中占比高达近75%，成为受影响最严重的年龄层。
据最新数据显示，马来西亚每年记录的毒品滥用案件高达19万余宗，按人口比例计算，每10万人中约有560人涉毒。尽管近期案件数量略有下降，但赛夫丁部长强调，整体毒品形势并未出现明显好转。
从区域分布来看，吉打州本同县的涉毒率位居全国之首，吉兰丹州瓜拉吉赖、万捷，登嘉楼州勿述以及柔佛州丰盛港也榜上有名。此外，全国41所监狱约8.7万名囚犯中，近七成与毒品案件相关，吉兰丹州马樟监狱的情况尤为严峻，几乎所有囚犯均涉毒。
赛夫丁部长还指出，合成毒品已取代传统毒品，成为马来西亚当前最大的毒品威胁，其中安非他命类兴奋剂约占全国毒品滥用案件的70%，东海岸地区尤其突出。值得警惕的是，执法单位近期在掺杂毒品的电子烟液体中检测到芬太尼，尽管尚未大范围扩散，但其致命效力远超吗啡，构成潜在巨大风险。
他强调，解决毒品问题绝非单一机构之力可为，需要医疗机构、学校、家庭及社区的共同参与，才能从预防、执法及戒毒康复等多个层面实现根本性改善。
