# ️ 阿布扎比警方：警惕假期租房诈骗高发

⚠️ 阿布扎比警方：警惕假期租房诈骗高发
阿布扎比警方近日发出紧急提醒，警告居民及游客警惕日益猖獗的假冒农庄和度假屋租赁骗局。
诈骗分子常利用社交媒体、短信及各类应用程序，在节假日和旅游旺季发布极具诱惑力的“超低价”租房信息，以此诱骗受害者支付定金或所谓的“保险费”。警方强调，此类虚假租房广告是当前网络诈骗的高发类型，提醒公众务必提高警惕，谨慎核实信息来源。
