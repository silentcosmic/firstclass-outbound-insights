# 柬埔寨恶性案件：嫌犯潜逃逾月，受害者家属陷恐惧

🚨 柬埔寨恶性案件：嫌犯潜逃逾月，受害者家属陷恐惧
柬埔寨茶胶省巴地县近日披露一起骇人听闻的案件。一名10岁孤女指控其78岁的继祖父郭崩多次强奸并以死亡威胁恐吓。
该案于今年6月9日因受害者哥哥发现而曝光，嫌犯随后潜逃。截至目前，案发已逾一月，当地警方仍未能将郭某归案。鉴于嫌犯曾发出死亡威胁，受害者及其家人当前安全状况令人担忧，深陷恐惧。
此案已在当地引发强烈愤慨与谴责。目前，社会各界正持续敦促柬埔寨国家警察总署及茶胶省警方加紧介入，加快追捕，以期早日将涉案人员绳之以法，还受害者一个公道。
