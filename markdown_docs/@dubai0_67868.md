# 胡志明市警方重拳出击：捣毁两大跨国诈骗基地，168名华人落网

🚨 胡志明市警方重拳出击：捣毁两大跨国诈骗基地，168名华人落网
2026年6月，越南胡志明市警方开展了一项为期45天的专项治安整治行动，重点打击外国人在当地的非法居留以及跨国网络诈骗活动。此次行动共检查了1616家住宿场所，发现160处违规，并成功侦破了311起违法案件。
行动中，警方成功捣毁了两个以中国籍人员为主的跨国网络诈骗筹备基地，共计抓获168名中国籍嫌疑人。其中，5月18日警方在平阳坊一家酒店查获85名嫌疑人，缴获电脑近200台、手机550多部，揭示了一个由戴飞（雷龙）、胡国宇、周文东等人领导的诈骗团伙，其目标是针对中国公民实施电商平台冒充、虚假金融及股票投资等诈骗。
随后在6月8日，警方在翡翠金酒店又发现了第二个团伙，共83名成员，其中多数为从柬埔寨非法入境。该团伙当时正在接受诈骗培训，组织者范清山及范氏黄如为其提供了场所支持。警方确认，这两个团伙在被截获时均已完成设备安装和人员集结，但尚未正式开始诈骗活动，此次及时行动有效阻止了大型跨国电信诈骗基地在胡志明市的形成。
胡志明市公安局表示，将持续加大力度打击跨国诈骗、高科技犯罪及非法入境行为，并严惩为犯罪团伙提供便利的个人和企业，以防范该市成为跨国犯罪活动的中心。
