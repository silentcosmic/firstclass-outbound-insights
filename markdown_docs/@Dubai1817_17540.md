# 泰国破获跨国偷渡网：6名中国公民被查

🚨 泰国破获跨国偷渡网：6名中国公民被查
泰国警方近日在廊磨喃蒲府开展打击非法移民行动，成功逮捕2名涉嫌运送非法移民的泰国司机，并查获6名中国籍男子。警方根据情报设卡拦截两辆可疑车辆，经查，车内6名中国公民均无法提供合法入境或居留证明，随即被依法控制。
调查揭示，两名司机受雇于一个跨国偷运网络，负责从泰柬边境的乌汶府接载人员，运送至巴吞他尼府，每趟可获得8000泰铢报酬，并承认此前曾多次参与类似运输。警方认为，该偷运网络可能涉及非法移民、人口贩运等跨国犯罪，目前正进一步追查幕后组织者及相关人员。
目前，两名泰国司机已因涉嫌协助非法入境被起诉，而6名中国公民则因涉嫌非法入境、非法居留等违法行为，正接受进一步法律处理。
