# 沙特12GWh储能项目公布新进展：七家中国企业成功入围，中东非市场成储能出海战略要地

沙特电力采购公司（SPPC）近日公布了其第二批储能电池系统项目的资格预审结果。此次共有27家企业成功入围，其中七家中国领先企业榜上有名，包括：国家能源集团旗下的龙源电力集团、南方电网国际（香港）有限公司、远景能源、合肥国轩高科动力能源有限公司、中国电力建设集团、国家电投旗下的上海电力股份有限公司，以及华能集团旗下的尚华投资有限公司。
本轮资格预审所涵盖的是六个独立的储能子项目，每个项目的规模均设定为500MW/2000MWh，且储能时长均为四小时，总计规模高达3GW/12GWh。这些项目将以建设-拥有-运营（BOO）模式进行开发。根据规定，成功中标的联合体需要为每一个独立的储能项目设立专门的项目公司（SPV），并持有其全部股权。届时，各SPV将分别与招标方签署长期的储能服务协议。
在这27家成功入围的企业中，有18家公司同时具备项目管理和技术开发双重资质，另有7家专注于管理成员资格，而仅有2家企业仅拥有技术成员资格。这一结构反映了市场对综合能力的偏好。
据市场观察与公开信息不完全统计，截至2026年上半年，中国企业在中东非地区斩获的储能订单已突破40GWh，实现同比增长17%，这一数字已占据同期中国企业新增海外储能订单总规模的四分之一以上。
上述统计数据基于企业公告及公开信息，为不完全统计，涵盖了储能电芯、储能系统及EPC总包等多种订单类型。对于同一项目涉及多个供应环节的情况，已按照行业惯例进行数据处理。需要指出的是，部分项目尚未披露具体容量，因此实际订单规模或高于现有统计。
中东非储能市场：可再生能源扩张驱动新增长
可再生能源的加速扩张，无疑是推动中东非地区储能需求激增的核心驱动力。在“2030愿景”的宏大框架下，沙特能源部明确提出，目标是在2030年前将可再生能源发电在其总发电量中的占比提升至50%，同时实现48GWh的储能容量目标。
阿联酋在其《国家能源战略2050》中也设定了雄心勃勃的目标：计划在2030年将清洁能源装机容量从14.2GW提升至19.8GW，并致力于在2050年前使清洁能源在其能源结构中的占比达到50%。
国际能源署的数据显示，2025年，中东地区的电力需求预计将实现近4%的同比增长，增速相比2024年进一步加快。伴随用电需求与可再生能源装机容量的同步扩张，电力系统对储能等灵活性资源的需求正在持续攀升。
SPPC迄今已连续推出了两批独立的储能项目。首批项目的规模为2GW/8GWh，而第二批则达到了3GW/12GWh，两批项目合计总规模已达5GW/20GWh，且全部采用了4小时的储能配置。
第二批项目采用的BOO模式，通过储能服务协议确立了长期的服务关系。这一创新机制将储能项目从单纯的一次性设备采购，转型为具备融资能力并可长期运营的基础设施资产，这对于吸引大型电力企业及长期资本的参与具有显著优势。
中国企业：凭借技术与规模优势，赋能中东储能发展
国际能源署报告指出，到2025年，中国在全球锂离子电池制造产能中的占比将超过80%。同时，2025年中国电池包的平均价格预计将分别比北美和欧洲市场低30%和35%。
凭借其规模化的制造能力、成熟的磷酸铁锂技术以及完善的产业链优势，中国储能企业在设备成本控制、产能保障以及项目交付效率方面展现出强大的竞争力。这些核心优势为中东地区大型储能项目的规模化开发提供了坚实的支撑。
中东非的储能市场正在经历显著转型，正从早期的示范项目和单一设备采购，迈向国家级批量开发、长时化配置、光储深度融合以及长期资产运营的新阶段。对于深耕此市场的中国企业而言，竞争的焦点也将随之演变，将逐渐从单纯的系统价格竞争，转向更侧重融资能力、长期履约保障以及本地化服务水平的综合较量。
沙特第二批项目的单批次规模已高达12GWh，这预示着中东非地区储能发展正从单一示范性项目向国家级批量部署模式转变。展望未来，统一规划、多项目同步招标以及标准化复制将成为大型储能市场开发的关键模式。
储能功能演进：从快速响应到跨时段能量调节
中东非地区储能的核心功能正逐步拓展，涵盖电量转移、峰值容量支撑以及新能源消纳。国际能源署的数据显示，全球新增电池储能项目中，以电量转移为主要应用的项目占比已从2015年的约40%跃升至2025年的90%以上。同时，2025年新投运的公用事业级储能项目的平均时长已从2023年的约2小时增至3小时，4小时及以上项目的占比亦持续攀升。这清晰表明，储能在电力系统中的角色正由单纯的快速响应，向更复杂的跨时段能量调节功能演进。
这一发展趋势在中东非的大型储能项目中得到了显著印证。沙特前两批独立的储能项目均采用了4小时的配置，总规模达到了5GW/20GWh。迪拜太阳能公园第七期招标方案中，也配置了1.4GW/8.4GWh的电池储能，其储能时长更是达到了6小时。
阿布扎比目前正着手建设一个超大型的5.2GW光伏与19GWh储能一体化项目，旨在实现全天候输出最高1GW的稳定清洁电力。在此类项目中，储能的角色已不再仅仅是光伏的辅助配套设备，它已然演变为将波动性新能源转化为可调度电力的核心基础设施。
BOO模式推动竞争维度升级：从设备到全生命周期服务
随着沙特等市场加速采纳建设—拥有—运营（BOO）模式，储能项目的竞争逻辑正在发生根本性转变。竞争已从一次性的设备采购，转向对项目全生命周期综合能力的全面较量。成功中标的联合体不仅需要负责项目的投资、融资和建设，更重要的是，他们需要长期持有并运营储能资产，并通过储能服务协议获得持续稳定的收益。
在BOO模式下，项目评估的标准已不再仅仅局限于初始系统价格。它已进一步扩展到对融资成本、电池衰减及容量补充策略、系统效率与可用性、长期运维能力，以及履约和性能担保等全方位的考量。
因此，中东大型储能市场的价值重心正从单纯的设备销售，逐步向项目开发、融资、EPC工程建设以及长期运营等更高价值环节转移。随着项目规模的持续扩大，单一设备企业独立承接大型BOO项目的难度日益增加。在此背景下，具备项目开发、资本组织、工程建设、设备供应和长期性能保障综合能力的联合体将更具竞争优势。这也预示着中国企业的出海模式将逐步向产业链协同和一体化能力输出的方向演进。
随着BOO模式和长期服务协议的日益普及，项目业主及金融机构的关注点将更加聚焦于：高温环境下的实际运行数据、科学的长周期衰减模型、获得银行认可的性能担保、完善的海外备件与运维能力，以及系统供应商的财务稳定性。
面对未来，中国企业亟需加快在中东非地区建立区域服务中心、备件仓储、专业的运维团队以及稳固的当地合作关系。中东非的大型储能项目通常拥有较长的运营周期，因此，项目业主不仅关注设备的及时交付，更看重迅速的故障响应机制、灵活的容量增补方案以及持续的长期技术支持能力。
未来展望：规模化、长时化、资产化与本地化
综合审视，中东非储能市场未来的发展将围绕四个核心关键词展开：规模化、长时化、资产化以及本地化。从目前的项目结构来看，沙特更倾向于国家级的独立储能BOO项目，而阿联酋则重点推动超大型光储一体化及全天候清洁电力项目。对于中国企业而言，成本优势和供应链效率依然是其基础竞争力，但未来市场竞争的胜负，将更多地取决于其融资能力、系统可靠性、长期履约承诺以及本地化服务水平。
