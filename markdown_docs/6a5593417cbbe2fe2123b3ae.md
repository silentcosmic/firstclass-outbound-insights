# 高端韩妆品牌折叠营销漏斗，加速抢占美国市场份额

韩妆中的“K”代表“韩国”（Korean）。然而，它同样能代表“爆炸”（kaboom），因为韩国美妆产品在全球范围，尤其是在美国市场，正经历着爆发式增长。
美国已成为韩国化妆品全球第一大进口国。更关键的是，这股韩妆热潮才刚刚拉开序幕。尽管韩妆在美国市场的营收预计将实现超过7%的增长，但在美国高端美妆产品总销售额中，其占比仍仅为3%，这预示着巨大的增长空间。
雪花秀（Sulwhasoo）正是众多力图抓住这一趋势、积极争取更多美国客户与货架空间的韩妆品牌之一。为实现这一目标，该品牌在拓展其美国消费者基础的过程中，正巧妙地运用营销漏斗两端策略。
雪花秀高级效果营销经理贾兹敏·冈萨雷斯（Jazmine Gonzalez）指出，对雪花秀而言，“品牌建设与营销效果并重”是实现长期成功的关键。
这种策略理念正日益普及，品牌方逐渐意识到，不能以牺牲客户基础建设为代价片面追求短期效果。尤其在韩妆这样竞争激烈的市场中，此点尤为重要。
雪花秀于2010年首次进军美国市场。目前，其产品可在品牌官网、亚马逊、丝芙兰、梅西百货以及其他多家百货商店购买。
为了维持增长势头，雪花秀必须触达那些最有可能产生购买意向的全新客户。为此，该品牌正将联网电视（CTV）和数字户外广告（DOOH）与在线再营销相结合，以提升漏斗上层认知度转化为漏斗下层销售额的转化概率。
为了让尽可能多的潜在客户了解其品牌存在，雪花秀与一家领先的视频广告平台合作，借力其运行联网电视和数字户外广告活动，并对移动端及桌面设备用户进行再营销。
去年，该品牌在两次独立的营销活动中测试了这一策略。其中一项活动以联网电视为起点，并对在线视频、移动设备和桌面设备上的观看者进行再营销。另一项活动则同时涵盖联网电视和数字户外广告，并同样包含数字再营销环节。
这项以联网电视为主导的营销活动于去年年末启动，旨在推广其人参系列美妆产品。其目标受众是购买高端护肤品、在销售雪花秀产品的零售店购物，以及/或对防衰老产品感兴趣的消费者。
为了精准触达目标人群，该视频广告平台根据来自多家数据供应商的消费者与社交媒体数据创建了受众细分。同时，为确保广告仅触达尚未开发的潜在客户群体，雪花秀根据自身客户数据排除了过往购买者。
该营销活动覆盖了多个主流流媒体平台。广告创意中包含一个二维码，用户扫描后可跳转至品牌微站浏览产品。该微站允许用户通过其偏好的零售商（包括品牌直销网站）完成购买。
此次活动还对接触过联网电视广告的家庭进行了再营销，在个人设备上以五到七次的频率投放情境相关广告，覆盖了多个时尚生活类网站。这些再营销广告具备更多互动功能，以鼓励用户参与和转化，例如“前后对比”标签，直观展示不同产品如何实现预期的美容效果。
冈萨雷斯表示，这项活动在很大程度上是一次“上层漏斗策略”的实践。但由于其中融入了可购物元素，雪花秀也期待看到获客成本等下层漏斗指标的改善。
根据该视频广告平台的报告，这项以联网电视为主导的活动实现了6.5%的互动率，这是基于移动和桌面设备上再营销广告内的网站互动数据测量的。尽管缺乏基准数据，很难完全解读这一数字的实际意义（该平台指出，互动率会因营销活动特征而异），但雪花秀品牌仍认为这一数据表现出积极潜力。
冈萨雷斯补充道，当品牌与多家零售商合作时，追踪增量转化效果可能会面临权衡。品牌很难比较自家网站与零售平台上的转化数据，因为在零售平台上品牌并不拥有完整的数据所有权。
她举例指出，如果一位现有客户开始从雪花秀的亚马逊店铺购买产品，而非其直销网站，亚马逊可能会将这些客户标记为新客户，即使他们已经是“忠诚的回头客”。
不过，冈萨雷斯也提到，一些零售商提供了定制化的数据集成方案，这使得效果追踪更为便捷。她提到近期与丝芙兰合作的一项社交媒体活动，双方共享了转化像素，以进行精准的效果衡量。
鉴于跨平台追踪增量效果的难度，雪花秀也在尝试利用广泛的上层漏斗认知度来推动实体店客流量和线下销售。
雪花秀去年在一次数字户外广告活动中测试了这一方法，该活动也包含了联网电视、在线视频、移动和桌面广告。一家广告技术服务商的客户合作总监瓦妮莎·罗特特（Vanessa Rottet）表示，数字户外广告部分的目标是触达“任何可能身处丝芙兰或梅西百货附近的户外人群”。
该活动在销售雪花秀产品的零售店附近（包括曼哈顿地区）的数字广告牌上投放，并根据经过地理围栏的设备ID，对联网电视、在线视频和移动设备上的用户进行再营销。品牌发现，对同一受众投放两到三次广告能获得最佳效果。
根据一家市场研究机构的数据，这项活动使得雪花秀的品牌认知度提升了4.1%，购买意向增加了3.4%。
冈萨雷斯表示，此次活动对客流量和店内销售也产生了积极影响，尽管具体量化其精确贡献存在一定难度。
例如，当某些丝芙兰门店举办店内活动时，雪花秀增加了在这些门店附近投放的数字户外广告曝光量。在这些活动期间，雪花秀发现，靠近品牌数字广告牌的门店，其客流量和销售收入均高于未靠近数字户外广告的门店。
然而，冈萨雷斯坦言：“在门店举办活动时，存在诸多变量，因此很难完全确定数字户外广告是否是直接驱动成果的唯一因素。”但她补充说，从趋势来看，可以合理推测数字户外广告在销售增长中发挥了辅助作用。
与此同时，当雪花秀持续深入分析联网电视、数字户外广告与再营销活动对增量销售和品牌认知度的综合贡献时，本刊编辑亦将关注这类高端美妆产品，以期提供更深度的市场洞察。
