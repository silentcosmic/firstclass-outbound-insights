# 星链越南开放预订：首年费用3140万，目标非大众市场

🚀 星链越南开放预订：首年费用3140万，目标非大众市场
美国太空探索技术公司SpaceX旗下卫星互联网服务“星链”（Starlink）已在越南正式开放个人用户预订。此举标志着该服务在越南政府批准的受控试点框架下取得了关键进展。
根据星链官网信息，越南用户首年使用成本约为3140万越南盾，涵盖了卫星接收终端设备和一年的网络订阅服务。这一价格明显高于越南市场主流的光纤宽带套餐。
从第二年起，用户无需重复购买设备，每月仅需支付约220万越南盾的订阅费，全年总计约2600万越南盾。
越南电信监管部门此前曾预测，星链的定价将是当地固定宽带的数倍，因此其主要目标客群并非普通城市家庭，而是地面通信基础设施薄弱的偏远山区、海岛、海上作业区域以及移动信号覆盖不足的地区。
目前，星链在越南仍处于受控试点阶段，具体的用户规模、服务区域及正式商业运营安排，尚待相关部门和企业进一步公布。星链已在全球100多个国家和地区提供服务，拥有数百万用户。
