# 新加坡数字银行业新里程：绿联银行2025年净利达1610万新元

绿联数字银行在2025财年录得税后利润1610万新元，成功扭转了上一财年的亏损局面。
该银行的2025财年财务报告显示，其总运营收入增至7180万新元，而2024财年为4780万新元。
净利息收入从3660万新元增长至5430万新元，同期，净手续费及佣金收入也从110万新元攀升至290万新元。
绿联数字银行披露，其已于2024年9月实现盈利，并在整个2025财年保持了持续盈利态势。
银行方面将此次盈利视为其围绕中小微企业（MSMEs）需求构建可持续业务战略的重要一环。尤其是在流动性管理、投资信心提振以及应对不断变化的市场条件保持韧性等方面，银行的业务模式正逐步显现成效。
绿联数字银行进一步指出，实现盈利有助于增强市场对新加坡数字银行领域的整体信心。
未来，该行计划继续加大投入，开发更多简化、加速并提升新加坡中小微企业银行服务可及性的解决方案。
绿联数字银行是新加坡金融管理局（MAS）许可的数字批发银行，专为包括中小微企业在内的各类企业提供银行及融资解决方案。
