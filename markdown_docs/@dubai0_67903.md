# 迪拜闯红灯事故频发：已致4死55伤，警方严厉警告

🚨 迪拜闯红灯事故频发：已致4死55伤，警方严厉警告
迪拜近期交通安全形势严峻。今年以来，迪拜全境已发生41起因闯红灯引发的交通事故，共造成4人死亡、55人受伤。
对此，迪拜警方已向驾驶员发出严厉警告，强调遵守交通规则的重要性，以遏制此类事故的持续上升趋势。
