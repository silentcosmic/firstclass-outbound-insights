# 老挝电诈再遭重击：磨丁一写字楼62名中国人被捕

🚨 老挝电诈再遭重击：磨丁一写字楼62名中国人被捕
老挝打击电信诈骗的行动持续升级。昨日下午，老挝磨丁经济特区内一ITC国际贸易中心写字楼遭遇突击检查，一批电信诈骗公司被成功捣毁。现场共逮捕62名嫌疑人，全部为中国籍。老挝公安部门调派部队货车，查获并运走了一整车用于诈骗的电脑设备。
