# 太子集团核心人物胡小伟日本二次被捕，永住权谋划落空

🚨 太子集团核心人物胡小伟日本二次被捕，永住权谋划落空
日本警视厅近日宣布，对柬埔寨太子集团高管胡小伟（又名陈小二，真名胡石，44岁，塞浦路斯籍华裔）实施二次逮捕，理由是其涉嫌违反《出入国管理法》。
警方指控，胡小伟涉嫌于今年4月至5月期间，串通他人非法提供并冒用其在留卡，在东京虚假登记户籍。胡小伟去年11月以“高级人才”身份赴日，此前被曝密谋通过户籍迁入东京以获取日本永住权。
此案背景深远：去年10月，美国政府曾将太子集团指控为“亚洲最大犯罪组织之一”并全面制裁，胡小伟名列其中。该集团主席陈志也已在柬埔寨落网并被遣返回中国。
目前，日本警方正与相关机构联手，深入追查太子集团在日本境内的电信诈骗网络活动。
