# 斯里兰卡移民高官涉护照造假案被捕

🚨 斯里兰卡移民高官涉护照造假案被捕
斯里兰卡移民和出入境管理局一名副局长近日因涉嫌参与伪造外国护照案件，被科伦坡欺诈调查局（FCID）逮捕。这起事件暴露出该国护照申请审核流程中的严重违规行为。
警方调查显示，该官员涉嫌协助他人非法办理护照，并在未完成必要审核程序的情况下处理相关申请。尤其引人关注的是，他被指控将一名被印度遣返人员的照片信息非法关联至他人个人资料，此举可能为有组织犯罪活动提供便利，利用虚假身份获取合法旅行证件。
目前，科伦坡欺诈调查局正根据法院指示，对这名51岁的马塔拉居民（其在移民局担任行政副局长职务）展开深入调查，以彻查此案背后的复杂网络。
