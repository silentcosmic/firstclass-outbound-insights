# 出海 · 头等舱深度报告：国产AI模型加速迭代，全球算力市场景气度持续高企

本刊观察，当前国内大模型技术正加速演进，例如GLM-5.2与Kimi K2.7 Code在百万级上下文、长程智能体（Agent）、智能体编码（Agentic Coding）及实际工程交付能力方面均实现显著提升。这标志着国产模型正从基础的通用问答功能，逐步拓展至开发者工具与企业级工作流的核心应用。同时，Kimi正积极强化其国际化运营布局，而DeepSeek的最新融资则进一步巩固了头部模型产业化的市场预期。此外，微信AI的灰度测试动态，揭示了AI服务入口正从单一独立应用向超级生态系统演进的趋势，预计这将显著提高模型调用频率与推理token的消耗量。
放眼全球市场，Anthropic的营收表现、大客户数量及融资规模均持续获得上调，表明企业工作流及智能体编码（Agentic Coding）的需求依然处于强劲的扩张周期。在算力基础设施层面，AI云服务提供商的订单积压（backlog）呈现增长态势，GPU租赁价格同步走高。与此同时，CPU、DRAM及NAND等关键组件的供应约束正逐步扩大。综合来看，全球算力、存储与租赁服务的市场景气度有望保持高位运行。
企业并购风险提示：*ST海利收购案条款调整引关注
6月21日晚间，*ST海利回复监管机构问询，针对其控股子公司以9360万元收购泽尔顿公司60%股权的事项，已紧急签署了补充协议。此项收购中，目标公司估值增值率达到352%，然而其业绩承诺却低于2025年的预期水平，构成一处关键疑点。此外，目标公司海外销售占比达到90%，其中超过七成依赖间接出口模式，且客户信息透明度不足。该公司方面表示，相关业绩承诺是合理的，并同时修改了原协议中的部分条款，以期更好地保障公司利益。
