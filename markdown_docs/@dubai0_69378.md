# 霍尔木兹海峡7月7日数据：商业航运繁忙，航线选择现新趋势

🔍 霍尔木兹海峡7月7日数据：商业航运繁忙，航线选择现新趋势
根据MarineTraffic最新数据，7月7日霍尔木兹海峡共录得41次已验证的海上穿越活动。其中，商业航运的西向东航行依然占据主导地位，显示该区域货物流通的持续活跃。
值得关注的是，满载原油、干散货和液化石油气（LPG）的船只往来尤为突出。在航线选择上，使用国际海事组织（IMO）推荐航线的比例有所增加，而经由阿曼水域的航线使用则呈现下降趋势。
