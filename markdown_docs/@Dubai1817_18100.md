# 怡保连破两诈骗窝点：18名中国籍男子被捕

🚨 怡保连破两诈骗窝点：18名中国籍男子被捕
马来西亚霹雳州怡保警方于7月13日采取联合行动，成功捣毁两处疑似网络诈骗窝点，并逮捕了18名中国籍男子。
据霹雳州总警长拿督莫哈末阿尔威透露，落网人员年龄介于26至45岁。警方首先突击了位于怡保山景霹雳公寓的一住宅单位，拘捕10人；随后又在打昔英达花园的另一住宅单位逮捕了8人。
行动中，警方查获多台笔记本电脑、手机及其他电子设备，总价值约7.8万令吉。初步调查显示，这些设备可能用于实施网络诈骗，涉及利用金融工具及他人银行账户进行资金流转。
目前，所有嫌疑人及涉案证物已被带回怡保警区总部接受进一步调查，案件将依据诈骗、刑事共谋及移民法令等展开侦办。警方尚未公布该诈骗团伙的具体作案手法、目标对象及涉案金额。
