# 泰国警方破获跨境非法代孕案，中国籍组织者芭提雅落网

🚨 泰国警方破获跨境非法代孕案，中国籍组织者芭提雅落网
泰国移民局于7月5日通报，两名涉嫌组织跨境非法代孕活动的中国籍犯罪嫌疑人已在芭提雅被成功抓捕。此二人目前正被羁押，等待完成相关法律程序后将被遣返中国接受进一步调查。
据中方执法部门通报，被捕者为43岁的鄢某和34岁的米某。他们涉嫌在未取得合法医疗执业资质的情况下，借用“杭州甥宝医疗健康科技公司”名义，于2026年3月至5月期间擅自设立非法医疗场所，从事人工授精、胚胎移植、代孕等违法医疗活动，并借此非法牟利，案件涉及人数众多。
案发后，两名嫌疑人潜逃至泰国。泰国移民局接获中方通报后迅速响应，依法撤销其在泰签证，并联合泰国中央调查局展开抓捕行动，最终在芭提雅一处豪华酒店内将两人成功控制。
此案已进入司法程序。有关部门强调，将持续加强跨境执法合作，严厉打击非法代孕、非法医疗及各类跨国犯罪活动，依法追究相关责任人的法律责任。
