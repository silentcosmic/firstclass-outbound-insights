# 出海洞察 | AI浪潮下微软碳排放量激增：科技巨头的绿色转型挑战

根据最新发布的微软2026年可持续发展报告显示，若排除部分干预措施的影响，该公司在2025年的碳排放量较上一年增长了25%，总量达到3400万吨。这一显著增长主要归因于其数据中心基础设施的持续扩张，同时，去年2月停止采购非额外性、非捆绑式可再生能源证书，也对排放量上升造成了影响。
早在数年前，微软便做出了到2030年实现碳负排放的庄严承诺，即清除的碳排放量须超越自身生产量。然而，迈向这一宏大目标的道路却并非坦途。此前的2024年可持续发展报告已揭示了其气候足迹的类似增长趋势。在今年的报告中，公司方面明确指出，尽管人工智能（AI）基础设施的迅速发展大幅推升了对能源、水资源、土地和关键材料的需求，但与之匹配的可持续发展解决方案的部署速度，尚未能有效满足当前需求。
在行业层面，其他科技巨头同样面临着来自可持续发展目标的巨大压力。谷歌在其2026年可持续发展报告中披露，其供应链排放量呈现25%的增长。而亚马逊公布的增幅相对较低，为16%。另一组数据显示，亚马逊在今年6月透露，其2025年数据中心用水量为95亿升，这一数字低于微软同类设施的用水量。
