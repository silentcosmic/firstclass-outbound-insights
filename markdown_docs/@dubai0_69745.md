# 东帝汶电诈窝点被端：24人落网19人逃逸

🚨 东帝汶电诈窝点被端：24人落网19人逃逸
近日，东帝汶国家警察刑事调查局在首都帝力成功捣毁一处大型电信网络诈骗窝点，累计抓获24名中国籍涉案人员。然而，另有19名嫌疑人在执法人员抵达现场时趁机逃逸。
据警方通报，该窝点藏匿于一栋两层民居内，自今年5月10日起被诈骗团伙占用。尽管房东当初仅与5名租客签订了每月3300美元的租赁合同，但团伙入住后私自改造房屋，内部违规设置43张床铺和18张办公工位，长期聚集数十人实施犯罪。现场查获包括41台台式电脑、多台笔记本及手机，甚至还有3台用于作案的星链卫星网络设备。
此次抓捕行动历时两天，共有19名男性和5名女性落网，所有嫌疑人及物证已统一移交刑事调查局核查。房东向媒体表示对房屋被非法改造毫不知情，并全程配合警方调查。东帝汶警方发言人指出，该窝点的诈骗作案模式与此前查获的同类案件高度相似。目前，警方正联合移民部门全力追查19名在逃人员，并深入调查该犯罪团伙的完整上下游链条。
