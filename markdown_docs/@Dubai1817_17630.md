# 印尼破获人口走私案：7名中国公民涉嫌偷渡澳洲被拘

🚨 印尼破获人口走私案：7名中国公民涉嫌偷渡澳洲被拘
印尼东南苏拉威西省肯达里市近日成功破获一起人口走私案件，7名中国公民在试图非法离境前往澳大利亚时被当地移民部门拘留。据初步调查，这批人员不仅涉嫌偷渡，其在印尼的合法停留期限也已届满。
此次行动是在东南苏拉威西省警方提供情报后展开，当地移民执法人员于6月9日在肯达里市多地实施联合行动，成功控制了这7名中国公民。调查发现，他们疑似受人口走私团伙安排，计划绕过正规出入境检查点，非法离境前往澳大利亚。执法部门同时指出，这些人员也可能属于人口走私案件中的受害者。
肯达里移民局负责人穆罕默德·诺夫里安·贾亚表示，现有证据表明案件存在人口走私犯罪网络介入，目前正追查幕后组织及涉案人员。被拘留人员已被安置在移民拘留中心，等待行政程序完成后将被依法驱逐出境，并禁止其5年内再次入境印尼。
