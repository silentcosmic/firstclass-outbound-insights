# 越南“净零挑战”2026创新大赛启动，80万美元巨额奖金助力气候科技发展

聚焦越南市场，该国规模最大的气候科技创新竞赛“净零挑战赛”已正式启动第四届申请通道。本届赛事设立了总额超过210亿越南盾（约合80万美元）的丰厚奖金池，项目申请期定于2026年7月1日至8月31日。
本次具有行业前瞻性的挑战赛，由越南本土的Touchstone Partners与新加坡的淡马锡基金会携手策划与执行。
Touchstone Partners方面表示，本届竞赛总奖金池的构成颇具战略考量。其中，150亿越南盾将作为非股权性质的催化资金，以无附加股权要求的赠款形式，直接授予排名前三的顶尖团队。余下的60亿越南盾则将以投资奖励的形式发放，并获得合作风险投资基金及相关企业的额外资源支持。
参赛团队将围绕三大核心赛道展开激烈角逐：其一，“可再生能源与碳去除”，细分领域涵盖太阳能、风能、储能技术、电网现代化、绿色氢能及碳排放测量方案；其二，“食品系统与可持续农业”，此赛道专为优化越南的农业及水产养殖业而设；其三，则是“循环经济与废物管理”，重点关注工业废弃物回收、塑料替代材料的开发以及城市回收基础设施的创新建设。
Touchstone Partners管理合伙人Khanh Tran指出，该计划的深远目标在于发掘、链接并扶持全球范围内卓越的绿色经济创新项目，从而为前瞻性技术在越南的落地试点与规模化应用铺设通路，创造更多机遇。
淡马锡基金会气候与宜居性部门负责人Heng Li Lang则强调，这项计划的关键在于构建一个多方协同平台，汇聚创新者、投资机构、产业界以及公共部门的力量，共同推动具备潜力的气候科技解决方案，将其转化为真实可见的社会与经济效益。
