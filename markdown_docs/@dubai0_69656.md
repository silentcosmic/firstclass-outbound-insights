# 阿联酋确认境外导弹威胁，国家系统保持最高戒备

🚨 阿联酋确认境外导弹威胁，国家系统保持最高戒备
阿联酋国家紧急危机与灾害管理局（NCEMA）近日发布声明，确认上午探测到的导弹威胁位于阿联酋境外。NCEMA强调，当前局势保持稳定，国家各系统持续维持最高级别的戒备状态。
该机构在与战略合作伙伴协调后，明确了威胁的地理位置，并表示目前没有发现任何令人担忧的迹象。阿联酋的国家监测系统正全天候运行，密切追踪事态发展，以确保公众安全。
