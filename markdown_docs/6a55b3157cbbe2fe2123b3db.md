# 东南亚数字经济前瞻：AWS重金布局菲律宾云基建，泰国激活多项关键产业

过去一周，东南亚科技市场涌现出多项值得深入分析的新动态。本期《出海·头等舱》将为您梳理从企业重大进展、关键产业趋势到投融资领域的最新情报。（涵盖2026年7月7日至2026年7月12日）
AWS拟在菲律宾投资最高50亿美元
亚马逊网络服务（AWS）正在与菲律宾政府高层探讨一项可能高达50亿美元的长期云基础设施投资，核心内容包括在当地设立AWS区域服务中心。这项战略性布局预计将在未来15年内逐步落实，旨在显著提升菲律宾的数字基础设施水平与云服务承载能力。尽管具体投资规模、建设规划及时间表仍在紧密磋商中，但此举无疑预示着菲律宾数字经济版图的巨大飞跃。
泰国批准9个产业项目，总投资19.9亿美元
泰国投资促进委员会（BOI）近日批准了九个总计达19.9亿美元的重大产业项目，涵盖人工智能基础设施、先进电子制造、航空航天、清洁能源及食品生产等多个战略领域。具体来看，日本Datasection将投入2.35亿美元用于构建先进的GPU服务器基础设施；雀巢公司则计划斥资6.89亿美元扩建其咖啡生产设施；此外，两座总装机容量达120兆瓦的风力发电项目也获得1.69亿美元投资，彰显了泰国在多元化产业升级上的决心。
淡马锡计划将AI投资占比提高至10%至15%
新加坡主权财富基金淡马锡（Temasek）在其最新年度投资回顾中透露，计划在2031年3月前，将其人工智能（AI）领域的直接投资占比从目前的约6%提升至10%至15%。截至2026年3月底，淡马锡的投资组合净值已达5180亿新元。其在AI生态系统中的布局极为广泛，涵盖了能源与数据中心、半导体、云服务、基础模型开发以及各类AI应用与软件基础设施。
Carousell首次实现全年EBITDA转正
新加坡领先的二手交易平台Carousell在2025财年取得了里程碑式的业绩突破，首次实现全年息税折旧摊销前利润（EBITDA）转正。公司收入强劲增长18%，达到1.41亿美元，相比2021财年几乎翻了两番。其中，核心的二手商品交易业务收入飙升40%，贡献了总收入的45%。截至财年末，Carousell已在新加坡、中国香港、马来西亚及印度尼西亚设立并运营29家实体门店，业务版图持续拓展。
马来西亚将建设年产3000吨稀土永磁体工厂
澳大利亚稀土巨头莱纳斯（Lynas Rare Earths）与韩国磁体制造商JS Link签署了一项长期合作协议，双方计划在马来西亚关丹共建一座稀土永磁体工厂。新工厂将选址在莱纳斯现有先进材料工厂附近，设计年产能为3000吨钕铁硼烧结永磁体，预计将创造多达400个新的就业机会。作为合作的一部分，莱纳斯还将向JS Link投资约5000万澳元（约合3470万美元），此举将进一步巩固马来西亚在全球稀土产业链中的地位。
新加坡拟将新类型基金审批时间缩短至三周
新加坡金融管理局（MAS）近期就零售基金监管规则的修订公开征求意见，核心目标是为那些现有框架难以完全涵盖的新类型基金建立一套独立的审批机制。根据该提案，监管机构预计将用大约三个月的时间来界定一类新基金所需的风险防护措施。一旦框架确立，后续符合同类要求的产品最快有望在三周内获得批准。与此同时，基金的流动性管理、资产保管、公司治理和信息披露等核心监管要求将继续得到严格执行，以确保市场稳定与投资者权益。
马来西亚公共电动汽车充电点增至6416个
马来西亚投资、贸易及工业部最新数据显示，截至2026年5月底，该国公共电动汽车充电点数量已增至6416个，其中包括2143个直流快速充电桩和4273个交流充电桩。马来西亚政府正积极推动电动汽车产业发展，目标是到2030年电动化汽车销量占比达到20%。同时，国家战略重点正从单纯的整车进口逐步转向本地组装、电池制造、电机以及高级驾驶辅助系统等关键供应链环节，以构建更具韧性的本土电动汽车生态系统。
新加坡首个量子科技加速器纳入4家公司
量子科技投资机构QAI Ventures近期在新加坡正式启动了其首个量子科技加速器项目——Singapore Quantum Accelerator。首批入选的四家公司从全球12个国家的63份申请中脱颖而出，其创新方向涵盖低温量子硬件、神经刺激技术、光子芯片制造以及先进材料研发。每家入选企业将获得30万新元的投资以及全面的商业化支持，项目孵化周期为五个月，旨在加速新加坡在全球量子科技领域的创新步伐。
马来西亚数据中心热度不减，但谁来供电、供水和供人？
马来西亚数据中心市场持续保持高热度，但其高速发展背后所需的能源、水资源及人才供应正成为关键挑战。根据最新分析报告预测，马来西亚的数据中心容量有望从2025年的约0.9至1吉瓦（GW）飙升至2029年的3至4吉瓦，其中柔佛州目前已运营容量占据全国的80%。然而，报告指出，一座100兆瓦（MW）的数据中心每日需消耗约420万升水，且单个设施通常仅需30至50名全职员工。因此，电力供应、水资源管理、专业技术人才匮乏以及本土供应链的成熟度，正成为制约该行业持续扩张的主要瓶颈。
九成企业押注数据治理，新加坡AI投入逻辑正在转变
新加坡在人工智能领域的投资逻辑正在悄然转变，数据治理的重要性日益凸显。一项针对新加坡IT负责人的最新调查揭示，高达78%的受访者认为，实时数据基础设施的不足是阻碍AI规模化落地的主要因素。同时，75%的企业已部署或正在试点AI智能体。在投资优先级上，90%的受访者将数据管理和治理视为重中之重，86%的企业则高度关注数据流处理能力。值得注意的是，超过73%的AI项目已遭遇瓶颈，其中约一半甚至已被彻底放弃，这反映出在追求AI效益时，基础数据能力建设的不可或缺性。
新加坡员工不排斥AI，但日常核心使用率仅6%
尽管新加坡员工对人工智能的接受度较高，仅有29%的受访职员对AI持怀疑态度，远低于全球平均水平的37%，但AI在日常工作中的核心使用率却出人意料地低，仅为6%。另有本地人力资源机构的调研数据指出，多达71.5%的私营企业尚未采纳AI技术。即便在已引入AI的企业中，也只有3.8%的企业能将其深度融入核心业务流程，这表明AI在新加坡的普及应用仍面临从初步接受到深度融合的巨大鸿沟。
IMF预计马来西亚和泰国2026年分别增长4.7%和1.9%
根据国际货币基金组织（IMF）最新发布的《世界经济展望》报告，韩国、中国台湾、泰国和马来西亚被视为AI相关硬件的主要净出口经济体，这些地区在2026年第一季度的经济表现平均超出此前预测4.4个百分点。报告维持对马来西亚2026年经济增长4.7%的预期，同时将泰国经济增长预期上调0.4个百分点至1.9%。越南则以7.5%的增长预期领跑，主要得益于其科技产品出口的强劲表现和内需的有力支撑。
马来西亚可再生能源装机占比有望达到32%
一项最新的行业研究预测，马来西亚有望在2026年实现可再生能源装机容量占比达到32%的目标，并进一步规划在2030年和2050年分别提升至35%和70%。数据中心日益增长的用电需求、屋顶太阳能的普及以及电池储能技术的进步，正成为推动可再生能源新增装机的重要动力。截至今年5月，马来西亚Solar ATAP屋顶太阳能计划的申请量已突破1万份，显示出市场对绿色能源的积极响应。
76%新加坡投资者使用AI辅助理财
一项对新加坡中高净值投资者的最新调研显示，高达76%的受访者已将人工智能（AI）工具融入其金融与投资活动中，这一比例高于全球平均水平的72%。在可投资资产超过200万美元的高净值群体中，AI的使用率更是达到90%。然而，仅有8%的新加坡受访者将AI视为其最近一次重大投资决策中最具决定性的信息来源，而57%的投资者则更倾向于结合AI工具的分析结果与人工顾问的专业建议，做出最终判断，显示出人机协作在财富管理领域的主导趋势。
