# 美军一周四度空袭伊朗，中东局势持续紧张

🔥 美军一周四度空袭伊朗，中东局势持续紧张
美国中央司令部日前宣布，美军于当地时间12日17时对伊朗发动了新一轮打击。据美方声明，此次行动旨在持续削弱伊朗在霍尔木兹海峡袭击商船的能力。
此次打击标志着本周内美军已对伊朗发动四次军事行动，凸显中东地区紧张局势的持续升级。
