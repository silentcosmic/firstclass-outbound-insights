# 迪拜海关新政：关税可分期，罚款最高减80%

📣 迪拜海关新政：关税可分期，罚款最高减80%
迪拜海关近期发布了两项优化贸易环境的重磅新规，旨在减轻企业运营负担，进一步激发市场活力。
其中，针对2026年3月1日至7月30日期间产生的进口关税，企业现可申请分期缴纳。此举允许企业在获得迪拜海关批准后，最长可在一年内完成税款支付。相关申请截止日期为2026年9月30日，企业需提前提交申请以获批。
同时，对于在2026年2月28日之前产生的海关罚款，符合条件的企业有机会获得高达80%的减免，即仅需补缴20%的罚款金额。该政策申请截止至2026年12月31日。需注意的是，此减免仅限于罚款本身，不包括欠缴的关税及其他相关费用。已结清的罚款或正在司法程序中的案件通常不适用此项减免，但存在特殊情况。完成剩余20%罚款的支付后，减免方可正式生效。在特殊情况下，这20%的罚款也可申请分期，但最迟须于2027年6月30日前全数付清。
