# 阿联酋首只零售伊斯兰债券认购火爆，发行量翻倍

🔥 阿联酋首只零售伊斯兰债券认购火爆，发行量翻倍
阿联酋首只面向零售投资者的伊斯兰债券（Sukuk）发行遭遇市场热烈追捧，原定5000万迪拉姆的发行规模，在认购阶段吸引了高达4.45亿迪拉姆的资金，超额认购近九倍。鉴于此，发行方决定将最终发行量翻倍至1亿迪拉姆，以满足强劲的市场需求。
这批两年期伊斯兰债券的年收益率为4.30%，每半年派发一次“利润”（而非传统意义上的利息，以符合伊斯兰教法）。其最低投资门槛仅为1000迪拉姆，极大地降低了个人投资者的参与难度。目前，该债券已在纳斯达克迪拜交易所上市交易，为未能参与首发认购的投资者提供了二级市场购买渠道。
