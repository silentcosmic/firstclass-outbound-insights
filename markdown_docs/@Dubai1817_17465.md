# 马来西亚女警不当言论引争议：警方启动纪律调查

🚨 马来西亚女警不当言论引争议：警方启动纪律调查
马来西亚全国警察总长莫哈末卡立于6月23日证实，警方已就一名女警员在中国旅游期间发表不当言论、涉嫌侮辱当地人的事件，启动正式纪律调查。此前，该女警在社交媒体上发布视频，内容涉及掩鼻并嘲讽当地人“好臭”“都不洗澡”，此举迅速引发网络大量批评与不满。
莫哈末卡立强调，此次调查不仅将审视警队内部纪律问题，更会核查涉事行为是否触犯当地法律。他明确指出，无论警员身处何地，都必须维护职业操守、警队纪律及国家整体形象，警方将严格依照相关法律与程序，秉持公平透明原则进行调查，并对任何违法行为采取相应行动。
涉事女警随后已公开道歉，承认其当时反应冲动、言行过火，并表示愿意承担由此造成的不适，承诺未来将更加谨慎言行。
