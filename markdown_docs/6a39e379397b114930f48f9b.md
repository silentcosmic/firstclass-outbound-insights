# 菲律宾SunAsia与越南VinEnergo达成战略合作：4.06亿美元共建水面光伏项目，赋能区域能源转型

菲律宾SunAsia能源公司与越南VinEnergo将携手，在菲律宾邦板牙省（Pampanga）和西内格罗斯省（Negros Occidental）共同开发总装机容量达422兆峰瓦（MWp）的水面光伏发电项目组合。此项合作的预计投资额高达4.06亿美元。
SunAsia能源公司周一发布声明，确认双方已就此项重要举措达成战略合作。这三个项目预计于2027年至2028年间投入商业运营，将全面采用先进的漂浮式光伏技术。其独特之处在于，太阳能电池板将架设于水体上方，同时不影响下方及周边区域的渔业养殖活动。
一旦全面投运，这些项目每年可为超过27.8万户家庭提供充足电力，同时每年减少约460,888吨的碳排放，对区域绿色发展贡献显著。
在接下来的18个月里，SunAsia能源公司与VinEnergo将共同推进工程设计和项目开发工作。这包括甄选海事工程师、技术供应商和承建商，以确保近70万块太阳能电池板的支架安装以及约62公里新建输电线路的铺设。
SunAsia能源公司首席执行官兼总裁Tetchi Capellan表示，此次合作源于双方在提升能源安全、支持经济增长及环境保护方面的共同决心。鉴于越南在虾类出口市场的领先地位以及菲律宾悠久的虾类养殖历史，将水产养殖与能源生产相结合，为“渔光互补”模式提供了极具说服力的发展前景。
VinEnergo首席执行官Nguyen Anh Khoa指出，水面光伏技术为推动可持续能源增长和实现负责任的资源利用提供了独一无二的机遇。
与此同时，越南VinEnergo近年来在全球可再生能源市场动作频频，其在丹麦、瑞典及菲律宾等地的可再生能源项目组合已累计新增10吉瓦（GW）装机容量。这进一步凸显了VinEnergo在国际清洁能源领域的战略布局与雄心。
