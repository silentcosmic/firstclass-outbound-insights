# 曝亚太集团高管涉人口贩卖，赌场洗码内幕浮出

🚨 曝亚太集团高管涉人口贩卖，赌场洗码内幕浮出
有市场消息指出，一名名为冯鑫（昵称“鑫哥”）的个人被指控为亚太集团旗下人口贩卖网络的头目，长期在亚太国际酒店活动。消息称，冯鑫行事隐秘，常戴口罩和棒球帽。
据爆料，冯鑫目前在百乐门赌场从事洗码业务，其所掌控的“三鑫公司”虽已更名，但仍活跃于亚太地区多家赌场。该公司被控诱骗中国公民参与赌博，一旦受害者无力偿还赌债，便将其扣押在千禧酒店，进而贩卖至各类非法黑园区。其中一受害者家属透露，其亲属被三鑫公司诱至百乐门赌场，以15000至20000美元的“出码”名义进行欺诈，随后遭绑架并贩卖至黑园区，目前伤势严重。
此外，冯鑫还被指与亚太及百乐门赌场的公关主管合谋，利用女主管的职务便利，将散客转归其洗码公司，共同瓜分洗码佣金。爆料者公开质问亚太集团主席何英雄，鉴于其曾声明集团不存在人口贩卖行为，应如何解释此次针对核心人员的指控。家属呼吁受害者亲友立即报警。
