# 百事泰国推“迪拜巧克力”可乐，口味大胆创新

⚡ 百事泰国推“迪拜巧克力”可乐，口味大胆创新
百事公司近日在泰国市场推出了一款引人注目的新口味碳酸饮料——Pepsi Treats迪拜巧克力风味可乐。这款零糖配方新品的定价约为每罐3.7元人民币。
此举体现了快消巨头在区域市场进行产品差异化和口味本土化创新的策略，旨在吸引对新奇口味有兴趣的消费者。
