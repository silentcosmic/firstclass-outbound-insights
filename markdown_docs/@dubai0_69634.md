# 水沟谷重拳打击赌诈：8层违建爆破，累计71栋

🚨 水沟谷重拳打击赌诈：8层违建爆破，累计71栋
7月10日，当地当局对水沟谷区域内一栋曾用于网络赌博诈骗活动的8层非法建筑实施了爆破拆除。
根据此前部署，该区域共有77栋目标违建，其中57栋计划通过机械拆除，另有20栋采取爆破方式。此次被拆除的8层楼即属于爆破范畴。截至目前，累计已有71栋违建完成拆除，剩余的6栋目标建筑也将陆续完成拆除工作。
