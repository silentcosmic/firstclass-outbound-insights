# 柬埔寨大规模打击网络诈骗，207人被捕

🚨 柬埔寨大规模打击网络诈骗，207人被捕
2026年7月3日下午，柬埔寨卜迭棉芷省统一指挥委员会展开大规模联合执法行动，对诗梳风市的五处涉嫌从事网络诈骗的窝点进行了突击搜查。此次行动中，执法人员共拘捕了207名不同国籍的外籍人员，他们均涉嫌参与网络诈骗活动。
