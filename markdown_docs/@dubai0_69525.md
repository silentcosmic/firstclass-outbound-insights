# 波贝突袭：16名外国人被拘，业主遭罚

🚨 波贝突袭：16名外国人被拘，业主遭罚
昨日（7月9日），柬埔寨波贝市当局采取突击行动，拘留了16名外国公民，行动中还涉及2条犬只。
据警方消息，此次行动在两处地点展开。在一栋三层七室公寓中，警方查获了12名印尼公民，该房屋业主因未按规定上报外国人信息而受到罚款。在另一栋四层22室的楼房中，警方又发现并拘留了4名印尼公民。
