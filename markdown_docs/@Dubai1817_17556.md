# 越南边境查获巨额走私iPhone，两嫌犯面临起诉

🚨 越南边境查获巨额走私iPhone，两嫌犯面临起诉
越南广宁省警方近日完成一起跨境非法运输手机案件的调查，并将两名涉案男子移送同级人民检察院，建议依法提起公诉。这批从中国非法运入越南的货物包括117部二手苹果手机，总价值超过10亿越南盾。
警方查明，涉案人员Bùi Văn Hưng和Nguyễn Quốc Toàn未经正常报关及进出口程序，非法将这些已使用过的手机运入越南。此案被广宁省警方视为当地在边境走私和非法运输专项整治行动中查办的典型案例。
近期，越南警方正重点打击利用中越边境地区非法运输商品的团伙和个人，并加强对货物运输路线、仓储地点及交易网络的调查，以遏制跨境走私活动。
