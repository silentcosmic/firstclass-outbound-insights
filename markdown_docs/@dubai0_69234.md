# 菲律宾上半年绑架案：华人受害华人作案凸显风险

🚨 菲律宾上半年绑架案：华人受害华人作案凸显风险
中国驻菲律宾大使馆近日通报，2026年上半年菲律宾共发生7起涉中国公民绑架案，与去年同期相比下降36%。这些案件呈现出显著共性：受害者与绑匪均为在菲长期工作、生活或学习的中国公民，案发地点集中在马尼拉及周边区域。
案件诱因多样，包括网络赌博、电信诈骗、债务纠纷，以及通过社交媒体交友、求职或联系学车等。其中，多名受害者在被掳走后，家属通过支付赎金获释；亦有受害者在被拘禁期间寻机逃脱或被释放。
中国驻菲律宾使馆对此高度重视，正全力协助两国执法部门打击此类跨国犯罪。今年上半年，中菲双方已联合抓获并遣返56名中国籍犯罪分子，其中包括4名绑匪。
