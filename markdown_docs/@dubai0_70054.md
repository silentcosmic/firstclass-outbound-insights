# ️ 缅甸星链设备或大规模停用，人道通讯受冲击

⚠️ 缅甸星链设备或大规模停用，人道通讯受冲击
据市场消息，缅甸境内广泛使用的星链卫星网络服务，或将自7月17日起面临大规模停用。SpaceX近期实施新规，要求设备在非注册国使用超过30天需更改地区设定，并强制进行护照旅行登记，否则将被停用。由于缅甸尚未正式批准星链服务，此举预计将导致下个月大量设备断网。此外，服务资费也已上调。
过去，星链在实皆省、掸邦等网络受限地区，对于人道主义通讯至关重要。然而，随着边境电诈团伙滥用该设备，星链的服务政策已被迫收紧，以应对合规挑战。
