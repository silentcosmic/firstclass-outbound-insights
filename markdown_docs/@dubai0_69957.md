# ️ 柬埔寨电诈核心：政府授予“公爵”身份者逍遥法外

⚠️ 柬埔寨电诈核心：政府授予“公爵”身份者逍遥法外
据路透社最新调查披露，尽管外界普遍认为柬埔寨的电诈园区与中国势力紧密相关，但其幕后真正的地主和保护者实为柬埔寨本地人，其中不乏拥有政府授予“公爵”身份的显赫人物，目前仍在法外。
此次调查揭示了“皇家山庄度假村”等多个电诈园区，其业主即为本地公爵林亨。与此隔路相望的国际知名“奥斯玛电诈园区”，则以“奥斯玛度假村”名义持有赌场牌照，其背后控制者是柬埔寨公爵李永法。值得关注的是，李永法及其多家关联公司已因涉嫌网络赌博诈骗及人口贩卖活动，遭到美国政府的制裁。
据报道，奥斯玛镇的赌场及园区自2018年起快速扩张，预计到2025年底将达到电诈活动的顶峰。近年来，中国媒体已多次披露有中国公民在奥斯玛电诈园区内遭受虐待致死的悲剧，例如2025年8月，湖南籍公民李峻旭在奥斯玛的利索赌场内被殴打三天后不幸身亡。然而，截至目前，柬埔寨政府尚未对奥斯玛地区的电诈活动展开任何调查或采取惩罚措施。
