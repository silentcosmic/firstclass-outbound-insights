# ️ 特朗普社媒出售数据：华尔街直通总统言论

⚠️ 特朗普社媒出售数据：华尔街直通总统言论
特朗普媒体科技集团（TMTG）近日宣布，旗下社交平台Truth Social将自8月1日起，面向机构客户推出其名为Truth API的数据服务。该服务以毫秒级速度，实时提供平台上排名前十账号的动态更新，旨在为高频算法交易者提供关键信息优势。公司尚未公布具体定价。
Truth Social已然成为美国前总统特朗普发布政策意向和重要声明的首要平台，其关于关税、国际关系等帖文曾多次引发全球股市与油市的剧烈波动。TMTG将此举视为货币化其专有资产、拓展高利润收入的战略步骤，然而，此举也再度引燃了外界对于总统候选人可能模糊商业利益与公共职责界限的深切担忧。此前，有媒体调查指出，特朗普曾利用该平台宣传自己投资的上市公司。
