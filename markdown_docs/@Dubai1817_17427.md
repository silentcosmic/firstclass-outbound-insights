# 阿联酋拟购印度旗舰防空系统，中东军贸格局生变

⚡ 阿联酋拟购印度旗舰防空系统，中东军贸格局生变
据路透社援引消息人士报道，印度正与阿联酋就出售其旗舰级武器系统展开深入谈判，包括“布拉莫斯”超音速巡航导弹及“阿卡什蒂尔”防空系统。若交易达成，这将是印度向中东地区出口高端武器的重大突破。
此次谈判由阿联酋方面主动发起，进程迅速。面对中东地区持续的冲突与新兴威胁，阿联酋正加速武器采购以强化自身防卫能力，尤其关注霍尔木兹海峡这一关键能源出口通道的安全。
近年来，印度与阿联酋关系持续升温，双方已在贸易、能源等领域达成多项合作，并签署了军事装备联合研发协议。分析认为，阿联酋寻求武器供应商多元化，旨在增强战略自主性，此举预计也不会影响其与美国同盟关系。
