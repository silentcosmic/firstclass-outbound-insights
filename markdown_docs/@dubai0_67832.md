# 特朗普：霍尔木兹海峡“守护天使”收费论引发关注

📣 特朗普：霍尔木兹海峡“守护天使”收费论引发关注
美国前总统特朗普近日就霍尔木兹海峡的通行费用问题发表了最新观点。他表示，在一个为期60天的停火期内，霍尔木兹海峡将不会收取任何通行费用，且停火期结束后也将维持免费。然而，这一承诺附带一个重要条件。
特朗普指出，如果相关协议未能达成，美国将有权为自身目的征收费用。他将此举定位为美国作为“守护天使”向中东各国提供服务的报酬，旨在偿付过去、现在及未来为此付出的成本。此番言论无疑将对全球航运及中东地缘政治格局产生深远影响。
