# 西港坠楼意外：男子触电命危，紧急送医抢救

🚨 西港坠楼意外：男子触电命危，紧急送医抢救
今天早上约6点，柬埔寨西港发生一起严重坠楼事故。一名外籍男子因不明原因从一处大楼高处坠下，但在坠落过程中，他意外被悬挂在路边的电线杆及变压器电线上，并瞬间遭受强烈高压电击。
据现场消息，该男子全身多处严重烧伤，伤势极其危急，被发现时仅存微弱呼吸。事发后，当地医护人员迅速赶往现场进行紧急处置，并已将伤者送往医院全力抢救。
