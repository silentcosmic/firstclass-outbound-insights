# 智能基础设施与城市共生：数据中心如何驱动区域经济可持续增长

当今世界，数据中心及其他人工智能相关基础设施的飞速发展，已成为最重要的趋势之一。非洲大陆亦是如此，其对数字服务的需求正加速攀升。尤其南非，凭借其卓越的互联互通性、庞大的市场规模以及在区域经济中的战略地位，已然成为非洲大陆数据中心投资的领先目的地。
对于南非行政首都茨瓦内等城市而言，这无疑带来了重要的发展机遇。数字基础设施有能力支撑创新，吸引投资，提升区域竞争力，并在经济的各个领域创造包括就业岗位在内的新机会。这也是全球众多城市竞相争取数据中心落户的核心原因。
然而，在全球围绕人工智能基础设施展开的讨论中，一个不容忽视的现实是：数据中心虽能支持数字经济的蓬勃发展，但其运营却高度依赖于实体的物理基础设施，包括电力网络、供水系统、道路交通以及各项市政服务。
人工智能相关基础设施的快速增长，给城市带来了平衡国家经济发展优先事项与地方公共服务供应责任的挑战。
企业倾向于在基础设施可靠的地区进行投资，产业会在具备确定性的环境中扩张，而就业机会则诞生于有能力支撑经济活动的城市。并非所有城市都具备这种承载力。如果主要的人工智能基础设施似乎与当地社区所承受的压力隔绝开来，居民们无疑会提出质疑。当家中灯光熄灭，而附近的数字中心依旧灯火通明时，公众的信任便会受到侵蚀。
茨瓦内等城市深知基础设施限制所带来的影响。非洲城市渴望参与到人工智能经济中，但我们不能重蹈覆辙，让基础设施先行落地，而城市随后才疲于应对其带来的各种后果。长期以来，影响非洲城市的决策往往在别处制定，地方政府未能及早参与协商，尽管它们肩负着基础设施建设与服务交付的重任。驱动人工智能发展的各项基础设施将塑造城市未来数十年的面貌，因此，市政当局必须在其发展过程中拥有话语权。
这正是“全球城市数据中心契约”致力于解决的核心问题。包括伦敦、芝加哥、孟买、内罗毕、里约热内卢和阿比让在内的四十多个城市已签署了这份协议，旨在推动数据中心实现可持续发展和资源高效利用；尊重当地社区需求；为地方带来经济效益；并积极投资于当地社区。
该契约在“伦敦气候行动周”期间启动，它还确保了非洲及其他发展中世界的城市，能够出席并参与到数据中心标准和预期制定等关键会议中。
全球近60%的人口居住在城市，且城市化进程预计将进一步加速。城市面临的诸多挑战日益呈现全球性特征，尽管其影响往往在地方层面感受最为强烈。无论是气候韧性、基础设施投资、融资渠道，抑或是数字技术的迅猛发展，城市都必须成为解决方案的积极塑造者，而非被动接受外部决策的实体。
经济增长离不开基础设施建设，然而，尚无任何城市能够完全解决发展与能源、土地和水资源压力之间的平衡难题。数据中心行业正迅速演进，而城市之间若能相互学习借鉴，便能更快地应对这些挑战。
非洲城市可能面临与欧洲或北美城市不同的具体情况，但其核心问题是共通的：我们如何在支持投资的同时保护当地居民？如何确保社区能够从中受益？以及如何在基础设施冲突爆发之前，与行业建立可行的合作关系？
这些都是至关重要的问题，及早加以解决有助于避免全球日益普遍的冲突发生。当家中灯光熄灭，而附近的数字中心依旧灯火通明时，公众的信任便会受到侵蚀。
对于茨瓦内以及其他发展中世界的城市而言，关于数据中心的讨论，最终都指向了经济增长这一核心议题。投资追随可靠的基础设施，而可持续增长则依赖于长期的规划。因此，数字基础设施应当融入更广泛的城市发展议程之中，以支持投资、提振商业信心、促进创新并创造就业。
企业期望稳定的城市环境、可预见的规划政策以及与地方政府建立强有力的长期合作关系。更优的协调机制既能造福居民，也能增强投资的确定性。规划得当的城市对长期投资更具吸引力，能够为产业发展、科技创新和可持续经济增长营造所需的有利环境。
人工智能相关基础设施的增长，给城市带来了平衡国家经济优先事项与地方公共服务供应责任的挑战。大规模投资必须以一种既惠及投资者又造福居民的方式，融入到地方规划之中。如此，城市便能通过增强基础设施韧性、提升服务可靠性、强化可持续性以及营造更具竞争力的投资环境，来有力支撑经济增长。
在展望数字基础设施的未来时，这种方法显得尤为关键。数据中心、人工智能相关投资以及其他技术驱动型产业，均离不开可靠的基础设施、监管确定性以及长远规划。
更为重要的是，机遇远不止于数据中心本身的建设。负责任的数字基础设施能够促进软件开发、网络安全、技术服务、维护产业、先进制造业以及数字技能培训等领域的发展。其目标不仅是承载基础设施，更要确保投资能够转化为有意义的经济参与和共享繁荣。
成功的城市将不仅仅是简单地承载基础设施的场所。它们将是那些积极参与塑造基础设施规划、治理并将其融入城市生活中的城市。茨瓦内等城市不应仅仅是根据他人优先事项而设计的基础设施的被动承载者。“全球城市数据中心契约”可以帮助确保非洲、亚洲和拉丁美洲的城市成为塑造数字经济未来的积极伙伴。
