# 西港诈骗逃亡：4名中国人支付高额费用仍边境被捕

🚨 西港诈骗逃亡：4名中国人支付高额费用仍边境被捕
7月8日晚，泰国军警在尖竹汶府泰柬边境开展夜间巡逻时，截获并逮捕了19名非法入境人员，其中包括4名中国公民、1名印度尼西亚公民和14名柬埔寨公民。
据泰国警方调查，4名中国公民和1名印尼公民供称，他们此前在柬埔寨西哈努克市（西港）从事经营活动，后因当地持续打击电信网络诈骗的专项行动而决定离境。他们通过中介安排偷渡进入泰国，每人支付了约4800美元（约合人民币3.4万元）的费用。警方强调，上述供述仍在进一步核实中。检查发现，部分人员持有过期护照，部分则无任何身份证明。
此外，被捕的14名柬埔寨籍人员则表示，他们计划前往泰国各地务工，并为此向偷渡组织支付了约7500至8000泰铢。
目前，所有涉案人员已被移交当地警方依法处理。泰国警方表示，将继续追查幕后偷渡集团，并深入调查此案是否涉及电信网络诈骗、人口贩运等跨国犯罪活动。
