# ️ 柬埔寨：中使馆前维权行动受阻，抗议物遭没收

⚠️ 柬埔寨：中使馆前维权行动受阻，抗议物遭没收
今日，中国驻柬埔寨大使馆门前，此前在此集结维权的储户，其现场展示的立牌、标语及画像，被柬埔寨当地执法人员强行收缴。据了解，储户对执法人员的行动反应激动，并当场质问维权诉求的解决途径，但警方在完成物品收缴后随即离去，未予回应。
