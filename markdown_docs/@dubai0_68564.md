# ️ 柬埔寨『网恋』陷阱：中国大叔20天被骗3000美元

⚠️ 柬埔寨『网恋』陷阱：中国大叔20天被骗3000美元
一位在柬埔寨谋生的中国籍男子，近期遭遇了一起精心策划的网恋诈骗，在短短20天内被骗取超过3000美元。该男子50岁，长期在柬埔寨从事水电安装工作，目前失业近四月，经济状况窘迫，全靠国内亲友接济度日。
据了解，他通过社交平台Facebook结识了一名自称19岁的柬埔寨女性。对方主动示好，迅速建立感情联系，并以年龄差不重要为由，与受害人发展为“夫妻”关系。在取得信任后，该女子及其“姐妹”开始编造悲惨身世，包括母亲重病、去世及急需丧葬费等，并辅以病床照片和视频博取同情。
在不到三周的时间里，受害人累计向对方转账逾3000美元，其中部分资金甚至是他向工地老板借贷而来。然而，款项到账后，对方态度骤变，最终完全失联，电话被拉黑，消息石沉大海。
事后，受害人察觉到诸多破绽，包括对方年龄、婚姻状况说辞前后矛盾，并多次拒绝视频通话或见面要求。他已保留全部聊天记录和转账凭证，并恳请当地警方介入调查，核实涉事女性的真实身份及动机，以查明这是否为一起有组织的感情诈骗。
