# 福州警方重拳出击：捣毁‘杀猪盘’引流团伙，10人落网

🚨 福州警方重拳出击：捣毁‘杀猪盘’引流团伙，10人落网
福建省福州市公安局台江分局近日对外通报，警方于6月10日成功打掉一个专门为“杀猪盘”提供引流服务的犯罪团伙。此次行动捣毁两处窝点，共抓获10名涉案人员，涉案金额累计逾30万元人民币。
据警方介绍，该团伙手段狡猾隐蔽，通过在各大社交平台物色目标，利用虚假人设和精心设计的“话术剧本”，以嘘寒问暖、刻意套近乎的方式骗取网友信任，随后将受害人推送给上游虚假投资、刷单等“杀猪盘”诈骗团伙，从中非法抽取分成牟利，沦为电信诈骗的“帮凶”。
行动中，警方现场查获作案手机、电脑各15部，并依法扣押一大批作案工具。目前，3名核心涉案人员已被依法刑事拘留，其余7名涉案人员则被采取取保候审强制措施，案件正在持续深挖侦办中。
