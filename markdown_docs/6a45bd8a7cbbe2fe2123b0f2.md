# 人工智能、能源与安全：全球战略投资新浪潮的核心驱动力

人工智能、能源及安全领域预计将成为下一轮战略投资浪潮的核心驱动力，支撑全球经济增长与企业盈利表现。
日益激烈的全球竞争与国家安全优先级正在催生维持资本市场活跃度所需的紧迫性、资金投入和政策支持。本刊分析认为，私募市场在此过程中将发挥更广泛的作用，以弥补投资资金缺口。
在对全球经济韧性的展望中，市场普遍保持乐观态度，认为当前市场波动性仍处于可控范围。
借鉴COVID-19疫情冲击和国际贸易格局变化的经验，各国政府和企业正积极推动供应链、贸易关系及能源来源的多元化，以期为经济增长提供稳固支撑。“全球市场对快速变化的风险和新闻头条反应迅速，波动性将长期存在。当务之急是保持严谨的投资纪律，构建具备韧性和多元化的多资产组合，从而抵御短期不确定性，同时把握结构性增长趋势中涌现的长期机遇。”一位资深市场观察家对此评论道。
在此背景下，有投资机构为高净值客户勾勒出一套聚焦人工智能、能源独立及航空航天等长期结构性趋势的投资组合策略。
这种策略着重强调了科技、工业、材料和公用事业等领域的投资机遇，这些行业在创新方面的投入依然强劲，并持续青睐成长型而非价值型企业。
本季度展望还预计，美国联邦储备委员会将在2026年全年维持政策利率不变，以平衡通胀与增长风险。
深耕人工智能引领的未来
强劲的盈利表现、变现担忧的缓解以及软件抛售后具有吸引力的估值，可能为半导体、数据中心及人工智能应用企业带来新的投资机会。
布局安全与能源独立
地缘政治格局的演变正在加速全球在能源供应多元化、电气化扩张以及电网投资方面的努力。
构建多元资产组合韧性
债券、黄金、另类投资及货币多元化依然是投资组合的核心要素，并可辅以基础设施投资，以提供稳定且与通胀挂钩的现金流。
把握亚洲创新与收益机遇
杠铃策略能够捕捉亚洲创新驱动的投资周期，同时平衡收益机会，并可在地域上对中国内地、香港、新加坡和韩国进行超配。
谈及亚洲市场前景，有亚洲区投资负责人指出：“随着全球人工智能投资的加速，亚洲凭借其在半导体领域的领先地位以及在大语言模型方面的快速进展，有望从中受益。除人工智能外，投资者还可以通过债券的收益潜力，以及日本、韩国、中国内地和新加坡企业治理改革的持续改善，发现更广泛的投资机会。”
关于马来西亚市场前景，分析显示，该国第一季度GDP同比增长5.4%，相较于2025年第四季度超过6%的增速有所放缓。这种减速被视为在异常强劲季度后的常态化调整，而非动能的显著丧失。市场机构维持马来西亚2026年全年GDP增长预测在4.5%。国内通胀总体保持温和，得益于充足的补贴措施有效控制了汽油价格。
“在外部环境方面，中东地区持续的冲突凸显了作为能源净出口国的优势。尽管马来西亚仍是原油的适度净进口国，但却是天然气的主要净出口国，这使其在当前环境下处于有利地位。”
尽管面临外部波动，马来西亚国内GDP的韧性增长和稳定的企业盈利为该国股市提供了支撑。预计未来风险将趋于平衡，市场展望保持中性。
此外，报告进一步强调，地缘政治发展正在塑造一个不均衡的经济格局，因此，区分不同发行方、行业和区域的潜在赢家与输家显得尤为重要。
