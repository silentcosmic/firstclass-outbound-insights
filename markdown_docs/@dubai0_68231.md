# 乌军情报战新招：伪装“寂寞主妇”钓俄指挥官

🚨 乌军情报战新招：伪装“寂寞主妇”钓俄指挥官
战场情报战再度升级。近日，有报道披露乌克兰军事情报部门正运用“情感诱捕”策略，通过网络钓鱼行动，诱使俄军士兵泄露敏感军事信息，进而引导精准打击。
据称，乌克兰军事情报局一名代号为“谢尔希”的男性官员，在通讯软件上伪装成35岁“寂寞家庭主妇”，成功与俄军一名车臣指挥官阿赫马德建立联系。当这位“主妇”提出希望了解其生活并索要照片时，阿赫马德发送了一张营房内部的自拍照。令人震惊的是，照片背景墙上赫然出现一张清晰标注着部队位置的军营地图。乌方随即根据获取的坐标，迅速调动无人机对该地点实施了精准打击。
