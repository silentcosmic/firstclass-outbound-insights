# ️ 阿布扎比海上活动严监管：最高罚款3万迪拉姆

⚠️ 阿布扎比海上活动严监管：最高罚款3万迪拉姆
阿布扎比海事局近日发布重要通知，详细列出了33种海上违规行为，并明确了相应的罚款标准，金额从500迪拉姆至最高3万迪拉姆不等。
此举旨在进一步加强阿布扎比水域的安全与秩序管理。当局提醒所有从事水上运动或航行的人士，务必在出海前充分了解并遵守各项规定，确保手续齐全，并全程穿戴救生衣，以避免面临严厉处罚。
