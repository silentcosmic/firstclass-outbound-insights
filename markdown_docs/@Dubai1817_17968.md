# 曼谷夜店大火致27死：公共安全警钟再敲响

🔥 曼谷夜店大火致27死：公共安全警钟再敲响
7月13日凌晨，泰国曼谷北部一处夜店突发严重火灾，造成至少27人死亡、63人受伤，其中22人伤势危重。
据目击者称，起火前舞台附近曾有电路断路器冒烟并伴随爆炸声，随后浓烟迅速蔓延。消防部门耗时约30分钟控制火势，多数遇难者被发现于夜店后方卫生间内。
泰国总理已表示，火灾原因仍在调查中，当局正全力核实遇难者及伤者身份。此次事件再度引发了外界对泰国公共场所消防安全及紧急疏散措施的深切关注。
