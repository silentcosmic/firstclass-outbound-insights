# 美方制裁太子集团核心：揭露跨境金融犯罪网络

🚨 美方制裁太子集团核心：揭露跨境金融犯罪网络
美国近日对“太子集团”多名核心人物实施制裁，揭露了一个涉及全球、深度卷入在线赌博诈骗和洗钱活动的庞大网络。其中，胡小伟被指通过其在英属维尔京群岛的“未来之王有限公司”等三家资产管理公司，在全球范围内构建了复杂的公司架构。
该网络包括香港的东储证券有限公司和东融财富有限公司。消息指出，东融财富公司涉嫌接收了数百万美元的加密货币非法所得。胡小伟还通过新加坡公司，在香港持有九玺云一号、九玺云二号等多家企业，并被指派下属何浩铭、江嘉安和李煇进行管理。
此次制裁名单还包括其他多名太子集团关键人物，他们普遍拥有多重国籍或频繁变更身份，以规避监管。例如，拥有柬埔寨、中国、缅甸及瓦努阿图国籍的陈波，在至少六家涉案企业中担任董事，同时也是香港九玺云四号租赁公司及柬埔寨CCU商业银行的大股东。邱伟仁（又名邱东）则被指是诈骗园区的重要幕后投资者，他拥有圣基茨和尼维斯及塞浦路斯国籍，长期常住英国伦敦。
其他被制裁者还包括戴安（又名戴德文），他在太子集团内部担任高层，并曾任职于此前已被美方制裁的太子寰宇地产集团；方智振（又名“KOFI”）被认为是网络博彩诈骗在线支付网关的核心操盘手，负责洗白非法收益；以及罗平华（Brendon Luo），作为核心投资人参与了大规模网络欺诈和诈骗园区的运作。这些制裁行动凸显了国际社会打击跨境金融犯罪和洗钱的决心。
