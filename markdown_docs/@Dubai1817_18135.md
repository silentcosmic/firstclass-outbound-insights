# 东帝汶重拳打击网络诈骗：已逮捕342人，行动不停

⚡ 东帝汶重拳打击网络诈骗：已逮捕342人，行动不停
东帝汶总理夏纳纳·古斯芒于7月16日表示，尽管政府已采取多项措施，当地网络诈骗活动仍呈现增长态势。他强调，政府将持续加强情报和安全力量，以更有效打击相关犯罪团伙。
古斯芒指出，东帝汶已具备打击网络诈骗的政策和法律框架，当前首要任务是提升情报收集、案件调查和执法能力，确保警方能迅速锁定并抓捕涉案人员。近期，东帝汶警方已连续查获多个疑似诈骗窝点，涉及中国、印度尼西亚和柬埔寨等多国公民，相关嫌疑人正依法接受调查处理。
在近期的多轮行动中，东帝汶安全部门累计抓获342名涉嫌参与网络诈骗等违法活动的人员。古斯芒总理重申，即便已逮捕大量嫌疑人，打击行动也绝不会停止。他要求各安全执法部门必须继续强化情报工作，持续追查涉案人员，彻底清除在东帝汶境内活动的所有网络诈骗团伙。
