# ️ 马来西亚女子不当言论引发跨国争议并致歉

⚠️ 马来西亚女子不当言论引发跨国争议并致歉
近日，一名马来西亚女子在中国旅游期间发布视频，其中包含“中国人不洗澡、很臭”等不当言论，迅速在网络上引发广泛批评。该视频被指带有歧视和冒犯性，激起了中国网民的强烈不满。
据市场消息，涉事者诺阿希金事后已公开道歉，承认其行为过火，并表示愿意为相关言论承担责任，强调并非刻意安排或有意挑起任何跨国冲突。此事件再次提醒公众，在跨文化交流中，言行需格外谨慎。
