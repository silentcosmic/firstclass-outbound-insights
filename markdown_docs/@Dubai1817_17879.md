# 越南捣毁跨国赌博集团：缴获百万USDT与黄金

🚨 越南捣毁跨国赌博集团：缴获百万USDT与黄金
越南警方近日宣布成功侦破一起依托OKWIN生态系统运营的跨国网络赌博大案。该犯罪网络集非法买卖银行账户、资金流转及网络赌博于一体，目前已有42名嫌疑人被依法起诉。
调查显示，该集团通过冒充电信运营商，以“数据清理”为名非法获取大量公民个人信息及身份证照片。随后利用这些资料在全国范围内开设银行及电子钱包账户，转售给境内外犯罪组织，用于网络赌博、诈骗等活动。主犯河进达（Hà Tiến Đạt）及其同伙杨鸿福（Dương Hồng Phúc）负责组织实施，在数月内便非法开设并出售了逾5000个银行账户，非法获利近60亿越南盾。
进一步侦查揭示，OKWIN是一个全球性的在线博彩平台，旗下管理着包括32win、Kuwin、789win、Okfun在内的60多个赌博网站及游戏平台。今年3月，越南警方在西宁、河内、胡志明市等多个省市同步展开收网行动，共传唤调查29名涉案人员。
行动中，执法人员查获了42两黄金、74亿越南盾现金、100万枚USDT，以及近百台电脑、手机等作案工具。河内警方指出，此次行动成功斩断了长期为网络诈骗、赌博及高利贷等犯罪活动提供银行账户的重要链条，有效切断了犯罪资金的流转渠道。案件仍在深入调查中。
