# 阿联酋脱离欧佩克后，石油出口量创历史新高

🔥 阿联酋脱离欧佩克后，石油出口量创历史新高
最新数据显示，阿联酋原油和凝析油出口量在6月份达到历史新高，平均每日约370万桶。这一数字显著高于中东冲突前每日310万至330万桶的水平，据船舶追踪机构Kpler和Vortexa的数据揭示。
此前，阿联酋已于5月1日正式退出石油输出国组织（欧佩克）及“欧佩克+”机制。此举赋予阿联酋更大的自主权，可根据市场动态自主决定增产时机与幅度，从而有望在高油价周期中进一步扩大其全球市场份额。
