# 迪拜华人遭遇绑架勒索：警方已介入调查

🚨 迪拜华人遭遇绑架勒索：警方已介入调查
据市场消息，一名海外华人于2026年6月28日在迪拜遭遇诱骗绑架及勒索。受害者表示，其在正常出行途中被陌生人员诱骗上车，随后即失去人身自由，手机和随身物品被强制没收。
绑匪对受害者进行言语恐吓，并强迫其交出手机密码，转走约2万美金，同时还胁迫其借贷转账虚拟货币。所幸，另一名受害者成功发出求救信号，绑匪在慌乱中将受害者释放并驾车逃离。
受害者已向当地警方报案。警方初步调查显示，绑匪曾租用车辆，并有监控记录。据称，两名主要嫌疑人及其雇佣的两名越南同伙已逃往越南。受害者目前正寻求公众协助，以期将涉案人员绳之以法。此事件再次提醒出海人员务必提高警惕，加强个人安全防范。
