# 柬埔寨CCU商业银行因太子集团关联遭美制裁

🚨 柬埔寨CCU商业银行因太子集团关联遭美制裁
美国财政部于当地时间6月23日宣布，对与柬埔寨太子集团相关的9名个人和26家企业实施新一轮制裁。
此次制裁名单中，柬埔寨CCU商业银行赫然在列。美方披露，该银行主要持有人之一陈波，曾担任多家太子系企业董事，是此次关联制裁的关键因素。根据制裁规定，CCU商业银行在美国境内或由美国人控制的资产将被冻结，同时美国个人和机构原则上不得与其进行任何交易。
