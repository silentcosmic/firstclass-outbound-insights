# 柬埔寨波贝捣毁跨国诈骗窝点，63人落网

🚨 柬埔寨波贝捣毁跨国诈骗窝点，63人落网
柬埔寨卜迭棉芷省波贝市警方近日联合多部门采取突击行动，成功捣毁了4处跨国诈骗窝点，并逮捕63名嫌疑人。此次行动由卜迭棉芷省副省长与省初级法院副检察官牵头，多省联合工作组于7月15日对市内目标地点实施了突击搜查。
行动中共逮捕60名泰国籍和3名中国籍嫌疑人，现场缴获电脑、手机以及一批伪造的泰国警用制服和徽章等大量作案工具。
目前，所有涉案人员已被拘留，相关犯罪窝点也已被查封，案件正在进一步调查中。
