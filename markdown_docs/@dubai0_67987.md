# 泰国重拳出击网络诈骗：9个月封停35万账户

🚨 泰国重拳出击网络诈骗：9个月封停35万账户
6月22日，泰国政府召开专题新闻发布会，正式将跨境网络诈骗及“代持公司”等灰色资本犯罪列为国家级重点打击对象。此举标志着泰国对此类犯罪活动的零容忍立场进一步升级，并公布了过去9个月的阶段性成果。
在副总理兼内政部长阿努廷的领导下，泰国政府联合国家警察总署、反网络诈骗中心等多个部门，持续开展专项整治。行动重点打击电信诈骗、虚假投资、跨境洗钱及非法代持企业，旨在全面切断犯罪链条。
数据显示，过去9个月内，执法部门已累计封停约35.1万个涉嫌用于违法犯罪活动的银行账户，冻结涉案资产总额超过36亿泰铢。同时，成功协助862名受害者止损，并实时拦截约8200万泰铢被骗资金。此外，警方在全国范围内逮捕超过2.9万名涉案人员，并锁定70余名跨国犯罪网络核心成员。
泰国政府指出，鉴于网络犯罪日益呈现跨境化、国际化趋势，泰国正积极加强与国际刑警组织、美国联邦调查局及东盟各国执法机构的合作，以提升打击效率，共同应对跨国网络犯罪威胁。
