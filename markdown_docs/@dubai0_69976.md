# 马来西亚A8体育陷管理泥潭，员工薪资停滞人心惶惶

🚨 马来西亚A8体育陷管理泥潭，员工薪资停滞人心惶惶
近日，有市场消息指出，位于马来西亚的A8体育公司正面临严重的内部管理危机与员工信任挑战。据匿名员工透露，该公司从管理层到团队运作均呈现出高度分散、缺乏协同的状态，专业能力和管理水平令人质疑。
多位员工反映，公司在招聘过程中所做的岗位、待遇及发展方向承诺，与实际入职后的情况存在显著差异。内部管理混乱、沟通不畅，制度朝令夕改，导致问题发生时责任难以界定，最终常由员工承担。
更令人担忧的是，公司近期内部动荡加剧。消息称，上月公司人事与人力资源相关人员之间发生严重事件，后续或将涉及法律程序，导致公司内部人心惶惶。此外，公司此前安排部分在职员工整体转至亚美尼亚办公，但到了发薪日，却出现数百名员工的薪资结算和发放严重滞后，引发广泛不满。这种连基本工资都无法准时发放的状况，进一步动摇了员工对公司管理能力的信心。
该爆料呼吁潜在求职者审慎评估，避免陷入类似的职场困境。
