# 新加坡金管局与中国证监会签署人员交流协议，深化资本市场监管合作

新加坡金融管理局（MAS）与中国证券监督管理委员会（中国证监会）已签署一项人员交流协议，旨在全面深化两国资本市场的监管合作关系。
该协议的签署，是新加坡金管局与中国证监会第十届年度监管圆桌会议的重要成果。
在此次会议中，两国监管机构及行业代表共同探讨了各自股权市场的最新发展，其中包括新加坡的股权市场发展计划，以及中国科创板与创业板的改革举措。
此外，双方还就金融基础设施韧性、市场不当行为、技术在执法中的应用，以及数字资产的监管策略等议题进行了深入交流。
行业参与者分享了多项跨境合作倡议的最新进展，尤其提到了新中两国之间的ETF产品互联互通机制及指数合作项目。
这项人员交流协议旨在进一步强化两国监管机构间的机构联系，并增进彼此的相互理解。
新加坡金管局金融监管副局长何升馨表示：“第十届监管圆桌会议标志着新加坡金管局与中国证监会长久合作关系中的一个重要里程碑。我期待未来能进一步深化监管合作，推动资本市场互联互通，以支持新加坡和中国市场的持续发展。”
