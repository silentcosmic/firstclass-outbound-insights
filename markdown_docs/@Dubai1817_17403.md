# 伊美谈判突破：石油制裁豁免，冻结资产解冻

⚡ 伊美谈判突破：石油制裁豁免，冻结资产解冻
伊朗方面周一宣布，经过卡塔尔和巴基斯坦斡旋的18小时密集谈判，伊朗与美国已就一份协议文件达成共识。伊朗外交部发言人巴加埃透露，该协议文本将由调解方发布。伊朗外长阿拉格齐更进一步表示，针对伊朗石油的制裁已获豁免，部分海外冻结资产也已解冻。
阿拉格齐在社交平台X上发文指出，一项旨在“重建与发展伊朗”的重大计划已启动。据伊朗半官方通讯社报道，德黑兰已与卡塔尔签署一份关于落实资产解冻事宜的新谅解备忘录，并已拟定关于暂时豁免美国对伊朗石油及衍生品制裁的协议草案。
然而，此次谈判过程并非一帆风顺。伊朗外交部披露，美国总统特朗普此前发表的威胁性声明一度导致伊朗拒绝继续四方会谈，凸显了谈判的脆弱性。尽管如此，各方仍就设立冲突降级协调机制以结束黎巴嫩军事行动达成一致，阿拉格齐称此为“首个真正的考验”。美国总统特朗普此前曾警告伊朗停止在黎巴嫩的“代理人”行动。
与此同时，以色列方面表示尚未收到美方要求其从黎巴嫩南部撤军的请求，但正考虑小规模撤军，并强调保留“黄线”以内控制区作为底线，以保护北部居民并保留谈判筹码。
