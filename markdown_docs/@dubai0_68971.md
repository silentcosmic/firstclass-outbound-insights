# 澜湄执法：中泰老合作遣返3名逃犯

🚨 澜湄执法：中泰老合作遣返3名逃犯
在澜湄执法合作中心的协调下，跨境打击诈骗犯罪取得显著成效。近期，泰国警方成功抓获一名中国警方通缉的诈骗嫌犯，并已于6月29日将其遣返回中国。随后，老挝警方也迅速行动，于7月2日将两名涉嫌诈骗的中国籍逃犯遣送回国。此次联合行动，再次彰显了澜湄区域各国在维护地区安全、打击跨国犯罪方面的坚定承诺与高效协作。
