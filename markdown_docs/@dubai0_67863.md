# 泰国突查非法挖矿：窃电近900万泰铢

⚡ 泰国突查非法挖矿：窃电近900万泰铢
泰国警方近日联合地方电力部门，在乌汶府展开专项执法行动，突击搜查了4处非法加密货币挖矿窝点。行动中，警方共查获45台比特币矿机，并确认这些团伙长期通过非法窃电方式运营，造成巨额国家电力资源损失。
调查揭示，涉案团伙通过私接线路、绕过计费系统等手段，为高耗能矿机非法供电，以降低成本谋取暴利。初步估算，此番窃电行为已导致地方电力局经济损失高达898万泰铢（约合24.5万美元）。警方还在一处窝点发现，约20台矿机疑似因消息泄露已提前被转移，现场仅剩遗留设备及相关痕迹。
警方强调，非法挖矿不仅构成窃电犯罪，更可能威胁当地电网稳定。目前，相关部门正扩大调查，追查涉案人员及幕后网络，并核查是否存在更多非法窝点。
