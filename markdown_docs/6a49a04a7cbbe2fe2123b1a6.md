# 新中资本市场监管合作深化：新加坡金管局与中国证监会签署人员交流协议

新加坡金融管理局（MAS）与中国证券监督管理委员会（CSRC）近日签署了一项人员交流协议，旨在全面深化两地资本市场间的监管联系与合作。
这项关键协议是在新加坡金管局与中国证监会共同举办的第十届年度监管圆桌会议期间正式签署的。
会议期间，两地监管机构代表与业界资深人士深入探讨了双方股票市场的最新进展。议题涵盖了新加坡的股权市场发展计划，以及中国科创板与创业板的改革动向。
此外，讨论还触及了金融基础设施的韧性、市场不当行为的防范、技术在监管执法中的应用，以及对数字资产的创新监管策略。
业界参与者亦分享了多项跨境合作倡议的最新进展，其中包括新中两地交易所交易基金（ETF）产品互联互通机制，以及在指数开发方面的紧密协作。
这项新签署的人员交流协议，核心目标在于强化双方机构间的制度性联系，并显著提升两地监管机构间的相互理解与协同效率。
新加坡金管局金融监管副局长何Hern Shin在会议上指出：“第十届监管圆桌会议标志着新加坡金管局与中国证监会长期伙伴关系中的一个重要里程碑。我们期待进一步加强监管合作，推动资本市场互联互通，以支持新加坡和中国市场持续健康发展。”
