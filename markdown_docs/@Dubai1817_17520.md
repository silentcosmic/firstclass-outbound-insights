# 中老警务合作升级：78名涉赌中国公民被移交

🚨 中老警务合作升级：78名涉赌中国公民被移交
老挝警方与中方开展跨境执法合作，正式向中国公安机关移交了78名涉嫌参与赌博活动的中国籍人员。此举标志着中老两国在打击跨境违法犯罪方面的协作持续深化。
据老挝《琅勃拉邦省警察杂志》披露，此次被移交人员均于此前在琅勃拉邦省首府巴邦市的班蓬赛村及班普拉巴特泰村落网。老挝外国人管制警察联合经济保护警察，经过严密侦查部署后，成功捣毁涉赌窝点，共抓获78名中国公民，其中包括75名男性、5名女性，以及3名非法入境者。
移交仪式在中老磨丁国际边境口岸与中国云南磨憨国际口岸同步举行，琅勃拉邦省警察局副局长Pho Somboun Thanavanh等两国代表共同参与。此次引渡依据《外国人出入境管理法》及《老挝人民民主共和国公安部与中华人民共和国公安部关于设立老中警务合作中心的协定》实施，为双方跨境执法合作提供了明确的法律框架。
行业观察人士指出，近年来中老两国警务协作持续加强，在联合打击跨境赌博、电信诈骗等违法犯罪活动方面力度不断加大，共同维护边境地区安全稳定，保障两国正常的社会秩序和人员往来安全。
