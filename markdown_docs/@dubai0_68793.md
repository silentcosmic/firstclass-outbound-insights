# ️ 印度僧侣深信能飞，坠崖身亡引警示

⚠️ 印度僧侣深信能飞，坠崖身亡引警示
近日，印度安得拉邦发生一起悲剧性事件。一名印度教僧侣因深信自己拥有神力可腾空飞翔，在完成特定宗教仪式后，试图从悬崖边缘“飞越”，不幸失足坠崖身亡。
据当地消息及目击者描述，该僧侣在事发前曾公开表示，相信自己能凭借修行之力腾空飞行。他在崖顶完成祈祷仪式后，走向悬崖边缘，随后失足滑落，当场死亡。
