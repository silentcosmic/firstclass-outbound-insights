# 马来西亚数字银行：普惠金融实践与盈利之道——专访GXBank副首席执行官Hildah Hamzah

数据显示，GXBank的小微企业借款人中，高达87%此前从未获得过银行贷款。GXBank副首席执行官Hildah Hamzah将深入剖析，该数字银行如何触达这些传统金融服务缺口人群，并同时实现可持续盈利。
作为马来西亚首家数字银行GXBank的副首席执行官兼首席运营官，Hildah Hamzah女士是该行的第三号员工，全程参与并推动团队从初创发展至约400人的规模。GXBank的业务核心，在于服务马来西亚收入最低的40%家庭（B40）和中等收入群体（M40）。在本次深度访谈中，Hamzah女士将与我们探讨数字银行是否真正兑现了其普惠金融的承诺，以及GXBank的具体实践。
