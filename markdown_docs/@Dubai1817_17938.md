# 泰警捣毁5大网赌平台：1.4亿泰铢流水曝光

🚨 泰警捣毁5大网赌平台：1.4亿泰铢流水曝光
7月10日，泰国警方在代号“粉碎暴富梦”的专项行动中取得重大突破。曼谷第二警区联合廊曼警署及巴吞他尼府警方，同步突击搜查曼谷及巴吞他尼府共12处目标地点，成功捣毁了5个大型网络赌博平台。
此次行动共逮捕了13名涉案人员，其中11人负责通过银行卡进行赌资转移与提取，另有2人则担任赌博网站管理员。警方透露，这5个平台——Black-888、gunx789、moscow-66、luckyheng66及spd-66.com，累计资金流水已超过1.4亿泰铢。
行动中，警方查扣了包括13部手机、5台电脑、17张银行卡在内的大量涉案物品，并扣押了3辆汽车、4辆摩托车、27万泰铢现金、10件黄金饰品以及1栋涉案房屋，查扣资产总价值约612.6万泰铢。
值得关注的是，此次大规模打击行动的线索竟源于今年4月16日曼谷廊曼区的一起街头枪击案。警方在调查一名24岁涉案男子的过程中，发现其经济来源异常，消费水平远超正常收入，持有大量黄金饰品并长期购买名牌商品。顺着资金流向深入调查，最终锁定了这5个网络赌博平台及其背后的犯罪团伙。
目前，所有落网人员及涉案证物已移交廊曼警署依法处理。警方表示将继续深挖，追查涉案资金流向、幕后组织者及其他在逃成员，以期扩大战果。
