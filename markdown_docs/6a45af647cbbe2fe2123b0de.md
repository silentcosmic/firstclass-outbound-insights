# Visa、Stripe等逾百家巨头联袂，共建开放美元稳定币新生态

由维萨（Visa）、万事达卡（Mastercard）和Stripe等逾百家机构组成的“开放标准”（Open Standard）联盟，日前宣布推出一款名为Open USD（OUSD）的新型稳定币。
该数字资产将向所有参与企业共享其储备金收益。
“开放标准”联盟将赋能企业，使其在铸造和赎回该稳定币时，无需支付任何费用，亦不受交易量限制。
Open USD将由一个独立的机构负责管理，其治理权由各合作公司共同享有，此举旨在取代传统由单一中心化发行方主导的模式。
此次合作倡议汇集了包括贝莱德（BlackRock）、渣打银行（Standard Chartered）在内的金融机构，谷歌（Google）等技术巨头，以及Coinbase等加密货币企业。
加入该联盟的企业将把此稳定币用作核心支付资产，并能获得相应的技术支持，同时根据其采纳和使用情况获取收益。“现有稳定币确有其独到优势，但若要实现大规模商业应用，企业需要的是一种开放、低成本、高吞吐量、广泛可及且符合自身利益的解决方案。我们很高兴能汇聚超过140家企业，共同推出Open USD。”
