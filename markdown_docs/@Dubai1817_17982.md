# 港人赴台涉诈骗案激增：‘车手’被捕三年暴涨七倍

🚨 港人赴台涉诈骗案激增：‘车手’被捕三年暴涨七倍
近年来，香港居民赴台湾涉嫌参与电信诈骗的案件显著增加，其中尤以充当诈骗集团“车手”的情况最为突出。数据显示，2025年香港人在台湾涉诈骗案件达262宗，2026年上半年已录得185宗。值得关注的是，担任诈骗集团“车手”的案件在2025年有180宗，2026年上半年已攀升至132宗，相较三年前此类案件数量暴增约七倍。
“车手”是诈骗链条中的关键环节，主要负责直接收取受害人现金或从银行ATM提取诈骗所得。台湾法律界人士指出，台湾司法实践中普遍采用“不确定故意”原则，即认为具正常认知能力者难以相信高薪收款等异常工作属于合法行为，因此“不知情”通常难以构成有效的抗辩理由。
此外，行业观察人士发现，有诈骗集团利用投资诈骗手法，先诱骗香港人成为受害者，随后再以“偿还损失”或“赚取佣金”为诱饵，将其诱骗至台湾充当“车手”，导致受害者最终沦为犯罪嫌疑人。台湾警政署已发出提醒，告诫公众切勿轻信“高薪、轻松赚钱”的工作机会，更不应以观光旅游名义赴台从事不明工作，以免误入诈骗陷阱并承担刑事责任。
