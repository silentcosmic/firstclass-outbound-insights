# 勐拉移交40名涉诈嫌犯：跨境打击电信诈骗再升级

🚨 勐拉移交40名涉诈嫌犯：跨境打击电信诈骗再升级
7月10日，缅甸勐拉第四特区警察局通过勐拉—打洛口岸，依法向中方移交了40名涉嫌电信网络诈骗的犯罪嫌疑人。此次行动不仅涉及人员移交，更同步完成了所有涉案电子设备与作案工具的全链条闭环交接，展现了中缅两国在联合打击跨境犯罪方面的深度合作。
