# 东盟携手伙伴打击跨国犯罪：AI技术将升级移民管理

🚨 东盟携手伙伴打击跨国犯罪：AI技术将升级移民管理
第29届东盟移民局长及外交部领事司长会议近日在柬埔寨暹粒省落下帷幕。会上，东盟成员国与中国、日本、澳大利亚、韩国等伙伴国就“数字化转型与人力创新”议题达成多项共识，承诺将通过加强信息共享与人才培养，携手打击人口贩卖、毒品及网络诈骗等跨国犯罪。
会议一致通过了柬埔寨关于设立“东盟移民培训中心”的提议，该中心未来将设在柬埔寨。柬埔寨内政部移民总局局长宋威斯那强调，在工业4.0时代，各国将积极引入人工智能（AI）技术，以升级移民和签证程序，更有效地应对高科技新型网络犯罪挑战。
