# 出海洞察：Horizon Media 携手 Intent IQ，在数据信号受限中实现55%的全新用户增量

增量增长是每一位营销负责人追求的核心目标，然而，如何有效论证并量化其真实成效，则始终是一个复杂的命题。在2026年戛纳国际创意节上，我们深入探讨了这一议题。Intent IQ 战略与合作高级副总裁 Fabrice Beer-Gabel 先生与 Horizon Media 平台执行副总裁兼负责人 John Koenigsberg 先生展开了一场深度对话，共同剖析了在当前数据信号日益受限的严峻背景下，受众拓展（audience amplification）策略如何才能真正转化为可验证的增量增长。
