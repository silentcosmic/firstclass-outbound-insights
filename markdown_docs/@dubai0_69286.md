# 以色列证实：铁穹系统首次部署至阿联酋

🚨 以色列证实：铁穹系统首次部署至阿联酋
以色列交通部长米里·雷格夫于7月5日向以色列媒体证实，在美以伊冲突爆发初期，以色列曾向阿联酋部署一套“铁穹”防空系统。
此举标志着“铁穹”系统首次部署到以色列和美国以外的国家，凸显了地区安全合作的深化，也为中东地缘政治格局带来了新的观察点。
