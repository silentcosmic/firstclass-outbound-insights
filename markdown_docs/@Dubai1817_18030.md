# 菲律宾：中国公民遭同胞绑架，NBI成功解救人质

🚨 菲律宾：中国公民遭同胞绑架，NBI成功解救人质
菲律宾国家调查局（NBI）于7月13日通报了一起中国公民绑架案的最新进展。该局成功解救了一名在马尼拉大都会地区被非法拘禁的中国籍男子，并逮捕了5名涉案的中国籍嫌疑人。据悉，绑匪曾向受害者家属索要200万菲律宾比索（约合人民币25万元）赎金。
根据NBI披露的信息，受害者为一名中国籍男子，被多名同胞非法拘禁。目前，仍有3名涉案人员在逃，执法部门正全力追捕。NBI表示，此案仍在深入调查中，将进一步查明犯罪团伙的组织分工及是否涉及其他犯罪行为。
