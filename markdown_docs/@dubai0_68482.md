# ️ 阿联酋国防部痛失一员：士兵训练中不幸殉职

⚠️ 阿联酋国防部痛失一员：士兵训练中不幸殉职
阿联酋国防部近日发布声明，确认士兵伊萨·古卢姆·阿尔·巴卢希（Issa Ghuloom Al Balushi）于周六（6月27日）在国内执行训练任务时不幸牺牲。
国防部通过其社交媒体平台向这位在任务中献身的士兵家属致以最深切的哀悼与慰问。
