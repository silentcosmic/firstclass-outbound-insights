# 阿联酋拨款千万美元，紧急驰援委内瑞拉震区

🚨 阿联酋拨款千万美元，紧急驰援委内瑞拉震区
委内瑞拉北部沿海地区近日遭遇两场强震，导致灾情严重，死亡人数已逼近千人。目前，当地民众与国际救援队正争分夺秒，在废墟中搜寻幸存者。
为响应国际人道主义呼吁，阿联酋已通过阿联酋援助署（UAE Aid Agency）迅速启动紧急人道主义援助计划，并承诺拨付1000万美元，用于支持受灾民众的救助与重建工作。
