# 法国诈骗头目泰国落网：涉案75亿铢惊动国际刑警

🚨 法国诈骗头目泰国落网：涉案75亿铢惊动国际刑警
近日，一名被国际刑警组织列入红色通缉令的法国籍网络诈骗头目多甘（Dogan，38岁）在泰国佛丕府一处别墅内被警方成功抓获。该团伙涉案金额高达75亿泰铢（约合数亿欧元），受害者超过900人。
调查显示，多甘是与阿联酋黑帮勾结的五人犯罪团伙核心，自2022年起设立虚假“数字金融公司”，以高额回报为诱饵，专门诱骗法国、比利时、瑞士及新西兰等国的土耳其裔移民进行投资。
值得注意的是，泰国近期已连续抓获多名来自中、日等国的跨国毒枭与诈骗头目。国际专家对此指出，泰国正日益成为外籍罪犯的藏匿避风港，呼吁当局必须正视并根除这一日益严峻的社会问题。
