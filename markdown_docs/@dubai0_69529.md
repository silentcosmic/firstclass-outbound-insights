# 中国"网络身份证"周年：4000万人申领，530+应用接入

⚡ 中国"网络身份证"周年：4000万人申领，530+应用接入
自《国家网络身份认证公共服务管理办法》施行一年以来，中国数字身份体系建设取得显著进展。截至目前，已有4000万人申领了数字身份，累计完成2.8亿次认证，并成功接入530余个各类App、小程序及网站。国家网络身份认证App下载量已达9000万次，用户基础持续扩大。
该数字身份服务已广泛覆盖多个核心领域。微信、QQ、淘宝、京东、抖音等主流互联网平台均已支持一键登录功能。同时，国家政务服务平台、17个省市级政务服务应用、六大国有银行以及北京协和医院等重要政务、金融及医疗机构也已陆续完成接入，此举显著提升了用户在多场景下的认证便利性与安全性。
该服务核心在于通过"可用不可见"机制，有效解决了传统实名认证中存在的"实名不实人"问题，实现了在实名认证中对用户隐私的有效保护。未来，该数字身份系统还将持续扩展应用场景，预计将接入铁路12306等更多民生服务平台。
