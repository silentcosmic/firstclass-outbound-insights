# 柬中深化执法合作：严打网络赌诈跨国犯罪

🚨 柬中深化执法合作：严打网络赌诈跨国犯罪
近日，柬埔寨国家警察总署副总监兼安全总局局长迪伟杰，在国家警察总署会见了中国云南省公安厅副厅长张庆星一行。双方围绕加强执法合作，共同打击跨国犯罪议题进行了深入会谈。
会谈重点聚焦于深化打击网络赌博、电信网络诈骗及其他相关跨境违法犯罪活动。此次高级别会晤进一步巩固了柬中两国执法部门在维护区域安全秩序、有效防范跨境犯罪方面的紧密合作关系。
