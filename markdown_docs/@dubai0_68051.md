# 西港省设专线打击网络诈骗，鼓励公众积极举报

📣 西港省设专线打击网络诈骗，鼓励公众积极举报
近日，柬埔寨西哈努克省政府正式对外公布了一项新的举报热线，旨在加强公共服务、维护社会治安，并重点打击日益猖獗的网络诈骗犯罪活动。
根据省政府通报，公众若对上述领域有任何疑问或掌握相关线索，特别是涉及网络诈骗的违法行为，可直接拨打专线电话092 557 117进行举报。省政府强调，所有提供咨询和线索的个人信息都将受到严格保密，以确保举报人的安全和权益。此举彰显了当地政府对打击网络犯罪、优化营商环境的决心。
