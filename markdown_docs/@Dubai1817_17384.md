# 伊朗关闭霍尔木兹海峡，回应以黎冲突与美国失信

🚨 伊朗关闭霍尔木兹海峡，回应以黎冲突与美国失信
伊朗紧急宣布关闭连接波斯湾和阿曼湾的战略航运通道——霍尔木兹海峡。此举是对以色列持续打击黎巴嫩的直接回应。据黎巴嫩国家通讯社报道，以色列周六对黎巴嫩多地发动空袭，造成至少38人死亡，进一步加剧了地区紧张局势。
此外，伊朗方面通过法尔斯通讯社表示，关闭海峡的另一重要原因在于“美国公然违背承诺、违反协议”，未能落实此前关于所谓“战争结束协议第一条款”的相关内容。这一系列行动无疑将对全球能源市场和地区稳定产生深远影响。
