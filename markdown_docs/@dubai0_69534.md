# 阿联酋枪杀案宣判：海湾男子被判死刑

🚨 阿联酋枪杀案宣判：海湾男子被判死刑
哈伊马角刑事法院近日宣判了一起震惊阿联酋的重大枪杀案。一名海湾国家男子因蓄意谋杀罪被判处死刑，该案件涉及一名66岁的母亲及其两名女儿被枪杀，导致一个家庭几乎覆灭。
据法院7月8日庭审宣判，该男子被认定犯有蓄意谋杀、谋杀未遂及发出死亡威胁等多项罪名。同案中，其儿子也因谋杀未遂和威胁受害者家庭另一成员被判处有罪，合并监禁九个月并处以1万迪拉姆罚款。法院同时下令没收涉案证据，包括一辆挂有哈伊马角车牌的车辆。根据阿联酋法律，死刑判决在生效前仍需经过完整的上诉司法程序。
这起致命枪击案发生在哈伊马角一处居民区，当局调查后确认，争执的导火索据称源于一起交通纠纷。
