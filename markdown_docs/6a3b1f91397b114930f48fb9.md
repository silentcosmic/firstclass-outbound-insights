# 泰国权衡收紧BNPL监管 小额消费信贷风险引关注

随着分期付款服务日益深入日常消费领域，泰国正在审慎权衡，准备收紧“先享后付”（BNPL）的监管规定。
相关机构正在为BNPL服务提供商筹备监管措施。监管机构担忧，线上信贷的便捷获取可能加剧家庭债务压力。
泰国中央银行行长Vitai Ratanakorn曾发出警示，当前消费者已将BNPL应用于包括餐饮在内的日常小额支出，而非仅限于高价商品的购买。
他指出，这种趋势可能助长人们在尚未具备偿付能力时即行消费的习惯。
目前，泰国家庭债务占国内生产总值（GDP）的比重约为87%，位居亚洲地区高位。其中大部分债务与消费紧密相关，使得借款人更容易受到收入冲击的影响。
Vitai估计，去年BNPL账户数量已增至约600万，几乎是2021年水平的十倍。
这一显著增长主要得益于电子商务、移动支付以及结算选项的普及，这些技术使消费者能够几乎即时地将购买转化为分期付款计划。
短期分期付款方案常以“零利率”作为宣传，然而，期限较长的分期计划年利率可能高达25%。
央行目前正在起草针对通过在线平台提供分期贷款的BNPL服务商的规则。
潜在措施可能涵盖年龄和收入限制、对适用商品的范围规定、最低购买金额以及费用上限等。这些新规预计将于今年年底前发布。
此次监管政策的推进，正值泰国家庭面临新的压力之际。
去年，泰国家庭月均收入下降2.5%至28,308泰铢，这是自2019年以来的首次下滑。与此同时，家庭支出也下降了5.4%。
经济学家分析认为，BNPL模式固然能够在一定程度上缓解短期的现金流压力，但其潜在风险在于可能助长不必要的消费，并将还款负担推迟至未来。
