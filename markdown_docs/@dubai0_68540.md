# 马来西亚沙巴：涉灰产人员街头裸体扰序，恐遭遣返重罚

🚨 马来西亚沙巴：涉灰产人员街头裸体扰序，恐遭遣返重罚
市场消息指出，马来西亚东部沙巴州近日发生一起公共秩序事件。一名涉嫌从事灰色产业的人员，被当地民众目击赤裸上街，并做出指挥交通的异常举动。
此行为已引起当地关注，预计将迅速招致执法部门介入。根据马来西亚相关法律及移民规定，该涉事人员恐将面临被举报、强制遣返出境以及相应的法律制裁。
