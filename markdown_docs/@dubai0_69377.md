# 伊朗警告：再遭袭击将关闭霍尔木兹海峡

🚨 伊朗警告：再遭袭击将关闭霍尔木兹海峡
据最新消息，伊朗方面近日发出严厉警告，声称如果其领土或利益再次遭受袭击，将采取行动关闭具有战略意义的霍尔木兹海峡。
霍尔木兹海峡是全球重要的石油运输要道，连接波斯湾与国际水域，其关闭将对全球能源供应和国际贸易造成严重冲击，加剧地区紧张局势。
