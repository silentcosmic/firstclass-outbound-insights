# 世界杯期间“鸭脖推广”遭员工控诉：涉嫌欺诈与内部腐败

🚨 世界杯期间“鸭脖推广”遭员工控诉：涉嫌欺诈与内部腐败
近日，某匿名消息源在公开社交群组爆料，详细揭露了“兰卡鸭脖推广”项目在世界杯期间存在的严重内部问题，引发广泛关注。
爆料指出，该项目涉嫌对新入职员工进行虚假宣传，承诺世界杯期间可月入数万并享受优厚福利，但实际却隐瞒了高达2000U的落地赔付风险。有员工质疑，公司此举或旨在通过新人的加入来填补资金缺口，而非真心发展业务。
此外，爆料还控诉公司在员工福利方面极度吝啬，甚至出现多人分享一个水果的情况。更严重的是，内部管理层被指控挪用员工资源和项目经费，用于“刷数据”以制造虚假业绩，而基层员工却未获得任何回报，形容为“吃肉不给汤，还砸锅碗”。
爆料者进一步指出，项目内部弥漫着浓厚的赌博风气，管理层在工作时间热衷研究球赛投注，而非专注于业务推广，导致整个部门陷入一种投机氛围，严重影响了正常工作秩序与业绩产出。这份控诉描绘了一个充满欺诈、腐败和管理混乱的内部环境，引发了对该项目运营合规性及员工权益保障的深切担忧。
