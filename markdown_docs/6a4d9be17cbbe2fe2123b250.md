# 微软进行全球性组织调整，裁员近5000人并澄清非AI直接替代岗位

科技巨头微软启动一项组织调整计划，将裁减约4800个职位，相当于其全球员工总数的2.1%。此轮调整核心在于优化资源配置，以更好地聚焦其战略核心业务。
据公司执行副总裁兼首席人事官艾米·科尔曼（Amy Coleman）在内部备忘录中披露，此次人员精简将主要集中于其商业销售与Xbox游戏部门。
科尔曼指出，面对不断演变的客户需求与技术格局，此次裁员是微软整体战略调整中的一环，旨在更灵活地优化人力资源、投资方向及运营模式。
微软方面特别澄清，此次裁撤的岗位并非直接由人工智能技术所替代。然而，人工智能的确正在深刻重塑公司内部的各项工作流程与协作方式。
微软承诺将持续加大对员工培训的投入，特别是在人工智能相关技能培养方面。
过去一年中，微软已成功为逾4000名员工重新分配了新岗位，其中本月便有500人完成转岗。
此前，超过30%的符合条件的员工亦主动参与了公司推出的自愿退休计划。
同时，微软计划将其旗下四家游戏工作室的管理权移交给新团队，此举旨在确保其宝贵的知识产权和各项在研项目能够得到妥善维护与持续发展。
对于受影响的员工，微软将提供必要的财务支持及其他辅助资源，以协助他们顺利过渡至职业生涯的下一阶段。
科尔曼进一步透露，随着微软其他业务部门陆续调整以适应公司的全新战略重心，未来预计还将有更多组织层面的变动。
