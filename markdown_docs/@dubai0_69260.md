# 佤邦大清查：192名中国籍诈骗嫌犯被移交中方

🚨 佤邦大清查：192名中国籍诈骗嫌犯被移交中方
近日，缅甸佤邦司法委员会组织全邦范围内的“大清查”专项行动，旨在持续加大对跨境电信网络诈骗犯罪的打击力度，并重点清剿那些非法偷渡潜入佤邦境内的电诈回流人员。
据通报，此次行动共抓获192名中国籍涉案人员。初步调查显示，这批人员大多是此前因其他境外区域持续清剿行动而流窜逃离，随后通过非法途径潜入佤邦，企图在当地重新组建电信网络诈骗窝点，继续实施跨境诈骗犯罪。
在抓捕行动中，有132名涉案人员由佤邦司法委员会反诈专项工作组独立部署并抓获；另有60名涉案人员则是在佤邦司法委员会与中国警方的联合行动中协同抓获。此次成果不仅展现了佤邦打击跨境犯罪的决心，也标志着中缅两国在跨境执法合作领域取得了显著进展。
