# 涉亿级刷单公司面临曝光：前员工手握核心证据

🚨 涉亿级刷单公司面临曝光：前员工手握核心证据
近日，一则匿名爆料指出，一家长期涉嫌“刷单”（虚假交易）业务的公司面临前员工的重磅指控。爆料人声称已掌握该公司大量内部核心证据，并威胁将直接提交给中国公安部处理，引发业界关注。
据爆料，这家公司每月产值过亿，涉案金额巨大。其运营模式一度神秘，曾频繁更换办公地点，先后进驻金贝园区、金玉满堂、浙商大酒店等多个场所，显示出其高度流动的特性。
爆料人声称，其手中握有包括一段长达7分钟的员工“杀客”（涉嫌欺诈或不当销售行为）视频，以及多名公司高层管理人员和股东的照片。公司核心人员包括大牛、老王、宇哥等老板，以及阿峰、星星、天权等股东均被点名。
该爆料人透露，此前曾就个人物品归还问题与公司负责人老王沟通，但对方态度强硬，导致事态升级。爆料人强调，若公司继续逃避或拒绝解决，将不再姑息，并承诺整理全部证据提交至国家公安部。
分析认为，一旦正式立案调查，该公司将面临不可挽回的法律后果，其员工回国后也可能面临被捕风险，牵涉范围或将非常广泛。
