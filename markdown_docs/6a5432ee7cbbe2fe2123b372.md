# Danantara印尼启动巴厘岛垃圾焚烧发电项目，引领可持续能源新篇章

印尼国有投资机构Danantara Indonesia已选定巴厘岛作为其垃圾焚烧发电厂（PSEL）项目的首个开发地点。此举标志着该国有投资机构正积极拓展其在可持续基础设施和清洁能源领域的战略布局。
Danantara Indonesia于本周三发布声明称，其通过子公司PT Danantara Investment Management (DIM) 和PT Daya Energi Bersih Nusantara (Denera)，已在南登巴萨的Pedungan村举行了项目启动仪式。这一里程碑事件，预示着印尼国家废弃物管理体系将迈入一个更加现代化、一体化及可持续发展的新篇章。
面对印尼每日产生逾14万吨废弃物的严峻挑战，PSEL项目被视为一项关键解决方案。它旨在缓解废弃物对环境、公众健康、气候变化以及经济生产力造成的深远影响。
对巴厘岛而言，可持续的废弃物管理体系至关重要。它不仅能有效保护岛屿生态系统、提升居民生活品质，更是支撑其核心经济支柱——旅游业长期可持续发展的基石，进而维护区域整体竞争力。
Danantara Indonesia首席执行官Rosan Roeslani表示：“正如普拉博沃·苏比安托总统所指示，废弃物管理是我们必须尽快解决的共同挑战，以避免给子孙后代带来负担。PSEL设施旨在通过采用成熟技术，有效减轻废弃物对环境、健康及安全的影响。”
他进一步强调：“在Danantara Indonesia推进这些项目时，我们不仅致力于快速交付，更以审慎的态度和最高标准的治理来确保项目的质量与可持续性。”
据介绍，巴厘岛PSEL项目将采用移动炉排焚烧技术，这是全球大多数垃圾焚烧发电设施普遍使用的成熟方案。
选择此项技术，正是因为它在全球范围内久经考验的运行可靠性，以及其与印尼城市生活垃圾特性高度契合的优势。
巴厘岛PSEL设施的设计严格遵循欧洲环保标准，即以欧盟工业排放指令（EU IED）为基准，对排放进行严苛控制。
完整的燃烧过程产生的烟气，在排放至大气前，将经过多层空气污染控制系统（APCS）的严格净化处理。
与传统的垃圾填埋方式相比，巴厘岛PSEL项目预计每吨废弃物可减少高达80%的排放量。同时，项目在建设及运营期间，将创造多达1200个绿色就业机会。
在项目启动仪式上，Danantara还与印尼国家电力公司PT PLN (Persero) 签署了购电协议（PPA）。
这份协议构成了巴厘岛PSEL设施所发电量并入PLN电网的商业框架。它不仅为项目提供了明确的电力购买保障，也为项目的长期运营可持续性奠定了基础。
在2025年第109号总统令发布后，DIM和Denera迅速加快了项目实施进程，并全程秉持最高标准的审慎、透明和良好治理原则。
在相对较短的时间内，DIM和Denera完成了一系列严格的先进技术合作伙伴甄选工作。随后，他们为首批三个项目地点设立了项目开发及运营公司（BUPP），与地方政府达成了合作协议，并加速了审批流程，同时启动了场地准备工作。
首批垃圾焚烧发电项目已被列为国家战略项目（PSN），这进一步确认了它们在印尼国家发展优先议程中的核心地位。
印尼共和国食物事务统筹部长Zulkifli Hasan指出：“该项目之所以能够顺利推进，是因为困扰废弃物管理多年的监管障碍，目前正通过放宽管制而得到简化。”
他强调：“随着法规的日益清晰、各方的紧密协作以及健全的治理体系，我坚信废弃物管理将得以加速，从而为社区和环境带来实实在在的利益。”
巴厘岛PSEL设施是各方紧密合作的结晶，由食物事务统筹部牵头，联合了环境部、内政部、能源与矿产资源部等相关部委机构、地方政府、Danantara Indonesia以及已成立的项目开发及运营公司（BUPP）共同推动。
声明强调，政策支持、负责任的治理以及公众积极参与废弃物管理解决方案，将是该体系取得全面成功的关键因素。
