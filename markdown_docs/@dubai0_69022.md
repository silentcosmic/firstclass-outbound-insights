# 迪拜信用卡盗刷案宣判：欧洲女子获刑并驱逐

🚨 迪拜信用卡盗刷案宣判：欧洲女子获刑并驱逐
迪拜轻罪法庭近日对一起信用卡盗刷案作出判决，一名欧洲女子因未经授权使用他人银行卡获取钱财，被判处一个月监禁并处罚款。
该判决依据阿联酋2021年第34号联邦法令《打击谣言和网络犯罪法》作出。该法明确规定，未经持卡人同意使用信用卡、借记卡或电子支付数据以获取金钱、财产或服务均属刑事犯罪。
法庭记录显示，该女子故意获取并利用他人支付卡数据进行未经授权的金融交易。最终，法院判决其一个月监禁，处以与其非法所得等额的18,750迪拉姆罚款，没收作案手机，并将在服刑期满后将其驱逐出境。此案再次强调了迪拜对金融犯罪的零容忍态度及严格的法律执行。
