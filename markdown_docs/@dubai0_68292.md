# 阿联酋：导弹警报系统故障，虚惊一场！

🚨 阿联酋：导弹警报系统故障，虚惊一场！
上周五，阿联酋阿布扎比和迪拜的居民经历了一场虚惊：手机上接收到导弹来袭警报后不久，又收到指令要求无视该警告。
阿联酋国家紧急危机与灾害管理局（NCEMA）随后解释称，此次错误警报是由于预警系统突发技术故障所致。NCEMA强调，专业团队发现问题后立即按预案采取了纠正措施，并表示故障已解决，将对用户的影响降至最低，同时对此次意外技术故障表示了歉意。
