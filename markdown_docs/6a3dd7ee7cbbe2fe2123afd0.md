# 启明创投胡奇：从AI算法工程师到资深投资人，深耕人工智能领域的十年洞察

展望未来数年，中国创投领域的潮流正经历深刻变革：人工智能（AI）已然从单纯的技术构想，全面渗透至产业发展的核心腹地；硬科技创业则从最初的细分领域，跃升为市场普遍认可的战略方向。新生代创业者正凭借其技术专长与创新实践，重塑中国创新版图的未来轨迹。在这样的背景下，我们观察到顶级投资机构、行业领袖及锐意进取的创业者齐聚一堂，通过一系列深度交流与专业分享，共同剖析人工智能、硬科技、出海与医疗健康等前沿赛道的深层逻辑。这些由少数先行者所坚守的愿景，正逐步汇聚成推动行业进化的巨大浪潮。
启明创投的资深投资人胡奇先生，以其深耕人工智能领域十余载的经验，分享了他对AI产业的独到见解。他强调，这份专注源自其学术背景——持有英国爱丁堡大学人工智能硕士学位，毕业后即投身AI模型算法研究，随后转向AI领域的投资工作，始终未曾脱离这一前沿科技。
谈及行业交流，胡奇先生笑言，投资人的公开分享总带着一份微妙的平衡。他坦诚地指出，当下的创投生态日趋活跃，投资人参与各类行业对话已成为常态，这反映出市场对专业洞察的需求日益增长。
胡奇先生的求学经历横跨中英两国，分别在武汉大学攻读计算机专业，后赴英国爱丁堡大学深造人工智能。他分享了一个温馨而充满时代印记的家庭故事：他的母亲曾对他的专业选择感到不解，只知道与‘电脑’相关。这位来自东北的母亲，时常劝他回归家乡，寻求一份‘稳定’的工作。而胡奇先生则幽默回应，在当时的家乡，他的专业技能可能仅限于‘修电脑’，就业机会相对有限，这也折射出早期AI行业发展区域不平衡的现实。
直至近年来AI热潮席卷全球，胡奇先生在家中与母亲谈及当前AI产业的蓬勃发展，甚至涌现出千亿乃至万亿市值企业时，母亲却冷静地回应道：‘儿子，你难道不知道吗？热得快的东西，往往也容易‘糊’。’ 这句质朴却深刻的哲理，令胡奇先生当场一怔，随后深以为然。他观察到，过去几年间，不乏一夜之间声名鹊起的企业，也常见两三年后便销声匿迹的案例。AI领域的创业竞争之激烈、风险之高，无疑是‘九死一生’的残酷现实。
回顾过去几年的投资布局，胡奇先生详细介绍了启明创投在AI领域的战略性进展。他指出，该机构已成功投资了一系列具有行业影响力的企业，尤其是在2021年底，启明创投便率先布局了智谱，成为中国最早一批关注并投资大模型技术的机构。此后，又相继投资了阶跃星辰、生数科技、无问芯穹以及与爱为舞等创新企业，构建起涵盖AI核心技术与应用生态的投资版图。
胡奇先生也坦言，在与同行和LP的交流中，他常被问及一个普遍的问题：‘既然你们投资了智谱、阶跃星辰、生数科技，为何没有选择其他一些同样知名的项目？’ 他将这种现象妙喻为他母亲的教育方式：小时候考了90分，母亲不会探究如何取得高分，只会追问‘剩下那10分去哪了？’ 这深刻反映出市场对投资人业绩评估的严苛与精细化要求，每一项决策都面临着全面的审视。
在经历了一段高强度的工作后，胡奇先生出于健康考虑开始健身，并从中领悟到了一个关于创业投资的深刻哲理。他发现，创业投资与健身锻炼有着惊人的相似之处：无论是在健身房办卡的第一天，还是完成一笔投资的最初瞬间，人们往往都充满必胜的信心。然而，随着时间的推移，一年之后，成功的秘诀便转向了对‘长期主义’的坚守与践行。
谈及对AI未来的洞察与展望，胡奇先生回顾了他与大模型的初次接触。早在2020年，当他首次研读GPT-3的论文时，便对其展现出的卓越泛化能力留下了深刻印象。同年底，他通过特定渠道获得了GPT-3的试用接口，得以亲身进行测试。他分享了一个有趣的实验：在未经任何模型训练或微调的情况下，GPT-3不仅准确理解了他的提问，还给出了流畅自然的回答，这与传统NLP模型形成了鲜明对比。随后，他又对用户标签、信息提取、翻译等多项任务进行了测试，结果显示，无论答案的最终准确性如何，GPT-3总能提供一个看似合理的回复。这种超越当时认知的泛化能力，令他尤为惊叹。
受此启发，启明创投于2021年初便积极着手在中国市场寻找顶尖的大模型研发团队。直至当年下半年，他们成功地与智谱团队建立了合作。胡奇先生将这段经历形容为一场‘双向奔赴’：一群对人工智能长期抱有坚定信念的投资者，最终与另一群同样深信AI未来潜力的创业者相遇，共同开启了合作的新篇章。
从2021年底投资智谱至今，不过短短四五年光景。然而，回溯这段历程，人工智能在技术创新、产业变革及商业化落地方面的进展，已远超当时多数人的预期。这四五年的时间，对AI领域而言，已足以将众多昔日的构想转化为现实，并引发整个行业的结构性巨变。启明创投始终坚信人工智能的长期价值，并期待未来能持续赋能杰出的创业者，共同推动AI生态的繁荣发展。
