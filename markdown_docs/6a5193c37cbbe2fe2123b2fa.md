# 新加坡投资者广泛拥抱AI赋能金融，但最终决策仍依赖专业顾问深度洞察

《出海·头等舱》独家洞察显示，新加坡的大众富裕及高净值投资者中，高达76%已将人工智能（AI）应用于其金融与投资活动，这一比例显著超越了全球平均水平的72%。然而，尽管AI在洞察生成方面日益成熟，这些投资者在做出最终投资决策前，仍普遍寻求专业金融顾问的验证与认可。
这项关键数据来源于一项最新的行业调研。该调研由一家全球领先的银行委托Ipsos于2026年1月至2月期间执行，覆盖了全球十大市场共计9,993名大众富裕及高净值投资者，其中新加坡的受访者数量为609人。
值得关注的是，新加坡市场对AI的采纳呈现出跨世代的广泛性。X世代投资者在金融领域使用AI的比例高达72%，远超全球同类群体的65%；而婴儿潮一代在新加坡的AI使用率同样达到72%，显著高于全球的59%。
在可投资资产超过200万美元的高净值投资者群体中，新加坡的AI采纳率更是高达90%，高于全球平均的82%。尤其引人注目的是，新加坡最富有的受访者平均将其过去12个月的投资回报中，有40%归因于AI的影响。
然而，高普及率并不等同于绝对依赖。数据显示，仅有8%的新加坡投资者表示，AI是他们最近一次重大投资决策中唯一或最具影响力的信息来源，这一比例低于全球的12%。
具体而言，投资者利用AI进行研究分析的比例为69%，获取策略支持的为44%，对投资理念进行压力测试的为34%。随后，他们会将这些AI生成的发现提交给专业顾问，以寻求79%的信心确认和71%的战略专业知识。超过半数，即57%的投资者倾向于AI与专业顾问协同工作，这一比例亦高于全球平均的50%。
一位专注于国际财富与私人银行业务的资深人士指出，这组数据反映出新加坡投资者正以审慎的态度运用AI工具，这使得他们在与专业顾问沟通时准备更加充分，并对专业服务提出了更高的期望。
在这些发现发布之际，一家全球金融服务巨头与一家领先的云计算提供商于2026年6月宣布建立一项为期多年的战略性AI合作关系。超个性化财富管理支持作为其首批三大重点领域之一，预计将在两年内在全球运营中催生200多个新的AI应用场景。
新加坡淡马锡：押注AI未来，设定投资组合新目标
呼应这一趋势，新加坡主权财富基金淡马锡（Temasek）正大力押注人工智能领域，并设定了宏大目标：到2031年，其投资组合中AI相关资产的配置比例将达到10%至15%。
