# 金边：三名中国籍男子入室盗窃未遂被捕

🚨 金边：三名中国籍男子入室盗窃未遂被捕
金边消息，柬埔寨警方于6月22日采取行动，逮捕了三名涉嫌入室盗窃未遂的中国籍男子。据警方调查，这三名嫌疑人于6月21日晚潜入金边朗哥区一处民宅，企图撬锁入室盗窃，但最终未能得手，也未造成受害者财物损失。
警方在锁定嫌疑人住处后，在菩森芷区成功将其抓获，并缴获了作案工具及多部手机和银行卡。目前，此案已移交朗哥区警察局，正依法进行后续处理。
