# Telegram全球监管风暴升级：多国限制，创始人遭法起诉

🚨 Telegram全球监管风暴升级：多国限制，创始人遭法起诉
凭借其严格的用户隐私保护政策，Telegram已成为全球领先的即时通讯平台之一，尤其受到抗议人士和新闻机构青睐，以规避政府监控。然而，其在用户匿名性与加密通信方面的坚持，也使其面临助长恐怖主义、犯罪活动及虚假信息传播的指控，引发世界各国政府陆续采取限制措施。
**印度**
作为Telegram最大市场之一，印度本周宣布暂时关闭该平台至2026年6月。此举旨在打击利用Telegram传播医学院入学考试虚假“泄题”的诈骗行为。尽管事后证实这些试题均为伪造内容，印度政府仍取消考试并重组，并以平台被用于传播和销售“考试泄题”为由暂停Telegram一周。创始人Pavel Durov对此批评，认为此举惩罚了普通用户而非诈骗者，并表示Telegram已删除数百个相关违规频道。
**俄罗斯**
Telegram在创始人杜罗夫的故乡俄罗斯经历多次波折。2018年，俄罗斯曾因Telegram拒绝提供用户聊天记录而将其封禁，但2020年又因Telegram同意加强打击极端主义内容而取消禁令。然而，2026年初，俄罗斯再次尝试限制Telegram，指责其未能有效打击诈骗、保护用户数据及阻止恐怖分子利用平台。杜罗夫则否认指控，称俄罗斯政府的真正目的是迫使民众转向国家控制的通讯软件以进行监控。
**乌克兰**
俄乌战争期间，Telegram成为乌克兰民众获取信息的重要工具。但乌克兰政府担忧俄罗斯利用该平台传播虚假信息或进行间谍活动，因此于2024年禁止军方人员、政府官员及关键基础设施工作人员在工作设备上使用Telegram。部分官员还要求Telegram公开大型匿名频道的运营者身份。
**欧洲**
挪威政府于2023年基于国家安全考量，正式通知政府工作人员不得在公务设备上安装或使用Telegram与TikTok。法国检方更于2024年对Telegram创始人杜罗夫提起刑事指控，认为Telegram未能有效打击平台上的儿童性侵内容传播、毒品交易、网络仇恨犯罪及其他严重刑事犯罪。杜罗夫一度被禁止离开法国，后获准返回迪拜。
**巴西**
巴西曾两次封禁Telegram。2022年，巴西最高法院因Telegram未能执行删除与前总统支持者相关账户（涉嫌传播虚假信息）的命令而下令全国封禁，后在Telegram配合后解除。2023年，一名法官再次下令封锁，原因是Telegram未能提供一个新纳粹极端主义聊天群的用户数据。尽管该群组已被删除，且禁令最终被上诉法院撤销，但Telegram仍面临高额罚款。
随着Telegram在全球范围内影响力不断扩大，其在隐私保护与执法监管之间的矛盾日益突出。支持者视其为保护言论自由和用户隐私的重要工具，而批评者则指其纵容诈骗、恐怖主义、极端主义宣传以及跨国犯罪活动。这场围绕隐私权、安全监管与平台责任的全球性争议，预计仍将持续下去。
