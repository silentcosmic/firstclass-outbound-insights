# 缅甸水沟谷反诈行动：再拆一栋楼，仅剩10栋待清

🚨 缅甸水沟谷反诈行动：再拆一栋楼，仅剩10栋待清
7月1日，缅甸克伦邦妙瓦底水沟谷地区的反诈清理行动持续推进。当地联合工作组再次拆除了一栋曾涉及网络诈骗和网络赌博的非法两层建筑，该建筑属于此前公布的机械拆除范围。
根据此前计划，水沟谷地区共锁定77栋涉诈非法建筑，其中57栋将通过机械拆除，另20栋则采用定向爆破。截至目前，涉诈建筑累计已拆除67栋，仅剩10栋待后续依法处理。当地仍在同步推进涉诈设备销毁及相关案件的法律程序，以彻底肃清该区域的非法活动。
