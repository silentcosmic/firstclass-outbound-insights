# ️ 斯里兰卡博睿集团陷欠薪风波：员工曝工资减半、支付受阻

⚠️ 斯里兰卡博睿集团陷欠薪风波：员工曝工资减半、支付受阻
近日，有市场消息指出，斯里兰卡博睿集团（Bory Group）正面临严重的劳资纠纷，数十名技术员工的工资遭到长期拖欠。
据受影响员工爆料，该公司先是单方面将工资条上的金额减半，随后又无故拖延发放日期。在员工追讨过程中，公司方面给出了多种解释，包括声称有技术负责人冒领工资，并要求所有员工在领取工资时必须进行视频语音验证，并手持身份证件，此举被员工视为前所未有的苛刻要求。然而，即便有员工按此要求操作，也未能如期收到薪资。
更令人质疑的是，员工在入职时已提供了U地址（数字货币钱包地址），且在发放工资条时也再次确认过支付信息。在这样的背景下，公司声称的“冒领”理由显得难以服众，被指为拖延支付的借口。此外，爆料还提及，该集团的负责人之一据称是某体育平台（Bty）的股东，加剧了外界对公司运营透明度和诚信度的担忧。
此次事件引发了对斯里兰卡地区部分企业商业信誉的广泛讨论，有受影响员工表示，当地一些公司在当前经济环境下，存在未能履行支付义务的情况。
