# 老挝向中国移交中国籍嫌犯

🚨 老挝向中国移交中国籍嫌犯
7月12日，老挝博胶省公安指挥部在老中边境磨丁—磨憨国际口岸举行了中国籍犯罪嫌疑人移交仪式。该嫌疑人因涉嫌违反老挝法律，已被依法移交至中方。此举体现了中老两国在执法领域的合作。
