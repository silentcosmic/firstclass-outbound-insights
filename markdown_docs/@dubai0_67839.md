# 柬埔寨官宣：中国公民2026年起免签入境

📣 柬埔寨官宣：中国公民2026年起免签入境
柬埔寨政府近日宣布，自2026年6月15日至10月15日期间，持有中华人民共和国普通护照的公民，无论从中国境内或第三国出发，均可免签入境柬埔寨。在此期间，旅客可多次入境，单次停留最长不超过14天。入境手续将简化，无需提前办理签证，仅需按要求填写电子入境卡（E-Arrival Card）即可完成。
中国驻柬埔寨使馆提醒计划赴柬的公民，务必提前做好出行安排。建议提前了解柬埔寨出入境规定、当地天气及风俗习惯。目前柬埔寨已启用电子入境系统，旅客可在入境前7天内通过“Cambodia e-arrival”手机应用或访问www.arrival.gov.kh网站填报信息。使馆特别强调，柬埔寨政府严禁口岸官员索要小费，中国公民如遇此类情况应坚决拒绝，并在确保自身安全的前提下收集证据，及时向使馆反映。
同时，使馆敦促旅客注意旅行安全，妥善保管护照、财物及个人证件，警惕陌生人，不替他人携带物品。严格遵守柬埔寨法律法规和宗教习俗，例如参观寺庙、王宫及吴哥古迹时需着装得体，不触摸或攀爬文物，不冒犯僧侣。此外，柬埔寨对无人机使用有严格管制，未经许可切勿擅自携带或使用。
鉴于柬埔寨政府近期持续加大打击电信网络诈骗、网络赌博及非法拘禁等犯罪活动的力度，使馆特别提醒公民洁身自好，远离涉“黄赌毒”活动，切勿轻信“高薪招聘”等虚假信息，谨防陷入非法务工风险。近期柬埔寨加强对外国公民合法身份的查验，请务必随身携带护照原件，积极配合检查。如遇误解或被带离核查，请保持冷静，通过合法途径维护自身权益，并及时向中国使馆通报情况。
最后，建议所有赴柬公民提前掌握当地报警、医疗急救电话以及中国驻柬使馆的应急联系方式，以便在紧急情况下及时寻求帮助。
