# 柬埔寨巴域市突袭网络诈骗窝点：33人被捕

🚨 柬埔寨巴域市突袭网络诈骗窝点：33人被捕
柬埔寨柴桢省巴域市于7月15日开展了一次大规模打击网络诈骗的专项行动。在巴域市副市长罗烈·拉萨的统一指挥下，联合柴桢省宪兵司令部，并在省副检察官多拉·昂普拉丹的法律协调下，成功查处了一处涉嫌网络诈骗的窝点，现场控制了33名涉案人员。
