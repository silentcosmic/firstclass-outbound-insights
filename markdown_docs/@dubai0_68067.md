# 柬埔寨捣毁“波贝观察”情报网：16名内线被捕

🚨 柬埔寨捣毁“波贝观察”情报网：16名内线被捕
柬埔寨警方近日成功捣毁一个名为“波贝观察”的秘密情报网络。该网络利用社交媒体平台实时监控警方执法行动，并向诈骗犯罪团伙通风报信。
据了解，该团伙专门收集警方突击检查线索，协助不法分子逃避打击，甚至为其策划新的非法活动。这种行为严重阻碍了当地警方的反诈骗工作。
在经过周密部署后，警方成功抓获该团伙16名核心成员，并已将其全部移送法庭起诉，彰显了柬埔寨当局打击跨境犯罪的决心。
