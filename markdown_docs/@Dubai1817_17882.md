# 泰国捣毁跨境假国籍团伙：中国客户伪造父子关系获身份

🚨 泰国捣毁跨境假国籍团伙：中国客户伪造父子关系获身份
7月9日，泰国警方启动“剥龙鳞”专项行动，成功捣毁一个长期运作的跨境犯罪网络。该团伙专门协助中国籍人士子女非法获取泰国国籍，通过伪造亲子关系、勾结医院及政府部门人员进行身份造假。
该网络已运作逾五年，其模式为招揽中国客户赴泰生子，随后安排泰国男子冒充孩子生父。团伙成员还包括医院病案人员和区办事处工作人员，负责伪造出生证明、补录父亲资料及办理户籍登记。客户需支付约7万泰铢分娩套餐费、2万泰铢协调费，以及2千至1.5万泰铢不等的后续登记费用，总计近10万泰铢。
警方调查发现，涉案医院在过去五年内共有164名中国籍产妇生产，并均使用泰国男子作为孩子父亲办理出生登记。这些“假爸爸”从未出现在产检记录中，往往在孩子出生后才突然出现。目前，通过DNA检测已确认至少20名泰国男子与孩子无血缘关系。警方已针对40名重点涉案人员申请逮捕令，并逮捕了16人，包括多名涉案工作人员、泰国籍“假爸爸”以及中国籍父母。
一起典型案例显示，一对中国籍夫妇在曼谷高档住宅区居住，其子于2024年1月出生后即取得中国国籍。然而，他们又于2026年1月通过该团伙，将一名泰国男子登记为孩子父亲，以获取泰国身份。DNA检测证实该泰国男子与孩子并无血缘关系。该案件最初是从一宗涉案金额超过700亿泰铢的中国诈骗集团洗钱案中被牵引出来的。
泰国警方指出，非法获取泰国国籍后，子女不仅能申领泰国身份证，享受教育、医疗及社会福利，还可合法持有土地及其他资产，甚至被用于代持公司、房产、洗钱及规避监管。目前，警方正持续追查幕后组织者、涉案医院、公职人员及更多中国客户。
