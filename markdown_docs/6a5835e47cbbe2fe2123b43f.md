# 斯年智驾斩获3亿元C轮融资，智能重卡赛道热度持续攀升

本刊从独家渠道获悉，专注于重载物流自动驾驶解决方案的斯年智驾，近期已成功完成3亿元人民币的C轮融资。此次融资由兴证资本与颐道资本联合领投，所募资金将主要用于其新一代车规级自动驾驶方案的深度研发。
聚焦该细分赛道，仅在2026年1月至6月期间，公开披露的融资事件便已达到5起，累计募资规模逼近70亿元人民币，同时，市场亦观察到密集的IPO动向。资本涌入的深层逻辑在于，自动驾驶重卡在商业化落地上正逐步呈现出明确的路径与成效。在港口、矿山、钢铁厂等特定封闭场景，无人重卡已形成稳定的刚性需求；与此同时，干线物流等更为复杂的开放场景也正随着政策的逐步放开而加速落地，预示着该领域即将迈入规模化应用的前夜。
成立于2020年4月的斯年智驾，自创立伊始便致力于重载物流领域的无人驾驶技术研发与商业落地，其业务核心是为港口、园区及干线物流等多元场景提供全面的自动驾驶解决方案。
产品层面，斯年智驾的主力阵容涵盖无人集卡、无人自卸车及无人平板车等系列。这些产品的核心支撑，源于其自主研发的“一段式世界模型”自动驾驶方案，并辅以域控制器、高精度定位板卡等关键硬件，共同构建了其技术壁垒。
在商业落地模式方面，斯年智驾提供双元策略：一方面可直接销售自动驾驶整车，另一方面亦可提供定制化的运力服务。其解决方案已在封闭场景中广泛应用，从传统的集装箱港口、散杂货堆场，逐步扩展至钢厂、纸厂、铝厂、铁路站场及各类物流枢纽。而在开放场景，公司也积极与多家物流企业及大型央国企建立合作关系，探索更多可能性。
突破“无人运输”至“无人作业”的边界
斯年智驾首席执行官何贝指出，公司当前的关键战略焦点之一，在于如何将目前局部的“无人运输”能力，进一步拓展至全局性的“无人作业”范畴。他阐述道：
“横向运输的自动化已基本实现，但我们正在思考如何通过引入世界模型与具身智能技术，对垂直运输及端侧协作进行全面一体化升级，最终在封闭场景中实现完全无人化的作业流程。预计在未来3到5年内，这一探索将有望形成明确的解决方案。”
何贝进一步强调了域控制器在无人驾驶系统中的核心地位，将其比作“无人驾驶的大脑”。他解释道，在无人驾驶的整个产品链条中，车辆的执行端，包括油门、刹车及线控技术等，已达到相当高的成熟度；而雷达、相机等传感器更多扮演着数据采集工具的角色。真正决定能否实现从“有人驾驶”向“无人驾驶”转变的关键，在于“所有的软件逻辑都固化并运行在控制器上。”
何贝透露，斯年智驾已成功推出车规级自动驾驶方案。他指出，车规级标准的达成极具挑战性，要求“硬件、操作系统、中间件乃至上层的软件模型板块，都需要进行全面而深入的调整与优化”。
对比封闭场景，开放的干线物流场景无疑复杂度更高，常需应对长距离颠簸、宽泛的温差波动及强电磁干扰等严峻挑战，这使得非车规级器件的故障风险显著增加。因此，要实现从小规模测试向大规模商业应用的跨越，从受控的封闭场景驶向广阔的开放道路，车规级标准的达成为自动驾驶企业构筑了一道必须攻克的“生死关隘”。
除了车规级标准，开放场景所面临的挑战还包括数据量级的显著提升。何贝补充道：“开放场景的数据量要比封闭场景高出一到两个量级。鉴于过去行业多数落地实践集中在封闭区域，我们当前的另一项重点任务是持续积累海量数据，并以此优化整体算法模型。”
斯年智驾此次3亿元C轮融资的达成，不仅是对其作为场景物流领域头部玩家的一次重要加持，更深层地折射出整个自动驾驶重卡赛道的行业共识：唯有深耕并做透封闭场景的基本盘，并在开放场景中成功突破车规级的严苛考验，企业方能真正具备迈向下个发展量级的资格。
从“无人运输”到“无人作业”的深层考量
关于如何将“无人运输”升级为“无人作业”这一宏大命题，何贝对此作了深入阐释：
“重卡本质上仅是一种运输工具，而非装卸工具。从整个物流作业流程来看，单纯的自动驾驶重卡，实则缺失了关键一环。只有将重卡所承担的水平运输与装卸服务这类垂直运输相结合，才能构建一个真正完整的无人化作业流程。目前在许多场景中，例如挂车的甩挂操作，仍需人工完成。我们正在构思未来是否能由机器人来执行这类操作。又如，叉车、吊车等设备，能否借助具身智能技术完全实现无人化？我们的愿景是实现转运与装卸全流程的无人化。”
对于这一愿景的实现时间，何贝表示：“这仍处于测试研发阶段，进展并非一蹴而就，难度系数极高。我认为可能需要3到5年的时间才能真正落地。不过，相较于无人驾驶技术从概念到落地耗时十年之久，无人化物流作业的推进速度已然显著加快。”
他进一步分析了驱动这一趋势的因素：“一方面，当前人工智能技术的飞速发展，正在显著加速我们整个研发与落地的全流程。另一方面，人力成本持续攀升亦是重要推力。观察国内诸多环卫场景，无人化实践已走在前列，部分功能模块甚至已基本实现‘去人工化’。因此，我们坚信全面无人化终将到来，‘晚干不如早干’。”
自动驾驶重卡竞争力的三大核心要素
在谈及当前自动驾驶重卡企业竞争的关键要素时，何贝总结了三点：
“首先是品牌。当前的市场格局下，新进入者或仅仅依靠测试场景的公司已难以立足，企业的现有声誉在很大程度上决定了能否获得客户的信任与合作。其次是数据。各家公司在不同应用场景的深耕，其背后体现的正是数据积累的优势。第三是算力与算力平台，这包括其上的数据闭环与飞轮系统。在我看来，这三点构成了当前各家公司展开竞争的核心所在。”
对于企业盈利能力，他坦率指出：“简而言之，销售费用受业务表现和外部支持力度的影响；而研发费用则取决于AI技术的有效应用程度以及公司内部产品化能力。我个人认为，这两点是制约公司盈利能力的关键因素。”
海外市场拓展的机遇与挑战
谈及海外市场的拓展，何贝分析了其首要意义与伴随的风险：
“海外市场首要的吸引力无疑在于盈利潜力，但其风险也与利润成正比。首先，若产品售价无法达到国内的3到4倍，那么在海外市场便难以实现盈利。其次，海外市场面临诸多限制。真正需要无人驾驶技术的地方，无疑是欧美、澳洲、日韩等发达经济体，然而这些地区普遍受限于强大的工会影响，导致无人化技术的推广速度相对缓慢。相比之下，中东、南亚、南美等区域多为发展中国家，当地劳动力成本相对低廉。因此，这些地区采用无人化技术的核心动因是显著提升运营效率。基于此，我们目前的海外订单也主要集中在这些区域。”
对市场周期与企业发展的判断
对于市场周期与未来的发展，何贝表示：“当前的市场环境是否为最后一波投资热潮，现在难以定论，但我们预计到2027年下半年市场可能会出现一定程度的收紧。整个行业的发展必然是波动的，即便明年有所收紧，后年也可能再次迎来开放。比起追求上市，我个人认为扎实做好核心业务显得更为重要。”
