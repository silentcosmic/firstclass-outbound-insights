# ️ 阿联酋网红诽谤餐厅被重罚8.1万迪拉姆

⚖️ 阿联酋网红诽谤餐厅被重罚8.1万迪拉姆
据阿布扎比司法部门（ADJD）消息，一名社交媒体内容创作者因发布针对某知名餐厅的视频内容，被法院裁定构成诽谤，而非正当的商业评论，最终被判处高达81,000迪拉姆（约合人民币16万元）的罚款。
该网红在其社交媒体账号上发布视频，质疑了餐厅经营者的诚信及商业行为。餐厅老板随即提起诉讼，指出该视频严重损害了其个人及企业的声誉。此次判决凸显了阿联酋在维护商业声誉和规范社交媒体内容方面的严格立场。
