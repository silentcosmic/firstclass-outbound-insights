# 泰航空姐涉毒被捕，总理震怒损国格

🚨 泰航空姐涉毒被捕，总理震怒损国格
泰国总理阿努廷在法国访问期间，对一名泰航空姐在澳大利亚涉嫌走私毒品被捕一事表达了强烈不满。他指出，此事件严重损害了泰国的国际形象，并敦促相关机构进行彻底调查。
据悉，这名26岁的泰籍乘务员于6月24日在墨尔本国际机场被捕。澳大利亚当局在其随身携带的12个手提包内衬中查获了逾1000克海洛因，黑市价值高达约34万美元。该空姐已被澳方正式起诉，面临两项跨境走私和持有受管制毒品的指控，最高可判处50年监禁。
