# 出海观察 | 智能出行新星OMOWAY获宁德时代系基金青睐，印尼电动摩托车市场首发即领跑

本刊独家获悉，智能两轮出行领域的新锐企业「OMOWAY」（目蔚科技）近期成功完成A轮及A+轮融资，两轮融资金额均达到数千万美元级别。其中，A+轮融资由宁德时代旗下美元基金Lochpine Capital领投，A轮融资则由Monolith牵头，并获得中金资本旗下基金以及现有股东真格基金的持续支持。
自成立不足两年，OMOWAY已迅速完成多轮融资。截至当前，其投资者阵容星光熠熠，不仅汇聚了红杉、真格、Monolith、中金等行业领先的财务投资机构，更吸引了Lochpine Capital、星航资本（小鹏汽车关联基金）、惠友资本（比亚迪联合创始人设立）等新能源产业战略投资方的深度参与。
凭借雄厚的资金储备与强大的产业资源协同，OMOWAY正加速其产品在全球市场的商业化布局。今年6月，OMOWAY首款旗舰级电动摩托车OMO-X已正式启动全球范围内的首次交付，其海外战略的首个落脚点选定在印度尼西亚。
审视全球摩托车市场，长期以来日系燃油品牌占据主导地位。这一领域普遍存在挑战：高端智能产品往往定价过高，而普惠型产品又普遍缺乏先进科技感。OMOWAY的迅速突破，正是通过智能化技术驱动所带来的“降维打击”式体验，成功颠覆了当地传统出行产品的既有市场版图。
深入产品设计层面，OMO-X在智能化与硬件配置上均展现出卓越水准。该系列车型全线标配数字钥匙、自动边撑及远程控车等功能，并创新性地搭载了摩托车领域罕见的10.25英寸汽车级智能交互大屏，远超行业主流的5-7英寸屏幕尺寸。此外，车辆的后悬挂系统采用自研的双叉臂后连杆结构，此设计常见于豪华汽车，显著提升了车辆的操控稳定性与骑行舒适度。
在产品线规划方面，OMO-X Smart（智能版）专注于打造极致的智能化出行体验，通过深度的数字化交互与智能生态系统连接，旨在为用户提供更便捷、高效的骑行方案。同时，其推出的OMO-X Balance（自平衡版）更是全球首款实现量产交付的自平衡摩托车。OMOWAY团队精准洞察到摩托车“易倾倒、难扶正”的核心痛点，创造性地引入源自航天领域的陀螺仪稳定技术，并将低速平衡辅助作为标准配置，从而显著增强了骑行稳定性。
为应对全球市场中复杂多变的真实路况，OMOWAY为OMO-X研发了先进的强化学习模型。该系统在云端仿真环境下，通过海量数据训练，持续优化电机输出与陀螺仪扭矩的协同逻辑。在行驶安全防护方面，OMO-X集成了由双前视摄像头与四个鱼眼摄像头构成的360°全景视觉解决方案。在低速行驶状态下，无论是遭遇湿滑路面、复杂弯道，还是面临潜在的碰撞风险，系统均能通过风险预判，精确控制电机、转向及刹车系统提前介入，从而及时修正行驶姿态，提供卓越的主动安全保障。
OMO-X卓越动态性能的底层逻辑，根植于OMOWAY自主研发的OMO-ROBOT通用技术架构。这一前瞻性架构将人工智能、机器人技术与车辆工程进行了深度融合，构建了一个涵盖感知、决策、执行与信息传递的全链路闭环系统。基于此核心技术架构，OMOWAY还推出了百分之百自主研发的轮式机器人Mobility One，其原型有望于今年内正式亮相。
依托其深厚的技术架构驱动力，OMOWAY正积极推进全球市场版图的扩张。作为其海外拓展的首个战略支点，OMOWAY已在印度尼西亚构建起覆盖数十家经销商的庞大网络，业务足迹全面深入雅加达、万隆、泗水及巴厘岛等关键区域。
展望未来，OMOWAY计划将业务版图逐步扩展至泰国、新加坡乃至欧洲市场。公司设定了宏伟愿景：以智能两轮出行为切入点，逐步向即时配送、公共服务、智慧家庭等多元场景进行业务延伸，致力于推动交通工具向更智能的伙伴角色演进，最终将高端智能化的产品体验带入千家万户。
