# 伊朗突发：霍尔木兹海峡航运通道关闭！

🚨 伊朗突发：霍尔木兹海峡航运通道关闭！
伊朗方面宣布，已关闭霍尔木兹海峡的航运通道。此举被伊朗视为对以色列持续打击黎巴嫩的直接回应。此前，据黎巴嫩国家通讯社报道，以色列于上周六对黎巴嫩多地发动空袭，已造成至少38人死亡，区域紧张局势持续加剧。
据伊朗国家媒体法尔斯通讯社消息，此次关闭海峡的另一原因在于“美国公然违背承诺、违反协议”，未能落实此前一项所谓“战争结束协议第一条款”的相关内容。此举无疑将进一步推高地区冲突风险，并可能对全球能源市场造成冲击。
