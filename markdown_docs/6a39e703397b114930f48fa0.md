# 行业风向 | 圆桌讨论：在番禺，我们造了什么？ | WAVES2026新浪潮

“2026年，创投圈的浪潮再次翻涌：AI从技术概念走进产业深水区，硬科技创业从“小众赛道” 变成“主流共识”，年轻的创业者们正在用代码和双手，重新定义中国创新的未来坐标。
每一年，由本站 · 暗涌主办的WAVES大会，都是中国创投圈的年度风向标。今年的 WAVES 2026以“今年盛夏”为主题，落地广州番禺良仓新造创意园，在两天的时间里，我们汇聚了顶级投资人、产业领袖、新锐创业者，用14场深度圆桌、数十场独立演讲，拆解 AI、硬科技、出海、医疗等核心赛道的底层逻辑，见证那些“少数人” 的坚持，如何汇聚成改变行业的浪潮。”
杨沙沙：今天请到的三位嘉宾虽然做的事情不一样，但是他们都有一个共同点，那就是扎根番禺，同时也在做产业里面的新变化，首先请三位简单介绍一下自己，然后再用一个具体的场景来讲一讲各自到底是在做什么，帮谁解决了什么问题。
翁淑蓁：大家好！探迹科技是一家专注于数字生产力的大模型智能体平台，通过人工智能跟大数据助力全球企业的成长。今天主要聚焦在智能销售领域，分享我们的智能体产品。从场景来看，在番禺可以看到非常多的大中小型的服装企业，仅南村镇塘埗西村都会有上百家的服装工厂。这些服装工厂经营模式非常灵活，也非常敢拼，但也会遇到问题，比如在获取客户以及营销成本较高的情景下。探迹科技造的是什么？就是去帮助这些工厂企业解决他们“下一单在哪里”的焦虑。我们的产品包括拓客Agent，帮客户去找到国内的客户，也有像Futern这样的全球销售智能体，去帮客户实现高效出海。以拓客Agent举例，番禺有一家服装的拉链工厂，我们通过系统快速地帮他定位全国的下游成衣厂，然后告诉他们哪些成衣厂可以成为精准的潜在客户，然后促成这个订单，从而让客户去告别过往需要大量的人工去拨打电话，从而去获客的场景。探迹科技所创造的价值，第一是提供了一张精准的生意地图，我们让数据跑在业务的前面。第二是销售场景里面，提供了高效的AI生产力，解决销售环节里繁琐的、高频的重复工作，让人解放出来，做更多定策略、谈大单等高价值、强转化的事情。
邱辉：大家好！浩云科技是2015年于创业板上市的一家物联网企业，是番禺首家，如果要定义浩云科技，现在核心是智慧物联、政企数字化、智慧能源三大业务板块。这三大板块是浩云科技最核心的能力，比如在政企端，基本上需要通过定制化能力去满足客户的个性化场景需求。基于个性化场景需求，让我们的业务覆盖越来越多的行业，所以后来为了满足更多行业，我们打造了低代码物联网平台，再通过平台结合AI，快速地去满足政企个性化需求。这是我们区别于传统通用化SaaS解决方案的地方。
柒成华：大家好！广州一小时科技有限公司有一个大家更熟悉的名称叫一小时快修，在全国很多地方都能看到我们的这个标签。我们是土生土长的番禺公司，2014年在番禺成立，经过12年的发展，已经从一个手机的维修门店变成了一个遍布在全国29个省、400多个城市的综合的服务平台。为全国的门店提供智能化的门店管理，以及为消费者提供从手机、家电到所有的电子产品，以及现在一些新领域：如无人机器人、新能源汽车的售后维修维保的一个服务公司，我们也通过公司AI智能店长深入到AI领域的应用场景里面。AI应用场景里，像在我老家广西桂平，当地很多人如果手机出问题，其实都不知道去哪里修，路边随机找一家维修店又要面临会不会被坑的问题。但是踏入1小时快修的门店，他的手机可以通过公司的系统快速进行诊断，确定问题之后快速匹配配件，半小时内便可在前置仓拿到配件，而且是正品带票的原装配件，确保维修质量、速度。我们团队是希望通过服务生态提供一小时快修、一小时配送，一小时上门，这样的智能化的、标准化的服务给到全国的消费者。我们目前在全国打造了10万家门店，以后还会更多地去覆盖全国的交付网络，同时交付节点通过AI智能店长提供标准化的、智能化的服务。
邱辉：浩云科技在番禺有20多年，公司从一个没有走出广州的小企业慢慢到布局全国，从最开始的金融物联，拓展到高校、能源、交通，用这20多年来的经历形容番禺，我认为是务实和敢闯，因为每当一个企业发展到一个关键阶段的时候，你会发现番禺它有非常多的这样子的氛围、土壤、资源和支撑，只要敢去想，敢去做，就有机会。在这里产业的生态环境非常好，大家虽然有竞争，但都是互相促进。
柒成华：我的第一反应其实也是务实。我们在番禺十几年，从体感上而言，可能说不上哪一方面特别强，但是把所有东西加在一起，会发现番禺是一个非常适合创业者起步的地方，性价比特别高。番禺每样东西都是靠前的，虽然不是第一，但也有第二、第三，所以一小时从番禺起步，也是各方面综合在一起的这种性价比。从人才需求层面，一小时科技主要需要两类人才，一类是算法工程师、一类是下沉市场的维修技师，这两类人才在番禺都特别容易找到，因为广州大学城在这里，像番职院等中专、大专院校番禺也有很多。并且，我们做服务、做平台的一些特征，和番禺的气质也是非常契合的。
翁淑蓁：我从业务场景里讲几个关键词，客户、产业和全球化视野。像探迹科技做的拓客Agent，打开番禺的地图，会发现其实番禺的客户体量挺大的，不管是说广汽埃安、希音这样的行业巨头，还是说制造企业里面成千上万的中小企业。基本上客户很大可能就在隔壁工业园，只要有产品，开车可能只需十几分钟，就能进行业务洽谈。这样的客户密度，对做生意是非常宝贵的。第二在产业链这块，番禺的产业底子是厚的，不仅是服装，番禺整体的制造业门类集聚度都挺高的。探迹做拓客Agent，做了很多垂直行业的版本，我们几乎在番禺都能找到各个细分行业里面的对标原型客户。说明番禺的产业拥有足够的密度也足够健全，这些要素让我们把AI训练做得更通透。最后是全球化视野，因为在番禺可能一家看似不起眼的工厂，也是在帮国际顶级品牌做代工，当时探迹科技做Futern这样的全球化产品，在番禺就能非常方便的开展调研。总的来说，番禺对探迹科技而言，是一个真实需求的发源地，也是产品的打磨场，在这里可以听到前线的炮火，客户的痛点，让我们把用户的声音揉进产品，才能把产品做得更好。
邱辉：浩云科技业务实现全国发展，其实是番禺给我们提供了好的环境和基础，公司每次遇到需要跨越的节点的时候，比如从广东走向全国，从单一行业到多个行业，每个关键节点的时候，在番禺都能找到支撑，支撑我们成长，在跨越的时候提供能力和资源支撑，这在其他地方未必能找到。在番禺发展对于我们来讲更多是一种幸运，在番禺这样的土壤里，把企业做起来，才能去面向更多的市场机会。
柒成华：我们觉得还是在于性价比。我们最近在建一个产业园，初衷也是围绕番禺，不管是区位优势、产业优势还是人才优势，番禺能够把整个生态形成一个聚集区。一小时科技围绕手机、电脑、家电这些3C的售后，后期还会延伸到新能源汽车、无人机以及机器人这些产业，这些产业在番禺都有前端非常成熟的市场，我们去解决前端产品的后市场问题，这种产业链闭环在番禺非常容易实现。同时，番禺在地理位置上是大湾区的中心区位，不管从交通，还是利用番禺产业集聚的优势，番禺我们觉得是95分以上。
翁淑蓁：探迹科技以前可能是“给人递锤子”，今天已经是直接“替人敲钉子”了。以前做产品还是帮忙找客户，具体销售跟进还是依赖销售本人，更像一个情报中心，提供精准的信息情报，但是现在随着AI的发展，拿Futern全球销售智能体举例，已经进化到在智能体里输入产品相关信息，就可以自动筛选全国乃至全球的客户，并且自动发信，沟通初步意向，整个过程不需要人为介入，像一个独立的数字员工自己干活。要实现这个转变，最难的地方在于如何让AI在真实商业场景中能够可靠地完成任务，这需要我们花大量时间，给AI训练行业的knowhow以及业务流程，让AI能够真正稳定、可靠地干活。而当AI掌握技能之后，就能够帮企业解决问题。人可以解放，聚焦在做决策、谈大单的高转化、高价值的流程里。
邱辉：浩云科技在政企端和中小民营企业确实有很大的不同，政企端对AI的态度，安全是第一要素，所以眼下阶段更多是锦上添花。但是AI的本质问题并不仅仅是降本增效，而是改变行业逻辑，长远来看如果哪个细分场景做好AI垂直大模型，那企业一定会非常领先，这又成了生死命题。
柒成华：一小时科技的AI智能店长面对的就是广大下沉市场的门店端，我们深刻感受到，AI工具正在面临从辅助工具向基础设施的转变。很多人一开始只是降低成本，提高效率，但是用多了之后，AI在很多场景里面能够解决非常多根本性问题，比如说定价策略、库存管理，甚至客户管理都能直接通过AI系统完成快速决策。实现了收益增加后，就离不开AI了。但有些地方却是AI很难改变的，比如说客户信任，像维修行业要建立信任，这其实是AI解决不了的，这是我觉得后面AI迭代需要考虑的地方，信任这种主观的东西，如何去标准化，这也是我们面临并在思考的问题。
杨沙沙：今天的大会叫WAVES，讲新浪潮，也想请几位分享一下，接下来一年你们会更关注什么？以及在番禺这个土壤里，你们还希望补上什么能力？
翁淑蓁：未来一年乃至更长的时间内，探迹科技仍会在B2B领域坚定地推进全销售场景的智能体化，比如整合国内和出海团队，构建一个能理解、会思考、可服务的全球产业链智能体。目前公司在国内服务了5万多家企业，但对于中国的企业体量而言，仍然有非常多的中小企业没有用上AI，我们希望把产品技术门槛降低，让AI产品的上手度提升，让更多中小企业享受智能化升级的红利。对于番禺而言，它已经为我们提供了客户、产业、全球化视野的土壤，如果还能为我们做些什么的话，期待它能帮助我们构建更多与需求方面对面的场景，让我们的产品功能能够更好地在产业场景中落地。我相信番禺敢闯敢试的精神下，一些新的技术、新的模式肯定能够落地到场景中跑起来，而番禺不管在智能制造还是数字经济上，都会是一个典范。
邱辉：浩云科技业务覆盖很多行业，接下来一年非常核心的关注点是能源行业，像去年和华中科技大学合作的怎么提高发电效率。布局聚焦能源，也是因为这是AI产业链的上游，未来是能源的效率，能源等于算力，可能就等于我们未来的能力。关于希望番禺提供的帮助方面，我们希望是支持公司在需要从0到1跨越的试点布局，能够在区里先行先试，再往全国复制。
柒成华：一小时科技后续重点关注的方向还是构建自身业务生态。我们也提出了人车家的生态，当然我们关注的是后市场生态，这个生态我们也是以番禺为中心，从番禺延展，包括建设人工智能终端的制造跟智能服务中心，和暨大、华南理工等高校共建产教融合中心，将后市场的产业闭环跟产业聚集落地在番禺，这是未来我们重点做的事情。
