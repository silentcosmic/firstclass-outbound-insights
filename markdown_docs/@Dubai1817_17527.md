# 伊朗对霍尔木兹海峡航运划定红线，警告处置

🚨 伊朗对霍尔木兹海峡航运划定红线，警告处置
伊朗伊斯兰革命卫队海军近日通过社交媒体发表声明，要求所有通过霍尔木兹海峡的船只必须与其协调。声明严厉指出，任何未经协调宣布的所谓“新航线”都不可接受且极其危险，违规船只将面临相应处置。革命卫队海军强调，航运仅限伊朗公布的航线，在指定航线以外航行将面临严重风险。
与此同时，伊朗外交部发言人巴加埃在社交媒体上强烈谴责美国政府在对待伊朗人民问题上缺乏诚意。他指出，美方就某谅解备忘录发表的相互矛盾言论，不仅加剧了伊朗民众的不信任感，更让人联想起美国屡次违约的历史。巴加埃强调，美国必须理解“承诺对等”原则，履行义务，而非曲解协议文本。尽管伊朗抱有疑虑，但仍以善意参与外交进程并签署了旨在结束“强加战争”的谅解备忘录，但考虑到过去50年的经验，伊朗将保持高度警惕。
国际社会关注的另一起事件是美国总统特朗普就今年2月伊朗一所女子小学遭袭事件的表态。特朗普表示，可能永远无法查明该袭击的责任方。此前，米纳卜市一所小学被“战斧”导弹击中，导致至少175人身亡，其中多数为儿童。伊朗初步调查指袭击由美军发动，但美方至今以调查未结束为由拒绝承认。特朗普此前在G7峰会上曾称，“没人故意那样做……犯错难免，战争是肮脏的”。
