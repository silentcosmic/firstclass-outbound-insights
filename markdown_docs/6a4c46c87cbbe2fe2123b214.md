# 小米深化全球布局：携手Adyen整合18个市场支付服务

为进一步优化全球市场运营效率，中国科技巨头小米已与全球支付平台Adyen达成战略合作，旨在全面简化其在18个主要市场的线上及线下零售渠道的支付流程。
根据此次合作协议，Adyen将负责为小米在新加坡、日本、阿拉伯联合酋长国、澳大利亚、墨西哥以及欧盟等关键市场的消费电子业务提供端到端的支付处理服务。
小米目前通过其官方电商平台及遍布全球的实体零售网络，已成功将其创新产品和服务覆盖至全球逾100个国家和地区。
为实现支付管理的现代化与统一化，小米将采用Adyen领先的统一商务（Unified Commerce）解决方案，通过单一集成平台，高效管理和处理所有线上及线下门店的交易支付。
Adyen方面表示：“我们非常高兴能够支持小米的国际化发展进程。小米作为消费电子领域的领军企业，正身处一个以快速创新和激烈竞争为特征的充满活力的行业。”
“通过提供安全且可扩展的支付能力，我们助力小米轻松便捷地处理全球范围内的交易，从而聚焦核心业务增长。”
