# 广州警方成功劝阻280万“发财大计”骗局

🚨 广州警方成功劝阻280万“发财大计”骗局
近日，广州市公安局黄埔分局反诈中心通过银行预警，成功劝阻一起新型网络投资理财诈骗，为一名市民挽回了280万元的经济损失。
据悉，黄埔区某银行发现客户侯某异常预约支取280万元现金后，立即通报警方。民警迅速赶赴现场拦截，发现侯某正深陷一种结合了网络交友、虚假投资与线下取现的复合型诈骗。
侯某因被深度洗脑，起初不仅以“商业秘密”为由刻意隐瞒，甚至怒斥劝阻民警“破坏发财大计”，并扬言将分次取现以规避监管。反诈民警随后上门进行深度劝阻，逐条拆解诈骗套路，最终使其彻底醒悟。
侯某承认此前已按骗子指令三次线下取现追加投资。此次警方及时介入，成功保住了这笔巨款。
