# 印尼特大网赌案：52亿资金流转，287名外籍人员涉案

🚨 印尼特大网赌案：52亿资金流转，287名外籍人员涉案
印尼警方近日宣布，成功破获一起特大跨国网络赌博案件，该案涉及高达52.82亿元人民币的充值总额，并逮捕了287名外籍嫌疑人。此次行动于5月7日在雅加达哈雅·乌鲁克区域一写字楼内进行，警方于6月26日召开发布会，公布了最新侦查细节。
据警方通报，这287名嫌疑人来自多个国家，其中越南籍185人，中国籍76人，其余来自缅甸、泰国、老挝和马来西亚。该团伙通过145个赌博网站，累计收取全球赌客充值达13.9万亿印尼盾（约合人民币52.82亿元），非法获利1.69万亿印尼盾（约合人民币6.42亿元）。
为逃避监管，该团伙伪装成科技或数字营销公司，利用代持账户及虚拟货币USDT进行多层资金洗白。其赌博网站的IP和服务器分散部署在巴西、菲律宾、中国、越南等地，并通过轮换站点规避印尼的网络封禁。此前，该团伙在柬埔寨、马来西亚、缅甸等国长期活动，因当地打击力度加大而转移至雅加达设立新据点。
目前，印尼警方已查封大量涉案本土银行账户，现场扣押85亿印尼盾现金（约合人民币323万元）及多国外币，并正与本国金融交易报告与分析中心（PPATK）合作，深入追查巨额非法资金的最终流向，全力缉捕幕后主使。
