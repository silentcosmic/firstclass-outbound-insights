# 泰国截获走私燃油：指向缅甸诈骗园区与犯罪集团

🚨 泰国截获走私燃油：指向缅甸诈骗园区与犯罪集团
泰国警方近期在清莱府清盛县成功破获一个大型燃油走私网络。7月13日，警方联合海关在湄公河水域截获一艘油轮，该船谎报目的地为老挝，实则驶向缅甸，船上查获近40万升柴油。
据泰国当局深入调查，该网络已向缅甸走私超过3650万升燃油，导致泰国税收损失高达2.26亿泰铢。警方指出，涉案船只数量不止一艘，这些走私燃油极有可能流向缅甸境内的诈骗园区以及盘踞当地的跨国犯罪集团，为他们的非法活动提供能源支持。
目前，泰国当局正全面扩大调查范围，旨在追查幕后资助者和组织者，并将依法追究相关责任。
