# 泰国清迈重拳打击非法导游：缅籍男子落网

🚨 泰国清迈重拳打击非法导游：缅籍男子落网
7月13日，泰国清迈警方在古城区域成功逮捕一名39岁的缅甸籍男子。该男子涉嫌长期持旅游签证非法从事导游工作，其活动据称已持续约三年。
警方调查发现，嫌疑人不仅未持有合法工作许可和导游资质，还利用不符合规定的车辆提供带团服务。目前，该男子面临非法务工、无证从事导游职业等多达5项指控。
值得注意的是，导游在泰国属于法律限制外国人从事的职业。近期，泰国有关部门持续加强对旅游、餐饮、零售等行业外籍劳工的检查力度，重点打击持旅游签证非法工作、无证经营及违规从事限制性职业等行为。此案的查处，再次凸显了泰国对非法劳务的零容忍态度。
该男子已被移交相关部门依法处理，案件正在进一步调查中。
