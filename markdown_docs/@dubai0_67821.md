# 日本签证费拟飙升5倍，出海成本将显著增加

⚡ 日本签证费拟飙升5倍，出海成本将显著增加
日本政府正计划从2026财年起大幅上调外国人签证发放手续费。其中，一次入境签证的费用预计将从目前的约3000日元提升至1.5万日元，涨幅高达五倍。多次入境签证的费用也在讨论中，可能设定在3万日元左右。此项调整已纳入2026年度预算案，值得注意的是，现有签证手续费自1978年以来一直未作调整。
与此同时，日本还计划提高国际观光旅客税，并同步将本国公民10年期护照的手续费从1.6万日元下调至约9000日元。这一系列政策调整，或将对赴日旅游、商务及长期居留的外国人产生直接影响，同时也反映出日本政府在入境管理和财政收入方面的最新考量。
