# 新加坡金管局启动2026年全球金融科技加速营及SFF卓越奖评选

2026年全球金融科技加速营（Global Fintech Hackcelerator）与新加坡金融科技节（SFF）卓越奖的申请通道已正式开启，为即将到来的新加坡金融科技节预热。新加坡金融管理局（MAS）于6月29日正式发布了这一征集通知。这两项计划的获奖名单，均定于2026年11月19日举行的新加坡金融科技节卓越奖晚宴上揭晓。
全球金融科技加速营：聚焦AI创新
由新加坡金管局（MAS）联合全球金融科技网络（GFTN）共同主办的“全球金融科技加速营”项目，正面向全球金融科技企业及解决方案提供商征集提案，重点关注人工智能（AI）如何有效应对金融服务领域的核心挑战。本年度挑战赛聚焦三大关键领域：数字银行中的信用与欺诈风险建模、面向数字原生代的财富管理解决方案，以及助力中小企业评估并管理商业风险的工具。
2026年的加速营得到了GXS、宝盛银行（Julius Baer）和苏黎世保险（Zurich Insurance）的鼎力支持，它们作为企业合作伙伴（corporate champions）参与其中，并各自在其专业领域贡献了具体的挑战命题。
该项目将选出最多20支决赛队伍。每支入围团队将获得2万新元的现金资助，并有机会与其企业合作伙伴紧密协作，接受专业指导。随后，他们将于2026年11月11日至13日赴新加坡参加现场集训。各团队将在2026年11月18日新加坡金融科技节期间举行的“全球金融科技加速营路演日”上，展示其创新解决方案。每个挑战命题将评选出一名获胜者，每位获胜者将获得8万新元的奖金。竞赛结束后，企业合作伙伴还可根据双方意愿及协议，从决赛队伍中挑选一支团队进行试点项目合作。
SFF金融科技卓越奖：表彰行业先锋
与此同时，新加坡金管局也同步开放了2026年新加坡金融科技节卓越奖的申请与提名通道。该奖项由新加坡金融科技协会（SFA）协办，并获得普华永道新加坡（PwC Singapore）的支持。该奖项旨在表彰那些在推动行业新实践、促进金融普惠、加速新兴技术采纳以及提升金融服务交付方面做出杰出贡献的金融科技解决方案及个人。
2026年新加坡金融科技节期间将揭晓八位获奖者。企业类奖项设“新兴金融科技”、“金融普惠”、“监管领导力”、“可持续创新者”以及“人工智能先锋”等类别。此外，还将有三位个人被授予“金融科技导师奖”。各企业类别的获奖者将获得5万新元的奖金，而三位金融科技导师奖获得者则每人获颁5千新元。
此次奖项的举办也得到了“金融科技回馈”（Fintech Gives Back）倡议的支持。该倡议于2023年启动，旨在鼓励行业内成熟企业积极支持金融科技领域的早期创新。2025年，参与捐助的企业包括Ripple、Visa、HeyMax、Aspire、益博睿（Experian）、律商联讯（LexisNexis）、Sumsub、Syfe和YouTrip。
“全球金融科技加速营”和“新加坡金融科技节卓越奖”的申请截止日期均为2026年8月14日。2026年新加坡金融科技节将于11月18日至20日举行。
