# 中泰联手：307名涉诈人员从泰遣返回国

🚨 中泰联手：307名涉诈人员从泰遣返回国
近日，共有307名中国籍人员经泰国被遣返回国。此次遣返人员中，239人来自缅甸妙瓦底地区的电信诈骗及网络赌博园区，另有68人系在泰国境内非法居留的中国公民。相关人员在完成身份核查及移交手续后，已统一从泰国来兴府湄索国际机场搭乘中国南方航空航班返回中国，后续将由中方有关部门依法进行调查和处理。
此次行动是中泰两国持续加强跨境执法合作、联合打击电信网络诈骗、网络赌博及非法出入境等跨国违法犯罪的重要成果。此举也再次释放出区域国家将持续深化联合执法，严厉打击跨境违法犯罪活动的明确信号。
