# ️ 出海企业警示：前高管涉嫌职务侵占及个人操守问题

⚠️ 出海企业警示：前高管涉嫌职务侵占及个人操守问题
近日，市场消息指出，一名曾就职于迪拜破浪集团（Silicon A7）的马来西亚籍前员工Mak Jun Wei（化名“阿喵”）被曝光涉嫌在任职期间私吞公司款项。据称，其涉案金额超过20万，且长时间未被察觉。
该员工被指擅长伪装，在迪拜“大扫荡”后回国，生活奢靡，并涉及吸毒等不法行为。此外，其还被曝以各种理由向身边朋友借钱不还，甚至利用护照骗取路费。这些行为不仅暴露了其个人操守问题，也为企业在背景调查和人才引进方面敲响了警钟。
鉴于当前出海市场环境复杂，业内人士呼吁各企业加强对员工的背景审查与风险管理，谨防类似事件再次发生，避免因识人不明而带来不必要的损失。
