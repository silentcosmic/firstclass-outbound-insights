# ️ 越南警方破获电诈案：11名中国籍嫌犯专骗法国人

⚠️ 越南警方破获电诈案：11名中国籍嫌犯专骗法国人
近日，越南北宁省警方成功捣毁一个跨境电信诈骗窝点，抓获11名中国籍嫌疑人。据了解，该团伙主要针对法国投资者实施诈骗。
现场查获包括11台笔记本电脑、12部手机及大量电子数据，这些均被视为关键作案工具。此次行动再次凸显了海外市场在跨境犯罪防范方面的挑战。
