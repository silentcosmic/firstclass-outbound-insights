# AI驱动公民社会远程驰援：委内瑞拉灾后数字化响应与挑战

在委内瑞拉北部地区上月遭遇双重地震灾害后，成千上万的民众涌向社交媒体，急切地寻求帮助以寻找失联的亲友。面对官方响应的迟缓，援助却意外地迅速来自一个群体：开发者与程序员。
“谁知道政府何时才会做出响应？”居住在布宜诺斯艾利斯的31岁委内瑞拉程序员豪尔赫·巴斯蒂达斯表示，“我们决定立即采取行动。”
巴斯蒂达斯与他的六人团队创建了“委内瑞拉地震失联者”（Desaparecidos Terremoto Venezuela）网站，旨在帮助公民报告、识别失联人员并与家人团聚。该网站利用Claude Opus 4.8模型以及墨西哥公司Lab-Co捐赠的面部识别软件构建，巴斯蒂达斯将其设计为快速加载，且无需用户注册或下载应用程序。他透露，网站上线前两天内，便收到了超过3万份失联人员报告。
巴斯蒂达斯指出，如果没有AI技术，“要构建一个耗时三小时完成的系统，我至少需要不眠不休地工作约24小时。”
此次地震造成超过3800人死亡，大量房屋和医院被毁。包括海底光缆和移动网络基站在内的电信基础设施遭受严重破坏，停电问题进一步加剧了互联网接入的挑战。
在国家支持几乎完全缺位的情况下，委内瑞拉公民和像巴斯蒂达斯一样的海外侨民纷纷介入。地震发生后的数小时内，身处迈阿密、旧金山、智利、阿根廷及其他地区的志愿开发者们利用AI技术，迅速搭建了各类网站，用于报告失联人员、协调物资捐赠、评估建筑物结构损坏、绘制医院容量图、定位避难所，甚至帮助宠物与主人重聚。
“AI绝不应取代国家的法律责任或其问责机制。”Fundación InternetBolivia.org研究员妮可·桑切斯强调。
身在加利福尼亚的委内瑞拉开发者塞缪尔·马里尼亚创建了“援助在路上”（Ayuda en Camino）网站，这是一个为援助组织、志愿者、捐赠中心和需要帮助的人们设计的平台。他利用Replit在短短四小时内完成了网站的搭建。考虑到受灾地区有限的网络连接，马里尼亚表示，用户可以通过一个WhatsApp助手查询捐赠和需求，而无需直接访问网站。
他解释道：“互联网上充斥着碎片化的信息，人们很难了解哪些物资可用以及哪些方面存在需求。我们希望集中信息，确保援助不被浪费，并精准送达最需要的地方。”
AI工程公司Henkki的首席执行官豪尔赫·路易斯·查孔，在地震发生数小时后便与宗教组织“国际陪伴部”（Ministerio de Acompañamiento International）共同开发了“我们陪伴”（Somos Acompañamiento）平台。除了失联人员报告，该平台还列出了医院中的幸存者和伤员名单。查孔透露，该平台已记录超过8.4万次注册。此外，他正与人权组织Laboratorio Migrante合作，开发一套AI系统来监测人口贩运活动。
官方资源匮乏的现实，也促使记者们开始利用AI技术收集数据。委内瑞拉女性记者网络利用AI工具Lovable创建了一个搜索工具，用于汇编来自医疗中心、医院和避难所的数据。该网络成员瓦莱里亚·佩迪奇尼表示，委内瑞拉本土和海外的记者会对所有数据进行核实。她认为，鉴于官方信息的缺失，额外的验证步骤“至关重要”。
数字安全专家、智利圣地亚哥非营利组织Acceso Libre的创始人玛丽安娜·迪亚斯·埃尔南德斯，在目睹社交媒体上充斥着幸存者绝望的求助信息后，随即搭建了terremotove.com。她利用Claude Code和Kobo Toolbox，仅用一小时便构建了一个应急响应地图，用于定位倒塌建筑物、搜索失联人员并发布与地震相关的任何信息。她回忆，当她推出该网站时，市面上至少已有三款类似平台。
埃尔南德斯感叹：“国家本应承担的所有职责，现在都由公民社会在履行。人们本可以将精力投入到其他地方，但我们所做的，是用我们所拥有的一切，尽力填补国家的角色空白。”
数字权利组织Fundación InternetBolivia.org的研究员妮可·桑切斯指出，在委内瑞拉正在经历的这类人道主义危机中，AI工具能够帮助分析海量数据，从而促进家庭团聚，为民众定位最近的可用资源，并优化援助分配。
她进一步阐释：“AI可以补充这些功能，使其更高效。但它绝不应取代国家的法律责任或其问责机制。”
公民们普遍认为，委内瑞拉当局响应迟缓，未能充分应对此次危机——尽管代理总统德尔西·罗德里格斯对此予以否认。
随着AI越来越多地应用于协助受冲突、灾难及其他人道主义危机影响的人群，专家警告称，这些系统需要人工监督，以确保数据隐私，并最大程度地减少可能对用户造成严重后果的错误。
桑切斯强调，在委内瑞拉，尽管速度至关重要，但开发者必须确保数据以负责任的方式收集和存储，并具备“比平时更强”的充分保障措施。
她指出：“这可能看起来像是我们可以稍后再考虑的问题，或者认为在人道主义危机中这不是优先事项。但这确实是一个实际问题，尤其涉及生物识别等敏感数据。”
“是人赋予了这些工具价值。”程序员豪尔赫·巴斯蒂达斯如是说。
巴斯蒂达斯表示，“委内瑞拉地震失联者”网站被设计得“简洁”，便于人们访问和上传信息，但也因此遭遇了网络攻击。随后，他们加强了安全防护。
三周过去了，国际援助正在陆续抵达委内瑞拉，但由公民主导的网站和应用程序仍是主要的信息来源。网络安全顾问豪尔赫·桑切斯-普里梅拉表示，尽管这些平台加速并民主化了关键数据的获取，但它们无法取代政府的援助职能。
他分析道：“目前，信息碎片化、不准确或重复，这还未考虑隐私和数据保护风险。国家应该集中精力，并保护民众的隐私。”
与此同时，一些开发者已将目光投向潜在的未来危机。阿图罗·涅韦，一位创建了失联人员登记系统Civis的开发者，正在对其进行调整，以应对他认为更严峻的问题：洪水和山体滑坡预警系统的几乎缺失。涅韦利用开源卫星数据和降雨预报，通过短信和WhatsApp向脆弱的农村社区发送警报。
巴斯蒂达斯透露，“委内瑞拉地震失联者”团队计划继续工作约三个月，并将在未来发生灾难时保持网站对公众开放。
尽管情况悲惨，他认为“看到公民的行动能力”是一个积极的结果。
巴斯蒂达斯表示：“不到一天的时间里，全球数百项不同的倡议通过AI被激活。但AI只是一种工具，它不能替代人类……是人赋予了这些工具价值。”
