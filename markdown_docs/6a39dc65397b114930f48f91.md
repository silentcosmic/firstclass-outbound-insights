# Sea集团联手OpenAI，为Shopee用户及商家拓展AI应用深度

扎根新加坡的电商与游戏巨头Sea集团宣布，将与OpenAI携手推进人工智能在东南亚及巴西市场的普及，尤其侧重于AI驱动的电商应用。
Sea集团周一发布声明指出，其旗下知名电商平台Shopee已与OpenAI签署一项全新战略合作协议。此举旨在为Shopee的用户与商家提供更广泛的ChatGPT访问权限，并加速开发者群体对OpenAI Codex的采纳与应用。
作为此次合作的实质性进展，Shopee应用程序现已在ChatGPT中上线，覆盖印度尼西亚、马来西亚、菲律宾、新加坡、泰国、中国台湾、越南及巴西等市场的用户。此举使Shopee成为首个在ChatGPT中，同时服务于东南亚及拉丁美洲市场的电商平台。
Sea集团强调，这项集成功能使用户能够通过对话式提示探索Shopee商品，随后在Shopee平台内继续其购物流程。
Sea集团董事长兼首席执行官李小冬表示，此次合作体现了双方共同致力于让AI在其服务社区中更易用、更有意义的承诺。通过融合OpenAI的AI能力与Sea的生态系统，将为消费者带来原生的AI体验，并为商家提供有力的增长工具。
OpenAI首席营收官Denise Dresser指出，此次合作将助力更多消费者体验AI赋能的创新，并为小型企业提供实用的增长工具。
此外，此次合作还将把ChatGPT商业版引入Shopee的商家生态系统，赋能商家更高效地创建商品列表与营销内容、优化客户服务流程，并通过试用、入驻资源及培训项目支持业务运营自动化。
更进一步，Sea集团与OpenAI于六月初在新加坡共同启动了亚太地区首个Codex系列编程马拉松活动，吸引了超过1200份申请。未来数月内，双方还计划在越南、中国台湾、印度尼西亚和马来西亚举办更多场次。
