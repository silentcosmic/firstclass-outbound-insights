# ️ 马来西亚导航失误致深坑坠车：5名中国公民重伤

⚠️ 马来西亚导航失误致深坑坠车：5名中国公民重伤
7月1日晚间，马来西亚吉兰丹州发生一起严重交通事故。五名中国籍公民在返程途中，疑似因导航错误驶入未开放的施工路段，导致车辆坠入约7米深的深坑中并发生翻覆，车上五人均受伤送院治疗。
据悉，受害者包括三名来自中国浙江的游客，以及两名在当地一家养鱼饲料生产企业工作的中国公民。事故造成多人不同程度受伤，其中一名女性伤势尤为严重，盆骨、手脚多处骨折，下半身暂时失去知觉，目前仍在高依赖病房接受密切观察。另有伤者左脚骨折，原定返华行程受阻，可能需进行手术。
事发后，中国驻马来西亚领事馆吉兰丹州志愿领保员已前往医院探望伤者，并协助处理相关事宜，持续关注伤者康复进展，同时感谢当地消防、警方和医护人员的及时救援。
