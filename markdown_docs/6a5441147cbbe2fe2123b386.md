# 膳食补充剂创新原料领军者纽邦生物完成近2亿元C+轮融资，年营收超4亿赋能全球600余品牌

南京纽邦生物科技（NNB Nutrition）近日成功完成近2亿元人民币C+轮融资。本轮投资由国寿股权领投，长期合作的明熙资本持续跟投，募集资金将重点用于深化产品研发、加速新工厂建设。这标志着纽邦生物在半年内实现了第二笔超亿元人民币融资，凸显其在健康科技领域的强劲增长势头。
纽邦生物深耕膳食补充剂创新原料的研发与生产，至今已成功推出逾40款自主研发的原创原料，并拥有超过100项在研产品管线。其产品应用广泛，涵盖运动营养、体重管理、认知健康及抗衰老等多个前沿健康领域。目前，公司已为雀巢、联合利华、百事等全球600余个知名终端品牌提供核心原料支持。为强化其产业布局，纽邦生物已自建制剂工厂与生物合成工厂，并已启动规划建设一座占地逾200亩、年产能达万吨级的综合性生产基地。据公司预测，到2025年，纽邦生物的全年营收有望突破4亿元人民币大关。
纽邦生物创始人廖琪林先生在膳食补充剂行业深耕逾十年，凭借其对市场的敏锐洞察，他指出近年来健康消费市场正经历两大核心转型。首先，消费者需求正从单纯的基础营养补充，转向更为个性化、精准化的健康调理方案。其次，便捷性已成为驱动消费决策的关键要素，消费者日益倾向于通过日常饮食和饮品，直接获取所需的营养与调理功效。这一趋势深刻影响了下游终端品牌的产品策略，促使他们从传统的片剂、粉剂、软糖等形态，积极转向布局更符合日常消费场景的功能性饮料与特色功能食品。“这类新型产品形态已然成为市场主流。下游需求端与产品形态的深刻革新，也重塑了终端品牌对我们这类上游原料供应商的甄选标准。如今，除了功能功效和法规合规等基本要求外，他们更加重视原料在这些新剂型、新应用场景中的实际表现，例如是否具备更卓越的稳定性、更佳的溶解性、更强的口感适配性以及更广泛的配方兼容性。”廖琪林先生如此阐释道。
这些市场变化向上游环节传导，意味着原料企业必须同步提升研发效率、强化研发能力并拓宽覆盖广度。为此，纽邦生物已对内部原料筛选标准进行了全面升级，确立了Pure（纯净）、Potent（高效）、Precise（精准）的“3P”核心标准。该标准严格要求所开发原料不仅需具备高纯净度和显著功效，更要在使用剂量上实现“足够精准、足够小”的卓越表现。
廖琪林进一步指出，纽邦生物的独特之处在于，公司始终坚持以创新药物的研发逻辑来开发膳食营养原料，这与行业内多数原料厂商的做法截然不同。其核心要点在于，纽邦生物会依据“比肩创新药研发的标准来设计并执行相关的临床试验研究，并将详尽数据提供给合作方”，以此严谨地验证产品的安全性和功效性。截至目前，纽邦生物已成功开展数十项临床前及临床试验，并计划以每年启动逾10项新试验的速度持续推进其科研验证工作。
支撑这种高标准临床研发体系的，是纽邦生物构建的多元技术融合研发矩阵，其核心涵盖合成生物学、化学合成、植物提取及制剂四大关键技术路线。“这种‘多端融合’的战略布局，让我们能够摆脱单一技术路径的束缚，真正以市场终端需求为导向，灵活调配各类工艺资源，显著提升研发效率。目前，我们已有多款创新产品通过这种协同模式成功完成开发。”
据内部资料披露，纽邦生物曾开发一款颇具代表性的创新产品。项目研发初期，团队原本规划依托合成生物学工艺实现规模化量产。然而，该品类在行业中属于全新创新领域，公开的技术研究储备极为稀缺，加之公司自身的配套工艺体系尚未完全成熟，使得直接采用生物合成路线面临诸多挑战。
为把握市场先机、迅速抢占赛道形成先发优势，纽邦生物团队果断决策，率先落地化学合成工艺，成功实现产品的商业化投放。此后，尽管团队成功攻克了合成生物学相关的技术瓶颈，但在实践中却发现，无论是单独采用化学合成还是合成生物学方法，都存在各自的局限性。最终，通过复合工艺的整合应用，纽邦生物找到了最优解决方案，实现了生产效率与产品稳定性的双重显著提升。
在谈及近两年大批合成生物概念企业涌入营养原料赛道时，廖琪林先生分析指出，两类企业的核心竞争领域并不完全重合。他强调，纽邦生物主要聚焦于“未曾被商业化的创新原料研发”，而部分合成生物公司则侧重于通过低成本发酵工艺，将“现有产品以新的生产方式进行重构”。因此，他认为这些新兴力量并不会对纽邦生物构成直接的竞争冲击。“事实上，我们也在积极构建自身的合作伙伴生态体系。如果这类合成生物学企业在某个特定技术点或品类上拥有显著优势，我们非常乐意与他们携手合作，共同开发新产品。同时，对于他们已成功研发并具备技术优势的产品，我们也可利用纽邦生物完善的商业化体系，助其实现更广泛的市场落地。”
在商业化战略层面，与传统的原料贸易销售模式不同，纽邦生物当前的核心差异化策略是其主推的“要素品牌（Ingredient Branding）”模式。这种模式的核心在于，核心供应商面向终端消费者塑造并推广其独立品牌，通过消费者对该关键要素品牌的认知与认可，进而驱动下游终端品牌主动选择并使用其产品，从而形成一种互惠互利的双向赋能效应。业界成功的范例包括以“Intel Inside”闻名的英特尔、影音技术领域的杜比（Dolby）、徕卡（Leica）光学镜头，以及宁德时代（CATL）的动力电池等。
廖琪林透露，经过多年的市场实践与深耕，纽邦生物的要素品牌“在行业市场中的影响力正持续攀升”。目前，已有数百家终端品牌选择在其产品标签上显著展示并宣传纽邦生物的原料品牌，这无疑是对其品质与创新的高度认可。他举例称：“我们曾与知名名人卡戴珊相关的品牌合作开发一款产品，其中便采用了纽邦生物的两款核心原料作为关键配方。卡戴珊本人也曾在多个场合、不同内容中提及并推广过这两款原料。”
此外，针对有志于全球化发展的跨境营养品牌，纽邦生物不仅提供优质原料，更可配套提供全面的全球化代工体系。这包括整合美国、欧洲等多地的合规工厂生产资源，实现海外成品的“一站式”落地服务。同时，纽邦生物还可凭借其丰富的海外渠道资源，协助合作品牌高效对接当地线下零售终端，打通海外上架销售通路，从而助力合作伙伴深度开拓国际消费市场。
国寿股权方面对此轮投资评价道：“创新功能原料是膳食营养补充剂行业构建坚实壁垒、实现品质提升与规模扩张的关键支点。伴随大众健康消费意识的不断增强，以及全球功能原料市场需求的稳步增长，整个行业正迈入由创新驱动的新发展阶段。纽邦生物长期致力于创新功能原料赛道，在多个细分领域持续深耕，为老年群体及更广泛消费者提供前瞻性的健康产品解决方案。我们期望通过本次投资合作，能够助力纽邦生物在全球膳食补充剂市场中开辟出独特的中国创新之路。”
明熙资本则表示：“膳食补充剂兼具大健康与消费品的双重属性。随着全球消费者健康管理意识的普遍提升，以及终端产品形态从传统的营养补充品向更日常化、场景化的功能食品与功能饮品拓展，创新功能原料正成为驱动行业升级的核心变量。纽邦生物始终坚持以创新药物的研发逻辑来开发膳食营养原料，并围绕多元技术路径，持续构建产品研发、临床验证、专利保护以及要素品牌等多维度壁垒，从而形成了兼具研发深度与商业化效率的平台型创新能力。作为长期支持的老股东，明熙资本持续看好纽邦生物对终端需求的深刻洞察、其持续推出高品质创新原料的卓越能力，以及其面向全球市场的长期成长潜力。”
