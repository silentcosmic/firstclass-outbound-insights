# 老缅建交70周年：两国深化多领域合作

🤝 老缅建交70周年：两国深化多领域合作
2026年7月3日至5日，缅甸总统敏昂莱应老挝总统唐伦思利邀请，对老挝进行正式访问。此访旨在纪念两国建交70周年，并全面推动双边合作深入发展。
访问期间，两国元首举行了正式会谈，回顾了过往合作成果，并就国防安全、经济贸易、教育等多个领域的未来合作前景进行了深入探讨。双方一致同意加强磋商，共同促进两国关系的持续发展。唐伦思利总统高度评价了此次访问的政治意义，并赞赏缅甸在新政府组建后所取得的发展成就。
两国领导人还就东盟、联合国及百万象河合作（MLC）等多边机制下的区域与国际合作交换了意见。会谈结束时，敏昂莱总统邀请唐伦思利总统适时访问缅甸，并获得了积极回应。访问期间，缅甸代表团还就旅游合作与新加坡签署了谅解备忘录，并参与了公共卫生合作活动。
此次高层访问不仅巩固了老缅两国深厚的传统友谊，也为双方在未来实现多领域互利共赢奠定了坚实基础。
