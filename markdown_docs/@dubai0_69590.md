# ️ 柬泰边境生变：泰军占领电诈园区引柬方抗议

⚡️ 柬泰边境生变：泰军占领电诈园区引柬方抗议
近日，柬埔寨媒体报道称，泰方军事力量已进入柬泰边境的奥斯玛（O'Smach）口岸区域，并占领了当地多个曾闻名国际的电信诈骗园区。此举引发柬埔寨政府强烈不满。
据了解，奥斯玛地区因其多个大型诈骗园区而臭名昭著。柬埔寨方面坚决反对泰方此番行动，视其为“侵犯柬埔寨领土”的行为，边境局势骤然升级。
