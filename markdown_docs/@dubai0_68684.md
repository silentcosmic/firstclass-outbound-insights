# ️ 迪拜雇佣纠纷警示：温超员工曝合同细节不符

⚠️ 迪拜雇佣纠纷警示：温超员工曝合同细节不符
市场消息显示，有员工在迪拜温超超市分享了其在职期间的真实经历，揭示了公司在薪资构成、工时计算及签证护照管理方面可能存在的争议。该员工指出，实际工作条件与入职前承诺存在显著差异，引发了对当地劳务合规性的关注。
据该员工披露，入职时劳动合同约定的转正薪资与实际工资单构成不符，其中“底薪”部分远低于预期，且严重影响年假及加班费的折算基数。此外，工作描述中提及的每日10小时（含1小时用餐）实际操作中变为11小时工作时长，用餐时间不计入工时，导致员工在职期间存在大量未获薪酬的工时。
在离职与签证方面，该员工提及公司合同条款规定，入职前六个月内离职需按天数赔偿公司，且签证办理滞后，离职前可能被取消，增加逾期滞留罚款风险。同时，公司“代为保管”员工护照，使用需经内部审批流程，引发对个人证件管理权的担忧。此外，工作场所的设备故障率较高也影响了日常运营效率。此案例提醒出海企业需审慎处理劳务合同细节，确保透明合规，避免潜在的雇佣纠纷及法律风险。
