# 阿布扎比首推自动驾驶车牌：商用测试分类明确

⚡ 阿布扎比首推自动驾驶车牌：商用测试分类明确
阿布扎比近日宣布推出其首个自动驾驶汽车专用车牌，此举标志着该地区在智能交通监管方面迈出了重要一步。
据悉，新的统一蓝色车牌将明确区分不同类型的自动驾驶车辆：“自动驾驶”字样将用于商用自动驾驶车辆，而“测试”字样则专为试验项目车辆设计。此举旨在为自动驾驶技术的商业化落地与研发测试提供清晰的监管框架，进一步推动智慧城市建设。
