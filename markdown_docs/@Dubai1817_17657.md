# 马来西亚加强外籍人员管理：突击检查行动持续进行

🚨 马来西亚加强外籍人员管理：突击检查行动持续进行
马来西亚沙巴州亚庇市伊纳南移民局于7月1日展开突击检查行动，对辖区内多处住宅区、商业场所及工地进行随机排查。此举旨在加强对外籍人员的管理与执法力度，重点核查是否存在非法居留、逾期逗留或违反移民条例的情况。
移民局方面表示，此类突击检查并非孤立事件，而是将持续常态化进行。其核心目标是确保所有在马外籍人员合法居留，并防范任何可能影响当地社会秩序的违法行为。
