# 泰国重拳出击：国际红通赌博头目落网，涉案年入超340万美元

🚨 泰国重拳出击：国际红通赌博头目落网，涉案年入超340万美元
泰国警方与国际刑警组织近日联手，在春武里府芭提雅成功逮捕了一名45岁韩国籍男子金某。该男子因涉嫌组织并运营大型跨境网络赌博及诈骗集团，长期以来被国际刑警组织列为红色通缉犯。
据调查，金某是该犯罪网络的主要头目之一，其控制的多个网络赌博平台，每个年收益最高可达340万美元，非法获利规模巨大。目前，警方正对案件进行深入调查，并追查其犯罪网络及资金流向。
此次逮捕行动，再次彰显了泰国在持续打击跨国犯罪方面的决心与成效。此前，泰国警方在曼谷也成功抓获了两名涉嫌跨国贩毒和洗钱的法国籍红色通缉犯。这表明泰国正不断加强与国际执法机构的合作，严厉打击各类跨境犯罪活动。
与此同时，泰国警方公布的最新数据显示，在过去九个月内，累计逮捕近3万名涉诈犯罪嫌疑人，查封1450家涉诈企业，并依法扣押了总价值约240亿泰铢的涉案资产。这些成果凸显了泰方在打击电信诈骗、网络赌博及相关黑灰产业链上的持续高压态势。
