# 特朗普回应加密财富质疑：强调全民皆获利

💰 特朗普回应加密财富质疑：强调全民皆获利
美国前总统特朗普近日在搭乘新空军一号专机时，公开回应了对其在2025年通过加密货币业务获利12亿美元的质疑，并否认利用职务敛财。特朗普强调，其财富增长主要得益于股市强劲表现及过往商业成就，并声称“大家都在获利”。
根据公开文件披露，特朗普被指在2025年将从其子参与创立的加密货币公司World Liberty Financial (WLF) 获得近5.5亿美元，并从“$TRUMP”代币授权中赚取6.35亿美元。批评者指出，其前总统任期内放宽加密货币监管政策，间接推动了相关资产价格飙升。《福布斯》杂志估算，其个人财富已从23亿美元激增至65亿美元。
