# 阿联酋全球首例：空中出租车起降场获官方认证

🚀 阿联酋全球首例：空中出租车起降场获官方认证
阿联酋近日创造历史，成为全球首个正式批准商用空中出租车起降场（vertiport）的国家，此举标志着城市交通进入新纪元。
位于迪拜国际机场旁的VDX起降场已正式获得阿联酋通用民航局（GCAA）的认证，将作为电动空中出租车（eVTOL）的核心枢纽。目前，迪拜正同步推进另外三个起降场的建设，以构建完整的空中出行网络。
GCAA强调，此次认证经过了涵盖建筑安全、操作流程及应急措施等多个方面的严格审查。民航局局长表示，这不仅是阿联酋航空业的里程碑，更彰显了该国在未来交通创新领域的领导地位。
