# ️ 台湾灵骨塔二次诈骗：教授四年被骗2300万

⚠️ 台湾灵骨塔二次诈骗：教授四年被骗2300万
台北警方近日侦破一起涉及黑帮的“二次诈骗”案，逮捕了包括竹联帮弘仁会和天道盟成员在内的9名嫌犯。
该诈骗集团专盯曾购买灵骨塔位或生基位的受害人，先以高价收购为诱饵，随后要求受害人补购生基罐、底座等殡葬周边产品。为营造交易真实性，诈骗团伙还安排成员假扮合资者或投资人，进行配合演戏，诱使受害人上当。
其中，一名国立台北教育大学副教授因急于脱手手中塔位，在四年内陆续支付超过2300万新台币，几乎耗尽多年积蓄。警方初步清查，全案至少有7名受害人，诈骗总金额逾4290万新台币。现场查获犯罪文件、手机、现金及保时捷等豪车，案件已依法移送侦办。
