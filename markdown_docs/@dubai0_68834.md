# 空姐涉毒案升级：牵出两大跨国贩毒集团

🔍 空姐涉毒案升级：牵出两大跨国贩毒集团
泰国航空一名乘务员于6月25日在澳大利亚因涉嫌走私海洛因被捕一案，调查持续深入。泰国国家禁毒委员会办公室缉毒局局长卡尼颂于7月2日披露了案件最新进展，指出调查正逐步揭示其背后复杂的犯罪网络。
据透露，在澳大利亚联邦警察与边境执法局和泰国禁毒办联合情报交流后，调查人员根据嫌疑人供述掌握了关键线索。嫌疑人称受一名使用Facebook账号“Rose”的人士雇佣，并约定抵达墨尔本后由一名昵称“Dear”的女子接货。目前，澳方正对嫌疑人手机进行司法鉴定，以期尽快锁定包裹最终接收人。
泰国禁毒办调取了嫌疑人住所公寓的监控录像，发现一名身穿黑色衣物、戴连帽衫的男子曾驾驶黑色轿车将包裹送至其住处。针对已注销的Facebook账号“Rose”，泰国方面正与Facebook美国总部及美国驻泰缉毒执法机构紧密合作，全力追查数字足迹，以锁定幕后雇佣者的真实身份。
卡尼颂还透露，该犯罪网络采用隐蔽手法，将条状海洛因加工压制成薄片，夹藏于布袋或地垫内衬中，以规避安检。更深层调查发现，目前至少有两个跨国贩毒集团正利用泰国作为毒品中转站，将毒品从邻国经泰国东北部边境运入，随后运至曼谷及周边地区暂存、重新包装，最终伺机偷运至澳大利亚。
