# ️ Telegram核心域名突遭禁用：疑与OFAC制裁相关

⚠️ Telegram核心域名突遭禁用：疑与OFAC制裁相关
即时通讯巨头Telegram的核心域名t.me近日遭遇全球范围内的访问障碍，因其已被.me域名管理机构禁用。Telegram官方团队尚未对此事件发表评论。
为应对此突发状况，Telegram建议用户暂时将t.me链接替换为telegram.me或telegram.dog。值得注意的是，Instagram平台会屏蔽telegram.me链接，但对telegram.dog则无此限制。Telegram已在其应用和官方网站上主动替换t.me链接，创始人帕维尔·杜罗夫也正尝试通过官方渠道联系.me域名管理机构。
此次禁用背后的原因指向美国财政部海外资产控制办公室（OFAC）的相关要求。尽管Telegram本身并未被OFAC列入制裁名单，但有市场消息指出，一份发布于2026年7月13日、涉及新增制裁企业的声明中，提及了一个名为“FIRST VPN SERVICE”的服务，其网站链接包含了一个位于t.me域名下的Telegram频道地址。这或暗示.me域名运营商在制裁名单中识别到t.me域名后，在未经深入调查的情况下便迅速采取了屏蔽措施。
对此，有观察人士发现，Telegram已于今日早些时候删除了上述制裁名单中提及的频道。目前尚不确定此举是否为导致屏蔽的直接原因，抑或是Telegram的预防性措施。
社区中曾流传一种说法，认为t.me域名被撤销是因Telegram官方Twitter账号发布了涉及9月11日恐怖袭击的玩笑。然而，行业分析师认为此说法可能性较低：若真如此，.me域名管理机构更可能全面屏蔽所有Telegram域名，包括telegram.me。此外，该图片并非公司官方玩笑，而是用户参与竞赛时提交的作品。
