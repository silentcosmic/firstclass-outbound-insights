# 沙迦奇案：男子误认路人妻施暴被捕

🚨 沙迦奇案：男子误认路人妻施暴被捕
沙迦轻罪法院近日正在审理一宗离奇的街头袭击案。一名亚洲籍男子被控对一名素不相识的阿拉伯女子及其子女进行了言语和肢体攻击。
据调查，该男子供称施暴原因竟是错将受害者误认为自己的妻子。这起无端暴力事件已引发当地社区的广泛关注。
