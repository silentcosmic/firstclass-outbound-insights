# 特朗普介入FIFA争议：美队射手红牌获“特赦”

⚡ 特朗普介入FIFA争议：美队射手红牌获“特赦”
据《纽约时报》报道，美国总统特朗普于7月1日致电国际足联（FIFA）主席因凡蒂诺，要求重新审查美国队头号射手巴洛贡的红牌停赛处罚。这一罕见举动引发了外界对体育赛事独立性及其受政治影响的广泛关注。
此前，巴洛贡在美国队2:0战胜波黑队的比赛中因踩踏对方球员被红牌罚下。然而，国际足联纪律委员会于7月5日宣布，依据其纪律准则，对巴洛贡的禁赛处罚将暂缓一年执行。这使得他有资格出战当地时间周一对阵比利时的比赛。值得注意的是，这是自1962年以来，国际足联首次允许一名原本应被停赛的球员继续参赛，此裁决的特殊性不言而喻。
特朗普随后在其社交媒体平台上对国际足联的决定表示欢迎，但并未公开提及此次通话或将该结果归功于自己。
