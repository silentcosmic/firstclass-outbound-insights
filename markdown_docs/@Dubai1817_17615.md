# 跨国诈骗新手法：中国公民被骗200万险遭虚拟绑架

🚨 跨国诈骗新手法：中国公民被骗200万险遭虚拟绑架
近日，一名中国公民在遭遇跨国电信诈骗后，不仅损失高达200万元人民币，更险些在泰国陷入一场由诈骗集团精心策划的“虚拟绑架”骗局。最终，泰国警方在6月29日成功将其安全解救。
据调查，受害人最初在马来西亚接到诈骗电话，对方冒充中国公检法人员，谎称其涉嫌刑事案件，并以此为由诱骗其分四次转账，总计约200万元人民币。随后，诈骗集团又将受害人诱骗至泰国，企图制造“假绑架”事件，向其家人勒索赎金。得益于一名在泰华人提供的关键线索，泰国警方联合多部门迅速行动，在曼谷一家酒店内找到了受害人。
目前，泰国警方正扩大调查范围，全力追查这一涉案的跨国电信诈骗集团，并已扣押受害人手机进行深入取证。
