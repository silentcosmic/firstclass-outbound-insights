# 曼谷素万那普机场突查：K9警犬严防毒品出境

🚨 曼谷素万那普机场突查：K9警犬严防毒品出境
7月8日上午约6时30分，泰国素万那普国际机场启动了一场严密的联合执法行动，旨在加强对航空渠道毒品走私活动的打击力度。
由泰国警方特别行动部门（191）警员、缉毒警犬（K9）以及机场安全保障部门工作人员组成的联合小组，在机场4楼出发大厅对出境旅客及机组人员的行李进行了随机抽查。此次突击检查是机场防范毒品出境措施的重要一环，旨在通过高强度震慑，有效阻断毒品通过航空渠道外流。
