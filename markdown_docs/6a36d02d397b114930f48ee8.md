# 深度解析：某数字银行用户画像揭示，逾四分之一客户年龄在55岁以上

某数字银行的百万客户群体中，逾四分之一的客户年龄在55岁以上，这一数据清晰地表明，数字银行服务的普及已远超传统认知中的年轻用户范畴。
该银行的客户覆盖所有年龄层，其中25至45岁之间的用户占据了客户总数的逾40%。
该数字银行的首席执行官此前曾深入探讨其业务的增长轨迹，以及在新加坡市场吸引各类客户的策略。
截至目前，该银行客户已在全球逾175个国家完成了超过6000万笔境外交易，并累计节省了逾1.42亿新元的境外汇兑费用。
在海外消费额和客户参与度方面，马来西亚位居榜首。其他重要的海外市场包括日本、中国、英国和美国。
通过在日常食品杂货、餐饮和交通等领域的消费，客户已累计获得超过90亿奖励积分及逾2700万新元的现金返还。
该银行已记录了超过43万笔分期付款交易。
这项便捷服务允许客户将消费款项分期支付，其中珠宝是最大的消费类别，紧随其后的是百货商店购物和旅行支出。
客户已自主开立超过15万个“储蓄罐”（Savings Pots），每个“储蓄罐”的平均余额约为3000新元。
此功能旨在帮助客户为旅行、日常账单支付以及未来长远规划等目标，有效地整理和管理储蓄。
该数字银行还创新性地引入了一款生成式人工智能（AI）聊天机器人，使得由人工客服处理的客户聊天量成功减少了50%，客户投诉量也下降了约40%。
该行首席信息官此前曾就银行业加速采纳人工智能技术的必要性进行了深刻阐述。
一位在银行业拥有35年资深经验的从业者指出，行业发展至今，从未面临过像人工智能这样迅猛的变革速度，并认为银行业尚未为此做好充分准备。他详细阐述了其中的深层原因。
该数字银行观察到，客户对在美国上市的股票和交易所交易基金（ETF）表现出早期投资兴趣。
客户已进行了数千笔股票交易，投资对象涵盖英伟达、特斯拉、苹果、Meta和微软等知名公司，同时也包括各类ETF产品。
该数字银行的首席数据与分析官表示：
“数据显示，我们的客户行为模式正在发生显著转变：他们更倾向于亲力亲为、更加注重成本效益，并在资金管理上拥有更明确的目的性。”
