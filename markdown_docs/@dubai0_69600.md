# ️ 吉隆坡营商警示：警务腐败与高物价风险

⚠️ 吉隆坡营商警示：警务腐败与高物价风险
近日有市场消息指出，对于计划在吉隆坡开展业务或已在此地运营的人士，当地的营商环境正面临严峻挑战。高昂的运营成本与普遍存在的警务腐败现象，构成了显著的风险。
据一位近期在吉隆坡的亲历者反馈，即便持有合法签证，仅因手机安装了如Telegram等特定应用程序，亦可能面临被执法人员带往警局的风险。若不愿配合，每人恐需支付高达3至4万马币方可免于扣押。更需警惕的是，支付“保护费”并非一劳永逸的解决方案，执法部门可能更换人员持续勒索，形成“唱双簧”式操作，前期收款者退居幕后，通过提供线索获取分成。
此外，当地的消费市场亦显混乱，例如一次娱乐场所包厢消费最低可达4至5千美元，且不包含其他服务费用。消息人士表示，未来将进一步揭露当地警务人员隐蔽而“教科书级别”的勒索手法。
