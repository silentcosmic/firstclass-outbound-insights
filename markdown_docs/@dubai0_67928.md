# 太子集团核心人物胡希日本落网：涉虚假登记被捕

🚨 太子集团核心人物胡希日本落网：涉虚假登记被捕
日本《朝日新闻》今日披露，日本警视厅已逮捕柬埔寨“太子集团（Prince Group）”的核心骨干、塞浦路斯籍嫌疑人胡希（Hu Xi，44岁）。该集团此前被指为“亚洲最大的国际犯罪组织之一”。胡希涉嫌于今年4月提交虚假的东京都中央区迁入登记，并于6月14日在大阪一家高级酒店被捕。
胡希向警方辩称此举旨在获取日本永住权，但对虚假登记事宜则推诿系由代理人处理。为此，警视厅已逮捕两名中国籍代理人。此案关联到美国财政部于2025年10月对太子集团因网络投资诈骗实施的制裁，当时制裁名单中的“陈小二”已被日本警方认定为胡希的化名。
此前，太子集团董事长陈志已在柬埔寨被捕并引渡回中国。胡希在潜逃日本后，曾频繁变更公司和个人地址以隐匿行踪。目前，日本警方已扣押其手机，正进行全面分析。
