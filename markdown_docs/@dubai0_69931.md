# 马来西亚霹雳州重拳打击电诈：18名中国籍嫌犯落网

🚨 马来西亚霹雳州重拳打击电诈：18名中国籍嫌犯落网
马来西亚霹雳州警方近日通报，怡保警方于7月13日成功突袭两处涉嫌网络诈骗的窝点，共逮捕18名中国籍男子。行动中，警方查扣了包括电脑、手机等在内的电子设备，总值约7.8万令吉。
据悉，被捕人员年龄介于26岁至45岁之间。初步调查显示，这些设备疑被用于操控网络诈骗活动，作案过程可能涉及盗用他人金融工具及银行账户。目前，所有嫌疑人已被警方控制，案件仍在深入调查中。
