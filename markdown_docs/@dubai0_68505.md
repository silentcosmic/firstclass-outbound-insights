# 泰国“杀猪盘”惊现FBI身影，高管急谋出路

🚨 泰国“杀猪盘”惊现FBI身影，高管急谋出路
市场消息披露，一家在泰国运营一年半、专门针对北美地区实施“杀猪盘”骗局的公司，正面临美国联邦调查局（FBI）的关注。该公司主要雇佣东南亚籍员工，以利用其英文优势与受害者沟通。
据内部员工透露，6月26日，一名泰国本地视频模特向中方管理层反映，其家人被当地警方及一名疑似美国便衣警察登门拜访，要求其配合调查数起针对美国公民的网络诈骗案。
该消息迅速在公司内部引发震动，有中方管理层成员立即萌生了撤离并转移部分资金的念头。据称，该公司在过去一年半内已创下逾2200万美元的营收。
该内部人士表示，在东南亚从事此类活动，常年面临当地及国际执法机构的追查，生存压力巨大，并已准备好在必要时披露内部信息以求自保。
