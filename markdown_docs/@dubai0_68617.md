# ️ 金边法院重判：20名园区保安涉非法武装获刑

⚖️ 金边法院重判：20名园区保安涉非法武装获刑
金边法院近日对一起涉嫌组建非法外籍武装力量案作出宣判，共21名男子被判处2至5年有期徒刑，其中包括1名中国籍、19名缅甸籍及1名柬埔寨籍人员。
主犯为42岁中国籍男子林景润，系贡布省波哥市某中资赌场前保安主管，他与19名缅甸籍赌场保安均被判处5年有期徒刑，刑满后将立即驱逐出境。涉案的柬埔寨籍大巴司机获刑2年。此外，另一名中国籍涉案男子李永缺席审判，被判5年并遭全国通缉。
据了解，该团伙于去年5月在前往蒙多基里省途中被警方拦截逮捕，当时他们身穿迷彩军装，携带对讲机及电警棍等装备。尽管嫌犯坚称是听从赌场老板指令，但法院最终认定其非法组建武装力量罪名成立。
